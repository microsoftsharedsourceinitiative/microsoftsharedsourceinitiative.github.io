MEDIA_SERIAL_NUMBER_DATA

The MEDIA_SERIAL_NUMBER_DATA structure is used in conjunction with the IOCTL_STORAGE_GET_MEDIA_SERIAL_NUMBER request to retrieve the serial number of a USB device. 

typedef struct _MEDIA_SERAL_NUMBER_DATA {
  ULONG  SerialNumberLength; 
  ULONG  Result;
  ULONG  Reserved[2];
  UCHAR  SerialNumberData[0];
} MEDIA_SERIAL_NUMBER_DATA, *PMEDIA_SERIAL_NUMBER_DATA;

Members

SerialNumberLength
Contains the size in bytes of the SerialNumberData array.
Result
Contains the status of the completed IOCTL_STORAGE_GET_MEDIA_SERIAL_NUMBER request.
Reserved
Reserved.
SerialNumberData
Contains a variable-sized array of characters that holds the serial number of the device.

Headers

Declared in ntddstor.h. Include ntddstor.h.
