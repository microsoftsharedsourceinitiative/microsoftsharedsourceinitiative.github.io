ScsiPortWmiGetReturnStatus

The ScsiPortWmiGetReturnStatus routine returns the status of a WMI SRB. 

UCHAR 
  ScsiPortWmiGetReturnStatus(
    PSCSIWMI_REQUEST_CONTEXT  RequestContext
    );

Parameters

RequestContext
Pointer to the request context for this SRB.

Return Value

The return value represents the return status for the SRB. The miniport driver should set Srb->SrbStatus to the value returned by ScsiPortWmiGetReturnStatus before completing the SRB.

Headers

Declared in srb.h. Include miniport.h and scsi.h.

Comments

A miniport driver must call ScsiPortWmiGetReturnStatus to obtain the value to put into Srb->SrbStatus. ScsiPortWmiGetReturnStatus should be called sometime after the miniport driver calls ScsiPortWmiPostProcess.
