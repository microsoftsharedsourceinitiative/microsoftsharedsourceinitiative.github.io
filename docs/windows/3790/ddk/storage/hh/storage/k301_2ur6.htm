ScsiPort Routines

An OS-specific SCSI port driver provides ScsiPortXxx routines to support OS-independent SCSI miniport drivers that are linked with a Microsoft-supplied, OS-specific SCSI port driver.

The SCSI port driver is a kernel-mode dynamic-link library.

References for the ScsiPortXxx routines are in alphabetical order.
