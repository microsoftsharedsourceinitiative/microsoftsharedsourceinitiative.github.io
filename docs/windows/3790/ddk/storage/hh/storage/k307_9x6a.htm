Storage Devices: Windows DDK

IOCTL_STORAGE_CHECK_VERIFY2

Operation

Determines whether the media has changed on a removable-media device which the caller has opened with FILE_READ_ATTRIBUTES. Because no file system is mounted when a device is opened in this way, this request can be processed much more quickly than an IOCTL_STORAGE_CHECK_VERIFY request.

Input

Input is identical to the input for IOCTL_STORAGE_CHECK_VERIFY.

Output

Output is identical to the output for IOCTL_STORAGE_CHECK_VERIFY.

I/O Status Block

I/O status is identical to the I/O status for IOCTL_STORAGE_CHECK_VERIFY.

Headers

Defined in ntddstor.h. Include ntddstor.h.
