ScsiPortWritePortUshort

The ScsiPortWritePortUshort routine transfers a USHORT value to the HBA.

VOID 
  ScsiPortWritePortUshort(
    IN PUSHORT  Port,
    IN USHORT  Value
    );

Parameters

Port
Pointer to the I/O port. The given Port must be in a mapped I/O-space range returned by ScsiPortGetDeviceBase.
Value
Specifies the value to be written to the HBA's I/O port.

Return Value

None

Headers

Declared in srb.h. Include miniport.h and scsi.h.
