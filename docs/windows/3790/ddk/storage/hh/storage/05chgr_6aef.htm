Storage Devices: Windows DDK

Sample Changer Miniclass Drivers

This section identifies the sample changer drivers included with the Windows DDK. These samples only function on Windows 2000 and later operating systems.

The Windows DDK includes the source code for the mcd.sys changer class driver and the following miniclass drivers:

  • adicsc.sys, the ADIC Scalar changer miniclass driver sample
  • ddsmc.sys, the DDS 4mm changer miniclass driver sample
  • elmsmc.sys, the Elms changer miniclass driver sample
  • exabyte.sys, the Exabyte changer miniclass driver sample
  • hpmc.sys, the Hewlett-Packard changer miniclass driver sample
  • nsmmc.sys, the NSM changer miniclass driver sample
  • sonymc.sys, the Sony changer miniclass driver sample

Source code for all changer driver samples is located in the \src\storage directory of the Windows DDK.
