FEATURE_DATA_CORE

The FEATURE_DATA_CORE structure holds data for the Core feature descriptor.

typedef struct _FEATURE_DATA_CORE {
  FEATURE_HEADER  Header;
  UCHAR  PhysicalInterface[4];
} FEATURE_DATA_CORE, *PFEATURE_DATA_CORE;

Members

Header
Contains a FEATURE_HEADER structure with header information for this feature descriptor.
PhysicalInterface
Must be set to the current communication path between initiator and device, as defined in the SCSI Multimedia - 3 (MMC-3) specification. The bytes of this array are arranged in big-endian order. PhysicalInterface[0] contains the most significant byte, and PhysicalInterface[3] contains the least significant byte.

Headers

Declared in ntddmmc.h. Include ntddmmc.h.

Comments

Indicates the feature named "Core" by the MMC-3 specification. This feature encompasses the basic functionality which is mandatory for all devices that support the MMC-3 standard. See the MMC-3 specification for a description of the capabilities included in the Core feature.
