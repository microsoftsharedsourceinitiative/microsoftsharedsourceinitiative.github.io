SCSI_ADAPTER_BUS_INFO

The SCSI_ADAPTER_BUS_INFO structure is used in conjunction with the IOCTL_SCSI_GET_INQUIRY_DATA request to retrieve the SCSI inquiry data for all devices on a given SCSI bus. 

typedef struct _SCSI_ADAPTER_BUS_INFO {
  UCHAR  NumberOfBuses;
  SCSI_BUS_DATA  BusData[1];
} SCSI_ADAPTER_BUS_INFO, *PSCSI_ADAPTER_BUS_INFO;

Members

NumberOfBuses
Contains the number of buses on the adapter for which inquiry data is being reported.
BusData
Pointer to a variable length buffer containing the inquiry data.

Headers

Declared in ntddscsi.h. Include ntddscsi.h.

Comments

The SCSI_ADAPTER_BUS_INFO structure is a header structure that is used to report inquiry data in the variable length buffer pointed to by the BusData member. The inquiry data, pointed to by BusData, consists of a series of NumberOfBuses-1 structures of type SCSI_BUS_DATA. Each of these structures is followed by the number equal to the value of NumberOfLUs structures of type SCSI_INQUIRY_DATA, where NumberOfLUs is a member of SCSI_BUS_DATA. See IOCTL_SCSI_GET_INQUIRY_DATA for further details concerning how the data in the buffer at BusData is laid out.
