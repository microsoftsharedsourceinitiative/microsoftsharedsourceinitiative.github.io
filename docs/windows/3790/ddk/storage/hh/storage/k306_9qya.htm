FEATURE_DATA_DVD_PLUS_RW

The FEATURE_DATA_DVD_PLUS_RW structure contains information about the DVD+RW feature. 

typedef struct _FEATURE_DATA_DVD_PLUS_RW {
  FEATURE_HEADER Header;
  UCHAR  Write:1;
  UCHAR  Reserved1:7;
  UCHAR  Reserved2[3];
} FEATURE_DATA_DVD_PLUS_RW, *PFEATURE_DATA_DVD_PLUS_RW;

Members

Header
Contains a FEATURE_HEADER structure with header information for this feature descriptor.
Write
Indicates, when set to 1, that the device can do background formatting of DVD+RW discs according to DVD+RW 4.7 Gbytes Basic Format Specifications, and can write to discs that have been formatted in this manner.
Reserved1
Reserved.
Reserved2
Reserved.

Headers

Declared in ntddmmc.h. Include ntddmmc.h.

Comments

This structure holds data for the feature named "DVD+RW" by the SCSI Multimedia - 3 (MMC-3) specification. Devices that support this feature can read a recorded DVD+RW disc that is formatted according to the DVD+RW 4.7 Gbytes Basic Format Specification.
