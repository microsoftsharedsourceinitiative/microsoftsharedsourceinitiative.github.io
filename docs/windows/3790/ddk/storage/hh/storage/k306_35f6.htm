STOR_PHYSICAL_ADDRESS

The SCSI_PHYSICAL_ADDRESS type is an operating system-independent data type that Storport miniport drivers use to represent either a physical addresses or a bus-relative address.

The following routines use this data type to represent physical addresses:

  • StorPortConvertPhysicalAddressToUlong64
  • StorPortConvertUlongToPhysicalAddress
  • StorPortConvertUlong64ToPhysicalAddress
  • StorPortGetPhysicalAddress
  • StorPortGetVirtualAddress

The following routines use this data type to represent bus-relative addresses:

  • StorPortValidateRange
