StorPortConvertUlong64ToPhysicalAddress

The StorPortConvertUlong64ToPhysicalAddress routine converts a ULONG64 value to a physical address.

STOR_PHYSICAL_ADDRESS
  StorPortConvertUlong64ToPhysicalAddress(
    IN ULONG64  UlongAddress
    );

Parameters

UlongAddress
Contains the ULONG64 value that is to be converted to a physical address.

Return Value

StorPortConvertUlong64ToPhysicalAddress returns the physical address corresponding to the input value.

Headers

Declared in storport.h. Include storport.h.

Comments

All addresses in Storport are expressed as 64-bit values.
