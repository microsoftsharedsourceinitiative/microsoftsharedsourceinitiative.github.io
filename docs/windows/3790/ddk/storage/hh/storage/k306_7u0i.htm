VERIFY_INFORMATION

The VERIFY_INFORMATION structure provides information used to logically format a disk extent. 

typedef struct _VERIFY_INFORMATION {
  LARGE_INTEGER  StartingOffset;
  DWORD  Length;
} VERIFY_INFORMATION, *PVERIFY_INFORMATION;

Members

StartingOffset
Specifies the starting offset, in bytes, of the disk extent.
Length
Indicates the length, in bytes, of the disk extent.

Headers

Declared in ntdddisk.h. Include ntdddisk.h.

Comments

VERIFY_INFORMATION is the output buffer for the IOCTL_DISK_VERIFY control code.
