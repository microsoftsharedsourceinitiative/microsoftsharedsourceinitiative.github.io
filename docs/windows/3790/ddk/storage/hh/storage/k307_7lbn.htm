Storage Devices: Windows DDK

IOCTL_STORAGE_EJECT_MEDIA

Operation

If the device supports ejection capabilities, this IOCTL causes the device to eject the media.

Input

None

Output

None

I/O Status Block

The Information field is set to zero. The Status field is set to STATUS_SUCCESS, or possibly to STATUS_NO_MEDIA_IN_DEVICE or STATUS_INVALID_DEVICE_REQUEST.

Headers

Defined in ntddstor.h. Include ntddstor.h.
