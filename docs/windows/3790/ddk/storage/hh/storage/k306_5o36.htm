SCSI Bus Structures

This section describes structures used by SCSI miniport drivers that are linked with a Microsoft-supplied, operating-system-specific SCSI port driver. SCSI miniport drivers drive SCSI adapters.

This section also contains structures used by storage class, miniclass, and filter drivers. For further information on storage structures, see Storage Structures.

Structures described in this section are in alphabetical order.
