IOCTL_DVD_GET_REGION

Operation

Returns Region Playback Control (RPC) information for a DVD device, such as whether the player supports the RPC2 standard, the current region code of the player, and the remaining number of times the player's region code can be changed by the user. It also indicates the region code of the currently mounted disc.

Input

None. Parameters.DeviceIoControl.OutputBufferLength indicates the size in bytes of the buffer at Irp->AssociatedIrp.SystemBuffer, which must be >= sizeof(DVD_REGION).

Output

The driver returns the DVD_REGION data in the buffer at Irp->AssociatedIrp.SystemBuffer.

I/O Status Block

The Information field is set to the number of bytes transferred. The Status field is set to STATUS_SUCCESS or possibly STATUS_INSUFFICIENT_RESOURCES.

Headers

Defined in ntddcdvd.h. Include ntddcdvd.h.
