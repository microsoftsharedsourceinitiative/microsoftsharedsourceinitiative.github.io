Storport Routines

The Storport driver fulfills much the same function as the SCSI port driver, while supplying a new I/O model tailored specifically for high-performance buses. Many of the SCSI port driver support routines have been rewritten to accomodate the new I/O model, and new routines have been designed. The guidelines for writing miniport driver routines have been modified as well.
