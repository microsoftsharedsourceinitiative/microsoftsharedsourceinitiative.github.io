Writing a Co-installer

A co-installer is a Microsoft® Win32® DLL that assists in device installation. Co-installers are called by Setup API as "helpers" for Class Installers. For example, a vendor can provide a co-installer to write device-specific information to the registry that cannot be handled by the INF file.
