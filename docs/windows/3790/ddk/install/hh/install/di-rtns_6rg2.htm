Device Installation: Windows DDK

SetupDiSetClassRegistryProperty

(Windows XP and later)

The SetupDiSetClassRegistryProperty function sets a specified device class property in the registry.

WINSETUPAPI BOOL WINAPI
  SetupDiSetClassRegistryProperty(
    IN  LPGUID ClassGuid,
    IN  DWORD Property,
    IN CONST BYTE* PropertyBuffer,  OPTIONAL
    IN  DWORD PropertyBufferSize,
    IN  PCTSTR MachineName,         OPTIONAL
    IN  PVOID Reserved
    );

Parameters

ClassGuid
Caller-supplied pointer to a GUID representing the device class for which a property is to be set.
Property
Caller-supplied value identifying the property to be set. This must be one of the following values:
SPCRP_SECURITY
The caller supplies the device's security descriptor as a SECURITY_DESCRIPTOR structure (described in Platform SDK documentation).
SPCRP_SECURITY_SDS
The caller supplies the device's security descriptor as a text string. The format of security descriptor strings is described in Platform SDK documentation.
SPCRP_DEVTYPE
The caller supplies the device type for the class. For more information, see Specifying Device Types.
SPCRP_EXCLUSIVE
The caller supplies a DWORD value indicating whether users can obtain exclusive access to devices for this class. The supplied value is 1 if exclusive access is allowed, or zero otherwise.
SPCRP_CHARACTERISTICS
The caller supplies flags indicating device characteristics for the class. For a list of characteristics flags, see the DeviceCharacteristics parameter to IoCreateDevice.
PropertyBuffer
Caller-supplied pointer to a buffer that supplies the specified property.
PropertyBufferSize
Caller-supplied size of the buffer pointed to by PropertyBuffer.
MachineName
Caller-supplied name of a remote machine on which to set the specified device class property. If NULL, the property is set on the local machine.
Reserved
Reserved, must be NULL.

Return Value

The function returns TRUE if it is successful. Otherwise, it returns FALSE and the logged error can be retrieved with a call to GetLastError.

Headers

Declared in setupapi.h. Include setupapi.h.

Comments

This function is supported in Windows XP and later operating system versions.

The caller of this function must be a member of the Administrators group.

To determine the data type for a device class property, call SetupDiGetClassRegistryProperty.
