Device Installation: Windows DDK

CM_PROB_NO_SOFTCONFIG

The device requires a forced configuration.

Error Code

34

Display Message – NT-based Systems

"Windows cannot determine the settings for this device. Consult the documentation that came with this device and use the Resource tab to set the configuration. (Code 34)"

Recommended Resolution – NT-based Systems

Change the HW settings (by setting jumpers or running a vendor-supplied utility) and then use the Device Manager's Resources tab to set the forced configuration.

Display Message – Windows 98/Me

Not used.

Recommended Resolution – Windows 98/Me

Not used.
