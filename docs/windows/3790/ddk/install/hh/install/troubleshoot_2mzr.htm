Device Installation: Windows DDK

CM_PROB_FAILED_POST_START

A driver has reported a device failure. (Windows XP and later)

Error Code

43

Display Message – NT-based Systems

"Windows has stopped this device because it has reported problems. (Code 43)"

Recommended Resolution – NT-based Systems

Uninstall and reinstall the device.

One of the drivers controlling the device told the OS the device failed in some manner in response to IRP_MN_QUERY_PNP_DEVICE_STATE.

Display Message – Windows 98/Me

Not used.

Recommended Resolution – Windows 98/Me

Not used.
