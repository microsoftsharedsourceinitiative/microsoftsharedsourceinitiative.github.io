SetupDiSetSelectedDevice

The SetupDiSetSelectedDevice function sets the specified device information element to be the currently-selected member of a device information set. This function is typically used by an installation wizard.

WINSETUPAPI BOOL WINAPI
  SetupDiSetSelectedDevice(
    IN HDEVINFO  DeviceInfoSet,
    IN PSP_DEVINFO_DATA  DeviceInfoData
    );

Parameters

DeviceInfoSet
Supplies a handle to the device information set for which the selected device should be set.
DeviceInfoData
Supplies a pointer to an SP_DEVINFO_DATA structure that specifies the device information element to select.

Return Value

The function returns TRUE if it is successful. Otherwise, it returns FALSE and the logged error can be retrieved with a call to GetLastError.

Headers

Declared in setupapi.h. Include setupapi.h.
