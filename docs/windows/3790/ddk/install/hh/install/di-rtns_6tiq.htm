SetupDiGetClassRegistryProperty

(Windows XP and later)

The SetupDiGetClassRegistryProperty function retrieves a property, for a specified device setup class, from the registry.

WINSETUPAPI BOOL WINAPI
  SetupDiGetClassRegistryProperty(
    IN LPGUID  ClassGuid,
    IN DWORD  Property,
    OUT PDWORD  PropertyRegDataType,  OPTIONAL
    OUT PBYTE  PropertyBuffer,
    IN DWORD  PropertyBufferSize,
    OUT PDWORD  RequiredSize,  OPTIONAL
    IN PCTSTR  MachineName,  OPTIONAL
    IN PVOID  Reserved
    );

Parameters

ClassGuid
Caller-supplied pointer to a GUID representing the device setup class for which a property is to be retrieved.
Property
Caller-supplied value identifying the property to be retrieved. This must be one of the following values:
SPCRP_SECURITY
The function returns the device's security descriptor as a SECURITY_DESCRIPTOR structure (described in Platform SDK documentation).
SPCRP_SECURITY_SDS
The function returns the device's security descriptor as a text string. The format of security descriptor strings is described in Platform SDK documentation.
SPCRP_DEVTYPE
The function returns the device type for the class. For more information, see Specifying Device Types.
SPCRP_EXCLUSIVE
The function returns a DWORD value indicating whether users can obtain exclusive access to devices for this class. The returned value is one if exclusive access is allowed, or zero otherwise.
SPCRP_CHARACTERISTICS
The function returns flags indicating device characteristics for the class. For a list of characteristics flags, see the DeviceCharacteristics parameter to IoCreateDevice.
PropertyRegDataType
(Optional) Caller-supplied pointer to a location that receives the property's data type as one of the REG_-prefixed registry data types. If NULL, the data type is not returned.
PropertyBuffer
Caller-supplied pointer to a buffer that receives the requested property.
PropertyBufferSize
Caller-supplied size of the buffer pointed to by PropertyBuffer.
RequiredSize
(Optional) Caller-supplied pointer to receive the required buffer size. If the buffer pointed to by PropertyBuffer is too small, and RequiredSize is not NULL, the function supplies the minimum buffer size required to receive the requested property.
MachineName
(Optional) Caller-supplied name of a remote machine from which to retrieve the specified device class property. If NULL, the property is retrieved from the local machine.
Reserved
Reserved, must be NULL.

Return Value

The function returns TRUE if it is successful. Otherwise, it returns FALSE and the logged error can be retrieved with a call to GetLastError.

Headers

Declared in setupapi.h. Include setupapi.h.

Comments

This function is supported by Windows XP and later operating system versions.
