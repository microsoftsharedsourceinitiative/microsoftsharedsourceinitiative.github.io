Device Classes

This section provides information about two device classes used to place devices and drivers into groups whose members have similar characteristics:

Device Setup Classes
Used to group together devices that are installed and configured in a similar manner.
Device Interface Classes
Used to group together devices that provide similar capabilities.

Because of similarities in the way both types of device classes are specified and used, the section provides a topic entitled Setup Classes Versus Interface Classes.

Setup functions use device information sets to expose the members of a class to user-mode code.
