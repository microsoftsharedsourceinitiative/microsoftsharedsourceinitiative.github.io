New for Still Image Drivers

Windows XP supports a new imaging architecture called Windows Image Acquisition (WIA). Vendors of imaging devices, such as digital cameras and scanners, should see Windows Image Acquisition Drivers for details on providing drivers for this new architecture.

The WIA Video Snapshot filter is an addition to WIA that ships with Windows XP and later operating systems. It is a filter driver that enables frames to be captured from the video stream. To capture high-resolution stills from a video stream, still pin support must be implemented in the video capture source filter. For more information, see USB Camera with a Still Pin.
