New for the Build Utility

Windows XP has many new features in the Build utility.

The Build utility should be used to compile and build all drivers that will run on NT-based operating systems, as well as on Windows 98/Me. The documentation of the Build utility has been completely revised.

For full details, see Using the Build Utility and Build Utility Reference.
