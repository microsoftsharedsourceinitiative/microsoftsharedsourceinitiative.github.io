Windows DDK

New for Kernel Streaming Drivers

In Windows XP SP1, the AVStream reference documentation has been enhanced and expanded. For more information, refer to AVStream Functions and AVStream Interfaces.

The WDM audio driver design guide now describes wave and MIDI filters in more detail. For more information, see Wave Filters and MIDI and DirectMusic Filters.

The Windows DDK now contains a sample driver for a GFX filter. For more information, see Sample Audio Drivers.

Support for video encoders has also been added. For more information, see Encoder Minidrivers.
