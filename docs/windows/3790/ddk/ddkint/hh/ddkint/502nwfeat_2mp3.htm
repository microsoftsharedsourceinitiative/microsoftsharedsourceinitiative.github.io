Windows DDK

New for Device Installation

INF files in Windows XP support the following new features:

  • Decorations for installations specific to 64-bit platforms. For more information, see Creating INF Files for Multiple Platforms and Operating Systems.
  • Operating system version numbers in INF Manufacturer sections.
  • An INF CopyINF directive, which is useful for copying INF files for multifunction devices.

Setup introduces "trusted" and "untrusted" hardware ID matches. For more information, see How Setup Selects Drivers.

Setup provides additional logging options. For more information, see Using SetupAPI Logging.

The Device Manager provides a new Details tab and new error messages. For more information, see Using the Device Manager.
