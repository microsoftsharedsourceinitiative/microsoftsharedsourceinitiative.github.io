Windows DDK

Using the DDK Viewer

DDK documentation uses the viewer supplied with Microsoft HTML Help (hh.exe). This viewer provides a table of contents and an index. Additionally, the viewer enables you to search for text strings, print one or several pages, and create subsets that hide topics you don't want to see.

The Contents Tab

The Contents tab provides an expandable view of the DDK documentation's table of contents. This view provides a single top node, called "Windows DDK."

The DDK is organized as a set of "books," and clicking the "Windows DDK" node's icon exposes a set of nodes representing these books. Each book contains one of the DDK's major topics.

Most books are divided into "Design Guide" and "Reference" sections.

The icon for each table of contents node is clickable. Some nodes display text when their icons are clicked; others just expand to a set of subnodes, and others do both.

The type of icon displayed for a table of contents node usually does not indicate whether clicking will display text, subnodes, or both.

The Index Tab

The Index tab displays the DDK documentation index. This index contains the following:

  • The title of every topic contained in the documentation
  • The name of every function, structure, IRP, IOCTL, or other item for which a "reference" page exists
  • Every glossary entry

The Search Tab

The Search tab gives you the ability to search for any text string contained in the documentation. You can search all text, or limit your search to topic titles. You can search for strings that are exact matches, strings that are similar to a specified string, or strings that are near other strings.

To search for strings of more than one word, enclose the string with quotation marks ("). You can refine your search by using the AND, OR, NOT, and NEAR operators. Use a question mark (?) as a single wildcard character, or an asterisk (*) to represent one or more wildcard characters, anywhere within a non-quoted string.

The Favorites Tab

The Favorites Tab lets you save the titles of commonly visited topics. While viewing a topic, select the Favorites tab and click the Add button. Later, you can use the Remove button to remove topics. You can also right-click entries in the Favorites window to rename them.

Printing Topics

To print one or more topics, select a topic icon on the Contents tab. Then click the Print button on the taskbar. You will be asked whether you'd like to print just the selected topic, or that topic and all of its subtopics. You can also print the topic you are viewing by right-clicking within the view window, and selecting Print.

Searching within a Topic

To search for a text string within the topic you are viewing, press CTRL+F, or select Find in this Topic from the Edit menu.

Creating Documentation Subsets

If you don't want to view the entire documentation set, you can create a subset by selecting Define Subset from the View menu. You can select the set of books you'd like the subset to contain, then assign a name to the subset. Then you can select the subset using the Active Subset dropdown list.

After you have selected a subset, only the books in the subset are available when you use the Contents, Index, and Search tabs.
