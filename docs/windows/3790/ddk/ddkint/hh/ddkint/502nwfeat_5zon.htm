New for Kernel Streaming Drivers

In Windows Server 2003, the documentation for Kernel Streaming and AVStream design has been significantly enhanced. The following topics are new:

AVStream Allocators

AVStream Child Devices

AVStream Clocks

AVStream Pipes and Circuits

New properties and methods are available in:

Kernel Streaming Method Sets

Kernel Streaming Property Sets

The reference section for Kernel Streaming Interface Sets is new.

In Stream Class Design documentation, the topic on Stream Class Registry Values is new.

The WDM audio driver design guide now contains more detailed guidelines for developing WavePci devices and miniport drivers. For more information, see Implementation Issues for WavePci Devices.
