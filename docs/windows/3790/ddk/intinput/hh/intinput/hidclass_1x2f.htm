Single-Interface USB or Non-USB Device That Supports Only One Top-Level Collection

The following hardware ID formats are used if the parent input device is enumerated as a single-interface USB device (or a non-USB device) and the parent device supports only one top-level collection:

  • Vendor Hardware ID Formats

    HID\Vid_v(4)&Pid_d(4)&Rev_r(4)

    HID\Vid_v(4)&Pid_d(4)

  • System Hardware ID Formats (Internal Use Only)

    HID_DEVICE_UP:p(4)_U:u(4)

    HID_DEVICE

The fields that are in italics are device-specific values which are defined in Hardware ID Field Definitions for Top-Level Collections.
