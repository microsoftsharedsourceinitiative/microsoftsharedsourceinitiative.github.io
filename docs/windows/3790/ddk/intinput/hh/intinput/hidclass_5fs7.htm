Interactive Input Devices: Windows DDK

Sample HID Minidrivers for Tablet PC Input Devices

The DDK includes the following sample HID minidrivers that demonstrate how to support pen devices and button devices that are installed on a Tablet PC. For more information about these sample drivers, see the readme.htm file in the src\wdm\hid\tablet directory of the DDK.

  • The HBtnKey driver is a HID minidriver for a button device that uses an i8042 port-compatible interface. The sample code for the HBtnKey driver is located in the src\wdm\hid\tablet\hbtnkey directory of the DDK. The corresponding sample INF file is hidbtn.inf, which is located in the src\wdm\hid\tablet\inf directory.
  • The MSTabBtn driver is the HID mindriver for a button device that is accessed through a 32-bit register in I/O space. The MSTabBtn driver is located in the src\wdm\hid\tablet\hbtnkey directory of the DDK. The corresponding sample INF file is hidbtn.inf, which is located in the src\wdm\hid\tablet\inf directory.
  • The MutohPen and WacomPen drivers are HID minidrivers for pen devices that use a 16550 UART-compatible interface. The sample code for the MutohPen driver is located in the src\wdm\hid\tablet\mutohpen directory of the DDK samples. The sample code for a WaComPen device is located in the src\wdm\hid\tablet\wacompen directory of the DDK. The corresponding sample INF file for both types of devices is hiddigi.inf, which is located in the src\wdm\hid\tablet\inf directory.
