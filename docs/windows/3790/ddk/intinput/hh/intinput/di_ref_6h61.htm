DirectInput Interfaces

Microsoft® DirectInput® defines the following interfaces:

  • IDirectInputEffectDriver. This interface exposes methods used by DirectInput for user-mode force feedback.
  • IDirectInputJoyConfig8. This interface exposes methods that allow hardware developers who are writing property sheets to get information in and out of the registry.
  • IDIGameCntrlPropSheet This interface exposes methods that define Control Panel property sheet.
