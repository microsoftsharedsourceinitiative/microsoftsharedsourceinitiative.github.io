HID Class-Specific Descriptors

A HID class-specific descriptor contains configuration information about a HIDClass device. Vendors use HID class descriptors to store device-specific information in the firmware of a device. A user-mode application or kernel-mode driver obtains different types of HID class descriptors by querying the HID class driver.
