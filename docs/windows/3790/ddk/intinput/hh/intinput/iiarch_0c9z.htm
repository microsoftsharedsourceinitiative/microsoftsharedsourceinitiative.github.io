HID Mapper Drivers

A HID mapper driver is a kernel-mode WDM filter driver that provides a bidirectional interface for I/O requests between a non-HIDClass driver and the HID class driver. The mapper driver maps the I/O requests and data protocols of one to the other.

The system-supplied HID mapper drivers for HID keyboard, mouse, and game port devices listed in the following table.

Windows Version Keyboard Mouse Game Port
Windows 2000 and later versions kbdhid.sys mouhid.sys None
Windows Me kbdhid.vxd mouhid.vxd joyhid.vxd
