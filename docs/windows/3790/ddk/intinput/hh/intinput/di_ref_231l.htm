IDIGameCntrlPropSheet

The IDIGameCntrlPropSheet interface consists of the following methods:

GetSheetInfo

GetPageInfo

GetID

SetID

In addition, all COM interfaces inherit the following IUnknown interface methods:

AddRef

QueryInterface

Release

These three methods allow additional interfaces to be added to the DirectInput game controller property sheet object without affecting the functionality of the original interface.
