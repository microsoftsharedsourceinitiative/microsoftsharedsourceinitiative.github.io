Hardware Capabilities Callback

int __stdcall HWCapsRoutine( int joyid, LPJOYOEMHWCAPS pjhwc );

HWCapsRoutine is called whenever the hardware capabilities of the device are requested. In particular, VJoyD calls HWCapsRoutine before returning from VJOYD_Register_Device_Driver, so it is important that any initialization upon which this call relies is complete before the device is registered. These values are usually constants. For a device that has four buttons and three axes, of which the third may return either a throttle or a rudder value, the following would be appropriate: 

pjhwc->dwMaxButtons = 4;    /* This should always be the number of buttons */
pjhwc->dwMaxAxes = 4;       /* The largest axis number which may be requested */
pjhwc->dwNumAxes = 3;       /* The number of axes the device has */
