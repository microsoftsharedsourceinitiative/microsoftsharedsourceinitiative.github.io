Operation of a HID Minidriver

A HID minidriver abstracts the operation of a hardware bus or port that an input device is attached to. After the HID minidriver registers with the HID class driver, the HID class driver communicates with the minidriver by calling the minidriver's standard driver routines.

For more information about the functionality that a HID minidriver must support in its standard driver routines, see the following topics:
