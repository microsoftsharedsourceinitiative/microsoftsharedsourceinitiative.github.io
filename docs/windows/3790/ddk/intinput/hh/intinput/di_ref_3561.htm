IDIGameCntrlPropSheet::AddRef

The IDIGameCntrlPropSheet::AddRef method increases the reference count of the DIGameCntrlPropSheet object by 1. This method is part of the IUnknown interface inherited by DIGameCntrlPropSheet. 

HRESULT
  IDIGameCntrlPropSheet::AddRef( 
    );

Parameters

None

Return Value

Returns the new reference count of the object.

Comments

When the DIGameCntrlPropSheet object is created, its reference count should be set to 1. Every time an application obtains an interface to the object or calls the AddRef method, the object's reference count is increased by 1. The application uses the Release method to decrease the reference count of the object by 1.
