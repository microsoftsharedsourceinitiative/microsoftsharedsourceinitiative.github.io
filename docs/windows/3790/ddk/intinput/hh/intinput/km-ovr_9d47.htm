Non-HIDClass Keyboard and Mouse Devices

This section includes the following topics about the Microsoft® Windows® 2000 and later system drivers for non-HID keyboard and mouse devices:

  • The features and standard configurations of the following drivers:
    • Kbdclass, the class driver for devices of GUID_CLASS_KEYBOARD device class
    • Mouclass, the class driver for devices of GUID_CLASS_MOUSE device class
    • I8042prt, the function driver for a PS/2-style keyboard device and a PS/2-style mouse device
  • Using an optional upper-level device filter driver to customize the operation of a PS/2-style keyboard or mouse device
  • The features of Kbfiltr, a sample upper-level keyboard filter driver in the Windows DDK
  • The features of Moufiltr, a sample upper-level mouse filter driver in the Windows DDK

This section does not cover detailed information about Sermouse, the Windows system function driver for a serial mouse. Note that the operational constraints that apply to I8042prt do not apply to Sermouse. In addition, upper-level device filter drivers are not used with Sermouse to customize the operation of a serial mouse. Instead, vendors need to install a device-specific function driver for the device. A device-specific function driver and Sermouse can operate at the same time, independent of one another.
