HID_DESCRIPTOR

The HID_DESCRIPTOR structure represents a HID descriptor for a HIDClass device.

typedef struct _HID_DESCRIPTOR {
  UCHAR  bLength;
  UCHAR  bDescriptorType;
  USHORT  bcdHID;
  UCHAR  bCountry;
  UCHAR  bNumDescriptors;
    struct _HID_DESCRIPTOR_DESC_LIST {
      UCHAR  bReportType;
      USHORT  wReportLength;
    } DescriptorList [1];
} HID_DESCRIPTOR, *PHID_DESCRIPTOR;

Members

For information about HID descriptors, see the Universal Serial Bus (USB) standard Device Class Definition for Human Interface Devices (HID) located at the USB Implementers Forum Web site.

Headers

Declared in hidport.h. Include hidport.h.

Comments

The HID class driver uses an IOCTL_HID_GET_DEVICE_DESCRIPTOR request to obtain a device's HID descriptor from a HID minidriver.
