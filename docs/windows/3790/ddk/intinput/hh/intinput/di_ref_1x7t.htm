VJOYD_GetPosEx_Service

The VJOYD_GetPosEx_Service macro allows kernel-mode components to get joystick data, thereby bypassing WinMM. 

VJOYDAPI_GetPosEx_Service, SERVICE
ENTRY:
eax = pointer to JOYINFOEX

ebx = joystick ID

EXIT:
eax = joystick return code
