HID Support Routines and IOCTLs

User-mode applications and kernel-mode drivers do the following to operate HID collections:

  • User-mode and kernel-mode drivers use HidD_Xxx HIDClass support routines to obtain information about a HID collection.
  • User-mode applications use HidP_Xxx HID support routines and kernel-mode drivers use HID class driver IOCTLs to handle HID reports.

For more information, see Operating HID Collections.
