VJOYD_OpenTypeKey_Service

The VJOYD_OpenTypeKey_Service macro is used by joystick drivers to open a type key. It should mirror the IDirectInputJoyConfig8::OpenTypeKey method. 

VJOYD_OpenTypeKey_Service, SERVICE
ENTRY:
eax = pointer to name of type

ecx = pointer to DWORD in which to store the key

EXIT:
eax = same as _RegOpenKey
