IRP_MJ_DEVICE_CONTROL (I8042prt Keyboard)

When Sent

The Power Manager uses device control requests to determine which power buttons a keyboard supports and to be notified of power-related button events.

Input Parameters

Request-specific.

Output Parameters

Request-specific.

I/O Status Block

Request-specific.

Headers

Defined in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Operation

If the device is in a Plug and Play stopped state or removed state, or a request is not supported, I8042prt completes the request with a status of STATUS_INVALID_DEVICE_REQUEST.
