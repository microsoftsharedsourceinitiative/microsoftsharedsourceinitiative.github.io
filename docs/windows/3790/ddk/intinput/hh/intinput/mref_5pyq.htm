IRP_MJ_FLUSH_BUFFERS (Mouclass)

When Sent

The Microsoft Win32 subsystem sends this request to flush the Mouclass input queue.

Input Parameters

None.

Output Parameters

None.

I/O Status Block

The Information member is set to zero.

The Status member is set to one of the following values:

STATUS_SUCCESS
The request completed successfully.
STATUS_NOT_SUPPORTED
The target device is associated with a subordinate class device that does not support flushing the internal data queue.
STATUS_PRIVILEGE_NOT_HELD

Headers

Defined in ntddk.h and wdm.h. Include ntddk.h or wdm.h.

Operation

The IRP_MJ_FLUSH_BUFFERS request clears the internal data queue.
