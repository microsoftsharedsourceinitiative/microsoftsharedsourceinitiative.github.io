IDIGameCntrlPropSheet::SetID

The IDIGameCntrlPropSheet::SetID method populates the m_nID member of DIGameCntrlPropSheet. 

HRESULT
  IDIGameCntrlPropSheet::SetID(
    USHORT  nID 
    );

Parameters

nID
Indicates the joystick ID assigned to the device that is represented by this page. This is sent by Gchand.dll after successful calls to IDIGameCntrlPropSheet::GetPageInfo and IDIGameCntrlPropSheet::GetSheetInfo.

Return Value

Returns a COM error code.
