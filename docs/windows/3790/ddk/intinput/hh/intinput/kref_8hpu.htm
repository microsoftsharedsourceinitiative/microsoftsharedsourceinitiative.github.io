KEYBOARD_ATTRIBUTES

KEYBOARD_ATTRIBUTES specifies the attributes of a keyboard.

typedef struct _KEYBOARD_ATTRIBUTES {
  KEYBOARD_ID  KeyboardIdentifier;
  USHORT  KeyboardMode;
  USHORT  NumberOfFunctionKeys;
  USHORT  NumberOfIndicators;
  USHORT  NumberOfKeysTotal;
  ULONG  InputDataQueueLength;
  KEYBOARD_TYPEMATIC_PARAMETERS  KeyRepeatMinimum;
  KEYBOARD_TYPEMATIC_PARAMETERS  KeyRepeatMaximum;
} KEYBOARD_ATTRIBUTES, *PKEYBOARD_ATTRIBUTES;

Members

KeyboardIdentifier
Specifies the keyboard type and subtype in a KEYBOARD_ID structure: 
typedef struct _KEYBOARD_ID {
  UCHAR  Type;
  UCHAR  Subtype;
} KEYBOARD_ID, *PKEYBOARD_ID;
Type
Specifies the keyboard type.
Subtype
Specifies the keyboard subtype, which is a vendor-specific value.
See the Comments section.
KeyboardMode
Specifies the scan code mode. See the Comments section.
NumberOfFunctionKeys
Specifies the number of function keys that a keyboard supports.
NumberOfIndicators
Specifies the number of LED indicators that a keyboard supports.
NumberOfKeysTotal
Specifies the number of keys that a keyboard supports.
InputDataQueueLength
Specifies the size, in bytes, of the input data queue used by the keyboard port driver.
KeyRepeatMinimum
Specifies the minimum possible value for the keyboard typematic rate and delay in a KEYBOARD_TYPEMATIC_PARAMETERS structure.
KeyRepeatMaximum
Specifies the maximum possible value for the keyboard typematic rate and delay in a KEYBOARD_TYPEMATIC_PARAMETERS structure.

Headers

Declared in ntddkbd.h. Include ntddkbd.h.

Comments

This structure is used with a IOCTL_KEYBOARD_QUERY_ATTRIBUTES request to return information about the attributes that a keyboard supports.

For information about keyboard types, subtypes, scan code modes, and related keyboard layouts, see the documentation in i8042prt.h, kbd.h, and the layout files in the input\layout directory of the DDK source samples. See also the Microsoft® specification Keyboard Scan Code Specification available at the key support and scan codes Web site.

For general information about querying and setting keyboard, see Non-HIDClass Keyboard and Mouse Devices and Query and Set a Keyboard Device.
