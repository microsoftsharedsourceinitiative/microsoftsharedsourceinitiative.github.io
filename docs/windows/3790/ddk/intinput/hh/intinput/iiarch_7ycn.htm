Vendor-Supplied Drivers for Input Devices

This section describes optional and required vendor drivers for input devices.

Vendor drivers are not required for the following devices:

  • Devices that comply with the USB HID Standard.
  • Non-HIDClass devices attached to ports or buses operated by a system-supplied HID minidriver.
  • Keyboard, mouse, or game port devices operated by the system-supplied non-HIDClass drivers.

A vendor-supplied HID minidriver is required if the system-supplied input drivers do not support an input device. For example, a custom device attached to a serial port would need a vendor serial port HID minidriver.

WDM function or filter drivers for HID collections are optional.
