IEEE 1284.4 Devices: Windows DDK

Dot4 Driver Features

This section summarizes the features of the Dot4 driver.

The Dot4 driver:

  • Operates an independent logical bus for each Dot4 device that is installed on a host system.

    Each Dot4 device supports one or more independent Dot4 functions that are child devices of the Dot4 device. Each Dot4 function supports one or more subfunctions, or Dot4 services. Each service is accessed over a Dot4 channel that is bound to an IEEE 1284.4 primary socket.

  • Supports Dot4 function I/O control codes that abstract the IEEE 1284.4 communication protocol.

    Clients must use these I/O control codes to access a Dot4 function. A client can be a Windows 2000 or later application, a subsystem of the operating system, or a kernel-mode driver.
