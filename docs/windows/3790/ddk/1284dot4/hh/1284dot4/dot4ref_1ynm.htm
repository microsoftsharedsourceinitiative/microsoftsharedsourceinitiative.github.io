DOT4_DC_OPEN_DATA

The DOT4_DC_OPEN_DATA structure specifies information Dot4 driver uses to open a Dot4 channel.

typedef struct _DOT4_DC_OPEN_DATA {
  unsigned char  bHsid;
  unsigned char  fAddActivity;
  CHANNEL_HANDLE  hChannelHandle;
} DOT4_DC_OPEN_DATA, *PDOT4_DC_OPEN_DATA;

Members

bHsid
Specifies an IEEE 1284.4 primary socket ID, which is obtained by using an IOCTL_DOT4_CREATE_SOCKET request.
fAddActivity
Specifies whether to add activity broadcast on a channel immediately after the channel is opened. To add activity broadcasting, fAddActivity is set to TRUE (nonzero), otherwise fAddActivity is set to FALSE (zero).
hChannelHandle
Specifies a handle to a Dot4 channel. The Dot4 driver sets this value.

Headers

Declared in d4drvif.h. Include d4drvif.h.

Comment

The client uses a DOT4_DC_OPEN_DATA structure to input and output information with an IOCTL_DOT4_OPEN_CHANNEL request. A client sets hHsid and fAddActivity, and the Dot4 driver sets hChannelHandle.
