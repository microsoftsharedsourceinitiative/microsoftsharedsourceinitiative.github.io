Image Device Setup Class and Dot4 Scanner Functions

A Dot4 scanner function, which is a Dot4 function that implements a scanner function, is a member of the Image device setup class.

Dot4 scanner functions do not require vendor drivers because Microsoft provides the following:

  • Dot4scan driver — the system function driver for Dot4 scanner functions.
  • Windows Image Acquisition (WIA) drivers, which support user-mode clients communicating with imaging devices such as digital cameras and scanners.

Note Microsoft supplies the Dot4scan driver to provide a software connection layer between existing system imaging components and Dot4 scanner functions.

A vendor must provide an INF file to install a Dot4 scanner function.
