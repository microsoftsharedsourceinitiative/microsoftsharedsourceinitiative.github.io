Dot4 Power Management Support

A Dot4 function supports the following power requests:

IRP_MN_QUERY_POWER

IRP_MN_SET_POWER

The Dot4 driver does not support the following requests, and completes them as indicated:
IRP_MN_POWER_SEQUENCE
Returns STATUS_NOT_IMPLEMENTED.
IRP_MN_WAIT_WAKE
Returns STATUS_NOT_SUPPORTED.

The parent bus driver of a Dot4 device is the power policy owner for the device.
