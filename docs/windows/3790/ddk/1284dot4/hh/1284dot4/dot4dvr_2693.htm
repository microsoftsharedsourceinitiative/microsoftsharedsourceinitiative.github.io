Dot4 WMI Support

The Dot4 driver registers the following types of WMI library callback routines for a Dot4 function:

DpWmiQueryReginfo

DpWmiQueryDataBlock

The Dot4 driver's DpWmiQueryReginfo routine registers a DOT4_WMI_BYTES_TRANSFERRED_GUID GUID, and the driver's DpWmiQueryDataBlock routine returns a WMI data block that contains a single instance of a DOT4_WMI_XFER_INFO structure. The DOT4_WMI_XFER_INFO structure contains information about data flow over Dot4 channels.

See Windows Management Instrumentation for more information about WMI operation.
