IEEE 1284.4 Devices: Windows DDK

Dot4 Data Link

The Dot4 driver requires that the drivers below it in a Dot4 device stack support an IEEE 1284.4 data link. The Dot4 driver uses internal device control requests to establish the required data link. In response to these internal device control requests, the system-supplied bus driver for parallel ports establishes an IEEE 1284.3 data link with a device. (An IEEE 1284.3 data link meets the requirements of an IEEE 1284.4 data link.)
