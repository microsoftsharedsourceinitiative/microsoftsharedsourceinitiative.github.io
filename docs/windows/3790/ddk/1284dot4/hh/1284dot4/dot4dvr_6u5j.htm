Dot4 Services Registry Key

The following Services key is located under a Dot4 device's Plug and Play hardware device key:

Services key
Contains a set of Service name subkeys for a Dot4 device.
Services\<Service name> key
Specifies the name of a service. The Service name key contains the following configuration entry values:
HostToPeriSize entry value (REG_DWORD)
Specifies the maximum packet size that can be used over a Dot4 channel to transfer data from the host to the peripheral. The default value is TO_PERI_PACKET_SIZE.
PeriToHostSize entry value (REG_DWORD)
Specifies the maximum packet size that can be used over a Dot4 channel to transfer data from the peripheral to the host. The default value is TO_PERI_PACKET_SIZE.

The Dot4 driver creates registry entries for Dot4 services that are statically defined in the driver. If a static service name is not present, Dot4 adds the service name to the registry. If a service name is present, but a configuration entry value is not present, Dot4 adds the entry value, and sets it to a default value.
