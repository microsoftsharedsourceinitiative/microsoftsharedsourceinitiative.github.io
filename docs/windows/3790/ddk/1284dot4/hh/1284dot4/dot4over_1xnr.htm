System-Supplied Drivers for Dot4 Devices

Microsoft® Windows® 2000 and later includes the following driver support for Dot4 devices:

  • The Dot4 driver is the function driver and bus driver for the Dot4 device setup class. The Dot4 driver enumerates and controls access to all its child devices, or Dot4 functions. The executable image of the Dot4 driver is dot4.sys.
  • The system-supplied bus driver for parallel ports is the parent bus driver for Dot4 devices that are attached to a parallel port. This bus driver supports the Dot4 data link that the Dot4 driver requires to access a Dot4 device. For more information about the parallel port bus driver, see System-Supplied Parallel Drivers.
  • The Dot4prt driver is the function and bus driver for the Dot4Print device setup class. The Dot4prt driver provides a connection layer between existing system printer components and Dot4 print functions. The Dot4prt driver creates a functional device object (FDO) for a Dot4Print device and a physical device object (PDO) for a single-child device, which is a member of the Printer device setup class. Upper-level system printer components and drivers access and operate Printer devices. The executable image of the Dot4prt driver is dot4prt.sys.
  • The Dot4scan driver is the function driver for Dot4 scanner functions, which are Dot4 functions that implement a scanner. The Dot4scan driver provides a connection layer between existing system imaging components and Dot4 scanner functions. A Dot4 scanner function is a member of the Image device setup class, and is operated through the services of the WIA drivers. The executable image of the Dot4scan driver is dot4scan.sys.
