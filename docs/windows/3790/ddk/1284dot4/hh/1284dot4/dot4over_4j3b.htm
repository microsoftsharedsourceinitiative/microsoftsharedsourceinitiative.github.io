Dot4Print Device Setup Class

A Dot4 print function, which is a Dot4 function that implements a print function, is a member of the Dot4Print device setup class. A Dot4Print device is a parent device that has a single child device; the child device is a member of the Printer device setup class.

Dot4Print devices do not require vendor drivers because Microsoft provides the following:

  • Dot4prt driver — the system function driver for a parent Dot4Print device
  • Dot4prt.inf file that installs a Dot4Print device
  • Port monitor support for Printer devices

Note  Microsoft supplies the Dot4prt driver to provide a software connection layer between existing system printer components and Dot4 print functions.

A vendor must provide an INF file for the child Printer device associated with the parent Dot4 print function.
