Dot4 PortName Entry Value

The following PortName entry value is located under the Plug and Play hardware device key for a Dot4 device and a Dot4 function:

PortName entry value (REG_SZ)
Specifies the name of the parallel port used by a Dot4 device. A Dot4 device, and all its child devices (Dot4 functions), use the same port name.
