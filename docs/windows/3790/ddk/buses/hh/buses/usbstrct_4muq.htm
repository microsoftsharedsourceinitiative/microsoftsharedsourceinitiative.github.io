System Support for Buses: Windows DDK

USB Structures

This section describes the system-defined structures used by Universal Serial Bus (USB) client drivers on Windows® Driver Model (WDM) platforms. See System Structures for information about system-defined structures that are not described here.

Drivers can use only the members of structures that are described in this section. All undocumented members of these structures are reserved for system use.

Transfer buffers that are members of structures defined in this section must be nonpageable memory.
