USB_DEVICE_TYPE

The USB_DEVICE_TYPE enumeration type is used in conjunction with the USB_DEVICE_INFORMATION_0 structure and the QueryDeviceInformation routine to report the type of a USB device. 

typedef enum _USB_DEVICE_TYPE {
  Usb11Device = 0,
  Usb20Device
} USB_DEVICE_TYPE;

Enumerators

Usb11Device
Indicates a USB 1.1-compliant device.
Usb20Device
Indicates a USB 2.0-compliant device.

Headers

Declared in usbhcdi.h. Include usbhcdi.h.
