Av61883_Listen

Operation

The Av61883_Listen request begins isochronous reception on the specified connection. This request will start capturing CIP packets, whether or not the packets have frames attached. The relevant members of the AV_61883_REQUEST structure for this request are:

typedef struct _AV_61883_REQUEST {
  ULONG Function;
  union {
    typedef struct _CIP_LISTEN {
      IN HANDLE  hConnect;
    } CIP_LISTEN, *PCIP_LISTEN;
    .
    .
    .
  };
} AV_61883_REQUEST, *PAV_61883_REQUEST;

AV_61883_REQUEST Input

Function
Av61883_Listen
hConnect
The handle of the connection to begin isochronous reception.

I/O Status Block

If successful, the IEC-61883 protocol driver sets Irp->IoStatus.Status to STATUS_SUCCESS.

If an incorrect parameter is passed in, the protocol driver sets Irp->IoStatus.Status to STATUS_INVALID_PARAMETER.

If the protocol driver is unable to allocate resources, it sets Irp->IoStatus.Status to STATUS_INSUFFICIENT_RESOURCES.
