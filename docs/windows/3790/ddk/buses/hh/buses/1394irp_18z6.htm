REQUEST_BUS_RESET

The REQUEST_BUS_RESET request initiates an IEEE 1394 bus reset.

The relevant members of the IRB for this request are:

typedef struct _IRB {
  ULONG FunctionNumber;
    .
    .
    .
  union {
    struct {
      ULONG    fulFlags;
    } BusReset;
    .
    .
    .
  } u;
} IRB;

IRB Input

FunctionNumber
REQUEST_BUS_RESET
u.BusReset.fulFlags
Set this flag to BUS_RESET_FLAGS_FORCE_ROOT to make the local node the root node. All other values indicate that the node is not root.
