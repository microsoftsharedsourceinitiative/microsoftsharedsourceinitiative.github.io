USB_CONTROLLER_INFORMATION_0

The USB_CONTROLLER_INFORMATION_0 structure is used in conjunction with the GetControllerInformation routine to retrieve information about a USB host controller. 

typedef struct _USB_CONTROLLER_INFORMATION_0 {
  ULONG  InformationLevel;
  ULONG  ActualLength;
  BOOLEAN  SelectiveSuspendEnabled;
  BOOLEAN  IsHighSpeedController;
} USB_CONTROLLER_INFORMATION_0, *PUSB_CONTROLLER_INFORMATION_0;

Members

InformationLevel
Indicates the level of information that the caller requests. For this structure, the level should be set to 0.
ActualLength
Indicates the length in bytes of the data returned.
SelectiveSuspendEnabled
Indicates whether the selective suspend feature is enabled for the controller.
IsHighSPeedController
Indicates whether the controller is a high-speed, USB 2.0-compliant controller.

Headers

Declared in hubbusif.h. Include hubbusif.h.

Comments

The USB_CONTROLLER_INFORMATION_0 structure is simply a header followed by a variable quantity of controller data returned by the port driver in response to a call to the GetControllerInformation routine. The ActualLength member indicates how much data the port driver returned, including the header.
