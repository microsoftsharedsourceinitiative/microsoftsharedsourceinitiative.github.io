USB_DEVICE_INFORMATION_0

The USB_DEVICE_INFORMATION_0 structure is used in conjunction with the QueryDeviceInformation routine to report information about a USB device. 

typedef struct _USB_DEVICE_INFORMATION_0 {
  ULONG  InformationLevel;
  ULONG  ActualLength;
  ULONG  PortNumber;
  USB_DEVICE_DESCRIPTOR  DeviceDescriptor;
  UCHAR  DD_pad[2];
  UCHAR  CurrentConfigurationValue;
  UCHAR  ReservedMBZ;
  USHORT  DeviceAddress;
  ULONG  HubAddress;
  USB_DEVICE_SPEED  DeviceSpeed;
  USB_DEVICE_TYPE  DeviceType;
  ULONG  NumberOfOpenPipes;
  USB_PIPE_INFORMATION_0  PipeList[1];
} USB_DEVICE_INFORMATION_0,  *PUSB_DEVICE_INFORMATION_0;

Members

InformationLevel
Indicates the level of information that the caller requests. For this structure, the level should be set to 0.
ActualLength
Indicates, on return, the length in bytes of the data output.
PortNumber
Indicates, on return, the number of the port to which the device is connected.
DeviceDescriptor
Contains, on return, the USB device descriptor for the device. For more information on the content of the device descriptor, see USB_DEVICE_DESCRIPTOR.
DD_pad
Forces double word alignment. This member does not contain meaningful data.
CurrentConfigurationValue
Contains the value that is used to select a configuration. This value is passed to the USB SetConfiguration request , as described in version 1.1 of the Universal Serial Bus Specification. The port driver does not currently expose a service that allows higher-level drivers to set the configuration.
ReservedMBZ
Reserved.
DeviceAddress
Contains the USB address that was assigned to the device.
HubAddress
Contains the USB address that was assigned to the hub device.
DeviceSpeed
Indicates the device speed. For a list of values that this member can take, see the USB_DEVICE_SPEED.
DeviceType
Indicates the device type. For a list of values that this member can take, see the USB_DEVICE_TYPE enumerator.
NumberOfOpenPipes
Indicates the number of open pipes.
PipeList
Contains information about the pipes connected to the interface for the device. For further information on the pipe data that is contained in this member, see USB_PIPE_INFORMATION_0.

Headers

Declared in hubbusif.h. Include hubbusif.h.
