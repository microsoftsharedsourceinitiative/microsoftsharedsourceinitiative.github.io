PHW_FIND_ADAPTER

The PHW_FIND_ADAPTER prototype declares a routine that uses supplied configuration to determine whether a specific HBA is supported and, if it is, to return configuration information about that adapter. 

typedef
ULONG
(*PHW_FIND_ADAPTER) (
  IN PVOID  DeviceExtension,
  IN PVOID  HwContext,
  IN PVOID  BusInformation,
  IN PCHAR  ArgumentString,
  IN OUT PPORT_CONFIGURATION_INFORMATION  ConfigInfo,
  OUT PBOOLEAN  Again
  );

Parameters

DeviceExtension
Pointer to the miniport driver's per-HBA storage area.
HwContext
Reserved member when used by one of the Storport driver's miniports.
With SCSI miniports this member points to a context value. For a description of the meaning of this member for a SCSI miniport, see HwScsiFindAdapter.
BusInformation
Reserved member when used by one of the Storport driver's miniports.
With SCSI miniports this member points to bus-type-specific information that the OS-specific port driver has gathered. For a complete description of the meaning of this member for a SCSI miniport, see HwScsiFindAdapter.
ArgumentString
Reserved member when used by one of the Storport driver's miniports.
With SCSI miniports this member points to a zero-terminated ASCII string that contains device information such as a base parameter or an interrupt level from the registry. For a complete description of the meaning of this member for a SCSI miniport, see HwScsiFindAdapter.
ConfigInfo
Points to a PORT_CONFIGURATION_INFORMATION structure that the miniport driver uses during initialization.
For more information on how a SCSI miniport uses this member, see the description of the ConfigId member of the PORT_CONFIGURATION_INFORMATION structure.
For more information on how this member is used by a miniport that works with the StorPort driver, see the description of the PortConfiguration member of the PORT_CONFIGURATION_INFORMATION (STORPORT) structure.
Again
Reserved member when used by one of the Storport driver's miniports.
With SCSI miniports this member points to a BOOLEAN variable that informs the port driver whether it should call this routine again. For more information on the meaning of this member for a SCSI miniport, see HwScsiFindAdapter.

Return Value

The routine declared by this prototype must return one of the following status values:

Value Meaning
SP_RETURN_FOUND Indicates a supported HBA was found and that the HBA-relevant configuration information was successfully determined and set in the PORT_CONFIGURATION_INFORMATION structure.
SP_RETURN_ERROR Indicates an HBA was found but there was error obtaining the configuration information. If possible, such an error should be logged with ScsiPortLogError.
SP_RETURN_BAD_CONFIG Indicates the supplied configuration information was invalid for the adapter.
SP_RETURN_NOT_FOUND Indicates no supported HBA was found for the supplied configuration information.

Headers

Declared in either srb.h or storport.h. Include srb.h or storport.h.

Comments

This declaration is used by both SCSI and StorPort miniport drivers.

For more information on the SCSI miniport driver's version of the routine associated with this declaration, see HwScsiFindAdapter.

For more information on the Storport driver's version of the routine associated with this declaration, see HwStorFindAdapter.
