Installing an SMB Client Driver

Installing an SMB client driver requires writing an INF file for the client driver and editing the battery.inf file supplied with the system. The INF file must specify the hardware ID, class name, and class GUID for the device. See Creating an INF File and Installing a Battery Driver for more information.

In battery.inf, the device controlled by the client driver must be listed in the Models section. If the device is not listed, add an entry like the following:

%ACPI\acpiNNNN.DeviceDesc% = SMBBCLIENT_Inst,ACPI\acpiNNNN

In the example, ACPI\acpiNNNN represents the ACPI hardware ID of the client device. The client device must be listed in the ACPI name space under the SMBus device, and its hardware ID must match the value of the _HID object in the ACPI name space.
