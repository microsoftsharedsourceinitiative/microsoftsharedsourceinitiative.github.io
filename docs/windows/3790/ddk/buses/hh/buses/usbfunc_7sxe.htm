System Support for Buses: Windows DDK

USBD_QueryBusTime

The USBD_QueryBusTime routine retrieves the current host controller frame. This routine is obsolete in Windows XP and later operating systems. Do not use. 

NTSTATUS
USBD_QueryBusTime(
  IN PDEVICE_OBJECT  RootHubPdo,
  IN PULONG  CurrentFrame
  )

Parameters

RootHubPdo
Pointer to the device object for the root hub.
CurrentFrame
Pointer to the a memory location holding the current frame number.

Return Value

USBD_QueryBusTime always returns STATUS_SUCCESS.

Headers

Declared in usbdlib.h. Include usbdlib.h.

Comments

Callers of this routine can be running at any IRQL level.
