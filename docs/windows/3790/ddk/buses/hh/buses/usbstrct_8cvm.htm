USB_COMMON_DESCRIPTOR

The USB_COMMON_DESCRIPTOR contains the head of the first descriptor that matches the search criteria in a call to USBD_ParseDescriptors. 

typedef struct _USB_COMMON_DESCRIPTOR {
  UCHAR  bLength;
  UCHAR  bDescriptorType;
} USB_COMMON_DESCRIPTOR, *PUSB_COMMON_DESCRIPTOR;

Members

bLength
Specifies the entire length of the descriptor, not of this structure.
bDescriptorType
Specifies the descriptor type code, as assigned by USB, for this descriptor.

Headers

Declared in usb100.h. Include usb100.h.
