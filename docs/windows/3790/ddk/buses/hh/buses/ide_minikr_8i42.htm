PciIdeX Library Routines

Microsoft supplies platform-specific PciIdeXxx library routines to support platform-independent IDE controller minidrivers.

The PciIdeXxx routines constitute a kernel-mode dynamic-link library. References for the PciIdeXxx routines are in alphabetical order.
