IEEE1394_API_REMOVE_VIRTUAL_DEVICE

Operation

The IEEE1394_API_REMOVE_VIRTUAL_DEVICE request number is used in conjunction with the IOCTL_IEEE1394_API_REQUEST request to remove a virtual device from the 1394 device stack. The IOCTL_IEEE1394_API_REQUEST request is only available in Windows XP and later operating systems.

The relevant members of the IEEE1394_API_REQUEST structure for this request are:

typedef struct _IEEE1394_API_REQUEST {
  ULONG  RequestNumber;
  ULONG  Flags;
  union {
    IEEE1394_VDEV_PNP_REQUEST  RemoveVirtualDevice;
    .
    .
    .
  } u;
} IEEE1394_API_REQUEST, *PIEEE1394_API_REQUEST;

RequestNumber
IEEE1394_API_REMOVE_VIRTUAL_DEVICE
Flags
See IEEE1394_API_REQUEST for a list of values that can be assigned to Flags.
u.RemoveVirtualDevice
See IEEE1394_VDEV_PNP_REQUEST for a list of values that can be assigned to AddVirtualDevice.

Output

None

I/O Status Block

The bus driver sets Irp->IoStatus.Status to STATUS_SUCCESS on success, or the appropriate error code on failure.
