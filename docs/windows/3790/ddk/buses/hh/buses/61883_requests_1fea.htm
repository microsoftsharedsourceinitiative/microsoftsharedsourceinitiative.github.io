IEC-61883 Protocol I/O Requests

An IEC-61883 client driver must communicate with its device by submitting IRPs down the device stack to the IEC-61883 protocol driver.

To use these I/O requests, include the header file 61883.h, which is shipped with the Microsoft® Windows® DDK.
