Sample USB Client Drivers

This topic identifies the sample USB client drivers that are included with the Microsoft® Windows® DDK.

The Windows DDK includes the source code for the following drivers in the USB section of the \src\wdm directory tree of the Windows DDK:

  • Isochronous client driver sample (isousb.sys)
  • Bulk transfer client driver sample (bulkusb.sys)
  • Selective suspend client driver sample (selsusp.sys)
