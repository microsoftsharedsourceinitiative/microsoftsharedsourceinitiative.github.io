Installing USB Client Drivers

This topic provides installation information that is specific to USB client drivers in Microsoft® Windows® XP and later versions of the operating system.

If you supply your own USB client driver, you should make that driver a member of the USB setup class in the INF Version Section of the driver's INF file. For example:

[version]
Signature="$WINDOWS NT$"
Class=USB
ClassGuid={36FC9E60-C465-11CF-8056-444553540000}

There are no other special requirements associated with installing USB client drivers.

For a list of USB hardware with native support in Windows 2000 and later, see the system-supplied INF files that are associated with each USB class. For example, see usbprint.inf for the USB printer class, usbstor.inf for the USB mass storage class, and input.inf for the USB HID class.

For general information about device installation in Windows 2000 and later versions of the operating system, see Device Installation Overview. For a discussion of how to use the ChkINF tool to validate an INF file, see ChkINF.
