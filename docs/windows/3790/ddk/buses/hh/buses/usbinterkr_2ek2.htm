System Support for Buses: Windows DDK

Routines Associated with Interfaces Exposed by the USB Bus Driver

Version 0 of the bus interface contains the following routines:

InterfaceReference

InterfaceDereference

GetUSBDIVersion

QueryBusTime

SubmitIsoOutUrb

QueryBusInformation

Version 1 of the bus interface contains all of the routines present in version 0 plus the following routine:

IsDeviceHighSpeed

Version 2 of the bus interface contains all of the routines present in version 1 plus the following routine:

EnumLogEntry
