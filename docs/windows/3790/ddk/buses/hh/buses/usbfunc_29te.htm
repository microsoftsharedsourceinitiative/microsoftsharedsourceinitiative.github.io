USBD_Debug_LogEntry

The USBD_Debug_LogEntry routine adds an entry to the USB bus driver log. This routine is obsolete in Windows XP and later operating systems. Do not use. 

VOID
USBD_Debug_LogEntry(
  IN CHAR  *Name,
  IN ULONG_PTR  Info1,
  IN ULONG_PTR  Info2,
  IN ULONG_PTR  Info3
  )

Parameters

Name
Contains a string that identifies the log entry.
Info1
Pointer to a value to be stored in this log entry. This can be any value of the caller's choice.
Info2
Pointer to a value to be stored in this log entry. This can be any value of the caller's choice.
Info3
Pointer to a value to be stored in this log entry. This can be any value of the caller's choice.

Return Value

None.

Headers

Declared in usbdlib.h. Include usbdlib.h.

Comments

Callers must be running at IRQL < DISPATCH_LEVEL.
