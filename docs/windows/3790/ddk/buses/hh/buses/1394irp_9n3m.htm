REQUEST_GET_GENERATION_COUNT

The REQUEST_GET_GENERATION_COUNT request retrieves the current bus reset generation. This count is incremented every time the bus is reset. This generation count must be passed in each asynchronous read, write, and lock request that a device driver submits to the 1394 bus driver. This request must be made at IRQL <= DISPATCH_LEVEL.

The relevant members of the IRB for this request are:

  union {
    struct {

} IRB;

IRB Input

FunctionNumber
REQUEST_GET_GENERATION_COUNT

IRB Output

u.GetGenerationCount.GenerationCount
Specifies the current generation count.

I/O Status Block

The bus driver sets Irp->IoStatus.Status to STATUS_SUCCESS on success, or the appropriate error code on failure.
