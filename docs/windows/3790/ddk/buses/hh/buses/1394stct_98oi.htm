IEEE1394_VDEV_PNP_REQUEST

The IEEE1394_VDEV_PNP_REQUEST structure contains data specific to an add or remove virtual device request.

typedef struct _IEEE1394_VDEV_PNP_REQUEST{
  ULONG  fulFlags;
  ULONG  Reserved;
  ULARGE_INTEGER  InstanceId;
  UCHAR  DeviceId;
} IEEE1394_VDEV_PNP_REQUEST,*PIEEE1394_VDEV_PNP_REQUEST;

Members

fulFlags
Flag Description
IEEE1394_REQUEST_FLAG_UNICODE The text string supplied in DeviceId is in Unicode.
Reserved
Reserved. Driver writers must set this member to zero.
InstanceId
A 64-bit number used to identify this instance of the virtual device.
DeviceId
Supplies a buffer containing a null-terminated string to be used for generating the device ID for this device.

Headers

Declared in 1394.h. Include 1394.h.

Comments

This structure contains the information needed to define or remove a virtual device.
