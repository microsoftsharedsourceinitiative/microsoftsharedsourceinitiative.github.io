CC Category

CC Category output pins provide a stream of decoded, closed-caption byte pairs from field 1, line 21.

When creating the category PinName, PINNAME_VIDEO_CC, the following parameters must be filled out.

Parameter Value
KS Datarange KS_DATARANGE
KS Dataformat KS_DATAFORMAT
KS MajorFormat KSDATAFORMAT_TYPE_AUXLine21Data
KS SubFormat (typical) KSDATAFORMAT_SUBTYPE_Line21_BytePair
KS Specifier KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
DS majortype None
DS formattype None
