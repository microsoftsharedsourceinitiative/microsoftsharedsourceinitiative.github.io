USB and HID Drivers

To create USB or kernel-mode HID client drivers, refer to the Microsoft® Windows® 2000 DDK documentation. USB and HID client drivers that conform to the Windows Driver Model (WDM) are source-compatible for Windows 98, Windows Millennium Edition (Windows Me), and Windows 2000 systems, provided they use only the system support routines that are declared in wdm.h.

However, source compatibility does not guarantee that an HID or USB driver will function properly on all WDM systems. Drivers should be thoroughly tested on all systems on which they are intended to run.

There are also significant differences in the Windows 2000 and Windows Me input architectures. Readers are referred to the next section for a discussion of the Windows Me input architecture.
