WMILib

WMILib is a set of functions that manage the IRP_MJ_SYSTEM_CONTROL IRPs and the WNODE data formats. WMILIB is appropriate only for those devices that do not need the flexibility of dynamic instance names. WMILib is only available on Windows NT/Windows 2000.

The device should build a WMILIBINFO structure that contains information about the GUIDs being registered, the device stack in which it resides, and callbacks for various WMI functions. The QueryWmiRegInfo and QueryWmiDataBlock callbacks must be implemented. However, if the other callbacks are set to NULL, then WMILIB will return an appropriate error message.
