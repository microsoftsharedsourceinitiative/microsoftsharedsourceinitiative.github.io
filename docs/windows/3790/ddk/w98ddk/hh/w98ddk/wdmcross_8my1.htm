Unavailable Functions in Windows Millennium Edition

Windows Me provides a subset of the functions provided by Windows 2000.

The following functions are not implemented by Windows Me:

  • IoReuseIrp
  • RtlFillMemory
  • RtlUlongByteSwap
  • RtlUlonglongByteSwap
  • RtlUshortByteSwap
  • ExInterlockedAddLargeInteger
  • KeAcquireSpinLockAtDpcLevel
  • KeReleaseSpinlockFromDpcLevel
  • KeEnterCriticalRegion
  • KeLeaveCriticalRegion
