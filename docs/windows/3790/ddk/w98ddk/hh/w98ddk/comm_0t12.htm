VCOMM_PM_API_ctx

include vcomm.inc

mov     ebx, hPort
mov     cl, chx
mov     ax, VCOMM_PM_API_ctx
call    [_VCOMMCallAddr]

Calls the _VCOMM_TransmitCommChar service.

  • Returns zero in AX if successful, or an error value otherwise.
hPort
Communications resource handle.
chx
Character to transmit.
