Bitmap Font Information

The cbBitMapFontDescriptor and cbBitMapCharDescriptor controls specify the size in bytes of the font, and character descriptors for a bitmapped font.

HP LaserJet IIP

For the HP LaserJet IIP printer, the font descriptor requires 64 bytes; therefore the cbBitMapFontDescriptor box contains the value 64. The character descriptor for each symbol in the font requires 16 bytes; therefore, the cbBitMapCharDescriptor contains the value 16.
