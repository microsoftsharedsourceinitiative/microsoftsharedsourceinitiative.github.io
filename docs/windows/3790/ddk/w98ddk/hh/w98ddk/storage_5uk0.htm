FSD File I/O Interface

Each file I/O routine and its I/O request parameters are given in the following. All of the FSD routines receive the ID of the requesting process in the ir_pid field and return success (zero) or failure (nonzero). If failure, the error code is returned in the ir_error field. The error code values that should be returned are the DOS extended error code set.
