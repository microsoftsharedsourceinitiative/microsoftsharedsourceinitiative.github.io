Enabing Duplex-On Printing

The PC_OCD_DUPLEX_ON box contains the printer command which the Universal Printer Driver issues to enable duplex (double-sided) printing. The Universal Printer Driver will not process the duplex-on command unless the MD_DUPLEX box is checked in the fGeneral dialog box.

Diconix 150 Plus

The Diconix 150 Plus does not support duplex printing; therefore the PC_OCD_DUPLEX_ON box is empty.

HP LaserJet IIP

For the HP LaserJet IIP, two separate commands are required to enable duplex printing. One command enables duplex printing in portrait mode and the other in landscape mode. These commands are entered in the PC_OCD_DUPLEX_VERT and PC_OCD_DUPLEX_HORZ boxes. The PC_OCD_DUPLEX_ON box is empty.
