IFSMgr_NetToWin32Time

IFSMgr_NetToWin32Time(
 unsigned long NetTime
 )

This service converts the time in net time format to the Win32 time format. Both these formats use UTC time.

NetTime
Time in net time format.
  • Returns time in Win32 time format.
