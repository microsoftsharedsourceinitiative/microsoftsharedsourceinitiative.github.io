DRV_GetVersion

Returns the driver version and pen packet size.

  • The high-order word of the return value contains the pen packet size in bytes. The low-order word of the return value contains the driver version information. The low-order byte is the major version number of the driver and the high-order byte is the minor version number.
lParam1
Should be set to zero.
lParam2
Should be set to zero.
