IFSMgr_DosToWin32Time

IFSMgr_DosToWin32Time(
 dostime_t DosTime
 )

This service converts the time passed in DOS format in local time to the Win32 style time in UTC.

DosTime
DOS time filled in the dostime_t structure.
  • Returns the Win32 time in UTC filled in the _FILETIME structure.
