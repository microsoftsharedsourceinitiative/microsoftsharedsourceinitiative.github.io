Step 3. Running Other IR-Capable Applications

Most applications that can communicate over a null modem cable that connects serial ports on two Windows 95 computers should also be able to communicate over an IR link. The procedure for setting up and using the IR link in these other communicating applications will probably be similar to the procedure you use in Step 2 of the Roadmap, when you use the HyperTerminal application to validate the installation of the IR communications driver. The procedure for running the Windows 95 Direct Cable Connection application is given in detail in Notes on Running the Direct Cable Connection Application Over an IR Link.
