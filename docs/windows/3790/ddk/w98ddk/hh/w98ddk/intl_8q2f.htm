Japanese EUDC

Japanese Windows 3.1 and Windows NT® 3.1 defined the EUDC code range (starting at F040) of the Japanese Microsoft Standard Character Set (ShiftJIS) in the End User Zone. The following table shows the mapping schema between the Japanese Microsoft Standard Character Set (starting at F040) and the Unicode (starting at U+E000).
