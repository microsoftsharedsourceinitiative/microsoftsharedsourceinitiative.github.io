KSPROPERTY_VIDEOPROCAMP_WHITEBALANCE

The KSPROPERTY_VIDEOPROCAMP_WHITEBALANCE read/write property is used to set and retrieve the white balance setting. The KSPROPERTY_VIDEOPROCAMP_S.Value member specifies the white balance setting expressed as a color temperature in degrees Kelvin. The range and default values for this property are device dependent.
