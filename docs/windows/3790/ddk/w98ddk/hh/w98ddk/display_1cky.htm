Adapters Without Frame Buffers

If you have a display adapter that does not have a frame buffer, you cannot use most of the DIB engine's capabilities. However, you can still use the DIB engine for drawings into memory bitmaps. For more information about using the DIB engine even if you don't have a frame buffer, see About the Device Independent Bitmap Engine.
