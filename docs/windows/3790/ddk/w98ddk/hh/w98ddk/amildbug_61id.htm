Windows 98/Me: Windows DDK

Examples

Type "dns" to dump the ACPI namespace.
Type "find _prw" to find all declarations of _PRW. This can be used for any declared object in the namespace.
Type "dns /s \_sb.pci0.dock". This will dump the namespace of everything just under dock.
Type "dns /s \_sb.pci0.dock._dck". This will dump the namespace of everything under _dck.
Type "u \_sb.pci0.dock._dck". This will disassemble the _dck method. This will only work for methods.
Type "ln" to list the nearest method.
Type "lc" for the current context.
Type "r" to display the context information.
Type "dl" to display the event log.
Type "cl" to clear the event log.
Type "run \_sb.pci0.dock._sta" to run control method _sta on device dock. This can only be done for methods.
Type "set verboseon" to turn up the debug spew from ACPI.
Type "set traceon" to turn on ACPI trace.
Type "bp \_sb.pci0.dock._dck" to set a breakpoint on _dck.
Type "bl" to list all set breakpoints.
Type "bd br0" to disable breakpoint 0.
Type "be br0" to enable breakpoint 0.
Type "bc br0" to clear breakpoint 0.
Type "p" to step through the code/method.
Type "g" to continue code execution.
