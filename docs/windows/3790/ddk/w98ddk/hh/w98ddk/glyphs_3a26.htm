INF File Syntax For Profile Installation

In the following example , the printer named “My printer model” has two ICM profiles: profile1.icm and profile2.icm. Note that the destination directory for color profiles is 23, which refers to the COLOR directory.

[Version]
Signature="$CHICAGO$"
Class=Printer
Provider=“My Company” 

[My Company]
“My printer model” = MYDRIVER,My_Printer_Model

[MYDRIVER]
DriverFile=DRVR.DRV
DataFile=DRVR.DRV
CopyFiles=@DRVR.DRV,MY COLOR PROFILES
DataSection=MYDATA

[MYDATA]
HelpFile=DRVR.HLP
DefaultDataType=EMF

[MY_COLOR_PROFILES]
profile1.icm
profile2.icm

[DestinationDirs]
DefaultDestDir=11
MY_COLOR_PROFILES =23
