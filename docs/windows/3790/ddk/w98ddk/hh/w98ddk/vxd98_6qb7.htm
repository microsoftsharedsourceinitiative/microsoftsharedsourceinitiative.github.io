VPOWERD_Get_Capabilities

VPOWERD_Get_Capabilities retrieves the APM bios capabilities bit mask. BIOS capabilities do not change while the system is running, so these values are stored in the reference data structure passed up from Real Mode Init.

POWERRET VPOWERD_Get_Capabilities( 
  APM_CAPABILITIES   *pAPM_CAPABILITIES
);

Parameters

pAPM_CAPABILITIES
Structure that is completed with the relevant capabilities.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
