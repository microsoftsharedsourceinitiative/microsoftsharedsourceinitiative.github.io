VPOWERD_W32_Set_System_Power_State

VPOWERD_W32_Set_System_Power_State sets the power state of the system.

POWERRET VPOWERD_W32_Set_System_Power_State(
  boolean   dwSuspendFlag,
  boolean   dwForceFlag
 );

Parameters

dwSuspendFlag
Set to TRUE if APM standby is not required.
dwForceFlag
Set to TRUE to suspend. By default, this should be FALSE.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
