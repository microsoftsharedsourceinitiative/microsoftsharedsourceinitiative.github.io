Spot Diameter

The sSpotDiameter box contains a value that specifies a special value used by the Universal Printer Driver to rasterize graphic data and compensate for overlapping pixels. The value entered will affect the lightness or darkness of printed raster-graphic images. The user can adjust this setting through Control Panel. Double-click on the Printers icon in Control Panel, choose the Setup button in the Printers dialog box, and choose the Options button to display the Options dialog box. The user can adjust the intensity of printed output using the Intensity scroll bar in the Options dialog box. The following table lists spot diameters for common resolutions supported by 9-pin, 24-pin, and ink-jet printers.

Printer Resolution Diameter
9-pin 240 x 144 170
  120 x 144 140

  180 x 180 140
