Return Values

Errors are represented by negative values and cannot be combined.The following list describes the values that can be returned by all IDsDriver, IDsDriverBuffer, and IDsDriverPropertySet methods. For a list of the error codes each method can return, see the individual method descriptions.

DS_OK
Everything worked.
DSERR_ALLOCATED
The requested object, service, or priority level is already allocated.
DSERR_ALREADYINITIALIZED
The object has already been initialized — it cannot be reinitialized.
DSERR_BADFORMAT
DirectSound cannot create a sound buffer of the specified wave format.
DSERR_BUFFERLOST
The sound buffer memory has been lost.
DSERR_CONTROLUNAVAIL
This function is not available on this sound buffer (probably because you did not specify the flag for this control in the DSBUFFERDESC structure when creating the buffer). For information about the DSBUFFERDESC structure, see the DirectX SDK documentation.
DSERR_GENERIC
An unspecified error has occurred and the operation can not be completed.
DSERR_INVALIDPARAM
This function was called with incorrect parameters.
DSERR_INVALIDCALL
The function request cannot be performed by the sound buffer in its current state or configuration.
DSERR_NODRIVER
The requested DirectSound driver could not be created.
DSERR_NOINTERFACE
The requested interface is not available.
DSERR_OUTOFMEMORY
The DirectSound object is out of memory.
DSERR_UNSUPPORTED
Returned by API functions that are not supported in this release.
