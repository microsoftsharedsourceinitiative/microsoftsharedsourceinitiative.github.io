Ordering of Commands

The Order group box contains 13 controls which you use to specify the order of the page-control commands. Some printers (such as the Hewlett-Packard printers that use PCL) require a specific ordering of the page-control commands in order to process pages correctly; other printers (such as the Diconix) work satisfactorily using the default ordering. Once you've entered the appropriate commands for your printer, you also need to specify the order in which the Universal Printer Driver should send them to the printer.

The default values that appear in these controls begin with the number 1 in the upper-left corner and proceed down the left column, to the top of the right column and then down the right column.

Diconix 150 Plus

The Diconix printer uses the default values.

HP LaserJet IIP

The HP LaserJet IIP printer requires the unique ordering specified in the following list:

  • PC_ORD_BEGINDOC
  • PC_ORD_PAPER_SOURCE
  • PC_ORD_PAPER_SIZE
  • PC_ORD_ORIENTATION
  • PC_ORD_RESOLUTION
  • PC_ORD_BEGINPAGE
  • PC_ORD_MULT_COPIES
  • PC_ORD_DUPLEX
  • PC_ORD_DUPLEX_TYPE
