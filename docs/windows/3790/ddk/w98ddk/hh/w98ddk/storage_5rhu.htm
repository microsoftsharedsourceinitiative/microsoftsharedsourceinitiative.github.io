ISP_DEV_REMOVED

#include <isp.h>

typedef struct ISP_DEV_REMOVED {  //
    ISP    ISP_d_removed_hdr;      // standard header
    PVOID  ISP_dev_removed_pdcb;   // physical DCB pointer
    ULONG  ISP_dev_removed_flags;  // flags; see below
} ISP_DEV_REMOVED, *PISP_DEV_REMOVED;

Contains information for a ISP_DEVICE_REMOVED function.

ISP_dev_removed_flags
Can be ISP_DEV_REMOVED_FL_MEDIA_ONLY to indicate only the medium is going away.
