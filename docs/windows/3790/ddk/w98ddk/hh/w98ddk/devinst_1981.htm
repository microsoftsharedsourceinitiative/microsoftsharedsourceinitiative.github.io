NDI_NDIDESTROY

NDI_NDIDESTROY

The NDI_NDIDESTROY message is sent when NETDI is about to deallocate the object.

This message has no parameters.

  • The return value is zero (OK) if successful; otherwise it is a nonzero NDI error value.
