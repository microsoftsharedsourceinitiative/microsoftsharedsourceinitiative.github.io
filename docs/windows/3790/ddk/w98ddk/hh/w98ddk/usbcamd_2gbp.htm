USBCAMD_SelectAlternateInterface

USBCAMD_SelectAlternateInterface is called by the cam minidriver to select an alternate interface. The RequestInterface parameter is filled in with the information from the SELECT_INTERFACE request on successful completion. This function is typically called from the cam minidriver in response to a request to allocate or free bandwidth.

NTSTATUS
  USBCAMD_SelectAlternateInterface(
    IN PVOID  DeviceContext,
    IN OUT PUSBD_INTERFACE_INFORMATION  RequestInterface
    );

Parameters

DeviceContext
Device-specific context.
RequestInterface
Appropriate interface to select.

Return Values

Returns the status from the SELECT_INTERFACE request.
