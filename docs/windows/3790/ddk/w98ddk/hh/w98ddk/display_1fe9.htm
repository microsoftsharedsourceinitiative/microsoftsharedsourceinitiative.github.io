Virtual Display Device Services Reference

The virtual display device (VDD) is a VxD supplied by Microsoft that performs all standard virtualization of display adapters. This section provides detailed descriptions of the VDD services that a virtual display minidriver (mini-VDD) may use. For an overview of the Virtual Display subsystem and information on VDD functions for use by display drivers for information, see about implementing a mini-VDD, see About Virtual Display Minidrivers.

This section provides detailed reference information for the VDD services. For the most part, a Windows 95 mini-VDD will only need the new Windows 95 VDD services. However, some of the Windows 3.x VDD services are still available for use by mini-VDDs.
