Delete All Fonts Box

The DEL_ALL_FONTS control contains the command which deletes all downloaded fonts from the printer's memory.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Font Control command (ESC*cnF) deletes downloaded fonts, makes downloaded fonts temporary, makes downloaded fonts permanent, and so on. If n is 0, all downloaded fonts are deleted.

To specify the Font Control command which deletes all downloaded fonts, type the following in the DEL_ALL_FONTS box:

\x1B*c0F

After entering the font information for your printer from the Printer Data menu, you choose the ModelData command and enter the model-specific data for your printer.
