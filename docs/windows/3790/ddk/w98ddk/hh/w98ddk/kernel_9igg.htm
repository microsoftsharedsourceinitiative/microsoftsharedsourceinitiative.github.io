VPICD_Test_Phys_Request

include vpicd.inc

mov     eax, IRQHandle
VxDcall VPICD_Test_Phys_Request
jc      irq_set

Determines whether the physical (hardware PIC) interrupt request is set for the specified IRQ. Uses Flags.

  • Clears the carry flag if the physical interrupt request is set.
IRQHandle
Handle of the IRQ.
