GetSystemMetrics

int
  GetSystemMetrics(
    int  nIndex
    );

Parameters

nIndex
Specifies the system metric or configuration setting to retrieve.

The following metrics refer only to the primary monitor:

Metric Description
SM_CXSCREEN, SM_CYSCREEN Width and height, in pixels, of the screen. Same as GetDeviceCaps(hdcPrimaryMonitor, HORZRES/VERTRES).
SM_CXFULLSCREEN, SM_CYFULLSCREEN Width and height of the client area for a full-screen window.
SM_CXMAXIMIZED, SM_CYMAXIMIZED Default dimensions, in pixels, of a maximized top-level window.

The following metrics refer to the entire desktop:

Metric Description
SM_CXMAXTRACK, SM_CYMAXTRACK Default maximum dimensions, in pixels, of a window that has a caption and sizing borders. The user cannot drag the window frame to a size larger than these dimensions. A window can override these values by processing the WM_GETMINMAXINFO message.

The following metrics have been added so the user can get the dimensions of the virtual screen:

Metric Description
SM_XVIRTUALSCREEN Left of the virtual screen.
SM_YVIRTUALSCREEN Top of the virtual screen.
SM_CXVIRTUALSCREEN Width of the virtual screen.
SM_CYVIRTUALSCREEN Height of the virtual screen.

The following metric has been added to help an application optimize color management:

Metric Description
SM_SAMEDISPLAYFORMAT Returns true if all the monitors have the same color format. Note that two displays can have the same bit depth, but different color formats, if the red, green, and blue pixels have different sizes or are located in different places in a pixel.

The following metric has been added for its general usefulness:

SM_CMONITORS Returns the number of monitors on the desktop.
