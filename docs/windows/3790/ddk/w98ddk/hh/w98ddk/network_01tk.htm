Where to Find More Information on Windows 95 Network Architecture

A good introduction to Windows 95 networking is in the Windows 95 Resource Kit. The CD-ROM version of Windows 95 contains the Windows 95 Resource Kit as a help file. The help file is located in the Admin\Reskit\Helpfile directory on the CD-ROM and is called Win95rk.hlp. Open this file and select the Windows 95 Reference section and then Windows 95 Network Architecture.

For a good overview of how the various components interface with the operating system, look in the MSDN Development Library CD.
