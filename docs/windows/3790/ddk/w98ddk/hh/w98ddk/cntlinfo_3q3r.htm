Enabling Duplex-Off Printing

The PC_OCD_DUPLEX_OFF box contains the command or commands which the Universal Printer Driver issues to disable duplex (double-sided) printing.

Diconix 150 Plus

The Diconix printer does not support duplex printing; therefore, the PC_OCD_DUPLEX_OFF box is empty.

HP LaserJet IIP

For the HP LaserJet IIP, the Duplex Off command (ESC&l0S) is required to disable duplex printing.

To specify the Duplex Off command, type the following in the PC_OCD_DUPLEX_OFF box:

\x1B&l0S
