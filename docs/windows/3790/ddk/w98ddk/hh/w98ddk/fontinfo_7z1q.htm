Enabling Underline Simulation

The FS_OCD_UNDERLINE_ON box contains the command which enables underline simulations for a printer.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Underline Mode command (ESC-n) enables and disables the underline font simulation. If n is 1, the font simulation is enabled; if n is 0, the font simulation is disabled.

To specify the Underline Mode command, type the following in the FS_OCD_UNDERLINE_ON box:

\x1B-\x01

HP LaserJet IIP

For the HP LaserJet IIP printer, the Underline command (ESC&dnD) enables fixed- and floating-position underlining. If n is 0, fixed-position underlining is enabled; if n is 1, floating-position underlining is enabled. The Universal Printer Driver will perform underline simulations.

To specify the Underline command, type the following in the FS_OCD_UNDERLINE_ON box:

\x1B&dD
