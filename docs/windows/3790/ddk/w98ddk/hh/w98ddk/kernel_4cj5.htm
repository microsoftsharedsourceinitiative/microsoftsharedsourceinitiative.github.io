_VPOWERD_Restore_Power_On_Defaults

POWERRET _VPOWERD_Restore_Power_On_Defaults(VOID);

Reinitializes all APM BIOS power-on defaults. This service is intended to be used only by the operating system.

  • Returns PR_SUCCESS if successful, or one of the error values beginning with PR_ listed in VPOWERD.H.
