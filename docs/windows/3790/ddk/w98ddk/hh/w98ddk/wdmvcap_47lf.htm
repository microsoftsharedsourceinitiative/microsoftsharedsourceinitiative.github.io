KSPROPERTY_CROSSBAR_CAPS

The KSPROPERTY_CROSSBAR_CAPS read-only property is used to retrieve the total number of input and output pins on the crossbar. It uses the following structure:

typedef struct {
  KSPROPERTY  Property;
  ULONG  NumberOfInputs;
  ULONG  NumberOfOutputs;
} KSPROPERTY_CROSSBAR_CAPS_S, *PKSPROPERTY_CROSSBAR_CAPS_S;

Members

Property
Specifies the property used.
NumberOfInputs
Specifies the number of audio and video input pins.
NumberOfOutputs
Specifies the number of audio and video output pins.
