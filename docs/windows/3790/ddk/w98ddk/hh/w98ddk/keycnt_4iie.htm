AnsiToOemBuff

void AnsiToOemBuff(
  LPSTR lpSrc, 
  LPSTR lpDst, 
  WORD nCount
 );

Translates a string of ANSI characters to OEM. However, use this function to translate fixed-length byte arrays, such as database records, which may contain null characters.

  • This function has no return value.
lpSrc and lpDst
Address of the input ANSI string, and address of the output OEM string.
nCount
Byte count.

The export ordinal for this function is 134.

The lpSrc and lpDst parameters may point to the same buffer.
