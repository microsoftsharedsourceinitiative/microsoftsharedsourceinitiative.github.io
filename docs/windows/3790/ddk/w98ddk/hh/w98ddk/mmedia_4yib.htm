JDD_GETNUMDEVS

JDD_GETNUMDEVS
lParam1 = 0;
lParam2 = 0;

Determines the number of devices supported by the driver.

  • Returns the maximum number of devices supported by the driver. This value does not necessarily match the number of devices physically connected to the computer.
