Text Quality

The Text Qualities dialog box contains controls which you use to identify a text quality and to specify the printer command which selects that text quality. Typically, text quality only varies for dot-matrix printers. Laser printers, such as the HP LaserJet IIP provide one text quality or resolution. The examples in this section for the Diconix 150 Plus printer are based on the commands and data corresponding to letter-quality output.
