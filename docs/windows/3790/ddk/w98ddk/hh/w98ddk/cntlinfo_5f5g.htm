Absolute-Horizontal Cursor Movement

The CM_OCD_XM_ABS box contains the printer command or commands which move the cursor horizontally with respect to the page origin. For most printers, the corresponding Units dialog box will contain values that the Universal Printer Driver will use to calculate the argument(s) for the absolute horizontal cursor-movement command.

Diconix 150 Plus

The Diconix 150 Plus printer does not support an absolute horizontal cursor-movement command; therefore, the CM_OCD_XM_ABS box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Horizontal Cursor Positioning command (ESC*pnX) moves the cursor to a new position with respect to the page origin. The variable n specifies a horizontal distance in 1/300 inch units.

To specify the absolute Horizontal Cursor Positioning command, type the following in the CM_OCD_XM_ABS box:

\x1B*p%dX
