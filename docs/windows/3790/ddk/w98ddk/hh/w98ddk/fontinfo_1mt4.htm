Select Font ID Box

The SELECT_FONT_ID box contains the command which selects the downloadable font identified by a particular integer value.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Font Selection by ID command (ESC(nX) selects the downloaded font with the ID value of n.

To specify the Font Selection by ID command, type the following string in the SELECT_FONT_ID box:

\x1B(%dX
