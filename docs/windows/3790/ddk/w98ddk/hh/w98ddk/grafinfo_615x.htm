Compression Type

The Compression Type box identifies the type of compression commands which appear in the dialog.

Diconix 150 Plus

The Diconix 150 Plus printer does not support compression; therefore the Compression Type box is empty.

HP LaserJet IIP

The HP LaserJet IIP printer supports run-length encoding as specified in the Compression Type box.
