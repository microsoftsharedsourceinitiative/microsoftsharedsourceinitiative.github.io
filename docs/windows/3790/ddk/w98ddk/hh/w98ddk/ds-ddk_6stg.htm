IDsDriver

The IDsDriver interface is the main control interface for the driver. It is used to create (or query for) other interfaces. The driver registers the vtable for this interface when it calls DSOUND_RegisterDeviceDriver in response to the CONFIG_START message from the Configuration Manager.

The IDsDriver interface contains the following methods:

  • AddRef
  • Close
  • CreateSoundBuffer
  • DuplicateSoundBuffer
  • GetCaps
  • GetDriverDesc
  • Open
  • QueryInterface
  • Release
