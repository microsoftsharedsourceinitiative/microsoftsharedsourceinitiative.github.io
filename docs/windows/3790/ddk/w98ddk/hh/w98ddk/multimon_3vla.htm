Multimonitor Recommendations for Graphics System Design

This section describes known problems with current graphics system designs, and offers suggestions for fixing these problems. It also summarizes PC 97 requirements for multimonitor support. As indicated in the suggested solutions, an important requirement is that a PCI graphics adapter on the motherboard behave the same as any other PCI graphics adapter.
