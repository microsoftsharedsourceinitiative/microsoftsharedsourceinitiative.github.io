EngineDeleteFont

The EngineDeleteFont function deletes an engine font. A driver calls this function, while processing a call to its RealizeObject function, to delete a font that was previously realized using the EngineRealizeFont function.

BOOL
  EngineDeleteFont(
    LPFONTINFO lpFont
    );

Parameters

lpFont
Points to a structure containing information defining the font to delete.

Return Value

The EngineDeleteFont function returns TRUE if it successfully deletes the font. Otherwise, it returns FALSE.
