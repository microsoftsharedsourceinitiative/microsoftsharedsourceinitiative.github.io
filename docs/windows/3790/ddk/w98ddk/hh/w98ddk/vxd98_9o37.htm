VPOWERD_Get_Battery_Unit_Status

VPOWERD_Get_Battery_Unit_Status returns the current battery status.

POWERRET VPOWERD_Get_Battery_Unit_Status( 
  DWORD   BatteryDeviceObject,
  Power_Status   pPower_Status
);

Parameters

BatteryDeviceObject
WDM DeviceObject for the battery.
pPower_Status
Structure to be completed with the relevant statuses.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
