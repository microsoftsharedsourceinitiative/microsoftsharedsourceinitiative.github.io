Other Tools

These tools are all for use on Microsoft® Windows® 98/Me. For debugging on NT-based operating systems in user mode, you should use WinDbg, CDB, NTSD, or the Microsoft Visual Studio® debugger. For debugging on NT-based operating systems in kernel mode, you should use WinDbg or KD. See Debugging Tools for NT-Based Operating Systems for more information on these tools.
