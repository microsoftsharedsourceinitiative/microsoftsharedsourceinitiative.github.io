IFSMgr_Get_NetTime

This service is provided for FSDs to find the time elapsed in seconds. This is the net time format.

Nothing.
[EAX] Number of seconds elapsed since 1/1/1970 with the time expressed in UTC.
[EDX] Millisecond remainder in the 1 second interval.

Register Usage

EAX, ECX, EDX, Flags.
