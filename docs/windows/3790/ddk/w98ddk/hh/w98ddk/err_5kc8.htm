The RESOLUTION Dialog Box

The following error messages are associated with the RESOLUTION dialog box.








List
Description





Every RESOLUTION must have a format string
UniTool issues this message if a format string for the selected resolution is not entered.





minidriver name Illegal Text X DPI value
UniTool issues this message if an invalid horizontal text resolution is entered.





minidriver name Illegal Text Y DPI value
UniTool issues this message if an invalid vertical text resolution is entered.





minidriver name Illegal Graphics X DPI value
UniTool issues this message if an invalid horizontal graphics resolution is entered.





minidriver name Illegal Graphics Y DPI value
UniTool issues this message if an invalid vertical graphics resolution is entered.





minidriver name: sNPins must be 1,8,16,32,40,48...
UniTool issues this message if an illegal value is entered in the sNPins box.





minidriver name: sPinsPerPass must be 1,8,16,32,40,48...
UniTool issues this message if an illegal value is entered in the sPinsPerPass box.





minidriver name: sMinBlankSkip must be >= 0
UniTool issues this message if an illegal value is entered in the sMinBlankSkip box.





minidriver name :sMinBlankSkip should be > 0
UniTool issues this message if a negative value is entered in the sMinBlankSkip box and the RES_BO_ENCLOSED_BLNKS box is checked in the fBlockOut dialog box.





minidriver name: Spot Diameters must be > 0
UniTool issues this message if a negative value is entered in the sSpotDiameter box.





minidriver name: Unknown iDitherBrush
UniTool issues this message if an undefined value is entered in the iDitherBrush box.





minidriver name: RES_DM_GDI conflicts with sNPins or sPinsPerPass
UniTool issues this message if a value greater than 1 is entered for sNPins or sPinsPerPass and the RES_DM_GDI box is checked in the fBlockOut dialog box.
