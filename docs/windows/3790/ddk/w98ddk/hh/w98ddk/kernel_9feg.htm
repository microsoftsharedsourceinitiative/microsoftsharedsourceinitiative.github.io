VDMAD_UnMask_Channel

include vdmad.inc

mov eax, DMA_Handle
mov ebx, VMHandle
VxDcall VDMAD_UnMask_Channel

Physically unmasks a channel so that DMA transfers can proceed. Uses flags

  • No return value.
DMA_Handle
Handle of the DMA.
VMHandle
Handle of the virtual machine.
