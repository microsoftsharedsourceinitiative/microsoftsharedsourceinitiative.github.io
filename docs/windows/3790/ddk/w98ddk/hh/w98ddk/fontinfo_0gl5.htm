Predefined Text Quality

The Text Quality Name box appears adjacent to the Predefined button. This box lists five common text qualities:

  • Letter Quality
  • Near-Letter Quality
  • Memo
  • Draft Quality
  • Text Quality

Diconix 150 Plus

For the Diconix 150 Plus printer, Letter Quality appears in the box.

HP LaserJet IIP

The Text Quality Name box is empty.
