The ModelData Dialog Box

The following error messages are associated with the ModelData dialog box.

List Description
Every Model must havea name! UniTool issues this message if a string is not entered in the Name of Model box.
sLeftMargin must be non-negative UniTool issues this message if a negative value is entered in the sLeftMargin box.
sMaxPhysWidth must be > 0 UniTool issues this message if a value less than or equal to zero is entered in the sMaxPhysWidth box.
Max Page Width must be equal to 0 or equal -1 UniTool issues this message if a value that is > 0 or less than -1 is entered in the Max-Width page size limits box.
Max Page Height must be > 0 or equal -1 UniTool issues this message if a value that is equal to 0 or less than -1 is entered in the Max-Height page size limits box.
Min Page Width must be > 0 UniTool issues this message if a value that is less than or equal to 0 is entered in the Min-Width page size limits box.
MODELDATA Min Page Height must always be > 0 UniTool issues this message if a value that is less than or equal to 0 is entered in the Min-Height page size limits box.
Default Font ID must refer to a valid font UniTool issues this message if a value in the Default Font ID box is entered that does not identify a valid font resource.
Default CTT ID must refer to a valid CTT UniTool issues this message if a value in the Default CTT ID box is entered that does not identify a valid character-translation table.
Look Ahead Region must be non-negative UniTool issues this message if a value that is negative is entered in the LookAhead Region box.
sCartSlots must be non-negative UniTool issues this message if a negative value is entered in the sCartSlots box.
minidriver name supports Font Cartridges, but has no Cartridge slots UniTool issues this message if the developer enters the value 0 in the sCartSlots box and selected one or more font cartridges in the supported-attributes Font Cartridges dialog box. The Font Cartridges dialog box is associated with the ModelData dialog box.
minidriver name has no PageControl! UniTool issues this message if the developer fails to identify supported page-control commands and data in the ModelData dialog box.
minidriver name has no CursorMove! UniTool issues this message if the developer fails to identify supported cursor-movement commands and data in the ModelData dialog box.
minidriver name has no Resolutions! UniTool issues this message if the developer fails to identify supported-resolution commands and data in the ModelData dialog box.
minidriver name has no Paper Sizes! UniTool issues this message if the developer fails to identify supported paper-size commands and data in the ModelData dialog box.
minidriver name: Memory Values not used for non-HPPCL devices UniTool issues this message if the minidriver chooses one of the listed memory configurations in the ModelData dialog box for a printer that does not use PCL.
