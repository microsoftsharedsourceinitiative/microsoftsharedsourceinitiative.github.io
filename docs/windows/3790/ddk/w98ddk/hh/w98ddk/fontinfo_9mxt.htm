Specifying Font-Cartridge Data

Some printers provide a slot for removable cartridges that contain device fonts. The Font Cartridge dialog box contains controls which you use to identify supported cartridges and the fonts which they contain.

The examples in the following sections are based on the Microsoft® 1A cartridge which is one of 33 cartridges supported by the HP LaserJet printer.
