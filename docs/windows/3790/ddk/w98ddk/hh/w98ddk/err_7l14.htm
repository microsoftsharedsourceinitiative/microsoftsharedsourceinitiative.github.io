The DOWNLOADINFO Dialog Box

The following error messages are associated with the DOWNLOADINFO dialog box.

List Description
minidriver name :Bitmap Font Descriptor must be > 0 UniTool issues this message if a bitmap font-descriptor value that is negative is entered.
minidriver name :Bitmap Char Descriptor must be > 0 UniTool issues this message if a bitmap character-descriptor value that is negative is entered.
minidriver name :fFormat DLI_FMT_PCL not set UniTool issues this message if the developer fails to check the DLI_FMT_PCL box for a PCL printer.
