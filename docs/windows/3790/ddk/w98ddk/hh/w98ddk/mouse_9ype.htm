Inquire

WORD Inquire(LPMOUSEINFO lpMouseInfo)

Retrieves information about the mouse hardware. The export ordinal for this function is 1.

  • Returns the size, in bytes, of a MOUSEINFO structure.
lpMouseInfo
Address of a MOUSEINFO structure that contains information about the mouse hardware that is present, including the number of buttons on the mouse, and the rate at which the mouse can issue interrupts.

See also MOUSEINFO
