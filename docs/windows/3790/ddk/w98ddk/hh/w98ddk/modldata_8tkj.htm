Default and Supported Font Cartridges

The Font Cartridges dialog box identifies the default and supported font cartridges.

Control Description
Default Specifies the default font cartridge.
List box Lists the available font cartridges for this printer.
Enable All Enables all font cartridges which are identified in the list box.
Inhibit All Inhibits all font cartridges which are identified in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus printer does not support detachable font cartridges.
