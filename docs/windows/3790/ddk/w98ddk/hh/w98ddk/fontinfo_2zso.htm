Extended Text Metrics Dialog Box

You may also enter optional values for extended text metrics. Only applications that specifically request this information from a driver will use the values entered here. To display the Extended Text Metrics dialog box, press the Ext.TextMetrics button. For more information about the controls in this dialog box, see the Microsoft Windows Printers and Fonts Kit.
