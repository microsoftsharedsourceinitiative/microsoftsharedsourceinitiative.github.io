Relative Horizontal-and-Vertical Movement

The CM_OCD_XY_REL box contains the printer command or commands which move the cursor horizontally and vertically with respect to the current cursor position. For some printers, the corresponding Units dialog box contains values by which the relative horizontal-and-vertical movement command arguments must be multiplied or divided. The Units dialog box also contains values which must be added to command arguments, maximum values for command arguments, and minimum values for command arguments. This command is not supported by version 1.0 of the Microsoft Windows Universal Printer Driver.
