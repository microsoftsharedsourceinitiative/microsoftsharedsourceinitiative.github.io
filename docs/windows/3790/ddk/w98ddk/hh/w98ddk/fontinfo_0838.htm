Font Weight

The Weight control contains a value that specifies the relative density or weight of characters rendered using the font. Valid weights can range from 100 to 1000 using increments of the value 100; 400 specifies normal weight, 700 specifies bold, and so on. Valid weights must be a multiple of 100.

Diconix 150 Plus

For the Diconix 150 Plus printer, this value is 400 since the bitmaps for the sample font render normal characters.
