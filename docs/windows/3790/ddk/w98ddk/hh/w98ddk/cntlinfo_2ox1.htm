Ending the Page

The PC_OCD_ENDPAGE box contains the printer command or commands which the Universal Printer Driver issues to end a page.

Diconix 150 Plus

For the Diconix 150 Plus, no command is required at the end of each page; therefore, the PC_OCD_ENDPAGE box is empty.

HP LaserJet IIP

For the HP LaserJet IIP, no command is required at the end of each page; therefore, the PC_OCD_ENDPAGE box is empty.
