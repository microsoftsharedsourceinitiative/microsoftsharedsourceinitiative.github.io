Disabling Bold Simulation

The FS_OCD_BOLD_OFF box contains the command which disables bold simulations for a printer.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Emphasized Mode Off command (ESCF) disables bold simulations.

To specify the Emphasized Mode Off command, type the following in the FS_OCD_BOLD_OFF box:

\x1BF

HP LaserJet IIP

For the HP LaserJet IIP printer, the Stroke Weight command (ESC(snB) specifies the thickness of the strokes that compose a character. If n is -3, the stroke is thin; if n is 0, the stroke is normal; if n is 3, the stroke is bold.

To specify the Stroke Weight command which disables bold output, type the following in the FS_OCD_BOLD_OFF box:

\x1B(s0B
