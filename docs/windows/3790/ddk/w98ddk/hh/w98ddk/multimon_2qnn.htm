GET_ADAPTER_POWER_STATE_CAPS

GET_ADAPTER_POWER_STATE_CAPS returns the valid power states of the display adapter or chip set.

Entry Meaning
ESP+4 Device node.

Exit Meaning
EAX Bitmask of the following:

CM_POWERSTATE_D0
CM_POWERSTATE_D1
CM_POWERSTATE_D2
CM_POWERSTATE_D3
