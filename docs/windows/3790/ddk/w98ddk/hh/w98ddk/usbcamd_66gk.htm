USBCAMD_AdapterReceivePacket

USBCAMD_AdapterReceivePacket is called by the cam minidriver from the AdapterReceivePacket function to allow USBCAMD to process an SRB.

PVOID
  USBCAMD_AdapterReceivePacket(
    IN PHW_STREAM_REQUEST_BLOCK  Srb,
    IN PUSBCAMD_DEVICE_DATA  DeviceData,
    IN PDEVICE_OBJECT  *DeviceObject,
    IN BOOLEAN  NeedsCompletion
    );

Parameters

Srb
SRB passed to the cam minidriver's AdapterReceivePacket routine.
DeviceData
USBCAMD_DEVICE_DATA structure with entry points to the cam minidriver.
DeviceObject
Pointer to the device’s PDO.
NeedsCompletion
If TRUE, processes and completes the SRB request. Otherwise, USBCAMD ignores the SRB request.

Return Values

Returns a pointer to the device-specific context for this instance of the camera Devicecontext.
