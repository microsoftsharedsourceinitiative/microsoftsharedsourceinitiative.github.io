Identifying Strikeout Fonts

The Strikeout control indicates whether the PFM file contains data for a strikeout font. The control should be set to 1 if it does; otherwise it should be set to 0.

Diconix 150 Plus

For the Diconix 150 Plus printer, this value is 0 since the font used for the example is not a strikeout font.
