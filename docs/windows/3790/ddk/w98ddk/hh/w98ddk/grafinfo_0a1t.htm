General Data

Choosing the fGeneral button displays the fGeneral dialog box. This dialog box contains four boxes which specify general rectangle-fill data.

The following table provides a brief description of each box.

Box Description
RF_WHITE_ABLE If this box is checked, the printer supports a white-fill command; if this box is not checked, the printer does not support this command.
RF_MIN_IS_WHITE If this box is checked, the minimum gray scale supported by the printer is actually white; if this box is not checked, the minimum gray scale contains black pixels.
RF_CUR_X_END If this box is checked, the cursor is aligned with the right edge of the rectangle after a rectangle-fill operation occurs. If this box is not checked, the cursor is positioned at the rectangle origin after the fill occurs.
RF_CUR_Y_END If this box is checked, the cursor is aligned with the bottom edge of the rectangle after a rectangle-fill operation occurs. If this box is not checked, the cursor is positioned at the rectangle origin after the fill occurs.

Diconix 150 Plus

None of the boxes are checked.

HP LaserJet IIP

The RF_WHITE_ABLE box is checked to indicate that the HP LaserJet IIP can use the White-fill command.
