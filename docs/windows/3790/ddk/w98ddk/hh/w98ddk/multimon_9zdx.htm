SET_MONITOR_POWER_STATE

SET_MONITOR_POWER_STATE sets the power state of the monitor. If this function is not implemented, the VDD assumes that the monitor can implement VESA DPMS functionality, and calls the display BIOS to set the power state of the monitor directly.

Entry Meaning
ESP+4 Device node.
ESP+8 Power state to set. One of:
CM_POWERSTATE_D0
CM_POWERSTATE_D1
CM_POWERSTATE_D2
CM_POWERSTATE_D3

Exit Meaning
EAX CR_DEFAULT

Routine did nothing and the main VDD should do a
VESA DPMS call to set the monitor power state.

CR_SUCCESS

Routine set the appropriate power state for the monitor.
