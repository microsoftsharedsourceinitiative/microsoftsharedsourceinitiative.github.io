Average Width

The Average Width control specifies the average character width for proportionally spaced and fixed-pitch fonts. If the font is a proportionally spaced font, UniTool will calculate this value based on the information entered in the font-width table. If the font is a fixed-pitch font, the developer must enter this value (which is equivalent to the width of each character).

Diconix 150 Plus

For the Diconix 150 Plus printer, this value is 57.
