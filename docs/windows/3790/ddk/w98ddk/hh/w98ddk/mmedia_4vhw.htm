MM_DRVM_CLOSE

Notifies a video-callback function or window that the video capture driver has received a DVM_STREAM_FINI message.

  • No return value.
dwParam1, dwParam2
Not used.

This message is used by video capture drivers, installable compressors, and other types of installable drivers whenever a device is closed.
