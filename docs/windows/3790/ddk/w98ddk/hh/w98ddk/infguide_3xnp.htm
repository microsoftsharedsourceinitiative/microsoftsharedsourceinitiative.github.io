Windows Me INF File Guide

This section includes basic information about INF files for Microsoft® Windows® 95/98/Me.

A basic INF file will have the following sections (more complex INF files will have additional sections). Bold section names are actual keywords. Italicized section names are arbitrary names created by the author of the INF file:

Section Description
[Version] Basic version information for validating the INF file.
[DefaultInstall] Installation section that is executed by default. Has pointers to other sections specifying, for example, what files to copy and delete, registry updates, and .ini file updates.
[OtherInstall] Uses same format as [DefaultInstall] section, but must be explicitly called. Useful for defining how a component should be uninstalled.
[DestinationDirs] Specifies the location on the hard drive where a section's files will be copied, deleted or renamed (for example, Windows dir or Windows\System dir.)
[OptionalComponents] Lists Install sections that are displayed when the user selects the INF file from the "Have Disk…" button during Windows Setup.
[FileCopy/Delete/RenameSection] Lists files to be copied, deleted or renamed.
[RegistryUpdateSection] Specifies registry additions or deletions.
[IniFileUpdateSection] Specifies updates to .ini files. Links are created in this section.
[SourceDisksNames] Lists disks that contain the files.
[SourceDisksFiles] Lists the specific disk where each file is located.
[Strings] Lists localizable strings used above.
