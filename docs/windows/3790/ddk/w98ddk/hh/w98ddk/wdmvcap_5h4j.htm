KSPROPERTY_TUNER_MODE_CAPS

The KSPROPERTY_TUNER_MODE_CAPS read-only property identifies the mode capacities of the tuner. In response to this property request, the driver returns the following structure.

typedef struct {
  KSPROPERTY Property;
  ULONG  Mode;
  ULONG  StandardsSupported;
  ULONG  MinFrequency;
  ULONG  MaxFrequency;
  ULONG  TuningGranularity;
  ULONG  NumberOfInputs;
  ULONG  SettlingTime;
  ULONG  Strategy;
} KSPROPERTY_TUNER_MODE_CAPS_S, *PKSPROPERTY_TUNER_MODE_CAPS_S;

Members

Property
Specifies the property used.
Mode
Specifies a KSPROPERTY_TUNER_MODES value.
StandardsSupported
Specifies the video standards supported by the device. Support for multiple standards by bitwise OR of KS_AnalogVideo_*
MinFrequency
Specifies the lowest frequency supported by the tuner expressed in Hz.
MaxFrequency
Specifies the highest frequency supported by the tuner expressed in Hz.
TuningGranularity
Specifies the granularity of tuning expressed in Hz.
NumberOfInputs
Specifies the number of inputs on the tuner.
SettlingTime
Specifies the settling time when changing frequencies expressed in milliseconds.
Strategy
Specifies the method of tuning using KS_TUNER_STRATEGY.
