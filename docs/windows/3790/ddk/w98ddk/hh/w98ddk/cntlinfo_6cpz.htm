Disabling Unidirectional Printing

The CM_OCD_UNI_DIR_OFF box contains the printer command or commands which disable unidirectional printing. (In order to enable unidirectional printing, the RES_BO_UNIDIR box in the fBlockOut resolution dialog box must be checked. If this box is not checked, the Universal Printer Driver will ignore these commands.)

Diconix 150 Plus

For the Diconix 150 Plus printer, the Unidirectional Printing Off command (ESCU0) disables unidirectional printing.

To specify the Unidirectional Printing Off command, type the following in the CM_OCD_UNI_DIR_OFF box:

\x1BU\x00
