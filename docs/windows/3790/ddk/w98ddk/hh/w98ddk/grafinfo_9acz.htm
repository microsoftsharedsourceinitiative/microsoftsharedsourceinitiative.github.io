Begin Graphics

The Begin Graphics box contains the command which is sent to the printer to enable graphics output. (Those printers that provide a specific command to select a graphics resolution may also provide a command to enter graphics mode.)

Diconix 150 Plus

The Diconix 150 Plus printer automatically begins rendering graphics when it receives a Send Block command; therefore, the Begin Graphics box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Start Raster Graphics command (ESC*rnA) signals the start of raster graphics and specifies the location of the left margin. If n is 1, the current cursor location defines the left margin for the graphics output; if n is 0, the left edge of the page defines the left margin for the graphics output.

To specify the Start Raster Graphics command using the current cursor location as the left margin, type the following in the Begin Graphics box:

\x1B*r1A
