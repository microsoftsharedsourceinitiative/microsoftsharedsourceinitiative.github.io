Windows 98 Driver Operation on Windows 95

For the Windows 98 driver to operate on Windows 95, the driver must operate as a current Windows 95 driver. The following describes the driver operation switching method.

To determine whether the operating system is Windows 98 or Windows 95, the driver verifies the Virtual Machine Manager (VMM) version prior to miniVDD registration using the following call to the VMMCall macro.

VMMCall Get_VMM_Version
EAX Value Meaning
< 040Ah The driver operates as a current Windows 95 driver. NBR_MINI_VDD_FUNCTIONS_40 is set as the function value (ECX) when the VDD_Register_Mini_VDD service call is made in miniVDD registration.
≥ 040Ah The driver operates as a Windows 98 driver. NBR_MINI_VDD_FUNCTIONS_41 is set as the function value (ECX) when the VDD_Register_Mini_VDD service call is made in miniVDD registration.
