CamConfigure

CamConfigure configures the ISO streaming interface, which is called just before the device is configured. At this point, USBCAMD can be instructed on which interface and alternate setting to use for the idle state. It is assumed that the device has a single interface with multiple alternate settings and each alternate setting has the same number of pipes.

NTSTATUS
  CamConfigure(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext,
    PUSBD_INTERFACE_INFORMATION  Interface,
    PUSBD_CONFIGURATION_DESCRIPTOR  ConfigurationDescriptor,
    PLONG  DataPipeIndex,
    PLONG  SyncPipeIndex
    );

Parameters

BusDeviceObject
Device object created by the hub. URBs can be submitted to the device through this device object.
DeviceContext
Minidriver device context.
Interface
USBD interface structure initialized with the proper values for select_configuration. This interface structure corresponds to a single ISO interface on the device. The minidriver is able to select a particular alternate setting and pipe.
ConfigurationDescriptor
USB configuration descriptor for this device.
DataPipeIndex
Indicates which pipe the data pipe is.
SyncPipeIndex
Indicates which pipe the sync pipe is, if one is needed.

Return Values

The return value is an NTSTATUS code.
