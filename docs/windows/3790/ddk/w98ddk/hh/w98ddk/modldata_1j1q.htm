Supported Download Information

The Download Info dialog box identifies the supported download-info commands. These commands download fonts to the printer. The following list describes each of the controls in this dialog box.

Control Description
List box Lists the available download-info commands and data for this printer.
Inhibit All Inhibits all download-info commands and data which are identified in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus does not support a download-info command.

HP LaserJet IIP

The HP LaserJet IIP does not support any of the three available download-info commands.
