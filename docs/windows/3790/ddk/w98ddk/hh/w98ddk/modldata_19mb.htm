Number of Cartridge Slots

The sCartSlots control contains a value which specifies the number of font-cartridge slots which appear on the printer.

Diconix 150 Plus

For the Diconix 150 Plus printer, the sCartSlots box contains the value 0.

HP LaserJet IIP

For the HP LaserJet IIP printer, the sCartSlots box contains the value 1.
