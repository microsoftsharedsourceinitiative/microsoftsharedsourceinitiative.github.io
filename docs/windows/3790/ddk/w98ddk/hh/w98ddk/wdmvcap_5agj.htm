DirectShow Filters Used with the Stream Class

There are four types of DirectShow™ filters that work with stream class drivers to perform video captures.

Type Purpose DLL
Video Capture Provides output streams of digital video data and ancillary data streams KSProxy.ax
TV Tuning Provides tuning control of TV, FM, and AM tuners KSTvTune.ax
Crossbar Provides routing of video and audio streams KSXBar.ax
TV Audio Provides control of TV audio such as SAP selection KSXBar.ax

Each of these functions appears in a DirectShow filter graph as a separate filter exposing unique control interfaces.
