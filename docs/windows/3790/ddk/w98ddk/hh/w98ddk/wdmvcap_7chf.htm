KSPROPERTY_VIDEOCONTROL_CAPS

The KSPROPERTY_VIDEOCONTROL_CAPS read-only property is used to identify the video control capabilities of the device. This property uses the following structure:

typedef struct {
  KSPROPERTY Property;
  ULONG    VideoControlCaps;      // KS_VideoControlFlags_*
} KSPROPERTY_VIDEOCONTROL_CAPS_S, *PKSPROPERTY_VIDEOCONTROL_CAPS_S;

Members

Property
Specifies the property used.
VideoControlCaps
Specifies the video control capabilities from the KS_VideoControlFlags_* of the property set.
