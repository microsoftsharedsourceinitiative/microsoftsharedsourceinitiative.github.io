Default and Supported Color Models

The Color Models dialog box identifies the default and supported color models. The following list describes each of the controls in this dialog box.

Control Description
Default Specifies the default color model.
List box Lists the available color models for this printer.
Enable All Enables all color models which are identified in the list box.
Inhibit All Inhibits all color models which are identified in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus does not support color output.

HP LaserJet IIP

The HP LaserJet IIP does not support color output.
