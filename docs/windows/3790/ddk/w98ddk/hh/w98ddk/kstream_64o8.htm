IRP_MJ_FLUSH

An object’s state can be flushed using IRP_MJ_FLUSH. This IRP is used to ensure that the object state and stream data have been flushed from permanent storage or completely processed by the underlying hardware. Any object can support this IRP through a KSDISPATCH_TABLE entry.
