IFSMgr_GetTimeZoneBias

IFSMgr_GetTimeZoneBias( )

This service returns the time zone bias. The time zone bias is the value that needs to be added to the local time to get UTC time. The time zone bias is returned in minutes.

Nothing.
  • Returns the time zone bias in minutes.
