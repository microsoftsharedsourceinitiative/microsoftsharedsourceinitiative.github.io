VxD_LOCKED_CODE_ENDS

include vmm.inc

VxD_LOCKED_CODE_ENDS

Defines the end of a code segment. Virtual devices use this macro in conjunction with the VxD_LOCKED_CODE_SEG macro to create segments for non-initialization code.

  • No return value.

This macro is equal to the VxD_CODE_ENDS macro.
