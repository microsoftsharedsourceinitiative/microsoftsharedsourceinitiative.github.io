The FONTSIMULATION Dialog Box

The following error messages are associated with the FONTSIMULATION dialog box.

List Description
minidriver name :sBoldExtra must be >= 0 UniTool issues this message if a negative value is entered in the sBoldExtra box.
minidriver name :sItalicExtra must be >= 0 UniTool issues this message if a negative value is entered in the sItalicExtra box.
minidriver name :sBoldItalicExtra must be >= 0 UniTool issues this message if a negative value is entered in the sBoldItalicExtra box.
