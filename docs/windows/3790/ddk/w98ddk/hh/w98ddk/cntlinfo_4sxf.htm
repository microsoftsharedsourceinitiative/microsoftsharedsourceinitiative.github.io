Formfeeds

The CM_OCD_FF box contains the printer command or commands which cause a formfeed.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Form Feed command (0x0C) causes a formfeed.

To specify the Form Feed command, type the following in the CM_OCD_FF box:

\x0C

HP LaserJet IIP

For the HP LaserJet IIP printer, the Form Feed command (0x0C) causes a formfeed.

To specify the Form Feed command, type the following in the CM_OCD_FF box:

\x0C
