Windows 98/Me: Windows DDK

WDM Audio Drivers

The Microsoft® Windows® 2000 DDK contains an updated version of the Windows Driver Model (WDM) audio development environment. Since this updated environment exists, the Windows® Me DDK does not support WDM audio driver development. Use the Windows 2000 DDK to create WDM audio drivers for Windows Me, Windows® 98 Second Edition (SE), and Windows 2000. Drivers written using the Windows 2000 DDK will run on Windows Me, Windows 98 SE, and Windows 2000.

For drivers that will be used exclusively on Windows 98 Gold (not Windows 98 SE), IHVs should write VxD drivers. Because most IHVs continue to support Windows® 95 (with VxDs), there is no extra work in supporting Windows 98.

The Windows DDK is available from the Windows DDK Web site.
