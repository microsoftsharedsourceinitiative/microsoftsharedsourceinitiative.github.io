VPOWERD_Disable_APM_Idle

VPOWERD_Disable_APM_Idle disables the APM idle. 

POWERRET VPOWERD_Disable_APM_Idle( VOID );

Parameters

None

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
