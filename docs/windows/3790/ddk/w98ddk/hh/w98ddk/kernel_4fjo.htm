VTD_Get_Interrupt_Period

include vtd.inc

VxDcall VTD_Get_Interrupt_Period
mov [Int_Period], eax

Retrieves the current timer interrupt frequency. Uses Flags.

  • Returns, in EAX, the length of time between ticks, in milliseconds.
