UniToUpper

UniToUpper(
 string_t pUniUpStr,
 string_t pUniStr,
 unsigned int length
 )

This service converts a mixed-case Unicode string into an uppercased Unicode string. For this service, the source and destination buffers can be the same, that is, the upcasing is done in place.

pUniUpStr
Supplies a flat pointer to the destination buffer to contain the upcased Unicode string.
pUniStr
Supplies a flat pointer to the Unicode string in mixed-case that needs to be upcased.
length
Supplies the length in bytes of the Unicode source string.
  • Returns the number of bytes in the uppercased Unicode string.
pUniUpStr Returns the upcased Unicode string.
