What to Do If Another Computer with a Serial Port is Not Available

You can use screen swapping. This requires that you set the resolution of your computer to 640 x 480 with 16 colors and add the following line to the [386Enh] section of System.ini: 

debugvga=1

To start the debugger, press the F12 key.
