DOSMGR_Get_Version

include dosmgr.inc
 
VxDcall DOSMGR_Get_Version

Retrieves the virtual MS-DOS manager version number. Uses EAX and Flags.

  • Returns the version number in EAX. AH contains the major version number, and AL contains the minor version number.
