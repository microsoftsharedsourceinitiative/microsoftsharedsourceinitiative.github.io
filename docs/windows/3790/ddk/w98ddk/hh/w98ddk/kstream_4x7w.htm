IRP_MJ_DEVICE_CONTROL

IRP_MJ_DEVICE_CONTROL is received by a device through a file object in order to support the predefined kernel streaming IOCTLs. The parameters depend on the subfunction passed. For more information, see Kernel Streaming I/O Control Codes.
