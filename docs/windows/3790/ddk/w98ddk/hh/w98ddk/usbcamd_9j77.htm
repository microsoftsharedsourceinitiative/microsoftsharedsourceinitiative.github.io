Limitations

Currently, USBCAMD handles only a camera with a single interface (with multiple alternate settings) controlling the video portion. Since all SRBs are seen by the cam minidriver before they go to USBCAMD, it should be possible to extend functionality within the cam minidriver. For example, the cam minidriver could support two streams: one managed by USBCAMD and one controlled independently.
