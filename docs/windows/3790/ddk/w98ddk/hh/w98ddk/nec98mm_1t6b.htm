Multimonitor Support for Display Adapters

Multimonitor support for display adapters is determined by the display driver. In the PC-9800 series, the VGA resource has been controlled by each display minidriver (miniVDD). This is the case in Windows 98 as well. However, resource conflict must be avoided, because all display adapters must be enabled for the multimonitor feature at the same time.
