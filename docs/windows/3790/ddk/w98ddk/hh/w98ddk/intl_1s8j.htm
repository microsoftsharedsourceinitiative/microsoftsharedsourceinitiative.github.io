IMC_SETSOFKBDFONT

An IME sends the IMC_SETSOFKBDFONT message to the soft keyboard window to specify a font to be used to display the reading characters of keys in the soft keyboard window. The UI window does not receive this message.

If successful, the return value is zero. Otherwise, it is nonzero.

lParam
Address of a LOGFONT structure that defines the font.
