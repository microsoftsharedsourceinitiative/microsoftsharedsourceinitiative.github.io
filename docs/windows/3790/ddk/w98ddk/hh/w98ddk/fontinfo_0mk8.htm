Device Fonts Character Widths Dialog Box

If the Pitch and Family value is set to indicate that you are editing a PFM file for a variable-pitch device font, you may enter values to describe the width of each character in the Device Font Character Widths dialog box. To display the dialog box, choose the FontWidth button. You may enter widths for all characters that fall between the First Char and Last Char ASCII character values already entered. The widths entered here are expressed in horizontal resolution units. If the font uses a character-translation table, the widths entered here should reflect the widths of the characters the printer actually produces when sent the values from the character-translation table.
