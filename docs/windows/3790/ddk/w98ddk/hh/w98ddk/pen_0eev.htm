VPEND_SETSAMPLINGRATE

Calls OEM-supplied, device-specific code to set the sampling rate.

  • If successful, returns the new sampling rate; otherwise, returns DRV_FAILURE.
lParam1
The low word contains the new sampling rate (the value that the virtual pen driver applies to the hardware).
lParam2
Should be set to zero.

The sample virtual pen driver sends a message with the requested new sampling rate to cOEM_API_Proc.
