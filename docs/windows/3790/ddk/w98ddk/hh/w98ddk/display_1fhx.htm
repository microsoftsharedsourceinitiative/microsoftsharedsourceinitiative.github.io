VDD_Set_HCurTrk

    include minivdd.inc
    VxDCall VDD_Set_HCurTrk

The VDD_Set_HCurTrk service sets the flag passed to VMDOSAPP indicating that VMDOSAPP should maintain cursor position within display window for this application. The keyboard driver calls this service when a keyboard interrupt is simulated into a virtual machine.

Parameters

EBX Specifies the virtual machine handle.

Return Values

This service has no return value.
