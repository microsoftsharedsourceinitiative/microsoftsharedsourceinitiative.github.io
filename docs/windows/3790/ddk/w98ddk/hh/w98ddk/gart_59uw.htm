PCIMP_GARTFlush

The PCIMP_GARTFlush service flushes the specified pages from the L2 cache. This service is always called after PCIMP_GARTMap.

VOID
  PCIMP_GARTFlush(
    PVOID  pHandle, 
    PVOID  pRefData, 
    ULONG  ulNumPages, 
    PVOID  pMapHandle, 
    ULONG  ulByteOffset, 
    PVOID  pSystemLin
    );

Parameters

pHandle
PCI device handle. This should be the same value used in the preceding PCIMP_GARTInit call.
pRefData
ULONG that was written by the PCIMP_GARTInit service.
ulNumPages
Specifies the number of pages. This should be the same value used in the PCIMP_GARTReserve service.
pMapHandle
Map handle obtained using the PCIMP_GARTReserve service.
ulByteOffset
Specifies the offset in bytes from the beginning of the memory block referred to by pMapHandle to the start of a page.
pSystemLin
Specifies the system linear address that was passed to a PCIMP_GARTMap service call.

Return Values

This service does not have a return value.

Comments

If the miniport’s PCIMP_GARTReset returned MP_FLUSH_L2_CACHE, the operating system will not flush the L2 cache by itself. If the miniport did not return MP_FLUSH_L2_CACHE, it should do nothing when this service is called.
