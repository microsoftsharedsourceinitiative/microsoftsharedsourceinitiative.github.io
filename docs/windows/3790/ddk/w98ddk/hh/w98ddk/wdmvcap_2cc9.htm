Videoport Category

Videoport Category streams are used to transfer image streams from an analog video decoder directly to DirectDraw surfaces through a hardware connection.

When creating the category PinName, PINNAME_VIDEO_VIDEOPORT, the following parameters must be filled out.

Parameter Value
KS Datarange KSDATARANGE
KS Dataformat KSDATAFORMAT
KS MajorFormat KSDATAFORMAT_TYPE_VIDEO
KS SubFormat (typical) KSDATAFORMAT_SUBTYPE_VPVideo
KS Specifier KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
Required Property Sets KSPROPSETID_VPConfig
Required Event Sets KSEVENTSETID_VPNotify
DS majortype None
DS formattype None
