Communications Port Drivers

Port drivers are virtual devices (VxDs) that the Windows 95 virtual communications driver (VCOMM) uses to access communications ports. Port drivers for common serial and LPT devices are included with Windows 95. Hardware vendors can create additional port drivers to enable Microsoft® Windows™-based applications and VxDs to use other communications devices.
