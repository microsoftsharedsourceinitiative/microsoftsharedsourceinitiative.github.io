Display Format

The Display Format box contains the conversion specification for the resolution string. The most common conversion specification is %d x %d. Values displayed with this string would appear as the following in a the Print Setup dialog: 320 x 96. In cases where two parameters are supplied in the display format string, the x-resolution will always be enumerated first. For more information about using device commands in UniTool, see Device Commands, in Overview of Microsoft Windows Printer Drivers.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Display Format box contains the following conversion specification:

%d x %d

HP LaserJet IIP

For the HP LaserJet IIP printer, the Display Format box contains the following conversion specification:

%d dots per inch
