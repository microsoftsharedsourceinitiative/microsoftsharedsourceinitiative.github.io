APM_CAPABILITIES

The APM_CAPABILITIES contains flags and information to set the APM capabilities of the device.

typedef struct APM_CAPABILITIES_S {
   WORD   Capabilities;
   BYTE   BatteryCount;
   BYTE   Reserved;
} APM_CAPABILITIES, *PAPM_CAPABILITIES;

Members

Capabilities
Sets the capabilities of the device. The field can be set to the capabilities listed in the following table. Unused capabilities must be set to zero.
Flag Name Description
GLOBAL_STANDBY_
SUPPORTED_BIT		 System can enter global standby state. Indicates BIOS will post standby and standby-resume events. If used, it must be set to GLOBAL_STANDBY_SUPPORTED.
GLOBAL_SUSPEND_
SUPPORTED_BIT		 System can enter global suspend state. Indicates BIOS will post suspend and suspend-resume events. If used, it must be set to GLOBAL_SUSPEND_SUPPORTED.
WAKE_ON_TIMER_
STANDBY_BIT		 Resume timer will wake up from standby. If used, it must be set to WAKE_ON_TIMER_STANDBY.
WAKE_ON_TIMER_
SUSPEND_BIT		 Resume timer will wake up from suspend. If used, it must be set to WAKE_ON_TIMER_SUSPEND.
RING_RESUME_
INTERNAL_STANDBY_
BIT		 Resume on ring indicator (internal COM or modem) will wake up from standby. If used, it must be set to RING_RESUME_INTERNAL_STANDBY.
RING_RESUME_
INTERNAL_SUSPEND_
BIT		 Resume on ring indicator (internal COM or modem) will wake up from suspend. If used, it must be set to RING_RESUME_INTERNAL_SUSPEND.
RING_RESUME_
PCMCIA_STANDBY_BIT		 PCMCIA ring indicator will wake up from standby. If used, it must be set to RING_RESUME_PCMCIA_STANDBY.
RING_RESUME_
PCMCIA_SUSPEND_BIT		 PCMCIA ring indicator will wake up from suspend. If used, it must be set to RING_RESUME_PCMCIA_SUSPEND.

BatteryCount
Number of batteries present.
Reserved
Reserved. Do not use.
