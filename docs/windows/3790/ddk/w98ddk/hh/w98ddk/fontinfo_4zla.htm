Disabling Double-Underline Simulation

The FS_OCD_DOUBLEUNDERLINE_OFF box contains the command which disables double-underline simulations for a printer.

Diconix 150 Plus

The Diconix 150 Plus printer does not support double-underlining; therefore, the FS_OCD_DOUBLEUNDERLINE_OFF box is empty.

HP LaserJet IIP

The HP LaserJet IIP printer does not support double-underlining; therefore, the FS_OCD_DOUBLEUNDERLINE_OFF box is empty.
