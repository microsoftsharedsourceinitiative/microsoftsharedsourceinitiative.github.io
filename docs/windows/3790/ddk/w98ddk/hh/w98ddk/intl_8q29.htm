GAIJISYSTEMGETFONT

Gets the bitmap data of the EUDC character from the system default WIFE font.

Returns zero for failure. Windows 95 always returns zero.

lpInData
The low-order word is the handle of the bitmap data.
lpOutData
A pointer to the memory block containing the EUDC character code.
