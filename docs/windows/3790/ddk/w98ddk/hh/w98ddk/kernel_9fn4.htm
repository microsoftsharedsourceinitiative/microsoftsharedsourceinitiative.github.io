PageFile_Set_Async_Manager

mov     eax, FSD_Entry             ; entry in FSD
VxDcall PageFile_Set_Async_Manager
jc      error_handler

An FSD offering asynchronous page-outs stores an internal entry that a VxD can call to register a callback function that the system calls when the FSD is idle. This service can only be called during initialization. Uses Flags.

The first caller of this service registers the callback function, pointing to the Async Manager for Pagefile. Subsequent calls to this routine will fail, returning with carry flag set.

VFAT typically calls this service, registering VFAT_Register_MMCallBack. Each subsequent call to the PageFile_Call_Async_Manager service is actually a call to the FSD_Entry callback function indicated in the sample code above.

See also the following KB article: Q223294 - INFO: The Windows 98 PageFile_Call_Async_Manager Service.

  • Clears the carry flag is successful, and sets it otherwise.
