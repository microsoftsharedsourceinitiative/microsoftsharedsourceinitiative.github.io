Printer Technology

The fTechnology button displays the fTechnology dialog box. This dialog box contains controls that identify the device technology used by the printer. Currently there is only one control in this dialog box: GPC_TECH_PCL4. This control should only be selected if a minidriver uses the HP PCL Level 4 page-description language.

Diconix 150 Plus

The GPC_TECH_PCL4 box is not checked.

HP LaserJet IIP

The GPC_TECH_PCL4 box is checked.
