Predefined Paper Source

The Source Name box specifies the names of 12 possible paper sources from which one can be selected. The following contains the names and a brief description of each.

Paper Source Description
DMBIN_UPPER/ONLY Upper bin
DMBIN_LOWER Lower bin
DMBIN_MIDDLE Middle bin
DMBIN_MANUAL Manual feed
DMBIN_ENVELOPE Envelope feed
DMBIN_ENVMANUAL Manual-envelope feed
DMBIN AUTO Automatic-sheet feeder
DMBIN_TRACTOR Tractor feed
DMBIN_SMALLFMT Small-format paper
DMBIN_LARGEFMT Large-format paper
DMBIN_LARGECAPACITY Large-capacity bin
DMBIN_CASSETTE Paper cassette

Diconix 150 Plus

For the Diconix 150 Plus, the primary paper source is the tractor-feed mechanism; therefore, DMBIN_TRACTOR is selected.

HP LaserJet IIP

For the HP LaserJet IIP, the primary paper source is the upper-paper tray; therefore, DMBIN_UPPER/ONLY is selected.
