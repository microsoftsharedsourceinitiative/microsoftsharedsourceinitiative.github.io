Specifying General Data

The fGeneral button displays the fGeneral dialog box which contains four checkboxes.

The following list describes the four checkboxes in this dialog box:

Control Description
PS_CENTER Specifies that the Universal Printer Driver should center the printable region on the page.
PS_ROTATE Specifies that the Universal Printer Driver should swap the x- and y- paper-size dimensions (if this control is set, the Universal Printer Driver will not swap the margin widths).
PS_SUGGEST_LNDSCP Specifies that landscape mode should be the suggested orientation for printing when this paper size is selected.
PS_EJECTFF Specifies that the Universal Printer Driver can eject a page of this paper size using the form-feed command instead of y-movement commands.

Diconix 150 Plus

For the Diconix 150 Plus printer, none of the boxes are checked.

HP LaserJet IIP

For the HP LaserJet IIP printer, none of the boxes are checked.
