KSPROPERTY_TUNER_MODES

Drivers can implement several different modes for tuning using the KSPROPERTY_TUNER_MODES types.

typedef enum {
    KSPROPERTY_TUNER_MODE_TV            = 0X0001,
    KSPROPERTY_TUNER_MODE_FM_RADIO      = 0X0002,
    KSPROPERTY_TUNER_MODE_AM_RADIO      = 0X0004,
    KSPROPERTY_TUNER_MODE_DSS           = 0X0008,
} KSPROPERTY_TUNER_MODES;
KSPROPERTY_TUNER_MODES Types Description
KSPROPERTY_TUNER_
MODE_TV		 Specifies the tuner is actively tuning broadcast or cable television channels.
KSPROPERTY_TUNER_
MODE_FM_RADIO		 Specifies the tuner is actively tuning FM radio channels.
KSPROPERTY_TUNER_
MODE_AM_RADIO		 Specifies the tuner is actively tuning AM radio channels.
KSPROPERTY_TUNER_
MODE_DSS		 Specifies the tuner is actively tuning DSS channels.
