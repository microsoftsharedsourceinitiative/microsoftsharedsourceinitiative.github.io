Default Character-Translation Table

The Default CTT ID box contains a value which identifies the default character-translation table (CTT). If a single character-translation table is to be used for all device fonts, the ID value for that CTT should be entered here. If the Universal Printer Driver should not use a CTT, this value should be 0; if the Universal Printer Driver should use its own CTT, this value should be that CTT's ID multiplied by -1; if the Universal Printer Driver should use a driver-supplied CTT, this value should be that CTT's ID.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Default CTT ID box contains the value 1.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Default CTT ID box contains the value 0.
