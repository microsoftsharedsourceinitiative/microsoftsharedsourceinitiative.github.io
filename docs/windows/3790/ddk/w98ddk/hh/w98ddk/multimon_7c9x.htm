VDD_ENABLE

VDD_ENABLE enables access to this device.

Note  Only one device at a time can have VGA memory, I/O, or ROM access. All other devices are  disabled if a secondary device has VGA access.

Entry Meaning
Client_EAX VDD_ENABLE equate from Minivdd.inc or Minivdd.h.
Client_EBX Device handle.
Client_EDX Enable flags:

ENABLE_ALL - enable all access to this device
ENABLE_IO - enable IO
ENABLE_MEM - enable memory
ENABLE_NONE - disable device
ENABLE_ROM - enable ROM at C000
ENABLE_VGA - enable VGA

Exit Meaning
Client_EAX Previous enable state or ENABLE_ERROR.

ENABLE_ERROR - enable fail code
