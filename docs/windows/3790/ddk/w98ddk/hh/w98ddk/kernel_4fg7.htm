VTDAPI_Get_Sys_Time

; Obtain the entry point address in VTDAPI_Addr
mov ax, VTDAPI_Get_Sys_Time
call VTDAPI_Addr

Retrieves the current system time.

  • Returns the current system time in milliseconds as a double-word in DX:AX.
