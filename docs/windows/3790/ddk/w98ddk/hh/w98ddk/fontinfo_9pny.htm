Font Format Information

The fFormat button displays the fFormat dialog box. This dialog box contains controls that the Universal Printer Driver uses for TrueType support on Hewlett-Packard printers that use PCL.

The following table provides a brief description of each control.

Control Description
DLI_FMT_PCL Specifies that the printer uses the Hewlett-Packard Page Control Language (PCL)

HP LaserJet IIP

For the HP LaserJet IIP printer, the DLI_FMT_PCL box is checked because the printer supports Hewlett-Packard's PCL.
