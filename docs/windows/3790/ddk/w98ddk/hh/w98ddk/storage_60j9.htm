IFSMgr_Win32_Get_Ring0_Handle

This service is provided only for the use of the Win32 subsystem and should not be used by anyone else.

[EBX]
File handle to be converted.
  • Carry flag clear, handle successfully mapped.
[EBX] Special Ring 0 handle corresponding to handle passed in.
[EDX] Current file position for the file handle passed in.

Carry flag set, handle was not mapped. EBX is not changed on failure.

Registers Used

EAX, ECX, Flags.
