PCIMP_GARTUnMap

The PCIMP_GARTUnMap service unmaps the specified GART entries.

VOID
  PCIMP_GARTUnMap(
    PVOID  pHandle, 
    PVOID  pRefData, 
    ULONG  ulNumPages, 
    PVOID  pMapHandle, 
    ULONG  ulByteOffset
    );

Parameters

pHandle
PCI device handle. This should be the same value used in the preceding PCIMP_GARTInit call.
pRefData
ULONG that was written by the PCIMP_GARTInit service.
ulNumPages
Specifies the number of pages to be unmapped.
pMapHandle
Map handle obtained using the PCIMP_GARTReserve service.
ulByteOffset
Specifies the offset in bytes from the beginning of the memory block referred to by pMapHandle to the start of a page.

Return Values

This service does not have a return value.

Comments

This function undoes the action of PCIMP_GARTMap. Parameters are the same as for PCIMP_GARTMap except PCIMP_GARTUnMap requires no pSystemLin parameter.

Unmapping is typically accomplished by filling the specified GART entries with the value passed in the pReservedValue parameter of PCIMP_GARTReset. After this function completes, the entries will still be reserved and the associated map handle will still be valid.
