LookAhead Region

The LookAhead Region box contains a value which specifies the length of the Lookahead Region in master units. This value should be 0 for most printer models; however, a nonzero value is required for HP DeskJet printers.

Diconix 150 Plus

For the Diconix 150 Plus printer, the LookAhead Region box contains the value 0.

HP LaserJet IIP

For the HP LaserJet IIP printer, the LookAhead Region box contains the value 0.
