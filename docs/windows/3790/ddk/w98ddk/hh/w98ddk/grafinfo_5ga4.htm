Minimum Gray Level

The wMinGray box contains the minimum percentage value that the Universal Printer Driver can pass as an argument in the printer's gray-fill command.

Diconix 150 Plus

The wMinGray box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the minimum percentage of black pixels is 1. (This means that there is 1 black pixel for every 99 white pixels in the minimum shade or fill pattern.)
