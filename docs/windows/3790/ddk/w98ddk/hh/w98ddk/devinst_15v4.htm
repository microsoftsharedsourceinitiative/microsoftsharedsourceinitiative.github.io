DiDestroyDeviceInfoList

Destroys a list of DEVICE_INFO structures. 

RETERR
  DiDestroyDeviceInfoList(
    LPDEVICE_INFO  lpdi
    );

Return Value

OK

Parameters

lpdi
Pointer to a list of DEVICE_INFO structures.

Comments

The list can contain only 1 structure. This function will free any compatible and/or class driver lists created using the lpdi(s) as well. Any class installer DLL's or property provider DLL's associated with the lpdi(s) list be freed also.

See Also
