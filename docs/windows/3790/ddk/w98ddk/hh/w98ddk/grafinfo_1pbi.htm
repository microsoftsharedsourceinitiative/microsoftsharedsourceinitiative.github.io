Specifying the Hatch Pattern

The Hatch Fill box contains the printer command or commands which fill a rectangle with a hatched pattern.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Hatch Fill box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Area Fill ID and the Fill Rectangular Area are combined into a single command. The Area Fill ID command (ESC*cnG) specifies the hatch pattern and the Fill Rectangular Area command (ESC*cnP) fills a rectangular area. The argument n specifies the hatch pattern is the Area Fill ID command; n specifies the type of fill in the Fill Rectangular Area command. The following table contains the command and its respective values and patterns.

Command Value Hatch or Fill
Area Fill ID 1 Horizontal hatch pattern
  2 Vertical hatch pattern
  3 Right-diagonal hatch pattern
  4 Left-diagonal hatch pattern
  5 Horizontal and vertical cross-hatch pattern
Fill Rectangular Area 0 Black
  1 White
  2 Shading
  3 Cross hatch
  5 Current pattern

To specify the combination, type the following in the Hatch Fill box:

\x1B*c%dG3P

Once you've entered the appropriate values in the rectangle-fill dialog boxes, you should begin entering font information for your printer.
