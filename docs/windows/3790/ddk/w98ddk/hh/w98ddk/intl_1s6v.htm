IMC_GETSTATUSWINDOWPOS

An application sends the IMC_GETSTATUSWINDOWPOS message to the IME window to get the position of the status window. The UI window receives the message.

The return value is a POINTS structure that contains the x and y coordinates of the position of the soft keyboard window. It is in screen coordinates.
