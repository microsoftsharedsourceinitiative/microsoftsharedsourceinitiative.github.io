GAIJIPHYSICALFONTSIZE

Gets the size of bitmap specified by the index.

If successful, the return value is 1. Otherwise, it is zero for failure. Windows 95 always returns zero.

lpInData
The low-order word is the index to the system default WIFE EUDC font.

The high-order word always passes zero.

lpOutData
A pointer to the POINT structure, which will contain the size of the bitmap.
