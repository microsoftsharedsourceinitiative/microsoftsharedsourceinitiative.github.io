Default and Supported Paper Qualities

The Paper Qualities dialog box identifies the default and supported paper qualities. The following list describes each of the controls in this dialog box.

Control Description
Default Specifies the default paper quality.
List box Lists the available paper qualities for this printer.
Enable All Enables all paper qualities which appear in the list box.
Inhibit All Inhibits all paper qualities which appear in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus does not support paper-quality commands.

HP LaserJet IIP

The HP LaserJet IIP does not support paper-quality commands.
