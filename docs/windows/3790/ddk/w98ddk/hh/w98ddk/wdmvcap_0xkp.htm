Timecode Category

Timecode Category output pins provide a stream of timecode.

When creating the category PinName, PINNAME_VIDEO_TIMECODE, the following parameters must be filled out.

Parameter Value
KS Datarange KS_DATARANGE_VIDEO_VBI
KS Dataformat KS_DATAFORMAT_VIDEO_VBI
KS MajorFormat To be determined
KS SubFormat (typical) To be determined
KS Specifier KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
DS majortype None
DS formattype None
