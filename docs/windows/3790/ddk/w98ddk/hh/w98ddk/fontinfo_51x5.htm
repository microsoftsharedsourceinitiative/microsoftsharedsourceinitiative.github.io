Specifying Text Quality

The Select String box contains the printer command which selects the text quality identified in the Text Quality Name box.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Text Quality command (ESCxn) selects a text quality. The argument n identifies the requested quality; if n is 0, draft quality is selected; if n is 1, near-letter quality is selected; if n is 3, letter-quality is selected.

To specify the Text Quality command, type the following in the Select String box:

\x1Bx\x03

HP LaserJet IIP

The Select String box is empty.

Once you have entered the text-quality commands and data for your printer, you should begin entering the font-cartridge commands and data.
