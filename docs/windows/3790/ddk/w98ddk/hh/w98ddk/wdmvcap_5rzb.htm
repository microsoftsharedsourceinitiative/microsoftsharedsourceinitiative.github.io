KSPROPERTY_VIDEODECODER_VCR_TIMING

The KSPROPERTY_VIDEODECODER_VCR_TIMING read/write property is used to control whether the VCR is expecting video from a tape source or a broadcast source. The timing accuracy of synchronization pulses on tape sources is typically poorer from tape sources than broadcast sources. This property uses the KSPROPERTY_VIDEODECODER_STANDARD_S structure to set or retrieve information. This property is optional.

typedef struct {
  KSPROPERTY  Property;
  ULONG  Value;
} KSPROPERTY_VIDEODECODER_S, *PKSPROPERTY_VIDEODECODER_S;

Members

Property
Specifies the property used.
Value
Specifies the value to be set or retrieved.

The KSPROPERTY_VIDEODECODER_S.Value member specifies whether to use VCR timing. Zero indicates a source other than tape, and 1 indicates a tape source.
