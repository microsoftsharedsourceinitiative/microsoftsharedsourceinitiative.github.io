ScsiPortWriteRegisterUchar

VOID ScsiPortWriteRegisterUchar(Register, Value)
IN PCHAR  Register;
IN UCHAR  Value;

Transfers an unsigned byte value to a memory mapped register.

  • No return value.
Register
Points to the register.
Value
Specifies the value to be written to the register.
