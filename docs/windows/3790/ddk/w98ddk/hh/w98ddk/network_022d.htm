Other Functions

Windows 95 supports all the miscellaneous network functions supported by Windows NT 3.51. Windows 95 also offers the WNetConnectionDialog1, WNetDisconnectDialog1, WNetGetNetworkInformation, and WNetGetProviderName functions, which are documented in Multiple Provider Router.
