IOS_BD_Register_Device

mov edi, BDD  ; BlockDev Device Descriptor
VxDCall IOS_BD_Register_Device
jc error

Registers a BlockDev device.

  • Carry flag is set on error.
BDD
Address of a BlockDev_Device_Descriptor structure.
