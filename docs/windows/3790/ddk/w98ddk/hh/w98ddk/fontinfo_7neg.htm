Permanent Font Box

The MAKE_PERM box contains the command which makes a downloadable font permanent.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Font Control command (ESC*cnF) deletes downloaded fonts, makes downloaded fonts temporary, makes downloaded fonts permanent, and so on. If n is 5, the font is made permanent.

To specify the Font Control command that makes a font permanent, type the following in the MAKE_PERM box:

\x1B*c5F
