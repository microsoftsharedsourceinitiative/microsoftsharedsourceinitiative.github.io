The Character Shift Amount

The Shift control contains a value that specifies a horizontal distance in horizontal-resolution units. This distance is an offset that the Universal Printer Driver uses to shift individual characters from a particular font. It is required for fonts with characters that are not centered in the bounding box. The Universal Printer Driver uses a negative shift value to shift characters to the right, and a positive shift value to shift characters to the left. Shift amounts are intended to be used when characters do not fall in the center of the bounding box defined by font metrics.

Diconix 150 Plus

For the Diconix 150 Plus printer, this control contains the value 0.
