Identifying the Break Character

The Break Char control contains a value that identifies the character used to define word breaks. This value is a zero-based offset from the first character identified in the First Char control.

Diconix 150 Plus

For the Diconix 150 Plus printer this value is 0, which corresponds to the space character.
