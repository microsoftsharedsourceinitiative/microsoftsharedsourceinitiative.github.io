IFSMgr_AssertFailed

IFSMgr_AssertFailed(
 char * pmsg,
 char * pfname,
 unsigned long line
 )

This service causes an assertion failure message to be printed and breaks into the debugger. This service is available only on debug versions of the IFS manager. On the nondebug version, this service returns without doing anything. This service is not available until the IFS manager has finished its Device_Init during device initialization.

pmsg
Supplies pointer to the ASCIIZ message string of the failure message.
pfname
Supplies pointer to the ASCIIZ file name string of the source file the failure occurred in.
line
Supplies line number in the source file the failure occurred at.
  • This service has no return value.
