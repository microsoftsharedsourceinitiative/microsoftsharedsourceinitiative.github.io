PCIFreePages

The PCIFreePages function frees the pages allocates with PCIAllocatePages.

VOID
  PCIFreePages(
    PVOID  pLin
    );

Parameters

pLin
Linear address previously returned by PCIAllocatePages.

Return Values

This function has no return value.
