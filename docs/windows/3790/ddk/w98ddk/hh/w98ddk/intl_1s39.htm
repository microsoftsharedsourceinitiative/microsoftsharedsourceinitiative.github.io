ImmDestroyIMCC

HIMCC WINAPI ImmDestroyIMCC(HIMCC hIMCC)

Destroys an IMC component that can be the member of IMC.

If successful, the return value is NULL. Otherwise, it is equal to HIMCC.

hIMCC
Handle of IMC component.
