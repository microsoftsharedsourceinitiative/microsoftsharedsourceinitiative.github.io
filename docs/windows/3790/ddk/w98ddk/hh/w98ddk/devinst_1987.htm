NdiIsNdi

RETERR NdiIsNdi(hndi)
HNDI hndi;    // NDI handle

The NdiIsNDI procedure determines whether the given NDI handle is valid.

hndi
Identifies an NDI object.
  • The return value is zero (OK) if the NDI handle is valid. Otherwise it is ERR_NDI_INVALID_HNDI.
