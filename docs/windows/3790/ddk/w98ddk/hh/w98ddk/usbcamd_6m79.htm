CamInitialize

CamInitialize is invoked upon using the peripheral for the first time in a session. This initializes the device and allows additional vendor commands to be provided. The device context needs to be previously configured and the initial alternate interface selected.

NTSTATUS
  CamInitialize(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext
    );

Parameters

BusDeviceObject
Device object associated with the peripheral.
DeviceContext
User-supplied value or structure relevant to the stream.

Return Values

The NTSTATUS code indicating the success or failure of the operation. Since the implementation is vendor specific, the errors messages may vary between manufacturers.
