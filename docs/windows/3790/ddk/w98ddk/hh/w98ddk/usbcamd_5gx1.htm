CamStopCapture

CamStopCapture is called after the ISO video stream is stopped. It allows alternate interface selection that uses no bandwidth.

NTSTATUS
  StopCapture(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext
    );

Parameters

BusDeviceObject
Device object created by the hub. URBs can be submitted to the device through this device object.
DeviceContext
Minidriver device context.

Return Values

The return value is the NTSTATUS completion code for the read IRP.
