VCache_RelinquishPage

Returns a page to the memory manager. Uses flags.

  • Returns the result in EAX. The value is the linear address of the located page or zero.

    This service tries to find a page it can return to the memory manager. In general it will only fail if all pages are currently reserved, held, or dirty.
