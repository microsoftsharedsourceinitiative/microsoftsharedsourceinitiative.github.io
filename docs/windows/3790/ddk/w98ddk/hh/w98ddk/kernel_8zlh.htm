Test_Debug_Installed

include vmm.inc

VMMCall Test_Debug_Installed

je      not_installed       ; zero flag set if not installed

Determines whether a debugger is currently connected. is running. This is not the same as checking whether the debugging version of the Windows VMM is running. Uses Flags.

  • Returns with the zero flag set if a debugger is not installed, clear otherwise.
