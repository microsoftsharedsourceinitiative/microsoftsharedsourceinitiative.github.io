BootConfig

BootConfig=binary-data

Identifies the boot configuration for a device. The binary-data is configuration data, in binary format, that specifies the IRQ, I/O ports, and other resources used by this device.

This value is used in HLM\ENUM\ROOT\device-id\instance.
