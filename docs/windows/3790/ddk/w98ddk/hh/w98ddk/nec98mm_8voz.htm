GET_NUM_UNITS

The GET_NUM_UNITS function obtains the number of monitors that can be connected to the display adapter. 

int get_num_units(
  DEVNODE dn
  );

Parameters

dn
Device node of the display adapter.

Return Value

Returns the number of connectable monitors.
