Wave Input

DirectSound does not currently define wave capture driver interfaces other than the Mmsystem.dll ring 3 widMessage wave driver interface that exists in the original release of Windows 95. The DirectSound API supports full-duplex (that is, audio can be played while it is being recorded) on drivers and cards that are capable of this feature. Full-duplex can add significantly to the interactivity of audio, especially with multiplayer games.
