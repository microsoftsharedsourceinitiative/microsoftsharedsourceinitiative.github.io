IMC_GETSOFTKBDSUBTYPE

An IME sends the IMC_GETSOFTKBDSUBTYPE message to the soft keyboard window to get the subtype of the soft keyboard window set by IMC_SETSOFTKBDSUBTYPE.

The return value is the subtype of the soft keyboard set by IMC_SETSOFTKBDSUBTYPE if successful. Otherwise, it is -1.
