Contention Management Functions

This section describes the _VCOMM_Get_Contention_Handler function, the VCOMM_Map_Name_To_Resource function, and the functions used to call a contention handler. The _VCOMM_Get_Contention_Handler service retrieves a pointer to one of the functions documented in this section.
