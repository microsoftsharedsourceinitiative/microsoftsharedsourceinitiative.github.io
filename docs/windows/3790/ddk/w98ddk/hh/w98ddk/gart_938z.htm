PCI Class Driver Services

The PCI class driver exports the following services that can be called from a GART miniport driver. They are described further in PCI Class Driver Service Reference.

Name Purpose
PCIGetVersion Gets the operating system version number.
PCIGetHandleInfo Retrieves the bus and device-function numbers.
PCIReadConfig Reads a configuration value.
PCIWriteConfig Writes a configuration value.
PCIAllocatePages Allocates memory.
PCIFreePages Frees memory.
PCILinToDev Converts linear address to device address.
