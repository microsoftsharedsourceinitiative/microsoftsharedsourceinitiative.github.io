General Paper Source Information

The fGeneral button displays the fGeneral dialog box. The controls in this dialog box allow you to specify general information relating to the paper-source selection.

The PSRC_EJECTFF box should be checked if the printer ejects a page of paper by sending a form-feed command; the checkbox should not be selected if the printer ejects the page by sending y-movement commands.

The PRSC_MAN_PROMPT box should be checked if the user should be prompted to insert a new piece of paper prior to printing each page.

Diconix 150 Plus

For the Diconix 150 Plus printer, the printer ejects a page by sending a form-feed command; therefore, the PSRC_EJECTFF box is checked.

HP LaserJet IIP

For the HP LaserJet IIP printer, the printer ejects a page by sending a form-feed command; therefore, the PSRC_EJECTFF box is checked.
