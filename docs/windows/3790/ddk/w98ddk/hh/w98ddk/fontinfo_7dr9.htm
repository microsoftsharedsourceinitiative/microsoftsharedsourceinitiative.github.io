Identifying a Character-Translation Table

The Trans Tab control identifies the character-translation table associated with a font. A character-translation table provides a method of remapping the extended characters used for device fonts to the Windows ANSI set. In the Trans Tab control, negative values identify predefined translation-tables, positive values identify minidriver-defined translation tables, and a value of zero indicates that the CTT is identified in the minidriver's model data. For more information about character-translation tables (CTT), see Creating a Character-Translation Table.

Diconix 150 Plus

For the Diconix 150 Plus printer, this control contains the value 0 since the font uses the CTT specified in the ModelData dialog box.
