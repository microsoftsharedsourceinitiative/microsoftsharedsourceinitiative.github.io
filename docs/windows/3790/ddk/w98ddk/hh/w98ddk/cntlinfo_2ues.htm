Ending the Document

The PC_OCD_ENDDOC box contains the printer commands which the Universal Printer Driver issues to end a document. The carriage return is a common end-document command--it's used to position the cursor in the left margin. Other end-document commands vary from printer to printer; however, it is typical to return a printer to its original (or default) state.

Diconix 150 Plus

For the Diconix 150 Plus, the Carriage Return (0x0d) command is required to return the cursor to the left edge of the page after printing a document.

To specify the Carriage Return command, type the following in the PC_OCD_ENDDOC box:

\x0D

HP LaserJet IIP

For the HP LaserJet IIP, the Reset command (\x1BE) is required to reset the printer to its default settings.

To specify the Reset command, type the following in the PC_OCD_ENDDOC box:

\x1BE
