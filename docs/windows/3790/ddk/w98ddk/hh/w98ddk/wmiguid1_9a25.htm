IRP_MN_CHANGE_SINGLE_ITEM

The driver receives an IRP_MN_CHANGE_SINGLE_ITEM to change a single data item in a single instance of a data block. The driver should use new values passed in the WNODE_SINGLE_ITEM to change the value of the data item and complete the IRP. The driver should ensure that any attempt to modify data items that are read-only are ignored.

Members

ProviderId
Contains the WMI provider ID of the device that should respond to this request.
DataPath
Points to a GUID that represents the data to be returned.
BufferSize
Specifies the size of the buffer that contains the new value for the data item in the data block.
Buffer
Points to a WNODE_SINGLE_ITEM that contains the new value for the data item in the data block.

Return Value

Returns STATUS_SUCCESS, STATUS_WMI_INSTANCE_NOT_FOUND, STATUS_WMI_INSTANCE_ID_NOT_FOUND, STATUS_WMI_GUID_NOT_FOUND, STATUS_WMI_READ_ONLY, or STATUS_WMI_SET_FAILURE.
