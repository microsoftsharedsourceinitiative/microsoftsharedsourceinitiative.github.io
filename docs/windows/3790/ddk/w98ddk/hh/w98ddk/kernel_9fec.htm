VDMAD_Set_Region_Info

include vdmad.inc

mov eax, DMA_Handle
mov bl, Buffer
mov bh, LockStatus
mov esi, OFFSET32 Region
mov ecx, RegionSize
mov edx, OFFSET32 PhysAddress
VxDcall VDMAD_Set_Region_Info

Sets information about the current region assigned to a DMA handle. Uses flags

  • No return value.
DMA_Handle
Handle of the DMA.
Buffer
Identifier for buffer.
LockStatus
Status of pages (zero if not locked, nonzero if locked).
Region
Address of region.
RegionSize
Size of region in bytes.
PhysAddress
Physical address for the transfer.

This service must be called before calling the VDMAD_Set_Phys_State service.
