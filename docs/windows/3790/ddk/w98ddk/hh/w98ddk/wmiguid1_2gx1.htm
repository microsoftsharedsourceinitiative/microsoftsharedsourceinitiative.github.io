IRP_MN_QUERY_SINGLE_INSTANCE

The driver receives an IRP_MN_QUERY_SINGLE_INSTANCE to query for a single instance of a data block.

Members

ProviderId
Contains the WMI provider ID of the device that should respond to this request.
DataPath
Points to a GUID that represents the data to be returned.
BufferSize
Specifies the maximum size available to write the WNODE_SINGLE_INSTANCE into the buffer.
Buffer
Points to a WNODE_SINGLE_INSTANCE that the driver should fill in with the result of the query.

Return Value

Returns STATUS_SUCCESS, STATUS_WMI_INSTANCE_NOT_FOUND, or STATUS_WMI_GUID_NOT_FOUND.
