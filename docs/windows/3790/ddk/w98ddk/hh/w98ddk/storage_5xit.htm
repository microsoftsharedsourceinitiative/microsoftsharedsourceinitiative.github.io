IFSMgr_Win32ToNetTime

IFSMgr_Win32ToNetTime(
 _FILETIME Win32Time
 )

This service converts the time in Win32 time format to the net time format. Both these formats use UTC time.

Win32Time
Time in Win32 time format in the _FILETIME structure.
  • Returns the time in net time format.
