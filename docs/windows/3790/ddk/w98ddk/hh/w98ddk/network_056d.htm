Enumeration

WNNC_ENUMERATION

Returns a mask of:

WNNC_ENUM_GLOBAL  
WNNC_ENUM_LOCAL  
WNNC_ENUM_CONTEXT  

If one or more of these values is set, then the NP must export and support NPOpenEnum, NPEnumResource, and NPCloseEnum.
