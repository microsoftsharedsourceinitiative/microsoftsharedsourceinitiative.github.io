PC 97 Requirements

The PC 97 Hardware Design Guide (Microsoft Press, 1996; ISBN 1-572321-381-1) lists several requirements for graphics adapters. These requirements must be implemented by January 1, 1998 for any adapter or PC system that receives the Designed for Windows NT/Windows 2000 and Windows 95 logo. The following list summarizes these requirements as defined in Chapter 15, Graphics Adapters, of the PC 97 Hardware Design Guide:

  • Item #7: Graphics operations use relocatable registers only
  • Item #11: Graphics adapters support DDC 2.0 Level 2 monitor detection
  • Item #14: Plug and Play conflict resolution for multiple monitors/multiple adapters
  • Item #15: VGA resources can be disabled by software
  • Item #16: Alternative mechanism to Int 10 for initialization provided
  • Item #43: Plug and Play device identifier (including Subsystem ID and Subsystem Vendor ID for PCI devices)
  • Item #44: Conflict resolution, VGA compatibility, and extended register
  • Item #45: PCI base address registers populated correctly for PCI graphics devices
