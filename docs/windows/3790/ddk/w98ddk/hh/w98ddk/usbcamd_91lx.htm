CamStartCapture

CamStartCapture, called just before the ISO video capture stream is started, selects the appropriate alternate interface, and sets up the device to stream. Called in connection with the stream class RUN command.

NTSTATUS
  CamStartCapture(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext
    );

Parameters

BusDeviceObject
Device object created by the hub. URBs can be submitted to the device through this device object.
DeviceContext
Mindriver context.

Return Values

The return value is the NTSTATUS completion code for the read IRP.
