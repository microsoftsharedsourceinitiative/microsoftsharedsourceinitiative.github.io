KSPROPERTY_CAMERACONTROL_TILT

The KSPROPERTY_CAMERACONTROL_TILT read/write property is used to set and retrieve the camera tilt setting.

The KSPROPERTY_CAMERACONTROL_S.Value member specifies the tilt setting in degrees. Values range from –180 to +180, and the default is zero. Positive values point the imaging plane up, and negative values point the imaging plane down. Note that a particular device may only implement a subset of this range.
