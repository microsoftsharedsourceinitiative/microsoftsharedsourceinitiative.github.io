HeapReAllocateEx

HeapReAllocateEx reallocates a block of memory. This function always allocates a new block even if the new block is smaller than the old block and is on the same heap.

PVOID HeapReAllocateEx(
  PVOID   pmem,
  ULONG   cBytes,
  PVOID   Reserved1,
  PVOID   Reserved2,
  ULONG   Flags
 );

Parameters

pmem
Pointer to memory previously returned by HeapAllocateEx.
cBytes
Size of the resized block.
Reserved1
Reserved for future use.
Reserved2
Reserved for future use.
Flags
One or more of HEAPxxx as defined in VMM.H.

Return Values

No value is returned.
