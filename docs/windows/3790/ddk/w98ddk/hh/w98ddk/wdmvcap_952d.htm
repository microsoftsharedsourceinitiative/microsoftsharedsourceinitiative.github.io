KSPROPERTY_TVAUDIO_MODE

The KSPROPERTY_TVAUDIO_MODE read/write property is used to set and retrieve the current audio mode. 

typedef struct {
  KSPROPERTY  roperty;
  ULONG  Mode;
} KSPROPERTY_TVAUDIO_MODE_S, *PKSPROPERTY_TVAUDIO_MODE_S;

Mode
Specifies the mode of the TV audio device, using values from the following.

KS_TVAUDIO_MODE_STEREO Specifies the audio is in stereo.
KS_TVAUDIO_MODE_LANG_A Specifies the audio is an additional language.
