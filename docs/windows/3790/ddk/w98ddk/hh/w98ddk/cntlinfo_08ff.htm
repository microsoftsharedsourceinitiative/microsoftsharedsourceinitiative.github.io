Popping the Stack

The CM_OCD_POP_POS box contains the printer command or commands which pops the current position off of the printer's internal stack.

Diconix 150 Plus

The Diconix 150 Plus printer does not provide an internal stack for printer commands; therefore, the CM_OCD_POP_POS box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Pop Cursor Position command (ESC&f1S) stores the current cursor position on an internal stack.

To specify the Pop Cursor Position command, type the following in the CM_OCD_POP_POS box:

\x1B&f1S
