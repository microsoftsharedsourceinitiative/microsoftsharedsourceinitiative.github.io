Multimedia Plug and Play

Instead of using a configuration dialog box to ask the user to supply I/O port addresses, IRQ levels, and DMA channels, Plug and Play devices obtain hardware configuration information from the Configuration Manager system component. Most of the requirements for implementing Plug and Play in a multimedia device driver are the same as for any other device driver; this section only discusses the procedures that are unique to multimedia devices. For more information about Plug and Play, see "Plug and Play Introduction," elsewhere in the Microsoft Windows 95 Device Driver Programmer's Reference.
