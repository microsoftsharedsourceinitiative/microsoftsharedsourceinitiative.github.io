VBI Category

VBI Category pins provide a stream of vertical blanking interval (VBI) waveform samples. This stream is typically passed to codecs that extract closed captioning, NABTS, WST, timecode, and other digital data streams.

When creating the category PinName, PINNAME_VIDEO_VBI, the following parameters must be filled out.

Parameter Value
KS Datarange KS_DATARANGE_VIDEO_VBI
KS Dataformat KS_DATAFORMAT_VIDEO_VBI
KS MajorFormat KS_DATAFORMAT_TYPE_VBI
KS SubFormat (typical) KSDATAFORMAT_SUBTYPE_RAW8
KS Specifier KSDATAFORMAT_SPECIFIER_VBI
Extended Header Size KS_VBI_FRAME_INFO
DS majortype MEDIATYPE_VBI
DS formattype FORMAT_VBI
