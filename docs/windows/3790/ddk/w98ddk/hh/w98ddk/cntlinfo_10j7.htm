Carriage Returns

The CM_OCD_CR box contains the printer command or commands which cause a carriage return.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Carriage Return command (0x0D) causes a carriage return.

To specify the Carriage Return command, type the following in the CM_OCD_CR box:

\x0D

HP LaserJet IIP

For the HP LaserJet IIP printer, the Carriage Return command (0x0D) causes a carriage return.

To specify the Carriage Return command, type the following in the CM_OCD_CR box:

\x0D
