_AtEventTime

include vmm.inc

VMMCall  AtEventTime
mov      fEvent, eax

Tests whether the current thread is processing an event. Uses EAX, ECX, EDX, and Flags

Returns non-zero in EAX if the thread is processing an event, zero otherwise.
