Upper-edge Functions

From the Windows NT/Windows 2000 network documentation: The NDIS library calls NIC driver upper-edge functions, which prepare the packet for transmission on the network and queue it to be transmitted by an underlying NIC. NIC driver function names are prefaced with "Miniport" or "Mac" (depending on the type of NIC driver).
