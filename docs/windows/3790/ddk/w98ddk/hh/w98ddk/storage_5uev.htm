ScsiPortReadRegisterUshort

USHORT ScsiPortReadRegisterUshort(Register)
IN PUSHORT  Register;

Reads a value from the specified memory mapped register address.

  • Returns a USHORT value from the HBA's register.
Register
Points to the register.

ScsiPortReadRegisterUshort ensures that the data is transferred correctly.

This function was not correctly exported in SCSIPORT.PDR. See KB Article Q160667, Miniport Driver Fails Using Certain ScsiPort APIs, for more information.
