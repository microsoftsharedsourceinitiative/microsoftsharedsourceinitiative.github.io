GAIJIAREASIZE

Returns the number of EUDC characters that can be used.

The return value is the total number of EUDC characters that can be used. Windows 95 always returns zero.

lpInData
Not used.
lpOutData
Not used.

This value must be determined from the EUDC character code range specified by WIFEMAN.INI.
