General Information

The fGeneral button displays the fGeneral dialog box. This dialog box contains controls that specify general information about a minidriver. Currently there is only one control in this dialog box: GPC_PRIVATE_HELP. This control should only be selected if a minidriver developer supplies a private help file that should replace the standard help file (UNIDRV.HLP).

Diconix 150 Plus

The GPC_PRIVATE_HELP box is not checked.

HP LaserJet IIP

The GPC_PRIVATE_HELP box is not checked.

Once you have initialized the controls in the MiniDriver Data dialog box, from the Printer Data menu, choose the Control Info command and begin typing information that describes your printer's page-control and cursor-movement commands and capabilities.
