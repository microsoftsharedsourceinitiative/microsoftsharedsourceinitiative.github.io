IFSMgr_GetRing0FileHandle

IFSMgr_GetRing0FileHandle gets the IFS ring 0 handle from an IOREQ. This function is called to obtain the IFSMgr's ring 0 file handle associated with an IFSMgr IOREQ. This function does not support IOREQs associated with path-based or find-handle-based APIs.

pfhandle IFSMgr_GetRing0FileHandle (
  pifsreq   pir
 );

Parameters

pir
Pointer to IFS IOREQ.

Return Values

The return value is a pointer to the IFS ring 0 file handle associated with this IOREQ. If NULL is returned, this indicates that either the IOREQ or the handle associated with the IOREQ is invalid.
