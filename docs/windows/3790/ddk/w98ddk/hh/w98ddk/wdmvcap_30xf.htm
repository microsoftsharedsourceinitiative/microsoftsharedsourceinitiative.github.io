Medium GUIDs

Breaking apart the Tuner, Crossbar, Capture, and TvAudio components into separate filters results in the need to provide a means to correctly describe the topological connectivity between components. Medium GUIDs are used to describe a hardware connection, and this value is used by graph builders to construct graphs.

Mediums are exposed by the KSPIN_MEDIUM structure, which consists of a GUID (Set) followed by two ULONGs (Id and Flags). The KSPIN_MEDIUM.Set should be assigned a unique GUID representing a hardware connection. The KSPIN_MEDIUM.Id should be set to an instance value, and KSPIN_MEDIUM.Flags should be zero. This allows installation of multiple instances of the same hardware.
