Specifying Printer Capabilities

The Capability Bitfields list box lists the names of eight selections. Of these eight, only three contain information used by the Microsoft® Windows® Universal Printer Driver. These three selections and their descriptions are provided in the following list.

Name Description
fGeneral Specifies the general capabilities of the printer model. These capabilities include double-sided printing, scrolling the cursor in the negative y-direction, and so on.
fText Specifies the text-rendering capabilities of the printer model in portrait mode.
fLText Specifies the text-rendering capabilities of the printer model in landscape mode.
