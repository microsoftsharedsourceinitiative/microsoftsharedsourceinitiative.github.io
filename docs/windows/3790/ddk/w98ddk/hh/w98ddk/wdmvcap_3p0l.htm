KSPROPERTY_VIDEODECODER_OUTPUT_ENABLE

The KSPROPERTY_VIDEODECODER_OUTPUT_ENABLE read/write property is used to tri-state the output of video decoders that reside on a shared video port bus. This property uses the KSPROPERTY_VIDEODECODER_S structure to set or retrieve information. This property is optional.

typedef struct {
  KSPROPERTY  Property;
  ULONG  Value;            
} KSPROPERTY_VIDEODECODER_S, *PKSPROPERTY_VIDEODECODER_S;

Members

Property
Specifies the property used.
Value
Specifies the value to be set or retrieved.

The KSPROPERTY_VIDEODECODER_S.Value member specifies whether to tri-state the output. Zero indicates tri-state output, and 1 indicates that the device is actively driving the video port bus.
