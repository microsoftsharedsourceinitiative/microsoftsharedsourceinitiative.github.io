Compatibility

Existing mouse drivers written for Windows version 3.x will run with Windows 95, including MOUSE.COM, MOUSE.SYS, MOUSE.DRV, VMOUSE, and OEM setup files (.INF). When the system starts, it detects whether a device driver written for Windows 3.x is installed and, if so, does not load VMOUSE.
