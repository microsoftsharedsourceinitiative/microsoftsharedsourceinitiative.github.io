PCIReadConfig

The PCIReadConfig function reads hardware-specific configuration information from a given offset in the configuration space.

ULONG
  PCIReadConfig(
    ULONG  ulBus, 
    ULONG  ulDevFunc, 
    ULONG  ulOffset
    );

Parameters

ulBus
Bus number.
ulDevFunc
Device/function number.
ulOffset
Offset into configuration space. Must be between zero and 252.

Return Values

Returns the ULONG read from the given offset in the configuration space.
