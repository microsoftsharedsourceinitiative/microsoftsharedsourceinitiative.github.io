Identifying the Default Character

The Default Char control contains a value that identifies the character which Windows renders when a character outside the range defined for First Char through Last Char is requested. This value is a zero-based offset from the first character identified in the Default Char control.

Diconix 150 Plus

For the Diconix 150 Plus printer this value is 14, which corresponds to the period (.).
