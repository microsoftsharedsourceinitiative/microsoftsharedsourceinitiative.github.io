IRP_MN_QUERY_ALL_DATA

The driver receives an IRP_MN_QUERY_ALL_DATA to query for all instances of a data block. It is passed a GUID as the DataPath. If DataPath is valid, the driver should fill the WNODE_ALL_DATA appropriately and complete the IRP.

Members

ProviderId
Contains the WMI provider ID of the device that should respond to this request.
DataPath
Points to a GUID that represents the data to be returned.
BufferSize
Specifies the maximum size available to fill the WNODE_DATA_ALL into the output buffer.
Buffer
Points to the buffer to return WNODE_ALL_DATA.

Return Value

Returns STATUS_SUCCESS or STATUS_WMI_GUID_NOT_FOUND.
