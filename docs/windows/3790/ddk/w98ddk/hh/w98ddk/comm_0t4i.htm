Legacy Contention Functions

These functions are provided for backward compatibility with Windows 3.x and should be used only for that purpose. These functions use only VCD for contention management and apply, therefore, only to ports managed by VCD. In general, the contention management functions specified earlier in "Contention Management Functions" are recommended.
