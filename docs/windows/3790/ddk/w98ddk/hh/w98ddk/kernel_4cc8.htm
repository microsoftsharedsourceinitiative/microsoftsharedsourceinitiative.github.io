VCache_GetSize

Retrieves the cache size. Uses EAX, EDX.

  • Returns information in the following locations:
Location Contents
EAX contains the current number of blocks in the cache if AH is zero. Contains the current quota of the FSD if AH contains an FSD ID (maximum number of block the FSD can own).
EDX contains the maximum number of cache blocks.

On entry, AH contains the FSD ID or zero.
