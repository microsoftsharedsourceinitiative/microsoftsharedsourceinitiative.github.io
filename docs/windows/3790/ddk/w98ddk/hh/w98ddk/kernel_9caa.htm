VxD_LOCKED_CODE_SEG

include vmm.inc

VxD_LOCKED_CODE_SEG

Defines the start of a code segment. Virtual devices use this segment for all code that is not explicitly for initialization of the device.

  • No return value.

This macro creates a 32-bit segment named _LTEXT. The segment is assembled for flat model memory so segment registers CS, DS, ES, and SS are assumed to be equal.

This macro is equal to the VxD_CODE_SEG macro.
