Send Character Descriptor Box

The SEND_CHAR_DESCRIPTOR box contains the command which downloads the character-descriptor and character-data components for a font.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Character Descriptor and Data command (ESC(snW) downloads n bytes of character descriptors and character data for a font.

To specify the Character Descriptor and Data command, type the following in the SEND_CHAR_DESCRIPTOR box:

\x1B(s%dW
