KS_TVTUNER_CHANGE_INFO

The KS_TVTUNER_CHANGE_INFO structure is used to provide VBI codecs with information about the currently tuned channel.

typedef struct tagKS_TVTUNER_CHANGE_INFO {
    DWORD    dwFlags;                // KS_TVTUNER_CHANGE_*
    DWORD    dwCountryCode;          // ITU country/region code
    DWORD    dwAnalogVideoStandard;  // KS_AnalogVideoStandard
    DWORD    dwChannel;              // Channel number
} KS_TVTUNER_CHANGE_INFO, *PKS_TVTUNER_CHANGE_INFO;

Members

dwFlags
Specifies flags indicating the status of the tuning operation. The flag can be one of the following.
Flag Description
KS_TVTUNER_CHANGE_
BEGIN_TUNE		 Indicates that a tuning operation has begun. VBI codecs should stop decoding at this time, since the video signal is likely to be corrupted.
KS_TVTUNER_CHANGE_
END_TUNE		 Indicates completion of a tuning operation. VBI codecs should resume decoding.

dwCountryCode
Specifies the country/region code in use. This corresponds to the ITU long distance dialing code.
dwAnalogVideoStandard
Specifies the current analog video standard, which is enumerated by the KS_AnalogVideoStandard.
dwChannel
Specifies the current channel.

Comments

A packet containing KS_TVTUNER_CHANGE_INFO structure normally originates at a TV Tuner filter, and is propagated through any crossbars to an analog video decoder capture filter. The capture filter in turn must propagate this packet to downstream VBI codecs.
