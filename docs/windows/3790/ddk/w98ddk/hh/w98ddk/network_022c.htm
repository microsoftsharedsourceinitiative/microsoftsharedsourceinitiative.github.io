Connection Functions

Windows 95 supports all the connection functions offered by Windows NT 3.51. Windows 95 also offers the WNetSetConnection and WNetUseConnection functions, which are documented in Multiple Provider Router.

Networking API Windows 95 Windows 3.51
WNetAddConnection Yes Yes
WNetAddConnection2 Yes Yes
WNetAddConnection3 Yes Yes
WNetCancelConnection Yes Yes
WNetCancelConnection2 Yes Yes
WNetGetConnection Yes Yes
WNetGetUniversalName Yes Yes
WNetSetConnection Yes No
WNetUseConnection Yes No
