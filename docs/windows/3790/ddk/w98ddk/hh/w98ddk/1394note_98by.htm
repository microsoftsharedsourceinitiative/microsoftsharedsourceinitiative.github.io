Windows 98/Me: Windows DDK

IEEE 1394 Bus Drivers

See the Windows 2000 DDK for information about creating IEEE 1394 client drivers. The IEEE 1394 documentation, samples, header files, and libraries in the Windows 2000 DDK can be used in developing 1394 client drivers for Windows Me and Windows 98 as well as for Windows 2000.

The 1394 samples included in the Windows 2000 DDK can be built and run on Windows 98 Gold, Windows 98 SE, and Windows Me as is.

For more information on IEEE 1394 driver development for Windows 98 Gold, Windows 98 SE, Windows Me, and Windows 2000, see the Microsoft IEEE 1394 technology Web site.
