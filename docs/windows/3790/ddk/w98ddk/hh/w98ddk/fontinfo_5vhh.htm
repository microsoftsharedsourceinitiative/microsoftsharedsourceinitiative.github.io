Cartridge Name

The Name of Cartridge box contains a string that specifies a font-cartridge name.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Name of Cartridge box contains the string Z: Microsoft 1A which identifies the Microsoft 1A font cartridge.
