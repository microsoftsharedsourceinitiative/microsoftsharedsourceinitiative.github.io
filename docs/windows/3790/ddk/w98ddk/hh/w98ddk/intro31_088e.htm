Introduction

This document describes Microsoft® Windows® UniTool — a new printer-driver development tool which you use to create a Windows printer driver that uses the Microsoft Windows Universal Printer Driver dynamic-link library (UNIDRV.DLL). The printer driver, referred to as a minidriver, contains data which describes the printer's capabilities, commands, and its resident fonts. The code in a minidriver forwards graphics device interface (GDI) calls to the Universal Printer Driver. The Universal Printer Driver carries out the appropriate actions on the device (such as printing text, rendering bitmaps, or advancing a page).
