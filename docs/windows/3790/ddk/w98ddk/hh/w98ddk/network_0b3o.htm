Setting Up the Session Management Module Overlay

The overlaying SMM can be installed for Windows 95 Dial-Up Networking as an external SMM. Dial-Up Networking treats the overlaying SMM as a separate SMM from the one it overlays. The user can choose to use the overlaying SMM by:

  • Open the connection icon's Property page.
  • Click on the Server Type button to bring up the Server type dialog.
  • Select the overlaying SMM from the list of Dial-Up Server type.
