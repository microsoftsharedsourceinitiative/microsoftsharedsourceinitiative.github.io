Default and Supported Printer Resolutions

The Resolutions dialog box identifies the default, the supported, and the unsupported printer resolutions. The default resolution appears in the dropdown list box that appears at the top of the dialog box. The supported resolutions are the highlighted resolutions that appear in the list box below the dropdown list box.

The following list describes each of the controls in this dialog box.

Control Description
Default Specifies the default printer resolution.
List box Lists the available resolutions for this printer.
Enable All Enables all resolutions which appear in the list box.
Inhibit All Inhibits all resolutions which appear in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus printer supports a low-, medium- and high-resolution of 80 x 96, 160 x 96, and 320 x 96 dpi respectively.

HP LaserJet IIP

The HP LaserJet IIP printer supports a low-, medium-, and high-resolution of 75 x 75, 150 x 150, and 300 x 300 dpi respectively.
