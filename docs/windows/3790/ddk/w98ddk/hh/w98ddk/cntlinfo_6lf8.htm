Relative-Up Cursor Movement

The CM_OCD_YM_RELUP box contains the printer command or commands which move the cursor in the negative y-direction with respect to the current position. For most printers, the corresponding Units dialog box will contain values that the Universal Printer Driver will use to calculate the argument for the relative-up cursor-movement command arguments from master units.

NOTE  A driver should not attempt to use the CM_OCD_YM_RELUP cursor movement command on a non-laser printer.

Diconix 150 Plus

The Diconix 150 Plus printer does not support a command that moves the cursor back up the page; therefore, the CM_OCD_YM_RELUP box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the relative Vertical Cursor Positioning command (ESC*p-nY) moves the cursor by n units in the negative y-direction with respect to the current cursor location. The variable n specifies the distance of movement in 1/300 inch units.

To specify the relative Vertical Cursor Positioning command, type the following into the CMD_OCD_YM_REL box:

\x1B*p-%dY
