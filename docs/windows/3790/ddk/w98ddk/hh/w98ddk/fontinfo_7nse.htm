Enabling Italic Simulation

The FS_OCD_ITALIC_ON box contains the command which enables italic simulations for a printer.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Italic Mode On command (ESC4) enables italic simulations.

To specify the Italic Mode On command, type the following in the FS_OCD_ITALIC_ON box:

\x1B4

HP LaserJet IIP

For the HP LaserJet IIP, in the FS_OCD_ITALIC_ON box, the font-style command is (ESC(snS). If n is 1, the characters are italicized; if n is 0, the characters are plain text.

However because the TC_IA_ABLE check box in the fText dialog box is not checked, the Universal Printer Driver will not perform italic simulations. The command to select italics is included in a font simulation structure to override the TC_IA_ABLE check box.

To specify italic output, type the following in the FS_OCD_ITALIC_ON box:

\x1B(s1S
