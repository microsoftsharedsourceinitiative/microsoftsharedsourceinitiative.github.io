FriendlyName

FriendlyNamenumber=string

Specifies a user-defined name for the given hardware profile. The number is a 4-digit, hexadecimal value identifying the hardware profile. The string can be any null-terminated string.

This value is used in SYSTEM\CurrentControlSet\Control\IDCONFIGDB.
