VPOWERD_W32_Get_System_Power_Status

VPOWERD_W32_Get_System_Power_Status retrieves the power status of the system.

POWERRET VPOWERD_W32_Get_System_Power_Status(
  Win32_System_Power_Status   *pWin32_System_Power_Status
 );

Parameters

pWin32_System_Power_Status
Structure that is completed with the relevant system information.

Return Values

The return value is TRUE, is successful. Otherwise, the return value is FALSE.
