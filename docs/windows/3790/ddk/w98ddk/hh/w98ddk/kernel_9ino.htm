V86MMGR_NoUMBInitCalls

include v86mmgr.inc

VxDcall V86MMGR_NoUMBInitCalls

Supports QEMM version 5.x. This service is called by the VMM before the Sys_Critical_Init message is done, and allows the XMSUMBINITCALLS=FALSE setting in the SYSTEM.INI file.

This service is only available for Windows version 3.1 and later.
