Adding the Cursor-Movement Data to Your Driver

Once you've specified the necessary commands and the ordering of those commands, you should add the cursor-movement data to your driver's data table. You can do this by choosing the Add or OK buttons.

The driver data table contains two types of control information: page-control data and cursor-movement data. Once you've set the controls in both of these dialog boxes, you can begin specifying paper information.
