VDD_Get_ModTime

    include minivdd.inc
    VxDCall VDD_Get_ModTime

The VDD_Get_ModTime service specifies whether any video activity has occurred. The virtual poll device uses it to determine if the virtual machine is idle.

Parameters

EBX Specifies the virtual machine handle.

Return Values

The EAX register contains the system time at last video modification.

Uses

Flags, EAX
