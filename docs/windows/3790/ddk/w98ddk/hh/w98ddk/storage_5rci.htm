ILB_enqueue_iop

cCall [ilb.ILB_enqueue_iop],<<OFFSET32 IOP>, <OFFSET32 DCB>>

Places the given IOP in the IOP queue. The position of the IOP depends on the queuing algorithm specified by the DCB_q_algo member. Uses EAX, ECX, EDX, and Flags registers.

  • No return value.
IOP
Address of the IOP structure to place in the queue.
DCB
Address of a physical DCB structure.
