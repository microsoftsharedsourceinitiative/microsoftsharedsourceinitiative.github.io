GET_MONITOR_INFO

The GET_MONITOR_INFO function obtains the EDID of the monitor.

CONFIGRET get_monitor_info(
  DEVNODE dn,
  int UnitNum,
  VESA_EDID *pedid
  );

Parameters

dn
Device node of the display adapter.
UnitNum
Monitor number.
*pedid
Pointer to the buffer that obtains the EDID information.

Return Value

Returns CR_SUCCESS if successful, or CR_FAILURE if an error has occurred, or *pedid to copy the EDID block content to the buffer.
