Page-Size Limits

The Page Size Limits controls specify the minimum allowable physical page size, and the maximum printable region widths and heights (in master units) that are allowed for user-defined paper sizes. If your minidriver supports user-defined paper sizes, and you have initialized the appropriate controls in the Paper Sizes dialog box, you will also need to set the Page Size Limits controls.

Diconix 150 Plus

For the Diconix 150 Plus printer, the maximum allowable printable width is imposed by the model itself, and is set to 6624 (6.9 inches as expressed in master units). The Diconix printer does not impose any paper-length constraints. For example, an arbitrary length of 24 inches was chosen that translates into 13,824 vertical master units (24 * 576 = 13,824).

The minimum allowable width for the Diconix printer is imposed by the rubber rollers on the bail arm. These rollers can be adjusted from 6 to 8 inches apart imposing the smallest allowable paper width of 6 inches which translates into 1,924 horizontal master units (6 * 320 = 1,924). The minimum allowable height is the distance from the entrance of the paper-feed slot to the top bail arm. This distance is 3.5 inches or 2,016 vertical master units (3.5 * 576 = 2,016).

HP LaserJet IIP

For the HP LaserJet IIP printer, the maximum allowable printable width is 8 inches. This value translates into 2400 horizontal master units (8 * 300 = 2400). The maximum allowable printable height is 13 inches. This value translates into 3900 master units (13 * 300 = 3900). The minimum allowable page width and height is one inch. This value translates into 300 master units (1 * 300 = 300).
