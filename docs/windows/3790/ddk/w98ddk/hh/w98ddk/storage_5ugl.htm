ScsiPortWritePortUshort

VOID ScsiPortWritePortUshort(Port, Value)
IN PUSHORT  Port;
IN USHORT  Value;

Transfers a USHORT value to an I/O port.

  • No return value.
Port
Points to the port.
Value
Specifies the value to be written to the port.
