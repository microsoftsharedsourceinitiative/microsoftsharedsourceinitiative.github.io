KSPROPERTY_TUNER_FREQUENCY

The KSPROPERTY_TUNER_FREQUENCY write-only property is used to get or set the tuning frequency. 

typedef struct {
  KSPROPERTY Property;
  ULONG  Frequency;  // Hz
  ULONG  LastFrequency;  // Hz (last known good)
  ULONG  TuningFlags;  // KS_TUNER_TUNING_FLAGS
  ULONG  VideoSubChannel;  // DSS
  ULONG  AudioSubChannel;  // DSS
  ULONG  Channel;  // VBI decoders
  ULONG  Country;  // VBI decoders
} KSPROPERTY_TUNER_FREQUENCY_S, *PKSPROPERTY_TUNER_FREQUENCY_S;

Members

Property
Specifies the property used.
Frequency
Specifies the frequency in Hz.
LastFrequency
Specifies the last known fine tune frequency. This field is provided only for drivers that implement the KS_TUNER_STRATEGY of KS_TUNER_STRATEGY_DRIVER_TUNES as a starting point when beginning a new fine tune operation.
TuningFlags
Specifies how the driver should proceed with the fine tuning operation using KS_TUNER_TUNING_FLAGS. The following flags are provided.
Flag Description
KS_TUNER_TUNING_EXACT Specifies that the tuner should tune directly to Frequency, bypassing any fine tuning logic.
KS_TUNER_TUNING_FINE Specifies the tuning operation should perform a comprehensive search for the best tuning. This flag is only used if the strategy is KS_TUNER_STRATEGY_DRIVER_TUNES.
KS_TUNER_TUNING_COARSE Specifies that the tuning operation should perform a fast search, attempting only to determine if a valid signal is present. This flag is only used if the strategy is KS_TUNER_STRATEGY_DRIVER_TUNES.

VideoSubChannel
Specifies a video subchannel for digital television sources.
AudioSubChannel
Specifies an audio subchannel for digital television sources.
Channel
Specifies the TV channel when the tuning mode is equal to KSPROPERTY_TUNER_MODE_TV.
Country
Specifies the current country/region code for the tuning operation. Country/region codes follow the TAPI country/region code values which correspond to the International long distance dialing codes.
