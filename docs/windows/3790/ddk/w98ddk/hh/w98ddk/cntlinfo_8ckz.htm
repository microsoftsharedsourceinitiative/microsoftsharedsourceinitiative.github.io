Multiple Copies

The PC_OCD_MULT_COPIES box contains the command or commands which the Universal Printer Driver issues to print multiple copies of a document. (The Universal Printer Driver will not process the multiple copies command unless the MD_COPIES box is checked in the fGeneral dialog box.)

Diconix 150 Plus

The Diconix 150 Plus printer does not support a command which enables multiple-copy printing; therefore the PC_OCD_MULT_COPIES box is empty.

HP LaserJet IIP

For the HP LaserJet IIP, the Number of Copies command (ESC&lnX) is required to print n copies of a document.

To specify the Number of Copies command, type the following in the PC_OCD_MULT_COPIES box:

\x1B&l%dX
