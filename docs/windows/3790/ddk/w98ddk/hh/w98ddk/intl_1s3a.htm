ImmLockIMCC

LPVOID WINAPI ImmLockIMCC(HIMCC hIMCC)

Retrieves the address of the IMC component that can be the member of IMC, and increases the lock count for IMCC.

If successful, the return value is the address of the IMC component. Otherwise, the return value is NULL.

hIMCC
Handle of IMC component.
