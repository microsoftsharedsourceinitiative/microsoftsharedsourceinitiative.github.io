VDD_OPEN_KEY

VDD_OPEN_KEY opens the setting key in the registry for the given device. The caller must close this key using the RegCloseKey function.

Entry Meaning
Client_EAX VDD_OPEN_KEY equate from Minivdd.inc or Minivdd.h.
Client_EBX Device handle.
Client_ECX Size of ES:EDI, Must be 4, or sizeof (HKEY)
Client_EDX Flags:

VDD_OPEN_KEY_GLOBAL - open the global (not per-user) settings.
VDD_OPEN_KEY_READ - will only be reading settings.
VDD_OPEN_KEY_USER - open the per-user settings.
VDD_OPEN_KEY_WRITE - will be writing settings.

Note If neither VDD_OPEN_KEY_USER nor VDD_OPEN_KEY_GLOBAL is specified, then the correct global/per user settings key will be opened.
Client_ES:EDI Points to location to store the opened registry key.

Exit Meaning
Client_EAX Zero for success, or Win32 error code.
