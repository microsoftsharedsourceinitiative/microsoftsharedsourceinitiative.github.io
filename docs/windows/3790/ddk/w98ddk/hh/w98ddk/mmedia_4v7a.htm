MCI_SET_PARMS

typedef struct {
    DWORD  dwCallback;
    DWORD  dwTimeFormat;
    DWORD  dwAudio;
} MCI_SET_PARMS;

The MCI_SET_PARMS structure contains parameters for the MCI_SET message. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
dwTimeFormat
Specifies the time format used by the device.
dwAudio
Specifies the audio output channel.
