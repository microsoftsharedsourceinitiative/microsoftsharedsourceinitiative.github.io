DRV_QUERYCONFIGURE

Checks if the driver provides a configuration dialog box.

  • Returns DRV_SUCCESS if the driver provides a dialog box that enables dynamic configuring; DRV_FAILURE if the driver does not support dynamic configuring.
lParam1
Should be set to zero.
lParam2
Should be set to zero.
