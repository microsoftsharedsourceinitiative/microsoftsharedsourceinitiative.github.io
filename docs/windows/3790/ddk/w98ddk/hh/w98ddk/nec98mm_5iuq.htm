PC-9801/9821-Series Display Adapter

The following three types of display adapters can be used for the PC-9801/9821 series:

  • Currently compatible graphics
  • Multiple color extension graphics
  • Window accelerator
  • 98 Core Graphics

Currently Compatible Graphics (Compatible Graph)

This type of display adapter is pre-installed in the PC-9801/9821 series and has the following features:

  • Plain format, 640 x 200/400, 8/16 colors
  • Overlapping with text is enabled

Multiple Color Extension Graphics (Extension Graph)

This type of display adapter is pre-installed starting from the PC-9821 series and has the following features (not installed in the PC-9821B series):

  • Packed pixel format, 640 x 400/480, 256 colors
  • Overlapping with text is enabled

Window Accelerator

The following three types of window accelerators can be used for the PC-9801/9821 series.

  • Internal window accelerator
  • NEC window accelerator board
  • IHV window accelerator board

98 Core Graphic (Core Graph)

This type of adapter includes all standard video systems (memory, I/O, etc.) in the PC9801/9821 series. This refers to the currently compatible graphics, the multiple color extension graphics, and text. The windows accelerator is not included.
