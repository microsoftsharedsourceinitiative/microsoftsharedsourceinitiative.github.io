VKD_Get_Kbd_Owner

include vkd.inc

VxDCall VKD_Get_Kbd_Owner
mov     VMHandle, ebx

Retrieves the virtual machine handle of the keyboard focus virtual machine. Uses EBX and Flags.

  • Returns the handle in EBX.
