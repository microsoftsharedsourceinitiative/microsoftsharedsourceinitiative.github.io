Supported Cursor Movements

The CURSORMOVE dialog box identifies the supported cursor-movement commands. These commands are sent to the printer to move the cursor (or print head) horizontally and vertically across a page of paper.

Control Description
List box Lists the available cursor-movement data for this printer.
Inhibit All Inhibits all cursor-movement data which is identified in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus supports a single set of cursor-movement commands.

HP LaserJet IIP

The HP LaserJet IIP supports the second of two sets of cursor-movement commands.
