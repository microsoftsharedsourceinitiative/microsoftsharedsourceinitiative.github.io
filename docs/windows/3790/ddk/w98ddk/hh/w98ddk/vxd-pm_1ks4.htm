System Power Management

Computers can be put into low power sleep states in response to a user command, or when the computer has been idle for some period of time. When a computer goes into a system sleeping state, most of the devices in the computer must either be turned off or put in a low power state. Devices go into low power states using the process described in Switching to Low Power State.
