Selecting the Paper Source

The Select String box contains the printer command or commands which select the paper source identified in the Paper Sources dialog box.

Diconix 150 Plus

For the Diconix 150 Plus printer, the default paper source uses the tractor-feed mechanism; therefore, it was not necessary to select this source.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Paper Source command (ESC&l1H) designates that the upper-paper tray should be used as the paper source.

To specify the Paper Source command, type the following in the Select String box:

\x1B&l1H
