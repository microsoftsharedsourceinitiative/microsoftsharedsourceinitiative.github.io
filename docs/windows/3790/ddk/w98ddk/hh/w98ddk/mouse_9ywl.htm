Minidriver Requirements

There are no requirements for naming a mouse minidriver, and there are no services that a minidriver must provide to other VxDs in the system. A minidriver can have any version number, its identifier should be set to Undefined_Device_Id, and its initialization order should be set to Undefined_Init_Order.
