Number of Bits per Pixel

The sBitsPixel box contains a value that specifies the number of bits of data required for each color pixel.

HP PaintJet

For the Hewlett-Packard PaintJet printer, 1 bit in each plane corresponds to a single pixel; therefore, the sBitsPixel box contains 1.
