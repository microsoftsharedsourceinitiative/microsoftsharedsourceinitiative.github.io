VPOWERD_W32_Get_Power_Status

VPOWERD_W32_Get_Power_Status retrieves the current power status of the device.

POWERRET VPOWERD_W32_Get_Power_Status(
  DWORD   Power_Device_ID, 
  Win32_System_Power_Status   pWin32_System_Power_Status
);

Parameters

Power_Device_ID
ID of the device to be polled.
pWin32_System_Power_Status
Structure to be completed with the relevant statuses.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
