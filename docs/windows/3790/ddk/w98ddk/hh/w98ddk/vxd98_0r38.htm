IFSMgr_DeregisterFSD

IFSMgr_DeregisterFSD unloads a previously registered FSD. This routine is called to dynamically unload an FSD that has already been loaded. Note there is no need for this function to be called when the system is shutting down. This call can handle the unloading of all the various types of FSDs that can be present in the system. A force level can also be specified to indicate what level of cleanup is allowed. If the specified force level is not high enough for all the cleanup to be done, the unload operation will be failed.

NTSTATUS IFSMgr_DeregisterFSD(
  DWORD   FSDType,
  DWORD   ProID,
  DWORD   Force,
  DWORD   Version 
 );

Parameters

FSDType
Type of FSD being deregistered.
ProID
Provider ID of FSD.
Force
Force level to use.
Version
Version of IFSMgr used by FSD.

Return Values

The return value is NTSUCCESS if successful. Otherwise, the appropriate error code is returned.
