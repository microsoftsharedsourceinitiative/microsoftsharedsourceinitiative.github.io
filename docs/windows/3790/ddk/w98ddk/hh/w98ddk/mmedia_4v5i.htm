MCI_OVLY_LOAD_PARMS

typedef struct {
    DWORD  dwCallback;
    DWORD  lpfilename;
    RECT  rc;
} MCI_OVLY_LOAD_PARMS;

The MCI_OVLY_LOAD_PARMS structure contains parameters for the MCI_LOAD message for video overlay devices. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
lpfilename
Specifies a far pointer to the buffer containing a null-terminated string.
rc
Specifies a rectangle.
