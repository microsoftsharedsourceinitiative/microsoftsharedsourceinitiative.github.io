VCache_FreeBlock

Put cache block onto free list. Uses ECX, EDX, and flags.

  • No return value.

On entry, ESI contains the cache block handle.

VCache_FreeBlock invalidates the contents of a buffer and places the buffer at the LRU end of the MRU/LRU list. VCache_FreeBlock assumes that the file system has already performed any cleanup required, so the buffer flush routine will not be called.
