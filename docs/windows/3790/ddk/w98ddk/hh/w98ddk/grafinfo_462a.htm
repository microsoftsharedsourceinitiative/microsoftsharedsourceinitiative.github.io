Adding Resolution Data to Your Minidriver

Once you've specified:

  • Display format
  • Scale factors
  • Dump data
  • Block data
  • Cursor data
  • Dither data
  • Minimum bitmap height
  • Vertical text offset
  • Spot diameter

you should add this data to the minidriver's data table. In order to do this, choose either the Add Res or the OK pushbuttons. There are four Graphics Info commands: Resolution, Compression, Color, and RectFill. Now that you've initialized the Resolution component, you should select the Compression option in the Graphics Info menu and enter the appropriate data for your printer.
