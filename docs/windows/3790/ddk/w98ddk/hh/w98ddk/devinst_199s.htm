NdiInstall

RETERR
 NdiInstall(hndi)
HNDI hndi;    // NDI handle

The NdiInstall function installs the NDI object by copying the NDI files and performing registry and system file modifications for the driver.

hndi
Identifies the NDI object.
  • The return value is zero (OK) if successful; otherwise it is a nonzero NDI error value.

This function causes the NDI_INSTALL message to be sent to the given NDI object.
