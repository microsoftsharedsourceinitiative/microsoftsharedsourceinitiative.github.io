OEM_APM_REGS

typedef union _OEM_APM_REGS {
    struct _OEM_APM_BYTE_REGS ByteRegs;
    struct _OEM_APM_WORD_REGS WordRegs;
    struct _OEM_APM_DWORD_REGS DwordRegs;
} OEM_APM_REGS;
typedef OEM_APM_REGS * LPOEM_APM_REGS;

Contains information about the CPU registers that can be used by OEM-specific APM functions. This structure is a representation in memory of the values of the CPU registers.
