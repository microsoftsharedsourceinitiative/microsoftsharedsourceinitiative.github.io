VPOWERD_Disable_Resume_On_Ring

VPOWERD_Disable_Resume_On_Ring disables the Resume on Ring capability of the BIOS.

POWERRET VPOWERD_Disable_Resume_On_Ring( VOID );

Parameters

None.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
