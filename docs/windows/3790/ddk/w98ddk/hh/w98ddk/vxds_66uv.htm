Introduction to Virtual Device Drivers

It is important to understand the architecture of the Microsoft Windows® 95 operating system as a whole, in order to understand the role of Windows 95 device drivers.

The Microsoft® Windows® 95 operating system supports multithreaded multitasking for applications. Windows provides a virtual machine manager (VMM) which, in conjunction with virtual devices, processes interrupts and carries out I/O operations for a given application without disrupting the execution of other applications. This section describes the Virtual Machine Manager (VMM), virtual devices, and other components of the operating environment for Windows.
