KS_TUNER_STRATEGY

Drivers can implement one of three methods for fine tuning, depending on their hardware capabilities. 

typedef enum {
    KS_TUNER_STRATEGY_PLL             = 0X01,
    KS_TUNER_STRATEGY_SIGNAL_STRENGTH = 0X02, 
    KS_TUNER_STRATEGY_DRIVER_TUNES    = 0X04,
}KS_TUNER_STRATEGY;
KS_TUNER_STRATEGY Types Description
KS_TUNER_STRATEGY_PLL Used by devices that can indicate tuning accuracy by measuring a PLL offset.
KS_TUNER_STRATEGY_
SIGNAL_STRENGTH		 Used by drivers that indicate tuning accuracy by signal strength. The DirectShow filter performs fine tuning based on feedback returned by the driver for both of these strategies.
KS_TUNER_STRATEGY_
DRIVER_TUNES		 Used by drivers that want total control over the fine tuning process.
