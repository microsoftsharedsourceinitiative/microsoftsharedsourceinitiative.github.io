Registry Layout - ISDN Devices

The Windows® 95 ISDN Support software is designed to support Windows 2000 NDIS WAN miniports unmodified. This means the various miniport drivers that exist today which read and write nonstandard, driver-specific data to the registry (in the driver’s registry key). However, for these drivers, the WANTSP will not allow TAPI apps to set or invoke the device properties.

This documentation describes a standard configuration interface using the WANTSP and a registry layout which must be adopted in order to support this interface. Old drivers must be modified to conform to the standard to be compatible with Windows 95.
