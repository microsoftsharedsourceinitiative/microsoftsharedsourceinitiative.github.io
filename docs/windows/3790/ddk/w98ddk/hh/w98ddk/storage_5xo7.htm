IFSMgr_KillEvent

IFSMgr_KillEvent(
 pevent pev,
 )

This service kills a pending event that could have been scheduled using the IFSMgr_SchedEvent or queued using the IFSMgr_QueueEvent services. If the event was scheduled and the time-out has not yet occurred, the time-out is also canceled.

pev
Supplies pointer to an event structure. This must be the same event structure that was previously used to register the event using the IFSMgr_SchedEvent or IFSMgr_QueueEvent services.
  • This service has no return value.
