PCISetPhysicalCacheTypeRange

The PCISetPhysicalCacheTypeRange function marks the specified range of memory with the specified attributes.

ULONG
  PCISetPhysicalCacheTypeRange(
    PVOID  pPhysAddress, 
    ULONG  ulNumBytes, 
    MEMORY_CACHING_TYPE  ulCacheType
    );

Parameters

pPhysAddress
Specifies the start of the memory block being typed.
ulNumBytes
Number of bytes in the block.
ulCacheType
Memory type. Can be a combination of the following values.
Value Meaning
PCIPAGEZEROINIT 0x00000001
PCIPAGECACHEDISABLE 0x00000002
PCIPAGECONTIG 0x00000004
PCIPAGEWRITETHROUGH 0x00000008

Return Values

Returns -1 if an error occurred.
