The DEVCOLOR Dialog Box

The following error messages are associated with the DEVCOLOR dialog box.

List Description
minidriver name :sPlanes must = 3 or 4 UniTool issues this message if a value is entered in the sPlanes box which is not equal to 3 or 4.
minidriver name :sBitsPerPixel must = 1 UniTool issues this message if a value is entered in the sBitsPerPixel box that is not equal to 1.
minidriver name :DC_PRIMARY_RGB conflicts w/ number of planes UniTool issues this message if the developer's DC_PRIMARY_RGB selection contradicts the specified number of planes.
minidriver name :DC_EXTRACT_BLACK conflicts w/ number of planes UniTool issues this message if the developer's DC_EXTRACT_BLACK selection contradicts the specified number of planes.
