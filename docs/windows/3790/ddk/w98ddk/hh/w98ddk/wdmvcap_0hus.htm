KSPROPERTY_TUNER_STANDARD

The KSPROPERTY_TUNER_STANDARD read only property is used to set and retrieve the current analog video standard. This property is only used when the current mode is KSPROPERTY_TUNER_MODE_TV. The following structure is used to communicate the standard.

typedef struct {
  KSPROPERTY  Property;
  ULONG      Standard;
} KSPROPERTY_TVTUNER_STANDARD_S, *PKSPROPERTY_TVTUNER_STANDARD_S;

Members

Property
Specifies the property used.
Standard
Specifies the video standard using KS_AnalogVideoStandard.
