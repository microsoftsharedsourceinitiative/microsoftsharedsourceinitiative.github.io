Minidriver Version Number

The Minor box contains the version number of the minidriver. This value can be any integer between 0 and 255.

If the driver has already been built and installed with Control Panel, changing this value and rebuilding the minidriver will cause the Universal Printer Driver to rebuild the driver's font-summary file. A font-summary file contains a list of fonts currently available for a particular printer model either internally or by the use of downloadable fonts and the font installer. For those printers using a Windows® printer driver that can download soft fonts using a font installer (as with the HP LaserJets and HP DeskJets), the Universal Printer Driver will build a font-summary file.

Diconix 150 Plus

The Minor box contains the value 1, specifying that this is the first version of the minidriver.

HP LaserJet IIP

The Minor box contains the value 41, specifying that this is the forty-first version of the minidriver.
