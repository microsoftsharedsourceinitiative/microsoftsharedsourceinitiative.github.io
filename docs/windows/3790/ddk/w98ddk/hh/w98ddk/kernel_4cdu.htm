VCache_MakeMRU

Make block "most recently used." Uses EAX, ECX, EDX, and flags.

  • No return value.

On entry, ESI contains the cache block handle.

This service is used when an FSD wants to update a buffer's access information without the overhead of calling VCache_FindBuffer.
