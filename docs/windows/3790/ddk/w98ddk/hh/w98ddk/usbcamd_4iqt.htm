CamUnInitialize

CamUnInitialize is invoked after using the peripheral for the last time in a session. While the Stream Class driver automatically resets values associated with the minidriver, this call allows the minidriver to deallocate specific resources used during its operation. This includes any memory it explicitly allocated.

NTSTATUS
  CamUnInitialize(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext
    );

Parameters

BusDeviceObject
Device object associated with the peripheral.
DeviceContext
User-supplied relevant value or structure.

Return Values

The NTSTATUS code indicating the success or failure of the operation. Since the implementation is vendor specific, the errors messages may vary between manufacturers.
