IME Interface

In Windows® 95, IMEs are provided as dynamic link libraries (DLLs) like device drivers. The Input Method Manager (IMM) should handle all the installed IMEs. Because IMEs are changeable at running time without rebooting, IMM will have a structure to maintain all entry points of each IME.

The list of IME functions is the description of all Asian common IME functions. These functions should not be called by an application directly.
