VMDAPI_GET_VERSION

include vmd.inc
 
mov     ax, VMDAPI_GET_VERSION
call    pVMD_API_Entry
mov     VersionNumber, ax

Retrieves the version number of VMOUSE. Uses AX and Flags; clears the carry flag.

  • Returns the version number in AX.
