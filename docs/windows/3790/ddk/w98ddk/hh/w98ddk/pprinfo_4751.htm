Driver-Defined Paper Source

The Source Name box for driver-defined paper sources contains the name of a unique paper source. This box is provided for printers with paper sources that are not identified in the list of predefined selections.

Diconix 150 Plus

For the Diconix 150 Plus printer, the paper source was predefined; therefore, the driver-defined box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the paper source was predefined; therefore, the driver-defined box is empty.
