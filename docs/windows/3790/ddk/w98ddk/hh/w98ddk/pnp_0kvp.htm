CONFIGMG_Run_Detection

CONFIGRET CONFIGMG_Run_Detection(ULONG ulFlags)

Ask CONFIGMG to spawn the detection DLL and run it.

  • Returns CR_SUCCESS if the function is successful. Otherwise, the return value can be CR_INVALID_FLAG.
ulFlags
Must be zero.

This API will be used by devloader that detects a legacy card has been inserted.
