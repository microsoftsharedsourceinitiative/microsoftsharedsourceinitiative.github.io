NdiRegCloseKey

RETERR NdiRegCloseKey(hkey)
HKEY hkey;    // registry key

The NdiRegCloseKey function closes a key opened with NdiRegOpenKey.

hkey
Identifies an opened registry key.
  • The return value is zero (OK) if successful; otherwise it is ERR_NDI_REG_API.
