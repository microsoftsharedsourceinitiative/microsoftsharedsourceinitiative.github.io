16-Bit LAN Manager APIs

LAN Manager 2.1 placed a layer (called the multinet DLL) above the Windows 3.1 WinNet interface which enable clients to simultaneously access NetWare and LAN Manager networks by routing requests to the appropriate network.

Windows 95 supports the 16-bit LAN Manager APIs in full (these are the APIs exported by Netapi.dll and Pmspl.dll). This includes support for network APIs, mailslots, and named pipes. Windows 95 uses the Client for Windows Networks to do this.
