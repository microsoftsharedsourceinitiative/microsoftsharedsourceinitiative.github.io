Infrared is a Part of Windows 95 Plug and Play

An Infrared device class has been added to the set of Windows 95 Plug and Play classes and IrEnum has been extended to be the enumerator for the "Infrared bus" on a Windows 95 machine. If your device provides a way for Windows 95 to retrieve a Plug and Play ID for the device, then your version of the Windows 95 Infrared Communications software will be Plug and Play-enabled.
