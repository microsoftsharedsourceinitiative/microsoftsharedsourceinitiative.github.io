CamFreeBandwidth

CamFreeBandwidth is called after the ISO video stream has been stopped and allows the caller to select an alternate interface that uses no bandwidth.

NTSTATUS
  CamFreeBandwidth(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext
    );

Parameters

BusDeviceObject
Device object created by the hub.
DeviceContext
Peripheral’s device context.

Return Values

Returns the NTSTATUS code indicating STATUS_SUCCESS if successful. Otherwise, it returns an appropriate error message.
