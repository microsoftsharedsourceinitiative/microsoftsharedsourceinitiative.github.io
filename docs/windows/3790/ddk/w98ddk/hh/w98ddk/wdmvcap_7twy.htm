PROPSETID_VIDCAP_VIDEODECODER

The PROPSETID_VIDCAP_VIDEODECODER property set is used to control an analog video decoder. An analog video decoder converts baseband analog video signals into a digital format such as RGB or YUV.

GUID PROPSETID_VIDCAP_VIDEODECODER
DirectShow Interface IAMAnalogVideoDecoder

The property set includes the following of type KSPROPERTY_VIDCAP_VIDEODECODER:

KSPROPERTY_VIDEODECODER_CAPS (Read Only)

KSPROPERTY_VIDEODECODER_STANDARD (Read/Write)

KSPROPERTY_VIDEODECODER_STATUS (Read Only)

KSPROPERTY_VIDEODECODER_OUTPUT_ENABLE (Read/Write)

KSPROPERTY_VIDEODECODER_VCR_TIMING (Read/Write)
