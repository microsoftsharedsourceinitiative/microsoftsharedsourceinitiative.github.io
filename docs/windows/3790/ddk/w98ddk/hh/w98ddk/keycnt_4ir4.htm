VKD_Local_Enable_Hot_Key

include vkd.inc

mov     eax, hHotKey
mov     ebx, VMHandle
VxDCall VKD_Local_Enable_Hot_Key

Enables a hot key in the specified virtual machine. Uses Flags.

  • No return value.
hHotKey
Handle of the hot key.
VMHandle
Handle of the virtual machine.
