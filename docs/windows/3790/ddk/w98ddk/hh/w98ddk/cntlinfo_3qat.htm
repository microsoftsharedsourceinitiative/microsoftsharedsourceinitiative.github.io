Landscape Mode

The PC_OCD_LANDSCAPE box contains the command or commands which the Universal Printer Driver issues to print output in landscape mode. A printer should only support the landscape-mode command if it also supports device fonts for both portrait and landscape modes.

Diconix 150 Plus

The Diconix 150 Plus printer does not support commands which set the page orientation; therefore, the PC_OCD_LANDSCAPE box is empty.

HP LaserJet IIP

For the HP LaserJet IIP, the Landscape Orientation command (ESC&l1O) is required to render output in portrait mode.

To specify the Landscape Orientation command, type the following in the PC_OCD_LANDSCAPE box:

\x1B&l1O
