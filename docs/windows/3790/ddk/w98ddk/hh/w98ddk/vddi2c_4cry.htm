I2C Devices and Multimonitor Configuration

When the VGA is running I2C devices in a Windows 98 multiple monitor configuration, the miniVDD must be installed as a miniVDD2 to support I2C on the secondary device.

During the I2C_OPEN call, the miniVDD2 must translate the physical device object to a device node by using the _NtKernPhysicalDeviceObjectToDevNode function.
