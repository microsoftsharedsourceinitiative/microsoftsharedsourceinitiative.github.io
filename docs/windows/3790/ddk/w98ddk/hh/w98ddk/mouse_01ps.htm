SETDISPLAYPAGE

include vmd.inc
 
mov     eax, SETDISPLAYPAGE
mov     cl, DisplayPage
Call    CursorProc

Sets the number of the video page on which the cursor VxD displays the cursor.

  • No return value.
DisplayPage
Number of the new video page.
