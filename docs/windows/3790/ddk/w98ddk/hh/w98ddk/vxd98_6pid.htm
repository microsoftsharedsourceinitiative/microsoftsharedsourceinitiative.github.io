VPOWERD_Get_Battery_Unit_Presence

VPOWERD_Get_Battery_Unit_Presence tests for the presence of a battery slot.

POWERRET VPOWERD_Get_Battery_Unit_Presence( 
  DWORD   BatteryDeviceObject
);

Parameters

BatteryDeviceObject
WDM DeviceObject for the battery.

Return Values

The return value is zero if a battery slot is found, and -1 for any other condition.
