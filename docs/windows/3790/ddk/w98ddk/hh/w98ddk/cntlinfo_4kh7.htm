Pushing the Stack

The CM_OCD_PUSH_POS box contains the printer command or commands which push the current position onto the printer's internal stack.

Diconix 150 Plus

The Diconix 150 Plus printer does not provide an internal stack for printer commands; therefore, the CM_OCD_PUSH_POS box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Push Cursor Position command (ESC&f0S) stores the current cursor position on an internal stack.

To specify the Push Cursor Position command, type the following string in the CM_OCD_PUSH_POS box:

\x1B&f0S
