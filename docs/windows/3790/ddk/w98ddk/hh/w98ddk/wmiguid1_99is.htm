IoWMIWriteEvent

The IoWMIWriteEvent routine queues the passed WNODE_EVENT_ITEM for delivery to the user-mode agent. Once the event is delivered, the WNODE_EVENT_ITEM buffer will be returned to the nonpaged pool. This routine must be called at a level equal to or less than DPC_LEVEL. 

NTSTATUS IoWMIWriteEvent(
    IN PVOID WnodeEventItem
    )

Parameters

WnodeEventItem
Pointer to WNODE_EVENT_ITEM that contains event information. This must be allocated from nonpaged pool if this routine is called at a level higher than PASSIVE.

The ProviderID and the Wnode size, which is the BufferSize field in the WNODE_HEADER structure, must be specified in the WNODE_HEADER structure passed in WnodeEventItem.

Return Value

Returns STATUS_SUCCESS or an error code.
