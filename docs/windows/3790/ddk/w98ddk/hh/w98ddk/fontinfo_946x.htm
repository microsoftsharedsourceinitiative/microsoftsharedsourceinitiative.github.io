Pitch and Family

The Pitch and Family control contains an integer value that corresponds to a 16-bit binary value. This value is set by choosing the appropriate controls in the Font Pitch and Family dialog box. You display the Font Pitch and Family dialog box by choosing the ? button. The Pitch value in the Font Pitch and Family dialog box indicates if the font is fixed pitch or proportionally spaced. The Family value affects which screen font Windows will display for this device font.

Diconix 150 Plus

For the Diconix 150 Plus printer, the pitch is fixed and the family is Roman in the Font Pitch and Family dialog box. The corresponding value of 16 appears in the PFM dialog box.
