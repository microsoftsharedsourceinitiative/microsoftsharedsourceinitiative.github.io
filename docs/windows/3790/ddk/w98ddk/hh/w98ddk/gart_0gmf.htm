PCIWriteConfig

The PCIWriteConfig writes a value at the specified offset in the configuration space.

VOID
  PCIWriteConfig(
    ULONG  ulBus, 
    ULONG  ulDevFunc, 
    ULONG  ulOffset, 
    ULONG  ulValue
    );

Parameters

ulBus
Bus number.
ulDevFunc
Device/function number.
ulOffset
Offset into the configuration space.
ulValue
Value to be written.

Return Value

This function has no return value.
