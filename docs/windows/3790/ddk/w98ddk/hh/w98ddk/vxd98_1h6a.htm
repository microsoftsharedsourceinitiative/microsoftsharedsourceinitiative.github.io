VPOWERD_Get_Resume_Timer

VPOWERD_Get_Resume_Timer should only be used within KERNEL32.dll since the list of users requesting to set and use the timer functions is maintained there.

POWERRET VPOWERD_Get_Resume_Timer(
  LPPOWERTIME   pPowerTime
 );

Parameters

pPowerTime
Retrieves the wake-up time of the machine.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
