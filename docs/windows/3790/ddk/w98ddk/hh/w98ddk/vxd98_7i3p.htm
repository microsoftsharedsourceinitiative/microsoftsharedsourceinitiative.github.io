VPOWERD_Get_Mode

VPOWERD_Get_Mode retrieves the operational mode of VPOWERD.

POWERRET VPOWERD_Get_Mode(
  PULONG   pulMode
 );

Parameters

pulMode
Indicates the power mode. This will be zero for APM mode and 1 for ACPI mode.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
