HeapGetSizeEx

HeapGetSizeEx gets the actual size of a block of memory.

ULONG HeapGetSizeEx(
  PVOID   pmem,
  PVOID   Reserved1
 );

Parameters

pmem
Pointer to memory previously returned by HeapAllocateEx.
Reserved1
Reserved. Must be NULL.

Return Values

The return value is the size of the block of memory pointed to by the pmem parameter.
