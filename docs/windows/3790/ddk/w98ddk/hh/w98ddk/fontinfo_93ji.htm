Disabling Underline Simulation

The FS_OCD_UNDERLINE_OFF box contains the command which disables underline simulations for a printer.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Underline Mode command (ESC-n) enables and disables the underline font simulation. If n is 1, the font simulation is enabled; if n is 0, the font simulation is disabled.

To specify the Underline Mode command, type the following in the FS_OCD_UNDERLINE_OFF box:

\x1B-\x00

HP LaserJet IIP

For the HP LaserJet IIP printer, the Disable Underline command (ESC&d@) disables fixed- and floating-position underlining.

To specify the Disable Underline command, type the following in the FS_OCD_UNDERLINE_OFF box:

\x1B&d@
