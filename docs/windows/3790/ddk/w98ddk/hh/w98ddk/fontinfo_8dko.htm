Delete Font Box

The DEL_FONT box contains the command which deletes a single downloaded font from the printer's memory.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Font Control command (ESC*cnF) deletes downloaded fonts, makes downloaded fonts temporary, makes downloaded fonts permanent, and so on. If n is 2, the font whose ID was last specified is deleted.

To specify the Font Control command that deletes the last identified downloaded font, type the following in the DEL_FONT box:

\x1B*c2F
