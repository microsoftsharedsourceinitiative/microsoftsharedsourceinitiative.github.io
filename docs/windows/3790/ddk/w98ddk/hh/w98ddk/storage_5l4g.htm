Application Programming Interface

The file I/O APIs provide support for the Win32 file I/O API functions as well as the DOS INT 21h API functions. The Win32 APIs are defined in the Microsoft Win32 Programmer's Reference. The INT 21h API is defined in the MS-DOS Programmer's Reference.
