Device Fonts

The Resident Fonts button displays a dialog box which contains two list boxes. The list box on the left side contains the names of portrait-mode device fonts. The list box on the right side contains the names of landscape-mode device fonts.

Diconix 150 Plus

For the Diconix 150 Plus printer, the only resident font is 10-point Times Roman for portrait mode.

HP LaserJet IIP

For the HP LaserJet IIP printer, several fonts are available in both portrait and landscape orientation and are selected in each control.
