Resources for WBEM/WMI SMBIOS Provider

  • For information about WBEM, WMI, and Win32® Extensions schema, see http://www.microsoft.com/management/wbem/ and http://wbem.freerange.com
  • See the overview of WMI and the white paper titled Adding WMI Code to a Driver in the Windows 98 DDK.
  • Windows Hardware Instrumentation Implementation Guide (WHIIG) from Intel and Microsoft. This documents the design requirements and recommendations for hardware developers who want to provide custom instrumentation data using the new Windows manageability infrastructure, including WBEM and WMI.
  • For information about this document, please send e-mail to whiig@microsoft.com with your name, e-mail name, company name, and address. You will be notified when this document is available.
