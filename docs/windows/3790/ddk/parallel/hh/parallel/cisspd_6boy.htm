Device Control Requests for Parallel Devices

This section describes how the system-supplied bus driver for parallel ports handles the following device control requests:

IOCTL_IEEE1284_GET_MODE

IOCTL_IEEE1284_NEGOTIATE

IOCTL_PAR_GET_DEFAULT_MODES

IOCTL_PAR_GET_DEVICE_CAPS

IOCTL_PAR_IS_PORT_FREE

IOCTL_PAR_QUERY_DEVICE_ID

IOCTL_PAR_QUERY_DEVICE_ID_SIZE

IOCTL_PAR_QUERY_INFORMATION

IOCTL_PAR_QUERY_LOCATION

IOCTL_PAR_QUERY_RAW_DEVICE_ID

IOCTL_PAR_SET_INFORMATION

IOCTL_PAR_SET_READ_ADDRESS

IOCTL_PAR_SET_WRITE_ADDRESS

IOCTL_SERIAL_GET_TIMEOUTS

IOCTL_SERIAL_SET_TIMEOUTS

The system-supplied bus driver does not support any other device control requests.
