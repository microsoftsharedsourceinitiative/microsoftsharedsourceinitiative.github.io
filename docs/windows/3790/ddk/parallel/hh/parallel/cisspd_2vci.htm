Internal Device Control Requests for Parallel Devices

This section describes how the system-supplied bus driver for parallel ports handles the following internal device control requests:

IOCTL_INTERNAL_DISCONNECT_IDLE

IOCTL_INTERNAL_LOCK_PORT

IOCTL_INTERNAL_LOCK_PORT_NO_SELECT

IOCTL_INTERNAL_PARCLASS_CONNECT

IOCTL_INTERNAL_PARCLASS_DISCONNECT

IOCTL_INTERNAL_UNLOCK_PORT

IOCTL_INTERNAL_UNLOCK_PORT_NO_DESELECT

The system-supplied bus driver does not support any other internal device control requests.
