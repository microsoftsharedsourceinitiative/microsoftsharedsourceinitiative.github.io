UPSCancelWait

The UPSCancelWait function cancels all waits initiated by calls to UPSWaitForStateChange. 

void
  UPSCancelWait(void);

Parameters

None

Return Value

None

Headers

Declared in upssvc.h. Include upssvc.h.

Comments

The call returns immediately.
