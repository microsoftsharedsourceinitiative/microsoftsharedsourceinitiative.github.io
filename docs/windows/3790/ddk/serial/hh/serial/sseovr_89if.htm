Serenum PortName Entry Value

The PortName registry entry value is located under a serial port's Plug and Play hardware device key.

PortName entry value (REG_SZ)
Specifies the name of a serial port. Serenum reads this value and returns the port name in response to an IOCTL_SERENUM_GET_PORT_NAME request.
