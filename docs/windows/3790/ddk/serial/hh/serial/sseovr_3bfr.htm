Serenum Identifer Entry Value

The Identifier registry entry value is located under a serial port's Plug and Play hardware device key.

Identifier entry value (REG_SZ)
Specifies the name of a serial port. Serenum reads this value. Support for the Identifer entry value is provided only for compatibility with some legacy PCMCIA devices. The use of the Identifer entry value is obsolete and should not be implemented with new drivers in Windows 2000 and later. Serenum returns the port name in response to an IOCTL_SERENUM_GET_PORT_NAME request.
