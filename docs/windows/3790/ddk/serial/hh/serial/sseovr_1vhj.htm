Open and Initialize a 16550 UART-Compatible Interface

When Serial is used as a lower-level device filter driver, the following considerations apply to opening and initializing the interface:

  • Serial supports only one open at a time on the interface.
  • The interface is in an undefined state when it is open.

    A client should initialize the interface to a known state before using the interface. A user-mode client can use the IOCTL_SERIAL_SET_Xxx requests. Note, however, that a Win32-compliant application must only use the Win32 Communications API. A kernel-mode client can use the IOCTL_SERIAL_Xxx and the IOCTL_SERIAL_INTERNAL_Xxx requests.
