Registry Settings for Serial

This section describes the registry settings that Serial uses to configure a COM port.

Serial uses the registry settings described in this section to configure a COM port in the following way:

  • If a device-specific entry value is present, Serial uses the device-specific entry value.
  • If a device-specific entry value is not present, but there is a corresponding Serial service entry value, Serial uses the value of the service entry value.
  • If a device-specific entry value is not present and there is no corresponding Serial service entry value, Serial uses a default service value that is statically defined in serial.sys.
