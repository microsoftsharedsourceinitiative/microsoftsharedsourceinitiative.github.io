PGET_DMA_ADAPTER

A PGET_DMA_ADAPTER-typed routine returns a DMA_ADAPTER structure for the target device.

typedef struct _DMA_ADAPTER *(*PGET_DMA_ADAPTER)(
  IN PVOID  Context,
  IN struct DEVICE_DESCRIPTION  *DeviceDescriptor,
  OUT PULONG  NumberOfMapRegisters
);

Parameters

Context
Pointer to interface-specific context information that the system provided when the caller queried for the interface.
DeviceDescriptor
Pointer to a DEVICE_DESCRIPTION structure, which describes the attributes of the physical device.
NumberOfMapRegisters
Pointer to, on output, the maximum number of map registers that the driver can allocate for any DMA transfer operation. Caller must allocate a LONG variable to receive this data.

Return Value

A PGET_DMA_ADAPTER-typed routine returns a pointer to a DMA_ADAPTER structure when successful. If an adapter structure cannot be allocated, the routine returns NULL.

Headers

Defined in ntddk.h. Include ntddk.h.

Comments

Normally, driver writers should call IoGetDmaAdapter to obtain this structure.
