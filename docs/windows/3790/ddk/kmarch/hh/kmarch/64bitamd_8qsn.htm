Unprototyped Functions

For functions that are not fully prototyped, the caller passes integer values as integers and floating-point values as double precision. For floating-point values only, both the integer register and the floating-point register contains the float value in case the callee obtains the value from the integer registers.

The following example shows how the caller passes parameters for unprotyped functions:

func1();
func2() {
  func1(2, 1.0, 7);
}

Func2 passes 2 in RCX, 1.0 in RDX and XMM1, and 7 in R8.
