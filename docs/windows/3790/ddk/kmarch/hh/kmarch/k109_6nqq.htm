RtlCharToInteger

The RtlCharToInteger routine converts a single-byte character to an integer value in the specified base. 

NTSTATUS 
  RtlCharToInteger(
    IN PCSZ  String,
    IN ULONG  Base  OPTIONAL,
    IN OUT PULONG  Value
    );

Parameters

String
Pointer to a zero-terminated, single-byte character string.
Base
Specifies decimal, binary, octal, or hexadecimal base.
Value
Pointer to a location to which the converted value is returned.

Return Value

RtlCharToInteger returns STATUS_SUCCESS if the given character is converted. Otherwise, it can return STATUS_INVALID_PARAMETER.

Headers

Declared in ntddk.h. Include ntddk.h.

Comments

RtlCharToInteger converts ANSI alphanumeric characters.

Callers of RtlCharToInteger must be running at IRQL = PASSIVE_LEVEL.
