Kernel-Mode Driver Architecture: Windows DDK

KAFFINITY

The KAFFINITY type specifies a bitmap that represents a set of processors.

Headers

Declared in basetsd.h. Include wdm.h or ntddk.h. (For Windows 2000, the KAFFINITY type is instead declared in wdm.h and ntddk.h. For Windows 98, this type is instead declared in wdm.h.)
