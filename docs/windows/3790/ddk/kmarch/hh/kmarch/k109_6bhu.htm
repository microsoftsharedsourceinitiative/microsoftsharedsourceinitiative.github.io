RtlIsZeroLuid

The RtlIsZeroLuid routine determines if the specified LUID is the zero LUID.

BOOLEAN
  RtlIsZeroLuid(
    PLUID  L1
    );

Parameters

L1
Specifies the LUID to check.

Return Value

TRUE if L1 is zero, FALSE otherwise.

Headers

Declared in ntddk.h. Include ntddk.h.
