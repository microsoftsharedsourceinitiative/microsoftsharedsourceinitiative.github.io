Increased ANSI/ISO C++ Compliance

The compiler for AMD64 supports increased compliance with the ANSI/ISO C++ Standard. Thus, prior ANSI/ISO-compliant code that did not compile with compilers in earlier DDK versions might compile with the compiler for AMD64 In addition, code that did compile on earlier compiler versions might not compile on the compiler for AMD64. The section on code compatibility for AMD64 contains a list of some common code compatibility changes from earlier versions that might affect how you write your code. If the compiler for AMD64 generates build error messages not seen in earlier compilers and the issue is not described in the Code Compatibility for AMD64 section, consult the ANSI/ISO standard first to see if the error is because the original code was not compliant.
