Template Friends And Specialization

The following code does not compile, because the friend declaration declares a new nontemplate function, which is never defined:

template<class T> void f(T) {
}
template<class T> struct S {
    friend void f(T);
};
main() {
    S<int> s;
    f(1);        //unresolved external.
}

The workaround is to explicitly specify the friend function’s template argument list:

template<class T> void f(T) {
}
template<class T> struct S {
    friend void f<T>(T);
};
main() {
    S<int> s;
    f(1); 
}
