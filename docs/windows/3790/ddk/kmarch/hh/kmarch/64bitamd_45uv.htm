Linker Long-Branch Region Optimizations

The compiler for AMD64 supports the removal of entries in the long-branch tables that are unused and not required. The linker can use another pass to remove these entries that are no longer necessary, which potentially can result in substantial size reduction of the long-branch regions of large programs (> 16MB). To use this optimization, specify the /opt:lbr linker switch.
