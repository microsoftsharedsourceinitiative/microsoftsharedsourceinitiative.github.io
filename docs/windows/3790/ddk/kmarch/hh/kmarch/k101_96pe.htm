WMI IRP Processing Routines

Drivers can use these routines in processing WMI IRPs.

WmiCompleteRequest
If a driver uses WMISystemControl to dispatch the WMI IRP to a callback routine, then the callback routine can use WMICompleteRequest to complete the IRP.
WmiSystemControl
Dispatches a WMI IRP to a driver-supplied callback routine.
