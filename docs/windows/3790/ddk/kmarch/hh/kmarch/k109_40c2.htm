RtlStoreUlong

The RtlStoreUlong routine stores a ULONG value at a particular address, avoiding alignment faults.

VOID 
  RtlStoreUlong(
    IN PULONG  Address,
    IN ULONG  Value
    );

Parameters

Address
Pointer to a location in which to store a given ULONG value.
Value
Specifies a ULONG value to be stored.

Return Value

None

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

The caller can be running at any IRQL if Address points to nonpaged pool. Otherwise, the caller must be running at IRQL < DISPATCH_LEVEL.
