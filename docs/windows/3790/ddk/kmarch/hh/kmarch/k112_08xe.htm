CM_MCA_POS_DATA

The CM_MCA_POS_DATA structure defines IBM-compatible MCA POS configuration information for a slot.

typedef struct _CM_MCA_POS_DATA {
  USHORT  AdapterId;
  UCHAR  PosData1;
  UCHAR  PosData2;
  UCHAR  PosData3;
  UCHAR  PosData4;
} CM_MCA_POS_DATA, *PCM_MCA_POS_DATA;
