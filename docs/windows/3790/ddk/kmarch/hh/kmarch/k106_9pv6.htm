MmGetMdlByteCount

The MmGetMdlByteCount routine returns the length in bytes of the buffer described by a given MDL.

ULONG 
  MmGetMdlByteCount(
    IN PMDL  Mdl
    );

Parameters

Mdl
Pointer to an MDL.

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Return Value

MmGetMdlByteCount returns the byte count of the buffer described by Mdl.

Comments

Callers of MmGetMdlByteCount can be running at any IRQL. Usually, callers are running at IRQL <= DISPATCH_LEVEL.
