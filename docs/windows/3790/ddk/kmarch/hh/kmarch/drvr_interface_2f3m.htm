PGET_SET_DEVICE_DATA

A PGET_SET_DEVICE_DATA-typed routine either reads data from or writes data to the device's configuration space. 

typedef ULONG (*PGET_SET_DEVICE_DATA)(
  IN PVOID  Context,
  IN ULONG  DataType,
  IN PVOID  Buffer,
  IN ULONG  Offset,
  IN ULONG  Length
);

Parameters

Context
Pointer to interface-specific context information that the system provided when the caller queried for the interface.
DataType
Indicates the type of bus data to be set. For further information, see the discussion of the WhichSpace parameter in the ref page for IRP_MN_READ_CONFIG.
Buffer
Pointer to the buffer that holds the configuration information to be read or to be written to the device's configuration space.
Offset
Contains the offset into the buffer where the data is located.
Length
Indicates the length in bytes of data to be written.

Return Value

A PGET_SET_DEVICE_DATA-typed routine returns the length, in bytes, of the configuration data read after a successful read. If the read operation is unsuccessful, a value of zero is returned. For write operations a PGET_SET_DEVICE_DATA-typed routine returns the number of bytes written when successful, and a value of zero when unsuccessful.

Headers

Defined in ntddk.h. Include ntddk.h.
