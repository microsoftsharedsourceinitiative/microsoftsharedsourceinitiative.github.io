Floating-Point Nontype Template Parameters Are Not Allowed

The following code that uses a floating-point, nontype template parameter produces errors:

template<class T, float f> void foo(T) {}

The workaround is a function template and uses a function argument to pass in the float. If the template is a class template, there is no easy workaround; therefore, use an alternative coding scheme. 

template<class T> void foo(T, float) {}
