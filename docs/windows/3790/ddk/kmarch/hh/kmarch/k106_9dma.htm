MmGetMdlPfnArray

The MmGetMdlPfnArray routine returns a pointer to the beginning of the array of physical page numbers associated with the MDL.

PPFN_NUMBER
  MmGetMdlPfnArray(
    IN PMDL  Mdl
    );

Parameters

Mdl
Pointer to an MDL.

Return Value

A pointer to the beginning of the array of physical page numbers associated with the MDL.

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

Callers of MmGetMdlPfnArray can be running at any IRQL.
