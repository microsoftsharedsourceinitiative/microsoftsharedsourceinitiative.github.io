ExIsResourceAcquiredLite

The ExIsResourceAcquiredLite routine returns whether the current thread has access (either shared or exclusive) to a given resource. 

USHORT 
  ExIsResourceAcquiredLite(
    IN PERESOURCE  Resource
    );

Parameters

Resource
Pointer to the resource to be queried.

Return Value

ExIsResourceAcquiredLite returns the number of times the caller has acquired the given resource for shared or exclusive access.

Headers

Declared in ntddk.h. Include ntddk.h.

Comments

The system considers exclusive access to be a subset of shared access. Therefore, a thread that has exclusive access to a resource also has shared access to the resource. Callers of ExIsResourceAcquiredLite must be running at IRQL <= DISPATCH_LEVEL.
