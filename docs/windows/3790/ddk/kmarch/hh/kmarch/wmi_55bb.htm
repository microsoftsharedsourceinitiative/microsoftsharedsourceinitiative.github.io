WMI Property Sheets

A user-friendly driver allows users to control its settings through its Device Manager property sheet. See Using the Device Manager for a description of the Device Manager.

Drivers can automatically expose any WMI classes they implement on their property sheet by using the WMI Generic Property Page Provider.

Drivers can enable certain controls on the Power Management tab of their Device Manager property sheet by supporting certain particular WMI class GUIDs. See WMI and the Power Management Tab for details.
