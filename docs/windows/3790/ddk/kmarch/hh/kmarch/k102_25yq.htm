ExReleaseResourceLite

The ExReleaseResourceLite routine releases a specified executive resource owned by the current thread.

VOID 
  ExReleaseResourceLite(
    IN PERESOURCE  Resource,
    );

Parameters

Resource
Pointer to an executive resource owned by the current thread.

Return Value

None

Headers

Declared in ntddk.h. Include ntddk.h.

Comments

Callers of ExReleaseResourceLite must be running at IRQL <= DISPATCH_LEVEL.
