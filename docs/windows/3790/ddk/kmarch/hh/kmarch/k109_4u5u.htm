RtlZeroBytes

The RtlZeroBytes routine fills a block of memory with zeros, given a pointer to the block and the length, in bytes, to be filled and is obsolete. For better performance, use RtlZeroMemory.

  RtlZeroBytes( 
    PVOID  Destination, 
    SIZE_T  Length
