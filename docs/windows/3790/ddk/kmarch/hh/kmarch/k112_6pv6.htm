LPGUID

The LPGUID structure represents a long pointer to a GUID. 

typedef struct _GUID {
  ULONG  Data1;
  USHORT  Data2;
  USHORT  Data3;
  UCHAR  Data4[8];
} GUID
 
typedef GUID *LPGUID;

Headers

Defined in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

A GUID is a 128-bit unique identifier.
