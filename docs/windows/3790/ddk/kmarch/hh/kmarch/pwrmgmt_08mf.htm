DeviceD1 and DeviceD2

The DeviceD1 and DeviceD2 fields indicate whether the device hardware supports these device states. Each is a single bit, which is set if the device supports the state and is clear if the device does not support the state. The operating system assumes that all devices support the D0 and D3 device power states.
