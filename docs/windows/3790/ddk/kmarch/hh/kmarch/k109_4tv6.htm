RtlEqualLuid

The RtlEqualLuid routine compares two LUIDs.

BOOLEAN 
  RtlEqualLuid( 
    LUID  Luid1, 
    LUID  Luid2 
    );

Parameters

Luid1
Pointer to a LUID to compare.
Luid2
Pointer to a LUID to compare.

Return Value

RtlEqualLuid returns TRUE if Luid1 and Luid2 are equivalent.

Headers

Declared in ntddk.h. Include ntddk.h.

Comments

Callers of RtlEqualLuid can be running at any IRQL.
