RtlUlongByteSwap

The RtlUlongByteSwap routine converts a ULONG from little-endian to big-endian, and vice versa.

ULONG FASTCALL
  RtlUlongByteSwap(
    IN ULONG  Source
    );

Parameters

Source
ULONG to convert.

Return Value

The converted ULONG value.

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

This routine is not supported on Windows 98 or Windows Me.
