Intrinsic Functions to Read and Write Registers

The intrinsic functions that read and write registers in the following table are implemented instead of inline assembly.

Function Operation Performed
ULONG64 __readcr0(VOID); Read CR0.
VOID __writecr0(ULONG64); Write CR0.
ULONG64 __readcr2(VOID); Read CR2.
ULONG64 __readcr2(VOID); Read CR3.
VOID __writecr3(ULONG64); Write CR3.
ULONG64 __readcr4(VOID); Read CR4.
VOID __writecr4(ULONG64): Write CR4.
ULONG64 __readcr8(VOID); Read CR8
VOID __writecr8(ULONG64); Write CR8
ULONG64 __readmsr(ULONG); Read model-specific register.
VOID __writemsr(ULONG, ULONG64); Write model-specific register.
ULONG64 __rdtsc(VOID); Generate an RDTSC instruction to read the time-stamp counter.
