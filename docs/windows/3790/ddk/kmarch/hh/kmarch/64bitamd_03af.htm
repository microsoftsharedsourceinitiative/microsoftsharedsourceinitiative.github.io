String MOV Intrinsic Functions

The following string MOV intrinsic functions are implemented instead of inline assembly:

VOID __movsb(PUCHAR, PUCHAR, byte-count);

VOID __movsw(PUSHORT, PUSHORT, word-count);

VOID __movsd(PULONG, PULONG, long-count);

VOID __movsq(PULONGLONG, PULONGLONG, quad-count);
