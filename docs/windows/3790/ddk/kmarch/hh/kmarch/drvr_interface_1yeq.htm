PTRANSLATE_BUS_ADDRESS

A PTRANSLATE_BUS_ADDRESS-typed routine translates addresses on the parent bus to logical addresses.

typedef BOOLEAN (*PTRANSLATE_BUS_ADDRESS)(
  IN PVOID  Context,
  IN PHYSICAL_ADDRESS  BusAddress,
  IN ULONG  Length,
  IN OUT PULONG  AddressSpace,
  OUT PPHYSICAL_ADDRESS  TranslatedAddress
);

Parameters

Context
Pointer to interface-specific context information that the system provided when the caller queried for the BUS_INTERFACE_STANDARD interface.
BusAddress
Contains the bus-relative address to be translated.
Length
Indicates the length, in bytes, of the input memory pointed to by BusAddress.
AddressSpace
Indicates, on input, the address space in which the device's hardware address resides. A value of 0x0 signifies memory space, and a value of 0x1 signifies port I/O space. On output, AddressSpace indicates the address space in which the TranslatedAddress resides.
TranslatedAddress
Contains the translated (logical) address that corresponds to the bus-relative address provided by the caller in BusAddress.

Return Value

A PTRANSLATE_BUS_ADDRESS-typed routine returns TRUE if the translation operation succeeded and FALSE otherwise.

Headers

Defined in ntddk.h. Include ntddk.h.
