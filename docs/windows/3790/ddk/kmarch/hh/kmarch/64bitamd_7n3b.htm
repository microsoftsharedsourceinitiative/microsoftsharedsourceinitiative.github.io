Private Base Classes Inaccessible

In the following example, A is private to B and thus should be inaccessible to C. However, A can be accessed with the scoping operator.

struct A {};
struct B: private A {};
struct C: B {
    void f() {
        A *p1 = (A*) this;        // Produces errors.
        ::A *p2 = (::A*) this;    // Valid
    }
};

The workaround is to use the scoping operator, as shown in the preceding code, to refer to the private base class’s type.
