Disallow Invalid Use Of Explicit Template Arguments

No template arguments are allowed as this is an overload, not a specialization. For example, adding the <int> in the following produces errors:

template<typename T>
void f(T) {}
void f<int>() {}

A workaround that compiles involves removing only the <int> as in the following:

void f() {}
