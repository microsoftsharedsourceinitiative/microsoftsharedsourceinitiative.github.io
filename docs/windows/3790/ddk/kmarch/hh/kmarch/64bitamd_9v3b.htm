Extra Commas In Macro Not Accepted

The extra commas in the following code are invalid:

#define FOO(x,y)
FOO(2,,,,,,3,,,,,,)    // Produces errors.

The workaround is to remove the extra commas:

#define FOO(x,y)
FOO(2,3)
