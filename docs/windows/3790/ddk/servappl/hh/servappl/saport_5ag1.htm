SaPortLockPagesForSystem

The SaPortLockPagesForSystem function obtains a virtual address for a given buffer that is locked down and that the miniport driver can use at all times.

PVOID
  SaPortLockPagesForSystem(
    IN PVOID MiniPortDeviceExtension,
    IN PVOID UserBuffer,
    IN ULONG UserBufferLength,
    IN OUT PMDL *Mdl
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
UserBuffer
Pointer to a buffer for which SaPortLockPagesForSystem obtains a virtual address.
UserBufferLength
Specifies the size, in bytes, of the buffer at UserBuffer.
Mdl
Pointer to a buffer that receives a pointer to a memory descriptor list (MDL) or a NULL pointer if the MDL was not allocated.

Return Value

Returns the base system-space virtual address that maps the physical pages described by the MDL in the pointer returned at Mdl. If the pages are not already mapped to system space and the attempt to map them fails, NULL is returned.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortLockPagesForSystem must be running at IRQL <= DISPATCH_LEVEL.
