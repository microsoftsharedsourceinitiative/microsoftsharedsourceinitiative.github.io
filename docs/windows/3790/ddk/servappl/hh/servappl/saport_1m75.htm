SaPortRequestDpc

The SaPortRequestDpc function queues the miniport driver's SaIsrForDpcRoutine function from the driver's SaInterruptServiceRoutine function to complete interrupt-driven I/O processing at a lower IRQL.

VOID
  SaPortRequestDpc(
    IN PVOID MiniPortDeviceExtension,
    IN PVOID Context
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
Context
Pointer to a driver-determined context to be passed to the DPC routine.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortRequestDpc must be running at DIRQL.

See Also

SaInterruptServiceRoutine, SaIsrForDpcRoutine
