SaPortFreePool

The SaPortFreePool function releases memory that was allocated for an SA miniport driver.

VOID
  SaPortFreePool(
    IN PVOID DeviceExtension,
    IN PVOID P
    );

Parameters

DeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
P
Pointer to memory that was allocated with the SaPortAllocatePool function.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortFreePool must be executing at IRQL <= DISPATCH_LEVEL.
