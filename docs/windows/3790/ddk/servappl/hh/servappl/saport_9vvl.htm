SaPortCreateUnicodeStringCat

The SaPortCreateUnicodeStringCat function initializes a UNICODE_STRING structure by using two given strings of 16-bit characters and concatenating these strings together.

NTSTATUS
  SaPortCreateUnicodeStringCat(
    IN PVOID MiniPortDeviceExtension,
    IN OUT PUNICODE_STRING DestinationString,
    IN PWSTR SourceString1,
    IN PWSTR SourceString2
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
DestinationString
Pointer to a UNICODE_STRING structure that receives a description for the concatenation of the two strings of 16-bit characters at SourceString1 and SourceString2.
SourceString1
Pointer to the first string of 16-bit characters.
SourceString2
Pointer to the second string of 16-bit characters.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortCreateUnicodeStringCat function uses the given source strings to perform the following actions:

  • Assigns the Length and MaximumLength members of the UNICODE_STRING structure.
  • Allocates a buffer from memory pool and assigns this buffer to the Buffer member of UNICODE_STRING. If SaPortCreateUnicodeStringCat is not able to allocate a buffer to hold the strings, STATUS_INSUFFICIENT_RESOURCES is returned.
  • Initializes this buffer.

Callers of SaPortCreateUnicodeStringCat must be running at IRQL < DISPATCH_LEVEL.
