SaPortGetVirtualAddress

The SaPortGetVirtualAddress function returns a virtual memory address that can be used to access the physical resource that is specified by the given physical address range. 

PVOID
  SaPortGetVirtualAddress(
    IN PVOID MiniPortDeviceExtension,
    IN PHYSICAL_ADDRESS PhysicalAddress,
    IN ULONG Length
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of its SA device.
PhysicalAddress
Specifies the starting physical address of the I/O range to be mapped to a virtual address.
Length
Specifies a value greater than zero, indicating the number of bytes to be mapped to a virtual address.

Return Value

Returns the base virtual address that maps the base physical address for the range; otherwise, it returns NULL if there is insufficient space for mapping the range.

Headers

Declared in saport.h. Include saport.h.

Comments

The memory that is mapped is not cached by the processor.

Callers of SaPortGetVirtualAddress must be running at IRQL = PASSIVE_LEVEL.
