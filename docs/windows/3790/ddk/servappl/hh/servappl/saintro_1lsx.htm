Local Display

A local display indicates the status of a server appliance to a user. In particular, the local display can provide information to the user during times when the server appliance is inaccessible from a client computer, specifically during start up, shut down, or when a network failure exists that prevents the client computer from communicating with the server appliance. On the other hand, if the server appliance is running and the network connection is operating correctly, then the user can use a Web browser on a client computer to obtain more detailed status and error-event information.

The local display can be implemented using light emitting diode (LED) indicators, a small bit-mapped liquid crystal display (LCD), or a combination of LED indicators and an LCD.

If the local display is implemented using only LED indicators, the following indicators are required:

  • Three status indicators: Starting, Ready, and Shutting Down
  • One error-event indicator: Networking Error

Because the server appliance presents message codes in a DWORD to control the LED indicators in a local display, a local display can be implemented using up to 32 indicators.

An LCD can provide more detail to the user by displaying status and error events in the form of icons and text. In addition, text displayed on an LCD can be localized to any language. If an LCD is used for the local display, the LCD panel must have a monochrome display area that is 64 pixels high and 128 pixels wide.
