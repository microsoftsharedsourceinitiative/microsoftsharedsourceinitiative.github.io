SaPortCreateUnicodeString

The SaPortCreateUnicodeString function initializes a UNICODE_STRING structure using a given string of 16-bit characters.

NTSTATUS
  SaPortCreateUnicodeString(
    IN PVOID MiniPortDeviceExtension,
    IN OUT PUNICODE_STRING DestinationString,
    IN PWSTR SourceString
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
DestinationString
Caller-supplied pointer to a UNICODE_STRING structure that receives a description for the string of 16-bit characters at SourceString.
SourceString
Pointer to a string of 16-bit characters.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortCreateUnicodeString function uses the given source string to perform the following actions:

  • Assigns the Length and MaximumLength members of the UNICODE_STRING structure.
  • Allocates a buffer from memory pool and assigns this buffer to the Buffer member of UNICODE_STRING. If SaPortCreateUnicodeString is not able to allocate a buffer to hold the string, STATUS_INSUFFICIENT_RESOURCES is returned.
  • Initializes this buffer.

Callers of SaPortCreateUnicodeString must be running at IRQL < DISPATCH_LEVEL.
