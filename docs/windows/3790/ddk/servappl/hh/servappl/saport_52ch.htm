SaPortWriteBinaryRegistryValue

The SaPortWriteBinaryRegistryValue function writes data into the binary-type registry value specified by the registry value's name.

NTSTATUS
  SaPortWriteBinaryRegistryValue(
    IN PVOID MiniPortDeviceExtension,
    IN PWSTR ValueName,
    IN PVOID RegistryValue,
    IN ULONG RegistryValueLength
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
ValueName
Pointer to a buffer that contains a null-terminated string of 16-bit Unicode characters that contains the name of the registry value to which information is written.
RegistryValue
Pointer to a buffer that contains the binary data that is written to the specified value.
RegistryValueLength
Specifies the size, in bytes, of the buffer at RegistryValue.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortWriteBinaryRegistryValue function writes binary (REG_BINARY) data in any form.

Callers of SaPortWriteBinaryRegistryValue must be running at IRQL = PASSIVE_LEVEL.
