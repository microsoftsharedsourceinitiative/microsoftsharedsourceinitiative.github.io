IOCTL_SADISPLAY_LOCK

Operation

The IOCTL_SADISPLAY_LOCK locks down the display surface for writing. The SA port driver handles this request.

Input

None

Output

None

I/O Status Block

The returned NTSTATUS value is set to STATUS_SUCCESS for a successful I/O operation; otherwise, the value is set to one of the error status values defined in ntstatus.h.

Headers

Defined in saio.h. Include saio.h.

Comments

Locking down a surface memory region identifies that region as "dirty". That is, when the IOCTL_SADISPLAY_UNLOCK code is submitted to unlock the region, the miniport driver updates any changes that were made to the region since the region was locked down with the previous IOCTL_SADISPLAY_LOCK request.
