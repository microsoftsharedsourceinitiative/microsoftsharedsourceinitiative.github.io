SaCancelRoutine

The SaCancelRoutine function is a required function if the miniport driver must cancel an I/O operation.

VOID
  SaCancelRoutine(
    IN PVOID DeviceExtension,
    IN PIRP Irp,
    IN BOOLEAN CurrentIo
    );

Parameters

DeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the device. The SaHwInitialize function sets up this area.
Irp
Specifies the IRP that describes the I/O operation being canceled.
CurrentIo
Specifies a Boolean value that is TRUE to indicate that the I/O operation to be canceled is the currently running I/O operation, and FALSE otherwise.

Headers

The prototype for SaCancelRoutine is declared in saport.h as (*PSA_CANCEL). Include saport.h.

Comments

The SaCancelRoutine function runs at IRQL <= DISPATCH_LEVEL.
