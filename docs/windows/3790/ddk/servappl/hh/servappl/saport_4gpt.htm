SaPortReadNumericRegistryValue

The SaPortReadNumericRegistryValue function reads the data contained in a 4-byte numerical registry value given the registry value's name.

NTSTATUS
  SaPortReadNumericRegistryValue(
    IN PVOID MiniPortDeviceExtension,
    IN PWSTR ValueName,
    OUT PULONG RegistryValue
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
ValueName
Pointer to a buffer that contains a null-terminated string of 16-bit Unicode characters that contains the name of the registry value from which information is read.
RegistryValue
Pointer to a variable that receives the 4-byte numerical data that is contained in the specified value.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.
Returns STATUS_OBJECT_TYPE_MISMATCH if the type of the value read is not REG_DWORD.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortReadNumericRegistryValue function reads a numeric (REG_DWORD) data value.

Callers of SaPortReadNumericRegistryValue must be running at IRQL = PASSIVE_LEVEL.
