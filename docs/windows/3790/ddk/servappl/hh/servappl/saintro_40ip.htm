Initializing the NVRAM Device

The SA port driver calls the NVRAM miniport driver's SaHwInitialize function to initialize the hardware resources that are specified in the miniport driver's device extension structure. The miniport driver can perform the following initialization tasks:

  • Assign the device extension's pointer to a DEVICE_OBJECT to the NVRAM device's object pointer that is passed in the SaHwInitialize call.
  • Initialize the device extension's synchronization object.
  • Set the size of the device extension's internal-buffer-size variable. For example, this variable can be set from the minimum size (SA_NVRAM_MINIMUM_SIZE) to the maximum size (SA_NVRAM_MAXIMUM_SIZE).
  • Call the SaPortAllocatePool function to allocate nonpageable system memory for the device extension's internal buffer. This buffer holds the system status that is transferred between the requesting application and the NVRAM device.
  • Call the RtlZeroMemory function to fill the internal buffer's memory with zeros.
