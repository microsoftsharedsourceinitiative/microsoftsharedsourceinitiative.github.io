Local Display Miniport Driver

The miniport driver for a local display device provides a standard interface between user-mode components in the operating system and the device. The driver primarily writes display information received from higher levels of the software to the local display device; the driver does not determine what information to display or when to display it.

The local display miniport driver provides a callback interface to the system-supplied server availability (SA) port driver and uses the port driver's services. The miniport driver controls low-level communication with the local display device. For more information on the standard architecture for all SA drivers, see Server Availability Driver Architecture.

When Plug and Play (PnP) detects a local display device, PnP loads the local display miniport driver. PnP also loads the SA port driver if it is not already loaded.
