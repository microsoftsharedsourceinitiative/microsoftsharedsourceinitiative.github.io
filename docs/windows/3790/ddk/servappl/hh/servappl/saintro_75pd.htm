Callback Functions Supplied by the Keypad Miniport Driver

The following table lists the callback functions that a keypad miniport driver must supply.

Function Description
DriverEntry Initializes the keypad miniport driver and registers it with the SA port driver.
SaHwInitialize Sets up the keypad device for I/O operations.
SaDeviceIoctl Uses IOCTL_SA_GET_VERSION to retrieve the version of the SA interface that the miniport driver supports.
SaRead Transfers key-press data from the keypad device to an application.
SaCancelRoutine Cancels an I/O operation.
SaInterruptServiceRoutine Quickly services an interrupt from the keypad device.
SaIsrForDpcRoutine Performs the deferred processing of an outstanding interrupt operation.
