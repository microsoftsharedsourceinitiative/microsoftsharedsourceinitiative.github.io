Keypad

A keypad allows a user to provide local input to the server appliance. This input can be as simple as scrolling through the alert messages that appear on the local display or as complex as entering network settings such as an Internet protocol (IP) address, subnet mask, or host name.

The keypad must have six keys as shown in the following diagram.

Server appliance keypad

These keys perform the following functions:

Cancel key
Cancels whatever has been done in the current display image and returns to the previous display image.
Enter key
Confirms the action or submits the data entry.
Left and right arrow keys
Steps through the data entry fields.
Up and down arrow keys
Steps through menu items and changes values in data entry fields.

The application layer can interpret each key-press input. The operating system receives the key-press input first, maps the input to Windows key messages, and then sends the input to the application layer.
