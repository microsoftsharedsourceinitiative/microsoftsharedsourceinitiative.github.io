IOCTL_NVRAM_READ_BOOT_COUNTER

Operation

The IOCTL_NVRAM_READ_BOOT_COUNTER request reads a value from a boot counter of a nonvolatile random access memory (NVRAM) device. The SA port driver handles this request by calling the NVRAM miniport driver's SaRead function.

Input

None

Output

The SA port driver specifies a pointer to the SA_NVRAM_BOOT_COUNTER structure in the DataBuffer parameter of the miniport driver's SaRead function. The Number member of this structure specifies the number of the boot counter from which to read. The NVRAM miniport driver indicates the count value that was read from the boot counter in the Value member and the values for the Advanced RISC Computing (ARC) path in the DeviceId member. For more information, see NVRAM.

I/O Status Block

The returned NTSTATUS value is set to STATUS_SUCCESS for a successful I/O operation; otherwise, the value is set to one of the error status values defined in ntstatus.h.

Headers

Defined in saio.h. Include saio.h.

Comments

Applications can use IOCTL_NVRAM_READ_BOOT_COUNTER to determine how many fail over instances have occurred.
