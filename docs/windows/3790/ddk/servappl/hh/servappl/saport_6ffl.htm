SaPortReleaseLockedPagesForSystem

The SaPortReleaseLockedPagesForSystem function unlocks physical pages described by a given MDL and then releases resources that were allocated for the MDL.

VOID
  SaPortReleaseLockedPagesForSystem(
    IN PVOID MiniPortDeviceExtension,
    IN PMDL Mdl
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
Mdl
Pointer to a memory descriptor list (MDL) to unlock and release.

Headers

Declared in saport.h. Include saport.h.

Comments

The miniport driver must have previously called the SaPortLockPagesForSystem function to allocate and lock a buffer that is described by the specified MDL. The miniport driver must unlock and release the MDL when operations on the buffer are done. As the physical memory pages are unlocked, the MDL is updated.

Callers of SaPortReleaseLockedPagesForSystem must be running at IRQL <= DISPATCH_LEVEL.
