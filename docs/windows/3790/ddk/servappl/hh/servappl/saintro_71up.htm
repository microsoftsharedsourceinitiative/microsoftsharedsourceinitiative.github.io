Initializing the Keypad Device

The server availability (SA) port driver calls the keypad miniport driver's SaHwInitialize function to initialize the hardware resources that are specified in the miniport driver's device extension structure and perform other initialization tasks. If the device extension contains, for example, a spin lock (KSPIN_LOCK object), SaHwInitialize calls KeInitializeSpinLock to initialize the spin lock.
