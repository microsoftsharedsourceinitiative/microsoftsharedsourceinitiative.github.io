SA_NVRAM_CAPS

The SA_NVRAM_CAPS structure contains the capabilities of a nonvolatile random access memory (NVRAM) device. 

typedef struct _SA_NVRAM_CAPS {
  ULONG  SizeOfStruct;
  USHORT  NvramSize;
} SA_NVRAM_CAPS, *PSA_NVRAM_CAPS;

Members

SizeOfStruct
Specifies the size in bytes of this structure.
NvramSize
Specifies the size, in bytes, of NVRAM storage. The minimum storage is 32 bytes and the maximum storage is 128 bytes. The miniport driver uses SA_NVRAM_MINIMUM_SIZE and SA_NVRAM_MAXIMUM_SIZE to specify minimum and maximum storage respectively.

Headers

Defined in saio.h. Include saio.h.

Comments

The SA port driver passes the IOCTL_SA_GET_CAPABILITIES code's corresponding function code in the FunctionCode parameter of a NVRAM miniport driver's SaDeviceIoctl function to retrieve the capabilities of the NVRAM device. The miniport driver then returns a pointer to a SA_NVRAM_CAPS structure in the OutputBuffer parameter.
