Viewing Warning Details

The Annotated Source window shows detailed information about an individual warning message, including which function contains the error, which line it occurs on, and where the path is. You can scroll through the entire warning detail or use the buttons on the left to go to a specific warning.

To view details about a specific warning

  • Double-click any warning message in the Message List window. The Annotated Source window appears.

To move through the warning details

  • Use the buttons at the top of the Annotated Source window to move to the previous or next warning message.

-or-

  • Close the Annotated Source window and return to the Message List window.
Note   If you are examining a defect log file that was generated by another person, you will receive a file not found error when trying to view the annotated source for the warning. To fix this issue, get the location of the original defect log file and enter the path information in your browser.
