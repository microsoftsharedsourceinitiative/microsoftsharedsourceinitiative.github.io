322 - Empty _except block.

This message indicates that there is no code in the _except block. Some exceptions may go unhandled.

Example

Defective Source

void mail ()

{

   __try

   {

      someFunction();

   }

   __except(1)

   {

   }

}

Corrected Source

void main()
{
   __try
   {
      someFunction();
   }
   __except(1)
   {
      handleException();
   }
}
