311 - Consider using <function> instead of <function>.

This message suggests a preferred function call that that may be semantically equivalent to the existing function call.

Example

Defective Source

char buff[MAX_PATH];

OemToChar(buff, input);  // if strlen(input) > MAX_PATH,
                         // leads to buffer overrun.

Corrected Source

char buff[MAX_PATH];

OemToCharBuff(buff, input, MAX_PATH);
