306 - Incorrect call to <function>. Consider using an alternate function which accepts a va_list as an argument.

This warning indicates an incorrect function call. The printf family includes several functions that take a variable list of arguments. However, these functions cannot be called with a va_list argument. There is a corresponding vprintf family of functions that can be used for such calls.

Example

Defective Source

    va_list v;
    va_start(v, pformat);
    printf(pformat, v);
    va_end(v);

Corrected Source

    va_list v;
    va_start(v, pformat);
    vprintf(pformat, v);
    va_end(v);
