Build Command Switches Reference

PREfast switches used with your build command line must be specified before the build command. Switches are case insensitive and can be preceded by either a slash (/) or a hyphen (-).

For example:

prefast /LOG=logfile nmake
prefast -HELP

The following PREfast switches are available for use with your build command.

Switch Description
? Opens a quick syntax help file.
HELP Opens a quick syntax help file.
FILTERPRESET=filtername Specifies using a filter preset.
LOG=logfile Specifies a location for the defect log file. If nothing is specified, a default location is used.
STACKHOGTHRESHOLD=nnnn Enables you to specify the stack usage threshold above which the Precedence defect module will flag a function as a stack hog.

For a complete list of commands and available switches, see Command Reference.
