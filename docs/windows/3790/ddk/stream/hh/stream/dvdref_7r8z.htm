KSPROPERTY_VPCONFIG_DECIMATIONCAPABILITY

The KSPROPERTY_VPCONFIG_DECIMATION_CAPABILITY property is used to indicate that the size of the video image can be reduced by the hardware.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY Boolean

The property value (operation data) is of type Boolean. TRUE specifies that the video image can be reduced in size. FALSE specifies that the image can still be resized, but would also be clipped to the rectangle instead of scaled.

Return Value

KSPROPERTY_VPCONFIG_DECIMATION_CAPABILITY property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.

When this property is used by KSPROPSETID_VPVBICONFIG, all property requests must return STATUS_NOT_IMPLEMENTED.
