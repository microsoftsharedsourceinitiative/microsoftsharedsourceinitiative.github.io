KSNODETYPE_SRC

The KSNODETYPE_SRC node represents a sample-rate converter, which produces an output stream that is a digitally resampled version of the input stream. The SRC node has one input stream and one output stream, each with the same number of channels.

A KSNODETYPE_SRC node can support the following optional properties:

KSPROPERTY_AUDIO_SAMPLING_RATE

KSPROPERTY_AUDIO_DYNAMIC_SAMPLING_RATE

KSPROPERTY_AUDIO_QUALITY
