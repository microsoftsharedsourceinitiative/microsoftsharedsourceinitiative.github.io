KSPROPERTY_MEDIASEEKING_FORMATS

The KSPROPERTY_MEDIASEEKING_FORMATS property retrieves the media time formats supported by a filter. This information is returned as a multiple item property.

A client can access this property using the following sample code:

DeviceIoControl(
    FilterHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &SeekingFormats,
    SeekingFormatLength,
    &BytesReturned,
    &Overlapped);

Comments

KSPROPERTY_MEDIASEEKING_FORMATS is a read-only property.
