KsPinAddEvent

The KsPinAddEvent function adds a specified event to Pin’s event list.

VOID __inline
  KsPinAddEvent(
    IN PKSPIN  Pin,
    IN PKSEVENT_ENTRY  EventEntry
    );

Parameters

Pin
Pointer to the KSPIN to which to add a specified event.
EventEntry
Pointer to a KSEVENT_ENTRY structure describing the event to add to Pin.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This function is an inline function call to KsAddEvent.

KsPinAddEvent must be called at IRQL = PASSIVE_LEVEL.
