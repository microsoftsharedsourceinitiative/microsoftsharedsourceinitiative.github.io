KsGateInitializeAnd

The KsGateInitializeAnd function initializes a KSGATE structure as an AND gate and attaches it to the OR gate specified by NextOrGate.

VOID __inline
  KsGateInitializeAnd(
    IN PKSGATE  AndGate,
    IN PKSGTE  NextOrGate OPTIONAL
    );

Parameters

AndGate
Pointer to the KSGATE structure to initialize as a new AND gate.
NextOrGate
Pointer to an existing KSGATE structure to which AndGate attaches. This parameter should be an OR gate and is optional.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

The next gate (following AndGate) must be an OR gate, whether it is specified in this call or later. For more information, see Flow Control Gates in AVStream.

KsGateInitializeAnd is an inline call to KsGateInitialize.

KsGateInitializeAnd may be called at any IRQL.
