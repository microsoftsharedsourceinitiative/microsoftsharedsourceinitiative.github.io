KSEVENT_CLOCK_INTERVAL_MARK

Clients enable the KSEVENT_CLOCK_INTERVAL_MARK event to be notified when an initial time value is reached, and then at fixed time increments after that.

Event Data

KSEVENT_TIME_INTERVAL, a structure of the form:

typedef struct {
    KSEVENTDATA EventData;
    LONGLONG TimeBase;
    LONGLONG Interval;
} KSEVENT_TIME_INTERVAL;

TimeBase specifies the initial time that triggers this event; an Interval specifies the interval between subsequent occurrences of the event.
