KsDeleteFilterFactory

KsDeleteFilterFactory deletes a given filter factory.

NTSTATUS
  KsDeleteFilterFactory(
    IN PKSFILTERFACTORY  FilterFactory
    );

Parameters

FilterFactory
Pointer to a KSFILTERFACTORY structure that represents the filter factory to be deleted.

Return Value

STATUS_SUCCESS is returned if the filter factory was able to be deleted. Otherwise, an appropriate error code is returned.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

KsDeleteFilterFactory must be called at IRQL = PASSIVE_LEVEL.
