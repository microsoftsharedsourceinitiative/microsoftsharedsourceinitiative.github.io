KSPROPSETID_Quality

Clients use properties in the KSPROPSETID_Quality property set to send and receive QM-related information to a driver.

The KSPROPSETID_Quality property set is enumerated by KSPROPERTY_QUALITY and includes:

KSPROPERTY_QUALITY_ERROR

KSPROPERTY_QUALITY_REPORT
