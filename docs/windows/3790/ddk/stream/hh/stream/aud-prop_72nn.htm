KSPROPSETID_Synth

The KSPROPSETID_Synth property set contains properties that are global to the configuration of a synth node (KSNODETYPE_SYNTHESIZER).

The property items in this set are specified by KSPROPERTY_SYNTH enumeration values.

The KSPROPSETID_Synth property set includes the following properties:

KSPROPERTY_SYNTH_CAPS

KSPROPERTY_SYNTH_CHANNELGROUPS

KSPROPERTY_SYNTH_LATENCYCLOCK

KSPROPERTY_SYNTH_MASTERCLOCK

KSPROPERTY_SYNTH_PORTPARAMETERS

KSPROPERTY_SYNTH_RUNNINGSTATS

KSPROPERTY_SYNTH_VOICEPRIORITY

KSPROPERTY_SYNTH_VOLUME

KSPROPERTY_SYNTH_VOLUMEBOOST
