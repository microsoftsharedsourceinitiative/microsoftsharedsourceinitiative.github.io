KSNODETYPE_SUPERMIX

The KSNODETYPE_SUPERMIX node represents a supermixer. A supermixer has one input stream with m channels and one output stream with n channels. For each output channel, the supermixer specifies a mix level for each of the input channels that adds to the mix in the output channel. The m-channel input stream is upmixed or down-mixed to n channels.

A KSNODETYPE_SUPERMIX node should support the following required properties:

KSPROPERTY_AUDIO_MIX_LEVEL_TABLE

KSPROPERTY_AUDIO_MIX_LEVEL_CAPS
