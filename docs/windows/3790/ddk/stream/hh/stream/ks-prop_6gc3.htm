KSPROPERTY_QUALITY_REPORT

The KSPROPERTY_QUALITY_REPORT property is an optional property that should be implemented if the pin supports quality management.

Property Data

A KSQUALITY structure, specifying the proportion of frames currently being used and the delta from optimal frame receipt time.

Comments

KSPROPERTY_QUALITY_REPORT may be read-only, write-only, or read/write.

The class driver does not handle this property; the stream minidriver must provide handling on its own.
