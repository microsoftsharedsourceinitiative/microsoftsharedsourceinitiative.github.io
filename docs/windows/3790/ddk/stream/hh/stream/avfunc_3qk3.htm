KsFilterGetParentFilterFactory

The KsFilterGetParentFilterFactory function returns the parent filter factory of the given filter.

PKSFILTERFACTORY __inline
  KsFilterGetParentFilterFactory(
    IN PKSFILTER  Filter
    );

Parameters

Filter
Pointer to the KSFILTER structure for which to return the parent filter factory.

Return Value

KsFilterGetParentFilterFactory returns a pointer to the parent KSFILTERFACTORY of Filter. This is the AVStream filter factory from which Filter was created.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This call is an inline function call to KsGetParent. Note that the object hierarchy is only guaranteed stable while the appropriate mutex is held, in this case the device mutex. For more information on mutexes, see Mutexes in AVStream.

KsFilterGetParentFilterFactory must be called at IRQL = PASSIVE_LEVEL.
