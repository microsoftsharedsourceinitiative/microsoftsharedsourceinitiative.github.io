KS_MPEG1VIDEOINFO

The KS_MPEG1VIDEOINFO structure describes an MPEG video stream without bob or weave settings.

typedef struct tagKS_MPEG1VIDEOINFO {
    KS_VIDEOINFOHEADER  hdr;
    DWORD              dwStartTimeCode;
    DWORD              cbSequenceHeader;
    BYTE               bSequenceHeader[1];
} KS_MPEG1VIDEOINFO;

Members

hdr
Specifies the details of the video stream.
dwStartTimeCode
Indicates a 25-bit “group-of-pictures” time code at the start of data.
cbSequenceHeader
Indicates the length in bytes of bSequenceHeader.
bSequenceHeader
Indicates the sequence header including quantization matrices, if any.

Headers

Declared in ksmedia.h. Include ksmedia.h.
