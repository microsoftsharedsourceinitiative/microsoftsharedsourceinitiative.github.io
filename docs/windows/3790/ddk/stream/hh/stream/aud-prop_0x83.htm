KSPROPERTY_AC3_LANGUAGE_CODE

The KSPROPERTY_AC3_LANGUAGE_CODE property specifies the language code of the AC3-encoded stream.

Get Set Target Property Descriptor Type Property Value Type
Yes No Pin KSPROPERTY KSAC3_LANGUAGE_CODE

The property value (operation data) is a KSAC3_LANGUAGE_CODE structure that specifies the language code.

Return Value

A KSPROPERTY_AC3_LANGUAGE_CODE property request returns STATUS_SUCCESS to indicate that it has completed successfully. Otherwise, the request returns an appropriate error status code.

Headers

Declared in ksmedia.h. Include ksmedia.h.
