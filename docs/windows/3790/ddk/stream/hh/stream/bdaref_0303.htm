KSPROPSETID_BdaTableSection

KSPROPSETID_BdaTableSection is the BDA table section property set. It is used to provide a table section to a node to use when delivering data on the node's output.

The following property is available:

KSPROPERTY_BDA_TABLE_SECTION
Informs a node about the table section to use when delivering data on the node's output.
