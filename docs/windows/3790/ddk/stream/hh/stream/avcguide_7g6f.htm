Peer Device Stack

The peer device stack (as opposed to the Virtual Device Stack) consists of the set of WDM drivers loaded to represent devices visible on the 1394 bus, and controlled from the computer. All external device control is initiated through this stack. The peer device stack also can contain Windows Driver Model (WDM) filter drivers that provide vendor or device specific target AV/C functionality. For more information on AV/C target functionality, see Peer Stack as Command Target.
