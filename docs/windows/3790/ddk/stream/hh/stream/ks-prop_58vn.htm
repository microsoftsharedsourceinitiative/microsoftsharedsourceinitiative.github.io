KSPROPERTY_MEDIASEEKING_CONVERTTIMEFORMAT

The KSPROPERTY_MEDIASEEKING_CONVERTTIMEFORMAT property converts the specified media time from one supported format to another.

A client can access this property using the following sample code:

DeviceIoControl(
    FilterHandle,
    IOCTL_KS_PROPERTY,
    &PropertyTimeFormat,
    sizeof(KSP_TIMEFORMAT),
    &Time,
    sizeof(LONGLONG),
    &BytesReturned,
    &Overlapped);

Comments

KSPROPERTY_MEDIASEEKING_CONVERTTIMEFORMAT is read-only.
