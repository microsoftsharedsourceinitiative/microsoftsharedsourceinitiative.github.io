IDmaChannelSlave::ReadCounter

The ReadCounter method reads the counter of a slave DMA object.

ULONG
  ReadCounter(
    VOID
    );

Parameters

None

Return Value

ReadCounter returns the DMA object’s counter value, which specifies the current byte count. When the object is started, this value is set to the map size (the IDmaChannelSlave::Start method's MapSize parameter). The value decrements until it reaches zero, at which time it is reset to the map size. This process repeats until the DMA object is stopped (by calling the IDmaChannelSlave::Stop method).

Headers

Declared in portcls.h. Include portcls.h.

Comments

Callers of ReadCounter must run at IRQL <= DISPATCH_LEVEL.
