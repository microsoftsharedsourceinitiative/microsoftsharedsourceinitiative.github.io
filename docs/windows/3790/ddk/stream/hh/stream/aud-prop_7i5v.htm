KSDS3D_ITD_PARAMS_MSG

The KSDS3D_ITD_PARAMS_MSG structure specifies the parameters used by the interaural time delay (ITD) algorithm in a 3D node (KSNODETYPE_3D_EFFECTS).

typedef struct
{
  ULONG  Enabled;
  KSDS3D_ITD_PARAMS  LeftParams;
  KSDS3D_ITD_PARAMS  RightParams;
  ULONG  Reserved;
} KSDS3D_ITD_PARAMS_MSG, *PKSDS3D_ITD_PARAMS_MSG;

Members

Enabled
Specifies whether to enable the ITD algorithm. A nonzero value enables the algorithm. Zero disables it. See the following Comments section.
LeftParams
Specifies the ITD parameters for the left channel (channel 0). This parameter is a structure of type KSDS3D_ITD_PARAMS.
RightParams
Specifies the ITD parameters for the right channel (channel 1). This parameter is a structure of type KSDS3D_ITD_PARAMS.
Reserved
Reserved. Set to zero.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

This structure is used by the KSPROPERTY_ITD3D_PARAMS property request.

The Enabled parameter should track the DirectSound buffer's 3D mode: Disable ITD processing when the 3D mode is disabled, and enable it when the 3D mode is enabled. For more information, see the description of the DirectSound3DBuffer::SetMode method in the Platform SDK documentation.
