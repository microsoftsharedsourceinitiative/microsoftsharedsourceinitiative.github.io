Minidriver Routines

This section contains reference pages for minidriver-supplied routines that run under Kernel Streaming. The minidriver exports these routines by providing pointers in the KSMETHOD_ITEM structure.

The names of these routines are vendor-defined;each is specified by a function pointer.
