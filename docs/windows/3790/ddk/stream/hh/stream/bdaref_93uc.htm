Polarization

The Polarization enumeration contains values that specify the polarity of the broadcast signal. 

typedef V1_ENUM enum Polarisation {
    BDA_POLARISATION_NOT_SET = -1,
    BDA_POLARISATION_NOT_DEFINED = 0,
    BDA_POLARISATION_LINEAR_H = 1, 
    BDA_POLARISATION_LINEAR_V, 
    BDA_POLARISATION_CIRCULAR_L, 
    BDA_POLARISATION_CIRCULAR_R, 
    BDA_POLARISATION_MAX,
} Polarisation;

Elements

BDA_POLARISATION_NOT_SET
Polarization on the signal is not set.
BDA_POLARISATION_NOT_DEFINED
Polarization on the signal is not defined.
BDA_POLARISATION_LINEAR_H
Polarization on the signal is linear horizontal.
BDA_POLARISATION_LINEAR_V
Polarization on the signal is linear vertical.
BDA_POLARISATION_CIRCULAR_L
Polarization on the signal is circular left.
BDA_POLARISATION_CIRCULAR_R
Polarization on the signal is circular right.
BDA_POLARISATION_MAX
Maximum value in the enumeration exceeded.

Headers

Defined in Bdatypes.h. Include Bdatypes.h.
