PROPSETID_ALLOCATOR_CONTROL

These properties are optional:

KSPROPERTY_ALLOCATOR_CONTROL_HONOR_COUNT

KSPROPERTY_ALLOCATOR_CONTROL_SURFACE_SIZE

KSPROPERTY_ALLOCATOR_CONTROL_CAPTURE_CAPS

KSPROPERTY_ALLOCATOR_CONTROL_CAPTURE_INTERLEAVE

DirectShow Interface

Properties in the PROPSETID_ALLOCATOR_CONTROL property set are used by the Overlay Mixer to control Microsoft® DirectDraw® surface allocation and dimensions. This pin property set should be implemented by devices that want to control the number of DirectDraw surfaces allocated and indicate to the Overlay Mixer that video scaling is not possible at the video decoder.
