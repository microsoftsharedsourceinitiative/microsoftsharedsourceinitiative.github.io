Streaming Devices (Video and Audio): Windows DDK

PROPSETID_EXT_DEVICE

The PROPSETID_EXT_DEVICE properties are used to control external devices. For example, a MiniDV camcorder.

KSPROPERTY_EXTDEVICE_ID

KSPROPERTY_EXTDEVICE_VERSION

KSPROPERTY_EXTDEVICE_POWER_STATE

KSPROPERTY_EXTDEVICE_PORT

KSPROPERTY_EXTDEVICE_CAPABILITIES

DirectShow Interface

Properties in the PROPSETID_EXT_DEVICE property set are accessible in user mode through the DirectShow IAMExtDevice interface.
