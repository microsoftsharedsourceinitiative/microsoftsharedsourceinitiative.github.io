KSPROPSETID_Connection

Clients use the properties in the KSPROPSETID_Connection property set to examine or set the state of a connection on a pin. Pin instances handle these properties.

Stream minidrivers do not have to handle the KSPROPERTY_CONNECTION_STATE, KSPROPERTY_CONNECTION_PROPOSEDATAFORMAT, and KSPROPERTY_CONNECTION_DATAFORMAT properties directly. The stream class driver handles them, using stream request blocks (SRB) to query for more information where necessary.

The KSPROPSETID_Connection property set includes:

KSPROPERTY_CONNECTION_ALLOCATORFRAMING

KSPROPERTY_CONNECTION_ACQUIREORDERING

KSPROPERTY_CONNECTION_DATAFORMAT

KSPROPERTY_CONNECTION_PRIORITY

KSPROPERTY_CONNECTION_PROPOSEDATAFORMAT

KSPROPERTY_CONNECTION_STATE
