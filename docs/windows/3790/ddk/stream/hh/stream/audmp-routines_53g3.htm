IInterruptSync::GetKInterrupt

The GetKInterrupt method gets a WDM interrupt object from a port-class synchronization object.

PKINTERRUPT
  GetKInterrupt(
    void
    )

Parameters

None

Return Value

GetKInterrupt returns a pointer to a the WDM interrupt object.

Headers

Declared in portcls.h. Include portcls.h.

Comments

A PKINTERRUPT pointer is one of the two parameters that are passed to every interrupt service routine (see InterruptService). Every IInterruptSync object has an associated PKINTERRUPT pointer. It points to the associated kernel interrupt object, which is opaque.

A driver typically calls GetKInterrupt only if it needs to obtain this pointer so that it can call KeSynchronizeExecution directly.
