KSPROPERTY_AUDDECOUT_CUR_MODE

The KSPROPERTY_AUDDECOUT_CUR_MODE property is used to indicate the current audio output mode.

Get Set Target Property descriptor type Property value type
Yes Yes Pin KSPROPERTY DWORD

The property value (operation data) is of type DWORD and can be one of the following mode constants defined in the header file ksmedia.h:

KSAUDDECOUTMODE_STEREO_ANALOG

Indicates that the output is in analog stereo.

KSAUDDECOUTMODE_PCM_51

Indicates that the output is in PCM 5.1 channel digital.

KSAUDDECOUTMODE_SPDIFF

Indicates that the output is SPDIFF format AC3 digital.

Return Value

KSPROPERTY_AUDDECOUT_CUR_MODE property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.

Comments

The audio miniport get-property handler returns the current mode of the decoder, while the audio miniport set-property handler requests that the decoder switch the output audio format to the requested mode.

It is recommended that a default value for this property be initially set in the minidriver’s serialized property set in the registry.

For more information, see Audio Miniport Drivers.
