PcRegisterSubdevice

The PcRegisterSubdevice function registers a subdevice to make it available for use by clients. 

PORTCLASSAPI NTSTATUS NTAPI
  PcRegisterSubdevice(
    IN PDEVICE_OBJECT  DeviceObject,
    IN PWCHAR  Name,
    IN PUNKNOWN  Unknown
    );

Parameters

DeviceObject
Pointer to the adapter driver's device object. This is a system structure of type DEVICE_OBJECT.
Name
Pointer to a null-terminated Unicode string that specifies the name of the subdevice. This string contains a short name that distinguishes the subdevice from any other subdevices registered on the same device. Each of the device's subdevices must have a unique name.
Unknown
Pointer to the subdevice's top-edge DDI (device driver interface). This is generally the port driver's IPort interface.

Return Value

PcRegisterSubdevice returns STATUS_SUCCESS if the call was successful. Otherwise, it returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.

Comments

For more information on the role of the PcRegisterSubdevice function in registering a subdevice, see Subdevice Creation.
