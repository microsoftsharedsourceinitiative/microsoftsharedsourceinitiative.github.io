KsQueryObjectAccessMask

The KsQueryObjectAccessMask function returns the access originally granted to the first client that created a handle on the associated object. Access can not be changed by duplicating handles.

KSDDKAPI ACCESS_MASK NTAPI
  KsQueryObjectAccessMask(
    IN KSOBJECT_HEADER  Header
    );

Parameters

Header
Points to a header previously allocated by KsAllocateObjectHeader whose access-granted mask pointer is to be returned.

Return Value

The KsQueryObjectAccessMask function returns an access mask.

Headers

Declared in ks.h. Include ks.h.
