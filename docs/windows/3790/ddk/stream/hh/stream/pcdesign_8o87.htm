Advanced MIDI DLS Miniport

The DirectMusic (DMus) miniport driver model is intended for wavetable and other more advanced MIDI devices, including those that support downloadable sounds (DLS). Multiple MIDI streams are supported, and each channel is unidirectional.

The main MIDI functionality is implemented in miniport drivers via the IMiniportDMus and IMXF interfaces. IMiniportDMus handles miniport driver initialization and the creation of MIDI streams. IMXF handles the passing of MIDI messages and the state of the stream.

Additional DLS functionality for software synthesizers is supported through the ISynthSinkDMus interface. ISynthSinkDMus handles the PCM interface out of the miniport driver.

The miniport driver calls the following additional interfaces on the port driver:

IPortDMus

IAllocatorMXF

IMasterClock
