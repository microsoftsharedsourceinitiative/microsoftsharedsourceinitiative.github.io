Kernel Streaming Proxy

The kernel-streaming (KS) proxy module represents KS filters of KS minidrivers as Microsoft® DirectShow® filters by assuming the characteristics of those KS filters. The proxy then sends control down to kernel mode while reflecting events in user mode. The KS proxy module also lets applications control and retrieve information from KS objects that a KS minidriver implements. KS objects include, for example, KS filters, KS pins, and KS clocks.
