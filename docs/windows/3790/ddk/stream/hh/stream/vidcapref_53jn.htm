KSPROPERTY_EXTDEVICE_POWER_STATE

The KSPROPERTY_EXTDEVICE_POWER_STATE property is used to set or get the power state of an external device.

Get Set Target Property descriptor type Property value type
Yes Yes Device KSPROPERTY_EXTDEVICE_S KSPROPERTY_EXTDEVICE_S

The PowerState member of the KSPROPERTY_EXTDEVICE_S structure is used to specify the external device's power setting. For example, on or standby. For example, a battery-powered external device, such as a DV camcorder may be powered off. An AC-powered DVHS device may be placed in standby. If a device is in standby, it may be powered-on later.

Headers

Declared in ksmedia.h. Include ksmedia.h.
