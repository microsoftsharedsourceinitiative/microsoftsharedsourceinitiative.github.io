KSDISPATCH_TABLE

The KSDISPATCH_TABLE structure is used for dispatching IRPs to various types of objects contained under a single DRIVER_OBJECT.

typedef struct {
    PDRIVER_DISPATCH        DeviceIoControl;
    PDRIVER_DISPATCH        Read;
    PDRIVER_DISPATCH        Write;
    PDRIVER_DISPATCH        Flush;
    PDRIVER_DISPATCH        Close;
    PDRIVER_DISPATCH        QuerySecurity;
    PDRIVER_DISPATCH        SetSecurity;
    PFAST_IO_DEVICE_CONTROL FastDeviceIoControl;
    PFAST_IO_READ           FastRead;
    PFAST_IO_WRITE          FastWrite;
} KSDISPATCH_TABLE, *PKSDISPATCH_TABLE;

Members

DeviceIoControl
Specifies the Device I/O Control handler.
Read
Specifies the Read handler.
Write
Specifies the Write handler.
Flush
Specifies the Flush handler.
Close
Specifies the Close handler.
QuerySecurity
Specifies the Query Security handler.
SetSecurity
Specifies the Set Security handler.
FastDeviceIoControl
Specifies the fast Device I/O Control handler.
FastRead
Specifies the fast Read handler.
FastWrite
Specifies the fast Write handler.

Headers

Declared in ks.h. Include ks.h.

Comments

A pointer to a dispatch table is contained in the opaque object header that is the first element of data pointed to by the file object's FsContext field.
