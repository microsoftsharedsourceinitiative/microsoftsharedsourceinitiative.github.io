KSEVENT_CONNECTION_DATADISCONTINUITY

The KSEVENT_CONNECTION_DATADISCONTINUITY event allows a client to receive notification of a data discontinuity. This event occurs when a data discontinuity flag is sent through the stream interface on a pin. There are no parameters to this event.
