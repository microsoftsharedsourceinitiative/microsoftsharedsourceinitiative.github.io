KS_TRUECOLORINFO

The KS_TRUECOLORINFO structure describes color palette and bit mask information for video images that also contain a palette.

typedef struct tagKS_TRUECOLORINFO {
    BYTE         dwBitMasks[KS_iMASK_COLORS];
    KS_RGBQUAD   bmiColors[KS_iPALLETE_COLORS];
} KS_TRUECOLORINFO;

Members

dwBitMasks
Specifies an array of color masks (one per each color component; red, green and blue).
bmiColors
Specifies an array of palette color triplets (red, green and blue components).

Headers

Declared in ksmedia.h. Include ksmedia.h.
