Topology Pins

The WDMAud system driver translates topology pins into the source and destination mixer lines that the mixer API exposes to applications. When a pin is queried for its KSPROPERTY_PIN_DATAFLOW property, a sink pin returns the value KSPIN_DATAFLOW_IN and a source pin returns KSPIN_DATAFLOW_OUT. Sink pins become source mixer lines, and source pins become destination mixer lines. Each pin found in the topology becomes either a source or destination line, but might not be shown in SndVol32 because of restrictions that are described in SysTray and SndVol32.
