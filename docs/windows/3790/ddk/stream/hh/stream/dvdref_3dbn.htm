KSPROPERTY_DVDSUBPIC_COMPOSIT_ON

The KSPROPERTY_DVDSUBPIC_COMPOSIT_ON property is used to enable or disable the display of the subpicture.

Get Set Target Property descriptor type Property value type
No Yes Pin KSPROPERTY KSPROPERTY_COMPOSIT_ON

The property value (operation data) is of type KSPROPERTY_COMPOSIT_ON (a type-defined Boolean).

Return Value

KSPROPERTY_DVDSUBPIC_COMPOSIT_ON property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.

Comments

If the subpicture display is disabled the decoder must still decode the subpicture data but not display it. This facilitates instant display when a subpicture-enable command is received.

There is a force-display subpicture command in the subpicture data command stream that can override the KSPROPERTY_DVDSUBPIC_COMPOSIT_ON property.
