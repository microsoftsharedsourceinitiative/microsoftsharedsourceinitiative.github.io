KSPROPERTY_VPCONFIG_GETVIDEOFORMAT

The KSPROPERTY_VPCONFIG_GETVIDEOFORMAT property is used to retrieve an array of supported pixel formats.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY DDPIXELFORMAT

The property value (operation data) is of type DDPIXELFORMAT

Return Value

KSPROPERTY_VPCONFIG_GETVIDEOFORMAT property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
