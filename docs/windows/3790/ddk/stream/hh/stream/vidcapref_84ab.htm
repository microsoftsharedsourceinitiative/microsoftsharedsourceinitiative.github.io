KSPROPERTY_CAMERACONTROL_FOCUS

User-mode clients use the KSPROPERTY_CAMERACONTROL_FOCUS property to get or set the camera's focus setting. This property is optional.

Get Set Target Property descriptor type Property value type
Yes Yes Pin KSPROPERTY_CAMERACONTROL_S KSPROPERTY_CAMERACONTROL_S

The Value member of the KSPROPERTY_CAMERACONTROL_S structure is used to specify the focus setting. This value is expressed in millimeters.

Every video capture minidriver that supports this property must define its own range and default value for this property. The value type must be VT_I4.

Headers

Declared in ksmedia.h. Include ksmedia.h.
