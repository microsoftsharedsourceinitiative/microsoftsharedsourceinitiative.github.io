IDmaChannelSlave::Stop

The Stop method stops a slave DMA object.

NTSTATUS
  Stop(
    VOID
    );

Parameters

None

Return Value

Stop returns STATUS_SUCCESS if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.

Comments

This call flushes any data remaining in the DMA controller's internal cache at the time that the DMA transfer operation is stopped.

Callers of Stop must run at IRQL <= DISPATCH_LEVEL.
