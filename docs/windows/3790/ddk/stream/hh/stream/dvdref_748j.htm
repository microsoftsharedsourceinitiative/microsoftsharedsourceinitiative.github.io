KSPROPERTY_DVDSUBPIC_HLI

The KSPROPERTY_DVDSUBPIC_HLI property is used to specify a rectangle of a subpicture or screen whose color or contrast is to be changed.

Get Set Target Property descriptor type Property value type
No Yes Pin KSPROPERTY KSPROPERTY_SPHLI

The property value (operation data) is of type KSPROPERTY_SPHLI.

Return Value

KSPROPERTY_DVDSUBPIC_HLI property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.

Comments

The KSPROPERTY_SPHLI used by this property describes the currently selected button from the DVD highlight information.
