KSNODETYPE_SUM

The KSNODETYPE_SUM node represents a simple summer (SUM). The SUM has multiple input streams and one output stream, all with the same data format. The input streams are mixed equally to produce the output stream.

A KSNODETYPE_SUM node does not currently support any properties.
