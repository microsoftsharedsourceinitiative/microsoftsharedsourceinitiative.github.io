KSEVENT_CONNECTION_TIMEDISCONTINUITY

The KSEVENT_CONNECTION_TIMEDISCONTINUITY event allows a client to receive notification of a time discontinuity. This event occurs when a time discontinuity flag is sent through the stream interface on a pin. There are no parameters to this event.
