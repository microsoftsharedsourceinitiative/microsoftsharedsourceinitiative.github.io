USBCAMD Structures

This section describes the structures used by the USBCAMD minidriver library. Of the structures documented in this section, USBCAMD version 1.0 supports only:

USBCAMD_DEVICE_DATA

The following three structures were introduced with USBCAMD version 2.0:

USBCAMD_DEVICE_DATA2

USBCAMD_INTERFACE

USBCAMD_Pipe_Config_Descriptor
