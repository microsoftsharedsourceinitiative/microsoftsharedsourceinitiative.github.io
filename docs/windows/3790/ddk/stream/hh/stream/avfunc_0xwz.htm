KsAcquireControl

The KsAcquireControl function acquires the filter control mutex for Object. 

VOID
  KsAcquireControl(
    IN PVOID  Object
    );

Parameters

Object
Pointer to the object for which to acquire the filter control mutex. This should be a pointer to either a KSFILTER or a KSPIN, cast to PVOID.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Minidrivers typically do not call this function directly, but instead call either KsFilterAcquireControl or KsPinAcquireControl. These functions provide the necessary typecasting to PVOID automatically.

For more information, see Mutexes in AVStream.

KsAcquireControl must be called at IRQL = PASSIVE_LEVEL.
