IMusicTechnology

The IMusicTechnology interface is used to change the DirectMusic synthesizer technology that is specified in the Technology member of each of the KSDATARANGE_MUSIC structures that specify the data ranges for the DMus miniport driver's pins. The Technology member specifies the type of MIDI synthesizer device that the pin represents. The DMus miniport driver implements the IMusicTechnology interface and exposes this interface to the DMus port driver. To determine whether the miniport driver supports this interface, the port driver calls the miniport driver's IMiniportDMus::QueryInterface method with REFIID IID_IMusicTechnology. IMusicTechnology inherits from the IUnknown interface.

In Microsoft® Windows® XP and later, the IMusicTechnology interface is supported by the system-supplied miniport drivers DMusUART and UART. For more information on these drivers, see PcNewMiniport.

During a call to the Windows Multimedia midiOutGetDevCaps function (see Platform SDK documentation), the WDMAud system driver converts the GUID in the Technology member to the corresponding integer constant that is loaded into the wTechnology member of the MIDIOUTCAPS structure that is output by the function.

In addition to the methods that IMusicTechnology inherits from the IUnknown interface, IMusicTechnology supports the following method:

IMusicTechnology::SetTechnology
