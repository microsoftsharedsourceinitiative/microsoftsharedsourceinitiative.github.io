BDA_CHANGE_STATE

The BDA_CHANGE_STATE enumeration contains values that specify the state of topology changes in a filter. 

typedef enum {
    BDA_CHANGES_COMPLETE = 0,
    BDA_CHANGES_PENDING
} BDA_CHANGE_STATE, * PBDA_CHANGE_STATE;

Elements

BDA_CHANGES_COMPLETE
BDA topology changes are complete.
BDA_CHANGES_PENDING
BDA topology changes are pending.

Headers

Defined in Bdatypes.h. Include Bdatypes.h.
