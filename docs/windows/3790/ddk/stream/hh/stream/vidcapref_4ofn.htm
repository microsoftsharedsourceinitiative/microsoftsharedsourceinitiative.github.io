PROPSETID_VIDCAP_CROSSBAR

The PROPSETID_VIDCAP_CROSSBAR property set is used to get or set information pertaining to routing video and audio signals. The crossbar is modeled on a general switching matrix with N inputs and M outputs. Any of the input signals can be routed to one or more of the outputs. Actual hardware implementations probably allow only a subset of this general routing capability.

The crossbar can be used to route either digital or analog signals. A single crossbar can route both video and audio signals. Video pins optionally can indicate an audio pin that is related to the video pin.

The following properties must be implemented by video capture minidrivers:

KSPROPERTY_CROSSBAR_CAN_ROUTE

KSPROPERTY_CROSSBAR_CAPS

KSPROPERTY_CROSSBAR_PININFO

KSPROPERTY_CROSSBAR_ROUTE

DirectShow Interface

Properties in the PROPSETID_VIDCAP_CROSSBAR property set are accessed through the DirectShow IAMCrossbar interface.
