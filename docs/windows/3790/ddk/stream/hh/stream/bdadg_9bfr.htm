Installing BDA Driver Samples

To install the portion of the Windows DDK that contains the Broadcast Driver Architecture (BDA) driver samples, follow these instructions:

  1. Run the Windows DDK setup application.
  2. From the list of component groups that the setup application displays, select "WDM Samples" under "Windows DDK", then select "BDA Samples".

When setup completes, each BDA driver sample will be in a subdirectory under the main directory of your DDK installation according to the following table.

BDA driver sample DDK subdirectory
Generic capture driver sample src\wdm\bda\CaptureSample
Generic tuner driver sample src\wdm\bda\GenericTuner
