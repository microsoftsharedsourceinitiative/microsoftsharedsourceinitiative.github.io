PROPSETID_EXT_TRANSPORT

The PROPSETID_EXT_TRANSPORT properties are used to control the transport of data to and from an external device.

DirectShow Interface

Properties in the PROPSETID_EXT_DEVICE property set are accessible in user mode through the DirectShow IAMExtTransport interface. This interface defines many methods, but only a subset are implemented for external device control.
