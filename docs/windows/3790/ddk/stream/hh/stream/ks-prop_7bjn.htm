KSPROPERTY_STREAM_QUALITY

The KSPROPERTY_STREAM_QUALITY property is an optional property that should be implemented if the pin generates Quality Management complaints. If a pin does not report quality problems, it does not need to support this property.

Comments

KSPROPERTY_STREAM_QUALITY is write-only.

A client can access this property using the following sample code:

DeviceIoControl(
    DeviceHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &Manager,
    sizeof(KSQUALITY_MANAGER),
    &BytesReturned,
    &Overlapped);

The pin connection in turn notifies the Quality Manager by providing KSQUALITY structures with the given context parameter.
