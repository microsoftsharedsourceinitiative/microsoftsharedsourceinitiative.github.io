KS_DATARANGE_ANALOGVIDEO

The KS_DATARANGE_ANALOGVIDEO structure describes an analog video stream.

} KS_DATARANGE_ANALOGVIDEO, *PKS_DATARANGE_ANALOGVIDEO;
