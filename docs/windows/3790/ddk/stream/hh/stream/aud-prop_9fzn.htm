KSDS3D_BUFFER_CONE_ANGLES

A KSDS3D_BUFFER_CONE_ANGLES structure specifies the inside and outside cone angles.

typedef struct
{
  ULONG  InsideConeAngle;
  ULONG  OutsideConeAngle;
} KSDS3D_BUFFER_CONE_ANGLES, *PKSDS3D_BUFFER_CONE_ANGLES;

Members

InsideConeAngle
Specifies the angle in degrees of the inside sound projection cone.
OutsideConeAngle
Specifies the angle in degrees of the outside sound projection cone.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

This structure is used to set or get the data value for the KSPROPERTY_DIRECTSOUND3DBUFFER_CONEANGLES property. DirectSound uses this property to implement the IDirectSound3DBuffer::GetConeAngles and IDirectSound3DBuffer::SetConeAngles methods, which are described in the Platform SDK documentation.
