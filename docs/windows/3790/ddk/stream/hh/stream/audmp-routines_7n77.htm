IResourceList::NumberOfEntries

The NumberOfEntries method returns the number of resource items in the resource list.

ULONG
  NumberOfEntries(
    void
    );

Parameters

None

Return Value

NumberOfEntries returns the number of items in the resource list.

Headers

Declared in portcls.h. Include portcls.h.

Comments

Callers of NumberOfEntries must run at IRQL PASSIVE_LEVEL.
