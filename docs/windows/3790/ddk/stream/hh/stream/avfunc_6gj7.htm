KsAcquireDevice

The KsAcquireDevice function gains synchronous access for Device by acquiring the device mutex.

VOID
  KsAcquireDevice(
    IN PKSDEVICE  Device
    );

Parameters

Device
AVStream device for which synchronous control should be acquired.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

KsAcquireDevice must be called at IRQL = PASSIVE_LEVEL.

For more information, see Mutexes in AVStream.
