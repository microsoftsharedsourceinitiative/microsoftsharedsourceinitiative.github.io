KsGateTerminateAnd

The KsGateTerminateAnd function deletes an existing AND gate and removes an input from any attached OR gate.

VOID __inline
  KsGateTerminateAnd(
    IN PKSGATE  AndGate
    );

Parameters

AndGate
Pointer to a KSGATE structure that is the AND gate to delete. This gate must be at the beginning of a gate chain.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Logical gates should be deleted in a front to back manner. KsGateTerminateAnd does not patch the gate chain for gates that are deleted at the end or in the middle. For more information, see Flow Control Gates in AVStream.

KsGateTerminateAnd may be called at any IRQL.
