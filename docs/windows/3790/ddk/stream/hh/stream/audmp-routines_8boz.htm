IMiniportMidiStream::Read

The Read method reads data from an incoming MIDI stream.

NTSTATUS 
  Read(
    IN PVOID  BufferAddress,
    IN ULONG  BufferLength,
    OUT PULONG  BytesRead
    );

Parameters

BufferAddress
Specifies the buffer address of the incoming MIDI stream. The size of this buffer must be greater than or equal to BufferLength.
BufferLength
Specifies the length in bytes of the buffer pointed to by BufferAddress.
BytesRead
Output pointer to a caller-allocated variable into which the method writes a count specifying the actual number of bytes successfully read from the stream.

Return Value

Read returns STATUS_SUCCESS if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.
