KsFilterAddEvent

The KsFilterAddEvent function adds an event to Filter’s event list.

VOID
  KsFilterAddEvent(
    PKSFILTER  Filter,
    PKSEVENT_ENTRY  EventEntry
    );

Parameters

Filter
Pointer to a KSFILTER structure representing the filter to which to add a specified event.
EventEntry
Pointer to an KSEVENT_ENTRY structure describing the event to add to Filter.

Headers

Declared inline in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This function is an inline function call to KsAddEvent.

KsFilterAddEvent must be called at IRQL = PASSIVE_LEVEL.
