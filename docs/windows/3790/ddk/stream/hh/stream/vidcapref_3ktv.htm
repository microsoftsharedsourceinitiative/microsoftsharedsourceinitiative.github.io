KSPROPERTY_VIDEOCONTROL_MODE

The KSPROPERTY_VIDEOCONTROL_MODE property is used to control the mode of image production. This includes settings for horizontal and vertical image flipping, and enabling frame capture by an external trigger. This property is optional.

Get Set Target Property descriptor type Property value type
Yes Yes Pin KSPROPERTY_VIDEOCONTROL_MODE_S KSPROPERTY_VIDEOCONTROL_MODE_S

The Mode member of the KSPROPERTY_VIDEOCONTROL_MODE_S structure is used to specify the video control mode.

Declared in ksmedia.h. Include ksmedia.h.
