Broadcast Driver Architecture Support Functions

This section documents the support functions that a BDA minidriver uses. The system-supplied BDA support library (Bdasup.lib) provides these functions. In order for your BDA minidriver to use these functions, you must include the Bdasup.h header file in your code and link to Bdasup.lib when you build your driver.
