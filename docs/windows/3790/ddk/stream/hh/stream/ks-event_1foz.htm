KSEVENT_CLOCK_POSITION_MARK

A KSEVENT_CLOCK_POSITION_MARK event occurs when a certain time on a clock is reached.

Event Data

KSEVENT_TIME_MARK, a structure of the form:

typedef struct {
    KSEVENTDATA EventData;
    LONGLONG MarkTime;
} KSEVENT_TIME_MARK;

The MarkTime member specifies the time stamp that triggers the event.
