Minidriver Entry Points

The camera minidriver defines all its entry points in a USBCAMD_DEVICE_DATA2 structure. The USBCAMD_DEVICE_DATA structure is supported in USBCAMD version 2.0 only for purposes of backward compatibility. For more information on the camera minidriver callback routines, see USBCAMD Callback Functions.
