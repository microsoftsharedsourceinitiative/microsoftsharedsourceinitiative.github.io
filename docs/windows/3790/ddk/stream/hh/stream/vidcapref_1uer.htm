Streaming Devices (Video and Audio): Windows DDK

PROPSETID_TUNER

User-mode clients use the members of the PROPSETID_TUNER property set to query or control devices that support TV or radio tuning. This property set supports multistandard video tuners, as well as tuners with multiple inputs.

The following properties must be implemented by video capture minidrivers:

KSPROPERTY_TUNER_CAPS

KSPROPERTY_TUNER_FREQUENCY

KSPROPERTY_TUNER_INPUT

KSPROPERTY_TUNER_MODE

KSPROPERTY_TUNER_MODE_CAPS

KSPROPERTY_TUNER_STANDARD

KSPROPERTY_TUNER_STATUS

The following property is optional:

KSPROPERTY_TUNER_IF_MEDIUM

DirectShow Interface

Properties in the PROPSETID_TUNER property set are accessible in user-mode through the DirectShow IAMTVTuner interface.
