KSPROPERTY_MEDIASEEKING_AVAILABLE

The KSPROPERTY_MEDIASEEKING_AVAILABLE property retrieves the media time span (the time span that a client can seek within) that is currently available on a filter.

A client can access this property using the following sample code:

DeviceIoControl(
    FilterHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &Available,
    sizeof(KSPROPERTY_MEDIAAVAILABLE),
    &BytesReturned,
    &Overlapped);

Comments

KSPROPERTY_MEDIASEEKING_AVAILABLE is a read-only property.
