KSPROPERTY_MEDIASEEKING_POSITION

The KSPROPERTY_MEDIASEEKING_POSITION read-only property retrieves the media time of a filter.

A client can access this property using the following sample code:

DeviceIoControl(
    FilterHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &Position,
    sizeof(LONGLONG),
    &BytesReturned,
    &Overlapped);
