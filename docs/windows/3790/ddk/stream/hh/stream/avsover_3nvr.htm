Sample AVStream Minidrivers

The DDK includes two sample AVStream minidrivers: Avshws and Avssamp. Avshws is a pin-centric capture driver for a simulated piece of hardware. Avshws also shows how to implement DMA through AVStream. Avssamp is a filter-centric capture driver that does not perform DMA.

These samples demonstrate many of the concepts that are described in this documentation. These sample drivers can be tailored to a driver developer's needs. Readmes corresponding to these samples can be found in the DDK, located in the same directories as the samples. These readmes are not included in the DDK documentation.
