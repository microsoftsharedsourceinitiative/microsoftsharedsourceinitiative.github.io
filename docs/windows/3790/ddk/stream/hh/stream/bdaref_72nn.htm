SpectralInversion

The SpectralInversion enumeration contains values that specify how spectral inversion is set on a signal demodulator node. 

typedef V1_ENUM enum SpectralInversion {
    BDA_SPECTRAL_INVERSION_NOT_SET = -1,
    BDA_SPECTRAL_INVERSION_NOT_DEFINED = 0,
    BDA_SPECTRAL_INVERSION_AUTOMATIC = 1,
    BDA_SPECTRAL_INVERSION_NORMAL,
    BDA_SPECTRAL_INVERSION_INVERTED,
    BDA_SPECTRAL_INVERSION_MAX
} SpectralInversion;

Elements

BDA_SPECTRAL_INVERSION_NOT_SET
Spectral inversion not set.
BDA_SPECTRAL_INVERSION_NOT_DEFINED
Spectral inversion not defined.
BDA_SPECTRAL_INVERSION_AUTOMATIC
Spectral inversion is automatic.
BDA_SPECTRAL_INVERSION_NORMAL
Spectral inversion is normal.
BDA_SPECTRAL_INVERSION_INVERTED
Spectral inversion is inverted.
BDA_SPECTRAL_INVERSION_MAX
Maximum value in the enumeration exceeded.
