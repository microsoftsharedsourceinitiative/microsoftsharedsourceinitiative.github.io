WaveCyclic Miniport

This class of miniport driver provides access to cyclic-DMA-based wave audio functions. The interface addresses cyclic buffers specifically, and it ignores streaming issues. Multiple channels are supported; each channel is unidirectional.

The miniport driver should support two interfaces:

  • The miniport interface, which supports miniport driver initialization, channel enumeration, and stream creation.
  • The stream interface, which exposes most of the miniport driver’s functionality.

The miniport interface, IMiniportWaveCyclic, is derived from IMiniport, and the stream interface, IMiniportWaveCyclicStream, is derived from IUnknown.
