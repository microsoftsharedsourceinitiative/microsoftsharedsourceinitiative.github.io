KSPROPERTY_VPCONFIG_SETVIDEOFORMAT

The KSPROPERTY_VPCONFIG_SETVIDEOFORMAT property is used to set the video format. The format must match one returned by an earlier by KSPROPERTY_VPCONFIG_GETVIDEOFORMAT.

Get Set Target Property descriptor type Property value type
No Yes Pin KSPROPERTY DDPIXELFORMAT

The property value (operation data) is of type DDPIXELFORMAT.

Return Value

KSPROPERTY_VPCONFIG_SETVIDEOFORMAT property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
