TransmissionMode

The TransmissionMode enumeration contains values that specify the carrier frequency of the broadcast transmission. 

typedef V1_ENUM enum TransmissionMode {
    BDA_XMIT_MODE_NOT_SET = -1,
    BDA_XMIT_MODE_NOT_DEFINED = 0,
    BDA_XMIT_MODE_2K = 1,
    BDA_XMIT_MODE_8K,
    BDA_XMIT_MODE_MAX,
} TransmissionMode;

Elements

BDA_XMIT_MODE_NOT_SET
Transmission mode not set.
BDA_XMIT_MODE_NOT_DEFINED
Transmission mode not defined.
BDA_XMIT_MODE_2K
Transmission uses 1705 carriers, that is a 2K FFT.
BDA_XMIT_MODE_8K
Transmission uses 6817 carriers, that is an 8K FFT.
BDA_XMIT_MODE_MAX
Maximum value in the enumeration exceeded.

Defined in Bdatypes.h. Include Bdatypes.h.
