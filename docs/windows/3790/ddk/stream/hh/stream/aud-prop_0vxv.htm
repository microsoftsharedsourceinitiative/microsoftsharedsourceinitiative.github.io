KSPROPSETID_Sysaudio_Pin

The KSPROPSETID_Sysaudio_Pin property set provides access to the properties of a pin instance on a virtual audio device. The SysAudio system driver manages virtual audio devices on behalf of DirectSound and other clients.

The following property is the only member of the KSPROPSETID_Sysaudio_Pin property set:

KSPROPERTY_SYSAUDIO_ATTACH_VIRTUAL_SOURCE
