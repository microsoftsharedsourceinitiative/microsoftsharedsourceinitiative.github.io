KsFilterGetDevice

The KsFilterGetDevice function returns the AVStream device to which Filter belongs.

PKSDEVICE __inline
  KsFilterGetDevice(
    IN PKSFILTER  Filter
    );

Parameters

Filter
Pointer to the KSFILTER for which to find the owning AVStream device.

Return Value

KsFilterGetDevice returns a pointer to the AVStream KSDEVICE to which Filter belongs.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This call is an inline function call to KsGetDevice.

KsFilterGetDevice must be called at IRQL = PASSIVE_LEVEL.
