WDM Audio Architecture: Basic Concepts

This section presents the basic architectural concepts of WDM (Windows Driver Model) audio drivers. The WDM audio architecture is based on KS (kernel streaming) services and is supported in Microsoft® Windows® 98/Me and in Windows 2000 and later. For information about KS services, see Kernel Streaming.
