KS_MPEGVIDEOINFO2

The KS_MPEGVIDEOINFO2 structure describes an MPEG video stream with bob or weave settings.

typedef struct  tagKS_MPEGVIDEOINFO2 {
    KS_VIDEOINFOHEADER2  hdr;
    DWORD                dwStartTimeCode;
    DWORD                cbSequenceHeader;
    DWORD                dwProfile;
    DWORD                dwLevel;
    DWORD                dwFlags;
    DWORD                bSequenceHeader[1];
} KS_MPEGVIDEOINFO2;

Members

hdr
Specifies the details of the video stream.
dwStartTimeCode
Indicates a 25-bit “group-of-pictures” time code at the start of data (not used for DVD).
cbSequenceHeader
Indicates the length in bytes of bSequenceHeader (zero for DVD).
dwProfile
Specifies the MPEG2 profile. This member must be a value from the KS_MPEG2Profile enumeration that is defined in ksmedia.h:
KS_MPEG2Profile_Simple
KS_MPEG2Profile_Main
KS_MPEG2Profile_SNRScalable
KS_MPEG2Profile_SpatiallyScalable
KS_MPEG2Profile_High
dwLevel
Specifies the MPEG2 level. This member must be a value from the KS_MPEG2Level enumeration that is defined in ksmedia.h:
KS_MPEG2Level_Low
KS_MPEG2Level_Main
KS_MPEG2Level_High1440
KS_MPEG2Level_High
dwFlags
Flags indicating preferences. This member may be set to one or more (logically ORed) values that are defined in ksmedia.h:
KS_MPEG2_DoPanScan
If set, the MPEG-2 video decoder should crop output image based on pan-scan vectors in picture_display_extension and change the picture aspect ratio accordingly.
KS_MPEG2_DVDLine21Field1
If set, the MPEG-2 decoder must be able to produce an output pin for DVD style closed caption data found in GOP layer of field 1.
KS_MPEG2_DVDLine21Field2
If set, the MPEG-2 decoder must be able to produce an output pin for DVD style closed caption data found in GOP layer of field 2.
KS_MPEG2_SourceIsLetterboxed
If set, indicates that black bars have been encoded in the top and bottom of the video.
KS_MPEG2_FilmCameraMode
If set, indicates "film mode" used for 625/50 content. If cleared, indicates that "camera mode" was used.
KS_MPEG2_LetterboxAnalogOut
If set and this stream is sent to an analog output, it should be letterboxed. Streams sent to VGA should be letterboxed only by renderers.
KS_MPEG2_DSS_UserData
If set, the MPEG-2 decoder must process DSS style user data.
KS_MPEG2_DVB_UserData
If set, the MPEG-2 decoder must process DVB style user data.
KS_MPEG2_27MHzTimebase
If set, the PTS, DTS timestamps advance at 27MHz rather than 90KHz.
bSequenceHeader
Specifies the sequence header including quantization matrices, if any.

Headers

Declared in ksmedia.h. Include ksmedia.h.
