KSEVENTSETID_StreamAllocator

The KSEVENTSETID_StreamAllocator event set specifies two events: one event used internally in the default allocator implementation to service outstanding allocation requests made through the IRP interface, and one public event used to notify clients of free frame availability.

The KSEVENTSETID_StreamAllocator event set includes:

KSEVENT_STREAMALLOCATOR_INTERNAL_FREEFRAME

KSEVENT_STREAMALLOCATOR_FREEFRAME
