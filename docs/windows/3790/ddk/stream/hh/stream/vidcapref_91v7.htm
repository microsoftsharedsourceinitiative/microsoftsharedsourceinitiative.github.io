PROPSETID_VIDCAP_VIDEODECODER

The PROPSETID_VIDCAP_VIDEODECODER property set is used to control an analog video decoder. An analog video decoder converts baseband analog video signals into a digital format such as RGB or YUV. Support for this property set is mandatory for analog video capture devices.

The following properties must be implemented by video capture minidrivers:

KSPROPERTY_VIDEODECODER_CAPS

KSPROPERTY_VIDEODECODER_STANDARD

KSPROPERTY_VIDEODECODER_STATUS

The following properties are optional.

KSPROPERTY_VIDEODECODER_OUTPUT_ENABLE

KSPROPERTY_VIDEODECODER_VCR_TIMING

DirectShow Interface

Properties in the PROPSETID_VIDCAP_VIDEODECODER property set are accessible in user mode through the DirectShow IAMAnalogVideoDecoder interface.
