KSPROPERTY_TUNER_STANDARD

The KSPROPERTY_TUNER_STANDARD property is used to get the current analog video standard. This property is used only when the current mode is KSPROPERTY_TUNER_MODE_TV. This property must be implemented.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY_TUNER_STANDARD_S ULONG

The Standard member of the KSPROPERTY_TUNER_STANDARD_S structure is used to specify the current analog video standard.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

Analog TV signals can be broadcast in accordance with several different analog TV standards such as NTSC, PAL, or SECAM. Clients use the KSPROPERTY_TUNER_MODES_CAPS property to query the supported standards, and use KSPROPERTY_TUNER_STANDARD to get or set the current standard for a TV tuner device.
