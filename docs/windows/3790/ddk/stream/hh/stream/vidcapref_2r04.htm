TIMECODE_SAMPLE

The TIMECODE_SAMPLE structure describes a complete timecode.

typedef struct tagTIMECODE_SAMPLE
{
  LONGLONG qwTick;
  TIMECODE timecode;
  DWORD dwUser;
  DWORD dwFlags;
} TIMECODE_SAMPLE, *PTIMECODE_SAMPLE;

Members

qwTick
Specifies a reference time, in 100-nanosecond units.
timecode
Specifies the TIMECODE structure.
dwUser
Specifies packed SMPTE user-bits.
dwFlags
Specifies any optional timecode flag masks. For example, ED_DEVCAP_TIMECODE_READ, ED_DEVCAP_ATN_READ or ED_DEVCAP_RTC_READ.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

Any ED_Xxx tokens are defined in xprtdefs.h in the Microsoft DirectX SDK.
