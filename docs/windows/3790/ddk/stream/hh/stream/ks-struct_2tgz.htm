KSFASTPROPERTY_ITEM

The KSFASTPROPERTY_ITEM structure is used with items for fast I/O dispatching.

typedef struct {
  ULONG  PropertyId;
  PFNKSFASTHANDLER  GetPropertyHandler;
  PFNKSFASTHANDLER  SetPropertyHandler;
  ULONG  Reserved;
} KSFASTPROPERTY_ITEM, *PKSFASTPROPERTY_ITEM;

Members

PropertyId
Specifies the identifier of the specific property within the set.
GetPropertyHandler
Specifies the fast handler for retrieving the property. If this is NULL, the property cannot be read with a fast handler.
SetPropertyHandler
Specifies the fast handler for setting the property. If this is NULL, the property cannot be set with a fast handler.
Reserved
Reserved and set to zero.

Headers

Declared in ks.h. Include ks.h.
