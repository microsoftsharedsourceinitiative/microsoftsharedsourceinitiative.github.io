IDmaChannel::FreeBuffer

The FreeBuffer method frees the buffer that was previously allocated by AllocateBuffer.

VOID
  FreeBuffer(
    VOID
    );

Parameters

None

Return Value

None

Headers

Declared in portcls.h. Include portcls.h.

Comments

Because the buffer is automatically freed when the DMA-channel object is deleted, the FreeBuffer method is not typically used.

Callers of FreeBuffer must run at IRQL PASSIVE_LEVEL.
