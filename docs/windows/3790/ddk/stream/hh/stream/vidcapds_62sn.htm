USB or 1394 Conferencing Camera

The simplest graph consists of a capture filter that exposes a single video stream with no TV/radio tuning capabilities. This type of graph is used by USB and 1394 conferencing cameras.

Microsoft provides a minidriver library for USB cameras (USBCAMD). For information about USBCAMD, see USB Camera Minidrivers and USB Camera Minidriver Library.

USB/1394 Video Conferencing Filter Graph

Note that the single capture stream from the camera is replicated by a user-mode DirectShow filter (the Smart Tee Filter) to provide separate capture and preview streams. This replication is performed without requiring a data copy.
