PCI Scatter/Gather Miniport

The PCI scatter/gather wave input and output class of miniport drivers supports access to scatter/gather DMA-based wave audio functions. The interface addresses physical mappings specifically, and ignores streaming issues. Multiple channels are supported; each channel is unidirectional.

The miniport driver should support two interfaces:

  • The miniport interface, which performs miniport driver initialization, channel enumeration, and stream creation.
  • The stream interface, which exposes most of the miniport driver’s functionality.

The miniport interface, IMiniportWavePci, is derived from IMiniport, and the stream interface, IMiniportWavePciStream, is derived from IUnknown.
