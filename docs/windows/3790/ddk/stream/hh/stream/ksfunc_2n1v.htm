Kernel Streaming Functions

This section describes the functions and macros available to kernel-mode drivers through the KS libraries. Some of these, such as pin-related items, are described in more detail elsewhere and references are made to the detailed explanations.

  • Allocator Functions

    Functions that assist with the management of allocators.

  • Clock Functions

    Functions that assist with the management of clocks.

  • Method Set Functions

    Functions that assist with the management of method sets. These functions enable a filter or pin to present a method set to a client as well as perform some of the basic parameter validation and routing.

  • Property Set Functions

    Functions that assist with the management of property sets. These functions can enable a filter or pin to present a property set to a client as well as perform some of the basic parameter validation and routing.

  • Event Set Functions

    Functions that assist with the management of event sets.

  • Connection Functions

    Functions that assist in creating and destroying connections.

  • IRP Management Functions

    Functions that assist in general processing of IRPs.

  • Topology Management Functions

    Functions that assist in the management of a topology.

  • Minidriver-Routines

    Callbacks that the minidriver provides.

  • General Utility Functions

    Functions that are general helper functions.
