Kernel Streaming Interface Sets

This section documents kernel streaming interface sets. It contains the following topics:

KSINTERFACESETID_Media

KSINTERFACESETID_Standard

These sets correspond to GUIDs that are defined in ks.h and ksmedia.h, and can be specified in the KSPIN_INTERFACE structure.

For information on how to specify interfaces in your minidriver, see KS Interfaces.
