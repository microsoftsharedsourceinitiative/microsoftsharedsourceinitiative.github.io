KSAC3_ROOM_TYPE

The KSAC3_ROOM_TYPE structure specifies the type of audio mixing room in which an AC3-encoded stream was produced.

typedef struct
{
  BOOL  fLargeRoom;
} KSAC3_ROOM_TYPE, *PKSAC3_ROOM_TYPE;

Members

fLargeRoom
Specifies the type and calibration of the mixing room used for the final audio mixing session. If TRUE, a large room with an X curve was used. If FALSE, a small room with a flat curve was used.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

This structure characterizes the production environment of an AC3-encoded stream. The information in the structure is not typically used by the decoder, but might be used by other audio components after decoding.

This structure is used by the KSPROPERTY_AC3_ROOM_TYPE property.

For more information about room types, see the AC3 specification at the Dolby Laboratories Web site. The specification is titled Digital Audio Compression Standard (AC-3).
