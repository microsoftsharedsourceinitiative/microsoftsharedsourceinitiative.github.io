AVStrMiniPinDisconnect

An AVStream minidriver's AVStrMiniPinDisconnect routine is called when the relevant KSPIN is serving as a sink pin and this sink pin is disconnected from an AVStream source pin.

VOID
  AVStrMiniPinDisconnect(
    IN PKSPIN  Pin,
    );

Parameters

Pin
Pointer to the relevant KSPIN, serving as a sink pin.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This routine is available in Windows XP and later operating systems and DirectX 8.0 and later versions.

The minidriver specifies this routine's address in the Disconnect member of its KSPIN_DISPATCH structure.

This routine is optional.

Also see KS Pins.
