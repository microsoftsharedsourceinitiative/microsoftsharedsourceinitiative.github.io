Property Set Range and Default Values

Several properties can support range and default values. This information can be used by UI elements such as sliders and scrollbars.

The range and default value information is provided in a KSPROPERTY_VALUES structure that is part of a property definition. This structure includes a pointer to a static table consisting of one or more KSPROPERTY_MEMBERSLIST items. Within the KSPROPERTY_MEMBERSLIST structure, the driver can specify either a value or a range of values member. Ranges include the minimum, maximum, and stepping value. Set the MembersFlags member of the KSPROPERTY_MEMBERSHEADER structure to the value KSPROPERTY_MEMBER_RANGES to indicate that the KSPROPERTY_MEMBERSLIST item is a range of values. A KSPROPERTY_MEMBERSLIST item is also used to specify a default value for the property. This is done by setting the MembersFlags member of the KSPROPERTY_MEMBERSHEADER to the value KSPROPERTY_MEMBER_VALUE.
