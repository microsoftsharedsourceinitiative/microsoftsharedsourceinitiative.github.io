IMiniportMidiStream

The IMiniportMidiStream interface represents the MIDI stream that flows through a MIDI pin on a MIDI or DirectMusic filter. The filter wraps a MIDI synthesizer or capture device and is implemented by pairing a MIDI or DMus port driver with a MIDI or DMus miniport driver. The miniport driver implements the IMiniportMidiStream interface and exposes it to the port driver. The port driver creates a stream object with this interface by calling the miniport driver's IMiniportMidi::NewStream method. IMiniportMidiStream inherits from the IUnknown interface.

The IMiniportMidiStream interface provides methods for reading and writing a MIDI stream and for setting the format and state of a MIDI stream.

In addition to the methods that IMiniportMidiStream inherits from the IUnknown interface, IMiniportMidiStream supports the following methods:

IMiniportMidiStream::Read

IMiniportMidiStream::SetFormat

IMiniportMidiStream::SetState

IMiniportMidiStream::Write
