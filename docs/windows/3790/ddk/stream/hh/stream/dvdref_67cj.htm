KSPROPERTY_VPCONFIG_NUMCONNECTINFO

The KSPROPERTY_VPCONFIG_NUMCONNECTINFO property is used to obtain the maximum number of electrical connection configurations.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY DWORD

The property value (operation data) is of type DWORD.

Return Value

KSPROPERTY_VPCONFIG_NUMCONNECTINFO property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
