BdaPropertyNodeDescriptors

The BdaPropertyNodeDescriptors function retrieves a list of nodes in a template topology. 

STDMETHODIMP_(NTSTATUS) 
  BdaPropertyNodeDescriptors(
    IN PIRP  Irp,
    IN KSPROPERTY  *pKSProperty,
    OUT GUID  *pguidProperty
    );

Parameters

Irp
Points to the IRP for the request to retrieve the list of nodes.
pKSProperty
Points to a KSPROPERTY structure that describes the property and request type of the property request.
pguidProperty
Points to an array that receives the list of GUIDs for the nodes in a template topology.

Return Value

Returns STATUS_SUCCESS or an appropriate error code.

Headers

Declared in Bdasup.h. Include Bdasup.h.

Comments

A BDA minidriver calls the BdaPropertyNodeDescriptors function to retrieve the list of nodes after the minidriver receives a KSPROPERTY_BDA_NODE_DESCRIPTORS request of the KSPROPSETID_BdaTopology property set from the network provider. Most BDA minidrivers can define dispatch and filter-automation tables so that those minidrivers dispatch the BdaPropertyNodeDescriptors function directly. See Defining Automation Tables and Determining BDA Device Topology for more information.

For a list of BDA nodes that are available to create in a template topology, see BDA Node Category GUIDs.

A BDA minidriver calls BdaPropertyNodeDescriptors at IRQL = PASSIVE_LEVEL.

BdaPropertyNodeDescriptors is available on Windows® XP and later. BdaPropertyNodeDescriptors is available on the Windows 2000 platform only if DirectX 9.0 or later is installed on that platform.
