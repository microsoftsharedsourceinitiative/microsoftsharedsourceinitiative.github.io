KsStreamPointerUnlock

The KsStreamPointerUnlock function unlocks a stream pointer that has previously been locked by an acquisition function (KsGet*EdgeStreamPointer) or by KsStreamPointerLock.

VOID
  KsStreamPointerUnlock(
    IN PKSSTREAM_POINTER  StreamPointer,
    IN BOOLEAN  Eject
    );

Parameters

StreamPointer
Pointer to a KSSTREAM_POINTER structure representing the stream pointer to unlock.
Eject
Specifies whether or not to advance StreamPointer to the next data frame. A value of TRUE advances the stream pointer.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

KsStreamPointerUnlock can also advance the stream pointer to the next data frame. See the description of Eject above.

Also see Stream Pointers.

KsStreamPointerUnlock must be called at IRQL <= DISPATCH_LEVEL.
