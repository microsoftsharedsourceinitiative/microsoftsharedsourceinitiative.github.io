KSPROPERTY_TUNER_IF_MEDIUM

KSPROPERTY_TUNER_IF_MEDIUM is used to describe the medium for the intermediate frequency pin for devices that support Digital TV tuning. This property is optional.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY_TUNER_IF_MEDIUM_S KSPIN_MEDIUM

The IFMedium member of the KSPROPERTY_TUNER_IF_MEDIUM_S structure is used to specify the medium of the intermediate frequency pin.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

If KSPROPERTY_TUNER_IF_MEDIUM is supported, kstvtune.ax creates an additional pin that represents the hardware-based MPEG-2 transport stream originating at the tuner. This pin is used solely to define graph topology. Data samples flowing through the user-mode stream from this pin on kstvtune.ax consist of KS_TVTUNER_CHANGE_INFO structures.
