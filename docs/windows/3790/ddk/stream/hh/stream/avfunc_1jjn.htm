KsTerminateDevice

The KsTerminateDevice function removes an AVStream device.

VOID
  KsTerminateDevice(
    IN PDEVICE_OBJECT  DeviceObject
    );

Parameters

DeviceObject
Pointer to a DEVICE_OBJECT structure corresponding to the device for which to remove AVStream support.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Normally, AVStream calls KsTerminateDevice upon receipt of a PnP remove device IRP (IRP_MN_REMOVE_DEVICE). Most minidrivers do not call this function directly; however, it can be used to remove AVStream support for the device given in DeviceObject.

KsTerminateDevice must be called at IRQL = PASSIVE_LEVEL.
