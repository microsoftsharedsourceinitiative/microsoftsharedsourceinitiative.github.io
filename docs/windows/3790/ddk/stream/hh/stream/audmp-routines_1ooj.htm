IInterruptSync::Connect

The Connect method initializes the synchronization object.

NTSTATUS
  Connect(
    void
    )

Parameters

None

Return Value

Connect returns STATUS_SUCCESS if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.

Comment

The Connect method connects the synchronization object to the interrupt. This interrupt is specified in the ResourceList and ResourceIndex parameters that are passed to the PcNewInterruptSync call that creates the synchronization object.
