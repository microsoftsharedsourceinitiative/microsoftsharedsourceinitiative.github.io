OID_WW_CDPD_CS_SERVICE_PREFERENCE

As a query, the OID_WW_CDPD_CS_SERVICE_PREFERENCE OID requests the miniport driver to return the current state of the CDPD circuit-switched service preference value.

As a set, this OID request the miniport driver to set the current state of the CDPD circuit-switched service preference value. Note that a change in this value can affect the currently established cellular connection if any. The setting value is returned as an integer.

In either case, one of the following designates the service preference value:

Value Meaning
0 Always use packet-switched CDPD.
1 Always use CS-CDPD via AMPS.
2 Always use CS-CDPD via PSTN.
3 Use CS-CDPD via AMPS only if packet-switched CDPD is not available.
4 Use packet-switched CDPD only if CS-CDPD via AMPS is not available.
5 This service-preference value is defined by the device manufacture.
6 This service-preference value is defined by the device manufacture.
-1 Unknown. If this value is specified for a set, the miniport driver returns NDIS_STATUS_INVALID_DATA.

All other values are reserved.

Indications are irrelevant for this OID.
