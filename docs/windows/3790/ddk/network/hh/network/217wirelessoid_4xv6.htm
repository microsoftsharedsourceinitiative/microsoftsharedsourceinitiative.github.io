OID_802_11_TX_POWER_LEVEL

When set, the OID_802_11_TX_POWER_LEVEL OID requests the miniport driver to set the transmit power level of the 802.11 NIC to a specified value. The NDIS_802_11_TX_POWER_LEVEL value is specified in milliwatts (mW).

When queried, this OID requests the miniport driver to return the transmit power level of the 802.11 NIC.

The NDIS_802_11_TX_POWER_LEVEL value is defined as follows:

typedef ULONG   NDIS_802_11_TX_POWER_LEVEL;

Note  The 802.11 NIC will not exceed regulatory power limits.
