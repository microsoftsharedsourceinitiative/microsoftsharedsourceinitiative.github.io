Installing the Windows Sockets Switch

Microsoft Windows installs the Windows Sockets switch as a layered service provider, with the Windows Sockets service provider interface (SPI) as both the top and bottom interface. The switch exports the protocol characteristics of TCP/IP just like the TCP/IP service provider. The switch is the first visible TCP/IP provider, which makes the switch the default choice for applications that open sockets for the AF_INET address family.
