Setting NDIS Debugging Options in the Debugger

One debugging option cannot be set through a registry entry. This debugging option is the NdisTraceAll flag and must be set during a debugger session. You are not required to set the NdisTraceAll flag; however, if it is enabled, the debugger produces verbose trace information.

The following sequence shows how to enable the NdisTraceAll flag:

To enable the NdisTraceAll flag

  1. Set the DebugBreak value in the registry to 1 (enabled). Doing so, causes NDIS to break at startup as described in Setting NDIS Debugging Options in the Registry.
  2. When NDIS hits the breakpoint, load symbols for NDIS and then set the NdisTraceAll flag as shown in the following code snippet: 
    kd> wa ndis
kd> e ndistraceall 1
