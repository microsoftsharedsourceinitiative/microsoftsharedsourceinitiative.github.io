Adding a Security Association to a NIC

After the TCP/IP transport determines that a NIC can perform Internet Protocol security (IPSec) operations (see Reporting a NIC's IPSec Capabilities), the transport must request the NIC's miniport driver to add one or more inbound and outbound security associations (SAs) to the NIC before the transport can offload IPSec tasks to the NIC. To request that a miniport driver add one or more SAs to its NIC, the TCP/IP transport sets OID_TCP_TASK_IPSEC_ADD_SA.
