NDIS Debug Extensions For Windows 2000 and Later

For Windows 2000 and later versions of the operating system, you can use extensions in the NDIS debugger DLL (Ndiskd.dll) along with a checked version of the NDIS library of functions (Ndis.sys) to debug various NDIS components. You can also use extensions in the NDIS debugger DLL to examine NDIS data structures. The Debugging Tools for Windows package contains the NDIS debugger DLL. For more information, see Debugging Tools for NT-Based Operating Systems.

Enabling NDIS Debug Tracing on Windows 2000 and Later

Note  To debug NDIS components, a checked version of Ndis.sys is required on the target computer. However, instead of installing an entire checked-build operating system, just copy the checked version of Ndis.sys onto an otherwise free-build operating system. Before you copy the checked version of Ndis.sys to the target computer, you must disable Windows File Protection (WFP). To ensure that WFP is disabled, start the operating system in safe mode.

Note  The NDIS debugging variables that are described in NDIS Debug Extensions For Windows NT 4.0 are also available on Windows 2000 and later versions of the operating system. However, the extensions in the NDIS debugger DLL provide an easier method of enabling debug tracing and allow you to debug more NDIS component types.
