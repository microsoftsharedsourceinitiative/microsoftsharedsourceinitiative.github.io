OID_WW_PIN_LOC_AUTHORIZE

The OID_WW_PIN_LOC_AUTHORIZE OID is obsolete and should not be used; it is included for informational and historical purposes.

This OID requests the miniport driver to return information about whether the device has been authorized to perform location fixes on itself as one of the following:

Value Meaning
0 Unauthorized
1 Authorized
-1 Unknown

Sets and indications are irrelevant for this OID.
