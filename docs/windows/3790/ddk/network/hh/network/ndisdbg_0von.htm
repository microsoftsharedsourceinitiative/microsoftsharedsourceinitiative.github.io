NDIS Debug Extensions For Windows NT 4.0

For Windows NT 4.0, you can use a checked version of the NDIS library of functions (Ndis.sys) plus debugging variables in NDIS to debug various NDIS 4.0 drivers. The following topics list acceptable values for these debugging variables and describe how to set up NDIS debug tracing:

Values for NDIS Debugging Variables in Windows NT 4.0

Setting Up NDIS Debug Tracing on Windows NT 4.0

Note  By enabling the trace levels within NDIS, you could adversely affect timing and performance parameters, which could cause or contribute to additional failures.

Note  These debugging variables are also available on Windows 2000 and later versions of the operating system. However, using extensions in the NDIS debugger DLL provides an easier method to enable debug tracing and allows you to debug more NDIS component types. For more information, see NDIS Debug Extensions For Windows 2000 and Later.
