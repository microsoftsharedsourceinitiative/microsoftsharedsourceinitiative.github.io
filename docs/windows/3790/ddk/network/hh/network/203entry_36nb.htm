Querying Miniport Driver

If MiniportInitialize returns NDIS_STATUS_SUCCESS, NDIS queries and saves the operational characteristics of the driver. These characteristics are represented by object identifiers (OIDs) maintained by the miniport driver. The miniport driver must support all the mandatory general and media-specific OIDs. For more information, see Obtaining and Setting Miniport Driver Information.

Protocol drivers can also issue queries and sets to the miniport driver once MiniportInitialize returns NDIS_STATUS_SUCCESS returns. In particular, a protocol driver can query or set OID_GEN_CURRENT_PACKET_FILTER.
