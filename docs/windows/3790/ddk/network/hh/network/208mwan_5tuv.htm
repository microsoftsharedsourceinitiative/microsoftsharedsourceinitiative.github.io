WAN Miniport Drivers

This section describes NDIS WAN miniport drivers, CoNDIS WAN miniport drivers, CoNDIS WAN call managers, CoNDIS WAN miniport call managers (MCMs), and the environments in which they operate. These WAN drivers support the Remote Access Service (RAS) and the PPP protocol over such media as ISDN, Frame Relay, or Switched 56.
