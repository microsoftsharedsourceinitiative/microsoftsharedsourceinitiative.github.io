Standard Miniport Driver Status Registered with WMI

NDIS automatically registers GUIDs with WMI for the following NDIS_STATUS codes that miniport drivers indicate with NdisM(Co)IndicateStatus:

NDIS_STATUS_RESET_START
NDIS_STATUS_RESET_END
NDIS_STATUS_MEDIA_CONNECT
NDIS_STATUS_MEDIA_DISCONNECT
NDIS_STATUS_MEDIA_SPECIFIC_INDICATION
NDIS_STATUS_LINK_SPEED_CHANGE
