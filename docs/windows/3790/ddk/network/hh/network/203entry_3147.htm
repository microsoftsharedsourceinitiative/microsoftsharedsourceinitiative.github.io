Miniport Driver DriverEntry Function and Initialization

This section describes the DriverEntry function and the MiniportInitialize function, both of which must be supplied by all NDIS miniport drivers. In the context of its DriverEntry function, a miniport driver associates itself with NDIS, specifies the NDIS version that it is using, and registers its entry points. In the context of its MiniportInitialize function, a miniport driver initializes each NIC that it manages, claims system resources (such as packets, buffers, and I/O ports), and registers an interrupt and a shutdown handler.
