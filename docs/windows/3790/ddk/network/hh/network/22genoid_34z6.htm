OID_GEN_MACHINE_NAME

The OID_GEN_MACHINE_NAME OID, which is optional, indicates the system name. The information buffer passed in this request contains an array of Unicode characters that represents the system name. The InformationBufferLength supplied to MiniportSetInformation specifies the length of this array in bytes, not including a NULL terminator.

NDIS sets OID_GEN_MACHINE_NAME only once after a miniport driver completes initialization. Under Windows® XP, NDIS does not dynamically notify miniport drivers of a change in the system name. After changing the system name, a user must reboot the system so that NDIS notifies miniport drivers of the new system name.

System support for OID_GEN_MACHINE_NAME is available in Windows XP and later operating systems.
