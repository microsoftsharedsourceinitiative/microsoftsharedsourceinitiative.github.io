TAPI Query Information Requests

Query-information requests differ from set-information requests in that one or more members in the associated structure, denoted by the "OUT" prefix in the structure definition, are reserved for information the underlying miniport driver should return to the initiator of the request. WAN NIC drivers should treat structure members prefixed with "IN" as read-only.

The first member of all the structures used in query-information requests, ulRequestID, is reserved for future use. Miniport drivers should ignore this member.
