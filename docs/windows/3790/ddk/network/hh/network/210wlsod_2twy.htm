OID_WW_GEN_LATENCY

This OID_WW_GEN_LATENCY OID requests the miniport driver to return its current estimate of the minimum latency, in milliseconds, for sending a net packet of the maximum size permitted by the network from one end point to the other within the wireless network. The return value is of type UINT.

A miniport driver can return a constant value for this query. However, a NIC driver that refines its latency estimates dynamically provides useful information to bound protocols that "autotune" themselves.

Sets and indications are irrelevant for this OID.
