Using NDIS Debugger Extensions

After you load the NDIS debugger DLL, you can examine NDIS data structures and set up NDIS debug tracing by using the following NDIS debugger extensions:

Extension Description
!dbglevel Selects the NDIS debug level.
!dbgsystems Selects NDIS components to debug.
!findpacket Finds specified packets.
!gminiports Displays a list of all miniport instances, including those that have not yet started.
!help Displays all Ndiskd.dll extension commands.
!mem Displays a log of all allocated memory.
!miniport Displays information about a miniport instance using the given address.
!miniports Displays a list of all miniport instances, except for those that have not yet started.
!mopen Displays information about an open miniport instance using the given address.
!ndis Displays NDIS information.
!opens Displays a list of all open objects.
!pkt Displays information about a packet.
!pktpools Displays a list of all allocated packet pools.
!protocol Displays the contents of a protocol block.
!protocols Displays a list of all protocols and their open objects.

For reference information on the NDIS debugger DLL's extensions, search the documentation in the Debugging Tools for Windows package for the "NDIS Extensions" string. For details on this package, see Debugging Tools for NT-Based Operating Systems.

For a quick reference of all the extension commands, enter the help extension at the debugger's command prompt.
