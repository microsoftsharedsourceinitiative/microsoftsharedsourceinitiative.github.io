IEnumNetCfgBindingPath::Skip

The Skip method skips over the next specified number of elements in the enumeration sequence.

HRESULT
  Skip(
    IN ULONG  celt
    );

Parameters

celt
Specifies the number of elements to skip.

Return Value

Returns one of the following values:

S_OK
The number of elements skipped was celt.
S_FALSE
The number of elements skipped was not celt.
E_OUTOFMEMORY
Insufficient memory exists to perform the operation.

Headers

Defined in Netcfgx.h. Include Netcfgx.h.
