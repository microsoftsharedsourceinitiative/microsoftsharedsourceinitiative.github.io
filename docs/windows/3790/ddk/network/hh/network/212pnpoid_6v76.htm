OID_PNP_WAKE_UP_ERROR

The OID_PNP_WAKE_UP_ERROR OID indicates the number of false wake-ups that are signaled by the miniport driver's NIC. A false wake-up occurs when the NIC wakes up the system when it shouldn't have. For example, the NIC could erroneously wake up the system due to an inexact pattern match.

An intermediate driver whose upper edge receives this OID request must always propagate the request to the underlying miniport driver by calling Ndis(Co)Request.
