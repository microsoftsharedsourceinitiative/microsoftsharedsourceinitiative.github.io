Promoting a Secondary Adapter After the Primary Adapter Fails

A miniport driver that supports LBFO can promote a secondary miniport adapter to the primary role if the primary adapter fails. If the miniport driver's primary adapter fails, the driver can call the NdisMRemoveMiniport function to remove the primary miniport adapter from the bundle. The driver can then call the NdisMPromoteMiniport function to promote the secondary miniport adapter to the primary role. NDIS will forward subsequent requests from transports to the miniport driver using the new miniport adapter as the primary.
