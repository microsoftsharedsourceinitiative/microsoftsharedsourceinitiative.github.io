NdisInitUnicodeString

NdisInitUnicodeString initializes a counted Unicode string.

VOID
  NdisInitUnicodeString(
    IN OUT PNDIS_STRING  DestinationString,
    IN PCWSTR  SourceString
    );

Parameters

DestinationString
Pointer to a caller-allocated NDIS_STRING type in which NdisInitUnicodeString should store the counted Unicode string. For Windows 2000 and later, NDIS defines the NDIS_STRING type as a UNICODE_STRING type.
SourceString
Pointer to a zero-terminated string with which to initialize the counted string.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

The DestinationString is initialized to point to the SourceString. The Length and MaximumLength members of NDIS_STRING for the DestinationString are initialized to the length of the string at SourceString. If SourceString is NULL, the length is zero.

Callers of NdisInitUnicodeString can be running at IRQL <= DISPATCH_LEVEL if the DestinationString buffer is allocated from nonpaged memory. Usually, callers are running at IRQL = PASSIVE_LEVEL during driver initialization.
