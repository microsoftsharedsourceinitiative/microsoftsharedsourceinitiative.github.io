Implementing CoNDIS WAN Miniport Drivers

CoNDIS WAN miniport drivers are the same as other CoNDIS drivers with the exception of features added to support WAN operations. For more information about connection-oriented NDIS, see Connection-Oriented NDIS. For more information on features that support WAN operations, see WAN-Specific Capabilities of CoNDIS WAN Drivers.
