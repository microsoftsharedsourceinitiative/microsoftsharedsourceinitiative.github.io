NDIS Debug Extensions For Windows 98

For Windows 98 and Windows 98 SE, you can use a checked version of the NDIS library of functions (Ndis.vxd) plus global debugging symbols to produce debug output for various network drivers as execution passes through NDIS. You enable these debugging symbols during a debugger session. The following topics list acceptable values for these debugging symbols and describe how to set up NDIS debug tracing:
