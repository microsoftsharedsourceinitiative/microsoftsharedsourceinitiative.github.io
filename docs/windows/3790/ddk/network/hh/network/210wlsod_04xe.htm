OID_WW_TAC_TRANSMIT_QUEUE_COUNT

The OID_WW_TAC_TRANSMIT_QUEUE_COUNT OID requests the miniport driver to return the number of frames that are currently queued by the NIC but not yet delivered to the network. This number is returned as an unsigned integer.

If any bound protocol has enabled indications with OID_WW_GEN_INDICATION_REQUEST, the miniport driver must call NdisMIndicateStatus to notify such protocol(s) according to the TriggerValue member that was specified in the OID_WW_GEN_INDICATION_REQUEST.

Sets are irrelevant for this OID.
