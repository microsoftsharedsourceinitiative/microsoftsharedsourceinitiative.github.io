Status Values

The Remote NDIS status values are generally equivalent to the 32-bit status values defined in the Windows 2000 DDK. The specific Remote NDIS status values used in this specification are listed below, others can be inferred from the Windows 2000 DDK or MSDN. A device may return any semantically correct Remote NDIS status value in a Status field of a message that it generates.

Status Identifier Value Description
RNDIS_STATUS_SUCCESS 0x00000000 Success
RNDIS_STATUS_FAILURE 0xC00000001 Unspecified error (equivalent to STATUS_UNSUCCESSFUL)
RNDIS_STATUS_INVALID_DATA 0xC0010015 Invalid data error
RNDIS_STATUS_NOT_SUPPORTED 0xC00000BB Unsupported request error
RNDIS_STATUS_MEDIA_CONNECT 0x4001000B Device is connected to network medium (equivalent to NDIS_STATUS_MEDIA_CONNECT from Windows 2000 DDK)
RNDIS_STATUS_MEDIA_DISCONNECT 0x4001000C Device is disconnected from network medium (equivalent to NDIS_STATUS_MEDIA_DISCONNECT from Windows 2000 DDK)
RNDIS_STATUS_Xxx … Equal to NDIS_STATUS_Xxx values defined in Windows 2000 DDK or MSDN
