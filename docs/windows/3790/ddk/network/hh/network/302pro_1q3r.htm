Medium-Dependent Queries

A protocol driver queries the medium-dependent current address with a medium-dependent OID. For example, a connectionless protocol might query one of the following: OID_WAN_CURRENT_ADDRESS, OID_802_3_CURRENT_ADDRESS, OID_802_5_CURRENT_ADDRESS, or OID_FDDI_LONG_CURRENT_ADDR. A connection-oriented protocol might query OID_WAN_CURRENT_ADDRESS or OID_ATM_HW_CURRENT_ADDRESS.
