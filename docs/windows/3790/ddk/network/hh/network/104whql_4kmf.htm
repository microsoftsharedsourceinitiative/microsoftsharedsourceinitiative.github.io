Prohibited NDIS Functions

WHQL restricts NDIS drivers, including NDIS-WDM, from calling the following NDIS functions. If an NDIS driver calls these functions, WHQL fails the driver and returns an error message that directs the use of the following alternate safe NDIS functions instead.

Restricted NDIS function Alternate safe NDIS function
NdisGetFirstBufferFromPacket NdisGetFirstBufferFromPacketSafe
NdisQueryBuffer NdisQueryBufferSafe
NdisBufferVirtualAddress NdisBufferVirtualAddressSafe
