Discouraged Kernel-Mode Routines

WHQL allows NDIS drivers—that are not also NDIS-WDM—to call the following kernel-mode support routines but discourages their use. If an NDIS driver—that is not also NDIS-WDM—calls one of these functions, WHQL returns a warning message that states, "This call should only be needed by NDIS WDM bus drivers." If possible, do not implement your NDIS driver using these functions because future versions of WHQL test kits might restrict their use.

Note  WHQL does not warn NDIS-WDM drivers against calling the following routines.

IoAllocateMdl

IoBuildPartialMdl
