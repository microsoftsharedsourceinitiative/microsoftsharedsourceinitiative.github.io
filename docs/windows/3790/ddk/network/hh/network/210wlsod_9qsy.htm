OID_WW_CDPD_TEI

The OID_WW_CDPD_TEI OID requests the miniport driver to return the temporary ID assigned by the network to a data-link connection with a particular M-ES. The returned value is of type ULONG, and is used for diagnostic purposes.

Sets and indications are irrelevant for this OID.
