OID_WW_TAC_STATISTICS

The OID_WW_TAC_STATISTICS OID requests the miniport driver to return DataTAC-specific statistics that the miniport driver retrieved from the NIC.

This request uses a WW_TAC_STATISTICS structure, defined as equivalent to the NDIS_VAR_DATA_DESC structure, which was defined previously. This buffer describes statistical information defined by the DataTAC protocol standard for MDC or RD-LAP statistics, based on the current channel attributes. For example, if the channel attribute is RD-LAP, the buffer contains the statistics report. The buffer contains a network-ordered sequence of binary byte values.

Sets and indications are irrelevant for this OID.
