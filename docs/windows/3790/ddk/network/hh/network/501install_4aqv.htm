Network INF File Naming Conventions

File names for INF files that ship with Windows 2000 must conform to the following requirements:

  • Cannot exceed eight characters. INF files that do not ship with Windows 2000 are not bound by this limitation.
  • Uses the .inf file name extension
  • The file name begins with net to indicate that the network class installer is used to process the INF file, except INF files for IrDA devices, which must begin with ir.
  • Indicates either the manufacturer of the network components installed by the INF file, describes the components, or includes the product model number.

The following are examples of the file name components that are used to create valid network INF file names:

File name components Valid file name
net + manufacturer netMsft.inf
net + product description netDlc.inf
net + product model number net999.inf
