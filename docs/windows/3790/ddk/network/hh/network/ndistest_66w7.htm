NDIS Verifier

The NDIS Verifier is a feature that is included in the checked version of the NDIS library of functions (Ndis.sys) for Windows XP and later versions of the operating system. You can use the NDIS Verifier to verify that your network driver is not making invalid or obsolete function calls. You can also use the NDIS Verifier to check the validity of values that are passed to certain functions.
