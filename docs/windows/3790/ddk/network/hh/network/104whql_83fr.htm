ZwXxx Routines

WHQL restricts NDIS drivers—that are not also NDIS-WDM—from calling the following ZwXxx routines:

Restricted routine Error message
ZwCreateDirectoryObject Do not use this call.
ZwCreateKey Do not use this call.
ZwDeleteKey Do not use this call.
ZwEnumerateKey Do not use this call.
ZwEnumerateValueKey Do not use this call.
ZwOpenKey Do not use this call.
ZwOpenSection Do not use this call.
ZwQueryInformationFile Do not use this call.
ZwQueryKey Do not use this call.
ZwQueryValueKey Do not use this call.
ZwReadFile Use NdisMapFile, NdisUnmapFile instead.
ZwSetValueKey Do not use this call.
