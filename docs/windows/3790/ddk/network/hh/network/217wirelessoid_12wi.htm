OID_802_11_TX_ANTENNA_SELECTED

When set, the OID_802_11_TX_ANTENNA_SELECTED OID requests the miniport driver to set, to a specified value, the antenna number that identifies the currently selected transmit (TX) antenna. A value of 0xffffffff indicates that all TX antennas are selected. In other words, a value of 0xffffffff indicates that the wireless LAN is in a state of "diversity."

When queried, this OID requests the miniport driver to return the antenna number that identifies the currently selected transmit antenna.

The antenna value is defined as follows: 

typedef  ULONG   NDIS_802_11_ANTENNA;
