OID_WW_CDPD_CS_SUSPEND

The OID_WW_CDPD_CS_SUSPEND OID requests that the miniport driver instruct the NIC to suspend (hang up) the current CS-CDPD connection. The value is represented as an unsigned integer.

The action requested is designated by one of the following values:

Value Meaning
0 No operation
1 Suspend (hang up) current connection

Queries and indications are irrelevant for this OID.
