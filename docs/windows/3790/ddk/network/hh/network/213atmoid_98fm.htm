OID_ATM_MAX_AAL34_PACKET_SIZE

The OID_ATM_MAX_AAL34_PACKET_SIZE OID requests an ATM miniport driver that supports the AAL34 type to return the maximum packet size that the miniport driver supports for the AAL34 type. If a miniport driver supports the AAL 3/4 type, it must support this OID.
