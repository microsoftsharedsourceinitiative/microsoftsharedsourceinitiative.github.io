OID_WW_GEN_HEADER_FORMATS_SUPPORTED

For the OID_WW_GEN_HEADER_FORMATS_SUPPORTED OID, a miniport driver returns an array of header formats that the miniport driver can support as one or more of the following system-defined values:

NdisWWDIXEthernetFrames
NdisWWMPAKFrames
NdisWWRDLAPFrames
NdisWWMDC4800Frames
NdisWWNCLFrames

DIX Ethernet is the default header format. Every wireless WAN miniport driver must support this format. Any miniport driver can support additional formats. However, a miniport driver is likely to support at most one other format that is native to its NIC.

Note that NdisWWRDLAPFrames and NdisWWMDC4800Frames types have been subsumed under the NdisWWNCLFrames type. That is, if a miniport driver and its NIC support NdisWWRDLAPFrames and NdisWWMDC4800Frames types, they also support the NdisWWNCLFrames type.

Sets and indications are irrelevant for this OID.
