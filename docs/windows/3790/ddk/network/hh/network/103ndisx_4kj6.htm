NdisRawReadPortUchar

NdisRawReadPortUchar reads a byte from a given I/O port on the NIC.

VOID
  NdisRawReadPortUchar(
    IN ULONG_PTR  Port,
    OUT PUCHAR  Data
    );

Parameters

Port
Specifies the I/O port. This address falls in a range that was mapped during initialization with NdisMRegisterIoPortRange.
Data
Pointer to a caller-supplied variable in which this function returns a byte read in from the port.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

NdisRawReadPortUchar runs faster than NdisImmediateReadPortUchar because it need not map a bus-relative I/O port address onto a host-dependent logical port address at every call.

Callers of NdisRawReadPortUchar can run at any IRQL.
