OID_GEN_TRANSMIT_QUEUE_LENGTH

The OID_GEN_TRANSMIT_QUEUE_LENGTH OID specifies the number of packets that are currently queued for transmission, whether on the NIC or in a driver-internal queue. For queries, the number returned is always the total number of packets currently queued. This number can include unsubmitted send requests queued in the NDIS library.
