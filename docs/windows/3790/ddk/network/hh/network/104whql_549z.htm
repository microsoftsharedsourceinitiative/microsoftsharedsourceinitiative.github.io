Memory Manager Routines

WHQL restricts NDIS drivers—that are not also NDIS-WDM—from calling the following memory manager routines:

Restricted routine Error message
MmBuildMdlForNonPagedPool Use NdisAllocateBuffer instead.
MmCreateMdl Use NdisAllocateBuffer instead.
MmGetSystemRoutineAddress Do not use this call.
MmLockPagableDataSection Do not use this call.
MmMapIoSpace Use NdisMMapIoSpace instead.
MmPageEntireDriver Do not use this call.
MmProbeAndLockPages Do not use this call.
MmProbeAndLockProcessPages Do not use this call.
MmQuerySystemSize Do not use this call.
MmResetDriverPaging Do not use this call.
MmSizeOfMdl Do not use this call.
MmUnlockPagableImageSection Do not use this call.
MmUnlockPages Do not use this call.
MmUnmapIoSpace Use NdisMUnmapIoSpace instead.
MmUnmapLockedPages Do not use this call.
