Disabling Task-Offload Capabilities

To disable all of a NIC task-offload capabilities, the TCP/IP transport sets OID_TCP_TASK_OFFLOAD and passes only an NDIS_TASK_OFFLOAD_HEADER in the InformationBuffer. The TCP/IP transport sets the OffsetFirstTask member of the NDIS_TASK_OFFLOAD_HEADER to zero.
