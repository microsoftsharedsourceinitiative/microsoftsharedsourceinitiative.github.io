Executive Support Routines

WHQL restricts NDIS drivers—that are not also NDIS-WDM—from calling the following executive support routines:

Restricted routine Error message
ExAcquireFastMutex Do not use this call.
ExAcquireFastMutexUnsafe Do not use this call.
ExAllocateFromPagedLookasideList Do not use this call.
ExAllocatePool Use NdisAllocateMemoryWithTag instead.
ExAllocatePoolWithQuota Do not use this call.
ExAllocatePoolWithQuotaTag Do not use this call.
ExDeletePagedLookasideList Do not use this call.
ExFreePool Use NdisFreeMemory instead.
ExFreeToPagedLookasideList Do not use this call.
ExInitializePagedLookasideList Do not use this call.
ExInterlockedAddLargeStatistic Do not use this call.
ExQueueWorkItem Use NdisScheduleWorkItem instead.
ExSetTimerResolution Do not use this call.
ProbeForRead Do not use this call.
ProbeForWrite Do not use this call.
