X.25 Framing

X.25 Framing with Protocol Field Compression On or Off

Address and Control Field Compression should not be enabled for X.25. If Protocol Field Compression is turned ON, the NLPID should be removed. It is the responsibility of the X.25 driver or hardware to provide for transparency, add any padding, add any FCS, and add any end flag.
