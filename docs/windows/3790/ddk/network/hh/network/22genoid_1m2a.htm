General Objects

This section describes the set of system-defined general object identifiers (OIDs) that all underlying NDIS drivers support. System-defined general OIDs can be used to get or set both the general operational and general statistics characteristics of miniport drivers and their NICs.

The general OIDs are defined in ntddndis.h. When building a driver that supports these OIDs, include ntddndis.h.
