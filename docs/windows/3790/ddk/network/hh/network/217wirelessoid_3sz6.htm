OID_802_11_RELOAD_DEFAULTS

The OID_802_11_RELOAD_DEFAULTS OID requests the miniport driver to reload the available defaults of a specified type.

For example, when the reload defaults value is set to Ndis802_11ReloadWEPKeys, the miniport driver should reload the available default WEP key settings from permanent storage. (Permanent storage may be a Windows registry key or flash settings on the NIC device.) This ensures that the WEP key settings in the NIC's running configuration return to the same values as those loaded from permanent storage during the initialization of the NIC from power on.

The NDIS_802_11_RELOAD_DEFAULTS enumeration defines the available reload options as follows:

Ndis802_11ReloadWEPKeys
Specifies that the driver should reload available default WEP keys from permanent storage.
