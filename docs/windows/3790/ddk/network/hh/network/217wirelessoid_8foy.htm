802.11 Wireless LAN Objects

The 802.11 wireless LAN (WLAN) object identifiers (OIDs), which are supported by NDIS 5.1, are available through Windows® Management Instrumentation (WMI). Miniport drivers for 802.11 NICs must support all mandatory WLAN OIDs.

The wireless LAN OIDs are defined in ntddndis.h. When building a driver that supports these OIDs, include ntddndis.h.
