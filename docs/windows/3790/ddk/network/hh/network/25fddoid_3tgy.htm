FDDI Objects

This section describes the mandatory OIDs for FDDI objects, both operational and statistical types.

The FDDI OIDs are defined in ntddndis.h. When building a driver that supports these OIDs, include ntddndis.h.

These OIDs are described in the following topics:

FDDI Operational Characteristics

FDDI Statistics

Note that FDDI short addresses are not supported in Microsoft® Windows XP (SP1 and later), Windows® Server 2003, and later operating systems.
