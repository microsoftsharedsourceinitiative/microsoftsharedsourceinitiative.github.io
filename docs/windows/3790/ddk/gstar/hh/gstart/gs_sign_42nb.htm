Windows File Protection (Windows 2000 and later)

Windows File Protection (WFP) protects system files that are required for operation in Windows 2000 and later versions. Protected system files include .sys, .exe, .ocx, and .dll files that ship "in the box" with the operating system.

During WHQL testing, the InfCatReady program checks a driver's INF file to ensure that it does not attempt to replace system files. A driver package that attempts to replace system files cannot receive a digital signature. A driver package can, however, contain updated versions of files that the vendor supplied to Microsoft to ship with Windows 2000 or later versions of the operating system.

For additional information about Windows File Protection, see the Platform SDK documentation.
