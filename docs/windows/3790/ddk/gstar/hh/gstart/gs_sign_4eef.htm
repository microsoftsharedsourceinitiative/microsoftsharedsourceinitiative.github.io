Getting Started with Windows Drivers: Windows DDK

Windows Update

You can distribute a driver package through the Windows Update program if the driver package:

  • Passes the WHQL test program and receives a WHQL digital signature.
  • Qualifies for the Windows Logo program.
  • Meets additional requirements that ensure that Windows Update can determine the correct driver package for the user's device, can legally distribute it, and can automatically download it.

Because the requirements of the Windows Update program are frequently updated, you should regularly check the Windows Update driver publishing Web site. This Web site offers downloadable drivers for Windows 2000 and later, and for Windows 98/Me.
