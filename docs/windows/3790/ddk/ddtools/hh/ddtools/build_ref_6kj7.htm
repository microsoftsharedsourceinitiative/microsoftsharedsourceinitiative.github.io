BUILD_MAKE_PROGRAM

Use the BUILD_MAKE_PROGRAM environment variable to specify the make program to execute.

The default is NMAKE. However, you can set this value to any make program you want (set it equal to the full path and file name).

Caution  Microsoft does not support the use of an alternate make program. It is highly recommended that you not change the value of this variable.
