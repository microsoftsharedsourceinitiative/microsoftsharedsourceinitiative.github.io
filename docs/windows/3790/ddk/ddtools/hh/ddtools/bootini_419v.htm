Driver Development Tools: Windows DDK

/pcilock

Prevents the HAL from reallocating I/O and IRQ resources on the PCI bus. The I/O and memory resources set by the BIOS are preserved.

[ /pcilock ]

Comments

This parameter is valid only on x86-based systems.

Example

multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Windows Server 2003, Standard" /fastdetect /pcilock

Bootcfg Command

bootcfg /raw "/pcilock" /A /ID 1
