Driver Development Tools: Windows DDK

SDK_LIB_PATH

Use the SDK_LIB_PATH environment variable to specify the path to the Platform SDK libraries.

SDK_LIB_PATH should contain an asterisk (*). When the Build utility looks for libraries, it will replace the asterisk with the target platform type — i386 for an x86 processor, ia64 for an Itanium-based processor, or amd64 for an AMD x86-64 processor.

The default is:

%DDKROOT%\lib\Version\*

For this default value, %DDKROOT% is the root installation directory of the DDK. Version indicates the version of Windows supported by the build environment — w2k for Windows 2000, wxp for Windows XP, or wnet for Windows Server 2003.
