COFFBASE

Use the COFFBASE macro to specify the name to look up in COFFBASE_TXT_FILE.

If you do not specify a name, it defaults to the value of TARGETNAME (if building a DYNLINK) or to "usermode" (if you are building a PROGLIB, or PROGRAM).

To specify an absolute base address for your image without using a global file for all, see DLLBASE or UMBASE.
