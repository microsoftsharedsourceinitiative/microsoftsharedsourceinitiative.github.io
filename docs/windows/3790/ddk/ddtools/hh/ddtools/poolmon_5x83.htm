PoolMon Commands

PoolMon has two separate, but related, sets of commands.

  • PoolMon Startup Command: To start PoolMon, type a command at the command line using the startup command syntax and parameters.
  • PoolMon Run-time Commands: To configure PoolMon while it is running, type a run-time command.
