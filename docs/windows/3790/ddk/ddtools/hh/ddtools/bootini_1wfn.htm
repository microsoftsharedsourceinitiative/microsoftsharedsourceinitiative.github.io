Exporting and Importing Boot Entries in NVRAM

Use the nvrboot x (export) command to export a boot entry to a backup file for safekeeping. Design a naming and storage convention that will make it easy to find the backup copy files when you need them.

Use the nvrboot i (import) command to import boot entries from the boot entry backup files that you exported or from the Bootxxxx boot entry backup files that Setup created.

Imported boot entries are assigned new NVRAM boot entry IDs, even when they are imported from a boot entry backup file that Setup created. For example, if you import an entry from the Boot0003 file, the imported boot entry receives a new boot entry ID, such as Boot000A.
