MSC_OPTIMIZATION

Use the MSC_OPTIMIZATION macro to override the default optimization that the Build utility uses on the compiler. By default, everything is optimized. If you want to turn off optimization in order to step through your code, you can set the MSC_OPTIMIZATION value to whatever is appropriate for your compiler.

When building in the Checked Windows DDK environment, this value is set to /Od /Oi, which avoids all optimization. All of these flags also have platform-specific variants.

If MSC_OPTIMIZATION is not specified, the default is 0xs, which fully optimizes the binaries, with a preference for smaller rather than faster binaries.

If you do not prefer these settings, you can always override the generic MSC_OPTIMIZATION macro with platform-specific macros. Available macros include I386_OPTIMIZATION for Intel x86, and IA64_OPTIMIZATION for Intel Itanium.

Note  Platform-specific settings (386_OPTIMIZATION, for example) will override the generic setting (MSC_OPTIMIZATION) if both are defined.
