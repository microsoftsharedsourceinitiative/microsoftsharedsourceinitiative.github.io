SYNCHRONIZE_DRAIN

The SYNCHRONIZE_DRAIN macro instructs the Build utility to build a certain directory last.

A directory whose sources file contains this macro will not be built until all other subdirectories have been built. To use this macro, specify the following in your sources file:

SYNCHRONIZE_DRAIN=1

For details, see Building on a Multiprocessor Computer.
