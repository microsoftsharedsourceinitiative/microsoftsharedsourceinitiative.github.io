NTTARGETFILE1

The NTTARGETFILE1 macro is exactly like NTTARGETFILE0, except that it occurs later in the build process. NTTARGETFILE0 occurs on pass zero; NTTARGETFILE1 occurs on pass two.

Both of these macros cause the Build utility to change directories to a specified subdirectory and to run NMAKE on pass zero or pass two in a case where it ordinarily would not do so.
