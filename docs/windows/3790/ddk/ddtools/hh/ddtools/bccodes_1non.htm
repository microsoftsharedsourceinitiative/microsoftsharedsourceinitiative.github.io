Bug Check 0x7D: INSTALL_MORE_MEMORY

The INSTALL_MORE_MEMORY bug check has a value of 0x0000007D. This indicates that there is not enough memory to boot Windows.

Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 The number of physical pages found
2 The lowest physical page
3 The highest physical page
4 0

Cause

Windows does not have sufficient memory to complete the boot process.

Resolving the Problem

Install more memory.
