Bug Check 0x80: NMI_HARDWARE_FAILURE

The NMI_HARDWARE_FAILURE bug check has a value of 0x00000080. This indicates that a hardware malfunction has occurred.

Parameters

None

Cause

This can be caused by a variety of hardware malfunctions. The exact cause is difficult to determine.

Resolving the Problem

Remove any hardware or drivers that have been recently installed. Make sure that all memory modules are of the same type.
