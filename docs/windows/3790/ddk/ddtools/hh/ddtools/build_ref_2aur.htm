(platform)mk.inc File

For each platform, there is a special make file:

  • i386mk.inc
  • ia64mk.inc

The current versions of these files reside on %ddkroot%\bin. In these files, there are optimization rules or assembler instructions that are specific for that target. There may also be extra link flags, resource compiler flags, or C preprocessor rules that are required. The makefile.inc files are not something you would usually edit because there are macros to override all the settings in makefile.inc in your sources file. If you want to change the optimization, use MSC_OPTIMIZATION. If you want to change linker flags, use LINKER_FLAGS.
