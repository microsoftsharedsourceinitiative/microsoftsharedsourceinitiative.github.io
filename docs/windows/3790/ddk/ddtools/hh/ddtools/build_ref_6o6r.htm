SOURCES_USED

Use the SOURCES_USED macro to indicate that there is another sources file or make file elsewhere in the tree. Use this macro if that additional file contains build dependencies.

When you specify:

SOURCES_USED=filename_someplace_else

NMAKE regenerates the objects file when changes occur in the sources file that resides in the specified directory. NMAKE will also regenerate the objects file when changes occur in any sources file or make file that you specified in the SOURCES_USED macro.

For example, a sources file typically includes some other file. Just beneath that directive you could specify:

SOURCES_USED=that_filename

This instruction establishes a link between that file and the sources file. When that other file changes, NMAKE regenerates the _objects.mac file.
