Bug Check 0x8F: PP0_INITIALIZATION_FAILED

The PP0_INITIALIZATION_FAILED bug check has a value of 0x0000008F. This indicates that the Plug and Play Manager could not be initialized.

Parameters

None

Cause

An error occurred during Phase 0 initialization of the kernel-mode Plug and Play Manager.
