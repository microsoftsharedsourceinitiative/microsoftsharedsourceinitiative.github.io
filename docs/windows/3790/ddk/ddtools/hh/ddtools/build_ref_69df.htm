TARGETPATHLIB

In the process of building a DLL, the Build utility also creates an import library to be used by other images that must use your DLL.

Use the TARGETPATHLIB macro to specify the target directory for this import library.

If TARGETPATHLIB is not specified, import libraries are placed in the same subdirectory as other build product files (that is, a subdirectory under TARGETPATH).
