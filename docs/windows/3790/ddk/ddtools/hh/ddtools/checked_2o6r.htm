Installing the Complete Checked Build

Probably the easiest way to use the checked build for debugging drivers is to install the complete checked build on your computer. Just install the check build by using the checked build distribution medium. This installs all available checked components on your computer.

This installation will result in maximum debugging checks, but it also cause the system to run considerably slower than it would if the free build of the operating system were installed.
