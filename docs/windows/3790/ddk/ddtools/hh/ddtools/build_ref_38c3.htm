NTDEBUGTYPE

Use the NTDEBUGTYPE macro to specify the method of linking and the kind of symbols that are used.

There are two supported values, and both have the same meaning. They are:

NTDEBUGTYPE=both

and

NTDEBUGTYPE=windbg

Both of these cause the Build utility to produce CodeView symbol information. All debuggers can understand these symbols.
