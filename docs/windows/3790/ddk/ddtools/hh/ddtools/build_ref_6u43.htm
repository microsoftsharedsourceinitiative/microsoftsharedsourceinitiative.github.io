UMBASE

The UMBASE macro can be used to set the base address for a program that you are creating.

If UMBASE is omitted or left blank, the base address will be determined by combining the file specified by COFFBASE_TXT_FILE and the executable name specified by COFFBASE. See COFFBASE and COFFBASE_TXT_FILE for details and for a description of how the base address is chosen if these macros are not defined.

If UMBASE is not defined and the base address cannot be determined through COFFBASE_TXT_FILE, the default base address of 0x01000000 will be used.

Note  Unless you are building a DLL, it is very unlikely that your program would need a different base address than the default.

If you are building a DLL, you cannot use UMBASE. You must use the COFFBASE_TXT_FILE file or the DLLBASE macro instead.
