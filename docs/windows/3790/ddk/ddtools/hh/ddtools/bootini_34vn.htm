Introduction to Boot Options

The concept and content of boot options are very similar on all computers that run Microsoft® Windows® operating systems. However, the options are stored differently on 32-bit x86-based computers and 64-bit Intel Itanium 2-based computers, and different tools are used to view and edit the options on each system.
