Bug Check 0x75: CANNOT_WRITE_CONFIGURATION

The CANNOT_WRITE_CONFIGURATION bug check has a value of 0x00000075. This indicates that the SYSTEM registry hive file cannot be converted to a mapped file.

Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 1
2 The NT status code that led Windows to assume that it had failed to convert the hive
3 Reserved
4 Reserved

Cause

This error usually happens if the system is out of pool and Windows cannot reopen the hive.

This error should almost never occur, since the conversion happens quite early during system initialization, so enough pool should be available.
