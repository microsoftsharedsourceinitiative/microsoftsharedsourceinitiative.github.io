MIDL_UUIDDIR

Use the MIDL_UUIDDIR macro to specify where the GUID file is placed when you generate an OLE interface description language (IDL) file. (UUIDs and GUIDs are the same thing.)

By default, everything built in pass zero is placed in the location that you defined as the TARGETPATH subdirectory. Use this macro to override that default.

To create the GUIDs in the current subdirectory, set MIDL_UUIDDIR to be dot ("."):

MIDL_UUIDDIR=.

It does not matter where you put the GUIDs so long as you set your #include directives to retrieve them from this location. Otherwise, the tools later used by the Build utility will not be able to find the GUIDs.
