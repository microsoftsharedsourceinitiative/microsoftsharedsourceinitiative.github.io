NO_BINPLACE

Use the NO_BINPLACE environment variable to control whether the Build utility automatically runs the BinPlace tool.

In all the DDK build environments, the default is:

NO_BINPLACE=TRUE

If this environment variable is deleted, NMAKE will automatically launch BinPlace.

For more details, see BinPlace.
