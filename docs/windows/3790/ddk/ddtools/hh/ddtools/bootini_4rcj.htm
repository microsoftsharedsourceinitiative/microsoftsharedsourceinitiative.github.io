Backing Up the boot.ini File

When you install or upgrade an operating system, the Windows installer creates a new boot.ini file for the computer. The new file retains some, but not all, previous customizations to the file.

To preserve your customized boot.ini file, make a backup copy before upgrading or installing the operating system.

After an update completes, you can replace the new file with your backup copy. If you have installed a new operating system, you can copy customized entries from your backup copy and paste them into the new boot.ini file.

An update or installation restores the default attributes on the boot.ini file, including the read-only attribute. To customize the file, use Bootcfg or change the file attributes. For instructions, see Editing the boot.ini File.
