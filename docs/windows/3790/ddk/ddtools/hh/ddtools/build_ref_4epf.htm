BUILD_MULTIPROCESSOR

When the BUILD_MULTIPROCESSOR environment variable is defined on a multiprocessor computer, Build will launch one thread for each processor.

This is equivalent to using the -M command-line option without an additional argument. See Build Utility Command-Line Options for information on the -M threads option.

On a single-processor machine, this variable has no effect.
