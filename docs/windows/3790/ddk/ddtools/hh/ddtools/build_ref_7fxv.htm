LANGUAGE

Use the LANGUAGE macro to specify the language when you set up dependencies so that you can include country/region-specific parts in your build. If you are building a product for the USA, you might want to include a USA icon, but if you are building for Korea, you would want to use a Korean icon.

The default is LANGUAGE=USA.
