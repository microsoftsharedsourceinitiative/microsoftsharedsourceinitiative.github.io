Note  The term build product denotes one or more binary files created by the Build utility. In the Windows DDK, the build product is typically a driver. However, the Build utility can be used to compile and build any kind of Windows binary. Indeed, Microsoft uses the Build utility to create the binary files for its NT-based operating systems.
