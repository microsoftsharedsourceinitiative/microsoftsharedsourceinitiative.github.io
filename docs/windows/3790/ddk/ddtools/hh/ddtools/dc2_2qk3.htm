HCT Test

Command-line parameter: /hct

The HCT Test is the combination of tests and test options that are used when you run Device Path Exerciser from the Test Manager tool in the Hardware Compatibility Test Kit.

Using the /hct parameter is equivalent to the following Device Path Exerciser command:

dc2 +al -c +dd /dl 0 /du 200 +e +f -fi /fl 0 +fn +fr /fu 400 +i +if /il 0 +im +in +ir /iu 512 +j -k +l /ltc /ldn /lrn +m -n +oc /ol 0 /ou 512 -p -pr +q -r +s +sd -sl /t 100000 /tt 10000 +y -w
