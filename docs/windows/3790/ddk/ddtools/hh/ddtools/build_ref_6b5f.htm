Driver Development Tools: Windows DDK

WDM_INC_PATH

Use the WDM_INC_PATH environment variable to specify the path to the WDM headers.

The default is:

%DDKROOT%\inc\ddk\wdm\Version

For this default value, %DDKROOT% is the root installation directory of the DDK. Version indicates the version of Windows supported by the build environment — w2k for Windows 2000, wxp for Windows XP, or wnet for Windows Server 2003.

If the DRIVERTYPE macro in your sources file has been set to WDM, the Build utility will use WDM_INC_PATH instead of DDK_INC_PATH.
