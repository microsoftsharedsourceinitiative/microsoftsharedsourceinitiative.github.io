Bug Check 0xA2: MEMORY_IMAGE_CORRUPT

The MEMORY_IMAGE_CORRUPT bug check has a value of 0x000000A2. This indicates that corruption has been detected in the image of an executable in memory.

Parameters

The following parameters are displayed on the blue screen. Parameter 1 indicates the type of violation. The meaning of the other parameters depends on the value of Parameter 1.

Parameter 1 Parameter 2 Parameter 3 Parameter 4 Cause of Error
0x02 If Parameter 3 is zero: The page number in the table page which failed

If Parameter 3 is nonzero: The page number with the failing page run index

Zero, or the index which failed to match the run 0 A table page check failure occurred.
0x03 The starting physical page number of the range The length (in pages) of the range The page number of the table page containing this run The checksum for the range of memory listed is incorrect.

Cause

A cyclic redundancy check (CRC) check on the memory range has failed.

On a system wake operation, various regions of memory may be CRC'd to guard against memory failures.
