Driver Development Tools: Windows DDK

Build

The Build utility is used to build the samples, drivers, and associated software components that are supplied in the Microsoft® Windows® DDK. The Build utility helps you maintain target-dependent information, such as source file names and paths to header files and output directories, in a single file. The utility automatically sets the compiler and linker switches that are required to create output files for specific platforms that you specify.
