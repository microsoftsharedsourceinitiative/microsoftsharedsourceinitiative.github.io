USE_NATIVE_EH

Use the US_NATIVE_EH macro if you are using standard C++ exception handling (try, catch, and throw). Specify the following in your sources file:

USE_NATIVE_EH=1

This instruction directs the Build utility to turn on some needed switches to the compiler.
