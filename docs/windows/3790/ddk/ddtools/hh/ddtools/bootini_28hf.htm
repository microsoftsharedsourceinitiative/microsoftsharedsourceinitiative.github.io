Boot Parameter Reference

Boot entry parameters, or boot parameters, are optional, system-specific switches that represent configuration options. You can add boot parameters to a boot entry for an operating system.

This section describes the boot parameters that are related to developing, testing, and debugging drivers on x86- and Itanium-based systems. You can add these parameters to the boot entries for Windows operating systems.

For a complete list of supported boot parameters, see Reviewing and Correcting Boot.ini Settings on x86-based Systems and Reviewing and Correcting NVRAM Startup Settings on Itanium-based Systems in the Windows XP Professional Resource Kit Documentation.
