Driver Development Tools: Windows DDK

/bootlog

Records the names of drivers as they load during the boot process and saves the ordered list in the %SystemRoot%\ntbtlog.txt file. 

[ /bootlog ]

Example

multi(0)disk(0)rdisk(0)partition(1)\WINDOWS="Microsoft Windows XP Professional" /fastdetect /bootlog

Bootcfg Command

bootcfg /raw "/bootlog" /A /ID 1
