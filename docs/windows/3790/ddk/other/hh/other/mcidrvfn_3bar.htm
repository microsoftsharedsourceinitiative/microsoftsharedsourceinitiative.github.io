mciSetDriverData

Do not write an MCI driver to support a multimedia device; write a WDM AVStream driver instead.

The mciSetDriverData function sets instance data for the given multimedia device.

BOOL
  mciSetDriverData(
    MCIDEVICEID wDeviceID,
    DWORD_PTR dwData
    );

Parameters

wDeviceID
Specifies the device identifier of the multimedia device for which to set instance data.
dwData
Pointer to a DWORD_PTR location that contains instance data to set for the multimedia device.

Return Value

Returns TRUE on success; otherwise, returns FALSE if the device identifier is unrecognized.

Headers

Declared in Mmddk.h. Include Mmddk.h.

Comments

Typically, when the device is opened, MCI device drivers call mciSetDriverData to set instance data. In response to messages, the driver subsequently calls the mciGetDriverData function to retrieve instance data.

For more information about MCI, see the Platform SDK documentation.
