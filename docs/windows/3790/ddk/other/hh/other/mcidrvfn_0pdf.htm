mciGetDriverData

Do not write an MCI driver to support a multimedia device; write a WDM AVStream driver instead.

The mciGetDriverData function retrieves instance data for a given multimedia device.

DWORD_PTR
  mciGetDriverData(
    MCIDEVICEID wDeviceID
    );

Parameters

wDeviceID
Specifies the device identifier of the multimedia device for which to retrieve instance data.

Return Value

Returns the instance data or a pointer to a DWORD_PTR location that contains instance data for the given multimedia device; otherwise, on error, returns zero. Note that the driver can only verify when instance data is zero if the returned value is a pointer.

Headers

Declared in Mmddk.h. Include Mmddk.h.

Comments

Typically, when the device is opened, MCI device drivers call the mciSetDriverData function to set instance data. In response to messages, the driver subsequently calls mciGetDriverData to retrieve instance data.

For more information about MCI, see the Platform SDK documentation.
