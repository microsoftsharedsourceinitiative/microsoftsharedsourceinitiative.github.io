VDDDeInstallIOHook

VDDDeInstallIOHook unhooks the I/O ports previously hooked by a VDD.

VOID
  VDDDeInstallIOHook(
    IN HANDLE  hVdd,
    IN WORD  cPortRange,
    IN PVDD_IO_PORTRANGE  pPortRange
    );

Parameters

hVdd
Identifies the VDD.
cPortRange
Specifies the number of VDD_IO_PORTRANGE structures.
pPortRange
Points to the VDD_IO_PORTRANGE array.

Comments

When VDDDeInstallIOHook cancels a hook, the default hook on those ports subsequently returns 0xFF when reading the port and ignores write operations.
