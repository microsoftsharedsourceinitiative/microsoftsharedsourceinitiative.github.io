WIA_DPC_THUMB_HEIGHT

The WIA_DPC_THUMB_HEIGHT property defines the height, in pixels, of a thumbnail image to use for newly captured images. An application reads this value to get an estimated size for displaying thumbnails in its user interface.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE or WIA_PROP_LIST

Access Rights: READONLY | WIA_PROP_NONE or READ/WRITE | WIA_PROP_LIST

Headers

Declared in wiadef.h. Include wiadef.h.

Comments

If the valid value is WIA_PROP_NONE, then the access rights must be READONLY. If the valid value is WIA_PROP_LIST, then the access rights must be READ/WRITE.
