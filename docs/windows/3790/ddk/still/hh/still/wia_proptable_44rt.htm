WIA_IPA_ICM_PROFILE_NAME

The WIA_IPA_ICM_PROFILE_NAME property contains the ICM profile name that is needed to properly decode the image. An application reads this property to determine the ICM profile to use when processing the image. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
