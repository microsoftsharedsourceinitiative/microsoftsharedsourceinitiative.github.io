WIA_DPC_EXPOSURE_TIME

The WIA_DPC_EXPOSURE_TIME property corresponds to the shutter speed, in units of seconds that are scaled by 10,000. This property is typically used only by the device when the WIA_DPC_EXPOSURE_MODE property is set to EXPOSUREMODE_MANUAL or EXPOSUREMODE_SHUTTER_PRIORITY.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE or WIA_PROP_LIST

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
