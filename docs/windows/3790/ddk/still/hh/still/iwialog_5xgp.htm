WIAS_LHRESULT

The WIAS_LHRESULT macro translates an HRESULT value into a string and writes the string to the diagnostic log file.

WIAS_LHRESULT(
  IWiaLog  *pIWiaLog,
  HRESULT  hr
  );

Parameters

pIWiaLog
Points to an IWiaLog interface.
hr
Specifies the HRESULT value to be translated into a string.

Headers

Declared in wia.h. Include wia.h.

Comments

The following is an example of how the macro would be used:

hr = wiasFreePropContext(*pContext);
if (hr != S_OK)
   WIAS_LHRESULT(g_pIWiaLog, hr);

This macro is available in Windows Me and in Windows XP and later.
