WIA_IPA_TYMED

The WIA_IPA_TYMED property contains the transfer method setting. An application reads this property to determine the minidriver's method of data transfer. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Value Definition
TYMED_CALLBACK Transfer an image to memory, in bands.
TYMED_FILE Transfer an image to a file.
TYMED_MULTIPAGE_CALLBACK Transfer multiple images to memory, in bands.
TYMED_MULTIPAGE_FILE Transfer multiple images to a file.

Headers

Declared in wiadef.h. Include wiadef.h.
