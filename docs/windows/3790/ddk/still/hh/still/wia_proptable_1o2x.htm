WIA_IPA_ITEM_TIME

The WIA_IPA_ITEM_TIME property contains the time that the image was originally captured. The minidriver creates and maintains this property. This property should be reported as a vector of eight WORD values in the form of a SYSTEMTIME structure (described in the Platform SDK documentation).

Property Type: VT_UI2 | VT_VECTOR

Valid Values: WIA_PROP_NONE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
