WIA_DPC_PICT_HEIGHT

The WIA_DPC_PICT_HEIGHT property contains the height in pixels to use for newly captured images. The list of valid values for this property has a one-to-one correspondence with the list of valid values for the WIA_DPC_PICT_WIDTH property. If the individual width and height are linearly settable and orthogonal to each other, they may be expressed as a range.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST or WIA_PROP_RANGE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
