INF File Entries

Using a microdriver for your device requires additional entries in the INF file. First, there must be an entry called “MicroDriver” in the DeviceData section of the INF file. (See INF Files for WIA Devices for more information.) This entry should be set to the name of the DLL that implements the microdriver.

In the INF AddReg directive where your WIA minidriver normally would be referenced, it should list wiafbdrv.dll as the driver. This is the component that implements the WIA Flatbed Driver.

Be sure that the INF CopyFiles directive includes both your microdriver and wiafbdrv.dll, which is copied from the Windows CAB files. The rest of the INF file is the same as for other WIA devices.
