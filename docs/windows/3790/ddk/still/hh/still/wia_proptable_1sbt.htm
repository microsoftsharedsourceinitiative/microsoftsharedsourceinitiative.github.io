WIA Properties

This section contains all the WIA properties and their attributes.

The reference pages for these properties list the following attributes: property type, valid values and access rights.

Possible property types are:

VT_BSTR
A string of Unicode characters, a wide character string.
VT_I4
A four-byte integer.
VT_R4
A four-byte real number, a C float.
VT_UI2
A two-byte unsigned integer.
VT_VECTOR
An array of elements of a particular type. If another property type is combined with VT_VECTOR using the OR operator, the value is a counted array consisting of a DWORD count of elements, followed by a pointer to the first of the elements in the array. For example, VT_UI2 | VT_VECTOR has a DWORD element count, followed by a pointer to an array of two-byte unsigned integer elements.

Possible valid value types are:

WIA_PROP_NONE
The property contains a single value.
WIA_PROP_RANGE
The property contains a range of values. A range of values includes a minimum value, a maximum value, a nominal value and an increment.
WIA_PROP_LIST
The property contains a list of values.

Possible access rights are:

READONLY
The driver has read-only access.
READ/WRITE
The driver has both read and write access.

The information presented here is not comprehensive. It is includes only those types used in the WIA properties that follow. For more information about these and other property types, see WIA_PROPERTY_INFO and PROPVARIANT (defined in the Platform SDK documentation).
