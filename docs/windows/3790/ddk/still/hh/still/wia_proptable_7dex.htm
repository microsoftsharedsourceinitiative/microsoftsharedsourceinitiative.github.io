WIA_DPC_FOCUS_DISTANCE

The WIA_DPC_FOCUS_DISTANCE property contains the distance, in millimeters, between the image-capturing plane of the digital camera, and the point of focus. A value of 0xFFFF corresponds to a setting greater than 655 meters.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE or WIA_PROP_LIST

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
