WIA_DPC_PICTURES_REMAINING

The WIA_DPC_PICTURES_REMAINING property contains the number of pictures that can be taken at the current property settings. If the current property settings change in a way that affects the size of the images produced by the camera device, the WIA minidriver should update the number of pictures that remain. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
