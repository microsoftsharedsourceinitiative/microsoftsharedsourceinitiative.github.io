wiasGetChildrenContexts

The wiasGetChildrenContexts function retrieves an array of item contexts belonging to the current item's children.

HRESULT 
  wiasGetChildrenContexts(
    BYTE  *pParentContext,
    ULONG  *pulNumChildren,
    BYTE  ***pppChildren
    );

Parameters

pParentContext
Pointer to the parent item.
pulNumChildren
Pointer to a memory location that receives the number of children contexts.
pppChildren
Pointer to a memory location that points to an array whose elements are addresses of IWiaItem objects (described in the Platform SDK documentation). Each IWiaItem object represents one child context.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
