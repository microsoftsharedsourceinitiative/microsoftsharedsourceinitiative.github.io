WIA_DPC_BATTERY_STATUS

The WIA_DPC_BATTERY_STATUS property defines the percentage of battery power that is left to operate the camera device. This value should be an integer from 0 through 100. An application reads this property to determine the remaining battery life of the camera device.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
