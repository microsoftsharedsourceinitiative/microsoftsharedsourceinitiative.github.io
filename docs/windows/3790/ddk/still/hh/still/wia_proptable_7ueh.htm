WIA_DPC_UPLOAD_URL

The WIA_DPC_UPLOAD_URL property describes a standard Internet Universal Resource Locator (URL) to which images or objects can be uploaded after they are acquired from the device. (Windows does not upload images by itself. There are two scenarios: WIA applications read this property and allow the user to upload images to the URL automatically; or an application sets the URL and other devices (kiosks, and so forth) use this property.)

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
