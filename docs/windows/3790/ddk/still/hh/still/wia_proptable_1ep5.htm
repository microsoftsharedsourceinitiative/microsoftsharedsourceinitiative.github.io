WIA_DPS_PAGE_HEIGHT

The WIA_DPS_PAGE_HEIGHT property contains the height of the current page selected, in 1000ths of an inch. An application reads this property to determine the physical dimensions of the page being scanned. If the extent settings are different from known page sizes, this property reports the height of the page whose WIA_DPS_PAGE_SIZE property is set to WIA_PAGE_CUSTOM (a value of the WIA_DPS_PAGE_SIZE property). The WIA_DPS_PAGE_HEIGHT property is created and maintained by the minidriver.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
