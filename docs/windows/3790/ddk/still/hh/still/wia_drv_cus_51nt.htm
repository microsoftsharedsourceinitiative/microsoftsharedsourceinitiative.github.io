Adding UI to a WIA Minidriver

Adding extended UI or replacing UI components for a WIA minidriver can be done by installing a separate DLL along with the WIA minidriver. Unlike a TWAIN driver, a WIA driver’s UI component is separate from the actual WIA minidriver. For Windows® Me, Windows XP, and later, the UI components run in the application’s process, while the WIA minidriver runs in the WIA service’s process.
