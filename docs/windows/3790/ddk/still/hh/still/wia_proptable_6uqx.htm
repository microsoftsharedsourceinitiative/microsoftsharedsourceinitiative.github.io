WIA_DPC_RGB_GAIN

The WIA_DPC_RGB_GAIN property contains a null-terminated Unicode string that represents the red, green, and blue gain that is applied to image data. For example, "4:25:50" represents a red gain of 4, a green gain of 25, and a blue gain of 50.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE, WIA_PROP_LIST or WIA_PROP_RANGE

Access Rights: READ/WRITE

Comments

This property is parsed as follows: R:G:B where R represents the red gain, G represents the green gain, and B represents the blue gain. For example, for an RGB ratio of red=4, green=2, blue=3, the RGB string could be "4:2:3" or "2000:1000:1500". These values are relative to each other. Larger numbers may be used for added granularity, but as long as the ratio of R, G and B remain the same, the color will be the same. The string parser for this property value should be able to support UINT16 integers for R, G, and B.
