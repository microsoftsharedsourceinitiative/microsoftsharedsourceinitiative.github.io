WIA_DPS_SHEET_FEEDER_REGISTRATION

The WIA_DPS_SHEET_FEEDER_REGISTRATION property contains the registration, or alignment and edge detection, for documents that are placed on the flatbed. The minidriver creates and maintains this property. See also WIA_DPS_HORIZONTAL_BED_REGISTRATION.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
