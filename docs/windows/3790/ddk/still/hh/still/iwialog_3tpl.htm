IWiaLog Interface

The IWiaLog interface provides methods to enable minidrivers to log trace, error, and warning messages to the diagnostic log file wiaservc.log. The prototypes for the methods appear in wia.h. The diagnostic log file wiaservc.log is found in the Windows directory, or in the directory returned by the GetWindowsDirectory system API call, which is described in the Platform SDK documentation.
