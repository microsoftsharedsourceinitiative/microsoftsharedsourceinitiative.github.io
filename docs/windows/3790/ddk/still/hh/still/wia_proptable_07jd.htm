WIA_DPC_FNUMBER

The WIA_DPC_FNUMBER property corresponds to the aperture of the lens, in units of the f-stop number scaled by 100. The setting of this property is typically valid only when the WIA_DPC_EXPOSURE_MODE property is set to EXPOSUREMODE_MANUAL or EXPOSUREMODE_APERTURE_PRIORITY.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Declared in wiadef.h. Include wiadef.h.
