WIA_DPC_COMPRESSION_SETTING

The WIA_DPC_COMPRESSION_SETTING property is intended to be approximately linear with respect to perceived image quality over a broad range of scene content, and is represented by either a range or a list of integers. Smaller integers are used to represent lower quality (that is, maximum compression), while larger integers are used to represent higher quality (that is, minimum compression). Any available settings on a device are relative to that device only and are therefore device-specific.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST or WIA_PROP_RANGE

Access Rights: READ/WRITE
