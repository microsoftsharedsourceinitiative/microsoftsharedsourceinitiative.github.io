WIA_DPS_PAGE_WIDTH

The WIA_DPS_PAGE_WIDTH property contains the width of the current page selected, in 1000ths of an inch. An application reads this property to determine the physical dimensions of the page being scanned. If the extent settings are different from known page sizes, this property reports the width of the page whose WIA_DPS_PAGE_SIZE property is set to WIA_PAGE_CUSTOM. This property is created and maintained by the minidriver.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
