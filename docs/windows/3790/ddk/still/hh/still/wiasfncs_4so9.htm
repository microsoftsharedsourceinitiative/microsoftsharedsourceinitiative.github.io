wiasSetValidRangeLong

The wiasSetValidRangeLong function specifies the range of valid values for a WIA_PROP_RANGE property of type VT_I4.

HRESULT
  wiasSetValidRangeLong(
    BYTE  *pWiasContext,
    PROPID  propid,
    LONG  lMin,
    LONG  lNom,
    LONG  lMax,
    LONG  lStep
    );

Parameters

pWiasContext
Pointer to a WIA item context.
propid
Specifies the identifier of the property to be updated.
lMin
Specifies the minimum value of the valid range.
lNom
Specifies the property's nominal value.
lMax
Specifies the maximum value of the valid range.
lStep
Specifies the increment between each valid value in the range.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
