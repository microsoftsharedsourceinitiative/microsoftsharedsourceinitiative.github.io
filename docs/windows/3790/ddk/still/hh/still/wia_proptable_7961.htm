WIA_IPA_NUMBER_OF_LINES

The WIA_IPA_NUMBER_OF_LINES property contains the number of lines contained in the image (the vertical height of the image in pixels). The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
