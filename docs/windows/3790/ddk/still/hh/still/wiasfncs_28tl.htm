wiasSetValidListFloat

The wiasSetValidListFloat function sets valid values for a WIA_PROP_LIST property of type VT_R4.

HRESULT
  wiasSetValidListLong(
    BYTE  *pWiasContext,
    PROPID  propid,
    ULONG  ulCount,
    FLOAT  fNom,
    FLOAT  *pfValues
    );

Parameters

pWiasContext
Pointer to a WIA item context.
propid
Specifies the identifier of the property to be updated.
ulCount
Specifies the number of items in the plValues array.
fNom
Specifies the property's nominal value.
pfValues
Pointer to the first element of an array of valid property values.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
