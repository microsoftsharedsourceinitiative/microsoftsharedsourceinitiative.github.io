Still Image Devices

This section presents information of interest to those developing drivers for imaging devices such as scanners and cameras.

The material in this section is divided into two major parts:

Design Guide

Reference

The Design Guide provides an overview, explanatory information and programming examples about the various types of drivers associated with imaging devices.

The Reference provides detailed information about all of the functions and structures available to driver developers. It also provides information about the functions that driver developers must implement in their drivers.
