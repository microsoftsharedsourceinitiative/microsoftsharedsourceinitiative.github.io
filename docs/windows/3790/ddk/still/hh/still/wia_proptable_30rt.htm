WIA_DPC_PICTURES_TAKEN

The WIA_DPC_PICTURES_TAKEN property contains the number of pictures taken by the camera. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
