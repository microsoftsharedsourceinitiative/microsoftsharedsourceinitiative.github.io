WIA_DIP_REMOTE_DEV_ID

The WIA_DIP_REMOTE_DEV_ID property contains the device ID of the WIA device that is installed on a remote machine. The WIA service creates and maintains this property. It is only used internally by the WIA service.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
