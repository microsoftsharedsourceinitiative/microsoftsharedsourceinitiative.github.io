WIA_IPS_ORIENTATION

The WIA_IPS_ORIENTATION property contains the current orientation setting. An application sets the WIA_IPS_ORIENTATION property to define the original orientation of a page or image to be acquired. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Value Definition
LANDSCAPE 90 degrees clockwise rotation, relative to PORTRAIT orientation
PORTRAIT 0 degrees
ROT180 180 degrees clockwise rotation, relative to PORTRAIT orientation
ROT270 270 degrees clockwise rotation, relative to PORTRAIT orientation

Headers

Declared in wiadef.h. Include wiadef.h.
