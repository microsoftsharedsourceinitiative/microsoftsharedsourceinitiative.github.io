WIA_IPA_FORMAT

The WIA_IPA_FORMAT property contains the current format of the image about to be transferred. An application reads this property to determine the format of the image that it is about to receive. An application writes this property to set the format. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Comments

If the device can be set to only a single value, create a WIA_PROP_LIST type, and place the valid value in it.

Headers

Declared in wiadef.h. Include wiadef.h.
