WIA_DPS_MIN_HORIZONTAL_SHEET_FEED_SIZE

The WIA_DPS_MIN_HORIZONTAL_SHEET_FEED_SIZE property contains the physical dimensions of the smallest page that the scanner’s document feeder can scan, in 1000ths of an inch. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
