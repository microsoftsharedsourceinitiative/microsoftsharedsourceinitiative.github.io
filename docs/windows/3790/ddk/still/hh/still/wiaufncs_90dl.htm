SKIP_AMOUNT

The SKIP_AMOUNT enumeration is used to indicate whether the file and informational headers of an image should be skipped over.

typedef enum {
  SKIP_OFF,
  SKIP_FILEHDR,
  SKIP_BOTHHDR
} SKIP_AMOUNT;

Enumerators

SKIP_OFF
Do not skip over either header.
SKIP_FILEHDR
Skip over the file header.
SKIP_BOTHHDR
Skip over both the file and info headers.

Headers

Declared in wiautil.h. Include wiautil.h and wiamindr.h.

Comments

This enumeration is available in Windows XP and later.
