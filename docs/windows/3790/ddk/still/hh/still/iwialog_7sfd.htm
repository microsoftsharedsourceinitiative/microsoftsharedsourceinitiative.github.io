IWiaLog::InitializeLog

The IWiaLog::InitializeLog method initializes the lWiaLog interface.

HRESULT
  InitializeLog(
    LONG  hInstance
    );

Parameters

hInstance
Specifies the module handle. This parameter indicates which module is calling the method.

Return Value

If the method succeeds, it returns S_OK. If the method fails, it returns a standard COM error code.

Headers

Declared in wia.h. Include wia.h.

Comments

The minidriver should use CoCreateInstance or CoCreateInstanceEx (documented in the Platform SDK documentation) to obtain the IWiaLog interface.

This method is available in Windows Me and in Windows XP and later.
