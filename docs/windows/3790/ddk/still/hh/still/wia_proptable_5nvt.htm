WIA_DPC_SHARPNESS

The WIA_DPC_SHARPNESS property indicates the perceived sharpness of captured images. This property can use either a list or a range of values. The minimum value is used to represent the least amount of sharpness, while the maximum value represents maximum sharpness. Typically a value in the middle of the range represents normal (default) sharpness.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST or WIA_PROP_RANGE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
