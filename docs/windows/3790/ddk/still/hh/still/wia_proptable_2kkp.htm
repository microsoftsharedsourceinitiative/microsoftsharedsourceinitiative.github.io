WIA_DIP_DEV_DESC

The WIA_DIP_DEV_DESC property contains the device description string for the WIA minidriver. The device description string is obtained from the INF file. An application reads this property to get a description of the device. The WIA service creates and maintains this property.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
