WIA_DPS_VERTICAL_BED_REGISTRATION

The WIA_DPS_VERTICAL_BED_REGISTRATION property contains the registration, or vertical alignment and edge detection, for documents placed on the flatbed. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Value Definition
BOTTOM_JUSTIFIED The paper is bottom-justified.
CENTERED The paper is centered.
TOP_JUSTIFIED The paper is top-justified.

Headers

Declared in wiadef.h. Include wiadef.h.
