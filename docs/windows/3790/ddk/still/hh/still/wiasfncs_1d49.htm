wiasWritePropFloat

The wiasWritePropFloat function writes a single floating point property value to a WIA item.

HRESULT
  wiasWritePropFloat(
    BYTE  *pWiasContext,
    PROPID  propid,
    FLOAT  fVal
    );

Parameters

pWiasContext
A Pointer to a WIA item context.
propid
Specifies the property identifier.
fVal
Specifies the floating point property value to write to the item.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
