wiauRegOpenData

The wiauRegOpenData function opens the DeviceData registry key.

HRESULT
  wiauRegOpenData(
    HKEY  hkeyAncestor,
    HKEY  *phkeyDeviceData
    );

Parameters

hkeyAncestor
Specifies the key handle of the parent. This parameter should be set to the value used in the call to the IStiUSD::Initialize method.
phkeyDeviceData
Pointer to a memory location that receives the opened key handle.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error.

Headers

Declared in wiautil.h. Include wiautil.h.

Comments

This function is available in Windows XP and later.

Call this function only in the STI IStiUSD::Initialize method. Call RegCloseKey (described in the Platform SDK documentation) when you are finished with the registry.
