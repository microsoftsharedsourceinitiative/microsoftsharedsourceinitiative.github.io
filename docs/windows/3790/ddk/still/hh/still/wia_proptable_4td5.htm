WIA_DPA_DEVICE_TIME

The WIA_DPA_DEVICE_TIME property contains the current clock time that is stored on the device. When this property is read, the minidriver should check the device's current clock time, and should always return the current time. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
