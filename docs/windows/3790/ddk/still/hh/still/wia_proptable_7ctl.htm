WIA_DPS_HORIZONTAL_BED_REGISTRATION

The WIA_DPS_HORIZONTAL_BED_REGISTRATION property contains the registration, or horizontal alignment, for documents placed on the flatbed. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Value Definition
CENTERED The paper is centered.
LEFT_JUSTIFIED The paper is left justified.
RIGHT_JUSTIFIED The paper is right justified.

Headers

Declared in wiadef.h. Include wiadef.h.
