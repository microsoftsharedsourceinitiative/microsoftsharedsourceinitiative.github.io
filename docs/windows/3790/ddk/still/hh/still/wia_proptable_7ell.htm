WIA_IPA_FILENAME_EXTENSION

The WIA_IPA_FILENAME_EXTENSION property contains the proper file extension needed to define the current setting of the WIA_IPA_FORMAT property. An application uses this information to append the proper file extension for WIA_IPA_FORMAT types that it does not understand. For example, if the XYZ development team created a WIA driver that transferred a new file format, they could specify a file extension of “XYZ”. This allows applications to transfer data in that format to a file, and to create a file name such as myfile.xyz that is useful to others who understand the new extension. This property is created and maintained by the minidriver.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
