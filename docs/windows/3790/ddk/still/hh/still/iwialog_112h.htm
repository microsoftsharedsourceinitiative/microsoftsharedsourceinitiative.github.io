WIAS_LWARNING

The WIAS_LWARNING macro writes a diagnostic WIA_WARNING message to the log file.

WIAS_LWARNING(
  IWiaLog  *pIWiaLog,
  LONG  lResId,
  (CONST CHAR  *format_string, ...)
  );

Parameters

pIWiaLog
Points to an IWiaLog interface.
lResId
Specifies the resource ID. This value should be set to WIALOG_NO_RESOURCE_ID.
format_string, ...
Specifies a variable argument list, which starts with an ANSI format string describing the message and any format identifiers. The ... specifies a variable number of arguments that need to be output. The error text should be prefixed with the full name of the method or function generating the message in the format of "class::method, error-text".

Headers

Declared in wia.h. Include wia.h.

Comments

The following is an example of how the macro would be used:

WIAS_LWARNING(g_pIWiaLog, WIALOG_NO_RESOURCE_ID,
  ("MyClass::MyMethod, This is my text and my lValue = %d", lValue));

This method is available in Windows Me and in Windows XP and later.
