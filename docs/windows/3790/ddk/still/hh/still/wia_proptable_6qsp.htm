WIA_DPS_SCAN_AHEAD_PAGES

The WIA_DPS_SCAN_AHEAD_PAGES property contains a value that indicates whether the scanner will cache pages in a scanner buffer before sending them to the application.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE of zero to the maximum number of pages that the document feeder can hold.

Access Rights: READ/WRITE

Note  A value of zero disables scan ahead, and no pages will be scanned ahead. Doing normal data transfers on the buffered scan-ahead item retrieves the buffered pages. WIA properties cannot be changed during a scan-ahead operation. This property is optional.
