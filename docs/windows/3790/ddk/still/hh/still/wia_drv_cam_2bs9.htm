Storing and Transferring Audio Data

Some WIA drivers for Windows Me and Windows XP used the following three WIA properties to store audio data:

WIA_IPC_AUDIO_AVAILABLE

WIA_IPC_AUDIO_DATA_FORMAT

WIA_IPC_AUDIO_DATA

These properties are obsolete and should no longer be used.

Audio for a picture item should be represented as an attachment. This provides easy access to all audio formats supported by the WIA minidriver. Audio content is transferred in the same way that other items in the WIA item tree are transferred.
