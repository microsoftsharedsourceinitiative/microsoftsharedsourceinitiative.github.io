WIA_IPA_DEPTH

The WIA_IPA_DEPTH property contains the bit depth setting of an image. An application reads this property to determine the bit depth setting of the image. The application might also have the ability to set this value to the desired bit depth. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Comments

If the device can be set to only a single value, create a WIA_PROP_LIST type, and place the valid value in it.

Headers

Declared in wiadef.h. Include wiadef.h.
