WIA_IPA_ACCESS_RIGHTS

The WIA_IPA_ACCESS_RIGHTS property contains the access rights for the WIA item. Access rights control the ability of the application to delete items in the WIA item tree. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READ/WRITE or READONLY, depending on the item's ability to have its access rights changed.

Flag Value Definition
WIA_ITEM_CAN_BE_DELETED This item can be deleted.
WIA_ITEM_READ Access to the item is read-only.
WIA_ITEM_WRITE Access to the item is read/write.
WIA_ITEM_RD WIA_ITEM_READ | WIA_ITEM_CAN_BE_DELETED
WIA_ITEM_RWD WIA_ITEM_READ | WIA_ITEM_WRITE | WIA_ITEM_CAN_BE_DELETED
