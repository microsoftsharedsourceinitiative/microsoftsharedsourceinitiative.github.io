WIA_IPC_THUMBNAIL

The WIA_IPC_THUMBNAIL property contains the 24-bits-per-pixel thumbnail data bits. The thumbnail data must be in uncompressed bitmap data and aligned on 32-bit boundaries. An application reads the values of the WIA_IPC_THUMBNAIL_WIDTH and WIA_IPC_THUMBNAIL_HEIGHT properties and creates a BITMAPINFOHEADER structure (described in the Platform SDK documentation). The application should be able to read the WIA_IPC_THUMBNAIL property (which represents the actual thumbnail image data), and use this property to write those bits directly into the newly created bitmap to create the thumbnail image. This property is used by WIA minidrivers and applications running on Windows Me and Windows XP and later operating systems. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
