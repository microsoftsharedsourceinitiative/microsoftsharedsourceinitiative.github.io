WIA_DPA_FIRMWARE_VERSION

The WIA_DPA_FIRMWARE_VERSION property contains the device firmware version. This value must be a string value, such as “1.0.4” or “1.0abc”.The minidriver creates and maintains this property.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
