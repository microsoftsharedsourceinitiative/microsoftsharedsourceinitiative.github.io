wiasCreateDrvItem

The wiasCreateDrvItem function creates an IWiaDrvItem object.

HRESULT
  wiasCreateDrvItem(
    LONG  lObjectFlags,
    BSTR  bstrItemName,
    BSTR  bstrFullItemName,
    IWiaMiniDrv  *pIMiniDrv,
    LONG  cbDevSpecContext,
    BYTE  **ppDevSpecContext,
    IWiaDrvItem  **ppIWiaDrvItem
    );

Parameters

lObjectFlags
Specifies the object item type, which must be WiaItemTypeFolder or WiaItemTypeFile (possibly the bitwise OR of these). These flags are described in the Platform SDK documentation.
bstrItemName
Specifies a string that contains the item name without path information.
bstrFullItemName
Specifies a string that contains the item name with path information.
pIMiniDrv
Pointer to the IWiaMiniDrv interface of the current minidriver.
cbDevSpecContext
Specifies the size in bytes of the device specific context.
ppDevSpecContext
Pointer to a memory location that receives the address of the device specific context. Set this to NULL if the information is not needed.
ppIWiaDrvItem
Pointer to a memory location that receives the address of an IWiaDrvItem interface for the newly created IWiaDrvItem object.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function creates and initializes an IWiaDrvItem object with the specified name and attributes. It also creates a context for the IWiaDrvItem object. Minidrivers typically use this function to build a tree of device items.

This function is available in Windows Me and in Windows XP and later.
