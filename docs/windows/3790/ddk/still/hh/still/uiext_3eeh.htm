IWiaUIExtension::GetDeviceBitmapLogo

The IWiaUIExtension::GetDeviceBitmapLogo method gets a custom bitmap logo for the device.

HRESULT
  GetDeviceBitmapLogo(
    THIS_
    IN BSTR  bstrDeviceId,
    OUT HBITMAP  *phBitmap,
    IN ULONG  nMaxWidth,
    IN ULONG  nMaxHeight
    ) PURE;

Parameters

bstrDeviceId
Specifies the device ID of the device that is to obtain the bitmap logo.
phBitmap
Points to a memory location that will receive a handle for the bitmap logo for the device.
nMaxWidth
Specifies the desired width of the bitmap.
nMaxHeight
Specifies the desired height of the bitmap.

Return Value

If the method succeeds, it returns S_OK. If the method fails, it returns a standard COM error code.

Headers

Declared in wiadevd.h. Include wiadevd.h.

Comments

This method is available in Windows Me and in Windows XP and later.
