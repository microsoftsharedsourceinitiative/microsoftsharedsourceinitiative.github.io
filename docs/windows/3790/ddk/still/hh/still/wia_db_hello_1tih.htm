"Hello World" Definition File

The definition file is used to export entry point functions. The hellowld.def file should contain the two following COM exports, DllGetClassObject and DllCanUnloadNow, which are described in the Platform SDK documentation.

LIBRARY HELLOWLD

EXPORTS
        DllGetClassObject   PRIVATE
        DllCanUnloadNow     PRIVATE
