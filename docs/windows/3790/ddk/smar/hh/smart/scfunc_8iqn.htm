SmartcardInvertData (VxD)

The SmartcardInvertData function converts the passed buffer from inverse to direct or vice versa.

This function performs the same for WDM and VxD drivers.

VOID 
  SmartcardInvertData(
    PUCHAR  Buffer,
    ULONG  Length
    );

Parameters

Buffer
Pointer to the buffer for which the data will be inverted.
Length
Length of the buffer, in characters.

Return Value

None

Headers

Declared in smclib.h. Include smclib.h.
