SmartcardExit (WDM)

The SmartcardExit function frees buffers that were allocated during a call to SmartcardInitialize (WDM).

VOID 
  SmartcardExit(
    IN PSMARTCARD_EXTENSION  SmartcardExtension
    );

Parameters

SmartcardExtension
Pointer to a structure of type SMARTCARD_EXTENSION that contains the device extension of the smart card device.

Return Value

None

Headers

Declared in smclib.h. Include smclib.h.

Comments

The library allocates internal buffers during a call to SmartcardInitialize (WDM). SmartcardExit frees these buffers.

SmartcardExit functions the same for both WDM and VxD drivers.
