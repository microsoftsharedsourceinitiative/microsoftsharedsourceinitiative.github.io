SmartcardReleaseRemoveLockAndWait (WDM)

SmartcardReleaseRemoveLockAndWait ensures that no unload code is executed as long as some other driver code is executing. 

VOID 
  SmartcardReleaseRemoveLockAndWait(
    PSMARTCARD_EXTENSION  SmartcardExtension
    );

Parameters

SmartcardExtension
Pointer to a structure of type SMARTCARD_EXTENSION that contains the device extension of the smart card device.

Return Value

None

Headers

Declared in smclib.h. Include smclib.h.

Comments

SmartcardReleaseRemoveLockAndWait should be called from the driver's Unload function to force the unload operation to wait until all other threads of the driver code have finished executing.
