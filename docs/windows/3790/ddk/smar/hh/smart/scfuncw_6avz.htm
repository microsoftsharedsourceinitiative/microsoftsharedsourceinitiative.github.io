SmartcardAcquireRemoveLock(WDM)

The SmartcardAcquireRemoveLock function is used to increment a reference count to the driver.

NTSTATUS 
  SmartcardAcquireRemoveLock(
    IN PSMARTCARD_EXTENSION  SmartcardExtension
    );

Parameters

SmartcardExtension
Pointer to a structure of type SMARTCARD_EXTENSION that contains the device extension of the smart card device. The extension contains the reference counter to be incremented.

Return Value

SmartcardAcquireRemoveLock returns an NTSTATUS value. Possible values are the following.

Value Meaning
STATUS_SUCCESS Lock was successfully acquired.
STATUS_DELETE_PENDING The driver is about to unload.

Headers

Declared in smclib.h. Include smclib.h.

Comments

If the driver is about to unload. In this case, no further code should be executed. Instead, the function that called SmartcardAcquireRemoveLock should return immediately. SmartcardAcquireRemoveLock should be called whenever an entry point function to the driver is called (for example, DeviceIoControl). This function ensures that the driver does not unload while some other driver code is being executed.
