IOCTL_SMARTCARD_EJECT

WDM driver version

Operation

The IOCTL_SMARTCARD_EJECT request ejects the currently inserted smart card from the smart card reader.

Input

None

Output

None

I/O Status Block

Information is set to zero. Status can be one of the following values.

Status Meaning
STATUS_SUCCESS Card successfully ejected.
STATUS_NO_MEDIA No card is in the reader.
STATUS_NOT_SUPPORTED Reader does not support card eject.
STATUS_IO_TIMEOUT Operation has timed out.

Headers

Defined in winsmcrd.h. Include winsmcrd.h.
