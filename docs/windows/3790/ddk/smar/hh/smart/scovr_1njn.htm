Smart Card Devices: Windows DDK

Sample Smart Card Reader Drivers

The Windows DDK includes the source code for the following drivers in the \src\smartcrd directory tree of the Windows DDK:

  • Bull TLP3 smart card reader driver sample (bulltlp3.sys)
  • PCMCIA smart card reader driver sample (pscr.sys)

These samples are binary-compatible between Windows 95/98/Me and NT-based versions of the operating system, and will remain compatible, even when altered, provided no Windows 95/98/Me VxD calls nor any NT-based system calls are added to them.
