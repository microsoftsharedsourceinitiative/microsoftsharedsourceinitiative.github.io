State Block Recording and Pure Devices

State block handling is different for a device operating in pure device mode. In this mode, the state block control DP2 token (D3DDP2OP_STATESET) is sent to the driver with a new operation type (in the dwOperations field). This new operation type is D3DHAL_STATESETCREATE.
