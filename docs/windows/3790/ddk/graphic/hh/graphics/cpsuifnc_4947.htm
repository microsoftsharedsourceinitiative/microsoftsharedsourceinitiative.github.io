CPSFUNC_SET_DMPUB_HIDEBITS

The CPSFUNC_SET_DMPUB_HIDEBITS function code causes the ComPropSheet function to "hide" a specified set of document property options, so they are not displayed.

ComPropSheet Parameters

hComPropSheet
Group parent handle.
Function
CPSFUNC_SET_DMPUB_HIDEBITS
lParam1
Bit array indicating the options to be hidden. This array must be created using the MAKE_DMPUB_HIDEBIT(DMPub) macro, where DMPub is one or more DMPUB_-prefixed constants (ORed together). The DMPUB_-prefixed constants are listed in the description of the OPTITEM structure. The macro and constants are defined in compstui.h.
lParam2
Not used. Must be zero.

ComPropSheet Return Value

If the operation succeeds, ComPropSheet returns the value specified for lParam1; otherwise it returns zero.

Headers

Declared in compstui.h. Include compstui.h.

Comments

Use the CPSFUNC_SET_DMPUB_HIDEBITS function code if you want to define OPTITEM structures for one or more document property sheet options, but you do not want the options to be user-modifiable. The property sheet page must be defined using the COMPROPSHEETUI structure, and the structure's pDlgPage member must be CPSUI_PDLGPAGE_DOCPROP or CPSUI_PDLGPAGE_ADVDOCPROP.

If this function code is used, it must be specified to ComPropSheet before the CPSFUNC_ADD_PCOMPROPSHEETUI or CPSFUNC_INSERT_PSUIPAGE function code is used to create the page.
