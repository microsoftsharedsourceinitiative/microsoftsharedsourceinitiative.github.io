SNMPDevice

The SNMPDevice property allows an ASP Web page to obtain a printer's SNMP device index (as defined by RFC 1759).

[propget, id(5), helpstring("property SNMPDevice")]
  HRESULT
    SNMPDevice(
    [out, retval] DWORD  *pVal
    );

Parameters

pVal
Caller-supplied location to receive a numeric value representing the printer's SNMP device index.

Return Value

S_OK The operation succeeded.
E_HANDLE The IAspHelp Open method has not been called.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

The name of the printer being queried must have been specified by a previous call to the IAspHelp Open method.

VBScript Example

    Dim objPrinter, SNMPDeviceIndex
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    objPrinter.Open strPrinter
    SNMPDeviceIndex = objPrinter.SNMPDevice
