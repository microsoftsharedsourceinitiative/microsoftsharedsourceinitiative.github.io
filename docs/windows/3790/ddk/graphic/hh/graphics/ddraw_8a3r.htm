Flagging Support for DMA Model Nonlocal Display Memory

In addition to specifying the DDCAPS2_NONLOCALVIDMEM flag to report AGP support, a DMA model driver must export the capability flag DDCAPS2_NONLOCALVIDMEMCAPS. This flag indicates that nonlocal (AGP) memory has different capabilities than local display memory.
