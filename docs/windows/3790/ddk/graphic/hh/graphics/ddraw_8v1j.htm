Windows Drop-down Menus

This topic applies only to Windows 98/Me.

When Windows 98/Me uses a drop-down menu, it saves the area under it in a special part of display memory so that it can restore it quickly. A flag (the RC_SAVEBITMAP bit in the dpRaster member of the GDIINFO structure) must be set to stop Windows 98/Me from saving the area under the menu. The area also can be stored elsewhere while the game is running, and restored to its original condition before Windows 98/Me takes control of the screen again, but this is not mandatory. For information on the Windows 98/Me version of the GDIINFO structure, see the Windows 98/Me section of the Windows DDK documentation.
