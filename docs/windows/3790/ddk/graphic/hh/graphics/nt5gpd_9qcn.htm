Printer Font Descriptions

The GPD language provides entries for describing hardware-resident fonts, font cartridges, and font substitution of TrueType fonts for similar hardware-resident or cartridge fonts.

You can optionally provide customized code for handling soft fonts. For information about this customized code, see Customized Font Management.
