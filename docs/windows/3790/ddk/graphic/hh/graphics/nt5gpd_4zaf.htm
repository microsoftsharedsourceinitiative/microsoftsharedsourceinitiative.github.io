Standard Options

Standard options are those that are associated with standard features. They are identified by predefined names that the GPD language recognizes. (Resource identifiers for strings representing these names are contained in stdnames.gpd, which is supplied with this DDK.)

The following table lists the standard option names that are permitted for each standard feature. Use these names as arguments to *Option entries.
