IPrintOemUni::PublishDriverInterface

The IPrintOemUni::PublishDriverInterface method allows a rendering plug-in for Unidrv to obtain the Unidrv driver's IPrintOemDriverUni interface.

STDMETHOD
  (PublishDriverInterface) (
    THIS_
    IUnknown  *pIUnknown
    ) PURE;

Parameters

pIUnknown
Caller-supplied pointer to the IUnknown interface of the driver's IPrintOemDriverUni COM interface.

Return Value

The method must return one of the following values:

Return Value Definition
S_OK The operation succeeded.
E_FAIL The operation failed.

Headers

Declared in prcomoem.h. Include prcomoem.h.

Comments

A rendering plug-in for Unidrv must implement the IPrintOemUni::PublishDriverInterface method and the method must return S_OK, or the driver will not call the plug-in's other IPrintOemUni interface methods.

If the rendering plug-in will be calling IPrintOemDriverUni interface methods, it must use the received IUnknown interface pointer to call IUnknown::QueryInterface (described in the Platform SDK documentation) in order to obtain a pointer to the driver's supported version of the IPrintOemDriverUni interface. For more information, see Interface Identifiers for Printer Drivers.
