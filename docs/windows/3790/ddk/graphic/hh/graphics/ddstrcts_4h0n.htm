DDGAMMARAMP

The DDGAMMARAMP structure contains red, green, and blue ramp data for the IDirectDrawGammaControl::GetGammaRamp and IDirectDrawGammaControl::SetGammaRamp methods.

typedef struct _DDGAMMARAMP {
  WORD  red[256];
  WORD  green[256];
  WORD  blue[256];
} DDGAMMARAMP, FAR * LPDDGAMMARAMP;

Members

red
Array of 256 WORD elements that describe the red gamma.
green
Array of 256 WORD elements that describe the green gamma.
blue
Array of 256 WORD elements that describe the blue gamma.
