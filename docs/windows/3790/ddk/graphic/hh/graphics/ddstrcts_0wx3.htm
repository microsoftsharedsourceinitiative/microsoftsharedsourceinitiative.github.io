DDGETKERNELCAPSOUT

The DDGETKERNELCAPSOUT structure contains the capabilities of the DirectDraw object. 

typedef struct _DDGETKERNELCAPSOUT {
  DWORD  ddRVal;
  DWORD  dwCaps;
  DWORD  dwIRQCaps;
} DDGETKERNELCAPSOUT, FAR *LPDDGETKERNELCAPSOUT;

Members

ddRVal
Specifies the location in which DirectDraw writes the return value. A return code of DD_OK indicates success.
dwCaps
Can be any combination of the capabilities in the dwCaps member of the DDKERNELCAPS structure.
dwIRQCaps
Can be a combination of the flags in the dwIRQCaps member of DDKERNELCAPS.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
