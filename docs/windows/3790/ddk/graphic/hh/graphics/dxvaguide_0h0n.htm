Frame-Rate Conversion Modes

Following are examples of the frame-rate conversion modes that can be supported by the DDI.

Mode Description
Frame Repeat/Drop This is not a recommended mode, because it uses extra memory by copying the selected source sample into the destination surface.
Linear Temporal Interpolation A future and a previous reference field are alpha blended together to produce a new frame.
Motion Vector Steered Motion vectors of the different objects in a scene are used to align individual movements to the time axis before interpolation takes place.
