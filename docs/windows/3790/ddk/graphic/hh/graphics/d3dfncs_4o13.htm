CPixel::ComputeSurfaceSize

The CPixel::ComputeSurfaceSize method determines the amount of memory required to allocate a surface. 

static UINT 
  ComputeSurfaceSize(
    UINT  cpWidth,
    UINT  cpHeight,
    D3DFORMAT  Format
    );

Parameters

cpWidth
Specifies the width of the surface.
cpHeight
Specifies the height of the surface.
Format
Uses a value from the D3DFORMAT enumeration to specify the surface format.

Return Value

Returns the size, in bytes, of the surface.

Headers

Declared in pixel.hpp. Include pixel.hpp.

Comments

For more information about D3DFORMAT, see the DirectX SDK documentation.
