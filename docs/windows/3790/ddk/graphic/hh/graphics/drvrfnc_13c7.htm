Display and Print Devices: Windows DDK

DrvSplAbort

The DrvSplAbort function enables a user-mode rendering DLL to respond to a request to cancel a print job.

VOID
  DrvSplAbort(
    HANDLE  hDriver
    );

This function is obsolete, and is supported only so that user-mode rendering DLLs, optionally supplied with Windows NT 4.0 drivers, will execute under Windows 2000 and later. To supply user-mode image rendering code for printers, see Choosing User Mode or Kernel Mode.
