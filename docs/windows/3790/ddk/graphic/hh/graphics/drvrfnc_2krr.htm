OEMDevMode

This function is obsolete for Windows XP and later. It is supported only for legacy plug-ins. For UI plug-ins, use IPrintOemUI::DevMode; for Pscript plug-ins, use IPrintOemPS::DevMode; for Unidrv plug-ins, use IPrintOemUni::DevMode.
