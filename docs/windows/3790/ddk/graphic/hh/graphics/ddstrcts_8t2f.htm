DDHAL_DDMISCELLANEOUSCALLBACKS

The DDHAL_DDMISCELLANEOUSCALLBACKS structure contains entry pointers to miscellaneous callback functions.

typedef struct _DDHAL_DDMISCELLANEOUSCALLBACKS {
  DWORD  dwSize;
  DWORD  dwFlags;
  LPDDHAL_GETAVAILDRIVERMEMORY  GetAvailDriverMemory;
  LPDDHAL_UPDATENONLOCALHEAP  UpdateNonLocalHeap;
  LPDDHAL_GETHEAPALIGNMENT  GetHeapAlignment;
  LPDDHALSURFCB_GETBLTSTATUS  GetSysmemBltStatus;
} DDHAL_DDMISCELLANEOUSCALLBACKS, *LPDDHAL_DDMISCELLANEOUSCALLBACKS;

Members

dwSize
Specifies the size in bytes of this DDHAL_DDMISCELLANEOUSCALLBACKS structure.
dwFlags
Indicates what DirectDraw miscellaneous callbacks the driver has implemented. For every bit set in dwFlags, the driver must initialize the corresponding function pointer member of this structure. This member can be one or more of the following flags:

DDHAL_MISCCB32_GETAVAILDRIVERMEMORY
DDHAL_MISCCB32_UPDATENONLOCALHEAP
DDHAL_MISCCB32_GETHEAPALIGNMENT
DDHAL_MISCCB32_GETSYSMEMBLTSTATUS

GetAvailDriverMemory
Points to the driver-supplied DdGetAvailDriverMemory callback.
UpdateNonLocalHeap
Points to the driver-supplied DdUpdateNonLocalHeap callback.
GetHeapAlignment
Points to the driver-supplied DdGetHeapAlignment callback.
GetSysmemBltStatus
Points to the driver-supplied DdGetSystemBltStatus callback.

Headers

Defined in ddrawi.h. Include ddrawi.h.

Comments

Entries that the display driver does not use should be set to NULL. The driver should initialize this structure when its DdGetDriverInfo function is called with the GUID_MiscellaneousCallbacks GUID.

The callback to which the GetSysmemBltStatus member points uses the same prototype as the DdGetBltStatus callback. Both GetSysmemBltStatus and the GetBltStatus member of the DDHAL_DDSURFACECALLBACKS structure can point to the same driver callback function.
