DDRAWI_DDRAWPALETTE_GBL

The DDRAWI_DDRAWPALETTE_GBL structure contains the global DirectDrawPalette data that can be shared between object instances. 

typedef struct _DDRAWI_DDRAWPALETTE_GBL {
  DWORD  dwRefCnt;
  DWORD  dwFlags;
  LPDDRAWI_DIRECTDRAW_LCL  lpDD_lcl;
  DWORD  dwProcessId;
  LPPALETTEENTRY  lpColorTable;
  union {
     ULONG_PTR  dwReserved1;
     HPALETTE  hHELGDIPalette;
  };
  DWORD  dwDriverReserved;
  DWORD  dwContentsStamp;
  DWORD  dwSaveStamp;
  DWORD  dwHandle;
} DDRAWI_DDRAWPALETTE_GBL;

Members

dwRefCnt
Used by DirectDraw and should not be filled in by the driver.
dwFlags
Specifies a set of palette control flags.
Flag Meaning
DDRAWIPAL_2 The palette has 2 entries.
DDRAWIPAL_4 The palette has 4 entries.
DDRAWIPAL_16 The palette has 16 entries.
DDRAWIPAL_256 The palette has 256 entries.
DDRAWIPAL_ALLOW256 The palette can be fully updated.
DDRAWIPAL_ALPHA The entries in the palette contain alpha information.
DDRAWIPAL_DIRTY This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWIPAL_EXCLUSIVE This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWIPAL_GDI The palette is allocated through GDI.
DDRAWIPAL_INHEL This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWIPAL_STORED_8 The palette is stored as 8 bpp per entry.
DDRAWIPAL_STORED_8INDEX The palette is stored as an 8-bit index into the destination palette.
DDRAWIPAL_STORED_16 The palette is stored as 16 bpp per entry.
DDRAWIPAL_STORED_24 The palette is stored as 24 bpp per entry.

lpDD_lcl
Used by DirectDraw and should not be filled in by the driver.
dwProcessId
Used by DirectDraw and should not be filled in by the driver.
lpColorTable
Specifies the address of an array of PALETTEENTRY structures that contain the RGB values that are the colors in the palette.
dwReserved1
Reserved for use by the display driver.
hHELGDIPalette
Used by DirectDraw and should not be filled in by the driver.
dwDriverReserved
Reserved for use by the HAL, regardless of who created the object.
dwContentsStamp
Incremented when the palette changes.
dwSaveStamp
Set to dwContentsStamp when the palette is saved to a surface file.
dwHandle
Handle used in D3dDrawPrimitives2 palette notification.
