USBPRINT Interface

Usbprint.sys is the Microsoft®-provided kernel-mode device driver for USB printers. Usbprint.sys works with usbmon.dll to provide end-to-end connectivity between USB printers and high-level printer drivers.

Unlike some USB device classes, usbprint.sys does not “drive” the printer. Instead, usbprint.sys provides a communication conduit by which higher-level drivers can control the printer. As is true for parallel printers, USB printers require a printer driver to render print jobs, and also may require a language monitor to manage high-level communication with the printer.

For general information about USB drivers, see Introduction to USB Drivers.
