GDI as a Rendering Engine

For rendering operations, the driver must first enable a surface for each PDEV structure that is enabled. A PDEV is a logical representation of a physical device. If the hardware can be set up as a GDI standard-format bitmap, GDI can be used to do some or all of the drawing to the bitmap surface. GDI can also handle advanced halftoning.

For information on enabling PDEVs and surfaces, refer to the DrvEnablePDEV and DrvEnableSurface functions.
