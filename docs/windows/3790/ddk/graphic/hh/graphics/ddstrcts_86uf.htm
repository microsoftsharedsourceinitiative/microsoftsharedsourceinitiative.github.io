DD_CANCREATEVPORTDATA

The DD_CANCREATEVPORTDATA structure contains the information required for the driver to determine whether a video port extensions (VPE) object can be created.

typedef struct _DD_CANCREATEVPORTDATA {
  PDD_DIRECTDRAW_LOCAL  lpDD;
  LPDDVIDEOPORTDESC  lpDDVideoPortDesc;
  HRESULT  ddRVal;
  VOID  *CanCreateVideoPort;
} DD_CANCREATEVPORTDATA;

Members

lpDD
Points to a DD_DIRECTDRAW_LOCAL structure that describes the DirectDraw API.
lpDDVideoPortDesc
Points to a DDVIDEOPORTDESC structure that contains a description of the VPE object being requested.
ddRVal
Specifies the location in which the driver writes the return value of the DdVideoPortCanCreate callback. A return code of DD_OK indicates success.
CanCreateVideoPort
Used by the DirectDraw API and should not be filled in by the driver.

Headers

Defined in ddrawint.h. Include winddi.h.
