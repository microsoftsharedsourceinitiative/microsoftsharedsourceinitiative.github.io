OEMLineTo

The OEMLineTo function draws a single, solid, integer-only cosmetic line.

BOOL APIENTRY
  OEMLineTo(
    SURFOBJ  *pso,
    CLIPOBJ  *pco,
    BRUSHOBJ  *pbo,
    LONG  x1,
    LONG  y1,
    LONG  x2,
    LONG  y2,
    RECTL  *prclBounds,
    MIX  mix
    );

Headers

Declared in printoem.h. Include printoem.h.

Comments

See DrvLineTo.

Do not directly hook out this drawing function. Instead, implement OEMEnableDriver so that it fills in a DRVENABLEDATA structure listing all of the drawing DDIs to be hooked out.

If you call into the core driver, cast the call using the PFN_DrvLineTo function pointer.
