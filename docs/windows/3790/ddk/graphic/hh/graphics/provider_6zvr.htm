Managing a Port

Port management activities consist of adding a port, configuring a port, and deleting a port in response to requests from applications (such as the print folder). These port management activities are handled by port monitors, not language monitors, and require Administrator privilege. There are special rules for storing port configuration information.

One additional, optional management activity involves Setting Port Timeout Values.
