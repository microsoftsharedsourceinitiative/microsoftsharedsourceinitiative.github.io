ToUnicode

The ToUnicode property allows an ASP Web page to convert one Unicode string to another using a specified code page.

[propget, id(4), helpstring("property ToUnicode")] 
  HRESULT 
    ToUnicode(
    [in] BSTR  bstrString,
    [in] long  lCodePage,
    [out, retval] BSTR  *pVal
    );

Parameters

bstrString
Caller-supplied string to be converted.
lCodePage
Caller-supplied code page to use for the conversion.
pVal
Caller-supplied pointer to a location to receive the converted Unicode string.

Return Value

S_OK The operation succeeded.
E_POINTER pVal does not point to a valid memory location.

VBScript Example

Convert using Japanese code page, if applicable
If strLang = "JP" Then
    tmpStr = OleCvt.ToUnicode (str, 932)
Else
    tmpStr = str
End If
