OEMRealizeBrush

The OEMRealizeBrush function requests that the driver realize a specified brush for a specified surface.

BOOL APIENTRY
  OEMRealizeBrush(
    BRUSHOBJ  *pbo,
    SURFOBJ  *psoTarget,
    SURFOBJ  *psoPattern,
    SURFOBJ  *psoMask,
    XLATEOBJ  *pxlo,
    ULONG  iHatch
    );

Headers

Declared in printoem.h. Include printoem.h.

Comments

See DrvRealizeBrush.

Do not directly hook out this drawing function. Instead, implement OEMEnableDriver so that it fills in a DRVENABLEDATA structure listing all of the drawing DDIs to be hooked out.

If you call into the core driver, cast the call using the PFN_DrvRealizeBrush function pointer.
