OEMQueryFontData

The OEMQueryFontData function retrieves information about a realized font. 

LONG APIENTRY
  OEMQueryFontData(
    DHPDEV  dhpdev,
    FONTOBJ  *pfo,
    ULONG  iMode,
    HGLYPH  hg,
    GLYPHDATA  *pgd,
    PVOID  pv,
    ULONG  cjSize
    );

Headers

Declared in printoem.h. Include printoem.h.

Comments

See DrvQueryFontData.

Do not directly hook out this function. Instead, implement OEMEnableDriver so that it fills in a DRVENABLEDATA structure listing all of the DDIs to be hooked out.

If you call into the core driver, cast the call using the PFN_DrvQueryFontData function pointer.
