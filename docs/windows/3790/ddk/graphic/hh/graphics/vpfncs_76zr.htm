VideoPortCreateSpinLock

The VideoPortCreateSpinLock function creates a spin lock.

VP_STATUS
  VideoPortCreateSpinLock(
    IN PVOID  HwDeviceExtension,
    OUT PSPIN_LOCK  *SpinLock
    );

Parameters

HwDeviceExtension
Pointer to the miniport driver's device extension.
SpinLock
Pointer to a memory location that will receive a pointer to the newly created spin lock. The caller must allocate the storage used to hold the spin lock.

Return Value

On success the function returns NO_ERROR. If an error occurs, the function returns an appropriate error code.

Headers

Declared in video.h. Include video.h.

Comments

This function is available in Windows XP and later.

This routine must be called before an initial call to VideoPortAcquireSpinLock or to any other support routine that requires a spin lock as an argument.

Storage for a spin lock object must be resident in the miniport driver's device extension.

Callers of this routine must be running at IRQL = PASSIVE_LEVEL.
