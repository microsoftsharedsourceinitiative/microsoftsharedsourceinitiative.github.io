CPSFUNC_SET_PSUIPAGE_ICON

The CPSFUNC_SET_PSUIPAGE_ICON function code causes the ComPropSheet function to add, replace, or remove the icon assigned a property sheet page's tab.

ComPropSheet Parameters

hComPropSheet
Group parent handle.
Function
CPSFUNC_SET_PSUIPAGE_ICON
lParam1
Caller-supplied CPSUI page handle, previously obtained by calling ComPropSheet with one of the following function codes:

CPSFUNC_ADD_HPROPSHEETPAGE

CPSFUNC_ADD_PROPSHEETPAGE

CPSFUNC_INSERT_PSUIPAGE (with the INSERTPSUIPAGE_INFO structure's Type member set to PSUIPAGEINSERT_HPROPSHEETPAGE or PSUIPAGEINSERT_PROPSHEETPAGE).

lParam2
Caller-supplied icon handle, or zero to remove the current icon.

ComPropSheet Return Value

If the operation succeeds, the ComPropSheet function returns 1. If an error is encountered, or if the specified page is not currently being displayed, the function returns zero.

Headers

Declared in compstui.h. Include compstui.h.

Comments

If lParam2 contains an icon handle, and if the page specified by lParam1 is currently being displayed, CPSUI adds the icon to the specified page's tab. If an icon is already assigned to the page, the old icon is replaced with the new one. If lParam2 is zero, the current icon (if one exists) is removed.

For all icons specified with the CPSFUNC_SET_PSUIPAGE_ICON function code, CPSUI sets the image size to 16 by 16 pixels.

Icon handles should be obtained by calling LoadImage, which is described in the Platform SDK documentation.
