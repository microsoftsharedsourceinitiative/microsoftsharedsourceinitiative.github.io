IAspHelp Close Method

The Close method allows an ASP Web page to close access to a printer.

[id(2), helpstring("method Close")]
  HRESULT
    Close(VOID);

Return Value

The return value is always S_OK.

Comments

The name of the printer being closed must have been specified by a previous call to the IAspHelp Open method.

VBScript Example

    Dim objPrinter
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    objPrinter.Open strPrinter
    ...
    objPrinter.Close
