IDirectDrawGammaControl::GetGammaRamp

The IDirectDrawGammaControl::GetGammaRamp method retrieves the red, green, and blue gamma ramps for the primary surface.

HRESULT
  IDirectDrawGammaControl::GetGammaRamp(
    THIS_
    DWORD  dwFlags,
    LPDDGAMMARAMP  lpRampData
    ) PURE;

Parameters

dwFlags
Not currently used and is set to 0.
lpRampData
Specifies the address of a DDGAMMARAMP structure to be filled with the new red, green, and blue gamma ramp entries. Each array maps color values in the frame buffer to the color values that are passed to the DAC.

Return Value

IDirectDrawGammaControl::GetGammaRamp returns DD_OK if it succeeds in retrieving the gamma ramp. If this method fails, it returns one of the following values:

  • DDERR_EXCEPTION
  • DDERR_INVALIDOBJECT
  • DDERR_INVALIDPARAMS
