Printer Options

Printer options represent the states that can be assigned to printer features. For example, if a printer can use both letter-sized and legal-sized paper, then LETTER and LEGAL are specified as options for its PaperSize feature.

This section discusses GPD language support for both standard options and customized options.

Additional topics in this section discuss the option entry format, option attributes, and option constraints.
