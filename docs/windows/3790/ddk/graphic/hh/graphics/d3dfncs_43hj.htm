CPixel::ComputeMipMapSize

The CPixel::ComputeMipMapSize method determines the amount of memory required to allocate a mipmap texture. 

static UINT 
  ComputeMipMapSize(
    UINT  cpWidth,
    UINT  cpHeight,
    UINT  cLevels,
    D3DFORMAT  Format
    );

Parameters

cpWidth
Specifies the width of the mipmap texture.
cpHeight
Specifies the height of the mipmap texture.
cLevels
Specifies the number of levels of the mipmap texture.
Format
Uses a value from the D3DFORMAT enumeration to specify the surface format.

Return Value

Returns the size, in bytes, of the mipmap texture.

Headers

Declared in pixel.hpp. Include pixel.hpp.

Comments

For more information about D3DFORMAT, see the DirectX SDK documentation.
