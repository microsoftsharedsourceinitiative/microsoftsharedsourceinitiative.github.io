DD_GETINTERNALMOCOMPDATA

The DD_GETINTERNALMOCOMPDATA structure contains the internal memory requirements. 

typedef struct _DD_GETINTERNALMOCOMPDATA {
  PDD_DIRECTDRAW_LOCAL  lpDD;
  GUID  *lpGuid;
  DWORD  dwWidth;
  DWORD  dwHeight;
  DDPIXELFORMAT  ddPixelFormat;
  DWORD  dwScratchMemAlloc;//..out
  HRESULT  ddRVal;
} DD_GETINTERNALMOCOMPDATA;

Members

lpDD
Points to a DD_DIRECTDRAW_LOCAL structure that describes the DirectDraw API.
lpGuid
Points to a GUID for which the internal memory requirement is requested.
dwWidth
Indicates the width of uncompressed output frame.
dwHeight
Indicates the height of uncompressed output frame.
ddPixelFormat
Points to a DDPIXELFORMAT structure that contains the pixel format of the uncompressed output frame.
dwScratchMemAlloc
Indicates the amount of internal memory that the display driver privately reserves to perform motion compensation
ddRVal
Specifies the location in which the driver writes the return value of the DdMoCompGetInternalInfo callback. A return code of DD_OK indicates success.

Headers

Defined in ddrawint.h. Include winddi.h.
