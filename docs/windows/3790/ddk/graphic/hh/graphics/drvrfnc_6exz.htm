IPrintCorePS2::DrvWriteSpoolBuf

The IPrintCorePS2::DrvWriteSpoolBuf method is provided by the Pscript5 driver so that rendering plug-ins can send printer data to the spooler.

STDMETHOD
  (DrvWriteSpoolBuf) (
    THIS_
    IN PDEVOBJ  pdevobj,
    IN PVOID  pBuffer,
    IN DWORD  cbSize,
    OUT DWORD  *pdwResult
    );

This method has the same behavior as IPrintOemDriverPS::DrvWriteSpoolBuf. This method is supported for any Pscript5 render plug-ins.

Headers

Declared in prcomoem.h. Include prcomoem.h.
