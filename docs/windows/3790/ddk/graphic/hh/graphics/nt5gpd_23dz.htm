Option Attributes for the OutputBin Feature

There are no attributes specifically associated with the OutputBin feature, although it is not uncommon to include an *OutputOrderReversed? entry for a specific output bin. For more information about the OutputBin feature, see Standard Features.

For information about additional option attributes, see Option Attributes for All Features.
