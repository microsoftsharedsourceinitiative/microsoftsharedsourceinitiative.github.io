DDGETAUTOFLIPIN

The DDGETAUTOFLIPIN structure contains the handle information. 

typedef struct _DDGETAUTOFLIPIN {
  HANDLE  hDirectDraw;
  HANDLE  hVideoPort;
} DDGETAUTOFLIPIN, FAR *LPDDGETAUTOFLIPIN;

Members

hDirectDraw
Specifies the DirectDraw handle.
hVideoPort
Specifies the video port extensions (VPE) object handle.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
