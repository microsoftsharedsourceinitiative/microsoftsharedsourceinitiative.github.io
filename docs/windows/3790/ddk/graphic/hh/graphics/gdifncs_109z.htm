GDI Functions for Graphics Drivers

This section provides reference information for system-supplied GDI functions that can be called by both display and printer drivers. This includes the following function groups:

  • EngXxx functions. These are general graphics engine services.
  • XXXOBJ_Xxx functions. These specialized service routines are related to a particular object.

GDI Support for Graphics Drivers lists the include files that contain declarations for the graphics driver functions described in this section.

Reference material for functions that are implemented by display and/or printer driver writers can be found in Graphics Driver Functions (DDI).
