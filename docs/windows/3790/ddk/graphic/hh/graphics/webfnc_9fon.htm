ISNMP Set Method

The Set method allows an ASP Web page to associate a value with an SNMP OID.

[id(7), helpstring("method Set")]
  HRESULT
    Set(
    [in] BSTR  bstrOID,
    [in] VARIANT  varValue
    );

Parameters

bstrOID
Caller-supplied pointer to an OID string.
varValue
Caller-supplied location containing the OID's value.

Return Value

S_OK The operation succeeded.
E_FAIL The ISNMP Open method has not been called.
E_INVALIDARG The specified OID is invalid.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

The ISNMP Open method must be called before the Set method can be called. The method calls SnmpMgrRequest (described in the Platform SDK documentation) to set OID values.

VBScript Example

    Dim StrIP, strCommunity, objSNMP, OIDValue
    strIP = Session("MS_IPaddress")
    strCommunity = Session ("MS_Community")
    Set objSNMP = Server.CreateObject("OlePrn.OleSNMP")
    objSNMP.Open strIP, strCommunity, 2, 1000
    ...
    Determine value to assign to OID, store in OIDValue.
    ...
    objSNMP.Set ("43.18.1.1.2", OIDValue)
