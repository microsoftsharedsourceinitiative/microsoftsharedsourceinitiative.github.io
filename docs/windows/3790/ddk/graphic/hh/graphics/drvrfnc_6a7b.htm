DrvUpdateUISetting

The DrvUpdateUISetting function is obsolete. Windows 2000 and later UI plug-ins should use IPrintOemDriverUI::DrvUpdateUISetting or IPrintCoreUI2::DrvUpdateUISetting. 

typedef BOOL 
  (APIENTRY PFN_DrvUpdateUISetting)(
    PVOID  pdriverobj,
    PVOID  pOptItem,
    DWORD  dwPreviousSelection,
    DWORD  dwMode
    );

The preceding function pointer prototype defines the type of the DrvUpdateUISetting member of the OEMUIPROCS structure.
