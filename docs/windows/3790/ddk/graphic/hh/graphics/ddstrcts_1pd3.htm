DDGETIRQINFO

The DDGETIRQINFO structure contains interrupt information for the miniport. 

typedef struct _DDGETIRQINFO {
  DWORD  dwFlags;
} DDGETIRQINFO, *PDDGETIRQINFO;

Members

dwFlags
Specifies the interrupt management status.
IRQINFO_HANDLED
The miniport is managing the interrupt.

Headers

Defined in dxmini.h. Include dxmini.h.
