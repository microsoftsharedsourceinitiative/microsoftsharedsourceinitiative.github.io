DrvUpgradeRegistrySetting

The DrvUpgradeRegistrySetting function is obsolete. Windows 2000 and later UI plug-ins should use IPrintOemDriverUI::DrvUpgradeRegistrySetting or IPrintCoreUI2::DrvUpgradeRegistrySetting.

typedef BOOL 
  (APIENTRY PFN_DrvUpgradeRegistrySetting)(
    HANDLE  hPrinter,
    PCSTR  pFeature,
    PCSTR  pOption
    );

The preceding function pointer prototype defines the type of the pfnUpgrade parameter of the OEMUpgradeRegistry function.

Declared in printoem.h. Include printoem.h.
