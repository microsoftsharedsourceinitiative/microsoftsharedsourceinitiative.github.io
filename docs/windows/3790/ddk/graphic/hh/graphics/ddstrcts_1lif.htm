DDHAL_Xxx Structures

This table contains all the DirectDraw driver structures whose names have the form DDHAL_Xxx, which is the form used on Windows 98/Me. These structures are identical to those whose names have the form DD_Xxx, which is the form used on Windows 2000 and later. The include file dx95type.h contains type defines from DD_Xxx to DDHAL_Xxx.
