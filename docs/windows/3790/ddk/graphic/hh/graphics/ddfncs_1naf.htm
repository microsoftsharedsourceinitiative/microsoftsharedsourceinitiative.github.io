DDHAL_SetInfo

The DDHAL_SetInfo function provides GDI with current information about the driver. 

extern BOOL DDAPI DDHAL_SetInfo(
  LPDDHALINFO  lpDDHALInfo,
  BOOL  reset
  );

Parameters

lpDDHALInfo
Points to the DDHALINFO structure.
reset
Specifies whether there has been a mode change. If set TRUE, there has been a mode change.
