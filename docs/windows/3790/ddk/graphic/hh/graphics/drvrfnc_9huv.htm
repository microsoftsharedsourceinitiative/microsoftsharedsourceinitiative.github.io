OEMQueryFont

The OEMQueryFont function is used by GDI to get the IFIMETRICS structure for a given font. 

PIFIMETRICS APIENTRY
  OEMQueryFont(
    DHPDEV  dhpdev,
    ULONG_PTR  iFile,
    ULONG  iFace,
    ULONG_PTR  *pid
    );

Headers

Declared in printoem.h. Include printoem.h.

Comments

See DrvQueryFont.

Do not directly hook out this function. Instead, implement OEMEnableDriver so that it fills in a DRVENABLEDATA structure listing all of the DDIs to be hooked out.

If you call into the core driver, cast the call using the PFN_DrvQueryFont function pointer.
