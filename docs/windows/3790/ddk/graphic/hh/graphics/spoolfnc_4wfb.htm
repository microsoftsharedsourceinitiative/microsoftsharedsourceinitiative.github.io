Bidi Functions, Structures, and Enumerations

This section contains definitions of the functions, structures, and enumerations used in bidirectional (bidi) requests and responses. The bidi functions are declared in winsplp.h. The bidi structures and enumerations are defined in winspool.h. All are available in Windows XP and later.
