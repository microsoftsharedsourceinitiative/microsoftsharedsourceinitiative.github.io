Display and Print Devices: Windows DDK

DDRAWI_DDRAWSURFACE_GBL

The DDRAWI_DDRAWSURFACE_GBL structure contains the global DirectDrawSurface data that can be shared between object instances. 

typedef struct _DDRAWI_DDRAWSURFACE_GBL {
  DWORD  dwRefCnt;
  DWORD  dwGlobalFlags;
  union {
     LPACCESSRECTLIST  lpRectList;
     DWORD  dwBlockSizeY;
  };
  union {
     LPVMEMHEAP  lpVidMemHeap;
     DWORD  dwBlockSizeX;
  };
  union {
     LPDDRAWI_DIRECTDRAW_GBL lpDD;
     LPVOID   lpDDHandle;
  };
  FLATPTR   fpVidMem;
  union {
     LONG   lPitch;
     DWORD  dwLinearSize;
  };
  WORD    wHeight;
  WORD    wWidth;
  DWORD   dwUsageCount;
  ULONG_PTR   dwReserved1;
  DDPIXELFORMAT   ddpfSurface;
} DDRAWI_DDRAWSURFACE_GBL;

Members

dwRefCnt
Used by DirectDraw and should not be filled in by the driver.
dwGlobalFlags
Specifies a set of global surface flags.
Flag Meaning
DDRAWISURFGBL_FASTLOCKHELD This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWISURFGBL_HARDWAREOPDEST This surface was the destination for an asynchronous hardware operation.
DDRAWISURFGBL_HARDWAREOPSOURCE This surface was the source for an asynchronous hardware operation.
DDRAWISURFGBL_IMPLICITHANDLE This hKernelHandle was created implicitly.
DDRAWISURFGBL_ISCLIENTMEM This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWISURFGBL_ISGDISURFACE The surface represents what to GDI is a front buffer.
DDRAWISURFGBL_LATEALLOCATELINEAR An optimized surface was allocate as a formless chunk. The lPitch member is invalid, but the dwLinearSize member is valid.
DDRAWISURFGBL_LOCKBROKEN This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWISURFGBL_LOCKNOTHOLDINGWIN16LOCK This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWISURFGBL_LOCKVRAMSTYLE This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWISURFGBL_MEMFREE Video memory has been freed.
DDRAWISURFGBL_READONLYLOCKHELD This flag is used by DirectDraw and should not be filled in by the driver.
DDRAWISURFGBL_SOFTWAREAUTOFLIP This surface is autoflipped using software.
DDRAWISURFGBL_SYSMEMEXECUTEBUFFER The driver sets this flag to inform DirectDraw that the surface was allocated in system memory.
DDRAWISURFGBL_SYSMEMREQUESTED The surface is in system memory at the request of the user.
DDRAWISURFGBL_VPORTDATA This surface received data from the video port.
DDRAWISURFGBL_VPORTINTERLEAVED This surface contains interleaved video port data.

lpRectList
Used by DirectDraw and should not be filled in by the driver.
dwBlockSizeY
Contains the block size of the y axis of display memory that the display driver requested.
lpVidMemHeap
Points to a VMEMHEAP structure from which the heap display memory was allocated.
dwBlockSizeX
Contains the block size of the x axis of display memory that the display driver requested.
lpDD
Points to a DDRAWI_DIRECTDRAW_GBL structure that contains information about the DirectDraw object.
lpDDHandle
Points to a handle to the DirectDraw object. This member is used by the display driver when calling functions in ddraw16.dll.
fpVidMem
Specifies the address of display memory.
lPitch
Specifies the distance in bytes to the start of the next line on the surface.
dwLinearSize
Specifies tthe linear size of a nonrectangular surface.
wHeight
Specifies the height of the surface in scanlines.
wWidth
Specifies the width of the surface in pixels.
dwUsageCount
Used by DirectDraw and should not be filled in by the driver.
dwReserved1
Reserved for use by the display driver for display memory surfaces only.
ddpfSurface
Specifies the address of a DDPIXELFORMAT structure that contains the pixel format information for this surface. This member is allocated only if the pixel format differs from that of the primary display; that is, it is not allocated unless the DDRAWISURF_HASPIXELFORMAT flag in the DDRAWI_DDRAWSURFACE_LCL structure is set.

Headers

Defined in ddrawi.h. Include ddrawi.h.

Comments

Driver writers should use the GET_LPDDRAWSURFACE_GBL_MORE macro to traverse from a DDRAWI_DDRAWSURFACE_GBL address to the DDRAWI_DDRAWSURFACE_GBL_MORE structure.

The DDRAWI_DDRAWSURFACE_GBL structure is similar in purpose to the Windows 2000 and later structure, DD_SURFACE_GLOBAL.
