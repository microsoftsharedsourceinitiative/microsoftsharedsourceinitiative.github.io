Customizing the Printer Details Web Page

If you want to replace the printer details page provided by Microsoft, you can provide one or more customized ASP files. If you replace the default page, you can also provide links to additional customized pages installed on the server, plus links to any other pages on the Web.

If Microsoft's standard TCP/IP port monitor (tcpmon.dll) is used with your printer, customized ASP files can be used to replace the default printer details pages on a per-printer type or per-manufacturer basis. If customized ASP files have not been installed, the default printer details page is used.

Customized ASP files can also be used to replace the default printer details pages for printers using other port monitors, but per-printer type and per-manufacturer replacements are not allowed.

The method used for installing customized ASP files determines whether the customized files replace the default files for a printer type, a manufacturer, or a port monitor. For more information, see Installing Customized Print Web Pages.
