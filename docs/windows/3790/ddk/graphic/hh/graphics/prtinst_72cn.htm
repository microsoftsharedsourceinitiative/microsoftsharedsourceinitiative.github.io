Printer Class Installer

The printer class is one of the built-in device setup classes in Windows 2000 and later. The printer class installer, ntprint.dll, installs printer driver files by calling the AddPrinterDriverEx function. This class installer makes printers available on the server by calling AddPrinter. Both functions are described in the Platform SDK documentation.

Associating a Printer with a Driver in a Custom Printer Class Installer

A custom printer class installer must associate the printer's hardware ID with the printer driver. In your custom printer class installer, set the pszHardwareID member of the DRIVER_INFO_6 structure to the printer's hardware ID. To install the printer driver, call the AddPrinterDriverEx function, passing the address of the DRIVER_INFO_6 structure in this function's pDriverInfo parameter. The AddPrinterDriverEx function and the DRIVER_INFO_6 structure are described in the Platform SDK documentation. For information about writing a class Installer, see Writing a Class Installer.
