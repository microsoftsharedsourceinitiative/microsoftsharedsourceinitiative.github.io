CPSUI Option Types

When using CPSUI to create a property sheet page, user-selectable options must be defined using CPSUI-supported window controls. These controls are identified by the following CPSUI option types:

TVOT_2STATES

TVOT_3STATES

TVOT_CHKBOX

TVOT_COMBOBOX

TVOT_EDITBOX

TVOT_LISTBOX

TVOT_PUSHBUTTON

TVOT_SCROLLBAR

TVOT_TRACKBAR

TVOT_UDARROW

An option's type is specified in the Type member of an OPTTYPE structure.
