GDI Support for Graphics Drivers

This section describes the Microsoft® NT-based operating system graphics device interface (GDI). It then details the support that GDI provides to graphics drivers.

References to GDI in this section are implicit references to kernel-mode GDI; Microsoft® Win32® GDI will be explicitly identified. Kernel-mode GDI is also known as the Graphics Engine.

GDI function and structure references are documented in the Windows DDK Graphics Driver Reference. Most of the GDI function declarations and structure definitions can be found in winddi.h. For display drivers, the DirectDraw heap manager functions are declared in dmemmgr.h. Both files are shipped with the Windows DDK.
