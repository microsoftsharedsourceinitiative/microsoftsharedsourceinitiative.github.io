Naming Windows 98/Me Device Drivers

This topic applies only to Windows 98/Me.

Microsoft has reserved certain file names for its default device driver suite. Device drivers from other vendors must not use these reserved file names. Vendors should use unique file names for all files that make up their driver, even if some of the components of the driver are identical to the components shipped by Microsoft.

Providing unique names for components ensures that end users can easily switch between device drivers from different manufacturers for a specific device. This ensures that no permanent loss of functionality occurs, regardless of version numbers, when a DirectX application installs its device drivers or when an upgrade of Windows is installed.

The current list of reserved driver filenames is available at the Windows Platform Development Web site.
