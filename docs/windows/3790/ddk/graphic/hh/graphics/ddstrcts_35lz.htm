DD_GETBLTSTATUSDATA

The DD_GETBLTSTATUSDATA structure returns the blit status information.

typedef struct _DD_GETBLTSTATUSDATA {
  PDD_DIRECTDRAW_GLOBAL  lpDD;
  PDD_SURFACE_LOCAL  lpDDSurface;
  DWORD  dwFlags;
  HRESULT  ddRVal;
  VOID  *GetBltStatus;
} DD_GETBLTSTATUSDATA;

Members

lpDD
Points to a DD_DIRECTDRAW_GLOBAL structure that describes the DirectDraw API.
lpDDSurface
Points to a DD_SURFACE_LOCAL structure representing the surface whose blit status is being queried. For GetSysmemBltStatus calls, the driver is assured that this member points to a system memory surface. For more information about GetSysmemBltStatus, see Asynchronous DMA Support in DirectDraw.
dwFlags
Specifies the blit status being requested. This member can be one of the following values:
Flag Meaning
DDGBS_CANBLT Queries whether the driver can currently perform a blit.
DDGBS_ISBLTDONE Queries whether the driver has completed the last blit.

ddRVal
Specifies the location in which the driver writes the return value of the DdGetBltStatus callback. A return code of DD_OK indicates success.
GetBltStatus
Used by the DirectDraw API and should not be filled in by the driver.

Headers

Defined in ddrawint.h. Include winddi.h.
