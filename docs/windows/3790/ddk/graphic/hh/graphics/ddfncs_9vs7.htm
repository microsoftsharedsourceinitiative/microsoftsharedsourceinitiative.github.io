DD_DXAPI_ADDVPCAPTUREBUFFER

Operation

A video capture driver passes DD_DXAPI_ADDVPCAPTUREBUFFER in the dwFunctionNum parameter of the DxApi function to add a new capture buffer to the internal video port capture queue.

Input

lpvInBuffer
Pointer to a DDADDVPCAPTUREBUFF structure that contains the information required to add a new buffer.

Output

lpvOutBuffer
Pointer to a DWORD that contains the DirectDraw return value.

Headers

Declared in ddkmapi.h. Include ddkmapi.h.

Comments

Each time a hardware video port V-sync occurs, the kernel-mode video transport determines whether the previous field needs to be captured and if so, initiates a bus master to the next buffer in the queue.

This function identifier can be called at raised IRQL.
