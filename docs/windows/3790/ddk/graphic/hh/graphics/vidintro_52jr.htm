Frame Buffers

A frame buffer is the dedicated memory on a graphics adapter. Simple graphics adapters can have a frame buffer but no accelerator. If the hardware is linearly addressable and it can support standard format bitmaps (DIB format), GDI can handle the rendering operations and write directly to the frame buffer. This can reduce the size of the display driver because the driver can let GDI do all or most of the drawing. Most frame buffers are linearly addressable, or “flat”.

Among the sample display drivers is a frame buffer driver in the directory named src\video\displays\framebuf. Because this driver supports linearly addressable frame buffer hardware, it allows GDI to do most of the work, and is therefore quite small and relatively simple.
