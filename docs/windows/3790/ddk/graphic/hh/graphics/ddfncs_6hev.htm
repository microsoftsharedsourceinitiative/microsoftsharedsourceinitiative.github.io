DdGetHeapAlignment

The DdGetHeapAlignment callback function retrieves alignment information about a particular heap. 

DWORD
  (FAR PASCAL *LPDDHAL_GETHEAPALIGNMENT)(
    LPDDHAL_GETHEAPALIGNMENTDATA  lpGetHeapAlignmentData
    );

Parameters

lpGetHeapAlignmentData
Points to a DDHAL_GETHEAPALIGNMENTDATA structure that contains data on required alignments from a particular heap.

Return Value

DdGetHeapAlignment returns one of the following callback codes:

DDHAL_DRIVER_HANDLED

DDHAL_DRIVER_NOTHANDLED

Headers

Declared in ddrawi.h. Include ddrawi.h.

Comments

The driver provides a pointer to the DdGetHeapAlignment function through a call to its DDGetDriverInfo function using the GUID_MiscellaneousCallbacks GUID. The driver specifies DdGetHeapAlignment in the GetHeapAlignment member of the DDHAL_DDMISCELLANEOUSCALLBACKS structure.
