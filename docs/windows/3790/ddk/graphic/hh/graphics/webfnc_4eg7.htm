PaperNames

The PaperNames property allows an ASP Web page to obtain a set of strings representing all the printer's paper forms.

[propget, id(16), helpstring("property PaperNames")]
  HRESULT
    PaperNames(
    [out, retval] VARIANT  *pVal
    );

Parameters

pVal
Caller-supplied location to receive a pointer to a set of strings representing all the printer's paper forms.

Return Value

S_OK The operation succeeded.
E_HANDLE The IAspHelp Open method has not been called.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

The name of the printer being queried must have been specified by a previous call to the IAspHelp Open method.

The list of paper forms is obtained by calling the printer driver's DrvDeviceCapabilities function with the DC_PAPERNAMES flag set.

VBScript Example

    Dim objPrinter, PaperNameArray
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    objPrinter.Open strPrinter
    PaperNameArray = objPrinter.PaperNames
