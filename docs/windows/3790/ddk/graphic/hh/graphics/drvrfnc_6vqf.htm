IPrintCoreUI2 Interface

This section describes the methods defined for the IPrintCoreUI2 COM interface. This interface inherits from the IPrintOemDriverUI COM interface. The methods in this interface are provided by the Windows XP Pscript5 driver, for the sole use of its user interface plug-ins. Method prototypes appear in prcomoem.h.
