DD_PALETTECALLBACKS

The DD_PALETTECALLBACKS structure contains entry pointers to the DirectDraw palette callbacks that a device driver supports.

typedef struct DD_PALETTECALLBACKS {
  DWORD  dwSize;
  DWORD  dwFlags;
  PDD_PALCB_DESTROYPALETTE  DestroyPalette;
  PDD_PALCB_SETENTRIES  SetEntries;
} DD_PALETTECALLBACKS, *PDD_PALETTECALLBACKS;

Members

dwSize
Specifies the size in bytes of this DD_PALETTECALLBACKS structure.
dwFlags
Indicates what DirectDrawPalette callbacks the driver has implemented. For every bit set in dwFlags, the driver must initialize the corresponding function pointer member of this structure. This member can be one or more of the following flags:

DDHAL_PALCB32_DESTROYPALETTE
DDHAL_PALCB32_SETENTRIES

DestroyPalette
Points to the driver-supplied DdDestroyPalette palette callback.
SetEntries
Points to the driver-supplied DdSetEntries palette callback.

Headers

Defined in ddrawint.h. Include winddi.h.

Comments

Entries that the display driver does not use should be set to NULL. The driver initializes this structure in DrvEnableDirectDraw.
