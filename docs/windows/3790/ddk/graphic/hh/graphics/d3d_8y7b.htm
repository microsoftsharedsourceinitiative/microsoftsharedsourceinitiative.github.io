Multiple Vertex Streams

DirectX 8.0 adds support for multiple vertex streams. Even if the driver and hardware combination does not support more than one stream of vertex data, the driver must still handle the stream binding DP2 tokens (D3DDP2OP_SETSTREAMSOURCE and D3DDP2OP_SETSTREAMSOURCEUM) and the new vertex stream based DP2 drawing tokens (see New DP2 Stream Drawing Tokens). These are the mechanisms for passing vertex data to the driver in drawing for DirectX 8.0 level drivers.
