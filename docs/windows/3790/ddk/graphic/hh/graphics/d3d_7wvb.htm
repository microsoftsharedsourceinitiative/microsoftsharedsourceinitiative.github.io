Rendering to Multiple Targets Simultaneously

A DirectX 9.0 version driver can render to multiple targets simultaneously if the driver indicates that its device supports multiple render targets. To indicate the number of render targets that the device supports, the driver sets this number in the NumSimultaneousRTs member of the D3DCAPS9 structure. The driver must set this number to 1, if only rendering to a single target is supported. The driver returns a D3DCAPS9 structure in response to a GetDriverInfo2 query similarly to how it returns a D3DCAPS8 structure as described in Reporting DirectX 8.0 Style Direct3D Capabilities. Support of this query is described in Supporting GetDriverInfo2.

Render targets in a multiple render target group must have identical dimensions but can have different surface formats.

The driver receives the D3DDP2OP_SETRENDERTARGET2 operation code if an application requests to set the color buffer for one of the render targets in the multiple group.
