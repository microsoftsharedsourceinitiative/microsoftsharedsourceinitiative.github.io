DDHAL_VidMemFree

The DDHAL_VidMemFree function frees a block of video memory previously allocated by DDHAL_VidMemAlloc.

extern void
  DDAPI DDHAL_VidMemFree(
    LPDDRAWI_DIRECTDRAW_GBL  lpDD,
    int heap,
    FLATPTR  fpMem
    );

Parameters

lpDD
Points to a DDRAWI_DIRECTDRAW_GBL structure.
heap
Specifies the index of the DirectDraw heap from which the surface was allocated.
fpMem
Specifies the FLATPTR offset of the allocated surface.
