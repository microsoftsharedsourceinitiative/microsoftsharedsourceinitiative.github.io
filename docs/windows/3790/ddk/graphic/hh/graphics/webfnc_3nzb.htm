ISNMP GetTree Method

The GetTree method allows an ASP Web page to obtain the values associated with a set of sub-nodes beneath a specified root OID.

[id(4), helpstring("method GetTree")]
  HRESULT
    GetTree(
    [in] BSTR  varTree,
    [out,retval]VARIANT  *varValue
    );

Parameters

varTree
Caller-supplied string identifying a root OID.
varValue
Caller-supplied location to receive the address of an two-dimensional array containing OID strings and associated values.

Return Value

S_OK The operation succeeded.
E_FAIL The ISNMP Open method has not been called.
E_INVALIDARG The specified OID is invalid.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

The ISNMP Open method must be called before the GetTree method can be called. The method calls SnmpMgrRequest (described in the Platform SDK documentation) to obtain OID values.

VBScript Example

    Dim StrIP, strCommunity, objSNMP, OIDValueArray
    strIP = Session("MS_IPaddress")
    strCommunity = Session ("MS_Community")
    Set objSNMP = Server.CreateObject("OlePrn.OleSNMP")
    objSNMP.Open strIP, strCommunity, 2, 1000
    OIDValueArray = objSNMP.GetTree ("43.18.1.1.2")
