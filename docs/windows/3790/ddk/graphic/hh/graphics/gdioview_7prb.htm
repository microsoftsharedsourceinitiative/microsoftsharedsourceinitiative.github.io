GDI Halftone Services

GDI Halftone support includes the services listed in the following table.

Function Description
HT_ComputeRGBGammaTable Causes GDI to compute device red, green, and blue intensities based on gamma numbers.
HT_Get8BPPFormatPalette Returns a halftone palette for use on standard 8-bits per pixel device types.
HT_Get8BPPMaskPalette Returns a mask palette for an 8-bits per pixel device type.
HTUI_DeviceColorAdjustment Displays a dialog box that allows a user to adjust a device's halftoning properties.
