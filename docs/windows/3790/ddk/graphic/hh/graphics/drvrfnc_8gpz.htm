OEMGetInfo

This function is obsolete for Windows XP and later. It is supported only for legacy plug-ins. For a UI plug-in, use IPrintOemUI::GetInfo; for a Pscript plug-in, use IPrintOemPS::GetInfo; for a Unidrv plug-in, use IPrintOemUni::GetInfo.
