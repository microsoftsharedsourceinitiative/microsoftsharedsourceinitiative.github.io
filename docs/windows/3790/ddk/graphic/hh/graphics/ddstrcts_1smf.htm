DD_CREATEVPORTDATA

The DD_CREATEVPORTDATA structure contains the information necessary to describe the video port extensions (VPE) object being created.

typedef struct _DD_CREATEVPORTDATA {
  PDD_DIRECTDRAW_LOCAL  lpDD;
  LPDDVIDEOPORTDESC  lpDDVideoPortDesc;
  PDD_VIDEOPORT_LOCAL  lpVideoPort;
  HRESULT  ddRVal;
  VOID  *CreateVideoPort;
} DD_CREATEVPORTDATA;

Members

lpDD
Points to a DD_DIRECTDRAW_LOCAL structure that describes the DirectDraw API.
lpDDVideoPortDesc
Points to a DDVIDEOPORTDESC structure that describes the created VPE object.
lpVideoPort
Points to a DD_VIDEOPORT_LOCAL structure that represents the created VPE object.
ddRVal
Specifies the location in which the driver writes the return value of the DdVideoPortCreate callback. A return code of DD_OK indicates success.
CreateVideoPort
Used by the DirectDraw API and should not be filled in by the driver.

Headers

Defined in ddrawint.h. Include winddi.h.
