DDGETCURRENTAUTOFLIPOUTINFO

The DDGETCURRENTAUTOFLIPOUTINFO structure provides the surface information. 

typedef struct _DDGETCURRENTAUTOFLIPOUTINFO {
  DWORD  dwSurfaceIndex;
  DWORD  dwVBISurfaceIndex;
} DDGETCURRENTAUTOFLIPOUTINFO, *PDDGETCURRENTAUTOFLIPOUTINFO;

Members

dwSurfaceIndex
Specifies the current index in the autoflip chain of the current surface.
dwVBISurfaceIndex
Specifies the current index in the autoflip chain of the current VBI surface.

Headers

Defined in dxmini.h. Include dxmini.h.
