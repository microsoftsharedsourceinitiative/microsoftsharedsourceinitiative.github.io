DD_SYNCSURFACEDATA

The DD_SYNCSURFACEDATA structure contains the surface information. 

typedef struct _DD_SYNCSURFACEDATA {
  DWORD  dwSize;
  PDD_DIRECTDRAW_LOCAL  lpDD;
  PDD_SURFACE_LOCAL  lpDDSurface;
  DWORD  dwSurfaceOffset;
  ULONG_PTR  fpLockPtr;
  LONG  lPitch;
  DWORD  dwOverlayOffset;
  DWORD  dwOverlaySrcWidth;
  DWORD  dwOverlaySrcHeight;
  DWORD  dwOverlayDestWidth;
  DWORD  dwOverlayDestHeight;
  ULONG  dwDriverReserved1;
  ULONG  dwDriverReserved2;
  ULONG  dwDriverReserved3;
  ULONG  dwDriverReserved4;
  HRESULT  ddRVal;
} DD_SYNCSURFACEDATA;

Members

dwSize (Windows 98/Me only)
Specifies the size of this structure. This member must be initialized before the structure is used.
lpDD
Points to a DD_DIRECTDRAW_LOCAL structure that describes the DirectDraw API.
lpDDSurface
Points to the DD_SURFACE_LOCAL structure that describes the surface with which to sync.
dwSurfaceOffset
Contains the byte offset from the start of the frame buffer to the start of the surface. This value is used only by the miniport. This member must contain data that is filled in by the driver.
fpLockPtr
Contains the pointer value to be returned by the DdLock call for accessing the surface. This value is used by a kernel-mode client. This member can be modified by the driver, but does not need to be.
lPitch
Contains the pitch passed to the client during a DdLock call. This member can be modified by the driver, but does not need to be.
dwOverlayOffset
Contains the byte offset from the start of the frame buffer to the start of the overlay. This value is used only by the miniport and may be different than the dwSurfaceOffset member if cropping is involved or if the overlay origin is not the top left of the surface. This member must contain data that is filled in by the driver.
dwOverlaySrcWidth (Windows 98/Me only)
Specifies the source width of the overlay. This value is used only by the miniport. This member can be modified by the driver, but does not need to be.
dwOverlaySrcHeight (Windows 98/Me only)
Specifies the source height of the overlay. This value is used only by the miniport. This member can be modified by the driver, but does not need to be.
dwOverlayDestWidth (Windows 98/Me only)
Specifies the destination width of the overlay. This value is used only by the miniport. This member can be modified by the driver, but does not need to be.
dwOverlayDestHeight (Windows 98/Me only)
Specifies the destination height of the overlay. This value is used only by the miniport. This member can be modified by the driver, but does not need to be.
dwDriverReserved1
Reserved for use by the display driver.
dwDriverReserved2
Reserved for use by the display driver.
dwDriverReserved3
Reserved for use by the display driver.
dwDriverReserved4 (Windows 2000 and later only)
Reserved for use by the display driver.
ddRVal
Specifies the location in which the driver writes the return value of the DdSyncSurfaceData callback. A return code of DD_OK indicates success.

Headers

Defined in ddrawint.h. Include winddi.h.
