Miscellaneous Callback Function

This section provides a reference page for the miscellaneous callback function. The DirectDraw driver supplies this callback function when the GUID_MiscellaneousCallbacks GUID is passed to its DdGetDriverInfo function. On Windows 98/Me, the driver supplies several callback functions when the GUID_MiscellaneousCallbacks GUID is passed. For more information, see the DDHAL_DDMISCELLANEOUSCALLBACKS structure.
