EngGetProcessHandle

The EngGetProcessHandle function retrieves a handle to the current client process.

HANDLE
  EngGetProcessHandle(
    VOID
    );

Parameters

None

Return Value

In Windows 2000 and later, the EngGetProcessHandle function always returns NULL.

Headers

Declared in winddi.h. Include winddi.h.
