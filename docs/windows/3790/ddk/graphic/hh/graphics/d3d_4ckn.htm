Direct3D Implementation Requirements

The purpose of this section and the following subsections is to provide detailed descriptions of the Windows Logo Program requirements for hardware with respect to exposed features. This section includes descriptions of features that are present in the PC2001 Design Guide.

Feature compliance is generally assured by comparing a scene rasterized by the hardware to an identical scene rasterized by the reference rasterizer. Each feature must conform to the reference rasterizer as described in the following topics:

Render Target Requirements
