EngGetPrinterDataFileName

The EngGetPrinterDataFileName function retrieves the string name of the printer's data file.

LPWSTR
  EngGetPrinterDataFileName(
    IN HDEV  hdev
    );

Parameters

hdev
Handle to the device. This is the GDI handle received by the driver as the hdev parameter for DrvCompletePDEV.

Return Value

EngGetPrinterDataFileName returns a pointer to the null-terminated string buffer in which the name of the printer's data file is specified. The system obtains and stores the printer's data file name from the DRIVER_INFO_2 structure (described in the Platform SDK documentation) when the driver is first installed through the Win32 AddPrinterDriver routine.

Headers

Declared in winddi.h. Include winddi.h.
