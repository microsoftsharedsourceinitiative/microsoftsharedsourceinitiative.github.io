ISNMP Automation Interface

The ISNMP object and Automation interface are available for use with printers that use Microsoft's standard TCIP/IP port monitor. They allow an ASP page to set and retrieve values associated with SNMP OIDs.

The programmatic identifier for the ISNMP object is OlePrn.OleSNMP.
