IF_COMP Instruction

The IF_COMP instruction begins an IF block with a comparison.

Format

instruction token that contains D3DSIO_IFC

first source parameter token

second source parameter token

Comments

The IF_COMP instruction skips a block of code, based on the comparison between sources.

Both sources require explicit use of replicate swizzle, that is, the X, Y, Z, or W swizzle component (or the R, G, B, or A equivalent) must be specified.

If the comparison between sources is TRUE, the code enclosed by the IF statement and the matching ELSE (or ENDIF if ELSE is not used) is executed; otherwise the execution passes to the code enclosed by the ELSE-ENDIF statements (ELSE exists) or out of the IF block (ELSE does not exist).
