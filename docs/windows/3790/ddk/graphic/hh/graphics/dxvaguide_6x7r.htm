ProcAmp Control Processing

The ProcAmp control DDI extends DirectX VA to support ProcAmp control and post processing of video content by graphics device drivers. The ProcAmp control DDI is an interface between the VMR and the graphics device driver. The DDI maps to the existing DirectDraw® and DirectX VA DDI. This interface is not accessible via the IAMVideoAccelerator interface. The ProcAmp control DDI is available in Microsoft® DirectX® version 9.0.

If a driver supports accelerated decoding of compressed video, the VMR will call the driver to create two DirectX VA devices, one to perform the actual video decoding work, and the other to be used strictly for ProcAmp adjustments.
