DDRAWI_DDRAWPALETTE_INT

The DDRAWI_DDRAWPALETTE_INT structure contains the DirectDrawPalette object's interface information. 

typedef struct _DDRAWI_DDRAWPALETTE_INT {
  LPVOID  lpVtbl;
  LPDDRAWI_DDRAWPALETTE_LCL  lpLcl;
  LPDDRAWI_DDRAWPALETTE_INT  lpLink;
  DWORD  dwIntRefCnt;
} DDRAWI_DDRAWPALETTE_INT;

lpLcl

Used by DirectDraw and should not be filled in by the driver.
