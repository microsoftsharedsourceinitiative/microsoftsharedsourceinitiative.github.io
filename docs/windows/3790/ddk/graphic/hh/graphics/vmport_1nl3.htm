Optional Video Miniport Driver Functions

The following system-defined functions can be optionally implemented in video miniport drivers. Functions are listed in alphabetic order.

Optional Function Name Description
HwVidBugcheckCallback Enables the miniport driver to append data to the dump file when a specified bug check occurs.
HwVidExecuteDma Performs a DMA operation.
HwVidLegacyResources Returns a list of legacy resources for a particular device.
HwVidQueryDeviceCallback and/or HwVidQueryNamedValueCallback Process configuration information stored in the registry. Optional if the miniport driver's HwVidFindAdapter function uses I/O bus-type-specific configuration information or driver-supplied defaults.
HwVidQueryInterface Returns a miniport driver-implemented functional interface into which a child device can call.
HwVidTimer Is called periodically at approximately one-second intervals by the video port driver.
SvgaHwIoPortXxx functions Validate accesses to I/O ports by full-screen MS-DOS applications in x86-based machines. These functions are for VGA-compatible SVGA miniport drivers. See VGA-Compatible Miniport's SvgaHwIoPortXxx for more information.
