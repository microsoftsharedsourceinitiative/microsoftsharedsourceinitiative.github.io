User-Mode Rendering DLL Functions

This section describes the functions defined by user-mode rendering DLLs. Function prototypes are defined in winddiui.h.

These functions are obsolete, and are supported only so that user-mode rendering DLLs, optionally supplied with Windows NT 4.0 drivers, will execute under Windows 2000 and later. To supply user-mode image rendering code for printers, see Choosing User Mode or Kernel Mode.
