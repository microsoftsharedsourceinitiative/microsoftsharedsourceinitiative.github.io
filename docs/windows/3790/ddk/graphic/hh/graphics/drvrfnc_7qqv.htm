DrvWriteAbortBuf

The DrvWriteAbortBuf function is obsolete. Windows 2000 and later Unidrv render plug-ins should use IPrintOemDriverUni::DrvWriteAbortBuf. 

typedef DWORD 
  (APIENTRY PFN_DrvWriteAbortBuf)(
    PDEVOBJ  pdevobj,
    PVOID  pBuffer,
    DWORD  cbSize,
    DWORD  dwWait
    );

The preceding function pointer prototype defines the type of the DrvWriteAbortBuf member of the DRVPROCS structure.

Declared in printoem.h. Include printoem.h.
