Notify Callback Function

This section provides a reference page for a notify callback function that is implemented in a video capture driver. The video capture driver supplies such a function to the DirectDraw runtime when the video capture driver calls the DxApi function for certain operations. For example, the video capture driver supplies a notify callback function when the driver opens a video port. After this video port closes, the DirectDraw runtime is notified and calls this notify callback. The video capture driver can then perform necessary operations that are related to this video port closing.
