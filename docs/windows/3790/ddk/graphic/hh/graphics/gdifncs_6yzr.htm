EngLpkInstalled

The EngLpkInstalled function determines whether the language pack is installed on the system.

BOOL
  EngLpkInstalled(
    VOID
    );

Parameters

None

Return Value

EngLpkInstalled returns TRUE when the language pack is installed. Otherwise, it returns FALSE.
