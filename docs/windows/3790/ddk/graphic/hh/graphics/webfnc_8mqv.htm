IsTCPMonSupported

The IsTCPMonSupported property allows an ASP Web page to determine if Microsoft's standard TCP/IP port monitor is being used with a printer.

[propget, id(11), helpstring("property IsTCPMonSupported")]
  HRESULT
    IsTCPMonSupported(
    [out, retval] BOOL  *pVal
    );

Parameters

pVal
Caller-supplied pointer to a location to receive TRUE if the TCP/IP port monitor is being used with the printer, or FALSE if it is not.

Return Value

S_OK The operation succeeded.
E_HANDLE The IAspHelp Open method has not been called.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

The name of the printer being queried must have been specified by a previous call to the IAspHelp Open method.

VBScript Example

    Dim objPrinter, UseStdMon
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    objPrinter.Open strPrinter
    UseStdMon = objPrinter.IsTCPMonSupported
