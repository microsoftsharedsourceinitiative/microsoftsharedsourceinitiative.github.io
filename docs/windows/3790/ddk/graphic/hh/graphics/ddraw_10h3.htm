Creating Device Drivers for Windows 98/Me that Work with DirectX

Microsoft® DirectX® is the high-performance game and multimedia API. The DirectX subsystem includes a new set of audio and display drivers that have been extended to support DirectX.

This topic contains important instructions on how OEMs and IHVs can install newer versions of these drivers and ensure a correct version number for the drivers.

Important  The latest release of DirectX is the only vehicle that should be used by hardware vendors who ship DirectX components to their customers. The methods summarized in this section for ensuring correct version numbers and installing drivers must be followed to ensure a good experience for end users.
