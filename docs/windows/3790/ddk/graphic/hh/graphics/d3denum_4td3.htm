TEXREG2RGB Instruction

The TEXREG2RGB instruction interprets the red, green, and blue (RGB) color components of the source as texture address data (U,V,W) in order to sample the texture at the stage corresponding to the destination register number. The result is stored in the destination.

Format

instruction token that contains D3DSIO_TEXREG2RGB.

destination parameter token

source parameter token

Comments

The TEXREG2RGB instruction is useful for color-space remapping operations. It supports two-dimensional (2-D) and three-dimensional (3-D) coordinates. It can be used just like the TEXREG2AR or TEXREG2GB instructions to remap 2-D data. However, the TEXREG2RGB instruction also supports 3-D data so it can be used with cube maps and 3-D volume textures.
