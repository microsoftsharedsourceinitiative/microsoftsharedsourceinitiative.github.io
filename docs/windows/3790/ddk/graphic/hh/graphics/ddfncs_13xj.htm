DdSyncVideoPortData

The DdSyncVideoPortData callback function sets and modifies VPE object data before it is passed to the miniport. 

DWORD
  (APIENTRY *PDD_KERNELCB_SYNCVIDEOPORT)(
    PDD_SYNCVIDEOPORTDATA  lpSyncVideoPort
    );

Parameters

lpSyncVideoPort
Points to a DD_SYNCVIDEOPORTDATA structure that contains the VPE object data.

Return Value

DdSyncVideoPortData returns one of the following callback codes:

DDHAL_DRIVER_HANDLED

DDHAL_DRIVER_NOTHANDLED

Headers

Declared in ddrawint.h. Include winddi.h.
