DDLOCKOUTINFO

The DDLOCKOUTINFO structure contains the surface information output from the DxLock function. 

typedef struct _DDLOCKOUTINFO {
  ULONG_PTR  dwSurfacePtr;
} DDLOCKOUTINFO, *PDDLOCKOUTINFO;
