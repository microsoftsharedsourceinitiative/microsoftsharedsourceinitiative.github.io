IPrintOemUI2 COM Interface

The IPrintOemUI2 COM interface extends the IPrintOemUI COM interface. In addition to all of the methods in the IPrintOemUI interface, the IPrintOemUI2 interface provides the following methods. Except for the IPrintOemUI2::HideStandardUI method, these methods can be implemented in Unidrv or Pscript5 plug-ins running on Microsoft® Windows® XP and later.

Method Description
IPrintOemUI2::DocumentEvent
  Allows a UI plug-in to replace the core driver UI module's default implementation of the DrvDocumentEvent DDI.
IPrintOemUI2::HideStandardUI
  Allows a Pscript5 UI plug-in to specify whether the standard property sheets should be displayed or hidden.
IPrintOemUI2::QueryJobAttributes
  Allows a UI plug-in to post-process the core driver's results after a call to the DrvQueryJobAttributes DDI.

For more information, see Implementing Printer Driver COM Interfaces.
