DDRAWI_DDRAWPALETTE_LCL

The DDRAWI_DDRAWPALETTE_LCL structure contains local data for each individual DirectDrawPalette object. 

typedef struct _DDRAWI_DDRAWPALETTE_LCL {
  DWORD  lpPalMore;
  LPDDRAWI_DDRAWPALETTE_GBL  lpGbl;
  ULONG_PTR  dwUnused0;
  DWORD  dwLocalRefCnt;
  IUnknown FAR  *pUnkOuter;
  LPDDRAWI_DIRECTDRAW_LCL  lpDD_lcl;
  ULONG_PTR  dwReserved1;
  ULONG_PTR  dwDDRAWReserved1;
  ULONG_PTR  dwDDRAWReserved2;
  ULONG_PTR  dwDDRAWReserved3;
} DDRAWI_DDRAWPALETTE_LCL;

Members

lpPalMore
Specifies the address of additional local data. Reserved for system use.
lpGbl
Specifies the address of a DDRAWI_DDRAWPALETTE_GBL structure that contains the palette data.
dwUnused0
Currently unused.
dwLocalRefCnt
Used by DirectDraw and should not be filled in by the driver.
pUnkOuter
Used by DirectDraw and should not be filled in by the driver.
lpDD_lcl
Specifies the address of the owning local driver object.
dwReserved1
Reserved for use by the display driver.
dwDDRAWReserved1
Reserved.
dwDDRAWReserved2

dwDDRAWReserved3
