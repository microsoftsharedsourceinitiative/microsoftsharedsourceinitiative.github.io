Compressed Picture Decoding

When the bDXVA_Func variable equals 1, the operation specified is compressed picture decoding. The DXVA_ConfigPictureDecode structure contains the DirectX VA connection configuration data for compressed picture decoding.

Compressed Picture Parameters

The parameters that must be sent once for each picture to be decoded are specified in the DXVA_PictureParameters structure.
