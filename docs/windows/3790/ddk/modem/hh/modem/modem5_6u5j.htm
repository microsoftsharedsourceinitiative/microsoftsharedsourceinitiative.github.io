Voice Modem Wave Devices

Digital playback and recording of audio required by voice modems is done by the Windows Multimedia System (MMSystem) using wave devices. The MMSystem stores the audio in .wav files. There are the two basic types of voice modem wave devices:

  1. Serial wave device
  2. Wave wrapper device

Windows 2000 and later versions of the operating system do not require a wave device INF file. The wave device settings are contained in the voice modem INF file on Windows 2000 and later versions of the operating system.

For more information about supporting a wave device on Windows 2000 and later versions of the operating system, see Platform-Specific Considerations for Voice Support.
