Cl1FCS

The Cl1FCS entry specifies the FCS checking behavior of the Class1 driver.

Format

Single command string.

Sample

HKR, Fax, Cl1FCS,,"0"

Note

The following values are valid for this entry:

Value Description
"0" Check
"1" Do not check – assume there is no FCS
"2" Do not check – assume there is an FCS (default )

The "0" value is not required for Windows 2000 and later operating systems.
