CallSetupFailTimer

The CallSetupFailTimer entry defines the AT command that is used to set the modem's call fail timer.

Format

A single command string.

Sample

HKR, Settings,CallSetupFailTimer,,"S7=<#>"

Note

If the modem does not support setting the call fail timer, set the value to 0 in the Properties INF file entry. Do not set this command to S7=<0>.

CallSetupFailTimer should not have a specific value. Instead, it should be set to "<#>". The "#" is a placeholder. Unimodem fills in the actual number that is sent to the modem based on the CallSetupFailTimeout member of the Properties INF file entry.
