Pulse, Tone

The Pulse and Tone entries specify the characters that are required to indicate Pulse or Tone dialing. These characters are inserted into the dialing string immediately following the DialPrefix characters.

Format

A single command string.

Sample

HKR, Settings,Pulse,, "P"
HKR, Settings,Tone,, "T"

Note

If the modem does not support both tone and pulse dialing, the corresponding INF file entry should not be included. The supported commands must be indicated in the ModemOptions member of the Properties INF file entry.
