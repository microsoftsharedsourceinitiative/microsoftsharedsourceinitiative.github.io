Modifying the Manufacturer Section

The INF Manufacturer section identifies the manufacturer of one or more modems that can be installed with the INF file. It also defines the Models section name for the installation of that manufacturer's modems.

The manufacturers listed in this section of the INF file appear in the Install New Modem Wizard's list of manufacturers.

For more information about the manufacturer section, see INF Manufacturer Section.

Example Manufacturer Section

An example adaptation of the modem INF file Manufacturer section follows: 

[Manufacturer]
%Mfg% = Models

In the preceding example, the manufacturer is specified by %Mfg% (%Microsoft% in SmpData.inf) in the INF Strings section and the Models section is called "Models" ("Microsoft" in SmpData.inf).
