Modem Devices: Windows DDK

Support for Modem ID Types

The following table lists the modem ID types that are supported by various versions of Windows.

Windows Version Supported ID Types
Windows 98 Compatible with the Plug and Play in Windows 95 OSR2. Supports additional buses, including 1394 and USB. All versions support detection of Unimodem IDs.
Windows Me Supports Plug and Play.
Windows NT 4.0 Requires a Unimodem ID to install most modems. Windows NT 4.0 supports a limited implementation of Plug and Play, including a Plug and Play Enabler that supports ISA Plug and Play devices. Requires user to manually select resources after a Plug and Play modem has been detected.
Windows 2000 Supports Plug and Play and detection of legacy Unimodem IDs.
Windows XP and later Supports Plug and Play. Does not support legacy Unimodem IDs
