Supporting Cellular Features

When you set the cellular flag in the Properties entry, cellular features are enabled.

To set the cellular flag

  1. Select Data Profile from the ProCalc menu.
  2. Click the Modem Options tab.
  3. Check the Cellular check box.

For more information about enabling cellular features, see Creating Data Modem INF File Entries with ProCalc.
