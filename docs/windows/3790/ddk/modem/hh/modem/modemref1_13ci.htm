Blind_Off, Blind_On

The Blind_Off and Blind_On settings define the AT commands that are used to set the modem's blind dialing modes.

Blind_Off
Detect dial tone before dialing.
Blind_On
Do not detect dial tone before dialing.

Format

A single command string.

Sample

HKR, Settings,Blind_Off,,"X4"
HKR, Settings,Blind_On,,"X3"

Note

An INF file entry should not be included if the modem does not support blind dialing. The supported commands must be indicated in the ModemOptions member of the Properties INF file entry.
