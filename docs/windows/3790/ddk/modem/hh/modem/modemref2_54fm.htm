Cl2SWBOR

If the Cl2SWBOR entry is not "0", and if Cl2RecvBOR is "1", "AT+FBOR=0" is issued and the received data is bit-reversed.

Format

Single command string.

Sample

HKR, Fax, Cl2SWBOR,,"0"

Note

Set this entry to "1" for modems that do not implement the "AT+FBOR" command. The default value for this entry is "0".
