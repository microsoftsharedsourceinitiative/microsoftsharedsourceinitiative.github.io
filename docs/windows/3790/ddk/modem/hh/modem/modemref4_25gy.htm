ISDN\SetEAZ

The ISDN\SetEAZ entry defines commands issued to set EAZ (Endgerateauswahlziffer).

EAZs are used by German 1TR6 switches.

Format

One or more command strings.

Sample

HKR, ISDN\SetEAZ, 1 ,,   "AT EAZ1=%s<cr>"
HKR, ISDN\SetEAZ, 2 ,,   "AT EAZ2=%s<cr>"

Note

If only one SetEAZ entry is included, the ISDN configuration UI only accepts one EAZ.
