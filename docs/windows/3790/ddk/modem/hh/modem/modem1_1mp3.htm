Determining a Modem ID

All modems are identified by at least one modem ID string. For Windows XP and later platforms, this string must be a Plug and Play device ID.

A modem INF file requires one or more modem ID strings for each modem that it supports. Any modem can produce multiple modem ID strings, and different types of modems use different ID string formats.

To help ensure error-free modem installations

  1. Identify the modem type (for example, PCMCIA, PCI, or USB).
  2. Obtain the appropriate modem ID strings.
  3. Choose a strategy for using the modem IDs.
  4. When upgrading, make appropriate changes to the modem ID.

For more information on modem IDs, see Modem ID Strings.

Note  Legacy Unimodem IDs are not supported in Windows XP and later versions of the operating system.
