Modem Devices: Windows DDK

VoiceAnswer

The VoiceAnswer entry identifies the AT commands sent to the modem to answer a call in voice mode.

Format

One or more command strings.

Sample

HKR, VoiceAnswer,               1,, "at#cls=8<cr>"
HKR, VoiceAnswer,               2,, "at#vls=0<cr>"
HKR, VoiceAnswer,               3,, "at#vbt=1<cr>"
HKR, VoiceAnswer,               4,, "ata<cr>"

Note

Usually, the following actions are implemented in VoiceAnswer:

  • Place the modem into voice mode.
  • Set the DTMF tone generation duration.
  • Make sure the speakerphone is on the hook.
  • Answer the modem.
