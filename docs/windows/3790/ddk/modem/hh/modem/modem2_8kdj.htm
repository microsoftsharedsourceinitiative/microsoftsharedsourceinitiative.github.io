Using Compatible IDs in Modem INF Files

A hardware ID string identifies a specific Plug and Play modem. Use a compatible ID to identify a set of modems that can be installed with the same driver and set of INF file entries.

The compatible ID enables a modem to be installed with an INF file that does not contain a match of the modem's hardware ID. For example, if you represent an OEM selling a modem to other vendors, you can provide a generic INF file that will work for the modem regardless of the vendor.

For general information about using compatible IDs, see INF Models Section.
