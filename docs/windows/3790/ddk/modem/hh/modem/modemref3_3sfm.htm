HandsetCloseDelay

The HandsetCloseDelay entry specifies the amount of time, in seconds, that the CloseHandset command will be issued after a call to the Multimedia function waveClose.

Format

One or more command strings.

Sample

HKR,, HandsetCloseDelay,         1, 1e,00,00,00

Note

If waveOpen is called before this delay expires, the OpenHandset command will not be reissued. This entry is used to work around a feature in some modems that require the handset to be on the hook when the OpenHandset command is issued. Using this entry keeps the handset wave device open during the interval that occurs between playing audio files through the handset. This technique eliminates a requirement for the handset to be put back on the hook.
