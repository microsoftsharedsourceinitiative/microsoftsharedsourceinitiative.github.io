InactivityTimeout

The InactivityTimeout entry specifies the InactivityTimeout command string. The equation for InactivityTimeout is as follows:

InactivityTimeout * InactivityScale = inactivity timeout in seconds

Format

A single command string.

Sample

HKR, Settings,InactivityTimeout,, "S30=<#>"

Note

InactivityTimeout should not have a specific value. Instead, it should be set to "<#>". The "#" is a placeholder. Unimodem fills in the actual number that is sent to the modem based on the InactivityTimeout member of the Properties INF file entry.
