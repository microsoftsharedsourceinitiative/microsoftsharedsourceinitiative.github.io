Voice Modem Support of Unimodem Features

A voice modem can support any or all of the features that Unimodem provides. These features are configured using modem INF file entries.

Unimodem supports the following features on Windows 2000 and later versions of the operating system:

  • Caller ID
  • Speakerphone capability
  • Wave playback and record to and from the phone line
  • Wave playback and record to and from the handset
  • Answering voice calls
  • Voice call dial-out
  • Generation of DTMF digits in voice mode
  • Voice-to-data answering switch

Note  The wave device entries are integrated into the modem INF file on Windows 2000 and later versions of the operating system.

For more information about sample voice INF files, see Voice Modem Wave Devices.
