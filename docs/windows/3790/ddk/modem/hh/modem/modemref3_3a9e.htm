VariableTerminator

The VariableTerminator entry identifies the characters that terminate a Caller ID response from the modem.

Format

One line of one or more characters.

Sample

HKR,, VariableTerminator,,<cr><lf>

Note

A variable length response is composed of a fixed response (which is listed in the Responses registry entries) followed by a variable-length terminator string. In order for the response to be interpreted as a variable-length response, the high bit of the first byte of the Responses binary data INF file entry must be set.
