Modem Devices: Windows DDK

Response State

Modems can respond to commands in various ways; therefore, it is necessary to describe the responses that a modem will send in response to various events. These responses are specified by Responses INF file entries. Unimodem uses the binary state structure in the Responses entries to interpret modem responses.

The Responses entry contains the following fields:

  • Response text
  • Response state
  • Negotiated options
  • Negotiated DCE rate
  • Negotiated DTE rate

Click the appropriate ProCalc tabbed property sheet to enter the corresponding data for each of the preceding items.

An example of a Responses entry follows:

HKR, Responses, "<cr><lf>CONNECT<cr><lf>",  1, 02, 00, 00,00,00,00, 00,00,00,00
