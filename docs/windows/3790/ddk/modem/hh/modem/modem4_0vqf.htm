Known Fax Problems

This topic documents known incompatibility problems. Solutions for many of these problems involve specifying one or more fax entries (for example, ResetCommand). For more information about these INF file entries, see Fax Modem INF File Entries.

The most common initialization problem occurs when the fax uses the wrong command for software flow control. For more information about configuring the fax modem reset and setup commands, see ResetCommand and SetupCommand.

Some modems offer, but do not properly support, class 1 drivers. Fax support can be provided by using the FixModemClass INF file entry.

Windows supports additional INF file entries to maintain backward compatibility with legacy fax modems. These entries provide workarounds for some problems that are not found in newer modems. For more information about these INF file entries, see Fax Modem INF File Entries.
