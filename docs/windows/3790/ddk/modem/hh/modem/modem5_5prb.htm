Adding Modem-Specific Voice Support

All of the INF file entries that are required to add voice capability to a modem must be added to add-registry-sections of the modem INF file. You must include the following items in all voice mode INF files:

  • Voice support INF file sections
  • Primary voice modem INF file entries
  • The VoiceProfile entry

The following features are optional:

  • Audio Mixer
  • Caller ID
  • Speakerphone capability
  • Phone message recording and playback
  • Support for answering voice calls
  • Voice call dial-out
  • Support for generation of DTMF digits in voice mode
  • Voice-to-data answering switch
