Disabling the Second MSN

To prevent the option for entering the second MSN from being active in the Unimodem TSP and user interface, include the following entry in the INF file:

HKR, ISDN\SetMSN, 1 ,, "AT MSN1=%s<cr>"

and omit the following entry:

HKR, ISDN\SetMSN, 2 ,, "AT MSN2=%s<cr>"

For more information, see ISDN\SetMSN.
