Cl2SendBOR

The Cl2SendBOR entry specifies value of "AT+FBOR=xxx" for send operations.

Format

Single command string.

Sample

HKR, Fax, Cl2SendBOR,,"0"

Note

The default value is modem-specific, but it is typically "0".
