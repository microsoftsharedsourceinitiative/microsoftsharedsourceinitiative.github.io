ISDN\SetSpid

The ISDN\SetSpid entry specifies the AT commands that are used to set the service provider identification number as required by certain US switch types.

Format

One or more command strings.

Samples

The following are valid forms for this entry:

HKR, ISDN\SetSpid, 1,,     "AT SPID1=%s<cr>"
HKR, ISDN\SetSpid, 2,,     "AT SPID2=%s<cr>"

Or:

HKR, ISDN\SetSpid, 1,,     "ATSS60=%s<cr>"
HKR, ISDN\SetSpid, 2,,     "ATSS61=%s<cr>"

Or:

HKR, ISDN\SetSpid, 1,,     "AT!C6=%s<cr>"  
HKR, ISDN\SetSpid, 2,,     "AT*1!C6=%s<cr>"

Note

The characters %s and <cr> must be entered in lowercase.
