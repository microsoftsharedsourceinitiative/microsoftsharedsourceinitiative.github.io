EnableV17Send

The EnableV17Send entry identifies the command that enables the use of V.17 during a FAX call.

Format

Single command string.

Sample

HKR, Fax, EnableV17Send,,"1"

Note

If this entry is set to "0", it disables the use of V.17 (14.4 Kbps, short train) when sending a fax. If this entry is set to "1", the V.17 capability is not disabled when sending. The default setting is "1".
