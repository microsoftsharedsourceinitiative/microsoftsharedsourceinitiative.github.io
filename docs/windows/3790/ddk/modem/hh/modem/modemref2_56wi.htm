EnableV17Recv

The EnableV17Recv entry identifies the command that disables the use of V.17 during a FAX call.

Format

Single command string.

Sample

HKR, Fax, EnableV17Recv,,"1"

Note

If this entry is set to "0", it disables the use of V.17 (14.4 Kbps, short train) when receiving a fax. If this entry is set to "1", the V.17 capability is not disabled when receiving a fax. The default setting is "1".
