DialSuffix

The DialSuffix entry indicates characters that are appended to dial strings before the Terminator characters.

Format

A single command string.

Sample

HKR, Settings,DialSuffix,, ";"
HKR, Settings,DialSuffix,, "" \\ NULL modem, see note below

Note

The DialSuffix key should cause the modem to return to command state after dialing, if possible. A null DialSuffix can be used if a dial suffix is not applicable. This null suffix would be used for devices that inadvertently listen for dial tone at each dial command. For voice modems, there is a setting in the VoiceProfile INF file entry that should use a null DialSuffix while dialing in voice mode.
