Speaker Phone Specs

The SpeakerPhoneSpecs entry specifies the range of values for speaker phone volume and gain.

The SpeakerPhoneSpecs entry contains the following fields:

  • Max. Volume
  • Min. Volume
  • Max. Gain
  • Min. Gain

An example of a SpeakerPhoneSpecs entry follows:

HKR,, SpeakerPhoneSpecs,1, 00,00,00,00, 0f,00,00,00, 03,00,00,00, 00,00,00,00
