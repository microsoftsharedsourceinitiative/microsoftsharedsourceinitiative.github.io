Cl2lsEx

The Cl2lsEx entry specifies the command string to suppress the "AT+FDIS?", "AT+FDIS=.xxx", and "AT+FPTS=xxx"strings.

Format

Single command string.

Sample

HKR, Fax, Cl2lsEx,,"0"

Note

The command expects a nonstandard FPTS response: FPTS:1 indicates a good page and FPTS:1,2,0,0 indicates a bad page.

The value for this entry is modem-specific, but it is generally "0".
