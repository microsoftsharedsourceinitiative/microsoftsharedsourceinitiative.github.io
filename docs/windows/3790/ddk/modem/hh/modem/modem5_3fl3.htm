Adding Support for Generation of DTMF Digits in Voice Mode

Unimodem uses the GenerateDigit entry to generate DTMF digits while in voice mode. The following example demonstrates the GenerateDigit entry:

HKR, GenerateDigit,    1,, "at#vts=<Digit><cr>"    (generate tone signal <Digit>)
