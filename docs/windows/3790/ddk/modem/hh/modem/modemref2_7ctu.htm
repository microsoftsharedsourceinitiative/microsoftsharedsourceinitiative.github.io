HardwareFlowControl

The HardwareFlowControl entry specifies the hardware flow control.

Format

Single command string.

Sample

HKR, Fax, HardwareFlowControl,, "1"

Note

If this entry is set to "1", DTE will be configured to use the hardware flow control. The default setting is "0".
