Supporting MSN Entry

You can support MSN entry by including a single registry entry containing AT<cr> as follows:

HKR, ISDN\SwitchType, SWITCH_DSS1,,"AT<cr>"

If your ISDN MODEM defaults to DSS1 and does not support any AT commands for switch selection, omit the ISDN\SwitchType entries from the INF file. The user interface for MSN entry is no longer available if the ISDN\SwitchType entries are omitted.
