InactivityTimeout/InactivityScale Settings Example

Use ProCalc to specify the Inactivity Timeout value in the Properties entry and the InactivityScale entry, as described in Creating Data Modem INF File Entries with ProCalc.

An inactivity timeout is the period of data inactivity after which a modem hangs up. If a modem does not have this feature, set the Inactivity Timeout value in the Properties entry to zero.

The InactivityScale value is multiplied by the Inactivity Timeout value in the Properties entry to obtain the inactivity timeout in seconds.

The following is an example of the InactivityScale entry:

HKR,, InactivityScale, 1, 3c,00,00,00

If the InactivityScale entry is omitted from an INF file, a default value of 10 seconds is assigned.
