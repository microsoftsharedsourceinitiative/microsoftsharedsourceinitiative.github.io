Modem Devices: Windows DDK

PreAnswerCommand

The PreAnswerCommand entry identifies the modem command string that is issued to the modem just before Unimodem issues the ATA command.

Format

Single command string.

Sample

HKR, Fax, PreAnswerCommand,,

Note

Microsoft Fax issues the Setup command, and the command to go to the appropriate Fax class (Class1 or 2), before issuing the PreAnswerCommand.

The default is the empty string.
