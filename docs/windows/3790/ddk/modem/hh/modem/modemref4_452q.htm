Modem Devices: Windows DDK

ISDN\DirectoryNo

The ISDN\DirectoryNo entry specifies the AT command sent to the ISDN modem to set the directory number (US Switch Types).

Format

One or more command strings.

Sample

HKR, ISDN\DirectoryNo,1,,  "AT N1=%s<cr>"
HKR, ISDN\DirectoryNo,2,,  "AT N2=%s<cr>"

Note

There are several issues related to using this entry:

  • %s represents the number being sent.
  • The entry of two directory numbers, either two MSNs or two EAZs, is supported.
  • The second entry can be used to assign a directory number to an analog port. The first entry should always be used to assign numbers to the data port.
  • The %s must be in lowercase. The letters <cr> must also be in lowercase.
  • If the device supports the entry of only one number, include only one entry to automatically disable the user interface for the second number.
