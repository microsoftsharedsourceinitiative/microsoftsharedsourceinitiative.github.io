ISDN Features Not Supported by Unimodem

Unimodem does not support the following B Channel protocols:

  • Bonding mode 0
  • Bonding mode 1
  • Cfos channel bundling
  • TurboPPP
  • AimUX
  • Manufacturer specific MLP X.75 bundling

There is no way for a user to specify the second number required to make multi-link PPP calls. For this reason, Unimodem supports only ISDN modems with the capability to set up multi-link calls by dialing a single hunt group number. Devices that require ATD########## <joining character> ########### are not supported.

You might have to write a WDM modem driver to add support for these features or for other features unique to your device. For more information about supporting WDM, see Adding WDM Modem Support.
