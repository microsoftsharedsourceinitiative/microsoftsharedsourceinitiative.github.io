Modem Devices: Windows DDK

ISDN, CompatibilityFlags

The CompatibilityFlags entry is used to change the order in which configuration information is sent to the ISDN modem during static configuration.

If this entry is not included, or if the entry is set as follows:

HKR,ISDN, CompatibilityFlags,  1,00,00,00,00

Unimodem will issue commands in the following order:

  • Switch type
  • Directory number(s)
  • SPIDs

If the INF file entry is as follows:

HKR,ISDN, CompatibilityFlags,  1,00,00,00,00

Unimodem will issue commands in the following order:

  • Directory number(s)
  • SPIDs
  • Switch type

Format

One or more command strings.

Sample

 HKR,ISDN, CompatibilityFlags,  1,00,00,00,00
Another possible configuration is as follows:
HKR,ISDN, CompatibilityFlags,  1,00,00,00,01
