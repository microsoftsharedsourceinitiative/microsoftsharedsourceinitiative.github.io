WaveHardwareID (NT Wavedriver)

The WaveHardwareID (NT Wavedriver) entry provides voice functionality for Windows 2000 and later versions of the operating system. It defines the ID string for the wave device.

Format

String

Sample

HKR, WaveDriver,    WaveHardwareID ,, "RockwellADPCMVoiceModemWave"

Note

This command is placed only in the modem INF file. This command should not be placed in a serial wave driver INF file because Windows 2000 and later versions of the operating system do not use a serial wave driver INF file.

In addition, the following keys are found in a modem INF file that is enabled for voice functionality in Windows 2000 and later versions of the operating system:

XformID (NT Wavedriver)

BaudRate (NT Wavedriver)

XformModule (NT Wavedriver)

WaveHardwareID (NT Wavedriver)

WaveDevices (NT Wavedriver)

The Windows 2000 VoiceProfile bit (0x02000000) must also be enabled.
