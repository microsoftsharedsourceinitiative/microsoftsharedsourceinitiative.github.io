Cl2lsSr

The Cl2lsSr entry specifies the command string to suppress "AT+FCQ=0".

Format

Single command string.

Sample

HKR, Fax, Cl2lsSr,,"0"

Note

If this entry is not "0", the driver will not issue "AT+FCQ=0". Some modems will not function correctly if they receive this command. The default value for this entry is "0".
