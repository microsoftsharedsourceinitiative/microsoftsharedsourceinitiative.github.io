Modem ID Strings

All modems are identified by modem ID strings. For Microsoft Windows XP and later versions, these strings must be Plug and Play device IDs.

A modem INF file must contain one or more modem ID strings for each modem it supports. Most modems can produce multiple modem ID strings, and different types of modems use different ID string formats.
