SaPortWriteUnicodeStringRegistryValue

The SaPortWriteUnicodeStringRegistryValue function writes a string of 16-bit characters into the Unicode-string-type registry value specified by the registry value's name.

NTSTATUS
  SaPortWriteUnicodeStringRegistryValue(
    IN PVOID MiniPortDeviceExtension,
    IN PWSTR ValueName,
    IN PUNICODE_STRING RegistryValue
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
ValueName
Pointer to a buffer that contains a null-terminated string of 16-bit Unicode characters that contains the name of the registry value from which information is read.
RegistryValue
Pointer to a UNICODE_STRING structure that contains a description for the string of 16-bit characters that is written to the specified value.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortWriteUnicodeStringRegistryValue function writes a Unicode-string (REG_SZ).

Callers of SaPortWriteUnicodeStringRegistryValue must be running at IRQL = PASSIVE_LEVEL.
