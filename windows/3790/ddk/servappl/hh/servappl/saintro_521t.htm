Watchdog Timer

The watchdog timer device counts down from a configurable value and resets or shuts down the operating system if the count reaches zero. Because operating system services periodically restart the timer, if the operating system, drivers, or services stop functioning, the watchdog timer automatically restarts or shuts down the system. For information on the hardware requirements of a watchdog timer in a server appliance, go to the watchdog timer hardware Web site. No third-party driver is required for the watchdog timer as long as it is implemented as defined in this specification.
