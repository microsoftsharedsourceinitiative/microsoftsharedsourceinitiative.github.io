IOCTL_SA_GET_VERSION

Operation

The IOCTL_SA_GET_VERSION request returns the version of the SA interface that the miniport driver supports. The SA port driver might require a particular version for interoperability. All SA miniport drivers must support this request.

Input

None

Output

The miniport driver returns a pointer to a ULONG in the OutputBuffer parameter. The value in this ULONG indicates the SA interface version. The current version of the SA interface is defined as SA_INTERFACE_VERSION (0x00000004) in saio.h.

I/O Status Block

The returned NTSTATUS value is set to STATUS_SUCCESS for a successful I/O operation; otherwise, the value is set to one of the error status values defined in ntstatus.h.

Headers

Defined in saio.h. Include saio.h.
