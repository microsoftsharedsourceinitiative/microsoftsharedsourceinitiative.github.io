Keypad Miniport Driver

The miniport driver for a keypad device provides a standard interface between user-mode components in the operating system and the device. The driver primarily reads key-press information received from the device and passes this information to higher levels of the software for interpretation; the driver does not interpret this information.

The keypad miniport driver provides a callback interface to the system-supplied server availability (SA) port driver and uses the port driver's services. The miniport driver controls low-level communication with the keypad device. For more information on the standard architecture for all SA drivers, see Server Availability Driver Architecture.

When Plug and Play (PnP) detects a keypad device, PnP loads the keypad miniport driver. PnP also loads the SA port driver if it is not already loaded.
