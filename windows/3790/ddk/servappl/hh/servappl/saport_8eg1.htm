SaPortCopyUnicodeString

The SaPortCopyUnicodeString function copies a description of a Unicode string from a given Unicode string to an empty Unicode string.

NTSTATUS
  SaPortCopyUnicodeString(
    IN PVOID MiniPortDeviceExtension,
    IN OUT PUNICODE_STRING DestinationString,
    IN PUNICODE_STRING SourceString
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
DestinationString
Caller-supplied pointer to a UNICODE_STRING structure that receives a description for a string of 16-bit characters from the UNICODE_STRING at SourceString.
SourceString
Pointer to a UNICODE_STRING structure that contains a description for a string of 16-bit characters.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortCopyUnicodeString function performs the following actions:

  • Allocates a destination buffer from memory pool for the destination Unicode string. If SaPortCopyUnicodeString is not able to allocate a buffer to hold the string, STATUS_INSUFFICIENT_RESOURCES is returned.
  • Copies a string of 16-bit characters from the given source buffer to the destination buffer.
  • Assigns the size of the source string to the size of the destination string.

Callers of SaPortCopyUnicodeString must be running at IRQL < DISPATCH_LEVEL.
