IOCTL_SA_GET_CAPABILITIES

Operation

The IOCTL_SA_GET_CAPABILITIES request returns the capabilities of the SA device. Local display and NVRAM miniport drivers must support this request.

Input

None

Output

The miniport driver returns a pointer to a device-specific structure in the OutputBuffer parameter. This structure describes the capabilities of the driver's device. The following table specifies the structures that can be returned by SA miniport drivers.

SA miniport driver Capabilities structure
Local display miniport driver SA_DISPLAY_CAPS
NVRAM miniport driver SA_NVRAM_CAPS

I/O Status Block

The returned NTSTATUS value is set to STATUS_SUCCESS for a successful I/O operation; otherwise, the value is set to one of the error status values defined in ntstatus.h.

Headers

Defined in saio.h. Include saio.h.
