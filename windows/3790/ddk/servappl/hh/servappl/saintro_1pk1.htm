Reporting NVRAM Capabilities

The NVRAM miniport driver reports the capabilities of its NVRAM device to inform applications about how to read from or write to the device. An application uses the IOCTL_SA_GET_CAPABILITIES code to retrieve the device's capabilities. When the SA port driver receives this IOCTL code for servicing, the port driver must pass the IOCTL code's corresponding function code in a call to the miniport driver's SaDeviceIoctl function. The miniport driver then returns a pointer to an SA_NVRAM_CAPS structure in the OutputBuffer parameter. This structure contains the capabilities of the NVRAM device.
