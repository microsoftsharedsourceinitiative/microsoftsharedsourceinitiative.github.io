SA_NVRAM_BOOT_COUNTER

The SA_NVRAM_BOOT_COUNTER structure describes the boot counter of a nonvolatile random access memory (NVRAM) device. 

typedef struct _SA_NVRAM_BOOT_COUNTER {
  ULONG  SizeOfStruct;
  ULONG  Number;
  ULONG  Value;
  ULONG  DeviceId
} SA_NVRAM_BOOT_COUNTER, *PSA_NVRAM_BOOT_COUNTER;

Members

SizeOfStruct
Specifies the size in bytes of this structure.
Number
Specifies the number of the boot counter. This number can be from 1 to 4.
Value
Specifies the count value in the boot counter. Valid values for boot counters are from 0 to 4.
DeviceId
Specifies the Advanced RISC Computing (ARC) path. For more information, see NVRAM.

Headers

Defined in saio.h. Include saio.h.

Comments

The IOCTL_NVRAM_READ_BOOT_COUNTER and IOCTL_NVRAM_WRITE_BOOT_COUNTER codes are used with SA_NVRAM_BOOT_COUNTER to direct an NVRAM miniport driver to read from and write to the boot counter of an NVRAM device respectively. For more information, see Reading from and Writing to an NVRAM Boot Counter.

A value is written to a boot counter using IOCTL_NVRAM_WRITE_BOOT_COUNTER to force a fail over to another bootable system partition.

When a value is read from a boot counter using IOCTL_NVRAM_READ_BOOT_COUNTER, this value indicates the number of fail over instances that have occurred.
