SaPortDebugPrint

The SaPortDebugPrint function prints formatted output to the debugger.

VOID
  SaPortDebugPrint(
    IN ULONG DeviceType,
    IN ULONG DebugLevel,
    IN PSTR DebugMessage,
    ...
    );

Parameters

DeviceType
Specifies the miniport driver's device type. This type can be one of the constants in the following table.
Device type Meaning
SA_DEVICE_DISPLAY The SA device is a local display.
SA_DEVICE_KEYPAD The SA device is a keypad.
SA_DEVICE_NVRAM The SA device is nonvolatile RAM.

DebugLevel
Specifies the level or amount of debug tracing to report in an SA miniport driver. This level can be one of the constants in the following table.
Level Meaning
SAPORT_DEBUG_ERROR_LEVEL Errors.
SAPORT_DEBUG_WARNING_LEVEL Warnings.
SAPORT_DEBUG_TRACE_LEVEL Log information.
SAPORT_DEBUG_INFO_LEVEL All available debug information. This is the highest level of trace.

DebugMessage
Pointer to an 8-bit character string to be copied to the debugger. This string can also include format specifications for any arguments that follow.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortDebugPrint must be executing at IRQL <= DISPATCH_LEVEL.
