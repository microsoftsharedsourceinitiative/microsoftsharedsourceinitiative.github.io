SaPortShutdownSystem

The SaPortShutdownSystem function shuts down the operating system. 

NTSTATUS
  SaPortShutdownSystem(
    IN BOOLEAN PowerOff
    );

Parameters

PowerOff
Specifies a Boolean value that is TRUE to shut down the operating system to a power-off state, and FALSE to shut down and restart.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in Ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortShutdownSystem must be executing at IRQL <= DISPATCH_LEVEL.
