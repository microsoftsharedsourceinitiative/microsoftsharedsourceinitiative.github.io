SaPortWriteNumericRegistryValue

The SaPortWriteNumericRegistryValue function writes 4-byte numerical data into the numerical registry value specified by the registry value's name.

NTSTATUS
  SaPortWriteNumericRegistryValue(
    IN PVOID MiniPortDeviceExtension,
    IN PWSTR ValueName,
    IN ULONG RegistryValue
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
ValueName
Pointer to a buffer that contains a null-terminated string of 16-bit Unicode characters that contains the name of the registry value to which information is written.
RegistryValue
Specifies the 4-byte numerical data that is written to the specified value.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

The SaPortWriteNumericRegistryValue function writes a numeric (REG_DWORD) data value.

Callers of SaPortWriteNumericRegistryValue must be running at IRQL = PASSIVE_LEVEL.
