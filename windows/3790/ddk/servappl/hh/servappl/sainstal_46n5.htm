Installing Server Availability Devices

Server availability (SA) devices make a server appliance that is powered by Microsoft® Windows® Server 2003 and later available to users even if the network fails. Each SA device must have an information (INF) file that the system uses to install the device and the driver for the device. The system looks for specific system-defined named sections within INF files. These named sections, in turn, reference sections that are defined by the INF writer. SA INF files are based on the common INF file format, so refer to INF File Sections and Directives for general help in creating an SA INF file.
