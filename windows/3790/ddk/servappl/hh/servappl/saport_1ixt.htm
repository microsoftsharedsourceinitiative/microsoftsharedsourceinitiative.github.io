SaPortDeleteRegistryValue

The SaPortDeleteRegistryValue function removes a named value from the registry.

NTSTATUS
  SaPortDeleteRegistryValue(
    IN PVOID MiniPortDeviceExtension,
    IN PWSTR ValueName
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
ValueName
Pointer to a buffer that contains a null-terminated string of 16-bit Unicode characters that contains the name of the registry value to remove.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortDeleteRegistryValue must be running at IRQL = PASSIVE_LEVEL.
