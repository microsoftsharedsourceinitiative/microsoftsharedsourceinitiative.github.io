SaPortFreeUnicodeString

The SaPortFreeUnicodeString function releases memory for the buffer that stores a string of 16-bit characters and nulls and zeros the members of the UNICODE_STRING structure that describes the string. 

NTSTATUS
  SaPortFreeUnicodeString(
    IN PVOID MiniPortDeviceExtension,
    IN PUNICODE_STRING SourceString
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
SourceString
Pointer to a UNICODE_STRING structure whose memory buffer is released and whose members are zeroed.

Return Value

Returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortFreeUnicodeString must be running at IRQL < DISPATCH_LEVEL.
