SaCreateRoutine

The SaCreateRoutine function is a required function if the miniport driver must initialize a file-system context area that represents the SA device object.

NTSTATUS
  SaCreateRoutine(
    IN PVOID DeviceExtension,
    IN PIRP Irp,
    IN PVOID FsContext
    );

Parameters

DeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the device. The SaHwInitialize function sets up this area.
Irp
Specifies the IRP that contains the IRP_MJ_CREATE request being handled.
FsContext
Pointer to a file-system context area that is to be initialized.

Return Value

SaCreateRoutine returns STATUS_SUCCESS on success; otherwise, it returns one of the error status values defined in ntstatus.h.

Headers

The prototype for SaCreateRoutine is declared in saport.h as (*PSA_CREATECLOSE). Include saport.h.

Comments

The SaCreateRoutine function runs at IRQL = PASSIVE_LEVEL.
