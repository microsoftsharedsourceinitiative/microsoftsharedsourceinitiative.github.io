SaPortCompleteRequest

The SaPortCompleteRequest function completes the given I/O request, the I/O request that is currently being processed, or all outstanding I/O requests. 

VOID
  SaPortCompleteRequest(
    IN PVOID MiniPortDeviceExtension,
    IN PIRP Irp,
    IN ULONG Information,
    IN NTSTATUS Status,
    IN BOOLEAN CompleteAll
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
Irp
Specifies the IRP that describes the I/O operation to be completed or NULL if the current or all outstanding I/O operations are to be completed.
Information
Specifies information that is specific to the I/O operation.
Status
Specifies the NTSTATUS value that is assigned to the completed I/O operation.
CompleteAll
Specifies a Boolean value that is TRUE to direct the completion of all outstanding I/O requests, and FALSE otherwise.

Headers

Declared in saport.h. Include saport.h.

Comments

If the CompleteAll parameter is set to TRUE to direct the completion of all outstanding I/O requests, SaPortCompleteRequest removes each IRP from the device queue and processes the IRP's I/O operation.

Callers of SaPortCompleteRequest must be executing at IRQL <= DISPATCH_LEVEL.
