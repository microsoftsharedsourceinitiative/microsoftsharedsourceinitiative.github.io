SaPortGetOsVersion

The SaPortGetOsVersion function returns the version of the operating system on which the miniport driver is running. 

ULONG
  SaPortGetOsVersion(
    VOID
    );

Parameters

None

Return Value

Returns the major and minor version numbers of the operating system on which the miniport driver is running in the upper and lower 16 bits of the ULONG value respectively.

Headers

Declared in saport.h. Include saport.h.

Comments

The miniport driver's DriverEntry function calls the SaPortGetOsVersion function to obtain the version number of the operating system.

The major and minor version numbers of the Windows Server 2003 operating system are 5.0 and 1.0 respectively.

Callers of SaPortGetOsVersion must be running at IRQL = PASSIVE_LEVEL.
