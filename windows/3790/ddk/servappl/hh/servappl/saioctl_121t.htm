IOCTL_SADISPLAY_CHANGE_LANGUAGE

Operation

The IOCTL_SADISPLAY_CHANGE_LANGUAGE request loads a bitmap that changes the text that is displayed to another language. The SA port driver handles this request.

Input

None

Output

None

I/O Status Block

The returned NTSTATUS value is set to STATUS_SUCCESS for a successful I/O operation; otherwise, the value is set to one of the error status values defined in ntstatus.h.

Headers

Defined in saio.h. Include saio.h.
