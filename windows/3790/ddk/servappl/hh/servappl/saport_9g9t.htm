SaPortSynchronizeExecution

The SaPortSynchronizeExecution function synchronizes the execution of a given function with that of the miniport driver's SaInterruptServiceRoutine function.

BOOLEAN
  SaPortSynchronizeExecution (
    IN PVOID MiniPortDeviceExtension,
    IN PKSYNCHRONIZE_ROUTINE SynchronizeRoutine,
    IN PVOID SynchronizeContext
    );

Parameters

MiniPortDeviceExtension
Pointer to a driver-allocated area in which the miniport driver maintains the state of the SA device.
SynchronizeRoutine
Specifies a caller-supplied SynchCritSection function whose execution is to be synchronized with the execution of the miniport driver's SaInterruptServiceRoutine.
SynchronizeContext
Pointer to a caller-supplied context area to be passed to the SynchCritSection function when it is called.

Return Value

Returns TRUE if the operation succeeded; otherwise, it returns FALSE.

Headers

Declared in saport.h. Include saport.h.

Comments

Callers of SaPortSynchronizeExecution must be running at IRQL <= DIRQL, that is, the DIRQL that was assigned when the driver called the IoConnectInterrupt function to initialize the interrupt object.
