Server Appliance High Availability Devices: Windows DDK

Reference

Server availability (SA) port and miniport drivers are used to communicate with server availability devices. These devices make a server appliance that is powered by Microsoft® Windows® Server 2003 and later available to users even if the network fails. SA miniport drivers implement callback functions and I/O control codes and use SA port driver library functions and structures that are described in the following sections:
