WIA_DPS_DOCUMENT_HANDLING_CAPABILITIES

The WIA_DPS_DOCUMENT_HANDLING_CAPABILITIES property contains the capabilities of the scanner. An application reads this property to determine whether the scanner has a flatbed, document feeder, or duplexer installed. This property is also used to further define the installed features. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Value Definition
DETECT_DUP The scanner can detect a duplex scan request from the user.
DETECT_DUP_AVAIL The scanner can detect when the duplexer is installed.
DETECT_FEED The scanner can detect a document in the feeder.
DETECT_FEED_AVAIL The scanner can detect when the document feeder is installed.
DETECT_FLAT The scanner can detect a document on the flatbed platen.
DETECT_SCAN The scanner can detect a document in the feeder only by scanning.
DUP The scanner has a duplexer.
FEED The scanner has a document feeder installed.
FLAT The scanner has a flatbed platen.
