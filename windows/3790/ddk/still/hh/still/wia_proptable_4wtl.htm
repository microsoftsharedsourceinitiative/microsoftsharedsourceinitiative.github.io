WIA_IPS_YPOS

The WIA_IPS_YPOS property contains the current y coordinate, in pixels, of the upper left corner of the selected image. An application sets this property to mark the upper left corner of the selection area. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
