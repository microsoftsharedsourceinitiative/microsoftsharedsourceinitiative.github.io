WIA_DIP_UI_CLSID

The WIA_DIP_UI_CLSID property contains the vendor-supplied CLSID for any UI extension COM object that is installed with the WIA minidriver. This value is obtained from the INF file. If no UI CLSID is specified, the WIA service supplies a default value. This property is only used internally by the WIA service when UI is being displayed. The WIA service creates and maintains this property.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
