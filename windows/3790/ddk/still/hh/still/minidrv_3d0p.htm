IWiaMiniDrv::drvValidateItemProperties

The IWiaMiniDrv::drvValidateItemProperties method validates an item's properties against the set of valid values for each property and will update those properties if necessary.

HRESULT
  drvValidateItemProperties(
    IN BYTE  *pWiasContext,
    IN LONG  lFlags,
    IN ULONG  nPropSpec,
    IN const PROPSPEC  *pPropSpec,
    OUT LONG  *plDevErrVal
    );

Parameters

pWiasContext
Pointer to a WIA item context.
lFlags
Is reserved. Set to zero.
nPropSpec
Indicates the number of items n the pPropSpec array.
pPropSpec
Points to the first element of an array of PROPSPEC structures (defined in the Platform SDK documentation).
plDevErrVal
Points to a memory location that will receive a status code for this method. If this method returns S_OK, the value stored will be zero. Otherwise, a minidriver-specific error code will be stored at the location pointed to by this parameter.

Return Value

On success, the method should return S_OK and clear the device error value pointed to by plDevErrVal. If the method fails, it should return a standard COM error code and place a minidriver-specific error code value in the memory pointed to by plDevErrVal.

The value pointed to by plDevErrVal can be converted to a string by calling IWiaMiniDrv::drvGetDeviceErrorStr.

Headers

Declared in wiamindr.h. Include wiamindr.h.

Comments

This method is available in Windows Me and in Windows XP and later.
