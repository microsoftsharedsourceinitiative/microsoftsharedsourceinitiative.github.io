WIA_IPS_CONTRAST

The WIA_IPS_CONTRAST property contains the current hardware contrast setting for a device. An application sets this property to the hardware's contrast value. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE

Access Rights: READ/WRITE

Values should be mapped in a range between -1000 and 1000.

Normal = 0

Minimum Contrast = -1000

Maximum Contrast = 1000

Headers

Declared in wiadef.h. Include wiadef.h.
