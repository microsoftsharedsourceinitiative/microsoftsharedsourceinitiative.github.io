WIA_DIP_HW_CONFIG

This property indicates the type of connection that the device is using. The WIA service creates and maintains this property. It may only be changed by the WIA service. An application reads this property to determine the device's connection type, if desired.

The possible values for this property are:

Value Definition
1 Generic WDM device
2 SCSI device
4 USB device
8 Serial device
16 Parallel device

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
