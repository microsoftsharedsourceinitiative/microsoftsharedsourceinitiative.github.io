WIA_IPA_DATATYPE

The WIA_IPA_DATATYPE property contains the current data type setting for the device. An application reads this property to determine the data type of the image. An application writes this property to set the current data type of the image about to be transferred. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Value Definition
WIA_DATA_COLOR Color data
WIA_DATA_COLOR_DITHER Color dithered data
WIA_DATA_COLOR_THRESHOLD Color threshold data
WIA_DATA_DITHER Dithered data
WIA_DATA_GRAYSCALE Grayscale data
WIA_DATA_THRESHOLD Threshold data (usually one-bit)

Comments

If the device can be set to only a single value, create a WIA_PROP_LIST type, and place the valid value in it.

Check the WIA_IPA_DEPTH property to determine the bit depth.

This property usually contains a single value for cameras.

Headers

Declared in wiadef.h. Include wiadef.h.
