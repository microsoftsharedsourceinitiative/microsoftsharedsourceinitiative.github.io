WIA_DIP_SERVER_NAME

The WIA_DIP_SERVER_NAME property contains the name of the server on which the WIA minidriver is running. This property should contain the string “local” when an application is connected to a device on the same machine. This property is optional for Windows XP and later. The default value of this property is "local".

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
