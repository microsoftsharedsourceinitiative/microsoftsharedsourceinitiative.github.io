WIA_IPS_BRIGHTNESS

The WIA_IPS_BRIGHTNESS property contains the current hardware brightness setting for the device. An application sets this property to the hardware's brightness value. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE

Access Rights: READ/WRITE

Values should be mapped in a range between -1000 and 1000, where -1000 corresponds to the brightest setting, zero corresponds to the normal setting, and 1000 corresponds to the darkest setting.

Headers

Declared in wiadef.h. Include wiadef.h.
