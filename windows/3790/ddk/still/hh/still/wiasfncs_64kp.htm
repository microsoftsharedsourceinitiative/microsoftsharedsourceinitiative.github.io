wiasPrintDebugHResult

The wiasPrintDebugHResult function prints an HRESULT string on the device manager debug console.

VOID __stdcall
  wiasPrintDebugHResult(
    HINSTANCE hInstance,
    HRESULT hr
    );

Parameters

hInstance
Is the module handle of the calling module.
hr
Specifies the HRESULT to be printed.

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

WIA driver writers should to use the WIAS_LHRESULT macro, rather than this function.

This function is available in Windows Me and in Windows XP and later.
