IOCTL_SCSISCAN_GET_INFO

Returns device information.

DeviceIoControl Parameters

hDevice
Device handle.
dwIoControlCode
IOCTL_SCSISCAN_GET_INFO
lpInBuffer
NULL.
nInBufferSize
Zero.
lpOutBuffer
Pointer to a SCSISCAN_INFO structure.
nOutBufferSize
Size of the buffer pointed to by lpOutBuffer.
lpBytesReturned
Pointer to a location to receive the number of bytes returned.
lpOverlapped
Optional pointer to an OVERLAPPED structure (described in the Platform SDK documentation).

Headers

Declared in scsiscan.h. Include scsiscan.h.

Comments

When the DeviceloControl function is called with the IOCTL_SCSISCAN_GET_INFO I/O control code, the caller must specify the address of a SCSISCAN_INFO structure as the function's lpOutbuffer parameter. The kernel-mode driver fills in the structure.
