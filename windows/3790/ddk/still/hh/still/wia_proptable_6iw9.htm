WIA_DIP_BAUDRATE

The WIA_DIP_BAUDRATE property contains the current baud rate setting for the device. The value should be “Empty” if the device is not connected by a serial cable. The WIA service creates and maintains this property.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
