WIA_DPC_FOCAL_LENGTH

The WIA_DPC_FOCAL_LENGTH property contains the 35mm-equivalent focal length. The values of this property correspond to the focal length in millimeters multiplied by 100. Focal length determines optical zoom.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
