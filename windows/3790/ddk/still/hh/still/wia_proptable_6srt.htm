WIA_DPC_POWER_MODE

The WIA_DPC_POWER_MODE property defines the current power source for the camera device. An application reads this property to determine what power source the camera is using.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Value Definition
POWERMODE_BATTERY The camera device is operating on battery power.
POWERMODE_LINE The camera device is operating on a power adapter.

Headers

Declared in wiadef.h. Include wiadef.h.
