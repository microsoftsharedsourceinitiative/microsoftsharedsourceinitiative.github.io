WIAS_CHANGED_VALUE_INFO

The WIAS_CHANGED_VALUE_INFO structure is used to store the current and previous values of a property.

typedef struct _WIAS_CHANGED_VALUE_INFO {
  BOOL  bChanged;
  LONG  vt;
  union {
    LONG  lVal;
    FLOAT fltVal;
    BSTR  bstrVal;
    GUID  guidVal;
  } Old;
  union {
    LONG  lVal;
    FLOAT fltVal;
    BSTR  bstrVal;
    GUID  guidVal;
  } Current;
} WIAS_CHANGED_VALUE_INFO, *PWIAS_CHANGED_VALUE_INFO;

Members

bChanged
Is a Boolean that indicates whether a property has changed. That is, if the property's current value is different from its value before IWiaMiniDrv::drvValidateItemProperties was called. Upon return from one of the wiasGetChangedValueXxx functions, this member is TRUE if the property changed, and FALSE if the property did not change.
vt
Specifies the variant data type for the property. This member can be one of the following:

VT_UI1

VT_UI2

VT_UI4

VT_I2

VT_I4

VT_R4

VT_R8

VT_CLSID

VT_BSTR

See the PROPVARIANT structure in the Platform SDK documentation for more information.

Old
Specifies the previous value of the property. This member is a union consisting of the following members. The particular union member used must be consistent with the type specified in the vt member.

lVal - the property type is LONG.

fltVal - the property type is a floating point type.

bstrVal - the property type is BSTR.

guidVal - the property type is GUID.

Current
Specifies the current value of the property. This member is a union consisting of the following members. The particular union member used must be consistent with the type specified in the vt member.

lVal - the property type is LONG.

fltVal - the property type is a floating point type.

bstrVal - the property type is BSTR.

guidVal - the property type is GUID.

Headers

Declared in wiamindr.h. Include wiamindr.h.

Comments

The wiasGetChangedValueXxx functions, use this structure to determine whether a property of a certain type has been changed by an application. These functions are used when the minidriver performs property validation, which occurs within the body of IWiaMiniDrv::drvValidateItemProperties.

This structure is available in Windows Me and in Windows XP and later.
