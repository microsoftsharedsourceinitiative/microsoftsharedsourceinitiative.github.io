wiasDebugTrace

The wiasDebugTrace function prints a debug trace string in the device manager debug console.

VOID __cdecl
  wiasDebugTrace(
    HINSTANCE  hInstance,
    (LPCSTR  pszFormat, ...)
    );

Parameters

hInstance
Is the module handle of calling module.
pszFormat, ...
Specifies a variable argument list, which starts with an ANSI format string containing the message and any format specifiers. The ellipsis (...) specifies a variable number of arguments that are to be output.

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

WIA device driver writers should to use the WIAS_LTRACE macro, rather than this function.

This function is available in Windows Me and in Windows XP and later.
