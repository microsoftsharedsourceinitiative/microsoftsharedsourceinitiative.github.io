WIA_DPC_FOCUS_MODE

The WIA_DPC_FOCUS_MODE property defines the current focus mode setting for the camera device. The device driver enumerates the supported values of this property. An application writes this property to set the focus mode for the camera device.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Value Definition
FOCUSMODE_AUTO The camera device is configured to focus automatically.
FOCUSMODE_MACROAUTO The camera device is configured to focus automatically using short-range macro settings.
FOCUSMODE_MANUAL The camera device is configured to allow the user to focus manually.
