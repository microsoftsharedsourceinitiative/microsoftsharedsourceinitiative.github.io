wiasGetRootItem

The wiasGetRootItem function retrieves the root item context of a specified WIA item.

HRESULT
  wiasGetRootItem(
    BYTE  *pWiasContext,
    BYTE  **ppWiasContext
    );

Parameters

pWiasContext
Pointer to a WIA item context.
ppWiasContext
Pointer to a memory location that receives the address of the WIA item's root item context.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
