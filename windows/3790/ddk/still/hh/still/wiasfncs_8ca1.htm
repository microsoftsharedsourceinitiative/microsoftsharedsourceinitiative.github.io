wiasReadPropFloat

The wiasReadPropFloat function retrieves a floating point property value from a WIA item.

HRESULT
  wiasReadPropFloat(
    BYTE  *pWiasContext,
    PROPID  propid,
    FLOAT  *pfVal,
    FLOAT  *pfValOld,
    BOOL  bMustExist
    );

Parameters

pWiasContext
Pointer to a WIA item context.
propid
Specifies the property identifier.
pfVal
Pointer to a memory location that receives the floating point value of the property.
pfValOld
Pointer to a memory location that receives the former floating point value of the property. If this information is not needed, set this parameter to NULL.
bMustExist
Indicates whether the property must exist. If set to TRUE, the property must exist; if set to FALSE, it does not have to exist.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
