WIA_DPC_BURST_NUMBER

The WIA_DPC_BURST_NUMBER property contains the number of images that the device attempts to capture during a burst operation.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE, WIA_PROP_LIST or WIA_PROP_RANGE

Access Rights: READ/WRITE

Headers

Declared in wiadef.h. Include wiadef.h.
