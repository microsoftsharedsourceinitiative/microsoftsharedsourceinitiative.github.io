WIA_IPS_WARM_UP_TIME

The WIA_IPS_WARM_UP_TIME property contains the maximum warm-up time, in milliseconds, needed by the device before starting the scanning operation. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Comments

An application can read this property to determine the maximum warm up time for this device, and then present a "waiting for the device to warm up" dialog to let the user know there might be a wait or pause before anything happens.

Headers

Declared in wiadef.h. Include wiadef.h.
