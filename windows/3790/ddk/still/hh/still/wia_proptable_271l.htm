WIA_DPA_CONNECT_STATUS

The WIA_DPA_CONNECT_STATUS property contains the current connection status for the device. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
