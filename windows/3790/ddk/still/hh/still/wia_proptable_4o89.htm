WIA_DPS_PAD_COLOR

The WIA_DPS_PAD_COLOR property contains the current pad color used when the minidriver pads unaligned data. This property should be reported as a vector of four BYTE values in the form of an RGBQUAD structure (described in the Platform SDK documentation). The minidriver creates and maintains this property.

Property Type: VT_UI2 | VT_VECTOR

Valid Values: WIA_PROP_NONE

Access Rights: READ/WRITE

Comments

An application reads this property to get the padding color used.

Headers

Declared in wiadef.h. Include wiadef.h.
