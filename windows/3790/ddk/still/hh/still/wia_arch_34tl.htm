WIA Utility Classes

This section describes the three helper classes that are part of the WIA Utility Library:

CWiauDbgFn class

CWiauFormatConverter class

CWiauPropertyList class

CWiauDbgFn Class

The CWiauDbgFn class is a helper class for function or method entry/exit point tracing.

Method Description
CWiauDbgFn::CWiauDbgFn Class constructor.
CWiauDbgFn::~CWiauDbgFn Class destructor.

CWiauFormatConverter Class

The CWiauFormatConverter class is a helper class for converting images to BMP format.

Method Description
CWiauFormatConverter::CWiauFormatConverter Class constructor.
CWiauFormatConverter::~CWiauFormatConverter Class destructor.
CWiauFormatConverter::ConvertToBmp Converts an image to BMP format.
CWiauFormatConverter::Init Initializes the class and GDI+ for converting images.
CWiauFormatConverter::IsFormatSupported Verifies that GDI+ supports the image format to be converted.

CWiauPropertyList Class

The CWiauPropertyList class is a helper class for defining and initializing WIA property lists.

Method Description
CWiauPropertyList::CWiauPropertyList Class constructor.
CWiauPropertyList::~CWiauPropertyList Class destructor.
CWiauPropertyList::DefineProperty Adds a property definition to the property list object.
CWiauPropertyList::GetPropId Gets the property ID for a property at a specified index.
CWiauPropertyList::Init Initializes the property list object.
CWiauPropertyList::LookupPropId Gets the property index for a property with a specified property ID.
CWiauPropertyList::SendToWia Calls the WIA service to define all the properties currently contained in a property list object.
CWiauPropertyList::SetAccessSubType Resets the access and subtype of a property.
CWiauPropertyList::SetCurrentValue(BYTE*) Sets the value of a property whose type is of BYTE array.
CWiauPropertyList::SetCurrentValue(BSTR) Sets the value of a property whose type is BSTR.
CWiauPropertyList::SetCurrentValue(CLSID) Sets the value of a property whose type is CLSID.
CWiauPropertyList::SetCurrentValue(FLOAT) Sets the value of a property whose type is FLOAT.
CWiauPropertyList::SetCurrentValue(LONG) Sets the value of a property whose type is LONG.
CWiauPropertyList::SetCurrentValue(SYSTEMTIME) Sets the value of a property whose type is SYSTEMTIME (described in the Platform SDK documentation).
CWiauPropertyList::SetValidValues(BSTR, list) Sets the type, as well as the default, current, and valid values of a list of BSTR values.
CWiauPropertyList::SetValidValues(CLSID, list) Sets the type, as well as the default, current, and valid values of a list of CLSID values.
CWiauPropertyList::SetValidValues(flag) Sets the type, as well as the default, current, and valid values of a property whose values are determined by flag settings.
CWiauPropertyList::SetValidValues(FLOAT, list) Sets the type, as well as the default, current, and valid values of a list of FLOAT values.
CWiauPropertyList::SetValidValues(FLOAT, range) Sets the type, as well as the default, current, and valid values of a range of FLOAT values.
CWiauPropertyList::SetValidValues(LONG, list) Sets the type, as well as the default, current, and valid values of a list of LONG values.
CWiauPropertyList::SetValidValues(LONG, range) Sets the type, as well as the default, current, and valid values of a range of LONG values.
