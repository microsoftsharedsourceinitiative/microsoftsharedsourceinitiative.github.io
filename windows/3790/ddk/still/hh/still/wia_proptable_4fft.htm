WIA_DPS_ENDORSER_STRING

The WIA_DPS_ENDORSER_STRING property contains a string that is to be endorsed (printed) on each page that is scanned by the minidriver. An application sets this property using the valid character set reported in the WIA_DPS_ENDORSER_CHARACTERS property. The minidriver should endorse documents only if a string is set in this property. An empty string means that the endorser functionality is disabled.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
