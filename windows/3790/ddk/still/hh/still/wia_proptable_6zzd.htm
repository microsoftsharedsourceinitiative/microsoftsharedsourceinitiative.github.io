WIA_IPS_YRES

The WIA_IPS_YRES property contains the current vertical resolution setting in pixels for the device. An application sets this property to set the vertical resolution. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE or WIA_PROP_LIST

Access Rights: READ/WRITE or READONLY

Comments

If the device can be set to only a single value, create a WIA_PROP_LIST type, and place the valid value in it. This is also a case where one resolution setting is dependent on another resolution. (The vertical resolution can be dependent on the horizontal resolution.)

Headers

Declared in wiadef.h. Include wiadef.h.
