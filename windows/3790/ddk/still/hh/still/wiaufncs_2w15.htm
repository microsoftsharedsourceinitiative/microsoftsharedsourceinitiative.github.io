WIA General Utility Functions

The WIA utility functions are a collection of functions that carry out common tasks. One group of functions in this collection, the wiauRegXxx functions, can be used to work with the system registry. A second group, the wiauStrXxx functions, can be used to copy ANSI or Unicode strings, or to convert strings from one of these formats to the other.
