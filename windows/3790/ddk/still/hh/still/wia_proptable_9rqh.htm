WIA_DPS_DOCUMENT_HANDLING_STATUS

The WIA_DPS_DOCUMENT_HANDLING_STATUS property contains current state of the scanner's installed flatbed, document feeder, or duplexer. An application reads this property to determine whether the scanner device is ready to be used. This is an ideal way to check whether there is paper in the feeder prior to acquiring an image. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Value Definition
DUP_READY The duplexer is enabled and ready to be used.
FEED_READY The document feeder has a page loaded and is ready for use.
FLAT_COVER_UP The flatbed cover is up.
FLAT_READY The flatbed is ready for use.
PAPER_JAM There is a document jammed in the document feeder.
PATH_COVER_UP The paper path is covered up and is preventing proper operation.

Headers
