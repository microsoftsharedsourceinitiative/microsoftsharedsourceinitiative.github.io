wiasGetItemType

The wiasGetItemType function indicates the item type.

HRESULT
  wiasGetItemType(
    BYTE  *pWiasContext,
    LONG  *plItemType
    );

Parameters

pWiasContext
Pointer to a WIA item context.
plItemType
Pointer to a memory location that receives a value indicating the type of the item. See the Platform SDK documentation for a list of the WIA item type flags.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
