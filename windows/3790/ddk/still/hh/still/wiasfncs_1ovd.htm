wiasWriteBufToFile

The wiasWriteBufToFile function writes from a specified buffer to an image file.

HRESULT
  wiasWriteBufToFile(
    LONG  lFlags,
    PMINIDRV_TRANSFER_CONTEXT  pmdtc
    );

Parameters

lFlags
Specifies a set of operation flags. This parameter should be set to 0.
pmdtc
Pointer to a MINIDRV_TRANSFER_CONTEXT structure.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

A WIA minidriver uses this function to write a buffer to any type of image file. A driver that intends to write a page of purely image data to a multipage TIFF file, delegating the addition of the appropriate tags, image file directory (IFD) entries, and other nonimage data to the WIA service, should use wiasWritePageBufToFile.

This function is available in Windows Me and in Windows XP and later.
