wiasCreateLogInstance

The wiasCreateLogInstance function creates an instance of a logging object.

HRESULT 
  wiasCreateLogInstance(
    BYTE  *pModuleHandle,
    IWiaLogEx  **ppIWiaLogEx
    );

Parameters

pModuleHandle
Pointer to the module handle, which is used to filter output.
ppIWiaLogEx
Pointer to a memory location that receives the address of the logging interface.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function is available in Windows Me and in Windows XP and later.
