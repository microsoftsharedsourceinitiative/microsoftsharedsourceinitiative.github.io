Still Image Devices: Windows DDK

SCSI Driver

The kernel-mode still image driver for SCSI buses supports ReadFile by creating a command descriptor block (CDB) that includes a SCSI Read command. It supports WriteFile by creating a CDB that includes a SCSI Write command. User-mode minidrivers can specify customized CDBs by calling DeviceIoControl. For more information, see SCSI Still Image I/O Control Codes. See the Platform SDK documentation for descriptions of ReadFile and WriteFile.
