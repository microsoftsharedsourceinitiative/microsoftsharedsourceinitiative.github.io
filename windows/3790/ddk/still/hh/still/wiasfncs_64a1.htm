wiasWriteMultiple

The wiasWriteMultiple function writes multiple property values to a WIA item.

HRESULT
  wiasWriteMultiple(
    BYTE  *pWiasContext,
    ULONG  ulCount,
    const PROPSPEC  *ps,
    const PROPVARIANT  *pv
    );

Parameters

pWiasContext
Pointer to a WIA item context.
ulCount
Specifies the total number of properties to write.
ps
Pointer to the first element of an array of PROPSPEC structures that indicate the properties to write.
pv
Pointer to the first element of an array of PROPVARIANT structures that contain the values to write to the item.

Return Value

On success, the function returns S_OK. If the function fails, it returns a standard COM error or one of the WIA_ERROR_XXX errors (described in the Platform SDK documentation).

Headers

Declared in wiamdef.h. Include wiamdef.h.

Comments

This function operates in a similar manner to IPropertyStorage::WriteMultiple, which is described in the Platform SDK documentation. The PROPSPEC and PROPVARIANT structures are also described in the Platform SDK documentation.

This function is available in Windows Me and in Windows XP and later.
