WIA_DIP_DEV_TYPE

The WIA_DIP_DEV_TYPE property contains the device type and device subtype. The WIA_DIP_DEV_TYPE property is created and maintained by the WIA service. The device type and subtype is taken from the INF file. An application reads this property to determine whether it is using a scanner, camera, or video device.

The possible values for device type are:

Device Type Value Definition
StiDeviceTypeDefault 0x0000 Default device
StiDeviceTypeScanner 0x0001 Scanner device
StiDeviceTypeDigitalCamera 0x0002 Camera device
StiDeviceTypeStreamingVideo 0x0003 Video device

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.

Comments

For more information on INF files see INF Files for WIA Devices. The StiDeviceTypeXxx are defined in sti,h
