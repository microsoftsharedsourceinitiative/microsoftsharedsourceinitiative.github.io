IWiaLog COM Interface

The IWiaLog interface is provided for backward compatibility only. The methods in this interface allow a minidriver to write error, trace, and warning messages to a log. The IWiaLog interface provides the following methods:

Method Description
IWiaLog::InitializeLog Initializes the logging utility.
IWiaLog::Log Logs a message to a file or other target.
IWiaLog::hResult Translates an HRESULT into a string.

For new WIA drivers, driver writers are encouraged to use the macros in the following table. The first three macros can be used to write a logging statement with a specified type of error, trace, or warning, respectively. The fourth macro can be used to translate an HRESULT into a descriptive string.

Macro Description
WIAS_LERROR Writes a log statement of type ERROR to the specified target.
WIAS_LHRESULT Translates an HRESULT value into a string and writes the string to the diagnostic log file.
WIAS_LTRACE Writes a log statement of type TRACE to the specified target.
WIAS_LWARNING Writes a log statement of type WARNING to the specified target.

See IWiaLog Interface and Diagnostic Log Macros for more information.
