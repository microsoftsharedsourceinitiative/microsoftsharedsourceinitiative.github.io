WIA_IPA_PLANAR

The WIA_IPA_PLANAR property contains image data packing options. An application reads this property to determine the image packing options, or sets the current image packing options. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Value Definition
WIA_PACKED_PIXEL Image data is in packed pixel format.
WIA_PLANAR Image data is in planar format.

Access Rights: READ/WRITE

Comments

If the device can be set to only a single value, create a WIA_PROP_LIST type, and place the valid value in it.

Headers

Declared in wiadef.h. Include wiadef.h.
