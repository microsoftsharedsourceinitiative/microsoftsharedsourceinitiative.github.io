WIA_DPC_EXPOSURE_MODE

The WIA_DPC_EXPOSURE_MODE property indicates the camera's current exposure mode. An application changes this property to control the exposure mode of the camera device.

Property Type: VT_I4

Valid Values: WIA_PROP_LIST

Access Rights: READ/WRITE

Value Definition
EXPOSUREMODE_APERTURE_PRIORITY The aperture is set manually by the user and the shutter speed is set automatically by the camera device.
EXPOSUREMODE_AUTO The shutter speed and aperture are set automatically by the camera device.
EXPOSUREMODE_MANUAL The shutter speed and aperture are set manually by the user.
EXPOSUREMODE_PROGRAM_ACTION The shutter speed and aperture are set automatically by the camera device, and optimized for moving subject matter (scenes containing fast motion).
EXPOSUREMODE_PROGRAM_CREATIVE The shutter speed and aperture are set automatically by the camera device, and optimized for still subject matter.
EXPOSUREMODE_PROGRAM_PORTRAIT The shutter speed and aperture are set automatically by the camera device, and optimized for portrait photography.
EXPOSUREMODE_SHUTTER_PRIORITY The shutter speed is set manually by the user, and the aperture is set automatically by the camera device.
