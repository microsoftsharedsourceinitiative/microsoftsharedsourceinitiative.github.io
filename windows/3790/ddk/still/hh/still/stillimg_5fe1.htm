Still Image COM Interfaces

Microsoft STI defines the following four COM interfaces:

IStillImage COM Interface – Provides access to the Still Image Event Monitor.

IStiDevice COM Interface – Provides access to still image data streams and device events.

IStiUSD COM Interface – Provides access to user-mode still image minidrivers.

IStiDeviceControl COM Interface – Provides utility operations to minidrivers.
