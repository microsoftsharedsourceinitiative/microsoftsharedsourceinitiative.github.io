SCSISCAN_INFO

The SCSISCAN_INFO structure is used as a parameter to DeviceIoControl (described in the Platform SDK documentation), when the specified I/O control code is IOCTL_SCSISCAN_GET_INFO.

typedef struct _SCSISCAN_INFO {
  ULONG  Size;
  ULONG  Flags;
  UCHAR  PortNumber;
  UCHAR  PathId;
  UCHAR  TargetId;
  UCHAR  Lun;
  UCHAR  AdapterName[MAX_STRING];
  ULONG  Reserved;
} SCSISCAN_INFO, *PSCSISCAN_INFO;
