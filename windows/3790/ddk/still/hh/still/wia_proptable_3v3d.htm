WIA_IPS_THRESHOLD

The WIA_IPS_THRESHOLD property contains the current hardware threshold setting for a device. An application sets this property to change the hardware threshold value. This value is valid only if the WIA_IPA_DATATYPE is equal to WIA_DATA_THRESHOLD. The minidriver creates and maintains this property.

Property Type: VT_I4

Valid Values: WIA_PROP_RANGE

Access Rights: READ/WRITE

Values should be mapped in a range of zero through 255. The default value is 128.

Headers

Declared in wiadef.h. Include wiadef.h.
