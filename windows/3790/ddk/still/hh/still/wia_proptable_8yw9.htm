WIA_DIP_DEV_NAME

The WIA_DIP_DEV_NAME property contains the name of the device. The device name is obtained from the INF file. An application reads this property to obtain the name of the device. The WIA service creates and maintains this property.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
