WIA_DIP_PORT_NAME

The WIA_DIP_PORT_NAME property contains the installed device's port name. The port name is assigned by the kernel-mode driver that is used to operate the device. An application reads this property to determine the installed device's port name. The WIA service creates and maintains this property.

Property Type: VT_BSTR

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Headers

Declared in wiadef.h. Include wiadef.h.
