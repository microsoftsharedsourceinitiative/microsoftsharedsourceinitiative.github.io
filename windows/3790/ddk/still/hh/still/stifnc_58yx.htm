IStillImage::SetupDeviceParameters

The IStillImage::SetupDeviceParameters method allows clients of the IStillImage COM interface to modify a still image device's stored characteristics, if the device's bus type is unknown.

HRESULT
  SetupDeviceParameters(
    PSTI_DEVICE_INFORMATION  pDevInfo
    );

Parameters

pDevInfo
Caller-supplied pointer to an STI_DEVICE_INFORMATION structure describing a still image device.

Return Value

If the operation succeeds, the method returns S_OK. Otherwise, it returns one of the STIERR-prefixed error codes defined in stierr.h.

Headers

Declared in sti.h. Include sti.h.

Comments

The IStillImage::SetupDeviceParameters method only allows modification of device parameters associated with still image devices for which a bus has not been identified. For such devices, the Still Image server sets the dwHardwareConfiguration member of the device's STI_DEVICE_INFORMATION structure to STI_HW_CONFIG_UNKNOWN when IStillImage::GetDeviceInfo is called.

Currently, the only device parameter that can be modified is the device's port name. When calling this method to modify the port name, the dwSize, szDeviceInternalName, and pszPortName members of the STI_DEVICE_INFORMATION must be specified. All other members are ignored.

Before calling IStillImage::SetupDeviceParameters, clients of the IStillImage COM interface must call IStillImage::StiCreateInstance to obtain an IStillImage interface pointer.
