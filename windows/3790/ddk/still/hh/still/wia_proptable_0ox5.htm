WIA_IPA_SUPPRESS_PROPERTY_PAGE

The WIA_IPA_SUPPRESS_PROPERTY_PAGE property specifies whether to suppress the general property pages for items on the device.

Property Type: VT_I4

Valid Values: WIA_PROP_NONE

Access Rights: READONLY

Value Definition
WIA_PROPPAGE_CAMERA_ITEM_GENERAL Suppress the general item property page for a camera.
WIA_PROPPAGE_SCANNER_ITEM_GENERAL Suppress the general item property page for a scanner.

Headers

Declared in wiadef.h. Include wiadef.h.
