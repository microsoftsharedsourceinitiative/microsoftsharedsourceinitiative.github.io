DBLNODE

The DBLNODE structure holds a node in a doubly-linked list of surface interfaces. 

typedef struct _DBLNODE {
  struct  _DBLNODE  FAR  *next;
  struct  _DBLNODE  FAR  *prev;
  LPDDRAWI_DDRAWSURFACE_LCL  object;
  LPDDRAWI_DDRAWSURFACE_INT  object_int;
} DBLNODE, FAR *LPDBLNODE;

Members

next
Points to the next node.
prev
Points to the previous node.
object
Points to a DDRAWI_DDRAWSURFACE_LCL structure that is the link to the object.
object_int
Points to a DDRAWI_DDRAWSURFACE_INT structure containing the object interface.

Headers

Defined in ddrawi.h. Include ddrawi.h.
