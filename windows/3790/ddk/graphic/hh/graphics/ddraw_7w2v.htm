Using Dxsetup.exe

This topic applies only to Windows 98/Me.

Dxsetup.exe, which is installed in the \Program Files\DirectX\Setup directory by DirectX Setup, has these functions:

  • Dxsetup.exe can be used to determine the version of the DirectX components installed, including whether the device drivers support DirectX and whether they are registered.
  • Dxsetup.exe can restore previously installed drivers.

Caution  When using dxsetup.exe for troubleshooting, always run the utility from the user's hard disk rather than from the DirectX distribution disk. Running dxsetup.exe from the DirectX distribution disk under Windows 2000 or later causes the system to stall.

When dxsetup.exe is run, it presents the current status of all of the DirectX components, including device drivers, plus three options: Quit, Restore Audio Drivers, and Restore Display Drivers. Unless the user selects Quit, dxsetup.exe forces a reboot after it has performed the requested function.
