Direct3D Driver Structures

This section describes the Direct3D® driver structures. It is divided into two main groupings: the first consists of the D3DHAL_Xxx structures defined in d3dnthal.h on Windows® 2000 and later and d3dhal.h on Windows 98/Me; the second grouping consists of the remaining structures, including device description structures, and other Direct3D structures defined in d3dcaps.h and d3dtypes.h. Direct3D lists the include files that contain Direct3D structure definitions and also contains information for writing a Direct3D driver.
