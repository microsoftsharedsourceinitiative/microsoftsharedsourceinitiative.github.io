Direct3D Driver-Supplied Functions

This section provides reference pages for the callback functions in the Windows® Direct3D® driver interface. All callback names are pseudonames to be decided upon by the display driver writer.

The end of this reference contains the D3DBuffer Callbacks that are used only in drivers that manage their own command and vertex buffers.
