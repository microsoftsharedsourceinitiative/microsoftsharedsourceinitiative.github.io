Video Port Functions for Miniport Drivers

This section describes the AgpXxx and VideoPortXxx functions. Together, these functions comprise the video port driver, a system-supplied component called by video miniport drivers. See Video Miniport Header Files, Sample Code, and References for a list of the include files that contain declarations for these functions. References are provided in alphabetic order.
