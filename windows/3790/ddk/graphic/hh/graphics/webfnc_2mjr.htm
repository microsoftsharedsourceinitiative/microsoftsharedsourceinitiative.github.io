LongPaperName

The LongPaperName property allows an ASP Web page to convert a short paper name to a long paper name.

[propget, id(19), helpstring("property LongPaperName")]
  HRESULT
    LongPaperName(
    [in] BSTR  bstrShortName,
    [out, retval] BSTR  *pVal
    );

Parameters

bstrShortName
Caller-supplied pointer to a string representing a short paper name.
pVal
Caller-supplied location to receive a pointer to a string representing a long paper name.

Return Value

S_OK The operation succeeded.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

VBScript Example

    Dim objPrinter, LongName
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    LongName = objPrinter.LongPaperName("iso-a0")
