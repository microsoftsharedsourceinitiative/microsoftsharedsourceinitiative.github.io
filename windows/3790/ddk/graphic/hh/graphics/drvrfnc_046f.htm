OEMStartPage

The OEMStartPage function is called by GDI when it is ready to start sending the contents of a physical page to the driver for rendering.

BOOL APIENTRY
  OEMStartPage(
    SURFOBJ  *pso
    );

Headers

Declared in printoem.h. Include printoem.h.

Comments

See DrvStartPage.

Do not directly hook out this drawing function. Instead, implement OEMEnableDriver so that it fills in a DRVENABLEDATA structure listing all drawing DDIs to be hooked out.

If you call into the core driver, cast the call using the PFN_DrvStartPage function pointer.
