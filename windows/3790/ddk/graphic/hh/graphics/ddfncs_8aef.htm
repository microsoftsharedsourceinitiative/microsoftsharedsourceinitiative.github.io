DDHAL_VidMemAlloc

The DDHAL_VidMemAlloc function allocates a block of video memory.

extern
  FLATPTR DDAPI DDHAL_VidMemAlloc(
    LPDDRAWI_DIRECTDRAW_GBL  lpDD,
    int  heap,
    DWORD  dwWidth,
    DWORD  dwHeight
    );

Parameters

lpDD
Points to a DDRAWI_DIRECTDRAW_GBL structure.
heap
Specifies the index of the DirectDraw heap from which to allocate the surface.
dwWidth
Specifies the width in bytes of the requested surface.
dwHeight
Specifies the height in scanlines of the requested surface.
