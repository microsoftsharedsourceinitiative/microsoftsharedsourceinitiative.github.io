DD_SURFACE_INT

The DD_SURFACE_INT structure contains the DirectDrawSurface object's interface information.

typedef struct _DD_SURFACE_INT {
  PDD_SURFACE_LOCAL  lpLcl;
} DD_SURFACE_INT;

Members

lpLcl
