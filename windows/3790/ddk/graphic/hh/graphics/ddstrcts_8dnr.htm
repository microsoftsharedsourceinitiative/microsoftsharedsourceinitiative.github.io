DDGETVERSIONNUMBER

The DDGETVERSIONNUMBER structure contains the version number of the kernel-mode video transport component of DirectDraw that is supported by the video miniport driver's DxApi interface. 

typedef struct _DDGETVERSIONNUMBER {
  DWORD  ddRVal;
  DWORD  dwMajorVersion;
  DWORD  dwMinorVersion;
} DDGETVERSIONNUMBER, FAR *LPDDGETVERSIONNUMBER;

Members

ddRVal
Specifies the location in which DirectDraw writes the return value. A return code of DD_OK indicates success.
dwMajorVersion
Specifies the major version number.
dwMinorVersion
Specifies the minor version number.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
