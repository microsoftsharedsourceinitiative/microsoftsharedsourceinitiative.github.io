IPrintOemUI COM Interface

The IPrintOemUI COM interface is the means by which the printer interface DLL for Unidrv or Pscript5 communicates with a UI plug-in. The IPrintOemUI interface is implemented by each UI plug-in.

The following table lists and describes all of the methods supplied by the IPrintOemUI interface. UI plug-ins must define all listed methods. If a method is not needed it can simply return E_NOTIMPL.

Method Description
IPrintOemUI::CommonUIProp
  Allows a user interface plug-in to modify an existing printer property sheet page or document property sheet page.
IPrintOemUI::DeviceCapabilities
  Allows a user interface plug-in to specify customized device capabilities.
IPrintOemUI::DevicePropertySheets
  Allows a user interface plug-in to add a new page to a printer device’s printer property sheet.
IPrintOemUI::DevMode
  Performs operations on a user interface plug-in’s private DEVMODE members.
IPrintOemUI::DevQueryPrintEx
  Allows a user interface plug-in to help determine if a print job is printable.
IPrintOemUI::DocumentPropertySheets
  Allows a user interface plug-in to add a new page to a printer device’s document property sheet.
IPrintOemUI::DriverEvent
  Called by the print spooler when processing driver-specific events that might require action by the printer driver.
IPrintOemUI::FontInstallerDlgProc
  Replaces the Unidrv font installer's user interface.
IPrintOemUI::GetInfo
  (Implementation required.) Returns a user interface plug-in's identification information.
IPrintOemUI::PrinterEvent
  Allows a user interface plug-in to process printer events.
IPrintOemUI::PublishDriverInterface
  (Implementation required.) Supplies a pointer to the Unidrv or Pscript5 driver's IPrintOemDriverUI COM interface.
IPrintOemUI::QueryColorProfile
  Allows a printer interface DLL to specify an ICC profile to use for color management.
IPrintOemUI::UpdateExternalFonts
  Allows a printer interface DLL to update a printer's Unidrv Font Format file.
IPrintOemUI::UpgradePrinter
  Allows a user interface plug-in to upgrade device option values that are stored in the registry.

For more information, see Implementing Printer Driver COM Interfaces.
