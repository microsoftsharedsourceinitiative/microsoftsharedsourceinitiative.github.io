DDSETFIELDNUM

The DDSETFIELDNUM structure contains the handles and the field number.

typedef struct _DDSETFIELDNUM {
  HANDLE  hDirectDraw;
  HANDLE  hVideoPort;
  DWORD  dwFieldNum;
} DDSETFIELDNUM, FAR *LPDDSETFIELDNUM;

Members

hDirectDraw
Specifies the DirectDraw handle.
hVideoPort
Specifies the video port extensions (VPE) object handle.
dwFieldNum
Specifies the hardware video port's field number to be set.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
