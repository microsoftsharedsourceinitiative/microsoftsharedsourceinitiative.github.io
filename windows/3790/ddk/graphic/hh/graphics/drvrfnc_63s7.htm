DrvWriteSpoolBuf

The DrvWriteSpoolBuf function pointed to by this function pointer is obsolete. Windows 2000 and later render plug-ins should use IPrintOemDriverUni::DrvWriteSpoolBuf (Unidrv plug-ins), IPrintOemDriverPS::DrvWriteSpoolBuf (PScript plug-ins), or IPrintCorePS2::DrvWriteSpoolBuf (PScript plug-ins). 

typedef DWORD 
  (APIENTRY PFN_DrvWriteSpoolBuf)(
    PDEVOBJ  pdevobj,
    PVOID  pBuffer,
    DWORD  cbSize
    );

The preceding function pointer prototype defines the type of the DrvWriteSpoolBuf member of the DRVPROCS structure.
