Using Gamma Controls

This topic applies only to Windows Me.

To determine whether a specific graphics card and monitor support gamma calibration, call IDirectDraw7::GetCaps and examine the dwCaps2 member of the associated DDCAPS structure. If the DDCAPS2_CANCALIBRATEGAMMA capability flag is present, then gamma calibration is supported.

You can retrieve the IDirectDrawGammaControl interface by querying the surface with the IID_IDirectDrawGammaControl reference identifier. The IDirectDrawGammaControl::GetGammaRamp method retrieves the red, green, and blue gamma ramps for the primary surface.
