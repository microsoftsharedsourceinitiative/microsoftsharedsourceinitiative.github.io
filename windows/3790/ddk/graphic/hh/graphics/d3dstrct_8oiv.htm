D3DNTHAL_Xxx Structures

This table contains all the Direct3D driver structures whose names have the form D3DNTHAL_Xxx. These structures are identical to those whose names have the form D3DHAL_Xxx, which is the form used on Windows 98/Me for DirectX 7.0 and on both Windows 98/Me and Windows 2000 and later for DirectX 8.0 and later versions. The include file dx95type.h contains type defines from D3DNTHAL_Xxx to D3DHAL_Xxx.
