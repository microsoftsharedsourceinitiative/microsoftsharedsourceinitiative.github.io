Third Part of Macroblock Control Command Structure

If the bPicIntra member of DXVA_PictureParameters is 1, the macroblock control command structure ends with the data described in the Second Part of Macroblock Control Command Structure. If bPicIntra is zero, the following additional data elements are included in the macroblock control command to control the motion compensation process. The data that follows is an array of DXVA_MVvalue structures contained in the MVector member of the macroblock control command structure. The number of elements in MVector depends on the type of picture specified by the members of DXVA_PictureParameters in the following table.

bPicOBMC bPicBinPB bPic4MVallowed Number of Elements in MVector
0 0 0 4
0 0 1 4
0 1 0 4
0 1 1 5
1 0 0 10
1 0 1 10
1 1 0 11
1 1 1 11

Note  The number of motion vectors specified in the MVector arrays for the macroblock control command structures defined in the dxva.h file is four, as this is the most commonly used form of the structure.
