DDGETSURFACESTATEIN

The DDGETSURFACESTATEIN structure contains the DirectDraw and DirectDraw surface handle information. 

typedef struct _DDGETSURFACESTATEIN {
  HANDLE  hDirectDraw;
  HANDLE  hSurface;
} DDGETSURFACESTATEIN, FAR *LPDDGETSURFACESTATEIN;

Members

hDirectDraw
Specifies the DirectDraw handle.
hSurface
Specifies the DirectDrawSurface handle.

Defined in ddkmapi.h. Include ddkmapi.h.
