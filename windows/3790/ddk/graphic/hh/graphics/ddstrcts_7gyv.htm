DDGETFIELDNUMIN

The DDGETFIELDNUMIN structure contains the DirectDraw and video port extensions (VPE) object handle information. 

typedef struct _DDGETFIELDNUMIN {
  HANDLE  hDirectDraw;
  HANDLE  hVideoPort;
} DDGETFIELDNUMIN, FAR *LPDDGETFIELDNUMIN;

Members

hDirectDraw
Specifies the DirectDraw handle.
hVideoPort
Specifies the VPE object handle.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
