Compatibility Testing Requirements for Display and Video Miniport Drivers

This section lists some of the requirements for NT-based operating system graphics drivers and associated components to meet the Windows Hardware Quality Lab (WHQL) test criteria. The drivers that meet or exceed these tests are authorized to display the Microsoft® Windows® Logo. For more information, go to the Windows Platform Development Web site.
