ISNMP SetList Method

The SetList method allows an ASP Web page to associate values with an array of SNMP OIDs.

[id(8), helpstring("method SetList")]
  HRESULT
    SetList(
    [in] VARIANT  *varName,
    [in] VARIANT  *varValue
    );

Parameters

varName
Caller-supplied pointer to an array of OID strings.
varValue
Caller-supplied pointer to an array of OID values.

Return Value

S_OK The operation succeeded.
E_FAIL The ISNMP Open method has not been called.
E_INVALIDARG The specified OID is invalid.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

The ISNMP Open method must be called before the SetList method can be called. The method calls SnmpMgrRequest (described in the Platform SDK documentation) to set OID values.

VBScript Example

    Dim StrIP, strCommunity, objSNMP, OIDArray, OIDValueArray
    strIP = Session("MS_IPaddress")
    strCommunity = Session ("MS_Community")
    Set objSNMP = Server.CreateObject("OlePrn.OleSNMP")
    objSNMP.Open strIP, strCommunity, 2, 1000
    OIDArray = Array("25.3.2.1.5", "25.3.5.1.1")
    ...
    Determine values to assign to OIDs, store in OIDArray.
    ...
    OIDValueArray = objSNMP.SetList (OIDArray)
