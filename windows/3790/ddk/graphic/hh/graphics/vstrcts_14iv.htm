Video Port/Miniport Structures

This section describes structures used by video miniport drivers that are linked with the video port driver. Structures are listed in alphabetic order. See Video Miniport Header Files, Sample Code, and References for the include file that contains definitions for these structures.

For more information about video requests and IOCTLs handled by the port-miniport driver pair, see Video I/O Control Codes.
