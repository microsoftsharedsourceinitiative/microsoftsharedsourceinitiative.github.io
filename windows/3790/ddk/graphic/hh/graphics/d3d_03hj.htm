Using DXVA with 2D Operations

DirectX 9.0 and later drivers use the D3DDP2OP_BLT operation code to perform blits between DirectX Video Acceleration (DXVA) surfaces. Therefore, if the runtime detects a DirectX 9.0 or later driver, the runtime must call the driver's D3dCreateSurfaceEx function to create any DXVA (or 2D-only) surface.
