OEMStretchBltROP

The OEMStretchBltROP function performs a stretching bit-block transfer using a ROP.

BOOL APIENTRY
  OEMStretchBltROP(
    SURFOBJ  *psoDest,
    SURFOBJ  *psoSrc,
    SURFOBJ  *psoMask,
    CLIPOBJ  *pco,
    XLATEOBJ  *pxlo,
    COLORADJUSTMENT  *pca,
    POINTL  *pptlHTOrg,
    RECTL  *prclDest,
    RECTL  *prclSrc,
    POINTL  *pptlMask,
    ULONG  iMode,
    BRUSHOBJ *pbo,
    ROP4  rop4
    );

Headers

Declared in printoem.h. Include printoem.h.

Comments

See DrvStretchBltROP.

This function is not supported on Windows NT 4.0 and earlier.

Do not directly hook out this drawing function. Instead, implement OEMEnableDriver so that it fills in a DRVENABLEDATA structure listing all of the drawing DDIs to be hooked out.

If you call into the core driver, cast the call using the PFN_DrvStretchBltROP function pointer.
