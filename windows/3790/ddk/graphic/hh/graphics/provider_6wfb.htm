Setting Port Timeout Values

If you are writing a port monitor for a port that has modifiable time-out values, the time-out values should be initialized from within the monitor’s OpenPort function. For example the OpenPort function in localmon.dll, the sample port monitor, calls the SetCommTimeouts function, described in the Platform SDK documentation, for this purpose.

Additionally, a port monitor can optionally provide a SetPortTimeOuts function, which can be called by language monitors. The function is called by pjlmon.dll, the sample language monitor. The print spooler does not call SetPortTimeOuts.
