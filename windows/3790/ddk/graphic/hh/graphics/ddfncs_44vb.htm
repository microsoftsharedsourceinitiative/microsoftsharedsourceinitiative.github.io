16-bit Helper Functions

This section contains general information about implementing a 16-bit DirectDraw HAL extension to the display driver for Windows 98/Me (that is, the 16-bit GDI driver).

The following helper functions are the only ones that may need to be included in the 16-bit HAL (depending on the capabilities of your display hardware):

buildDDHALInfo16

DDCreateDriverObject

HALDestroyDriver

Note  Although any of the callback functions of the 32-bit DirectDraw HAL (32-bit Callback Functions) can be implemented in 16-bit code, in the interest of speed — and, potentially, forward compatibility — it is wisest to leave them in the 32-bit HAL.
