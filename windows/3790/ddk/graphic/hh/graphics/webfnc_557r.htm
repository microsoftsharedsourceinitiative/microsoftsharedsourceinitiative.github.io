ISNMP Close Method

The Close method allows an ASP Web page to close a communication path to an SNMP agent.

[id(5), helpstring("method Close")]
  HRESULT
    Close(VOID);

Return Value

The method always returns S_OK.

Comments

The method closes the communication path that was created by the last call to the ISNMP Open method. It calls SnmpMgrClose, which is described in the Platform SDK documentation.

VBScript Example

    Dim StrIP, strCommunity, objSNMP
    strIP = Session("MS_IPaddress")
    strCommunity = Session ("MS_Community")
    Set objSNMP = Server.CreateObject("OlePrn.OleSNMP")
    objSNMP.Open strIP, strCommunity, 2, 1000
    ...
    objSNMP.Close
