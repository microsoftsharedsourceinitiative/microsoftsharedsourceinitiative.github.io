GET_ADAPTER_POWER_STATE_CAPS

The GET_ADAPTER_POWER_STATE_CAPS function returns the power states supported by the display adapter.

DWORD
  cdecl GetAdapterPowerStateCaps(
    DEVNODE  devnode
    );

Parameters

devnode
Specifies the device node for the device.

Return Value

GET_ADAPTER_POWER_STATE_CAPS returns a bitmask containing an ORed combination of the following flags. The CM_POWERSTATE_HIBERNATE flag is used to indicate support for the Hibernate feature.

Flag Value (hexadecimal)
CM_POWERSTATE_D0 0x00000001
CM_POWERSTATE_D1 0x00000002
CM_POWERSTATE_D2 0x00000004
CM_POWERSTATE_D3 0x00000008
CM_POWERSTATE_HIBERNATE 0x00000010
