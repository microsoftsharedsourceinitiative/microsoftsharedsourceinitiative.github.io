Plotter Driver Minidrivers

Model-specific minidrivers for the Microsoft Plotter Driver are vendor-supplied binary .pcd files created from text files that describe a device's characteristics.

PCD Files

To generate a .pcd file, you must first create a text file using the PCD source file format. You must then run plotgpc.exe, which is included with this DDK. This program will convert a text file into a binary .pcd file. Use the following command syntax:

plotgpc source-file-path.txt target-file-path.pcd

For both the source and destination files, you must explicitly specify file name extensions; defaults are not supported.

A sample text file that can be used as input to plotgpc.exe is included in the sample plotter driver files.
