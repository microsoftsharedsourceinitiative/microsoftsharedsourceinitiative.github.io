DDGETPREVIOUSAUTOFLIPOUTINFO

The DDGETPREVIOUSAUTOFLIPOUTINFO structure provides the surface data. 

typedef struct _DDGETPREVIOUSAUTOFLIPOUTINFO {
  DWORD  dwSurfaceIndex;
  DWORD  dwVBISurfaceIndex;
} DDGETPREVIOUSAUTOFLIPOUTINFO, *PDDGETPREVIOUSAUTOFLIPOUTINFO;

Members

dwSurfaceIndex
Specifies the current index in the autoflip chain of the current surface.
dwVBISurfaceIndex
Specifies the current index in the autoflip chain of the current VBI surface.
