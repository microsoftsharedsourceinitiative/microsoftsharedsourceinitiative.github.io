DdSyncSurfaceData

The DdSyncSurfaceData callback function sets and modifies surface data before it is passed to the miniport. 

DWORD
  (APIENTRY *PDD_KERNELCB_SYNCSURFACE)(
    PDD_SYNCSURFACEDATA  lpSyncSurface
    );

Parameters

lpSyncSurface
Points to a DD_SYNCSURFACEDATA structure that contains the surface data.

Return Value

DdSyncSurfaceData returns one of the following callback codes:

DDHAL_DRIVER_HANDLED

DDHAL_DRIVER_NOTHANDLED

Headers

Declared in ddrawint.h. Include winddi.h.
