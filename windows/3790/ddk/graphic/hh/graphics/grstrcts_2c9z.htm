Display and Print Devices: Windows DDK

DRVENABLEDATA

The DRVENABLEDATA structure contains a pointer to an array of DRVFN structures and the graphics DDI version number of the structures.

typedef struct tagDRVENABLEDATA {
  ULONG  iDriverVersion;
  ULONG  c;
  DRVFN  *pdrvfn;
} DRVENABLEDATA, *PDRVENABLEDATA;

Members

iDriverVersion
Specifies the graphics DDI version number for the NT-based operating system for which the driver is targeted. This member can be set to one of the following values:
Value Operating System Version
DDI_DRIVER_VERSION_NT4 Windows NT 4.0
DDI_DRIVER_VERSION_SP3 Windows NT 4.0 Service Pack 3
DDI_DRIVER_VERSION_NT5 Windows 2000
DDI_DRIVER_VERSION_NT5_01 Windows XP

See the Comments section for more information.

c
Specifies the number of DRVFN structures in the buffer pointed to by the pdrvfn member.
pdrvfn
Pointer to a buffer containing an array of DRVFN structures.

Headers

Declared in winddi.h. Include winddi.h.

Comments

To run on these NT-based operating systems versions, the iDriverVersion member must be set as follows:

Windows Version Value of iDriverVersion
Windows NT 4.0
  iDriverVersion == DDI_DRIVER_VERSION_NT4
Windows NT 4.0 SP3
  DDI_DRIVER_VERSION_NT4 <= iDriverVersion <= DDI_DRIVER_VERSION_SP3
Windows 2000
  DDI_DRIVER_VERSION_NT4 <= iDriverVersion <= DDI_DRIVER_VERSION_NT5
Windows XP
  DDI_DRIVER_VERSION_NT4 <= iDriverVersion <= DDI_DRIVER_VERSION_NT5_01

As the table shows, a driver for which iDriverVersion is set to DDI_DRIVER_VERSION_NT4 can run on any of these operating system versions, but a driver for which iDriverVersion is set to DDI_DRIVER_VERSION_NT5_01 can currently run only on Windows XP.

Note:  If a driver implements a DrvXxx graphics DDI that is not supported in all versions of Windows, the driver may not specify a DRVFN entry for that graphics DDI when running on versions of Windows that do not support it. If the driver does specify a DRVFN entry for such a graphics DDI, Windows will reject the driver. The permedia2 sample demonstrates how to specify different DRVFN structures for different versions of Windows.
