DDOPENSURFACEIN

The DDOPENSURFACEIN structure contains the DirectDrawSurface object information. 

typedef struct _DDOPENSURFACEIN {
  HANDLE  hDirectDraw;
  ULONG_PTR  dwSurfaceHandle;
  LPDD_NOTIFYCALLBACK  pfnSurfaceClose;
  PVOID  pContext;
} DDOPENSURFACEIN, FAR *LPDDOPENSURFACEIN;

Members

hDirectDraw
Specifies the DirectDraw object to which the surface handle is associated.
dwSurfaceHandle
Specifies the DirectDrawSurface handle passed down from user mode.
pfnSurfaceClose
Points to a pfnSurfaceClose callback function that is called if the DirectDraw object goes away.
pContext
Contains a value that is passed if the pfnSurfaceClose callback function is ever called.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
