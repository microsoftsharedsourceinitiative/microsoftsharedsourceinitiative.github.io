MibErrorDscp

The MibErrorDscp property allows an ASP Web page to convert an SNMP MIB error code into a text description of the error.

[propget, id(22), helpstring("property MibErrorDscp")]
  HRESULT
    MibErrorDscp(
    [in] DWORD  dwError,
    [out, retval] BSTR  *pVal
    );

Parameters

dwError
Caller-supplied SNMP MIB error code.
pVal
Caller-supplied location to receive a pointer to a string containing a text description of the error.

Return Value

S_OK The operation succeeded.
E_OUTOFMEMORY Out of memory.

Win32 error codes can also be returned.

Comments

VBScript Example

    Dim objPrinter, MIBErrorCode, MIBErrorString
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    ...
    Get error code from MIB.
    ...
    MIBErrorString = objPrinter.MibErrorDscp(ErrorCodeMIB)
