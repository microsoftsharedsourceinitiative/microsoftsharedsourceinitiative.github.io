DD_KERNELCALLBACKS

The DD_KERNELCALLBACKS structure contains entry pointers to the DirectDraw kernel-mode callbacks that the driver supports.

typedef struct _DD_KERNELCALLBACKS {
  DWORD  dwSize;
  DWORD  dwFlags;
  PDD_KERNELCB_SYNCSURFACE  SyncSurfaceData;
  PDD_KERNELCB_SYNCVIDEOPORT  SyncVideoPortData;
} DD_KERNELCALLBACKS, *PDD_KERNELCALLBACKS;

Members

dwSize
Specifies the size in bytes of this DD_KERNELCALLBACKS structure.
dwFlags
Indicates what DirectDraw kernel callbacks the driver has implemented. For every bit set in dwFlags, the driver must initialize the corresponding function pointer member of this structure. This member can be one or more of the following flags:

DDHAL_KERNEL_SYNCSURFACEDATA
DDHAL_KERNEL_SYNCVIDEOPORTDATA

SyncSurfaceData
Points to the driver-supplied DdSyncSurfaceData object callback.
SyncVideoPortData
Points to the driver-supplied DdSyncVideoPortData object callback.

Headers

Defined in ddrawint.h. Include winddi.h.

Comments

Entries that the display driver does not use should be set to NULL. The driver should initialize this structure when its DdGetDriverInfo function is called with the GUID_KernelCallbacks GUID.
