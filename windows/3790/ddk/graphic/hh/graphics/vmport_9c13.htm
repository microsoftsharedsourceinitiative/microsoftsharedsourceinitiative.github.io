Releasing Memory from AGP Use

The miniport driver is responsible for freeing and releasing all virtual and physical memory that it has reserved and committed as outlined in Accessing System Memory through AGP. It does so by calling the following video port driver AGP service routines:

  • AgpFreeVirtual unmaps virtual memory locked by a prior call to AgpCommitVirtual.
  • AgpReleaseVirtual releases a range of virtual addresses reserved by a prior call to AgpReserveVirtual.
  • AgpFreePhysical unmaps physical memory locked by a prior call to AgpCommitPhysical.
  • AgpReleasePhysical releases a range of physical addresses reserved by a prior call to AgpReservePhysical.
