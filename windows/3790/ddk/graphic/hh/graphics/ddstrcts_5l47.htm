DDRAWI_DDVIDEOPORT_LCL

The DDRAWI_DDVIDEOPORT_LCL structure contains local data for each individual DirectDraw video port object. 

typedef struct _DDRAWI_DDVIDEOPORT_LCL {
  LPDDRAWI_DIRECTDRAW_LCL  lpDD;
  DDVIDEOPORTDESC  ddvpDesc;
  DDVIDEOPORTINFO  ddvpInfo;
  LPDDRAWI_DDRAWSURFACE_INT  lpSurface;
  LPDDRAWI_DDRAWSURFACE_INT  lpVBISurface;
  LPDDRAWI_DDRAWSURFACE_INT *lpFlipInts;
  DWORD  dwNumAutoflip;
  DWORD  dwProcessID;
  DWORD  dwStateFlags;
  DWORD  dwFlags;
  DWORD  dwRefCnt;
  FLATPTR  fpLastFlip;
  ULONG_PTR  dwReserved1;
  ULONG_PTR  dwReserved2;
  HANDLE  hDDVideoPort;
  DWORD  dwNumVBIAutoflip;
  LPDDVIDEOPORTDESC  lpVBIDesc;
  LPDDVIDEOPORTDESC  lpVideoDesc;
  LPDDVIDEOPORTINFO  lpVBIInfo;
  LPDDVIDEOPORTINFO  lpVideoInfo;
  DWORD  dwVBIProcessID;
} DDRAWI_DDVIDEOPORT_LCL;

Members

lpDD
Points to a DDRAWI_DIRECTDRAW_LCL structure that contains the local DirectDraw object's data.
ddvpDesc
Is a DDVIDEOPORTDESC structure that describes the video port object.
ddvpInfo
Is a DDVIDEOPORTINFO structure that describes the transfer of video data to a surface.
lpSurface
Points to a DDRAWI_DDRAWSURFACE_INT structure for the surface receiving the data.
lpVBISurface
Points to a DDRAWI_DDRAWSURFACE_INT structure for the surface receiving the VBI data.
lpFlipInts
Used by DirectDraw and should not be filled in by the driver.
dwNumAutoflip
Specifies the number of current autoflip surfaces. The driver should use this member for information purposes only.
dwProcessID
Used by DirectDraw and should not be filled in by the driver.
dwStateFlags
Used by DirectDraw and should not be filled in by the driver.
dwFlags
Used by DirectDraw and should not be filled in by the driver.

dwRefCnt
Used by DirectDraw and should not be filled in by the driver.
fpLastFlip
Used by DirectDraw and should not be filled in by the driver.
dwReserved1
Reserved for use by the display driver.
dwReserved2
Reserved for use by the display driver.
hDDVideoPort
Used by DirectDraw and should not be filled in by the driver.
dwNumVBIAutoflip
Specifies the number of VBI surfaces currently being autoflipped. The driver should use this member for information purposes only.
lpVBIDesc
Points to a DDVIDEOPORTDESC structure that describes the video port object that is used for VBI data.
lpVideoDesc
Points to a DDVIDEOPORTDESC structure that describes the video port object that is used for video.
lpVBIInfo
Points to a DDVIDEOPORTINFO structure that describes the transfer of VBI data to a surface.
lpVideoInfo
Points to a DDVIDEOPORTINFO structure that describes the transfer of video to a surface.
dwVBIProcessID
Specifies the ID of the process owning this video port.

Headers

Defined in ddrawi.h. Include ddrawi.h.

Comments

The DDRAWI_DDVIDEOPORT_LCL structure is similar in purpose to the Windows 2000 and later structure, DD_VIDEOPORT_LOCAL.
