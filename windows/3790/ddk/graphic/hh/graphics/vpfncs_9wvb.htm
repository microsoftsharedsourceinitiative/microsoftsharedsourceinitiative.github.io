VideoPortClearEvent

The VideoPortClearEvent function sets a given event object to the nonsignaled state.

VOID
  VideoPortClearEvent(
    IN PVOID  HwDeviceExtension,
    IN PEVENT  pEvent
    );

Parameters

HwDeviceExtension
Pointer to the miniport driver's device extension.
pEvent
Pointer to the event object.

Return Value

None

Headers

Declared in video.h. Include video.h.

Comments

This function is available in Windows XP and later.

To set the state of an event object to the signaled state, use VideoPortSetEvent.

Callers of VideoPortClearEvent must be running at IRQL <= DISPATCH_LEVEL.
