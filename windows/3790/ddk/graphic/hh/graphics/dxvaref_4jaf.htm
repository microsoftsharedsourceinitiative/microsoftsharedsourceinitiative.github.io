DXVA_DeinterlaceQueryModeCaps

The DXVA_DeinterlaceQueryModeCaps structure is sent by the driver to the host decoder to specify the capabilities allowed for driver deinterlacing.

typedef struct __DXVA_DeinterlaceQueryModeCaps {
  DWORD  Size;
  GUID  Guid;
  DXVA_VideoDesc  VideoDesc;
} DXVA_DeinterlaceQueryModeCaps;

Members

Size
Indicates the size of this structure.
Guid
Specifies which type of deinterlacing that the driver should return capabilities for.
VideoDesc
Specifies a DXVA_VideoDesc structure that describes the type of video that is to be deinterlaced.

Headers

Declared in dxva.h. Include dxva.h.
