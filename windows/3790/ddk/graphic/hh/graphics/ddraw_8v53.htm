DDIs and APIs Related to Gamma Ramp Support

This topic applies only to Windows Me.

To add gamma ramp support to your Windows 98/Me 16-bit GDI driver:

  1. Export DDIGammaRamp as ordinal 32.
  2. Set the C1_GAMMA_RAMP bit in the dpCaps1 member of the GDIINFO structure.
  3. Implement DDIGammaRamp.

The functions, methods, and structure in the following list play roles in gamma ramp support.

DDIGammaRamp

GetDeviceGammaRamp

SetDeviceGammaRamp

IDirectDrawGammaControl::GetGammaRamp
