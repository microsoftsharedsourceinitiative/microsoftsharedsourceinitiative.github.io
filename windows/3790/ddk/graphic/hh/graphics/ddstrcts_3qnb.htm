DDRAWI_DDRAWSURFACE_INT

The DDRAWI_DDRAWSURFACE_INT structure contains the DirectDrawSurface object's interface information. 

typedef struct _DDRAWI_DDRAWSURFACE_INT {
  LPVOID  lpVtbl;
  LPDDRAWI_DDRAWSURFACE_LCL  lpLcl;
  LPDDRAWI_DDRAWSURFACE_INT  lpLink;
  DWORD   dwIntRefCnt;
} DDRAWI_DDRAWSURFACE_INT;

Members

lpVtbl
Used by DirectDraw and should not be filled in by the driver.
lpLcl
Points to a DDRAWI_DDRAWSURFACE_LCL structure that contains the local object.
lpLink
Used by DirectDraw and should not be filled in by the driver.
dwIntRefCnt
Used by DirectDraw and should not be filled in by the driver.
