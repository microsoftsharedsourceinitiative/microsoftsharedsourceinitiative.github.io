IAspHelp Open Method

The Open method allows an ASP Web page to open access to a printer.

[id(1), helpstring("method Open")]
  HRESULT
    Open(
    [in] BSTR  bstrPrinterName
    );

Parameters

bstrPrinterName
Caller-supplied pointer to a string representing a printer name.

Return Value

S_OK The operation succeeded.

Win32 error codes can also be returned.

Comments

The method obtains access to the specified printer by calling the print spooler's OpenPrinter function (described in the Platform SDK documentation). The printer remains open until the IAspHelp Close method is called, or until Open is called again with a different printer name.

VBScript Example

    Dim objPrinter
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    objPrinter.Open strPrinter
