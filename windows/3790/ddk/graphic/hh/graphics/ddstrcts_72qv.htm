DDLOCKIN

The DDLOCKIN structure contains the DirectDraw object and DirectDraw surface handle information. 

typedef struct _DDLOCKIN {
  HANDLE  hDirectDraw;
  HANDLE  hSurface;
} DDLOCKIN, FAR *LPDDLOCKIN;

Members

hDirectDraw
Specifies the DirectDraw handle.
hSurface
Specifies the DirectDrawSurface handle.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
