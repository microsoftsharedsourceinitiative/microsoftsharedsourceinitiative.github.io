DD_MISCELLANEOUS2CALLBACKS

The DD_MISCELLANEOUS2CALLBACKS structure is used to return the addresses of the four new callback routines for DirectX® 7.0 that are exposed through DdGetDriverInfo by responding to the GUID_Miscellaneous2Callbacks GUID.

typedef struct _DD_MISCELLANEOUS2CALLBACKS {
    DWORD                dwSize;
    DWORD                dwFlags;
    PDD_ALPHABLT         AlphaBlt;
    PDD_CREATESURFACEEX  CreateSurfaceEx;
    PDD_GETDRIVERSTATE   GetDriverState;
    PDD_DESTROYDDLOCAL   DestroyDDLocal;
} DD_MISCELLANEOUS2CALLBACKS, *PDD_MISCELLANEOUS2CALLBACKS;

Members

dwSize
Specifies the size, in bytes, of this structure.
dwFlags
Specifies the callbacks supported and flags concerning their usage.
Value Meaning
DDHAL_MISC2CB32_ALPHABLT The DdAlphaBlt callback is supported.
DDHAL_MISC2CB32_CREATESURFACEEX The D3dCreateSurfaceEx callback is supported.
DDHAL_MISC2CB32_GETDRIVERSTATE The D3dGetDriverState callback is supported.
DDHAL_MISC2CB32_DESTROYDDLOCAL The D3dDestroyDDLocal callback is supported.

AlphaBlt
Points to the driver's DdAlphaBlt implementation.
CreateSurfaceEx
Points to the driver's D3dCreateSurfaceEx implementation. This callback creates an association between a DirectDraw® surface and a small integer handle.
GetDriverState
Points to the driver's D3dGetDriverState implementation.
DestroyDDLocal
Points to the driver's D3dDestroyDDLocal implementation. Used to destroy the local copy of the device context.

Headers

Defined in ddrawint.h. Include winddi.h.
