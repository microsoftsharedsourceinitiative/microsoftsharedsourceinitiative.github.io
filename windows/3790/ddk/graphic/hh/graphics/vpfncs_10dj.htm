VideoPortWriteRegisterUchar

The VideoPortWriteRegisterUchar function writes a byte to a mapped register.

VOID
  VideoPortWriteRegisterUchar(
    IN PUCHAR  Register,
    IN UCHAR  Value
    );

Parameters

Register
Pointer to the register. The given Register must be in a mapped memory-space range returned by VideoPortGetDeviceBase.
Value
Specifies a byte to be transferred to the adapter.

Return Value

None

Headers

Declared in video.h. Include video.h.

Comments

A miniport driver's HwVidInterrupt or HwVidSynchronizeExecutionCallback function can call VideoPortWriteRegisterUchar.

Callers of VideoPortWriteRegisterUchar can be running at any IRQL, provided that the memory pointed to by the Register parameter is resident, mapped device memory.
