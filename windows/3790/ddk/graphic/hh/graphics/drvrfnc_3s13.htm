IPrintOemPS::DevMode

The IPrintOemPS::DevMode method, provided by rendering plug-ins for Pscript5, performs operations on private DEVMODE members.

STDMETHOD
  (DevMode) (
    THIS_
    DWORD  dwMode,
    POEMDMPARAM  pOemDMParam
    ) PURE;

If you are providing a user interface plug-in for Pscript5, and if you are adding private members to the driver's DEVMODE structure, you must implement both the IPrintOemUI::DevMode and the IPrintOemPS::DevMode methods. The code implementing these methods must be identical and can be placed in a library that is statically linked to both the UI plug-in and the rendering plug-in.

For a description of the IPrintOemPS::DevMode method, see IPrintOemUI::DevMode.
