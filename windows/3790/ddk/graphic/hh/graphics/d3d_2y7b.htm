Display and Print Devices: Windows DDK

Present and GetBltStatus

For DX8 the runtime no longer calls DdGetBltStatus on blts involving system memory surfaces. This was always the behavior on Windows® 2000 and is now the behavior on Windows 98/Me. The result is that asynchronous DMA to or from system memory surfaces is no longer possible (see Asynchronous DMA Support in DirectDraw for details). DX8 drivers should not page lock system memory surfaces by themselves, and system memory to video memory transfers should be synchronous.
