EngDeleteSemaphore

The EngDeleteSemaphore function deletes a semaphore object from the system's resource list.

VOID
  EngDeleteSemaphore(
    IN OUT HSEMAPHORE  hsem
    );

Parameters

hsem
Handle to the semaphore to be deleted. The semaphore was created in EngCreateSemaphore.

Return Value

None

Headers

Declared in winddi.h. Include winddi.h.

See Also

EngAcquireSemaphore, EngCreateSemaphore, EngReleaseSemaphore
