SetPortTimeOuts

A port monitor server DLL's SetPortTimeOuts function sets port time-out values for an open port.

BOOL
  (WINAPI *pfnSetPortTimeOuts)(
    HANDLE  hPort,
    LPCOMMTIMEOUTS  lpCTO,
    DWORD  reserved
    );

Parameters

hPort
Caller-supplied handle to the open port on which to set the time-out values.
lpCTO
Caller-supplied pointer to a COMMTIMEOUTS structure (described in the Platform SDK documentation).
reserved
Reserved for future use. Must be set to zero.

Return Value

If the operation succeeds, the function should return TRUE. Otherwise it should return FALSE.

Headers

Declared in winsplp.h. Include winsplp.h.

Comments

A port monitor server DLL's SetPortTimeOuts function allows a language monitor to specify port time-out values for an open port. The function is optional, and only needs to be provided if the port monitor controls a port that allows modifying of port time-out values. If the function is defined, its address must be included in a MONITOR2 structure.

The function is called by pjlmon.dll, the sample language monitor, and you can write a customized language monitor that calls it. The print spooler does not call SetPortTimeOuts.

The port monitor should initialize the port's time-out values from within its OpenPort function.
