DXVA_SampleFormat

The DXVA_SampleFormat enumeration type describes the format of data that the input sample contains.

typedef enum _DXVA_SampleFormat {
  DXVA_SamplePreviousFrame  = 0x0001,
  DXVA_SampleProgressiveFrame  = 0x0002,
  DXVA_SampleFieldInterleavedEvenFirst  = 0x0003,
  DXVA_SampleFieldInterleavedOddFirst  = 0x0004,
  DXVA_SampleFieldSingleEven  = 0x0005,
  DXVA_SampleFieldSingleOdd  = 0x0006,
} DXVA_SampleFormat;

Members

DXVA_SamplePreviousFrame
Specifies the sample that contains the previous output frame.
DXVA_SampleProgressiveFrame
Specifies the sample that contains a progressive frame.
DXVA_SampleFieldInterleavedEvenFirst
Specifies the sample that contains two interleaved fields, the even field is temporally first.
DXVA_SampleFieldInterleavedOddFirst
Specifies the sample that contains two interleaved fields, the odd field is temporally first.
DXVA_SampleFieldSingleEven
Specifies the sample that contains an even interleaved field.
DXVA_SampleFieldSingleOdd
Specifies the sample that contains an odd interleaved field.
