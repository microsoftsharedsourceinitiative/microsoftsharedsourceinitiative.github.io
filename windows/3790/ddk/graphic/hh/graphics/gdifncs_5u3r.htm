EngReleaseSemaphore

The EngReleaseSemaphore function releases the specified semaphore.

VOID
  EngReleaseSemaphore(
    IN HSEMAPHORE  hsem
    );

Parameters

hsem
Handle to the semaphore to be released.

Return Value

None

Headers

Declared in winddi.h. Include winddi.h.

Comments

EngReleaseSemaphore releases the semaphore's exclusive lock on a driver's resource and reenables the delivery of special kernel asychronous procedure calls.

The lock and asynchronous procedure call suspension were acquired in a call to EngAcquireSemaphore.
