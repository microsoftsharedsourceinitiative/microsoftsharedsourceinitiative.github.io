DDCLOSEHANDLE

The DDCLOSEHANDLE structure contains the DirectDraw object, surface, video port extensions (VPE) object, or VPE capture handle. 

typedef struct _DDCLOSEHANDLE {
  HANDLE  hHandle;
} DDCLOSEHANDLE, FAR *LPDDCLOSEHANDLE;

Members

hHandle
Handle to be closed.
