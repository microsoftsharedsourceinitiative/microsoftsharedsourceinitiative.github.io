WMV8_A, WMV8_B, WMV9_A, WMV9_B, WMV9_Ai, and WMV9_Bi

The WMV8_A, WMV8_B, WMV9_A, WMV9_B, WMV9_Ai, and WMV9_Bi restricted profiles contain the sets of features required for support of Windows Media Video, versions 8 and 9. For more information on these profiles, see the Windows Media Porting Kit.
