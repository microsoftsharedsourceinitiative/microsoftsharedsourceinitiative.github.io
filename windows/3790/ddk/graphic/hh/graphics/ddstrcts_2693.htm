DDGETTRANSFERSTATUSOUTINFO

The DDGETTRANSFERSTATUSOUTINFO structure contains the transfer status information. 

typedef struct _DDGETTRANSFERSTATUSOUTINFO {
  DWORD  dwTransferID;
} DDGETTRANSFERSTATUSOUTINFO, *PDDGETTRANSFEROUTINFO;

Members

dwTransferID
Contains the transfer ID of the bus master that completed. The transfer ID was originally supplied to the driver in the dwTransferID member of the DDTRANSFERININFO structure. The driver receives a pointer to DDTRANSFERININFO in a call to its DxTransfer function.

Headers

Defined in dxmini.h. Include dxmini.h.
