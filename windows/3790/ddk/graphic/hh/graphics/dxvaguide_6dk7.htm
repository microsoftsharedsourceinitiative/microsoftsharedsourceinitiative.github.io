Additional Encouraged Configuration Set

Implementation of additional configurations for software decoders is encouraged. These configurations may exist in hardware and can provide a significant performance benefit relative to those in the minimal interoperability configuration sets.

This additional configuration set is defined in terms of the members of the DXVA_ConfigPictureDecode structure.
