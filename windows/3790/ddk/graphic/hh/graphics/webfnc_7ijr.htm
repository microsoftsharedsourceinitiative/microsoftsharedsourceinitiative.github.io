AspPage

The AspPage property allows an ASP Web page to obtain the directory path to the initial ASP file used for describing printer-specific details.

[propget, id(3), helpstring("property AspPage")]
  HRESULT
    AspPage(
    [in] DWORD  dwPage,
    [out, retval] BSTR  *pVal
    );

Parameters

dwPage
Must be 1.
pVal
Caller-supplied pointer to a location to receive a size-prefixed Unicode string specifying the directory path to the initial Web page describing printer-specific details.

Return Value

S_OK The operation succeeded.
E_HANDLE The IAspHelp Open method has not been called.
E_NOTIMPL An ASP file is not available.
E_POINTER Invalid pVal pointer.

Comments

The name of the printer being queried must have been specified by a previous call to the IAspHelp Open method.

To determine where to find the page's ASP file, the method uses the algorithm described in Which Printer Details Page is Displayed?.

VBScript Example

    Dim objPrinter, str
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    objPrinter.Open strPrinter
    str = objPrinter.ASPPage(1)
