Video Miniport Driver Interface

The following sections list the functions that comprise the video miniport driver interface. Some functions are required, while others are conditionally required or optional, and their inclusion in a miniport driver depends on the adapter's features and the driver writer.

Except for DriverEntry, this documentation uses pseudonames for standard miniport driver functions that suggest the required functionality of each. In other words, the name given to each standard miniport driver function, except DriverEntry, can be anything the miniport driver writer chooses.
