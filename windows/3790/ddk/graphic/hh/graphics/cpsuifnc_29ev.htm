CPSFUNC_SET_RESULT

The CPSFUNC_SET_RESULT function code causes the ComPropSheet function to pass a specified result value to all PFNPROPSHEETUI-typed functions associated with a specified page and its parents.

ComPropSheet Parameters

hComPropSheet
Group parent handle.
Function
CPSFUNC_SET_RESULT
lParam1
Caller-supplied CPSUI handle to the page for which a result value is being passed. If lParam1 is NULL, the CPSUI uses hComPropSheet.
lParam2
Caller-supplied 32-bit DWORD result value.

ComPropSheet Return Value

The ComPropSheet function returns the number of PFNPROPSHEETUI-typed functions that were called. If the handle specified for lParam1 is invalid, the function returns -1.

Headers

Declared in compstui.h. Include compstui.h.

Comments

For more information about setting result values, see the description of the SETRESULT_INFO structure.
