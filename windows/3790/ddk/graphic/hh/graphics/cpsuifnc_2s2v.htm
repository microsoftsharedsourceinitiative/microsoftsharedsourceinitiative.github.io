CPSFUNC_ADD_PROPSHEETPAGE

The CPSFUNC_ADD_PROPSHEETPAGE function code causes the ComPropSheet function to add a property sheet page that is described by a PROPSHEETPAGE structure (described in the Platform SDK documentation).

The ComPropSheet function creates the page by calling the CreatePropertySheetPage function (described in the Platform SDK documentation), passing the PROPSHEETPAGE structure's address.

ComPropSheet Parameters

hComPropSheet
Group parent handle.
Function
CPSFUNC_ADD_PROPSHEETPAGE
lParam1
Pointer to a PROPSHEETPAGE structure.
lParam2
Not used, must be zero.

ComPropSheet Return Value

If the operation succeeds, ComPropSheet returns a CPSUI handle to the added page; otherwise, it returns NULL.

Headers

Declared in compstui.h. Include compstui.h.
