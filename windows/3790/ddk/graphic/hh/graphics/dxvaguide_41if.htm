Sample Functions for ProcAmp Control

The following sample ProcAmp functions are provided to show how to implement ProcAmp control functionality. The ProcAmp control code for the sample functions in this section shows how the deinterlace container device could be implemented by a driver. The deinterlace container device is reused for the ProcAmpControlQueryCaps and ProcAmpControlQueryRange sample functions.

Function Description
ProcAmpControlQueryCaps Signals the VMR to query the driver to determine input requirements of the ProcAmp control device. This function is called on the deinterlace container device.
ProcAmpControlQueryRange Signals the VMR to query the driver to determine the minimum, maximum, step size and default value for each ProcAmp property. This function is called on the deinterlace container device.
ProcAmpControlOpenStream Creates a ProcAmpStream object. This function is called on the ProcAmp control device.
ProcAmpControlBlt Performs the ProcAmp adjustment operation by writing the output to the destination surface. This function is called on the ProcAmp control device.
ProcAmpControlCloseStream Closes the ProcAmp stream object and instructs the device driver to release any hardware resource associated with this stream. This function is called on the ProcAmp control device.

The sample functions in this section map to the callback functions defined in the DD_MOTIONCOMPCALLBACKS structure as follows.

Function DD_MOTIONCOMPCALLBACKS member
ProcAmpControlQueryCaps RenderMoComp
ProcAmpControlQueryRange RenderMoComp
ProcAmpControlOpenStream CreateMoComp
ProcAmpControlBlt RenderMoComp
ProcAmpControlCloseStream DestroyMoComp
