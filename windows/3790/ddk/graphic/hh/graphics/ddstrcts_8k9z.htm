DDHALDDRAWFNS

The DDHALDDRAWFNS structure contains pointers to ddraw16.dll entry points.

typedef struct {
  DWORD  dwSize;
  LPDDHAL_SETINFO  lpSetInfo;
  LPDDHAL_VIDMEMALLOC  lpVidMemAlloc;
  LPDDHAL_VIDMEMFREE  lpVidMemFree;
} DDHALDDRAWFNS, FAR *LPDDHALDDRAWFNS;

Members

dwSize
Specifies the size of this structure.
lpSetInfo
Points to the DDHAL_SetInfo function.
lpVidMemAlloc
Points to the DDHAL_VidMemAlloc function.
lpVidMemFree
Points to the DDHAL_VidMemFree function.
