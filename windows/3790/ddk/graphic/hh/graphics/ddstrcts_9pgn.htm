DDGETAUTOFLIPOUT

The DDGETAUTOFLIPOUT structure contains the handle and polarity information returned from the DD_DXAPI_GET_CURRENT_VP_AUTOFLIP_SURFACE and DD_DXAPI_GET_LAST_VP_AUTOFLIP_SURFACE function identifiers of the DxApi function. 

typedef struct _DDGETAUTOFLIPOUT {
  DWORD   ddRVal;
  HANDLE  hVideoSurface;
  HANDLE  hVBISurface;
  BOOL  bPolarity;
} DDGETAUTOFLIPOUT, FAR *LPDDGETAUTOFLIPOUT;

Members

ddRVal
Specifies the location in which DirectDraw writes the return value. A return code of DD_OK indicates success.
hVideoSurface
Handle for the current video surface.
hVBISurface
Handle for the current VBI data.
bPolarity
Specifies whether the field is an even or odd field of an interlaced video signal.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
