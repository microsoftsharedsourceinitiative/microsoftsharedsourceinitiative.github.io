VGA-Compatible, x86-Based Miniports

A VGA-compatible miniport driver has the following features:

  • It is based on the system-supplied VGA miniport driver, with code modified to support adapter-specific features. The system-supplied VGA display drivers use the support provided by VGA-compatible miniport drivers, so the developer of a new miniport driver for a VGA-compatible adapter need not write a new display driver.
  • It provides support for full-screen MS-DOS® applications to do I/O directly to the adapter registers. It also functions as a video validator to prevent such applications from issuing any sequence of instructions that would hang the machine.

Miniport drivers of most SVGA adapters fall in this category, but any new miniport driver for a nonSVGA adapter can provide VGA compatibility support at the discretion of the driver designer.

Self-declared "VGA-compatible" miniport drivers are set up to configure themselves in the registry with VgaCompatible set to one (TRUE).
