ReEnable

The ReEnable function reinitializes the driver object. 

UINT FAR PASCAL _loadds 
  ReEnable(
    LPVOID  lpPDevice,
    LPVOID  lpGDIInfo
    );

Parameters

lpPDevice
Points to a DIBENGINE structure for the display card.
lpGDIInfo
Points to a GDIINFO structure that receives the display card's changed mode information.

Return Value

ReEnable returns TRUE if successful; otherwise, it returns FALSE.

Comments

A graphics driver uses 31 as the export ordinal to export a ReEnable function.

ReEnable can be implemented to call Enable as shown in the following code:

UINT FAR PASCAL _loadds ReEnable(LPVOID lpPDevice, LPVOID lpGDIInfo){
    DPF("ReEnable");
    if (Enable(lpPDevice, EnableDevice, NULL, NULL, NULL)) {
        Enable(lpGDIInfo, InquireInfo, NULL, NULL, NULL);
        return TRUE;
    }
    return FALSE;
}

For information on Enable and other graphics driver functions, GDIINFO, and DIBENGINE, see the Graphics Device Interface and Display Drivers sections of the Windows 98/Me section of the Windows DDK documentation.
