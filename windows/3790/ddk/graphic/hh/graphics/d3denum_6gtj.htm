Direct3D Driver Enumerations

This section describes the enumerated types that are used by Direct3D® drivers. Enumerated types that are used by both Direct3D drivers and applications, such as D3DRENDERSTATETYPE, are described in the DirectX SDK documentation.

Direct3D lists the include files that define Direct3D enumerated types.
