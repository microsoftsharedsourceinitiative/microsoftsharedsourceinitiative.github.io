DD_DESTROYVPORTDATA

The DD_DESTROYVPORTDATA structure contains the information necessary for the driver to clean up.

typedef struct _DD_DESTROYVPORTDATA {
  PDD_DIRECTDRAW_LOCAL  lpDD;
  PDD_VIDEOPORT_LOCAL  lpVideoPort;
  HRESULT  ddRVal;
  VOID  *DestroyVideoPort;
} DD_DESTROYVPORTDATA;

Members

lpDD
Points to a DD_DIRECTDRAW_LOCAL structure that represents the DirectDraw object.
lpVideoPort
Points to a DD_VIDEOPORT_LOCAL structure that represents this video port extensions (VPE) object.
ddRVal
Specifies the location in which the driver writes the return value of the DdVideoPortDestroy callback. A return code of DD_OK indicates success.
DestroyVideoPort
Used by the DirectDraw API and should not be filled in by the driver.

Headers

Defined in ddrawint.h. Include winddi.h.
