ComputerName

The ComputerName property allows an ASP Web page to obtain a print server's name.

[propget, id(18), helpstring("property ComputerName")]
  HRESULT
    ComputerName(
    [out, retval] BSTR  *pVal
    );

Parameters

pVal
Caller-supplied pointer to a location to receive a pointer to a computer name string.

Return Value

S_OK The operation succeeded.
E_OUTOFMEMORY Out of memory.

VBScript Example

    Dim objPrinter, CompName
    strPrinter = Session("MS_printer")
    Set objPrinter = Server.CreateObject ("OlePrn.AspHelp")
    CompName = objPrinter.ComputerName
