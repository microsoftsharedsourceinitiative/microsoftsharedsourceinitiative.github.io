DDCAPBUFFINFO

The DDCAPBUFFINFO structure contains the capture information. 

typedef struct _DDCAPBUFFINFO {
  DWORD  dwFieldNumber;
  DWORD  bPolarity;
  LARGE_INTEGER  liTimeStamp;
  DWORD  ddRVal;
} DDCAPBUFFINFO, FAR * LPDDCAPBUFFINFO;

Members

dwFieldNumber
Indicates the internal field number of the field captured.
bPolarity
Specifies whether the captured field is an even or odd field.
liTimeStamp
Used by DirectDraw and should be ignored by the driver.
ddRVal
Specifies the location in which DirectDraw writes the return value. Contains DD_OK if the capture buffer contains valid data.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
