DDGETFIELDNUMOUT

The DDGETFIELDNUMOUT structure contains the hardware video port's field number. 

typedef struct _DDGETFIELDNUMOUT {
  DWORD  ddRVal;
  DWORD  dwFieldNum;
} DDGETFIELDNUMOUT, FAR *LPDDGETFIELDNUMOUT;

Members

ddRVal
Specifies the location in which DirectDraw writes the return value. A return code of DD_OK indicates success.
dwFieldNum
Specifies the hardware video port's field number.

Headers

Defined in ddkmapi.h. Include ddkmapi.h.
