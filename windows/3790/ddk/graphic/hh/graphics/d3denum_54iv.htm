BREAK PRED Instruction

The BREAK PRED instruction breaks out of the current loop at the nearest endloop or endrep by using one of the components of the predicate register as a condition to determine whether or not to perform the instruction.

Format

instruction token that contains D3DSIO_BREAKP

source parameter token using the D3DSPR_PREDICATE register type with a replicate swizzle (each channel uses the same source component). The NOT (0xd) source modifier in bits 27:24 is permitted.
