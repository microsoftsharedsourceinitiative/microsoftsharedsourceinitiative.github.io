ComPropSheet Function Codes

The following function codes can be passed to CPSUI's ComPropSheet function:

CPSFUNC_ADD_HPROPSHEETPAGE – Adds property sheet pages described by handles.

CPSFUNC_ADD_PCOMPROPSHEETUI – Adds property sheet pages described by a COMPROPSHEETUI structure.

CPSFUNC_ADD_PFNPROPSHEETUI – Requests CPSUI to call a PFNPROPSHEETUI-typed callback.

CPSFUNC_ADD_PROPSHEETPAGE – Adds property sheet pages described by PROPSHEETPAGE structures.

CPSFUNC_DELETE_HCOMPROPSHEET – Deletes property sheet pages described by handles.

CPSFUNC_DO_APPLY_CPSUI – Simulates delivery of the PSN_APPLY notification message.

CPSFUNC_GET_HPSUIPAGES – Obtains property sheet page handles.

CPSFUNC_GET_PAGECOUNT – Obtains a count or property sheet pages.

CPSFUNC_GET_PFNPROPSHEETUI_ICON – Obtains an icon handle.

CPSFUNC_IGNORE_CPSUI_PSN_APPLY – Disables or re-enables CPSUI's handling of the PSN_APPLY notification message.

CPSFUNC_INSERT_PSUIPAGE – Inserts property sheet pages at a specified position.

CPSFUNC_LOAD_CPSUI_ICON – Loads a CPSUI-supplied icon.

CPSFUNC_LOAD_CPSUI_STRING – Loads a CPSUI-supplied text string.

CPSFUNC_QUERY_DATABLOCK – Retrieves a previously stored data block.

CPSFUNC_SET_DATABLOCK – Makes one property sheet page's information available to other pages.

CPSFUNC_SET_DMPUB_HIDEBITS – "Hides" a specified set of document property options, so they are not displayed.

CPSFUNC_SET_FUSION_CONTEXT – Sets a Fusion activation context for the specified page.

CPSFUNC_SET_HSTARTPAGE – Sets a property sheet's initial top page.

CPSFUNC_SET_PSUIPAGE_ICON – Adds, replaces, or removes a property sheet page's tab icon.

CPSFUNC_SET_PSUIPAGE_TITLE – Sets a property sheet page's tab title.

CPSFUNC_SET_RESULT – Sets a result value.
