Sample Rendering Plug-Ins

Source code for sample rendering plug-ins are provided with this DDK. Code is contained in the src\oemdll\oemuni and src\oemdll\oemps source code subdirectories. The code in the first subdirectory creates a plug-in that can be used with Unidrv. The code in the second subdirectory creates a plug-in that can be used with Pscript5.
