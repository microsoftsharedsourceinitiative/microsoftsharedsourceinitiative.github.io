IPrintOemDriverUI COM Interface

The IPrintOemDriverUI COM interface allows a UI plug-in to view and modify information managed by the printer interface DLL for Unidrv or Pscript.

The following table lists and describes all of the methods defined by the IPrintOemDriverUI interface.

Method Description
IPrintOemDriverUI::DrvGetDriverSetting
  Allows a user interface plug-in to obtain the current status of printer features and other internal information.
IPrintOemDriverUI::DrvUpdateUISetting
  Allows a user interface plug-in to notify the driver of a modified user interface option.
IPrintOemDriverUI::DrvUpgradeRegistrySetting
  Allows a user interface plug-in to update device settings stored in the registry.

For more information, see Implementing Printer Driver COM Interfaces.
