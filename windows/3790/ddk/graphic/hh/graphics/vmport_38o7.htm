Supporting Video Miniport Drivers on Multiple Windows Versions

If you plan to modify a video miniport driver written for Windows NT 4.0 to run on a later NT-based operating system version, see Converting a Windows NT 4.0 Miniport Driver to Windows 2000. You can develop a video miniport driver on a current version of Windows, but have it run on an earlier NT-based Windows version. For details, see Using VideoPortGetProcAddress.
