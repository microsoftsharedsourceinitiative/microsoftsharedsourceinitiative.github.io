Standard Bus Interface

The standard bus interface allows device drivers to make direct calls to parent bus driver routines rather than using I/O request packets (IRP) to communicate with the bus driver. In particular, it allows drivers to access routines provided by the bus driver for translating bus addresses, for retrieving a DMA adapter structure in cases where the bus adapter supports DMA, and for reading and setting the bus configuration space for a particular device on the bus.

This section contains the following information:

  • The BUS_INTERFACE_STANDARD Interface Structure
  • PTRANSLATE_BUS_ADDRESS
  • PGET_DMA_ADAPTER
  • PGET_SET_DEVICE_DATA

The BUS_INTERFACE_STANDARD interface is an extension of the INTERFACE interface template.
