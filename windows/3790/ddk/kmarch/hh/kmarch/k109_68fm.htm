Kernel-Mode Driver Architecture: Windows DDK

RtlStoreUshort

The RtlStoreUshort routine stores a USHORT value at a particular address, avoiding alignment faults.

VOID 
  RtlStoreUshort(
    IN PUSHORT  Address,
    IN USHORT  Value
    );

Parameters

Address
Pointer to a location in which to store a USHORT value.
Value
Specifies a USHORT value to be stored.

Return Value

None

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

The caller can be running at any IRQL if Address points to nonpaged pool. Otherwise, the caller must be running at IRQL < DISPATCH_LEVEL.
