READ_PORT_USHORT

The READ_PORT_USHORT macro reads a USHORT value from the specified port address.

USHORT 
  READ_PORT_USHORT(
    IN PUSHORT  Port
    );

Parameters

Port
Pointer to the port address, which must be a mapped range in I/O space.

Return Value

READ_PORT_USHORT returns the USHORT value read from the specified port address.

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

Callers of READ_PORT_USHORT can be running at any IRQL, assuming the Port is resident, mapped device memory.
