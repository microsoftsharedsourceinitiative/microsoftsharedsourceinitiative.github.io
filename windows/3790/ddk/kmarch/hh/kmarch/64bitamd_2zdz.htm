Scalar Types

Although the access of data can stem from any alignment, it is recommended that data be aligned on its natural boundary to avoid performance loss (or a multiple thereof). Enumerated types are constant integers and are treated as 32-bit integers. The following table describes the type definition and recommended storage for the type as the type pertains to alignment using the following alignment values:

  • Byte – 8 bits
  • Word – 16 bits
  • Double Word – 32 bits
  • Quad Word – 64 bits
  • Octal Word – 128 bits
