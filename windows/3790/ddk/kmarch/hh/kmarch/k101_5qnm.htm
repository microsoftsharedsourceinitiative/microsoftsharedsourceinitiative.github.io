Summary of Kernel-Mode Support Routines

This section summarizes the kernel-mode support routines that can be called by kernel-mode drivers.

Support routines are categorized as follows:

  • Initialization and Unload
  • IRPs
  • Synchronization
  • Memory
  • Plug and Play Routines
  • Windows Management Instrumentation Routines
  • DMA
  • PIO
  • Driver-Managed Queues
  • Driver-dedicated system threads and system worker threads in Driver System Threads
  • Strings
  • Data Conversions
  • Access to and access rights on driver-managed objects in Access to Driver-Managed Objects
  • Error Handling

Some routines are listed in more than one section or subsection of this section.
