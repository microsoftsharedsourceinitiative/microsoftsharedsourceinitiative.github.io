Intrinsic Functions for AMD64

Because the compiler for AMD64 does not support inline assembly (see Intrinsic Functions and Inline Assembly for more information), tasks that typically require inline assembly are implemented as compiler intrinsic functions.

The compiler for AMD64 supports a large number of standard intrinsic functions that are common with other architectures. For more information on these standard intrinsic functions, see the reference sections of the DDK and the Platform SDK documentation.
