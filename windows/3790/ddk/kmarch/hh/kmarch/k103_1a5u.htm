WRITE_REGISTER_USHORT

The WRITE_REGISTER_USHORT macro writes a USHORT value to the specified address.

VOID 
  WRITE_REGISTER_USHORT(
    IN PUSHORT  Register,
    IN USHORT  Value
    );

Parameters

Register
Pointer to the register, which must be a mapped range in memory space.
Value
Specifies a USHORT value to be written to the register.

Return Value

None

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

Callers of WRITE_REGISTER_USHORT can be running at any IRQL, assuming the Register is resident, mapped device memory.
