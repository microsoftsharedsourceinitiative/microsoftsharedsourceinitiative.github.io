D1Latency, D2Latency, and D3Latency

The D1Latency, D2Latency, and D3Latency fields contain the approximate time, in 100-microsecond units, that the device requires to return to the D0 state from each of the sleeping states. A driver should specify a latency time of zero for any device state that it does not support.
