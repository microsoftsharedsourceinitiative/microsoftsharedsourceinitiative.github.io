Executive Support Routines

This section describes the executive support routines. These routines use the ExXxx naming convention and are listed in alphabetical order.

For an overview of the functionality of these routines, see Summary of Kernel-Mode Support Routines.

The following executive support routines are reserved for system use. Use the corresponding KeXxxSpinLock routines instead.

ExAcquireSpinLock

ExAcquireSpinLockAtDpcLevel

ExReleaseSpinLock

ExReleaseSpinLockFromDpcLevel
