Compiler Helper Intrinsic Functions

The compiler helper intrinsic functions in the following table are implemented instead of inline assembly.

Function Return Value
int _setjmpex(jmp_buf); Returns the number of jumps.
void __va_start(va_list*, ...); None
unsigned int __getcallerseflags(); Returns the contents of the EFLAGS register at function entry.
