Initialization of Static Members of Base Classes

Static members cannot be initialized through a derived class; they must be initialized through a base class. This applies to templates as well. In the following example, s cannot be initialized through B:

struct A {
static int s;
};
struct B: A {};
int B::s = 1;    // Produces errors. Use int A::s instead.

The workaround is to use the base class:

struct A {
static int s;
};
struct B: A {};
int A::s = 1;
