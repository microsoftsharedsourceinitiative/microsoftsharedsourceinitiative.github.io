Special and Privileged Instruction Intrinsic Functions

The special and privileged instruction intrinsic functions in the following table are implemented instead of inline assembly.

Function Operation Performed
VOID __wbinvd(VOID); Generates the WBINVD instruction to write back and flush internal caches, and initiate write back and flush on external caches.
VOID __invlpg(PVOID); Generates the INVLPG instruction to invalidate the TLB entry for the page pointed to by the argument.
