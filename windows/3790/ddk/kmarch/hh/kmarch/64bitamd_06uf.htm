Kernel-Mode Driver Architecture: Windows DDK

Return Values

Most return values that can fit into 64 bits (including __m64 types) are returned through the RAX register. However, the types __m128, __m128i, __m128d, floats, and doubles are returned in the XMM0 register. If the return value does not fit within 64 bits, then the caller must allocate and pass a pointer for the return value as the first argument. Subsequent arguments are then shifted one argument to the right. User-defined types to be returned must be 1, 2, 4, 8, 16, 32, or 64 bits in length.
