RtlConvertUlongToLargeInteger

The RtlConvertUlongToLargeInteger routine converts the input unsigned integer to a signed large integer.

LARGE_INTEGER 
  RtlConvertUlongToLargeInteger(
    IN ULONG  UnsignedInteger
    );

Parameters

UnsignedInteger
Specifies a value of type ULONG.

Return Value

RtlConvertUlongToLargeInteger returns the converted large integer.

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

Callers of RtlConvertUlongToLargeInteger can be running at any IRQL.
