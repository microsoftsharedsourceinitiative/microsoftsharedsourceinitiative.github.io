Bit Fields for AMD 64 Bit

Structure bit fields are limited to 64 bits and can be of type signed int, unsigned int, _int64, or unsigned _int64. Bit fields that cross the type boundary will skip bits in order to align the bit field to the next type alignment. For example, integer bit fields may not cross a 32-bit boundary.
