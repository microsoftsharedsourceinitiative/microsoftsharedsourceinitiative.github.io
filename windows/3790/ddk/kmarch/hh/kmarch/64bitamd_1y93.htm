Arithmetic Intrinsic Functions

The arithmetic intrinsic functions in the following table are implemented instead of inline assembly.

Function Return Value
LONG64 __mulh(LONG64, LONG64); Return the high 64 bits of the multiplication of the two 64-bit arguments (signed).
ULONG64 __mulh(ULONG64, ULONG64); Return the high 64 bits of the multiplication of the two 64-bit arguments (unsigned).
