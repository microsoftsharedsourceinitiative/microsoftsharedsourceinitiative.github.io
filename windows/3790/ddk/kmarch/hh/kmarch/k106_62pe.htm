MmInitializeMdl

The MmInitializeMdl routine initializes the header of an MDL.

VOID 
  MmInitializeMdl(
    IN PMDL  MemoryDescriptorList,
    IN PVOID  BaseVa,
    IN SIZE_T  Length
    );

Parameters

MemoryDescriptorList
Pointer to the MDL to be initialized.
BaseVa
Pointer to the base virtual address of a buffer.
Length
Specifies the length in bytes of the buffer to be described by the MDL.

Return Value

None

Headers

Declared in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Comments

Callers of MmInitializeMdl must be running at IRQL <= DISPATCH_LEVEL.
