Explicit Specializations No Longer Find Template Parameters from the Primary Template

The use of T from the primary template is not allowed in the explicit specialization. For example, declaring an instance of T in the following produces errors:

template <class T> struct S;
template <> struct S<int> {
    T m_t;    
};

A workaround involves replacing all instances of the template parameter in the specialization with the explicitly specialized type:

template <class T> struct S;
template <> struct S<int> {
    int m_t;
};
