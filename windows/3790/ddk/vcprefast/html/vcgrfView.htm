PREfast User Guide  

VIEW

Displays the user interface for the PREfast defect log. 

prefast VIEW logfile
prefast /LOG=logfile VIEW

Parameters

filtername
Specifies the use of the filter preset to apply to the Message List window.
logfile
Specifies a location for the defect log file. If nothing is specified, a default location is used.

Remarks

The default log file is called defects.xml and is in the directory where you installed PREfast.

For more information, see Viewing the Defect Log.
