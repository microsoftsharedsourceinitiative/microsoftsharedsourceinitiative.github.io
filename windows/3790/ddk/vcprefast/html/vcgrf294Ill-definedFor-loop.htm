294 - Ill-defined for-loop.

Additional Information: Initial condition does not satisfy test.

This warning indicates that PREfast has detected a for-loop that may not function as intended.

The for-loop: 

     for (initializer; continuation; increment ) {
          ...
          }

is equivalent to the following: 

     {
          initializer;
          while (continuation) {
          ...
              increment;
          }
     }

Thus, if initializer does not satisfy continuation, the loop body will not be executed.

The consequence of this error is that the body is not executed.

Example

Defective Source

signed char i;

for (i = 0; i > 100; i++) { ; }

Corrected Source

signed char i;

for (i = 0; i < 100; i++) { ; }
