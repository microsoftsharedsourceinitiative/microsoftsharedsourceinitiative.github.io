Troubleshooting Device Installation

To verify that your device installed properly, you should do the following:

  • Use the Device Manager to view system information about the device.
  • Use SetupAPI Logging to identify installation errors.
  • Run test programs to exercise the device, including the testing and debugging tools supplied with this DDK.

Additionally, a co-installer or class installer can provide a troubleshooter that helps users troubleshoot problems with your device. See DIF_TROUBLESHOOTER for more information.
