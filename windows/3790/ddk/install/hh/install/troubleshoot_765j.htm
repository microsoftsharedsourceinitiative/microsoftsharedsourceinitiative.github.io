Device Installation: Windows DDK

CM_PROB_REGISTRY

A registry problem was detected.

Error Code

19

Display Message – NT-based Systems

"Windows cannot start this hardware device because its configuration information (in the registry) is incomplete or damaged. (Code 19)"

Recommended Resolution – NT-based Systems

This can be set if more than one service is defined for a device, there is a failure opening the service key, or the driver name can’t be obtained from the service key.

Display Message – Windows 98/Me

"Your registry may be bad. (Code 19)"

"To check your registry, click Check Registry. If the registry is bad, Windows will restart your system and go back to a previous registry that is good."

Recommended Resolution – Windows 98/Me

Select Check Registry, which will start the registry checker utility.
