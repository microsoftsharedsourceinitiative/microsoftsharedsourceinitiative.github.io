Device Installation: Windows DDK

CM_PROB_DRIVER_FAILED_LOAD

The driver could not be loaded. (Windows XP and later)

Error Code

39

Display Message – NT-based Systems

"Windows cannot load the device driver for this hardware. The driver may be corrupted or missing. (Code 39)"

Recommended Resolution – NT-based Systems

Reinstall or obtain a new driver.

Reasons for this error include a driver that is not present, a binary file that is corrupted, a file I/O problem, or a driver that references an entry point in another binary that could not be loaded.

Display Message – Windows 98/Me

Not used.

Recommended Resolution – Windows 98/Me
