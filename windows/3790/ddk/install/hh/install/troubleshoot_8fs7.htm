CM_PROB_ENTRY_IS_WRONG_TYPE

Registry entries are incorrect.

Error Code

4

Display Message – NT-based Systems

Not used.

Recommended Resolution – NT-based Systems

Not used.

Display Message – Windows 98/Me

"This device is not working properly because one of its drivers may be bad, or your registry may be bad. (Code 4)"

To Update the drivers for this device, click Update Driver. If that doesn't work, run Scanregw.exe (click Start, click Run, type Scanregw.exe, and then click OK) to check your registry."

Recommended Resolution – Windows 98/Me

Select Update Driver, which starts the Update Driver wizard, or run scanregw.exe.
