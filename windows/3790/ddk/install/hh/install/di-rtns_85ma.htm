Device Installation Functions

The device installation functions support device installation on Windows® 2000 and later versions of the Windows NT-based operating system. Many, but not all, of these functions were introduced with Windows NT® 4.0. Most of these functions are not supported on Windows 9x/Me.

For more information about the SetupDi-prefixed device installation functions described in this section, see Using Device Installation Functions. To perform device installation operations that are not supported by these functions, call the appropriate PnP Configuration Manager Functions (CM_Xxx functions).
