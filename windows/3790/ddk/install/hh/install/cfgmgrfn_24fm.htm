Device Installation: Windows DDK

CM_Set_DevNode_Registry_Property

Do not use this function. Use device installation functions instead.

The CM_Set_DevNode_Registry_Property function sets a specified device property in the registry.

CMAPI CONFIGRET WINAPI
  CM_Set_DevNode_Registry_Property(
    IN DEVINST  dnDevInst,
    IN ULONG  ulProperty,
    IN PCVOID  Buffer, OPTIONAL
    IN ULONG  ulLength,
    IN ULONG  ulFlags
    );

Parameters

dnDevInst
Caller-supplied device instance handle that is bound to the local machine.
ulProperty
Supplies a CM_DRP_-prefixed constant value that identifies the device property to be set in the registry. These constants are defined in cfgmgr32.h.
Buffer
Pointer to a caller-supplied buffer that supplies the requested device property, formatted appropriately for the property's data type.
ulLength
Supplies the length, in bytes, of the supplied device property.
ulFlags
Not used, must be zero.

Return Value

If the operation succeeds, the function returns CR_SUCCESS. Otherwise, it returns one of the CR_-prefixed error codes that are defined in cfgmgr32.h.

Headers

Declared in cfgmgr32.h. Include cfgmgr32.h.

Comments

Use SetupDiSetDeviceRegistryProperty instead of CM_Set_DevNode_Registry_Property.

For information about using device instance handles that are bound to the local machine, see CM_Get_Child.
