Device Installation: Windows DDK

CM_PROB_DRIVER_SERVICE_KEY_INVALID

Information in the registry's service key for the driver is invalid. (Windows XP and later)

Error Code

40

Display Message – NT-based Systems

"Windows cannot access this hardware because its service key information in the registry is missing or recorded incorrectly. (Code 40)"

Recommended Resolution – NT-based Systems

Reinstall the driver.

Not used.

Recommended Resolution – Windows 98/Me
