Device Installation: Windows DDK

CM_PROB_WILL_BE_REMOVED

The system will remove the device.

Error Code

21

Display Message – NT-based Systems

"Windows is removing this device. (Code 21)"

"Close this dialog box, and then wait a few seconds. If this problem continues, restart your computer."

Recommended Resolution – NT-based Systems

Select Restart Computer, which will restart the computer. (Note that this problem code is transitory, existing only during attempts to query and then remove a device.)

Display Message – Windows 98/Me

"Windows is removing this device. (Code 21)"

"Close this dialog box, and then wait a few seconds. If this problem continues, restart your computer."

Recommended Resolution – Windows 98/Me

Select Restart Computer, which will restart the computer.
