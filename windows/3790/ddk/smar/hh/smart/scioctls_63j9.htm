IOCTL_SMARTCARD_SET_ATTRIBUTE

WDM driver version

Operation

The IOCTL_SMARTCARD_SET_ATTRIBUTE request sets various attributes in the driver. For a list of all defined attributes, refer to Part 3 of the Interoperability Specification for ICCs and Personal Computer Systems

Input

Irp->AssociatedIrp.SystemBuffer
Contains the tag and value to be set.
Parameters.DeviceIoControl.InputBufferLength
Contains the length of the whole TLV structure.

I/O Status

Information must be set to sizeof(ULONG). Status can be one of the following values.

Status Meaning
STATUS_SUCCESS The attribute has been set successfully.
STATUS_NOT_SUPPORTED The attribute is not supported.
STATUS_INVALID_PARAMETER The attribute to be set is in the wrong format.

Headers

Defined in winsmcrd.h. Include winsmcrd.h.

Contains the tag and value to be set.

Following are possible return values.

Value Meaning
STATUS_SUCCESS The attribute has been set successfully.
STATUS_NOT_SUPPORTED The attribute is not supported.
