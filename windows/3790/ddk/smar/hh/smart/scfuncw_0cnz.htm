SmartcardReleaseRemoveLock (WDM)

SmartcardReleaseRemoveLock is the inverse function to SmartcardAcquireRemoveLock (WDM). It decrements the reference count to the driver, incremented by the call to SmartcardAcquireRemoveLock.

VOID 
  SmartcardReleaseRemoveLock(
    PSMARTCARD_EXTENSION  SmartcardExtension
    );

Parameters

SmartcardExtension
Pointer to a structure of type SMARTCARD_EXTENSION that contains the device extension of the smart card device.

Return Value

None

Headers

Declared in smclib.h. Include smclib.h.

Comments

SmartcardReleaseRemoveLock should be called to release the lock acquired with SmartcardAcquireRemoveLock.
