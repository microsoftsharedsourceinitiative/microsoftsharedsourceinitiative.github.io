Modem Devices: Windows DDK

FixSerialSpeed

The FixSerialSpeed entry specifies the DTE-DCE baud rate to be used when the modem is in Class1 or Class2 mode.

Format

Single command string.

Sample

HKR, Fax, FixSerialSpeed,,"19200"

Note

The default value for this entry is "19200".
