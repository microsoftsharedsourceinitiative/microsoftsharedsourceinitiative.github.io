Dynamic Configuration

Dynamic configuration is performed on the basis of per-call configuration requirements. Unimodem sends AT commands to the ISDN modem during dynamic configuration to select the protocol and authentication method. For more information on commands issued during dynamic configuration, see the Protocol\ISDN entry in ISDN-TA Modem INF File Entries.
