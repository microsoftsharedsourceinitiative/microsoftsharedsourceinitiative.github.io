Modifying SmpVoice.inf

SmpVoice.inf is a sample voice modem INF file that is included with the DDK. SmpVoice.inf demonstrates a modem with a serial wave driver that plays and records audio to the handset and phone line.

In SmpVoice.inf, the add-registry-section named MICROSOFT_VOICE contains all the voice modem INF file entries with sample values.

For further information about each INF file entry, see the Voice Modem INF File Entries.
