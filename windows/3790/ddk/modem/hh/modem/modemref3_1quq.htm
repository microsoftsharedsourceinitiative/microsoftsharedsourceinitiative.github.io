Modem Devices: Windows DDK

BaudRate (NT Wavedriver)

The BaudRate (NT Wavedriver) entry provides voice functionality for Windows 2000 and later versions of the operating system. It specifies port speed.

Format

A single command of type DWORD.

Sample

HKR, WaveDriver,  BaudRate       ,1, 00, c2, 01, 00

Note

The preceding example sets the port speed to 115200 bps.

This command is placed only in the modem INF file. Windows 2000 and later versions of the operating system do not use a serial wave driver INF file.

In addition, the following entries are collectively found in a modem INF file that is enabled for voice functionality in Windows 2000 and later versions of the operating system:

XformID (NT Wavedriver)

BaudRate (NT Wavedriver)

XformModule (NT Wavedriver)

WaveHardwareID (NT Wavedriver)

WaveDevices (NT Wavedriver)

The VoiceProfile bit (0x02000000) must also be enabled for Windows 2000 and later versions of the operating system.
