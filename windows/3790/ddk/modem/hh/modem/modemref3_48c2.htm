AbortPlay

The AbortPlay entry identifies the command that is issued to the modem to stop playing the audio data immediately.

Format

One or more command strings.

Sample

HKR,, AbortPlay,,            "<h10><h18>"

Note

This command is issued after StartPlay, and before StopPlay.

This command is only called for modems that use a serial wave device. It is not used for modems that use a wave wrapper.
