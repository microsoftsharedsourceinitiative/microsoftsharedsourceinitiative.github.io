Modem Devices: Windows DDK

XformID (NT)

Use the XformID (NT) selection in ProCalc to create an XformID (NT Wavedriver)modem INF file entry for Windows 2000 and later versions of the operating system.

An example of an XformID entry for Windows 2000 and later versions of the operating system follows:

HKR,WaveDriver,XformID, 1, 00,00

For more information about the XformID entry in Windows 2000 and later versions of the operating system, see XformID (NT Wavedriver).
