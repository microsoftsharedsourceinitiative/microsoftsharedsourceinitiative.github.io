Setting the Maximum DCE Speed for a 56 Kbps Modem

Use ProCalc to set the maximum DCE speed in the modem INF Properties entry. For more information about ProCalc and the Properties entry, see Creating Data Modem INF File Entries with ProCalc.

Set the maximum DCE rate to 56 Kbps if the modem responses that are available in firmware go to 56 Kbps. If your modem firmware contains responses up to 60 Kbps and the modem can be expected to connect at speeds that high, set the maximum DCE rate to 60 Kbps.
