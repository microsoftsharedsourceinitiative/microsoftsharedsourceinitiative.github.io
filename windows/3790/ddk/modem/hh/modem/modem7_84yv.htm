Inactivity Scale

The InactivityScale value is multiplied by the InactivityTimeout value in the Properties entry to obtain the inactivity timeout in seconds.

The ProCalc inactivity scale feature provides predefined values of 1, 10, and 60 seconds for InactivityScale. If required, other values can be typed into the provided edit box and ProCalc will translate them.

An example of an InactivityScale entry follows:

HKR,, InactivityScale, 1, 0A,00,00,00
