EnableCallerID

The EnableCallerID entry defines the AT command used to enable Caller ID reporting.

Format

One or more command strings.

Sample

HKR, EnableCallerID, 1,, "at#cid=1<cr>"

Note

If the modem does not have the ability to report Caller ID, this entry should be omitted from the modem INF file. The VoiceProfile INF file entry can also indicate that the modem supports Caller ID. If Caller ID is enabled, make sure that Caller ID responses are in the Responses INF File entries.
