Modem Devices: Windows DDK

Sample Modem INF Files

Any installed Windows operating system is a good source for sample modem INF files. Look for INF files with file names that match mdm*.inf.

The following sample INF files are included in the DDK:

smpdata.inf
Installs a data modem.
smpvoice.inf
Installs a voice modem.
