Supporting Bongtone Settings

Use ProCalc to set the wait-for-bongtone feature in the dial options field of the Properties entry.

DialSettings Bongtone on many modems requires some alternate AT-style character rather than the dollar sign ($). If your modem supports bongtone but does not use the $ character, you cannot specify bongtone in your modem INF file and you should disable bongtone in the Properties entry.

Fore more information about ProCalc and the Properties entry, see Creating Data Modem INF File Entries with ProCalc.
