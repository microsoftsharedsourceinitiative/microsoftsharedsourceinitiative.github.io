Cl2RecvBOR

The Cl2RecvBOR entry specifies a value of "AT+FBOR=xxx" for receive operations.

Format

Single command string.

Sample

HKR, Fax, Cl2RecvBOR,,"1"

Note

The default value is modem-specific, but it is typically "1".
