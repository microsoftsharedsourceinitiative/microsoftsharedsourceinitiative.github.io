VoiceBaudRate

The VoiceBaudRate entry specifies port speed prior to switching into voice mode with VoiceAnswer or VoiceDialNumberSetup. The BaudRate setting is used at the beginning of serial wave operation.

Format

A single command of type DWORD.

Sample

HKR,, VoiceBaudRate,  1, 00, c2, 01, 00

Note

The preceding example sets port speed to 115200 bps.

This command is placed in the modem INF file only.

Generally the VoiceBaudRate and BaudRate values should be the same.
