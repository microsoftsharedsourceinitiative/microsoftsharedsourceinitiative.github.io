Voice Profile

The VoiceProfile entry is a bit-field that is used to define voice modem capabilities. ProCalc provides a simple interface to set the VoiceProfile entry option fields.

An example of a VoiceProfile entry follows:

HKR,, VoiceProfile,    1 , 07,a1,02,02
