TerminatePlay

The TerminatePlay entry identifies the command that is issued to the modem to empty (finish playing) the local buffer and stop playing the audio data.

Format

One command string.

Sample

HKR,, TerminatePlay,,        "<h10><h03>"

Note

This command is issued after StartPlay, and before StopPlay. This command is called only for modems that use the serial wave device. It is not used for modems that use the wave wrapper.
