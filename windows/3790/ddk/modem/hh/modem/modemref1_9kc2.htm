Modem Devices: Windows DDK

MaximumPortSpeed

The MaximumPortSpeed entry specifies the maximum DTE port speed for the modem.

Format

A single command of type DWORD.

Sample

HKR,,MaximumPortSpeed,    1, 00, c2, 01, 00

Note

This entry is supported in Windows 2000 and later versions of the operating system.
