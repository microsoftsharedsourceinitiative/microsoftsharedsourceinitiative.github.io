Configuring the Modem Volume Level

Use ProCalc to set the modem volume levels in the modem INF Properties entry. For more information about ProCalc and the Properties entry, see Creating Data Modem INF File Entries with ProCalc.

Some modems support only one volume level. Such modems might respond to the usual ATL1-3 command but do not adjust the volume level. When configuring the Properties entry in the INF file for such a modem, do not indicate that the modem supports any of the volume levels. Doing so would disable the Volume Control Slider of the modem program in Control Panel. For modems that support only one volume level, enable the appropriate bits in the speaker mode data, and do not enable any bits in the speaker volume data.
