FixModemClass

The FixModemClass entry specifies the preferred fax class.

Format

Single command string.

Sample

HKR, Fax, FixModemClass,,"1"

Note

The default setting for this entry is "1". Some modems claim to support Class1 but do not support it correctly. In such cases, specify a value of "2" for this entry.
