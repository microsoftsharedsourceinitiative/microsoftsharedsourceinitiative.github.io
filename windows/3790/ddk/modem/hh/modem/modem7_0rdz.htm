WHQL Testing for Modems

Microsoft Windows Hardware Quality Labs (WHQL) ensures the compatibility of hardware with Windows. WHQL provides Hardware Compatibility Test (HCT) kits that are used to qualify devices, including modems, for the Microsoft Windows logos and for inclusion on the Microsoft Hardware Compatibility List (HCL).

Modem INF files that pass WHQL testing are eligible for distribution in the Windows Driver Library on the World Wide Web and in future Windows operating system releases.

You will receive a test report when testing is complete.

When the modem and INF file pass WHQL testing, you will receive:

  • A signed Windows Logo License Agreement and a Logo Kit containing camera-ready artwork.
  • Addition of the modem product to the Windows Hardware Compatibility Lists (HCL).
  • An opportunity to distribute the INF file through Windows Update.

To obtain information about the latest WHQL modem tests, check the following resources.
