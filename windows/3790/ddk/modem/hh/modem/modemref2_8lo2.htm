HighestSendSpeed

The HighestSendSpeed entry specifies the highest over-the-wire speed acceptable for sending a fax, in bits per second.

Format

Single command string.

Sample

HKR, Fax, HighestSendSpeed,,"0"

Note

A value of "0" for this entry indicates the highest available speed that the modem supports. The default value is "0". The T.30 protocol engine uses this setting when creating the DIS frame.
