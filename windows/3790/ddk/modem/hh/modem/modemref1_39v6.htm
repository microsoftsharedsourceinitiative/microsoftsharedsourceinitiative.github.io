Modem Devices: Windows DDK

Prefix

The Prefix entry specifies the command string that indicates which characters are prefixed to every Settings command sent to the modem.

Format

A single command string.

Sample

HKR, Settings,Prefix,, "AT"

Note

The initialization commands should include appropriate Prefix (generally AT) and Terminator (generally <cr>) characters. Do not exceed the maximum allowable length for individual command strings.
