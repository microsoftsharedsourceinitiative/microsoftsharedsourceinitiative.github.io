Modem Devices: Windows DDK

ISDN\NvSave

The ISDN\NvSave entry defines AT commands that are used to configure the ISDN modem for storage of the current configuration in NvRAM. This entry will always store ISDN\init and ISDN\SwitchType. Depending on the switch type, it may also store MSN, EAZs, SPIDs, and directory number.

Format

One or more command strings.

Samples

The following forms are valid for this entry:

HKR, ISDN\NvSave,1,,"AT&W<cr>"
Or:
HKR, ISDN\NvSave,1,,"AT&W<cr>" 
HKR, ISDN\NvSave,2,,"ATZ<cr>"

Or:

HKR, ISDN\NvSave,1,, "AT&W0<cr>"  
HKR, ISDN\NvSave,2,, "AT>W<cr>"    
HKR, ISDN\NvSave,3,, "ATZ0<cr>"  
HKR, ISDN\NvSave,4,, "AT>Z<cr>"

Note

As demonstrated by the samples, a multi-line format can be used if it is required to cause the ISDN modem to reset the D channel and network.

The device has 10 seconds to recover from this reset or Unimodem will assume the device is failing to respond and return an appropriate error.
