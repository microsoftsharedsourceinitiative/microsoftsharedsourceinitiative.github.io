Supporting V.23 Override

Use ProCalc to modify the Properties entry and enable V.23 Override.

To activate V.23 Override with ProCalc

  1. Select Data Profile from the list.
  2. Click the Modem Options tab and set V.23 Override to ON.
  3. Include the appropriate registry entries in the appropriate add-registry-section of your INF file. The content of each entry depends on the chip set used. For example: 
    HKR, Settings, Modulation_CCITT_V23,, "+MS=3"

    or 

    HKR, Settings, Modulation_CCITT_V23,, "B0 +MS=3,0,1200,1200"

    or 

    HKR, Settings, Modulation_CCITT_V23,, "S27=16"
  4. Include the appropriate Responses entries, for example: 
    HKR,Responses,"<cr><lf>CONNECT 75/1200<cr><lf>" ,1,2,0, B0,04,0,0, 0,0,0,0
HKR,Responses,"<cr><lf>CONNECT 1200/75<cr><lf>" ,1,2,0, B0,04,0,0, 0,0,0,0
HKR,Responses,"<cr><lf>CONNECT 75/1200/NONE<cr><lf>"    ,1,2,0, B0,04,0,0, 0,0,0,0
HKR,Responses,"<cr><lf>CONNECT 1200/75/NONE<cr><lf>"    ,1,2,0, B0,04,0,0, 0,0,0,0
HKR, Responses, "CONNECT 1200TX/75RX",   1, 02, 00, B0,04,00,00, 00,00,00,00
HKR, Responses, "CONNECT 1200RX/75TX",   1, 02, 00, B0,04,00,00, 00,00,00,00
