Modem Devices: Windows DDK

Cl2SkipCtrlQ

The Cl2SkipCtrlQ entry indicates that the modem will not wait for a Ctrl-Q character before sending page data.

Format

Single command string.

Sample

HKR, Fax, Cl2SkipCtrlQ,,"0"

Note

If this entry is not "0", the driver will not wait to receive a Ctrl-Q character before sending page data. The default value for this entry is "0".
