Tools for Creating a Modem INF File

You can use a simple text editor to perform most of the work required to create a modem INF file. This section describes procedures to follow to configure modem features.

The DDK includes an application called ProCalc (ProCalc.exe), in the \tools directory. You can use ProCalc to create some common modem INF file registry entries. For more information about using ProCalc to create modem INF file entries, see Using ProCalc.
