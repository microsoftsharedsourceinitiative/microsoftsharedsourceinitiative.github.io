Modem Devices: Windows DDK

GenerateDigit

The GenerateDigit entry specifies the AT command that is used to generate a DTMF digit during a voice call.

Format

One or more command strings.

Sample

HKR, GenerateDigit,  1,, "at#vts=<Digit><cr>"

Note

This command is only issued during an active voice call. This entry can be omitted if the modem cannot generate DTMF digits during a voice call.
