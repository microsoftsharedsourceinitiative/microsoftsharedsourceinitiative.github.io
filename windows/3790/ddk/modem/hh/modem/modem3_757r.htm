Assigning MSN/DN/EAZ/SPID to the Analog Port

There are two keys provided for setting MSN/DN/EAZ/SPID: 

HKR, ISDN\SetSpid, 1,,     "AT SPID1=%s<cr>"
HKR, ISDN\SetSpid, 2,,     "AT SPID2=%s<cr>"

The first key should always be used for your device's data port; the second can be used for the voice port on your ISDN modem.

For more information, see ISDN\SetSpid.
