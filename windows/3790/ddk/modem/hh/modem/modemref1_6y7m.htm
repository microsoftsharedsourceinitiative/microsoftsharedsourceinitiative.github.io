PowerDelay

The PowerDelay entry specifies the number of milliseconds that elapse between the time when the modem powers up, and when the modem begins to send data to the serial driver.

Format

A single command of type DWORD.

Sample

HKR,,PowerDelay,    0x10001, 1500

Note

In the preceding example, the delay is set to 1500 milliseconds.

This entry should only be used for PC card modems.
