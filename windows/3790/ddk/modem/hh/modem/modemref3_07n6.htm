TerminateRecord

The TerminateRecord entry identifies the command that is issued to the modem to finish recording the local buffer and stop recording the audio data.

Format

One command string.

Sample

    HKR,, TerminateRecord,,     "<h10>E<cr>"

Note

This command is issued after StartRecord, and before StopRecord. This command is called only for modems that use the serial wave device. It is not used for modems that use the wave wrapper.
