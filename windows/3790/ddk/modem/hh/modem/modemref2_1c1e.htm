Modem Devices: Windows DDK

AnswerCommand

The AnswerCommand entry identifies commands that are sent to the modem to configure it for adaptive answer and to answer a call.

Format

One or more command strings.

Sample

HKR, Fax\Class1\AdaptiveAnswer\AnswerCommand, 1,, "ATS7=60S19=0M1&M4"
HKR, Fax\Class1\AdaptiveAnswer\AnswerCommand, 2,, "ATA"
