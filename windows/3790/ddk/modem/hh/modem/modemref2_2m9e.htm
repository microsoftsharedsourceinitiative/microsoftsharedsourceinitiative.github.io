PreDialCommand

The PreDialCommand entry specifies the modem command string that is issued to the modem just before Unimodem issues the ATD command.

Format

Single command string.

Sample

HKR, Fax, PreDialCommand,,

Note

Microsoft Fax issues the setup command and the command to go to the appropriate fax class (Class1 or 2) before issuing the PreDialCommand.

The default is the empty string.
