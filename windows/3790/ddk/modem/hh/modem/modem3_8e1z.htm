Modifying the DDInstall Sections

This topic provides information about modifying the DDInstall sections in the sample modem INF file SmpData.inf. Each per-model DDInstall section contains one or more directives that reference additional named sections in the INF file. The most commonly specified INF directives are CopyFiles and AddReg. The sections referenced by these directives contain instructions for installing driver files and writing any device-specific or driver-specific information into the registry.

For Windows 2000 and later versions of the operating system, DDInstall sections in modem INF files must include an ".NT" decoration. If this decoration is not included, the modem class installer will not install the modem.

For more information on DDInstall sections, see INF DDInstall Section.
