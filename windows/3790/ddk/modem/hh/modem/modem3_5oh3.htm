Setting the D Channel Command Sequence

The Unimodem default sequence for D channel configuration commands is the switch type followed by the directory number and SPID. Some ISDN devices, however, require D channel configuration commands to be issued in the reverse order:

  1. Directory number and SPID
  2. Switch type

To reverse the order in which these commands are issued, include the following INF entry in the add-registry-section that contains your ISDN registry entries:

HKR,ISDN, CompatibilityFlags, 1,00,00,00,01

For more information, see ISDN, CompatibilityFlags.
