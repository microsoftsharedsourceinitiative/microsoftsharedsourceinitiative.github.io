BaudRate

The BaudRate entry is used in the serial wave device INF file. It specifies the port speed.

Format

A single command of type DWORD.

Sample

HKR,Config,BaudRate,    1, 00, c2, 01, 00

Note

The preceding example sets the port speed to 115200 bps.
