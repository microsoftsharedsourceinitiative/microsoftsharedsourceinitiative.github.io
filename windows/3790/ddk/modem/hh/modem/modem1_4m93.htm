How to Create a Modem INF File

This topic summarizes the activities involved in creating a modem INF file for a new modem.

When creating a modem INF file, you should:
  • Keep the file small to improve performance.
  • Create modem INF file entries using ProCalc (which is supplied as part of the Windows Driver Development Kit) whenever possible.
  • Double-check your files for common errors using Chkinf.

Several sample modem INF files can be adapted for use with your modem by modifying INF file sections.

For additional information on configuring data modem features, see Creating INF Files for Data Modems. For detailed information about supported data modem features, see Data Modem INF File Entries.

If your modem supports fax or voice communication in addition to data, you should read the following sections:

Adding FAX Support

Adding Voice Support

If you need to provide a driver for a modem, such as, a controllerless modem or softmodem, read the following section:

Adding WDM Modem Support

After you have created your modem INF file, you should test it using the guidelines supplied in Testing a Modem Installation.
