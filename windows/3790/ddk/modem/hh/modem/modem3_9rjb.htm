Creating INF Files for Data Modems

This section provides information about creating information (INF) files for data modems. A basic understanding of Microsoft® Windows® support for INF files and modems is required to understand this information. For an overview of modem INF files, see Overview of Modem INF Files.

This section also provides modem INF file information for a subset of the data modem features supported by Windows. This subset includes the most commonly supported features and a few solutions configuration problems. For detailed information about the features supported, see Data Modem INF File Entries.
