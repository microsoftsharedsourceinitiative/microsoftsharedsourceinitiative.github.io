Enable Modem Answering

For some modems, the NVRAM setting of the S0 register defaults to values greater than zero after startup. Setting the S0 register to zero will enable modem answering.

Add the following entries to your INF file MfgAddReg section to force S0 to zero:

  HKR, Monitor, 1,, "ATS0=0<cr>"
  HKR, Monitor, 2,, "None"
