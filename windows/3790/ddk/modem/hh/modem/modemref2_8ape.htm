LowestSendSpeed

The LowestSendSpeed entry specifies the lowest over-the-wire speed that is acceptable for sending a fax, in bits per second.

Format

Single command string.

Sample

HKR, Fax, LowestSendSpeed,,"0"

Note

A value of "0" indicates that negotiated settings should be used. The default value is "0". The T.30 protocol engine uses this setting when determining if transmission should continue at a lower speed after transmission errors (after receiving FTT, RTP, and so forth).
