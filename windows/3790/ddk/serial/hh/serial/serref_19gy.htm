IOCTL_SERIAL_INTERNAL_CANCEL_WAIT_WAKE

Operation

The IOCTL_SERIAL_INTERNAL_CANCEL_WAIT_WAKE request disables the wait/wake operation of a COM port.

Input

None.

Output

None.

I/O Status Block

The Information member is set to zero.

The Status member is set to one of the Generic Status Values for Serial Device Control Requests.

Headers

Defined in ntddser.h. Include ntddser.h.
