mciFreeCommandResource

Do not write an MCI driver to support a multimedia device; write a WDM AVStream driver instead.

The mciFreeCommandResource function releases memory for the command table that is identified by the given index number.

BOOL
  mciFreeCommandResource(
    UINT wTable
    );

Parameters

wTable
Specifies the table index that was previously returned from a call to the mciLoadCommandResource function.

Return Value

Returns TRUE on success; otherwise, returns FALSE if the table index is invalid.

Headers

Declared in Mmddk.h. Include Mmddk.h.

Comments

For more information about MCI, see the Platform SDK documentation.
