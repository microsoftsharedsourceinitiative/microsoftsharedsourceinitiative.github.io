VdmUnmapFlat

VdmUnmapFlat releases a mapping returned by VdmMapFlat.

BOOL
  VdmUnmapFlat(
    IN USHORT  Segment,
    IN ULONG  Offset,
    IN PVOID  Buffer,
    IN VDM_MODE  Mode
    );

Parameters

Segment
The segment or selector, depending on the value of Mode, of the x86 address.
Offset
The offset of the x86 address.
Buffer
The linear address returned by VdmMapFlat.
Mode
Either VDM_PM or VDM_V86.

Return Value

VdmUnmapFlat returns TRUE if the given mapping was released.

Comments

VdmUnmapFlat releases a linear address mapping obtained in a previous call to VdmMapFlat.

After calling VdmMapFlat, a VDD should call VdmUnmapFlat when such a returned linear pointer is no longer needed.
