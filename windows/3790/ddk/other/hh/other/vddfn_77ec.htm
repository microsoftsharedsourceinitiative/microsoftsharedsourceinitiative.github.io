VdmMapFlat

VdmMapFlat returns a linear address for a given x86 address.

PVOID
  VdmMapFlat(
    IN WORD  Segment,
    IN ULONG  Offset,
    IN VDM_MODE  Mode
    );

Parameters

Segment
The segment or selector, depending on the value of Mode, of the x86 address.
Offset
The 32-bit or 16-bit offset of the x86 address.
Mode
Either VDM_PM or VDM_V86.

Return Value

VdmMapFlat returns the linear address.

Comments

VDDs call this function to form a pointer to an x86 "client" memory range for the VDM. Note that such a memory range must not be based at zero in the process, so use of VdmMapFlat makes the VDD platform independent.

If the VDD modifies code within such a range, it should call VdmFlushCache first to ensure cross-platform independence and consistency of operation.
