Parallel Ports and Devices: Windows DDK

Connecting to an IEEE 1284.3 Data Link Device

The system-supplied function driver for parallel ports does not provide full support for the IEEE 1284.3 data link layer services; however, it does provide internal device control requests that an upper-level driver can use to connect, disconnect, reset, and signal an IEEE 1284.3 data link device. These services are under development and are intended to support new devices that are not generally available. See the sample code \src\kernel\parport in the Windows DDK for more information about support for IEEE 1284.3 data link services.
