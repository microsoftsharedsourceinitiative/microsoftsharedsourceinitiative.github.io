Setting Attributes on a Parallel Device

A client uses following device control requests to set the indicated operations of a parallel device:

  • IOCTL_PAR_SET_INFORMATION initializes a parallel device.
  • IOCTL_SERIAL_SET_TIMEOUTS sets time outs for a parallel device.
  • IOCTL_PAR_SET_READ_ADDRESS sets an ECP or EPP read address (channel) for a parallel device.
  • IOCTL_PAR_SET_WRITE_ADDRESS sets an ECP or EPP write address (channel) for a parallel device.
