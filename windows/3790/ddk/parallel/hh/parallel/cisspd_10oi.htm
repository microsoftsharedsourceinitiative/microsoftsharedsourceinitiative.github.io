I/O Requests for Parallel Devices

This section describes the device-specific operations that the system-supplied bus driver for parallel ports does for the following I/O requests:

IRP_MJ_CREATE (Parallel Device)

IRP_MJ_DEVICE_CONTROL (Parallel Device)

IRP_MJ_INTERNAL_DEVICE_CONTROL (Parallel Device)

IRP_MJ_QUERY_INFORMATION (Parallel Device)

IRP_MJ_READ (Parallel Device)

IRP_MJ_WRITE (Parallel Device)

If an I/O request is not supported, the parallel port bus driver returns the status STATUS_NOT_SUPPORTED.
