Parallel Ports and Devices: Windows DDK

Sample Parallel Drivers

This section identifies the sample parallel drivers that are included with the Microsoft® Windows® DDK.

The Windows DDK includes the source code for the Parport system driver supplied by Windows XP and later. Parport is the parallel port function driver and the parallel port bus driver. See the sample code in the \src\kernel\parport directory of the Windows DDK.
