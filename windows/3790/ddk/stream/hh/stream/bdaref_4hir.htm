HierarchyAlpha

The HierarchyAlpha enumeration contains values that specify the hierarchy alpha used by a signal demodulator node. 

typedef V1_ENUM enum HierarchyAlpha {
    BDA_HALPHA_NOT_SET = -1,
    BDA_HALPHA_NOT_DEFINED = 0,
    BDA_HALPHA_1 = 1,
    BDA_HALPHA_2,
    BDA_HALPHA_4,
    BDA_HALPHA_MAX,
} HierarchyAlpha;

Elements

BDA_HALPHA_NOT_SET
Hierarchy alpha not set.
BDA_HALPHA_NOT_DEFINED
Hierarchy alpha not defined.
BDA_HALPHA_1
Hierarchy alpha set to 1.
BDA_HALPHA_2
Hierarchy alpha set to 2.
BDA_HALPHA_4
Hierarchy alpha set to 4.
BDA_HALPHA_MAX
Maximum value in the enumeration exceeded.
