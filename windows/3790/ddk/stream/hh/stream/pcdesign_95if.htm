WavePci Port

The WavePci port driver provides wave audio functionality for an audio device that resides on a PCI adapter card that can perform scatter/gather DMA transfers. The adapter driver provides a corresponding WavePci miniport (see PCI Scatter/Gather Miniport) driver object that binds itself to the WavePci port driver object to form a wave filter that can capture or render a wave stream. The port and miniport driver objects communicate with each other through their respective IPortWavePci and IMiniportWavePci interfaces.
