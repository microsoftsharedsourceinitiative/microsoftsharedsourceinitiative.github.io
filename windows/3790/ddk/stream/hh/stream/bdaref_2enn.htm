KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER

Clients use KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER along with KSPROPERTY_BDA_RF_TUNER_FREQUENCY to control the frequency setting of the tuner node.

Usage Summary Table

Get Set Target Property descriptor type Property value type
No Yes Filter KSP_NODE ULONG

Comments

The NodeId member of KSP_NODE specifies the identifier of the tuner node.

The property value specifies the frequency multiplier to set.

If the KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER property specified a multiplier of BDA_FREQUENCY_MULTIPLIER_NOT_SET (–1), BDA_FREQUENCY_MULTIPLIER_NOT_DEFINED (0), or 1, then the KSPROPERTY_BDA_RF_TUNER_FREQUENCY property specifies the frequency in Hertz. If KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER specified a multiplier of greater than 1, then the frequency in Hertz is obtained by multiplying the value of the KSPROPERTY_BDA_RF_TUNER_FREQUENCY property by the value of the KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER property.
