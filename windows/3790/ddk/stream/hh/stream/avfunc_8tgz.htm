KsFilterCreatePinFactory

The KsFilterCreatePinFactory function creates a new pin factory on the specified filter.

KSDDKAPI NTSTATUS NTAPI
  KsFilterCreatePinFactory (
    IN PKSFILTER Filter,
    IN const KSPIN_DESCRIPTOR_EX *const PinDescriptor,
    OUT PULONG PinID
    );

Parameters

Filter
Pointer to a KSFILTER structure for which to create a new pin factory.
PinDescriptor
Pointer to a KSPIN_DESCRIPTOR_EX that describes the pins this factory will create.
PinID
Pointer to the location containing the ID of the new factory.

Return Value

KsFilterCreatePinFactory returns the success or failure of the attempt to create the pin factory. Failure may occur due to invalid parameters or low memory.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Note that the filter control mutex must be held before calling this function. Fore more information, see Mutexes in AVStream.

KsFilterCreatePinFactory must be called at IRQL = PASSIVE_LEVEL.
