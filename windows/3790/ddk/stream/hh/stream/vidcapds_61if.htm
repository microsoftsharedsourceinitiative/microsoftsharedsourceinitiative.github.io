Streaming Devices (Video and Audio): Windows DDK

CC Category

Category/PinName Description
PINNAME_VIDEO_CC CC category output pins provide a stream of decoded, closed-caption byte pairs from field 1, line 21.

When creating the category PinName, PINNAME_VIDEO_CC, the following structures must be used.

Attribute Value
DataRange Structure KS_DATARANGE
DataFormat Structure KS_DATAFORMAT
Major Format GUID KSDATAFORMAT_TYPE_AUXLine21Data
Sub-Format GUID KSDATAFORMAT_SUBTYPE_Line21_BytePair
Specifier GUID KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
Required Property Sets None
Required Event Sets None
DirectShow majortype None
DirectShow formattype None
