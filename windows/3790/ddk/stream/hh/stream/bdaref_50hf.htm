BdaPropertyNodeMethods

The BdaPropertyNodeMethods function retrieves a list of methods that a node supports. 

STDMETHODIMP_(NTSTATUS) 
  BdaPropertyNodeMethods(
    IN PIRP  Irp,
    IN KSPROPERTY  *pKSProperty,
    OUT GUID  *pguidProperty
    );

Parameters

Irp
Points to the IRP for the request to retrieve the list of methods for a node.
pKSProperty
Points to a KSPROPERTY structure that describes the property and request type of the property request.
pguidProperty
Points to an array that receives the list of GUIDs for the methods supported by a node.

Return Value

Returns STATUS_SUCCESS or an appropriate error code.

Headers

Declared in Bdasup.h. Include Bdasup.h.

Comments

A BDA minidriver calls the BdaPropertyNodeMethods function to retrieve the list of methods for a node after the minidriver receives a KSPROPERTY_BDA_NODE_METHODS request of the KSPROPSETID_BdaTopology property set from the network provider. Most BDA minidrivers can define dispatch and filter-automation tables so that those minidrivers dispatch the BdaPropertyNodeMethods function directly. See Defining Automation Tables and Determining BDA Device Topology for more information.

A BDA minidriver calls BdaPropertyNodeMethods at IRQL = PASSIVE_LEVEL.

BdaPropertyNodeMethods is available on Windows® XP and later. BdaPropertyNodeMethods is available on the Windows 2000 platform only if DirectX 9.0 or later is installed on that platform.
