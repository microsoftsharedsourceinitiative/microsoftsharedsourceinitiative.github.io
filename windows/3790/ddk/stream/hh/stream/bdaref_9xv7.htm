ComponentStatus

The ComponentStatus enumeration contains values that specify the status of a substream of the received broadcast stream. 

typedef enum ComponentStatus {
    StatusActive,
    StatusInactive,
    StatusUnavailable,
} ComponentStatus;

Elements

StatusActive
Specifies the substream is active.
StatusInactive
Specifies the substream is inactive.
StatusUnavailable
Specifies the substream is not available.

Headers

Defined in Bdatypes.h. Include Bdatypes.h.
