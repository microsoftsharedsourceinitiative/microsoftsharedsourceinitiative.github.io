Audio Port Class Structures

This section describes the structures that are defined by the header file portcls.h for use with PortCls port and miniport drivers:

PCAUTOMATION_TABLE

Contains a master table of properties and events for an audio filter or pin.

PCCONNECTION_DESCRIPTOR

Describes a single data-path connection inside an audio filter.

PCEVENT_ITEM

Specifies an event supported by a particular audio filter, pin, or node.

PCEVENT_REQUEST

Specifies an event request in a call to an event handler.

PCFILTER_NODE

The null node ID (a constant) for the FromNode or ToNode field of a connection descriptor.

PCFILTER_DESCRIPTOR

Describes an audio filter's pins, nodes, connections, and properties.

PCMETHOD_ITEM

Specifies a method supported by a particular filter, pin, or node.

PCMETHOD_REQUEST

Specifies a method request in a call to a method handler.

PCNODE_DESCRIPTOR

Describes a topology node inside an audio filter.

PCPIN_DESCRIPTOR

Describes a pin factory in an audio filter.

PCPROPERTY_ITEM

Specifies a property supported by a particular filter, pin, or node.

PCPROPERTY_REQUEST

Specifies a property request in a call to a property handler.
