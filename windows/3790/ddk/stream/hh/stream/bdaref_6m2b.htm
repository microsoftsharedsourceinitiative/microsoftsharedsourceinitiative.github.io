BdaPropertyTemplateConnections

The BdaPropertyTemplateConnections function retrieves a list of connections that describe how pin types and node types are connected in a template topology. 

STDMETHODIMP_(NTSTATUS) 
  BdaPropertyTemplateConnections(
    IN PIRP  Irp,
    IN KSPROPERTY  *pKSProperty,
    OUT KSTOPOLOGY_CONNECTION  *pConnectionProperty
    );

Parameters

Irp
Points to the IRP for the request to retrieve list of connections.
pKSProperty
Points to a KSPROPERTY structure that describes the property and request type of the property request.
pConnectionProperty
Points to an array that receives the list of KSTOPOLOGY_CONNECTION structures that describe the connections in a template topology.

Return Value

Returns STATUS_SUCCESS or an appropriate error code.

Headers

Declared in Bdasup.h. Include Bdasup.h.

Comments

A BDA minidriver calls the BdaPropertyTemplateConnections function to retrieve the list of template connections after the minidriver receives a KSPROPERTY_BDA_TEMPLATE_CONNECTIONS request of the KSPROPSETID_BdaTopology property set from the network provider. Most BDA minidrivers can define dispatch and filter-automation tables so that those minidrivers dispatch the BdaPropertyTemplateConnections function directly. See Defining Automation Tables and Determining BDA Device Topology for more information.

The BDA minidriver provided a list of connections to the BDA support library in an array of KSTOPOLOGY_CONNECTION structures when the BDA minidriver passed the BDA filter template (BDA_FILTER_TEMPLATE) in BdaCreateFilterFactory and BdaInitFilter function calls. This array is a representation of all the possible connections between node and pin types that can be made within the filter or between the filter and adjoining filters. The BdaPropertyTemplateConnections function returns this array.

A BDA minidriver calls BdaPropertyTemplateConnections at IRQL = PASSIVE_LEVEL.

BdaPropertyTemplateConnections is available on Windows® XP and later. BdaPropertyTemplateConnections is available on the Windows 2000 platform only if DirectX 9.0 or later is installed on that platform.
