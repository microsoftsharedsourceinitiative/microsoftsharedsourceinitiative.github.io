KSPROPERTY_TVAUDIO_CURRENTLY_AVAILABLE_MODES

The KSPROPERTY_TVAUDIO_CURRENTLY_AVAILABLE_MODES property is used to retrieve TV audio modes that are available for the device. This property must be implemented.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY_TVAUDIO_S ULONG

The Mode member of the KSPROPERTY_TVAUDIO_S structure is used to specify the audio mode. It contains a bitwise ORing of the KS_TVAUDIO_MODE_* flags indicating the modes supported by the device at the time the information was requested.

Headers

Declared in ksmedia.h. Include ksmedia.h.
