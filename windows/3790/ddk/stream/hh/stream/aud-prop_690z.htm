KSPROPERTY_DIRECTSOUND3DBUFFER_MAXDISTANCE

The KSPROPERTY_DIRECTSOUND3DBUFFER_MAXDISTANCE property specifies the maximum distance for a 3D sound buffer.

Get Set Target Property Descriptor Type Property Value Type
Yes Yes Pin KSNODEPROPERTY FLOAT

The property value (operation data) is of type FLOAT and specifies the maximum distance. For information on distance units, see KSPROPERTY_DIRECTSOUND3DLISTENER_DISTANCEFACTOR.

Return Value

A KSPROPERTY_DIRECTSOUND3DBUFFER_MAXDISTANCE property request returns STATUS_SUCCESS to indicate that it has completed successfully. Otherwise, the request returns an appropriate error status code.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

At a distance that exceeds the maximum distance from a sound source, the sound from that source is reduced to silence. For more information on the maximum distance for a DirectSound 3D buffer, see the following in the Platform SDK documentation:

  • The flMaxDistance member of the DS3DBUFFER structure.
  • The IDirectSound3DBuffer::GetMaxDistance and IDirectSound3DBuffer::SetMaxDistance methods.
