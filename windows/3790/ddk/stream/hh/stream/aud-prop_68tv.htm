KSNODETYPE_MUX

The KSNODETYPE_MUX node represents a multiplexer (MUX). The MUX has multiple input streams and one output stream, all with the same data format. Only one input stream at a time is routed to the output stream.

A KSNODETYPE_MUX node should support the following required property:

KSPROPERTY_AUDIO_MUX_SOURCE
