PROPSETID_VIDCAP_VIDEOCONTROL

The PROPOSETID_VIDCAP_VIDEOCONTROL property set is used to control additional aspects of the video capture operation such as enumerating available frame rates and image orientation.

Support of this property set is mandatory for devices that cannot capture video at arbitrary frame rates. Generally, analog digitizers can support any frame rate request that USB and 1394 conferencing cameras support, but with a limited set of frame rates.

The following property must be implemented by video capture minidrivers:

KSPROPERTY_VIDEOCONTROL_CAPS

The following properties are optional.

KSPROPERTY_VIDEOCONTROL_ACTUAL_FRAME_RATE

KSPROPERTY_VIDEOCONTROL_FRAME_RATES

KSPROPERTY_VIDEOCONTROL_MODE

DirectShow Interface

This property set currently is not available through a DirectShow interface.
