EVENTSETID_TUNER

The EVENTSETID_Tuner event set is used to notify clients when a tuner change event occurs, such as switching a channel.

The event item in this set is specified in the KSEVENT_TUNER enumeration.

The only event in this set is KSEVENT_TUNER_CHANGED.
