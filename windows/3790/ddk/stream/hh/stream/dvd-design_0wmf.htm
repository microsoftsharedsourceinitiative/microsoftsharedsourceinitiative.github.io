Streaming Devices (Video and Audio): Windows DDK

Output Streams

The following table describes the video port output stream media types used by DVDs:

Attribute Value
Major Format GUID KSDATAFORMAT_TYPE_VIDEO
Minor Format GUID KSDATAFORMAT_SUBTYPE_VPVideo
Format Block Specifier GUID KSDATAFORMAT_SPECIFIER_NULL
No format block.

Kernel-mode interface provides control of video port extensions (VPE) settings. For more information, see Video Port Extensions Background.

The following table describes the closed caption (CC) output stream media type used by DVDs:

Attribute Value
Major Format GUID KSDATAFORMAT_TYPE_AUXLine21Data
Minor Format GUID KSDATAFORMAT_SUBTYPE_Line21_
GOPPacket
Format Block Specifier GUID KSDATAFORMAT_SPECIFIER_NONE
No format block.

A frame size of 200 (decimal) in the SampleSize member of the KSDATAFORMAT structure must be specified. For more information, see Closed Captioning Streams.
