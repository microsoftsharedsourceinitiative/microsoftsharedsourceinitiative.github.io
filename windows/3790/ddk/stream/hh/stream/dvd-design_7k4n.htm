DVD Data Streams

DVD decoder minidrivers support both input and output data streams. Streams are uniquely defined by three GUIDs (major type, minor type, and specifier) and a format block. Stream formats are also referred to as media types. For more information on format blocks and media types, see DirectShow in the Platform SDK documentation
