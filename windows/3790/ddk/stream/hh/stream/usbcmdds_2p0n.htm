USB Camera Minidrivers

The USB camera minidriver library (USBCAMD) is a kernel-mode minidriver library that can be used by developers of video camera minidrivers for the USB bus. USBCAMD simplifies the development of USB bus camera drivers by managing all stream class driver and USB bus driver interfaces for the camera minidriver.

USBCAMD is intended for streaming video cameras (web cameras). Web cameras with snapshot capability (still frames) are also supported by USBCAMD. See New in USBCAMD Version 2.0 for more information.

Hybrid cameras (those that primarily take stills but that can also capture video) may be supported by writing a USBCAMD driver and a separate Windows Image Acquisition (WIA) still camera driver (for still image storage and management). See Windows Image Acquisition Drivers for more information about WIA.

See Introduction to USB Drivers for more information about supporting USB devices.

For more information on WDM video capture minidriver development see Video capture minidrivers and the video capture technologies Web site.
