KSPROPERTY_TUNER_MODE_CAPS

The KSPROPERTY_TUNER_MODE_CAPS property describes the capabilities of a tuning mode for a device that supports TV or radio tuning. For each separate mode (analog TV, digital TV, FM, AM, or DSS), the minidriver reports capabilities such as minimum and maximum frequency, tuning granularity, settling time, and number of inputs. This property must be implemented.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY_TUNER_MODE_CAPS_S KSPROPERTY_TUNER_MODE_CAPS_S

The StandardsSupported member of the KSPROPERTY_TUNER_MODE_CAPS_S structure is used to specify the current analog video standard.
