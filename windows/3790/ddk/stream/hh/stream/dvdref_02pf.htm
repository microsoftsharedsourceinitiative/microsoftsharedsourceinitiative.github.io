KSPROPERTY_VPCONFIG_SETCONNECTINFO

The KSPROPERTY_VPCONFIG_SETCONNECTINFO property is used to set the hardware to user-defined connection information for the video port. It is a pointer to an array of DDVIDEOPORTCONNECT structures as returned by the KSPROPERTY_VPCONFIG_GETCONNECTINFO property.

Get Set Target Property descriptor type Property value type
No Yes Pin KSPROPERTY DDVIDEOPORTCONNECT

The property value (operation data) is of type DDVIDEOPORTCONNECT, which describes a hardware video port connection.

Return Value

KSPROPERTY_VPCONFIG_SETCONNECTINFO property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
