Windows Me Support for WDM Audio

Windows Me supports the following WDM audio capabilities:

  • WDM version 1.05
  • All the features that Windows 2000 supports (See Windows 2000 Support for WDM Audio.)
  • Copying, or splitting, an input capture stream to produce two identical output streams (See Splitter System Driver.)
  • Kernel-mode digital rights management (see DRM Functions.)
  • DirectSound support for non-PCM formats (See Differences in Support for Non-PCM Formats.)
  • USB audio: All the features of Windows 2000, plus MIDI

Note  In Windows Me, a conversion error in the KMixer system driver causes 24-bit and 32-bit output samples to be truncated to 16 bits of precision. In Windows 2000, KMixer has a similar problem (see Windows 2000 Support for WDM Audio). However, this problem does not occur in Windows 98 or in Windows XP and later. A hotfix is available to fix the problem in Windows Me. For more information, see Microsoft Knowledge Base Article Q316638.
