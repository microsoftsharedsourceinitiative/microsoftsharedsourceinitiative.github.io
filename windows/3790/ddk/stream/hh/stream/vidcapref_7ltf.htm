EVENTSETID_CROSSBAR

The EVENTSETID_CROSSBAR event set is used to notify clients when a crossbar change event occurs, such as when a new signal routing has been selected.

The event item in this set is specified in the KSEVENT_CROSSBAR enumeration.

The only event in this set is KSEVENT_CROSSBAR_CHANGED.
