VBI Category

Category/PinName Description
PINNAME_VIDEO_VBI VBI category pins provide a stream of vertical blanking interval (VBI) waveform samples. Typically, this stream is passed to codecs that extract closed captioning, NABTS, WST, timecode, and other digital data streams.

When creating the category PinName PINNAME_VIDEO_VBI, the following structures must be used.

Attribute Value
DataRange Structure KS_DATARANGE_VIDEO_VBI
DataFormat Structure KS_DATAFORMAT_VIDEO_VBI
MajorFormat GUID KS_DATAFORMAT_TYPE_VBI
SubFormat GUID KSDATAFORMAT_SUBTYPE_RAW8
Specifier GUID KSDATAFORMAT_SPECIFIER_VBI
Extended Header Size KS_VBI_FRAME_INFO
Required Property Sets None
Required Event Sets None
DirectShow majortype MEDIATYPE_VBI
DirectShow formattype FORMAT_VBI
