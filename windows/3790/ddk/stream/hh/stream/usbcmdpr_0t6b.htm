USBCAMD Services and Functions

The following topics list the USBCAMD services and functions available to camera minidrivers.

USBCAMD version 1.0 supports only the following services and functions:

USBCAMD_AdapterReceivePacket

USBCAMD_ControlVendorCommand

USBCAMD_DriverEntry

USBCAMD_GetRegistryKeyValue

USBCAMD_SelectAlternateInterface

USBCAMD version 2.0 supports all the previous services and functions (to ensure backward compatibility with camera minidrivers using USBCAMD version 1.0) in addition to the new ones added below:

USBCAMD_BulkReadWrite

USBCAMD_CancelBulkReadWrite

USBCAMD_Debug_LogEntry

USBCAMD_InitializeNewInterface

USBCAMD_SetIsoPipeState

USBCAMD_SetVideoFormat

USBCAMD_WaitOnDeviceEvent
