IInterruptSync::RegisterServiceRoutine

The RegisterServiceRoutine method registers the routine to be called when an interrupt occurs. 

NTSTATUS
  RegisterServiceRoutine(
    IN PINTERRUPTSYNCROUTINE  Routine,
    IN PVOID  DynamicContext,
    IN BOOLEAN  First
    );

Parameters

Routine
Pointer to the routine that is to be called. This parameter is a function pointer of type PINTERRUPTSYNCROUTINE (see IInterruptSync).
DynamicContext
Specifies a context value that is passed to the routine when it is called.
First
Specifies whether the routine is added at the head or tail of the list. If TRUE, the routine is added at the head of the list. If FALSE, the routine is added to the tail of the list. When an interrupt occurs, the routine at the head of the list is called first, and the routine at the tail is called last.

Return Value

RegisterServiceRoutine returns STATUS_SUCCESS if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.
