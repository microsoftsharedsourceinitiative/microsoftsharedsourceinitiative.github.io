KsGateTerminateOr

The KsGateTerminateOr function deletes an existing OR gate and removes an input from any attached AND gate.

VOID
  KsGateTerminateOr(
    IN PKSGATE  OrGate
    );

Parameters

OrGate
Pointer to a KSGATE structure that is the OR gate to delete. This gate must be at the beginning of a gate chain.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Logical gates should be deleted in a front to back manner. KsGateTerminateOr does not patch the gate chain for gates that are deleted at the end or in the middle. For more information, see Flow Control Gates in AVStream.

KsGateTerminateOr may be called at any IRQL.
