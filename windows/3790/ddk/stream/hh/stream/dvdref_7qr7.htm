KSPROPERTY_AUDDECOUT_MODES

The KSPROPERTY_AUDDECOUT_MODES property is read-only. The return value enumerates all the available output modes for the audio decoder and is a DWORD. It can be any combination of the bits listed in the following table.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY DWORD

The property value (operation data) is of type DWORD and can be one of the following mode constants defined in the header file ksmedia.h:

KSAUDDECOUTMODE_STEREO_ANALOG

Indicates that the output is in analog stereo.

KSAUDDECOUTMODE_PCM_51

Indicates that the output is in PCM 5.1 channel digital.

KSAUDDECOUTMODE_SPDIFF

Indicates that the output is SPDIFF format AC3 digital.

Return Value

KSPROPERTY_AUDDECOUT_MODES property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
