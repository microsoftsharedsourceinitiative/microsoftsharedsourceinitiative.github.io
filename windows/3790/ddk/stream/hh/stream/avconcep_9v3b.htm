Mutexes in AVStream

In AVStream, minidrivers synchronize access to objects by using three types of mutexes and process control gates. All three mutexes are directly accessible to the minidriver. For more information on process control gates, see Flow Control Gates in AVStream.

Synchronize hierarchy objects from device to filter with the device mutex, and synchronize objects from filter to pin using the control mutex. See related details in the sections listed below.

Several AVStream API functions require that associated mutexes be held. Notes are included in the Comments sections of relevant function reference pages stating which mutex should be held before calling that particular function.
