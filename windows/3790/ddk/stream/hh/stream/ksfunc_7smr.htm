Connection Functions

Several WDM Streaming functions assist with creating and destroying connections. The following functions are provided:

KsCreatePin

KsValidateConnectRequest

KsPinPropertyHandler

KsPinDataIntersection

KsHandleSizedListQuery

For more information on functions that handle IRPs, see the IRP Management Functions.
