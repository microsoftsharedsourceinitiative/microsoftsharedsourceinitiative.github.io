AVStream Minidriver Routines

This section contains reference pages for minidriver-supplied AVStream minidriver routines. This material is relevant only for Windows® XP and DirectX® 8.0 releases and later.

The names of these routines are vendor-defined; each is specified by a function pointer in the relevant routine or structure.
