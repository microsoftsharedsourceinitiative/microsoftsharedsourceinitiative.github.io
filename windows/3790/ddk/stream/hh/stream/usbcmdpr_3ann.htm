USBCAMD_CancelBulkReadWrite

The USBCAMD_CancelBulkReadWrite service cancels a pending bulk read or write request.

NTSTATUS
  USBBCAMD_CancelBulkReadWrite(
    IN PVOID  DeviceContext,
    IN ULONG  PipeIndex
    );

Parameters

DeviceContext
Pointer to device-specific context.
PipeIndex
Specifies the index of the bulk pipe to cancel the read or write request.

Return Value

USBCAMD_CancelBulkReadWrite returns STATUS_SUCCESS or an appropriate error code.

Headers

Declared in usbcamdi.h. Include usbcamdi.h.

Comments

USBCAMD_CancelBulkReadWrite may only be called at IRQL = PASSIVE_LEVEL.

USBCAMD_CancelBulkReadWrite is not available in USBCAMD version 1.0.
