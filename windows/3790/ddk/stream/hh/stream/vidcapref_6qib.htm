KSPROPERTY_VIDEODECODER_STANDARD

The KSPROPERTY_VIDEODECODER_STANDARD property is used to set and retrieve the current analog video standard. This property must be implemented.

Get Set Target Property descriptor type Property value type
Yes Yes Pin KSPROPERTY_VIDEODECODER_S ULONG

The Value member of the KSPROPERTY_VIDEODECODER_S structure is used to specify the current analog video standard.

Headers

Declared in ksmedia.h. Include ksmedia.h.
