KsPinHandshake

The KsPinHandshake function attempts a protocol handshake with a connected pin.

NTSTATUS
  KsPinHandshake(
    IN PKSPIN  Pin,
    IN PKSHANDSHAKE  In,
    OUT PKSHANDSHAKE  Out
    );

Parameters

Pin
Pointer to the KSPIN object that is initiating the handshake. The handshake request is passed on to the pin connected to this object.
In
Pointer to the KSHANDSHAKE structure containing the handshake information to be passed to the connected pin.
Out
Pointer to a structure that is filled in with handshake information by the connected pin.

Return Value

KsPinHandshake returns STATUS_SUCCESS if the connected pin is accepting the negotiated connection. Otherwise, it returns an appropriate error code.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Protocol handshakes can be used to negotiate private interfaces between two pins that are both AVStream pins.

Connections between AVStream pins are performed using this type of handshake.

KsPinGetHandshake must be called at IRQL = PASSIVE_LEVEL.
