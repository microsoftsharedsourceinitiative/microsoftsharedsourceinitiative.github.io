IOCTL_KS_ENABLE_EVENT

Operation

An application can use IOCTL_KS_ENABLE_EVENT to request notification of a KS event type, or to determine the events supported by a KS object. The application specifies IOCTL_KS_ENABLE_EVENT in the IoControl parameter of a call to KsSynchronousDeviceControl.

Input

The application places a pointer to a structure of type KSEVENT in the InBuffer parameter and the size of this structure at InLength. The application places a pointer to a structure of type KSEVENTDATA in the OutBuffer parameter and the size of this structure at OutLength. To determine events supported by a KS object, specify NULL and 0 respectively for InBuffer and InLength.

Output

None

Headers

Defined in ks.h. Include ks.h.
