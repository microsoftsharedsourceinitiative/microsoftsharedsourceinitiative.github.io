KsPinRegisterPowerCallbacks

The KsPinRegisterPowerCallbacks function registers power management callbacks for Pin.

VOID
  KsPinRegisterPowerCallbacks(
    IN PKSPIN  Pin,
    IN PFNKSPINPOWER  Sleep,
    IN PFNKSPINPOWER  Wake
    );

Parameters

Pin
Pointer to the KSPIN for which to register power callbacks. Note that the pin must actually process (be pin-centric) in order to receive power notification messages.
Sleep
Supplies the address of a AVStrMiniPinPower function that handles sleep requests for the device. Optional.
Wake
Supplies the address of a AVStrMiniPinPower function that handles wake requests for the device. Optional.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

At least one of the callbacks must be specified when calling KsPinRegisterPowerCallbacks.

Do not attempt to obtain the filter control mutex from within either the Sleep or Wake callback. Doing so induces a risk of deadlock. For more information on mutexes, see Mutexes in AVStream.
