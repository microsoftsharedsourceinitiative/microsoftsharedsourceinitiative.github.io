KSPROPSETID_BdaFrequencyFilter

KSPROPSETID_BdaFrequencyFilter is the BDA frequency filter property set. It is used to control the RF tuner node in a receiver topology.

The following properties are available:

KSPROPERTY_BDA_RF_TUNER_FREQUENCY
Informs the tuner node about the base frequency of the signal carrier. Multiply the base frequency by the value of the KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER property to obtain the actual frequency.
KSPROPERTY_BDA_RF_TUNER_POLARITY
Informs the tuner node about the polarization that is used on the transmitted signal.
KSPROPERTY_BDA_RF_TUNER_RANGE
Sets the tuner range.
KSPROPERTY_BDA_RF_TUNER_TRANSPONDER
Informs the tuner node of the appropriate transponder number.
KSPROPERTY_BDA_RF_TUNER_BANDWIDTH
Informs the tuner node of the bandwidth of the transmitted signal.
KSPROPERTY_BDA_RF_TUNER_FREQUENCY_MULTIPLIER
Informs the tuner node about the value with which to multiply the value of the KSPROPERTY_BDA_RF_TUNER_FREQUENCY property to obtain the actual frequency.

Comments

The KSPROPSETID_BdaFrequencyFilter property set is generic across almost all tuners. It is used to inform the tuner node how to tune the RF signal.
