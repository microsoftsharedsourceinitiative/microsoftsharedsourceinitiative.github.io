KSPROPERTY_WAVE_OUTPUT_CAPABILITIES

The KSPROPERTY_WAVE_OUTPUT_CAPABILITIES property is used to return a device's wave output capabilities.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY KSWAVE_OUTPUT_CAPABILITIES

The property value (operation data) is of type KSWAVE_OUTPUT_CAPABILITIES.

Return Value

KSPROPERTY_WAVE_OUTPUT_CAPABILITIES property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
