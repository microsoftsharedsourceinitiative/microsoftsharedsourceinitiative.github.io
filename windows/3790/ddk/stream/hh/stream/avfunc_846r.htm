KsFilterCreateNode

The KsFilterCreateNode function creates a new topology node on the specified filter.

KSDDKAPI NTSTATUS NTAPI
  KsFilterCreateNode (
    IN PKSFILTER Filter,
    IN const KSNODE_DESCRIPTOR *const NodeDescriptor,
    OUT PULONG NodeID
    );

Parameters

Filter
Pointer to a KSFILTER structure on which to create a new topology node.
NodeDescriptor
Pointer to a KSNODE_DESCRIPTOR structure that describes the new node.
NodeID
Pointer to a ULONG where AVStream places the ID of the new node.

Return Value

KsFilterCreateNode returns the success or failure of creating the node. The call may fail because of invalid parameters, low memory, or other reasons.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Note that the filter control mutex must be held before calling this function. For more information, see Filter Control Mutex in AVStream.

KsFilterCreateNode must be called at IRQL = PASSIVE_LEVEL.
