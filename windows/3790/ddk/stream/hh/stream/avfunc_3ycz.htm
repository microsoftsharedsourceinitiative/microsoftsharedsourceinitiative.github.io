KsPinGetAndGate

The KsPinGetAndGate function returns the processing control gate for Pin.

PKSGATE
  KsPinGetAndGate(
    IN PKSPIN  Pin
    );

Parameters

Pin
Pointer to the KSPIN for which to return the processing control gate.

Return Value

Returns a pointer to the KSGATE representing the processing control gate for Pin.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

For more information, see Flow Control Gates in AVStream.

KsPinGetAndGate may be called at any IRQL.
