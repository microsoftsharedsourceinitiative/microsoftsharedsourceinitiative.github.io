KsPinGetDevice

The KsPinGetDevice function returns the AVStream device to which Pin belongs.

PKSDEVICE
  KsPinGetDevice(
    IN PKSPIN  Pin
    );

Parameters

Pin
Pointer to a KSPIN structure representing the pin for which to return the owning AVStream device.

Return Value

KsPinGetDevice returns a pointer to a KSDEVICE representing the AVStream device to which Pin belongs.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This call is an inline function call to KsGetDevice.

KsPinGetDevice must be called at IRQL = PASSIVE_LEVEL.
