KSPROPSETID_DrmAudioStream

The KSPROPSETID_DrmAudioStream property set specifies the DRM content ID and DRM content rights on a KS audio stream. For background information on DRM content IDs, see Digital Rights Management.

The KSPROPSETID_DrmAudioStream property set contains only a single property item:

KSPROPERTY_DRMAUDIOSTREAM_CONTENTID
