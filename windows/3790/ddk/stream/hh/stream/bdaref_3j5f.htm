FECMethod

The FECMethod enumeration contains values that specify the forward error correction (FEC) type used by a signal demodulator node. 

typedef V1_ENUM enum FECMethod {
    BDA_FEC_METHOD_NOT_SET = -1,
    BDA_FEC_METHOD_NOT_DEFINED = 0,
    BDA_FEC_VITERBI = 1,
    BDA_FEC_RS_204_188,
    BDA_FEC_MAX,
} FECMethod;

Elements

BDA_FEC_METHOD_NOT_SET
FEC method not set.
BDA_FEC_METHOD_NOT_DEFINED
FEC method not defined.
BDA_FEC_VITERBI
FEC is Viterbi binary convolution.
BDA_FEC_RS_204_188
FEC is Reed-Solomon 204/188 (outer FEC).
BDA_FEC_MAX
Maximum value in the enumeration exceeded.

Headers

Defined in Bdatypes.h. Include Bdatypes.h.
