KSPROPSETID_CopyProt

All DVD decoder input streams must support the KSPROPETID_CopyProt property set. This property set indicates the level of copyright protection in all DVD streams. The following properties are defined:

KSPROPERTY_DVDCOPY_CHLG_KEY

KSPROPERTY_DVDCOPY_DVD_KEY1

KSPROPERTY_DVDCOPY_DEC_KEY2

KSPROPERTY_DVDCOPY_TITLE_KEY

KSPROPERTY_COPY_MACROVISION

KSPROPERTY_DVDCOPY_REGION

KSPROPERTY_DVDCOPY_SET_COPY_STATE

KSPROPERTY_DVDCOPY_DISC_KEY
