KSNODETYPE_DEMUX

The KSNODETYPE_DEMUX node represents a demultiplexer. This control has a single input stream and multiple output streams.

A KSNODETYPE_DEMUX node should support the following property:

KSPROPERTY_AUDIO_DEMUX_DEST
