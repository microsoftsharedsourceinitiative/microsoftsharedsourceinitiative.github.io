KSEVENTSETID_VIDCAP_TVAUDIO

The KSEVENTSETID_VIDCAP_TVAUDIO event set is used to notify clients when a TV audio change event occurs, such as when a newly tuned-to channel supports a secondary audio program (SAP).

The event item in this set is specified in the KSEVENT_TVAUDIO enumeration.

The only event in this set is KSEVENT_TVAUDIO_CHANGED.
