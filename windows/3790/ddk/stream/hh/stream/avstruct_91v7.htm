KSNODE_DESCRIPTOR

The KSNODE_DESCRIPTOR structure describes a topology node within a filter.

typedef struct {
  const KSAUTOMATION_TABLE*  AutomationTable;
  const GUID*  Type;
  const GUID*  Name;
} KSNODE_DESCRIPTOR, *PKSNODE_DESCRIPTOR;

Members

AutomationTable
Pointer to a KSAUTOMATION_TABLE structure for this topology node. The automation table contains the properties, methods, and events supported by this topology node.
Type
Pointer to a GUID defining the topology node.
Name
Pointer to a GUID that represents the name of this topology node. This is used to return information on a KSPROPERTY_TOPOLOGY_NAME query.

Headers

Declared in ks.h. Include ks.h.

Comments

This structure is available in Windows XP and later operating systems and in DirectX 8.0 and later versions.
