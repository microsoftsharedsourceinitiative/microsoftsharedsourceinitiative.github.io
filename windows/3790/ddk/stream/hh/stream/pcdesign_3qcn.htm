Installing in Windows 98/Me

In the following example of an INF device-driver-installation section, the AlsoInstall directive adds the system-driver information that installs the core system components for an audio adapter in Windows 98/Me:

  [XYZ-Audio-Device.Registration]
  AlsoInstall=KS.Registration(ks.inf),WDMAUDIO.Registration(wdmaudio.inf)

The AlsoInstall directive should only be used if the INF file is provided by a vendor and the device is installed in Windows 98/Me. For Windows 2000 and later, use the Include and Needs directives instead.
