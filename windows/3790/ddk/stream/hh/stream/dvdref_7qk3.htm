KSPROPERTY_DVDSUBPIC_PALETTE

The KSPROPERTY_DVDSUBPIC_PALETTE property is used to set the palette that is used by the subpicture stream.

Get Set Target Property descriptor type Property value type
No Yes Pin KSPROPERTY KSPROPERTY_SPPAL

The property value (operation data) is of type KSPROPERTY_SPPAL.

Return Value

KSPROPERTY_DVDSUBPIC_PALETTE property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.

Comments

The structure used by this property is KSPROPERTY_SPPAL. The format is described in the DVD specification. This structure contains an array of 16 YUV elements. These elements correspond to the 4-bit color numbers requested within the subpicture command stream.
