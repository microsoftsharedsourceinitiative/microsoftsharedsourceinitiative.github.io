KSPROPERTY_VIDEODECODER_CAPS

The KSPROPERTY_VIDEODECODER_CAPS property describes the basic capabilities of a video decoder. This property must be implemented.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY_VIDEODECODER_CAPS_S KSPROPERTY_VIDEODECODER_CAPS_S

Information pertaining to this property is specified in a KSPROPERTY_VIDEODECODER_CAPS_S structure.

Headers

Declared in ksmedia.h. Include ksmedia.h.
