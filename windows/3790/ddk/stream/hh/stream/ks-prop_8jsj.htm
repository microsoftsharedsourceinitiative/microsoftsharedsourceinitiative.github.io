KSPROPERTY_STREAMALLOCATOR_STATUS

The KSPROPERTY_STREAMALLOCATOR_STATUS property retrieves the current status of the given allocator in the KSSTREAMALLOCATOR_STATUS structure.

A client can access this property using the following sample code:

DeviceIoControl(
    AllocatorHandle,
    IOCTL_KS_PROPERTY,
    &Allocator,
    sizeof(KSPROPERTY),
    &AllocatorStatus,
    sizeof(KSSTREAMALLOCATOR_STATUS),
    &BytesReturned,
    &Overlapped);

The STREAMALLOCATOR_STATUS structure description is as follows:

typedef struct {
  KSALLOCATOR_FRAMING  Framing;
  ULONG  AllocatedFrames;
  ULONG  Reserved;
} KSSTREAMALLOCATOR_STATUS;

Members

Framing
Specifies the framing specified when the allocator was created.
AllocatedFrames
Specifies the current number of allocated frames. Obviously, this number is dynamic in nature.
Reserved
Reserved and set to zero.

Comments

The status of the allocator indicates the framing specifications and the currently allocated frames.
