KSPROPERTY_MEDIASEEKING_POSITIONS

The KSPROPERTY_MEDIASEEKING_POSITIONS property sets the media time and/or the stop time on a filter.

A client can access this property using the following sample code:

DeviceIoControl(
    FilterHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &Positions,
    sizeof(KSPROPERTY_POSITIONS),
    &BytesReturned,
    &Overlapped);

Comments

KSPROPERTY_MEDIASEEKING_POSITIONS is a write-only property
