KsDispatchIrp

This function is used to multiplex an Irp to a specific file context.

KSDDKAPI
NTSTATUS
NTAPI
KsDispatchIrp(
    IN PDEVICE_OBJECT DeviceObject,
    IN PIRP Irp
    );

Parameters

DeviceObject
Irp

Return Value

Headers

Declared in ks.h. Include ks.h.

Comments

KsDispatchIrp will either make a call into the file object's KSDISPATCH_TABLE to handle the IRP, or if the file object refers to an AVStream device, it will directly dispatch the IRP to the appropriate AVStream device. Very few AVStream minidrivers will use this function. Class drivers written under AVStream may use it.
