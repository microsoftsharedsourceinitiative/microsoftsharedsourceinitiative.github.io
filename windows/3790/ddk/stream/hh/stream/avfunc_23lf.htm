KsFilterFactoryGetDevice

The KsFilterFactoryGetDevice function returns the AVStream device to which FilterFactory belongs.

PKSDEVICE __inline
  KsFilterFactoryGetDevice(
    IN PKSFILTERFACTORY  FilterFactory
    );

Parameters

FilterFactory
Pointer to a KSFILTERFACTORY for which to find the corresponding AVStream device.

Return Value

Returns a pointer to a KSDEVICE structure representing the AVStream device to which FilterFactory belongs.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This call is an inline function call to KsGetDevice.

For more information on the AVStream object hierarchy, see AVStream Overview.

KsFilterFactoryGetDevice must be called at IRQL = PASSIVE_LEVEL.
