KSPROPERTY_CONNECTION_DATAFORMAT

Clients use the KSPROPERTY_CONNECTION_DATAFORMAT property to get or set the current data format.

Specifying This Property

KSPROPERTY

Property Data

KSDATAFORMAT, specifying the current data format.

Comments

KSPROPERTY_CONNECTION_DATAFORMAT is read/write. KS filters only need to support this property if they allow clients to reset the current property, or if connections can be made with the data format incompletely specified.
