KsFilterGetAndGate

The KsFilterGetAndGate function returns Filter's AND gate.

PKSGATE 
  KsFilterGetAndGate(
    IN PKSFILTER  Filter
    );

Parameters

Filter
Pointer to the KSFILTER for which to acquire the corresponding AND gate.

Return Value

KsFilterGetAndGate returns Filter's AND gate as a pointer to a KSGATE structure.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

A minidriver can use AND gates on a filter as a processing control mechanism for that filter.
