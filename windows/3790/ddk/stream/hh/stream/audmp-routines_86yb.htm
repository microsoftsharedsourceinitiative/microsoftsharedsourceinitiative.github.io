IPortWavePciStream::TerminatePacket

The TerminatePacket method terminates the packet currently being mapped.

NTSTATUS
  TerminatePacket(
    void
    );

Parameters

None

Return Value

TerminatePacket returns STATUS_SUCCESS if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.

Comments

The TerminatePacket method is used primarily for capture situations in which packets need to end at specific points in the stream (for example, for video frames).

Callers of TerminatePacket must run at IRQL <= DISPATCH_LEVEL.
