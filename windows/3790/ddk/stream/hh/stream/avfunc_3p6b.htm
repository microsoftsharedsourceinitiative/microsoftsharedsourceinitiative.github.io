KsPinSetPinClockTime

The KsPinSetPinClockTime function sets the current time on the clock exposed by Pin.

VOID
  KsPinSetPinClockTime(
    IN PKSPIN  Pin,
    IN LONGLONG  Time
    );

Parameters

Pin
Pointer to a KSPIN structure representing the AVStream pin object for which the exposed clock is set to the time specified by Time.
Time
Specifies the time that is set on the clock exposed by Pin. Use IKsReferenceClock_GetResolution to verify granularity.

Return Value

None

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

KsPinSetPinClockTime modifies the current time returned by the clock. If an external clock is used, this function can still be used to force a resetting of the current timer when the external timer is not being used. In this case, the time provided is ignored and must be set to zero.

KsPinSetPinClockTime must be called at IRQL <= DISPATCH_LEVEL.
