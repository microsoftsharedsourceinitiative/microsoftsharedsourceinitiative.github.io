GuardInterval

The GuardInterval enumeration contains values that specify the type of guard interval used by a signal demodulator node. 

typedef V1_ENUM enum GuardInterval {
    BDA_GUARD_NOT_SET = -1,
    BDA_GUARD_NOT_DEFINED = 0,
    BDA_GUARD_1_32 = 1,
    BDA_GUARD_1_16,
    BDA_GUARD_1_8,
    BDA_GUARD_1_4,
    BDA_GUARD_MAX,
} GuardInterval;

Elements

BDA_GUARD_NOT_SET
Guard interval not set.
BDA_GUARD_NOT_DEFINED
Guard interval not defined.
BDA_GUARD_1_32
Guard interval set to 1/32.
BDA_GUARD_1_16
Guard interval set to 1/16.
BDA_GUARD_1_8
Guard interval set to 1/8.
BDA_GUARD_1_4
Guard interval set to 1/4.
BDA_GUARD_MAX
Maximum value in the enumeration exceeded.
