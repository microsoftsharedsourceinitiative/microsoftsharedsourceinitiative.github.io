Windows 98 Second Edition Support for WDM Audio

Windows 98 SE supports the following WDM capabilities:

  • WDM version 1.10
  • Complete support for the following audio port drivers: Topology, WaveCyclic, DMus, and MIDI
  • Partial support for the WavePci port driver, which is subject to the following limitations:
    • It supports a maximum of 64 streams.
    • IMiniportWavePciStream::GetPosition does not work in all situations. For details, see Differences in Support for Non-PCM Formats.
  • System-supplied MIDI miniport drivers: FMSynth, DMusUART, and UART (See PcNewMiniport.)
  • DirectSound hardware mixing and 3D acceleration
  • KMixer sample-rate conversion (SRC) for capture streams (See KMixer Driver Sample Rate Conversion and Mixing Policy.)
  • Multichannel formats (See the white paper entitled Multiple Channel Audio Data and WAVE Files at the audio technology Web site.
  • Other DirectMusic kernel support, including the DMusic system driver (dmusic.sys)
  • USB audio: All the features of Windows 98, plus multichannel support
