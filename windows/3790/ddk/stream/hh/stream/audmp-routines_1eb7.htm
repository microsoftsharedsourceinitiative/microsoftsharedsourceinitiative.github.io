IMiniportMidiStream::Write

The Write method writes data to an outgoing MIDI stream.

NTSTATUS
  Write(
    IN PVOID  BufferAddress,
    IN ULONG  BytesToWrite,
    OUT PULONG  BytesWritten
    );

Parameters

BufferAddress
Pointer to the buffer that contains the MIDI data for the stream
BytesToWrite
Specifies the length in bytes of the MIDI data stream buffer at BufferAddress.
BytesWritten
Output pointer to a caller-allocated variable into which the method writes a count specifying the actual number of bytes successfully written to the stream. This value should be either the same value as BytesToWrite or the next greater multiple of four.

Return Value

Write returns STATUS_SUCCESS if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in portcls.h. Include portcls.h.
