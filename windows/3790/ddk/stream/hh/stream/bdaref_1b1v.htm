KSPROPSETID_BdaVoidTransform

KSPROPSETID_BdaVoidTransform is the BDA void transform property set. It is used for controlling when nodes start and stop. The network provider filter uses this property set to control the operation of nodes.

The following properties are available:

KSPROPERTY_BDA_VOID_TRANSFORM_START
Starts operation of a node.
KSPROPERTY_BDA_VOID_TRANSFORM_STOP
Stops operation of a node.
