KSE_NODE

Just as KSP_NODE is used for properties on a node, the KSE_NODE structure is used for events on a node.

typedef struct {
    KSEVENT Event;
    ULONG NodeId;
    ULONG Reserved;
} KSE_NODE;

Members

Event
Specifies the event.
NodeId
Specifies the node ID.
Reserved
Reserved for system use. Should be set to zero.

Headers

Declared in ks.h. Include ks.h.
