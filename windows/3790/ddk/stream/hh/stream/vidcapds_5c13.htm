TELETEXT Category

Category/PinName Description
PINNAME_VIDEO_TELETEXT TELETEXT category output pins provide a stream of decoded World Standard Teletext (WST) data.

When creating the category PinName PINNAME_VIDEO_TELETEXT, the following structures must be used.

Attribute Value
DataRange Structure KS_DATARANGE_VIDEO_VBI
DataFormat Structure KS_DATAFORMAT_VIDEO_VBI
Major Format GUID KS_DATAFORMAT_TYPE_VBI
SubFormat GUID KSDATAFORMAT_SUBTYPE_TELETEXT
Specifier GUID KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
Required Property Sets None
Required Event Sets None
DirectShow majortype MEDIATYPE_VBI
DirectShow formattype FORMAT_VBI
