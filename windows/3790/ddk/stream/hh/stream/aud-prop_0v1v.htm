KSNODETYPE_LOUDNESS

The KSNODETYPE_LOUDNESS node represents a loudness control that adjusts the dynamic range of the output stream. The loudness control has one input stream and one output stream, and each of the two streams has the same data format.

A KSNODETYPE_LOUDNESS node should support at least one of the following properties:

KSPROPERTY_AUDIO_LOUDNESS

KSPROPERTY_AUDIO_DYNAMIC_RANGE
