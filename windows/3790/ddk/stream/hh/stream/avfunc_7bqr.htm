KsFilterGetNextSiblingFilter

The KsFilterGetNextSiblingFilter function returns the next instantiated filter belonging to the parent filter factory of Filter.

PKSFILTER __inline
  KsFilterGetNextSiblingFilter(
    IN PKSFILTER  Filter
    );

Parameters

Filter
Pointer to the KSFILTER structure for which to find the next sibling filter.

Return Value

KsFilterGetNextSiblingFilter returns a pointer to the next sibling KSFILTER of Filter. If no such instantiated filter exists, it returns NULL.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This call is an inline function call to KsGetNextSibling. Note that the object hierarchy is only guaranteed stable while the appropriate mutex is held, in this case the device mutex. For more information, see Mutexes in AVStream.

KsFilterGetNextSiblingFilter must be called at IRQL = PASSIVE_LEVEL.
