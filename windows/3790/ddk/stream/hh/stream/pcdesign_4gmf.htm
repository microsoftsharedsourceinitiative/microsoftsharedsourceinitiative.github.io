WaveCyclic Port

The WaveCyclic port driver provides wave audio functionality on a DMA-based audio device that processes audio data in a cyclic buffer. This device is typically a hardware function on an internal adapter card that plugs into an ISA or PCI system bus. The adapter driver provides a corresponding WaveCyclic miniport driver object that binds itself to the WaveCyclic port driver object to form a wave filter that can capture or render a wave stream. The port and miniport driver objects communicate with each other through their respective IPortWaveCyclic and IMiniportWaveCyclic interfaces.
