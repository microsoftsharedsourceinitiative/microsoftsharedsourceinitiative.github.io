KsDeviceRegisterAggregatedClientUnknown

This inline function is a wrapper for KsRegisterAggregatedClientUnknown.

KSDDKAPI PUNKNOWN NTAPI
  KsDeviceRegisterAggregatedClientUnknown(
    IN PKSDEVICE  Device,
    IN PUNKNOWN  ClientUnknown 
    );

Parameters

Device
Pointer to a KSDEVICE structure representing the specified AVStream device.
ClientUnknown
Pointer to the client's undelegated IUnknown interface.

Return Values

Returns the newly created aggregate object.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

This inline function performs a typecast and then calls KsRegisterAggregatedClientUnknown.
