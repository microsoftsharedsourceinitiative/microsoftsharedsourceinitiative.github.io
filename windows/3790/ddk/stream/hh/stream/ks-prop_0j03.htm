KSPROPERTY_TOPOLOGY_NAME

The KSPROPERTY_TOPOLOGY_NAME property provides the localized Unicode string name of the node.

Specifying This Property

KSP_NODE. The NodeId member specifies the node ID of the node.

Property Data

A buffer to hold the string name.

Comments

KSPROPERTY_TOPOLOGY_NAME is read-only.
