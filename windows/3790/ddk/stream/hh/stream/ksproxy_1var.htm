IKsPinFactory

The IKsPinFactory interface provides a method that retrieves the identifier of a pin factory.

The IID for this interface is IID_IKsPinFactory.

When to Implement

It is not necessary to implement the method of this interface. It is implemented in ksproxy.lib.

When to Use

Use this interface to retrieve the identifier of a pin factory.

Methods

The following methods are listed in Vtable order.

IUnknown method Description
QueryInterface Returns pointers to supported interfaces.
AddRef Increments reference count.
Release Decrements reference count.

IKsPinFactory method Description
KsPinFactory Retrieves the identifier of a pin factory.
