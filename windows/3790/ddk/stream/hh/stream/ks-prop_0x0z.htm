KSPROPERTY_MEDIASEEKING_STOPPOSITION

The KSPROPERTY_MEDIASEEKING_STOPPOSITION property retrieves the stop media time on a filter.

A client can access this property using the following sample code:

DeviceIoControl(
    FilterHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &StopPosition,
    sizeof(LONGLONG),
    &BytesReturned,
    &Overlapped);

Comments

KSPROPERTY_MEDIASEEKING_STOPPOSITION is a read-only property.

The stop media time is the time that can be set if a stop time is supported by the filter.
