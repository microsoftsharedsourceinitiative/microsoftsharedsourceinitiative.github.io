KSPROPERTY_TUNER_CAPS

The KSPROPERTY_TUNER_CAPS property is used to describe the basic capabilities of a tuner. This property must be implemented.

Get Set Target Property descriptor type Property value type
Yes No Pin KSPROPERTY_TUNER_CAPS_S KSPROPERTY_TUNER_CAPS_S

Information pertaining to this property is expressed in the KSPROPERTY_TUNER_CAPS_S structure.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

A single tuning device may support tuning digital and analog TV and AM/FM radio, as well as digital satellite systems (DSS). The video capture minidriver indicates which tuning modes it supports through the KSPROPERTY_TUNER_CAPS property. Properties of the individual tuning modes are indicated through the KSPROPERTY_TUNER_MODE_CAPS property.
