KsGetFilterFromIrp

The KsGetFilterFromIrp function returns the AVStream filter object associated with a given IRP.

PKSFILTER
  KsGetFilterFromIrp(
    PIRP  Irp
    );

Parameters

Irp
Pointer to the IRP structure for which to return the associated filter.

Return Value

KsGetFilterFromIrp returns either a pointer to the KSFILTER structure associated with the given IRP or NULL. NULL indicates that either the IRP was not associated with an AVStream object or that it was associated with a topology node. Currently, IRPs associated with topology nodes cannot be queried for their associated filter.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

If the IRP is I/O for a pin, KsGetFilterFromIrp returns the filter to which the pin belongs. If the IRP is I/O for a topology node, KsGetFilterFromIrp returns NULL.

Currently, IRPs associated with topology nodes cannot be queried for associated filters. This may change in a future revision of AVStream.
