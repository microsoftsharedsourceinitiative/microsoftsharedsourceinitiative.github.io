KSAC3_DIALOGUE_LEVEL

The KSAC3_DIALOGUE_LEVEL structure specifies the average volume level of spoken dialog within the audio program encoded in an AC3 stream.

typedef struct
{
  ULONG  DialogueLevel;
} KSAC3_DIALOGUE_LEVEL, *PKSAC3_DIALOGUE_LEVEL;

Members

DialogueLevel
Specifies how far the average dialog level is below digital 100 percent. Valid values are 1 through 31. The values 1 through 31 are interpreted as -1 decibels through -31 decibels with respect to digital 100 percent.

Headers

Declared in ksmedia.h. Include ksmedia.h.

Comments

This structure is used by the KSPROPERTY_AC3_BIT_STREAM_MODE property.

For more information on dialog levels, see the AC3 specification at the Dolby Laboratories Web site. The specification is titled Digital Audio Compression Standard (AC-3).
