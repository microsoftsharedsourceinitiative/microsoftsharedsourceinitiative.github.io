Allocator Functions

This section specifies the helper functions for allocators. These functions allow a device object to create allocators, interface to the system default allocator, and validate creation requests. The following functions are provided:

KsCreateAllocator

KsCreateDefaultAllocator

KsValidateAllocatorCreateRequest

KsCreateDefaultAllocatorEx
