ComponentCategory

The ComponentCategory enumeration contains values that specify a substream of the received broadcast stream. 

typedef V1_ENUM enum ComponentCategory {
    CategoryNotSet = -1,
    CategoryOther = 0,
    CategoryVideo,
    CategoryAudio,
    CategoryText,
    CategoryData,
} ComponentCategory;

Elements

CategoryNotSet
Specifies that the category of the substream is not set.
CategoryOther
Specifies a category other than video, audio, text, or data.
CategoryVideo
Specifies that the category of the substream is video.
CategoryAudio
Specifies that the category of the substream is audio.
CategoryText
Specifies that the category of the substream is text.
CategoryData
Specifies that the category of the substream is data.
