AV/C Protocol and Subunit Driver Support

This section describes the Microsoft implementation of the Audio/Video Control (AV/C) protocol and provides guidelines for developing subunit minidrivers. Examples of AV/C subunit types include TV tuners, tape, disk and cameras.

The AV/C protocol is built upon the IEEE 1394 bus and 61883 protocol. For more information about these topics, see The IEEE 1394 Driver Stack and IEC-61883 Client Drivers. Subunit drivers are WDM drivers that are either stream class-based (Kernel Streaming 1.0) or AVStream class-based (Kernel Streaming 2.0). See AVStream Overview for further information on AVStream Minidrivers. The AVStream class is the preferred interface to use.

See Kernel Streaming and Streaming Minidrivers for more information about the stream class and streaming drivers.
