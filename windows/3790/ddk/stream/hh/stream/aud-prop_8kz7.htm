DRMRIGHTS

The DRMRIGHTS structure specifies the DRM content rights assigned to a KS audio pin or to a port-class driver's stream object.

typedef struct tagDRMRIGHTS
{
  BOOL  CopyProtect;
  ULONG  Reserved;
  BOOL  DigitalOutputDisable;
} DRMRIGHTS, *PDRMRIGHTS;

Members

CopyProtect
Specifies one of the following copy-protection values:
TRUE
Enable copy protection. A software component (1) should not store the content in any form in any nonvolatile storage, and (2) should not pass the content by reference or value to any other component within the host system that is not authenticated by the DRM system.
FALSE
Disable copy protection. No restrictions apply to copying of content.
Reserved
Reserved for future use. Initialize to zero.
DigitalOutputDisable
Specifies one of the following digital output protection values:
TRUE
Disable digital outputs. A software component must not transfer the content out of the host system through any type of digital interface. Note that digital output protection does not affect USB devices because the host system includes USB devices.
FALSE
Enable digital outputs. No restrictions apply to transferring the content from the host system to an external component.

This flag applies primarily to S/PDIF outputs.

Headers

Declared in drmk.h. Include drmk.h.

Comments

If a structure of type DRMRIGHTS is declared a constant, the DEFINE_DRMRIGHTS_DEFAULT macro can be used to statically assign default values to the members of the structure.
