KSPROPERTY_RTC_READER

The KSPROPERTY_RTC_READER property is used to retrieve the relative time counter (RTC) for the current tape position.

Get Set Target Property descriptor type Property value type
Yes No Device KSPROPERTY_TIMECODE_S KSPROPERTY_TIMECODE_S

The TimecodeSamp member of the KSPROPERTY_TIMECODE_S structure is used to describe the relative time counter for the current tape position.

Headers

Declared in ksmedia.h. Include ksmedia.h.
