ATSCComponentTypeFlags

The ATSCComponentTypeFlags enumeration contains a value that specifies a bitmask of the various properties of an ATSC stream. 

typedef enum ATSCComponentTypeFlags {
    ATSCCT_AC3 = 0x00000001,
} ATSCComponentTypeFlags;

Element

ATSCCT_AC3
Bitmask that specifies an ATSC AC3 audio stream type.
