AV/C Streaming

This section describes the AV/C Streaming filter driver. Avcstrm.sys is a lower level subunit filter driver that is located immediately above avc.sys and 61883.sys in the driver stack and below any subunit drivers. The AV/C Stream filter driver provides additional support for the AV/C protocol driver. This support is strictly optional.
