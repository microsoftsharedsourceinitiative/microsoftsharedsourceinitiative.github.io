KSEVENTSETID_Clock

Clients can request to be notified of clock state events on a clock by registering these events with the clock's file object:

KSEVENT_CLOCK_INTERVAL_MARK

KSEVENT_CLOCK_POSITION_MARK

When a client submits the IOCTL_KS_ENABLE_EVENT request to register for event notification, it submits as a data buffer the structure documented in the EventData section of each event reference page.
