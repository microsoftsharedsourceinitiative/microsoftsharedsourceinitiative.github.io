AV/C Streaming Resources

The AV/C Stream filter driver relies on the 61883.sys protocol driver. It uses DDIs provided by 61883.sys to do isochronous connection and isochronous data streaming, and uses avc.sys to issue AV/C commands for external device control. The following list shows related documentation and references:
