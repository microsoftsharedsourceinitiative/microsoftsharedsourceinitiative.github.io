KSPROPSETID_AudioDecoderOut

The audio stream should support the KSPROPSETID_AudioDecoderOut property set. This property set indicates the forms of audio output available from the decoder. The following properties are available:

KSPROPERTY_AUDDECOUT_MODES

KSPROPERTY_AUDDECOUT_CUR_MODE
