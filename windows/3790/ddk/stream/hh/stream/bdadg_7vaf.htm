Understanding BDA Driver Samples

The DDK contains the following Broadcast Driver Architecture (BDA) driver samples:

  • BDACap

    This sample generic video capture driver demonstrates how to write a pin-centric AVStream minidriver. The sample also shows how to implement DMA by using the related functionality provided by AVStream. The BDACap sample is designed to be used with the BDATuner sample.

  • BDATuner

    This sample generic tuner driver provides hardware vendors assistance in writing their own BDA minidrivers. This sample does not have any associated hardware. The BDATuner sample is designed to be used with the BDACap sample.

For details on these samples, see the source code for each sample in the DDK.
