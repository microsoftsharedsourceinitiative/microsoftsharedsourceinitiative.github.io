KSPROPERTY_TOPOLOGY_NODES

KSPROPERTY_TOPOLOGY_NODES provides a list of the topology nodes and node types GUIDs supported by the filter.

Specifying This Property

KSPROPERTY

Property Data

A KSMULTIPLE_ITEM structure, followed by a sequence of GUIDs representing the node types. The index within the sequence must match the node ID number.

Comments

KSPROPERTY_TOPOLOGY_NODES is read-only.
