Streaming Devices (Video and Audio): Windows DDK

KSEVENT_TVAUDIO_CHANGED

The KSEVENT_TVAUDIO_CHANGED event is used to propagate an action, such as a newly tuned-to channel supports stereo audio, from the kernel-mode video capture minidriver to DirectShow in user-mode.

Get Set Target Event descriptor type Event value type
No Yes Pin KSE_NODE KSEVENTDATA
