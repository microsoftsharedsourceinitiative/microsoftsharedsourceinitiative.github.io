IAllocatorMXF::GetBuffer

The GetBuffer method returns a pointer to a buffer. The miniport driver calls this method only when it needs to send large chunks of data to the capture sink.

HRESULT
  GetBuffer(
    OUT PBYTE  *ppbBuffer
    );

Parameter

ppbBuffer
Output pointer for the buffer. This parameter points to a caller-allocated pointer variable into which the method writes the buffer pointer.

Return Value

GetBuffer returns S_OK if the call was successful. Otherwise, the method returns an appropriate error code.

Headers

Declared in dmusicks.h. Include dmusicks.h.

Comments

This method is used whenever a component needs to package more data than can be stored in the uData member of DMUS_KERNEL_EVENT. The allocator returns a pointer to a buffer when GetBuffer is called. The size of this buffer is determined by calling IAllocatorMXF::GetBufferSize. This will only need to be done once because the buffer size is constant for any allocator implementation.
