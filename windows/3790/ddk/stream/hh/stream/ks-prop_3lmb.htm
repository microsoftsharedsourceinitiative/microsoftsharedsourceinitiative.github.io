KSPROPERTY_STREAM_TIMEFORMAT

The KSPROPERTY_STREAM_TIMEFORMAT property is used to retrieve the time format used on a particular pin connection.

A client can access this property using the following sample code:

DeviceIoControl(
    DeviceHandle,
    IOCTL_KS_PROPERTY,
    &Property,
    sizeof(KSPROPERTY),
    &Guid,
    sizeof(GUID),
    &BytesReturned,
    &Overlapped);

Return Values

The property returns a GUID specifying the time format used in the connection and indicating the format of the presentation time and extent. The defined time formats correspond to those defined by DirectShow.

Comments

KSPROPERTY_STREAM_TIMEFORMAT is an optional property that should be implemented if the pin supports the rate, presentation time/extent, or skip degradation properties (for more information on these properties, see Quality Management). This allows a client to determine the time format used for connection and the format of the time stamp information used in rate, presentation time/extent, and skip degradation operations.
