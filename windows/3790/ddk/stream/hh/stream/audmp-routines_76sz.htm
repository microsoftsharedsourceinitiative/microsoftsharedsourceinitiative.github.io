IDmaChannel::MaximumBufferSize

The MaximumBufferSize method returns the size in bytes of the largest buffer that this DMA-channel object is configured to support.

ULONG
  MaximumBufferSize(
    VOID
    );

Parameters

None

Return Value

MaximumBufferSize returns the size in bytes of the largest buffer that this DMA channel supports.

Headers

Declared in portcls.h. Include portcls.h.

Comments

Callers of MaximumBufferSize can run at any IRQL.
