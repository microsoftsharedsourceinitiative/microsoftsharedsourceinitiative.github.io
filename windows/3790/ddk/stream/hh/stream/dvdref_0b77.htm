KSPROPSETID_DvdSubPic

The following list of properties control the color, contrast, and output of the subpicture display. The GUID of this property set is KSPROPERTY_DvdSubPic. The following properties are defined:

KSPROPERTY_DVDSUBPIC_PALETTE

KSPROPERTY_DVDSUBPIC_HLI

KSPROPERTY_DVDSUBPIC_COMPOSIT_ON
