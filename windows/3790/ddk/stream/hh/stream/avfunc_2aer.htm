KsFilterRegisterAggregatedClientUnknown

This inline function is a wrapper for KsRegisterAggregatedClientUnknown.

KSDDKAPI PUNKNOWN NTAPI
  KsFilterRegisterAggregatedClientUnknown(
    IN PKSFILTER  Filter,
    IN PUNKNOWN  ClientUnknown 
    );

Parameters

Filter
Pointer to the specified AVStream KSFILTER structure.
ClientUnknown
The client IUnknown interface object.

Return Values

KsFilterRegisterAggregatedClientUnknown returns the newly created aggregate object.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

Note that this inline function only performs a typecast and then calls KsRegisterAggregatedClientUnknown.
