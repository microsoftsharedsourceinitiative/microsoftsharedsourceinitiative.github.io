IKsAllocatorEx

The IKsAllocatorEx interface is for proxy use and not recommended for application use. IKsAllocatorEx inherits all the methods of the IKsAllocator interface and extends IKsAllocator to provide methods that further control and query an allocator.

The IID for this interface is IID_IKsAllocatorEx.

When to Implement

It is not necessary to implement the methods of this interface. They are implemented in Ksproxy.lib.

When to Use

Not recommended for application use.

Methods

The following methods are listed in Vtable order.

IUnknown method Description
QueryInterface Returns pointers to supported interfaces.
AddRef Increments reference count.
Release Decrements reference count.

IKsAllocator method Description
KsGetAllocatorHandle Retrieves a file handle to an allocator.
KsGetAllocatorMode Returns the mode in which an allocator allocates memory.
KsGetAllocatorStatus Retrieves the status of an allocator.
KsSetAllocatorMode Sets the mode in which an allocator allocates memory.

IKsAllocatorEx method Description
KsGetProperties Returns the properties for an allocator.
KsSetProperties Sets the properties for an allocator.
KsSetAllocatorHandle Sets the handle for an allocator.
KsCreateAllocatorAndGetHandle Creates an allocator for a pin and retrieves a file handle to the allocator.
