Timecode Category

Category/PinName Description
PINNAME_VIDEO_TIMECODE Timecode category output pins provide a stream of timecode.

When creating the category PinName PINNAME_VIDEO_TIMECODE, the following structures must be used.

Attribute Value
DataRange Structure KSDATARANGE
DataFormat Structure KSDATAFORMAT
Major Format GUID None defined. Define GUID using Direct Show's MEDIATYPE_Timecode.
Sub-Format GUID KSDATAFORMAT_SUBTYPE_NONE
Specifier GUID KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
Required Property Sets None
Required Event Sets None
DirectShow majortype MEDIATYPE_Timecode
DirectShow formattype MEDIASUBTYPE_None
