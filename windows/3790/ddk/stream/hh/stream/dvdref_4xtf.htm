KSPROPERTY_VPCONFIG_SURFACEPARAMS

The KSPROPERTY_VPCONFIG_SURFACEPARAMS property is used to specify the video-port settings, such as x and y origin and surface pitch.

Get Set Target Property descriptor type Property value type
No Yes Pin KSPROPERTY KSVPSURFACEPARAMS

The property value (operation data) is of type KSVPSURFACEPARAMS.

Return Value

KSPROPERTY_VPCONFIG_SURFACEPARAMS property requests return STATUS_SUCCESS to indicate successful completion. Otherwise, requests return an appropriate error status code.
