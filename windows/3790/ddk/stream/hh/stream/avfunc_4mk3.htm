KsPinGetConnectedPinFileObject

The KsPinGetConnectedPinFileObject function returns the file object for the pin to which Pin is connected. Works only for source pins.

PFILE_OBJECT
  KsPinGetConnectedPinFileObject(
    IN PKSPIN  Pin
    );

Parameters

Pin
Pointer to a KSPIN structure that is the AVStream pin object for which to acquire the file object for the connected pin.

Return Value

If Pin is a source pin, KsPinGetConnectedPinFileObject returns a pointer to the FILE_OBJECT for the pin to which Pin is connected. If Pin is not a source pin, it returns NULL.

Headers

Declared in ks.h. Include ks.h.

Comments

This function is available in Windows XP and DirectX 8.0 and later.

KsPinGetConnectedPinFileObject must be called at IRQL = PASSIVE_LEVEL.
