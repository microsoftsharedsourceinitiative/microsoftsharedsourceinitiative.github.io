Beginning the Page

The PC_OCD_BEGIN_PAGE box contains the printer command or commands that the Universal Printer Driver issues prior to printing each page of a document. For some printers, the begin-page command may be as simple as a single carriage return; for other printers, it may be necessary to send other commands as well.

Diconix 150 Plus

For the Diconix 150 Plus, the Carriage Return command (0x0d) is required to return the cursor (or print head) to the left edge of the page.

To specify the Carriage Return command, type the following in the PC_OCD_BEGIN_PAGE box:

\x0D

HP LaserJet IIP

For the HP LaserJet IIP, no command is required at the beginning of a new page.
