EXE2BIN

Strips the .EXE header from an executable file. You can use EXE2BIN to create .COM files.

Syntax

exe2bin filename

Parameters

filename Specify the name of the file to strip.

Restrictions

  • The executable file must not contain a stack segment.
  • The combined size of code and data must not exceed 64K
