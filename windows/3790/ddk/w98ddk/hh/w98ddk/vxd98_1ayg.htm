HeapAllocateEx

HeapAllocateEx allocates a block of RAM from a heap. 

PVOID HeapAllocateEx(
  ULONG   cBytes,
  PVOID   CallBack,
  PVOID   Reference,
  ULONG   flags
 );

Parameters

cBytes
Number of bytes required
CallBack
To be provided.
Reference
To be provided.
flags
One or more of HEAPxxx as defined in VMM.H.

Return Values

The return value is a pointer to allocated memory, or is NULL if no memory is allocated.
