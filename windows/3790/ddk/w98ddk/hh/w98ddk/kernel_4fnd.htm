VWIN32_GetCurrentProcessHandle

Retrieves the current ring 3 process handle.

  • Returns the current process handle.

VxDs that support device input and output control (IOCTL)services use this service to tag the current device IOCTL connection or request with an ID. By using a process ID, the device can cleanup more easily on process termination.
