CamProcessRawVideoFrame

CamProcessRawVideoFrame, called at the PASSIVE level, allows the minidriver to perform any decoding of the raw video frame. 

NTSTATUS
  CamProcessRawVideoFrame(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext,
    PVOID  FrameContext,
    PVOID  FrameBuffer,
    ULONG  FrameLength,
    PVOID  RawFrameBuffer,
    ULONG  RawFrameLength,
    ULONG  NumberOfPackets,
    PULONG  BytesReturned
    );

Parameters

BusDeviceObject
Device object created by the hub.
DeviceContext
Minidriver context.
FrameContext
Minidriver context for this frame.
FrameBuffer
Pointer to the buffer that should receive the final processed video frame.
FrameLength
Length of the frame buffer (from the original read request).
RawFrameBuffer
Pointer to buffer containing the received USB packets.
RawFrameLength
Length of RawFrameBuffer.
NumberOfPackets
Number of USB packets received into RawFrameBuffer.
BytesReturned
Pointer to the number of bytes read.

Return Values

The return value is the NTSTATUS completion code for the read IRP.
