ILB_service_rtn

cCall [ilb.ILB_service_rtn],<<OFFSET32 ISP>>

Carries out the task specified by the ISP structure. A device driver or client uses this routine to carry out tasks such as allocating memory for an IOP structure.

  • No return value.
ISP
Address of the ISP structure that contains information about the task to carry out. For more information about the possible tasks, see the ISP structure.
