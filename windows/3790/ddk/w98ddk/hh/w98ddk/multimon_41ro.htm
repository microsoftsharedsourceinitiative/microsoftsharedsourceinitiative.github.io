MonitorFromPoint

HMONITOR
  MonitorFromPoint(
    POINT  pt, 
    DWORD  dwFlags
    );

Parameters

pt
To be provided.
dwFlags
To be provided.

Return Values

Returns the monitor that contains pt. If no monitor contains pt, the return value depends upon the dwFlags field:

Value Description
MONITOR_DEFAULTTONULL Returns NULL.
MONITOR_DEFAULTTOPRIMARY Returns the HMONITOR of the primary monitor.
MONITOR_DEFAULTTONEAREST Returns the HMONITOR nearest to the point.
