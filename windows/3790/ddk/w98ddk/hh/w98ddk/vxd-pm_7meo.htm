CONFIG_SET_POWERSTATE_D0

The configuration manager sends the CONFIG_SET_POWERSTATE_D0 message to a device driver to tell it to switch to the D0 (fully on) power state. By the time the device driver receives this message, its parent drivers have already been sent the message and signaled the physical hardware to turn on.

When the device driver receives this message, it must restore any hardware context needed to completely restore the device to D0. If any proprietary commands are required to turn on the device and they are not handled by bus standards, the device driver can send them at this time.
