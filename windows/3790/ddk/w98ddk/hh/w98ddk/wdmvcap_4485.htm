KSPROPERTY_TUNER_MODE

The KSPROPERTY_TUNER_MODE read/write property set can be used to set the mode of the tuner. For example, if a tuner is capable of supporting both FM and TV tuning, this property is used to select and report the current mode of operation.

typedef struct {
  KSPROPERTY  Property;
  ULONG  Mode;
} KSPROPERTY_TUNER_MODE_S, *PKSPROPERTY_TUNER_MODE_S;

Members

Property
Specifies the property used.
Mode
Specifies the mode using values from KSPROPERTY_TUNER_MODES.
