VKD_API_Get_Version

include vkd.inc

VxDCall VKD_API_Get_Version
mov     byte ptr [MajorVersion], ah
mov     byte ptr [MinorVersion], al

Retrieves the version number of the VKD. Uses EAX and Flags; clears the carry flag.

  • Returns the major version number in AH and the minor version number in AL.
