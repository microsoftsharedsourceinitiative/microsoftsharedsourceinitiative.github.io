Character Set

The Char Set control contains a value that identifies the character set associated with a font. This value is 0 if the font uses the Windows ANSI character set; this value is 1 if the font is an IBM PC hardware font (code page 437); this value is 2 if the font uses a special symbol set. Whenever possible, a minidriver should support the Windows ANSI character set.

Diconix 150 Plus

For the Diconix 150 Plus printer, this value is 0 because the Diconix device fonts use the Windows ANSI character set.
