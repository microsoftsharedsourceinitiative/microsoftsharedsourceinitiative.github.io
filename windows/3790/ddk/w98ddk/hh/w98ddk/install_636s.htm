Windows 98/Me: Windows DDK

Getting Started on Windows 98/Me

The Microsoft® Windows® 98/Me section of the Windows DDK documentation contains information about developing device drivers for Windows® 95/98/Me. For some types of devices, additional Windows 98/Me information can be found in the appropriate device-type-specific topics within the Windows DDK documentation.

Note that some of the information in the Windows 98/Me sections may be outdated or made obsolete by the newer material that is contained in other sections of the documentation.

This section makes occasional references to the file paths and file names that are unique to the Windows 95 DDK CD-ROM, Windows 98 DDK CD-ROM and the Windows Me CD-ROM. If those files or folders are needed, refer to the original CD-ROM.

The build environments for Windows 95, Windows 98 and Windows Me are no longer supported, and are not available on the Windows DDK CD-ROM. If you need the build environment and its related tools, refer to the Windows XP, or previous, DDK CD-ROM.

Information on the Microsoft Web Site

The Windows Platform Development Web site provides the following information on hardware and driver development:

  • Supplemental driver development information.
  • Third-party driver development and training resources.
  • Advice from "Dr. Iver" on special driver topics, such as WDM, Plug and Play, and power management.
  • Specifications for power management, Plug and Play, and other special topics.

The MSDN (Microsoft Developers Network) Web site provides the following downloads and information:

  • Service packs or patches for Microsoft development tools.
  • Special developer tools for MSDN Online Members to use and evaluate. These tools are designed to complement and integrate with the Microsoft programming environments you already use.
  • Up-to-date releases of the Platform SDK.
  • Access to user groups, chats, event information, the MSDN Library Online, and a host of products and technologies for developers.

The Microsoft DirectX Web site provides information for DirectX hardware and software developers, including:

  • Microsoft DirectX SDK download.
  • Connections to technical articles, support information, and news groups.

The Windows DDK Web site provides information and resources for driver development, including:

  • Current versions of various DDKs.
  • Online help for DDKs.
  • Announcements of new DDK developments.
  • Feedback for reporting bugs, suggesting content, and rating the DDK.

The Windows DDK Support Web site provides information on developing and maintaining drivers, including:

  • Additional driver samples.
  • Tools for device drivers and debugging.
  • FAQs on Windows driver development.
