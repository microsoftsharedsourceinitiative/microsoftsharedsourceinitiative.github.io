HeapFreeEx

HeapFreeEx frees a block of RAM to a heap.

VOID HeapFreeEx(
  PVOID   mem,
  PVOID   Reserved1
 );

Parameters

mem
Pointer to a piece of memory previously returned by HeapAllocateEx or HeapReAllocateEx.
Reserved1
Reserved. Must be NULL.

Return Values

No value is returned.
