End Graphics

The End Graphics box contains the command which is sent to the printer to disable graphics output. (Those printers which provide a specific command to disable a graphics resolution may also provide a command to end the processing of graphics data.)

Diconix 150 Plus

The Diconix 150 Plus printer automatically ends graphics output after it processes the last byte of graphics data; therefore, this control is not initialized.

HP LaserJet IIP

For the HP LaserJet IIP printer, the End Raster Graphics command (ESC*rB) disables raster graphics output. To specify the End Raster Graphics command, type the following in the End Graphics box:

\x1B*rB
