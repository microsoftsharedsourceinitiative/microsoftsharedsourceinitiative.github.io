Using the IShellPropSheetExt::AddPage Function

You can use the AddPage member function of the IShellPropSheetExt interface to add property-sheet pages to Control Panels. The system calls AddPages when it is about to display a property sheet. The system calls each property-sheet handler registered to the file type to allow the handlers to add pages to the property sheets. This technique is explained in the Shell Extensions section of the Windows 95 Guide to Programming.
