MonitorFromWindow

HMONITOR
  WINUSERAPI MonitorFromWindow(
    HWND  hwnd, 
    DWORD  dwFlags
    );

Parameters

hwnd
To be provided.
dwFlags
To be provided.

Return Values

Returns the monitor to which a window belongs. If a window does not belong to any monitor, the return value depends upon the dwFlags field as in MonitorFromPoint.
