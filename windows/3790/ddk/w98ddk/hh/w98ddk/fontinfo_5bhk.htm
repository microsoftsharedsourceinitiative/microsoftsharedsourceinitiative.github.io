Pixel Width

The PixWidth control contains a value that specifies the character width for fixed-pitch raster fonts. This value is specified using the horizontal-resolution units. (If the font is a variable-width device font, this value is 0.)

Diconix 150 Plus

For the Diconix 150 Plus printer, this value is 57.
