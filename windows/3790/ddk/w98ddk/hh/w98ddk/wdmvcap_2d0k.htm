KSPROPERTY_VIDEODECODER_STANDARD

The KSPROPERTY_VIDEODECODER_STANDARD read/write property is used to set and retrieve the current analog video standard. The following structure is used to indicate the standard:

typedef struct {
  KSPROPERTY  Property;
  ULONG  Value;
} KSPROPERTY_VIDEODECODER_S, *PKSPROPERTY_VIDEODECODER_S;

Members

Property
Specifies the property used.
Value
Specifies the value to be set or retrieved.

The KSPROPERTY_VIDEODECODER_S.Value member specifies the current analog video standard using one of the values in KS_AnalogVideoStandard.
