Channel Settings Dialog

The Channel Settings dialog allows the user to assign a phone number and SPID to the selected channel.

Group box.
The group box label indicates which channel is being set.
Switch Type.
This is a static text field that reflects the current switch type of the physical line. This field serves to remind the user, in case she questions the derived SPID number.
Phone Number.
The edit box accepts standard keystrokes acceptable for a phone number in canonical address form. If the user enters a noncanonical phone number, the dialog will convert it once OK is clicked.
SPID.
This edit box accepts numerical data which should contain the Service Profile ID provided by the telephone company.
Determine SPID automatically.
When marked, the SPID is generated according to the phone number (not currently implemented). Otherwise, the SPID field is enabled to allow the user to manually enter the SPID.
