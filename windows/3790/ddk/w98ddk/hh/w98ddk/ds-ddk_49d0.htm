How to Instantiate Drivers

This section describes how to instantiate drivers, and what drivers must do to register themselves with DirectSound. This section assumes knowledge of the Windows 95 audio device driver interface, Plug and Play, and so on. This information can be obtained from the Windows 95 Device Driver Kit (DDK). It is strongly recommended that a functional Windows 95 driver be written for the sound device before DirectSound driver support is added. The DirectSound portion of the driver is a small addition to (and relies on several components of) the Windows 95 driver.

The following topics are discussed in this section:

  • The Configuration Handler
  • DirectSound VxD Exports
  • Registering a Driver
  • Deregistering a Driver
