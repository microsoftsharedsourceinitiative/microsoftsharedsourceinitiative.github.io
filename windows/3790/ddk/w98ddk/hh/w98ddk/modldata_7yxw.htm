Default Font ID

The Default Font ID box contains a value which identifies the default font that the Universal Printer Driver should use for text output when an application fails to request a specific font. It is suggested that the default font be set to a fixed pitch font: 10 point and 10 characters per inch, if at all possible.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Default Font ID box contains the value 1.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Default Font ID box contains the value 225.
