NdiGetNext

HNDI NdiGetNext(hndi)
HNDI hndi;    // NDI handle

The NdiGetNext function returns the next NDI object list after the given NDI object.

hndi
Identifies the original NDI object.
  • The return value is the NDI object handle if the function is successful. Otherwise it is NULL.
