IMN_SETCOMPOSITIONWINDOW

The IMN_SETCOMPOSITIONWINDOW message is sent when the composition form of the input context is updated. When the UI window receives this message, the new conversion mode can be retrieved from cfCompForm. The composition component of the UI windows uses cfCompForm to show the composition window.

There is no return value.
