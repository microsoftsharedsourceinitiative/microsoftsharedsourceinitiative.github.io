DiInstallClass

Install the [ClassInstall] section of the specified INF. 

RETERR
  DiInstallClass(
    LPCSTR  lpszInfFileName,
    DWORD  dwFlags
    );

Return Value

Returns OK if successful, otherwise an ERR_DI_xxx error code will be returned.

Parameters

lpszInfFileName
The name of the INF file containing a [ClassInstall] section.
dwFlags
Flags to control the geninstall operation.
DI_NOVCP This flag should be specified if a VCP queue is already opened.
DI_NOBROWSE This flag should be specified if no file browsing should be allowed in the event a copy operation cannot find a specified file.

Comments

This API is generally called the Device Manager when it installs a device of a new class.
