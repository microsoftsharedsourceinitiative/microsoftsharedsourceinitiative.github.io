Supported Attributes in the ModelData Dialog Box

The Supported Attributes list box lists the names of fourteen dialog boxes. The values and strings that appear in these dialog boxes correspond to values and strings which were entered in the corresponding dialog boxes accessible through the cascading menus in the Printer Data menu. By double-clicking any of the entries in the Supported Attributes list box, you display a dialog box with data previously entered through the commands and corresponding dialog boxes in the Printer Data menu. Here, you may indicate which data applies to your device and which does not; you may also specify values to which the driver will default when it is first installed. The following list briefly describes each of the fourteen dialog boxes accessible from the Supported Attributes list box.

Dialog Box Description
Resolutions Specifies the default, supported, and unsupported printer resolutions. The values which appear in this dialog correspond to values which were entered in the RESOLUTION dialog box.
Paper Sizes Specifies the default, supported, and unsupported paper sizes. The values which appear in this dialog box correspond to values which were entered in the Paper Sizes dialog box.
Paper Qualities Specifies the default, supported, and unsupported paper qualities. The values which appear in this dialog box correspond to values which were entered in the Paper Qualities dialog box.
Paper Sources Specifies the default, supported, and unsupported input paper trays. The values which appear in this dialog box correspond to values which were entered in the Paper Sources dialog box.
Paper Destinations Specifies the default, supported, and unsupported output paper trays. The values which appear in this dialog box correspond to values which were entered in the Paper Destination dialog box.
Text Qualities Specifies the default, supported, and unsupported text qualities. The values which appear in this dialog box correspond to values which were entered in the Text Qualities dialog box.
Compression Modes Specifies the default, supported, and unsupported compression modes. The values which appear in this dialog box correspond to values which appear in the Compression Modes dialog box.
Font Cartridges Specifies the default, supported, and unsupported font cartridges. The values which appear in this dialog box correspond to values which appear in the Font Cartridge dialog box.
Color Models Specifies the default, supported, and unsupported color models. The values which appear in this dialog box correspond to values which were entered in the DEVCOLOR dialog box.
Page Control Specifies the supported page-control command set.
Cursor Move Specifies the supported cursor-movement data.
Font Simulation Specifies the supported font simulation.
RectFill Specifies the supported rectangle fills.
Download Info Specifies the supported download information for printer fonts.
