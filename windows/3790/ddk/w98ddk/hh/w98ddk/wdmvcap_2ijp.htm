KSPROPERTY_CROSSBAR_ROUTE

The KSPROPERTY_CROSSBAR_ROUTE write-only property is used to route a video or audio stream. The property uses the following structure:

typedef struct {
  KSPROPERTY  Property;
  ULONG  IndexInputPin;
  ULONG  IndexOutputPin;
  ULONG  CanRoute;
} KSPROPERTY_CROSSBAR_ROUTE_S, *PKSPROPERTY_CROSSBAR_ROUTE_S;

Members

Property
Specifies the property used.
IndexInputPin
Specifies the zero-based index of the input pin.
IndexOutputPin
Specifies the zero-based index of the output pin.
CanRoute
Returns nonzero on CanRoute if routing is possible.
