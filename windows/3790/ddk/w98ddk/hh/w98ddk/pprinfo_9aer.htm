5.4 Specifying Paper Destinations

After you've initialized the controls in one or more Paper Source dialog boxes for your printer, you should specify paper-destination data and commands in the Paper Destinations dialog box (if your printer supports the ability to control sending output to different destinations).

Neither the Diconix 150 Plus or the HP LaserJet IIP printer supports a paper-destination command (or command set); therefore, the controls in the respective dialog box for each printer's minidriver are empty.
