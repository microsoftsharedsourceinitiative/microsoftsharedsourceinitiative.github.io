IMC_SETSTATUSWINDOWPOS

An application sends the IMC_SETSTATUSWINDOWPOS message to the IME window to set the position of the status window.

If successful, the return value is zero. Otherwise, it is nonzero.

lParam
A POINTS structure that contains the x and y coordinates of the position of the status window. This is in screen coordinates.
