Vertical Text Offset

The sTextYOffset box contains a value that specifies the difference between the baseline of printer fonts and the baseline of GDI fonts. If the baseline of device fonts differs from the baseline of GDI fonts and an application mixes the two fonts on a line, the text will appear uneven. In order to prevent this from occurring, the Universal Printer Driver requires a value that specifies the difference between the two baselines. This value is specified in master units.

Diconix 150 Plus

For the Diconix 150 Plus printer, the baseline of printer fonts is identical to the baseline of GDI fonts; therefore, the sTextYOffset box contains the value zero.

HP LaserJet IIP

For the HP LaserJet IIP printer, the baseline of printer fonts is identical to the baseline of GDI fonts; therefore, the sTextYOffset box contains the value zero.
