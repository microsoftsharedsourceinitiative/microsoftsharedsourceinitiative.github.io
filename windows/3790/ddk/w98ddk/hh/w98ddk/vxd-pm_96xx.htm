Get_DevNode_PowerState

Device drivers call the Get_DevNode_PowerState service to get the current power state of a device.

CONFIGRET
  CONFIGMG_Get_DevNode_PowerState(
    DEVNODE  dnDevNode, 
    PFARULONG  pulPowerState, 
    ULONG  ulFlags
    );

Parameters

dnDevNode
Handle to the device node from which the configuration manager returns the current power state.
pulPowerState
Pointer to a ULONG that receives the device's power state. The result is one of the following:

CM_POWERSTATE_D0
CM_POWERSTATE_D1
CM_POWERSTATE_D2
CM_POWERSTATE_D3

ulFlags
Reserved. Currently must be zero.
