SCAN

typedef struct {
    short scnPntCnt;    // count of x coordinates (not x pairs)
    short scnPntTop;    // top line in a group of scanlines (inclusive)
    short scnPntBottom; // bottom line (exclusive)
    short scnPntX[2];   // variable-length array of x pairs 
                        // (inclusive-exclusive)
    short scnPntCntToo; // same as scnPntCnt
} SCAN;

Contains the drawing information for a group of scanlines. This is not a true structure, because the scnPntX member is a variable-length array.
