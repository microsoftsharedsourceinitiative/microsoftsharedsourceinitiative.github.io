Font ID Value Limits

The HP PCL requires that a special identifier be assigned to a soft font prior to downloading. The Min and Max ID boxes specify the minimum and maximum allowable font ID value.

Note  The ID values listed will only apply to downloaded TrueType fonts. Other downloaded fonts installed with the font installer use ID values supplied to the font installer.

HP LaserJet IIP

The Minimum ID box specifies the minimum allowable font ID value which is 1; the Maximum ID box specifies the maximum allowable font ID value.
