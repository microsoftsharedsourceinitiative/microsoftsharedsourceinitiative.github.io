Handling System Wake Events

When bus drivers receive a CONFIG_RESUME_SUSPEND message, they must put their buses into the lowest power state needed to detect insertions and removals. Bus drivers must not do any re-enumeration at this time. If re-enumeration is needed, the configuration manager sends a CONFIG_ENUMERATE message.
