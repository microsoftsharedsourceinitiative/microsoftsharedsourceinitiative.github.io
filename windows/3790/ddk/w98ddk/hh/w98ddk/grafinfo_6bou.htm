Select Resolution

The Select Resolution box contains the command which selects the designated printer resolution.

Diconix 150 Plus

The Diconix 150 Plus printer integrates the selection of the resolution with the Send Block command; therefore this box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Raster Graphics Resolution command (ESC*tnR) selects the resolution for graphics output where n specifies a resolution of 75, 100, 150, or 300 dpi. To specify the Select Resolution command for 300 dpi graphics, type the following in the Select Resolution box:

\x1B*t300R
