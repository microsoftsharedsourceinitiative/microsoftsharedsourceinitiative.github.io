GET_NUM_UNITS

GET_NUM_UNITS gets the number of "display units" that one device (or chip set) can support as separate display surfaces. Implement this function only if your device implements two or more units.

Entry Meaning
ESP+4 Device node.

Exit Meaning
EAX Number of display units.
