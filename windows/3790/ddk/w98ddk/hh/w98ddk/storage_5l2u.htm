Design Requirements

The design must not break Win 3.x redirectors (typically a shell that hooks INT 21h and INT 2Fh calls).

All existing INT 21h file I/O functions must be supported.

The Win32 file I/O API functions must be supported.

The components necessary to implement the design must use as little low, real mode memory (memory below 1 MB) as possible.
