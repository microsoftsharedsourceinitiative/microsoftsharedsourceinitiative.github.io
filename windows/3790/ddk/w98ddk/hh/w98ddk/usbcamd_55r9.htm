CamRestoreState

CamRestoreState recalls a previously saved state and makes it current.

NTSTATUS
  CamRestoreState(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext
    );

Parameters

BusDeviceObject
Device object created by the hub. May be NULL.
DeviceContext
Peripheral’s device context.

Return Values

Returns STATUS_SUCCESS.
