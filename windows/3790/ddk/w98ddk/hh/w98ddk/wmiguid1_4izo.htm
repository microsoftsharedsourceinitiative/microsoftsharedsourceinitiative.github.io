IRP_MN_EXECUTE_METHOD

The driver receives an IRP_MN_EXECUTE_METHOD to execute a method associated with a data block.

Members

ProviderId
Contains the WMI provider ID of the device that should respond to this request.
DataPath
Points to a buffer that specifies the GUID that identifies the method to be executed.
BufferSize
Specifies the maximum size available to fill the WNODE_METHOD_ITEM that is returned from the method invocation.
Buffer
Points to the WNODE_METHOD_ITEM that on entry describes the method called and contains the input data block. On exit, it is filled with the returned data block. If the WnodeMethodItem->DataBlockOffset is zero on entry, then no data is passed as input to the method. If it is set to zero on return, then no data is returned from the method call.

Return Value

Returns STATUS_SUCCESS, STATUS_WMI_INSTANCE_NOT_FOUND, or STATUS_WMI_GUID_NOT_FOUND.
