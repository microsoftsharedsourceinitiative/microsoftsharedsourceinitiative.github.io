KSPROPERTY_VIDEOCOMPRESSION_GETINFO

The KSPROPERTY_VIDEOCOMPRESSION_GETINFO read-only property gets information on video compression supported by a device. It uses the following structure:

typedef struct {
  KSPROPERTY  Property;
  LONG  DefaultKeyFrameRate;
  LONG  DefaultPFrameRate;
  LONG  DefaultQuality;
  LONG  Cpabilities;
} KSPROPERTY_VIDEOCOMPRESSION_GETINFO_S,
*PKSPROPERTY_VIDEOCOMPRESSION_GETINFO_S;

Members

Property
Specifies the property used.
DefaultKeyFrameRate
Specifies the estimated frames per key frame.
DefaultPFrameRate
Specifies the zero-based index of the output pin.
DefaultQuality
Specifies the default quality value, ranging from zero to 10000.
Capabilities
Specifies the compression capabilities, from the KS_CompressionCaps.

The KSPROPERTY_VIDEOCOMPRESSION_GETINFO_S.Capabilities can be one of the following:

typedef enum {
    KS_CompressionCaps_CanQuality = 1,
    KS_CompressionCaps_CanCrunch = 2,
    KS_CompressionCaps_CanKeyFrame = 4,
    KS_CompressionCaps_CanBFrame = 8
} KS_CompressionCaps;
