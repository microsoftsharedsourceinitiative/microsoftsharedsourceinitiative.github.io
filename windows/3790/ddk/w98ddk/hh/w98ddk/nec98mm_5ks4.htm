Operating Environment

Hardware candidates for the new multimonitor feature include the PC-9801/9821 series models that are equipped with PCI expansion slots. In addition, the models must be capable of running Windows 98. To use the multimonitor feature, several display adapters must be available for multiple monitors.

The mulitmonitor feature operates on Windows 98 and requires a display driver that supports the multimonitor environment.
