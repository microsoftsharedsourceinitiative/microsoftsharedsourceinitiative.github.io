Specifying Whether the Quality is Predefined or Driver Defined

The Predefined and Driver Defined buttons indicate whether a Text Qualities dialog box contains data for one of the five predefined qualities or whether the dialog box contains data for a driver-defined text quality. These buttons are mutually exclusive — only one can be selected.

Diconix 150 Plus

For the Diconix 150 Plus printer, the predefined button is selected.

HP LaserJet IIP

For the HP LaserJet IIP printer, the predefined button is selected by default. (All other controls in this dialog box are empty.)
