KSPROPERTY_VIDEOPROCAMP_GAMMA

The KSPROPERTY_VIDEOPROCAMP_GAMMA read/write property is used to set and retrieve the gamma setting.

The KSPROPERTY_VIDEOPROCAMP_S.Value member specifies the gamma value expressed gamma * 100. Value ranges from 1 to 500, and the default is 100 (gamma = 1). Note that a particular device may only implement a subset of this range.
