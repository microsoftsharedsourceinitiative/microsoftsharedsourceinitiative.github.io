Vertical Adjustments

The Y Moved and the Y Adjust controls contain vertical offsets that are required to align a font's baseline with the GDI baseline. If these baselines differ, Windows will shift the font baseline by the Y Adjust value prior to rendering text. Once the text is rendered, Windows will shift the font baseline back to the proper position using the Y Moved value. Typically, it is not necessary to adjust the Y position for device fonts.

Diconix 150 Plus

For the Diconix 150 Plus printer, these controls contain the value 0.
