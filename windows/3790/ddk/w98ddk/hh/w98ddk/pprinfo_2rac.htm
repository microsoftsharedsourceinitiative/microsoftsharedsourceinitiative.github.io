Specifying Whether the Paper Source Is Predefined or Driver-Defined

The Predefined and Driver Defined Source Name options specify whether the data and commands in the Paper Sources dialog box are predefined or driver defined. The Predefined and Driver Defined options also specify whether the description of a single paper quality presented to the user is selected from the predefined names supplied by the Universal Printer Driver or by the minidriver.

Diconix 150 Plus

For the Diconix 150 Plus, the Predefined button is selected.

HP LaserJet IIP

For the HP LaserJet IIP, the Predefined button is selected.
