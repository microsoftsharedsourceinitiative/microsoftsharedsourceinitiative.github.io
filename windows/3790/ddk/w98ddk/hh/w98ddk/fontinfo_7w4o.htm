Maximum Width

The Max. Width control contains a value that specifies the width of the widest character in a font. For fixed-pitch fonts, this value will always be equivalent to the value specified in the Average Width control. For proportionally spaced fonts, UniTool will calculate and insert this value in the control (using values from the character-width table).

Diconix 150 Plus

For the Diconix 150 Plus printer, this value is 57.
