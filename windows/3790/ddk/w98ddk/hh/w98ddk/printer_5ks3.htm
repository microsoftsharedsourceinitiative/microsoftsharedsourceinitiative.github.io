Overview of ICC 3.0 Profile Tags Used by Windows 95

For ICC version 3.0 color profiles to be used with Windows 95, a profile must support some tags and can, optionally, support some additional tags for greater flexibility. Information discussing how printer drivers map to various profiles is discussed in Mapping Color Profiles to Printer Drivers.
