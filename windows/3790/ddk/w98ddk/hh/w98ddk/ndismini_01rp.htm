Windows 95 NDIS WAN Miniports

The primary documentation for developing NDIS WAN miniport drivers is the Network Drivers section in the Windows NT® 3.51 DDK. The following information assumes that the reader is already familiar with the Windows NT® 3.51 DDK documentation and network driver development. This documentation provides additional details for developing NDIS 3.1 WAN miniport drivers for Windows® 95 as noted in the Windows NT 3.51 DDK documentation:

If your MAC driver will have both a Windows NT/Windows 2000 and Windows 95 version, it is best to develop and perfect the Windows NT/Windows 2000 version first. Code written for Windows NT/Windows 2000 easily translates to Windows 95, but the reverse is not necessarily true.
