MCI_INFO_PARMS

typedef struct {
    DWORD  dwCallback;
    LPSTR  lpstrReturn;
    DWORD  dwRetSize;
} MCI_INFO_PARMS;

The MCI_INFO_PARMS structure contains parameters for the MCI_INFO message. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
lpstrReturn
Specifies a long pointer to a user-supplied buffer for the return string.
dwRetSize
Specifies the size in bytes of the buffer for the return string.
