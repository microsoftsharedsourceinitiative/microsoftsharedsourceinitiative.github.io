TELETEXT Category

TELETEXT Category output pins provide a stream of decoded WST (World Standard Teletext) data.

When creating the category PinName, PINNAME_VIDEO_TELETEXT, the following parameters must be filled out.

Parameter Value
KS Datarange KS_DATARANGE_VIDEO_VBI
KS Dataformat KS_DATAFORMAT_VIDEO_VBI
KS MajorFormat KS_DATAFORMAT_TYPE_VBI
KS SubFormat (typical) KSDATAFORMAT_SUBTYPE_TELETEXT
KS Specifier KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
DS majortype MEDIATYPE_VBI
DS formattype FORMAT_VBI
