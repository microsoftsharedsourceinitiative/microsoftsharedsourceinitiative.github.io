IR Communications DDK Contents

The Infrared Communications for Windows 95 Device Driver Kit (DDK) includes the following:

  • Documentation
    • README.TXT, which gives a quick overview of the contents of the IR Communications DDK and how the DDK can be used.
    • PRODUCT.DOC, which describes Microsoft's current and future infrared communications products for Windows 9x.
    • DDKRELNO.DOC (this documentation), which contains instructions on how to use the IR Communications DDK to customize the IR communications device driver.
    • DRVRELNO.DOC, which contains instructions on how to install the IR communications device driver that is part of the DDK, and how to test the driver installation.
  • Software on diskette
    • A complete set of Windows 95 IR communications driver binary files. For a list of the infrared devices supported by these binary files, see Instant Support for Some IR Hardware in Individual VxDs.
    • Windows 95 IR communications Setup program and the device information files (INF files) used by the setup program.
    • A set of on-line Help files for the Infrared Monitor (IrMon) application that is bundled with the IR communications driver.
    • Sample source code for the OEM-provided part of the serial IrFramer. This is the source code for ACT200L.VXD, which is shipped with the DDK to support the OEM-provided part of IrFramer for ACTiSYS ACT-220L Infrared Wireless Interface infrared dongles.

The Microsoft Windows 95 DDK provides header files for the creation of IR Communications for Windows 95. The Windows 95 DDK is available from MSDN.
