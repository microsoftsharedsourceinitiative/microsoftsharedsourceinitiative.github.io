Data Structures

A common data structure is needed to transfer device configuration between the WANTSP and the device properties. The data structure must be extensible for future devices and capabilities. For an ISDN device, the data structure must also support multiple lines and channels.

The following ISDN structures are provided:

ISDNDEVCAPS

ISDNLINECAPS
