IMC_SETSOFKBDDATA

An IME sends the IMC_SETSOFKBDDATA message to a the soft keyboard window to specify the character code to be used for displaying reading characters of key in the soft keyboard window. The UI window does not receive this message.

If successful, the return value is zero. Otherwise, it is nonzero.

lParam
Address of a SOFTKBDDATA structure.
