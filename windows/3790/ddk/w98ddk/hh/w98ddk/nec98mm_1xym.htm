VDD_OPEN

The VDD_OPEN function opens a device, given a name.

Parameters

ES:EDI: device name

EDX: flags

VDD_OPEN_EXIST
Checks whether the device name is valid.
VDD_OPEN_ENUM
Returns the Nth device.
VDD_OPEN_LOCK
Locks the device (default).

EBX: device ID

Return Value

EAX: device handle
