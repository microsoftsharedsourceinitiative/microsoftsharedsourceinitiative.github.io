Clear_Mono_Screen

include vmm.inc

VMMCall Clear_Mono_Screen

Clears the secondary display screen by filling it with spaces, and setting character attributes to normal. Uses Flags.

No return value.

This service has no effect in the retail version of Windows. It is intended to be used with the debugging version.
