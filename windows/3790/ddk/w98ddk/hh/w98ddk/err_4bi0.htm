The MiniDriver Data Dialog Box

The following error messages are associated with the MiniDriver Data dialog box.

List Description
Illegal GPC Major Version UniTool issues this message if the value Number, must be 0<=n<=1 specified in the Major box is not 0 or 1.
Illegal GPC Minor Version UniTool issues this message if the value Number, must be 0<=n<=255 specified in the Minor box is outside of the specified range (0 - 255).
Structure-Name Conversion error UniTool issues this message whenever the new master units result in a round-off error after they are divided into a specified member in the MODELDATA, RESOLUTION, PAPER-SIZE, PAPER-SOURCE, CURSOR-MOVEMENT, or RECTANGLE-FILL data structures. For example, if the sLeftMargin box of the ModelData dialog box contains the value 179, and the horizontal master unit was changed from 360 to 180, the conversion error message would appear because 89.5/180 cannot be expressed as an integer value. UniTool also replaces the data structure identified by Structure-Name with the appropriate string.
