Device_Wakeup

Device drivers call the Device_Wakeup service for each child device that is signaling wake up. The configuration manager sends a CONFIG_APM_WAKEUP message to the child device driver to inform the driver that the device needs to handle the wake up event. 

CONFIGRET
  CONFIGMG_Device_Wakeup(
    DEVNODE  dnDevNode, 
    ULONG  ulFlags
    );

Parameters

dnDevNode
Handle to the device node that identifies the driver to which the configuration manager sends the CONFIG_APM_WAKEUP message.
ulFlags
Reserved. Currently must be zero.
