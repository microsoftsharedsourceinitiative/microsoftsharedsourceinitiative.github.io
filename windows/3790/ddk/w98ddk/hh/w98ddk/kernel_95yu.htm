Get_Cur_Thread_Handle

include vmm.inc

VMMCall Get_Cur_Thread_Handle
mov     [hThread], edi

Returns a handle identifying the currently executing thread. Uses EDI and Flags.

  • Returns a handle in EDI that identifies the currently executing thread.

This service can be called at interrupt time.
