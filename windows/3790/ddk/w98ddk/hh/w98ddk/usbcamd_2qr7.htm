Minidriver Entry Points

The following is a list of entry points provided by the cam minidriver to support the ISO stream and configuration processes handled by USBCAMD. All the calls are defined as pointers to functions and appear as members in the structure USBCAMD_DEVICE_DATA. This structure must be initialized prior to use.

CamInitialize

CamUnInitialize

CamProcessUSBPacket

CamNewVideoFrame

CamProcessRawVideoFrame

CamStartCapture

CamStopCapture

CamConfigure

CamSaveState

CamRestoreState

CamAllocateBandwidth

CamFreeBandwidth
