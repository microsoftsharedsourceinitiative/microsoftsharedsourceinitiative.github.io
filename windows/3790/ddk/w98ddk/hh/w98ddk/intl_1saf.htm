IMN_SETCONVERSIONMODE

The IMN_SETCONVERSIONMODE message is sent when the conversion mode of the input context is updated. An application or the UI window uses this message to retrieve information about the status window by calling the ImmGetConversionStatus function. The UI window redraws its status window, if the status window shows the conversion mode.

There is no return value.
