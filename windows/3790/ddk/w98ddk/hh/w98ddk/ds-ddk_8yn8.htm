DSVOLUMEPAN

Describes the volume and pan settings in their original (hundredths of dB) API form, in fixed-point linear form, and in combined (that is, applying both volume and pan) fixed-point linear values. 

typedef struct tDSVOLUMEPAN{
    DWORD    dwTotalLeftAmpFactor;
    DWORD    dwTotalRightAmpFactor;
    LONG     lVolume;
    DWORD    dwVolAmpFactor;
    LONG     lPan;
    DWORD    dwPanLeftAmpFactor;
    DWORD    dwPanRightAmpFactor;
} DSVOLUMEPAN, *PDSVOLUMEPAN;

Members

dwTotalLeftAmpFactor and dwTotalRightAmpFactor
Specifies the total combined linear factors (since volume and pan are cumulative).
lVolume
Specifies the input volume in hundredths of dB.
dwVolAmpFactor
Provides the linear equivalents in a fixed point 16.16 representation (for volume).
lPan
Specifies the input pan in hundredths of dB.
dwPanLeftAmpFactor and dwPanRightAmpFactor
Provides the linear equivalents in a fixed point 16.16 representation (for pan).
