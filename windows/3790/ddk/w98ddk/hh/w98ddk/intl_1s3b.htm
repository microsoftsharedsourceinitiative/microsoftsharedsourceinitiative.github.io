ImmUnlockIMCC

BOOL WINAPI ImmUnlockIMCC(HIMCC hIMCC)

Decrements the lock count for IMCC.

The return value is FALSE if the lock count of IMCC was decremented to zero. Otherwise, it is TRUE.

hIMCC
Handle of IMC component.
