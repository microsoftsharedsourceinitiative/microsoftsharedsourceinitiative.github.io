Secondary Display Adapter

Accelerators other than the primary will be the secondary. However, they may not be used for the desktop due to the following limitations:

  • Limitations due to shared resources

    Accelerators that share resources, such as VRAM, with the core graph cannot be used at the same time as the core graph. This means the accelerator cannot be used as a secondary display adapter.

  • Limitations due to a common CRT output

    Accelerators that share the CRT output (CRT connector) with the core graph cannot be used as a secondary display adapter because they cannot be used at the same time as the core graph.

    (If an external accelerator is set as the primary display adapter, the internal accelerator cannot be used because the core graph output needs to be allocated to the external accelerator.)

  • Accelerators that use Vflatd.vxd

    When accessing VRAM, accelerators that use vflatd.vxd to switch banks cannot be used as secondary, according to the accelerator CHIP specification.

For secondary display adapters that cannot be used as a desktop due to the reasons previously listed, a “!” is displayed in the device manager.

Core graph with the primary as the accelerator does not display “!” or “x” in the device manager.
