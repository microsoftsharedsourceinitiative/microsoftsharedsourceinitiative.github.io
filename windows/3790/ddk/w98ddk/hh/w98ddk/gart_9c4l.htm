PCIMP_GARTFree

The PCIMP_GARTFree service frees entries in the GART table.

VOID
  PCIMP_GARTFree(
    PVOID  pHandle, 
    PVOID  pRefData, 
    ULONG  ulNumPages, 
    PVOID  pMapHandle
    );

Parameters

pHandle
PCI device handle. This should be the same value used in the preceding PCIMP_GARTInit call.
pRefData
ULONG that was written by the PCIMP_GARTInit service.
ulNumPages
Specifies the number of pages. This should be the same value used in the PCIMP_GARTReserve service.
pMapHandle
Map handle obtained using the preceding PCIMP_GARTReserve service.

Return Values

This service does not have a return value.

Comments

PCIMP_GARTFree undoes the work of PCIMP_GARTReserve by filling the previously reserved ulNumPages identified by pMapHandle with the value specified in the ulFreeValue parameter in a previous PCIMP_GARTReset call.

Note that since this function does not return a value, there is no way to notify the system of an error, such as the ulNumPages parameter not matching the number of pages originally specified in the PCIMP_GARTReserve call.

Entries that have been freed must be reserved again before they can be remapped.
