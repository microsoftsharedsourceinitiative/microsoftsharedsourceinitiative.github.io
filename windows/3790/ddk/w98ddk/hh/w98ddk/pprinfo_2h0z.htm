Specifying Driver-Defined Paper Sizes

The Driver Defined control identifies a driver-defined paper size. If the Driver Defined control is set, the appropriate string describing the name of the paper size should be entered in this box.

Diconix 150 Plus

The example for the Diconix 150 Plus printer is based on a predefined paper size; therefore, this box is empty.

HP LaserJet IIP

The example for the HP LaserJet IIP printer is based on a predefined paper size; therefore, this box is empty.
