The Character Translation Table Dialog Box

The following error messages are associated with the Character Translation Table (CTT) dialog box.

List Description
Must be 0 < n < 255 UniTool issues this message if the specified character code is not an integer value between 1 and 255.
Can't have LastChar < FirstChar UniTool issues this message if the specified character code for the first character is greater than the specified character code for the last character.
