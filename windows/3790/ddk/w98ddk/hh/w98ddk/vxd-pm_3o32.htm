Register_DevNode_For_Idle_Detection

Device drivers can register to have the configuration manager automatically send the driver a CONFIG_SET_POWERSTATE_Dx message after some period of inactivity on a child device. Device drivers indicate activity by periodically clearing the counter pointed to by pulCounterVariable. 

CONFIGRET
  CONFIGMG_Register_DevNode_For_Idle_Detection(
    DEVNODE  dnDevNode, 
    ULONG  ulConservationTime, 
    ULONG  ulPerformanceTime, 
    PFARULONG  pulCounterVariable, 
    ULONG  ulState, 
    ULONG  ulFlags
    );

Parameters

dnDevNode
Handle to the device node that identifies the driver to which the configuration manager sets idle detection.
ulConservationTime
Time-out (in seconds) to be used when the system is optimizing for power consumption.
ulPerformanceTime
Time-out (in seconds) to be used when the system is optimizing for performance.
pulCounterVariable
Pointer to a ULONG to hold the timer counter.
ulState
Power state to go into when the time-out threshold is reached.
ulFlags
Reserved. Currently must be zero.
