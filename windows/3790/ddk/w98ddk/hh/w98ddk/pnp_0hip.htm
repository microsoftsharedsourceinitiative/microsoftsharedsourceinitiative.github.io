Driver

Driver=registry-path

Identifies the branch (under the SYSTEM\CurrentControlSet\Service\Class branch) that contains information about the device driver to load for the new device. The registry-path is a subkey that typically consists of a class name and a 4-digit, hexadecimal, device-instance number.

This value is used in HLM\ENUM\enumerator\device-id\instance and HLM\ENUM\ROOT\device-id\instance.
