CamAllocateBandwidth

CamAllocateBandwidth is called just before the isochronous video capture stream is started. The caller selects the appropriate alternate interface and sets up the device to stream. It is called in connection with the owning stream class RUN command.

NTSTATUS
  AllocateBandwidth(
    PDEVICE_OBJECT  BusDeviceObject,
    PVOID  DeviceContext,
    PULONG  RawFrameLength,
    PVOID  Format
    );

Parameters

BusDeviceObject
Device object created by the hub.
DeviceContext
Peripheral’s device context.
RawFrameLength
Size, in bytes, of the raw frame data from the packet stream.
Format
Pointer to PKS_DATAFORMAT_VIDEOINFOHEADER associated with this stream.

Return Values

Returns the NTSTATUS code indicating STATUS_SUCCESS if successful. Otherwise, it returns an appropriate error message.
