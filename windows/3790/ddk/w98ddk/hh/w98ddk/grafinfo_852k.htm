Color Mode Control

The ocdSetColorMode control contains the command or commands which are required to initialize a printer to process color data.

HP PaintJet

For the PaintJet printer, the Set Number of Planes Per Row command (ESC*rnU) is required to initialize the printer for color data processing. The argument n specifies the number of planes of color data.

To specify the Set Number of Planes Per Row command, type the following in the ocdSetColorMode box:

\x1B*r3U

After you specify the color-selection commands for your printer, from the Printer Data menu, choose the Graphics Info cascading menu. From the Graphics Info menu, choose the RectFill command. You can then begin entering rectangle-fill commands and data.
