KSPROPERTY_TVAUDIO_CAPS

The KSPROPERTY_TVAUDIO_CAPS read-only property is used to retrieve the capabilities of the TV audio device. It uses the following structure:

typedef struct {
  KSPROPERTY  Property;
  ULONG  Capabilities;
  KSPIN_MEDIUM  InputMedium;
  KSPIN_MEDIUM  OutputMedium;
} KSPROPERTY_TVAUDIO_CAPS_S,    *PKSPROPERTY_TVAUDIO_CAPS_S;
