System-Controlled Color Management

The recommended approach for printer vendors is to let the operating system handle color management. This is the default situation if the driver does not fully conform to ICM. In this case, if ICM is turned on, the operating system automatically corrects the colors of all pens, brushes, and DIBs. These automatic decisions are based on the input and output color spaces before passing these objects to the device driver. This results in vastly improved color reproduction on the printer. By having the operating system control these functions, the print driver is not required.
