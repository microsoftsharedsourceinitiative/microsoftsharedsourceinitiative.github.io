CANDIDATEINFO

typedef struct tagCANDIDATEINFO {
  DWORD  dwSize;
  DWORD  dwCount;
  DWORD  dwOffset[32];
  DWORD  dwPrivateSize;
  DWORD  dwPrivateOffset;
} CANDIDATEINFO;

Contains candidate information. This structure can have up to 32 candidate lists, and these candidate lists must reside in the same memory block.

dwSize
Size, in bytes, of this structure.
dwCount
Number of candidate lists included in the memory block.
dwOffset[32]
Array offsets from the beginning of this structure to the beginning of each candidate list.
dwPrivateSize
Private area in this memory block.
dwPrivateOffset
Offset from the beginning of this structure to the private area.
