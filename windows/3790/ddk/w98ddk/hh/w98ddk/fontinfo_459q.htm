Specifying Font Information

This section describes how you should specify font information for your printer. This information includes the Printer Font Metric (PFM) file, the Character-Translation Table (CTT), the text-quality commands and data, the font-cartridge commands and data, the font-simulation commands and data, and the downloadable-font commands and data.
