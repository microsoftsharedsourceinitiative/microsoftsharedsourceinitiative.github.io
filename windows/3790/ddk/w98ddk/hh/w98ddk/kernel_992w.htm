Destroy_Semaphore

include vmm.inc

mov     eax, Semaphore
VMMCall Destroy_Semaphore

Destroys the specified semaphore. Uses Flags.

  • No return value.
Semaphore
Handle of the semaphore to delete.

Destroying a semaphore while threads are still blocked on it will crash the system.
