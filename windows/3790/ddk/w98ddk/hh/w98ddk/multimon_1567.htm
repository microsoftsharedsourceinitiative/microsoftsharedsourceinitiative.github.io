GetMonitorInfo

GetMonitorInfo allows applications to get metric information relevant to a particular monitor: It enables applications to position themselves correctly on monitors other than the primary monitor. 

BOOL
  GetMonitorInfo(
    HMONITOR  hmonitor,
    LPMONITORINFO  lpmi
    );

Parameters

hmonitor
Specifies the handle of the monitor for which to get the information.
lpmi
Points to a MONITORINFO or MONITORINFOEX structure (both defined in the Platform SDK documentation) that will receive the information.
