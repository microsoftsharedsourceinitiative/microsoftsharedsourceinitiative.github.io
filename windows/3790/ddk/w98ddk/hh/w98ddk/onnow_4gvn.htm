GET_ADAPTER_POWER_STATE_CAPS

The GET_ADAPTER_POWER_STATE_CAPS function should return the power states supported by the display adapter.

DWORD __cdecl 
  GetAdapterPowerStateCaps(
    DEVNODE  devnode
  );

Parameters

devnode
Specifies the device node for the device.

Return Value
