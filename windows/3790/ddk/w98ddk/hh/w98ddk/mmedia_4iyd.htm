Calling Windows Functions at Interrupt Time

The only Windows functions a driver can call at interrupt time are:

  • PostMessage
  • PostAppMessage
  • DriverCallback
  • timeGetSystemTime
  • timeGetTime
  • timeSetEvent
  • timeKillEvent
  • midiOutShortMsg
  • midiOutLongMsg
  • OutputDebugStr
