Supported Font Simulations

The FontSimulation dialog box identifies the supported font-simulation commands. These commands are sent to the printer to simulate bold, italic, underline, double-underline, strike-through, and proportional-space fonts. The following list describes each of the controls in this dialog box.

Control Description
List box Lists the available font-simulation data for this printer.
Inhibit All Inhibits all font-simulation data which is identified in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus supports a single set of font-simulation commands.

HP LaserJet IIP

The HP LaserJet IIP supports a single set of font-simulation commands.
