Parameters for EUDC

The EUDC key in the following Registry path contains entries that define the parameters for the EUDC. 

HKEY_CURRENT_USER\EUDC\CodePage

The EUDC key contains a CodePage subkey. The values of the CodePage subkey is either 932 (Japan), 936 (Simplified Chinese), 949 (Unified Hangeul), or 950 (Traditional Chinese). The CodePage subkey may contain the following entries. 

SystemDefaultEUDCFont  REG_SZ     TrueTypeEUDCFontFileName

Specifies the system default EUDC font. The TrueTypeEUDCFontFileName is the file name of a TrueType EUDC font. The system default EUDC font does not work without this entry. 

TrueTypeFontTypeface    REG_SZ  TrueTypeEUDCFontFileName

Specifies a DBCS TrueType font typeface and its typeface-specific TrueType EUDC font file name. The TrueTypeFontTypeface is a DBCS TrueType font typeface and TrueTypeEUDCFontFileName is a typeface-specific TrueType EUDC font file name.

For example: 

HKEY_CURRENT_USER\EUDC\932
SystemDefaultEUDCFont=EUDC.TTF
MS Mincho=MINEUDC.TTF
MS Gothic=GOTEUDC.TTF
