Send Font Descriptor Box

The SEND_FONT_DESCRIPTOR control contains the command which downloads the font-descriptor data to a printer.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Font Descriptor command (ESC)snW) downloads font descriptor data to the printer for the font whose ID is specified by the argument n. Font Descriptor data is data that is common to all characters in a font; this data includes: the dimensions of the character cell, the orientation of the characters, the x-height, the stroke weight, the first character code, the last character code, and so on.

To specify the Font Descriptor command, type the following string in the SEND_FONT_DESCRIPTOR box:

\x1B)s%dW
