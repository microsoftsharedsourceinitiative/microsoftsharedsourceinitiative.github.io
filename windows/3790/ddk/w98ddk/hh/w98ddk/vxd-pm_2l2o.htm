Disable_DevNode_Wakeup

Components that want to disable wake up on a device call the Disable_DevNode_Wakeup service. Calling this service causes the configuration manager to clear the flag in the device node to indicate that the device is enabled for wake up and to send a CONFIG_DISABLEWAKEUP message to the device. 

CONFIGRET
  CONFIGMG_Disable_DevNode_Wakeup(
    DEVNODE  dnDevNode, 
    ULONG  ulFlags
    );

Parameters

dnDevNode
Handle to the device node in which the configuration manager disables wake up.
ulFlags
Reserved. Currently must be zero.
