VPOWERD_Disable_Timer_Based_Requests

VPOWERD_Disable_Timer_Based_Requests prevents the BIOS from initiating timer-based changes in the power state. It is used only in APM mode.

POWERRET VPOWERD_Disable_Timer_Based_Requests( VOID );

Parameters

None.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
