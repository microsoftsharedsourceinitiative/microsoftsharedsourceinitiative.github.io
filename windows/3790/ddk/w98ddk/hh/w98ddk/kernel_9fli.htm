PageFile_Test_IO_Valid

VxDcall PageFile_Test_IO_Valid
jc      paging_not_permitted

Determines whether paging to or from disk can be performed. Uses Flags.

  • Returns with carry clear if paging is permitted. Otherwise, the carry flag is set, indicating you should avoid operations that may cause paging.
