Windows 95 Support for Large IDE Drives

Microsoft® Windows® 95 supports drives whose characteristics exceed the standard INT 13h limitations. It does this by using geometry translation, INT 13 extensions, Logical Block Addressing (LBA) mode, or software mechanisms. Windows 95 Support for Large IDE Drives describes the methods Windows 95 uses to detect whether a drive needs support and explains how this support is implemented.
