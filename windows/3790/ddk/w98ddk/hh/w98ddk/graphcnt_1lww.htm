Output(OS_ENDNSCAN)

WORD Output(
  LPPDEVICE lpDestDev, 
  WORD wStyle, 
  WORD wCount, 
  LPPOINT lpPoints, 
  LPPPEN lpPPen, 
  LPPBRUSH lpPBrush, 
  LPDRAWMODE lpDrawMode, 
  LPRECT lpClipRect
 );

Carries out any action needed to complete a series of scan line output.

The return value is 1.

lpDestDev
Address of a PDEVICE or PBITMAP structure that specifies the destination device or bitmap.
wStyle
Must be OS_ENDNSCAN
wCount
Must be 0.
lpPoints
Must be NULL.
lpPPen
Must be NULL.
lpPPBrush
Must be NULL.
lpDrawMode
Must be NULL.
lpClipRect
Must be NULL.

GDI calls this function to mark the end of a series of calls to the Output function in which the OS_SCANLINES style is specified. GDI marks the start of the series with a matching OS_BEGINNSCAN style.

Every graphics driver (for Windows versions 2.0 and later) should support the OS_ENDNSCAN style.
