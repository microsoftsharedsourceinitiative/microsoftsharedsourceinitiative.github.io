CONFIGMG_Get_Performance_Info

CONFIGRET CONFIGMG_Get_Performance_Info(PCMPERFINFO pPerfInfo, ULONG ulFlags)

This API works only in MAXDEBUG. It returns or reset performance info in a buffer.

  • Returns CR_SUCCESS if the function is successful. Otherwise, the return value can be CR_INVALID_FLAG, CR_FAILURE (CM_PERFORMANCE_INFO was not defined) or CR_INVALID_POINTER.
pPerfInfo
Address that will receive the size (in bytes) of the resource-descriptor header.
ulFlags
Must be one of CM_PERFORMANCE_INFO_GET_DATA CM_PERFORMANCE_INFO_RESET, CM_PERFORMANCE_INFO_START or CM_PERFORMANCE_INFO_STOP.
