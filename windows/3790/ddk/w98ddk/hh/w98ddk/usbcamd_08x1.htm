USB Camera Minidriver Interface Reference

This documentation is written for developers of the Universal Serial Bus drivers for Microsoft® Windows® 98, and Windows® 2000. The USB Camera Driver is described in this documentation and is customized for Windows® 98 applications.

Hardware vendors are responsible for creating a stream class minidriver that implements the Universal Serial Bus (USB) Camera Minidriver Interface for all USB devices. The IHV-provided minidriver and the USBCAMD driver combine to create functionality equivalent to that of the test cap sample in the DDK.
