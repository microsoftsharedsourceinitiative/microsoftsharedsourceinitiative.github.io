KSPROPERTY_TUNER_INPUT

The KSPROPERTY_TUNER_INPUT read/write property is used to specify the input connector. 

typedef struct {
  KSPROPERTY  Property;
  ULONG  InputIndex;
} KSPROPERTY_TUNER_INPUT_S, *PKSPROPERTY_TVTUNER_INPUT_S;

Members

Property
Specifies the property used.
InputIndex
Specifies the input connection. This value must be in the range of zero to (NumberOfInputs – 1).
