Landscape Font Names

The Landscape Fonts box contains strings that specify the names of landscape fonts supported by a cartridge. UniTool copies the names of all the PFM font files used by the minidriver into this control and the minidriver developer specifies which fonts are supported on a particular cartridge by choosing the correct names with the cursor and then clicking the mouse button.

HP LaserJet IIP

For the HP LaserJet IIP printer, one of the 303 device fonts is supported in landscape mode on the Microsoft 1A cartridge. The string which identifies this font (numbered 192) is highlighted.
