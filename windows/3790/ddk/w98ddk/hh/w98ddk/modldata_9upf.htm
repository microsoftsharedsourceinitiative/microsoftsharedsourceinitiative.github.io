Default and Supported Paper Sizes

The Paper Sizes dialog box identifies the default and supported paper sizes. The default paper size appears in the dropdown list box at the top of the dialog box. The supported paper sizes are highlighted in the list box below the dropdown list box.

The following list describes each of the controls in this dialog:

Control Description
Default Specifies the default paper size.
List box Lists the available paper sizes for this printer.
Enable All Enables all paper sizes which appear in the list box.
Inhibit All Inhibits all paper sizes which appear in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus printer supports 8.5 x 11 inch as well as user-defined paper sizes.

HP LaserJet IIP

The HP LaserJet IIP printer supports 8.5 x 11 inch, 8.5 x 14 inch, and 210 x 297-millimeter paper sizes.
