IFSMgr_DosToNetTime

IFSMgr_DosToNetTime(
 dostime_t DosTime
 )

This services converts the time passed in DOS local time format to net time format which is in UTC.

DosTime
DOS time filled in the dostime_t structure.
  • Returns the time in the net time format in UTC.
