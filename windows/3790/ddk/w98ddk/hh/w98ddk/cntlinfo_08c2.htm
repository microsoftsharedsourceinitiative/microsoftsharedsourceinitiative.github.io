Adding Page-Control Data to Your Driver

Once you've specified the necessary commands and the ordering of those commands, you can add the page-control data to your driver's data table. You can do this by choosing the Add or OK buttons.

The driver-data table contains two types of control information: page-control data and cursor-movement data. Once you've set the controls in the PAGECONTROL dialog box, from the Control Info cascading menu in the Printer Data menu, choose the CursorMove command. You can then begin specifying cursor-movement data.
