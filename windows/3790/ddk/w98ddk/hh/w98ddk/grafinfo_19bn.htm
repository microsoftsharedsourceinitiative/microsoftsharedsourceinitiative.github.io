Number of Pins

The sNPins box in the RESOLUTION dialog box contains a value that specifies the minimum height (in pixels) of lines of raster data that the Universal Printer Driver will evaluate at one time. This is typically, but not always, the number of pins (or jets) used by the printer for each vertical slice of graphic output produced by one pass of the print head. When the value of sNPins is equal to the value of sPinsPerPass, this is referred to as noninterlaced graphics output.
