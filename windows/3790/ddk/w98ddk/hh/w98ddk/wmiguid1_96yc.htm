IoWMIFireEvent

The IoWMIFireEvent routine will fire a WMI event using the data buffer passed. This routine may be called at or below DPC_LEVEL.

NTSTATUS IoWMIFireEvent(
    IN PWMILIB_INFO WmiLibInfo,
    IN PDEVICE_OBJECT DeviceObject,
    IN ULONG GuidIndex,
    IN ULONG EventDataSize,
    IN PVOID EventData
    );

Parameters

WmiLibInfo
WMI information control block.
DeviceObject
Supplies a pointer to the device object for this event.
GuidIndex
Index into the GUID List within the WmiLibInfo control block for the GUID of the event.
EventDataSize
Number of bytes of data that is being fired with the event.
EventData
Data that is fired with the events.

Return Value

Returns a status code.
