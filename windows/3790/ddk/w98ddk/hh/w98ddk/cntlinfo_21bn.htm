Maximum Number of Copies

The Max # Copies box specifies the maximum number of copies that the user can request.

Diconix 150 Plus

If a printer does not support multiple copies, this box can be left empty; however, for the sake of clarity, the box contains the value 1 for the Diconix minidriver.

HP LaserJet IIP

The HP LaserJet IIP printer allows a maximum of 99 copies; therefore, the Max # Copies box contains the value 99.
