Display Driver

When using a Windows 95 driver or a driver that does not support the multimonitor feature for Windows 98, the following limitations apply:

  • When a driver that does not support multimonitor is used as the primary, “Use as desktop” cannot be set for other display adapters (secondary).
  • When a driver that does not support multimonitor is used as a secondary “Use as desktop” cannot be set for the display adapter that uses this driver. (Other display adapters can use the multimonitor feature.)

In addition, it must be determined which display driver is calling the function within the baseVDD interface, because several display drivers are operating simultaneously in a multimonitor environment.
