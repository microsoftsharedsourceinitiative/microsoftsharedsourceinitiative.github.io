VPOWERD_Get_Timer_Based_Requests_Status

VPOWERD_Get_Timer_Based_Requests_Status gets the status of the timer-based requests flag in the BIOS. It is used only in APM mode.

POWERRET VPOWERD_Get_Timer_Based_Requests_Status( 
  DWORD   *pStatusVOID 
);

Parameters

pStatus
Retrieves the time request status with CAPABILITY_ENABLED or CAPABILITY_DISABLED.

Return Values

The return value is POWERRET. This is PR_SUCCESS if the call successfully completes. Otherwise, an appropriate error message is returned.
