WNODE_EVENT_REFERENCE

typedef struct tagWNODE_EVENT_REFERENCE {
    struct _WNODE_HEADER WnodeHeader;
    GUID TargetGuid;
    ULONG TargetDataBlockSize;
    Union {
        ULONG TargetInstanceIndex;
        WCHAR TargetInstanceName[];
    };
} WNODE_EVENT_REFERENCE, *PWNODE_EVENT_REFERENCE;

Comments

In the case of a call where the data buffer passed was too small, the data provider should return success with a WNODE_TOO_SMALL. This is composed of a filled in WNODE_HEADER with the WNODE_FLAG_TOO_SMALL flag set directly followed by a DWORD that specifies the size needed to contain the entire result.
