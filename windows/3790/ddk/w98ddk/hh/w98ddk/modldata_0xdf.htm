Supported Page Controls

The PAGECONTROL dialog box identifies the supported page-control commands. These commands are sent to the printer to begin a document, begin a new page, and cancel the printing of a document.

Control Description
List box Lists the available page-control data for this printer.
Inhibit All Inhibits all page-control data which is identified in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus supports a single set of page-control commands.

HP LaserJet IIP

The HP LaserJet IIP supports a single set of page-control commands.
