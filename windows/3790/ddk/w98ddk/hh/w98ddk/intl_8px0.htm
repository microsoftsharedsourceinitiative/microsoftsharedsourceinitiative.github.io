End User Defined Characters

In Asian versions of Windows® 95, there is a special code range or ranges in each DBCS character set (code page) in which the end user can define special characters called End User Defined Characters (EUDC). This section describes these code ranges and explains the affect of using these characters with Windows functions and structures.
