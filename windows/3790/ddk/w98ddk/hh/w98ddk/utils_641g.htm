.dosmgr (Display Real Mode DOS structures)

Dumps various data structures related to real-mode DOS.

[1] Display SFTs

[2] Display DPBs

[3] Display MCBs (dump the real-mode memory map)

[4] Display CDSs

[5] Display PDBs (dump the real-mode process map)

[6] Display DEVs

[7] Display BDSs

[8] Display INT 21h status
