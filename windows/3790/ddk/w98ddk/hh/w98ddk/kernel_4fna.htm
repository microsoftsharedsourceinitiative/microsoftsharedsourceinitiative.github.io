VWIN32_EnterCrstR0

The VWIN32_EnterCrstR0 function waits for ownership of the specified critical section object. The function returns when the calling thread is granted ownership. Uses EAX, ECX, EDX.

  • Returns with the carry flag set if there was a fault accessing critical section.
EDX
Pointer to critical section object.
