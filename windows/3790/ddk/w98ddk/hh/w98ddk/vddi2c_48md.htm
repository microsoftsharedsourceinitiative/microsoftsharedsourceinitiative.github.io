Exporting the I2C Interface

The miniVDD can use the standard MiniVDDDispatch macro to export the I2C_ACCESS and I2C_OPEN functions, as shown in the following code sample: 

MiniVDDDispatch    I2C_ACCESS, i2c_access

The miniVDD can also load the function table directly, in the following manner: 

mov     [edi+(I2C_OPEN*4)], OFFSET32 i2c_openfunc
