Point Size

The Points control contains the point size for the font. The point size is determined by measuring the vertical distance from the bottom edge to the top edge of a character cell for a font. The value is specified in printer's points; each printer's point is equivalent to approximately 1/72 of an inch.

Note  Most applications do not use this box to display point sizes to the user because it is incapable of expressing fractional point sizes. Most applications calculate point size as the sum of the value of the PixHeight and Internal Leading controls in the PFM dialog box.

Diconix 150 Plus

For the Diconix 150 Plus printer, the point size is 10.
