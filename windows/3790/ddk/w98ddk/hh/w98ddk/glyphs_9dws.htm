Internet Printing Protocol

Windows® Millennium Edition (Windows Me) ships with an Internet Printing Protocol (IPP) client. The IPP client is located in the Add-Ons directory of the CD, and is named IPPClient.exe. The IPP client can be selectively installed.

The Internet Printing Protocol client supports the industry-standard IPP 1.0 protocol and allows a user to print over the Internet. With the IPP client, your Windows computer can print to any IPP-enabled printer or to any printer running on a Windows 2000 or later print server over the Internet. Microsoft's IPP 1.0 client uses standard Internet protocols (HTTP on TCP port 80), so no special configuration of proxies or firewalls is required. If you can browse to the print server, you can also use IPP to print to it.

To install the Internet Printing Protocol client

  1. Double click IPPClient.exe.
  2. Click OK to reboot your machine.

    After you have installed the Internet Printing Protocol client, you can connect to a printer in either of two ways:

  3. Using a browser.
  4. Using the Add Printer Wizard.

To connect to a printer using a browser

  1. From your browser, go to the URL of the printer you want to connect to. (If you are connecting to a printer on a Windows 2000 or later print server, you can go to http://servername/printers to browse through a list of all the printers on that server.
  2. Click the name of the printer you want to connect to.
  3. Under Printer Actions on the left hand side, click Connect.

To connect to a printer using the Add Printer Wizard

  1. Open the Printers Folder.
  2. Double-click the Add Printer Wizard.
  3. Click Next on the welcome screen.
  4. Select Networked Printer, and then click Next.
  5. In the Network path or queue name text box, type the URL of the printer (i.e., http://servername/printers/myprinter), and then click Next.
  6. Select the manufacturer and model of the printer you are connecting to, and then click Next.
  7. Select Yes to print a test page, if you wish, then click Next.
  8. Click Finish.
