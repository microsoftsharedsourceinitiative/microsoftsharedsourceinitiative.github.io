IFSMgr_DebugMenu

IFSMgr_DebugMenu(
 pIFSMenu pm
 )

This service prints the debug service menu that is passed in and returns the address of the handler routine for the menu item that the user selected. The FSD passes in a table of menu strings and handler addresses, one for each menu item. The IFS manager displays the entire menu and then returns the address of the handler routine associated with the menu item the user selects. The user can hit ESC to quit the menu without selecting anything. This service is available only on debug versions of the IFS manager. On the nondebug version, this service returns without doing anything. This service is not available until the IFS manager has finished its Sys_Crit_Init during device initialization.

pm
Supplies pointer to menu structure which is an array of IFSMenu structures.
IFSMenu Structure:  
Value Meaning
pMenu_string Supplies menu string to display. A NULL pointer for the string indicates end of the list of menu items.
pMenuHandler Supplies handler for this menu item.

  • Returns the address of the menu handler for the menu item selected by the user. Returns NULL if the user did not select any menu item.
