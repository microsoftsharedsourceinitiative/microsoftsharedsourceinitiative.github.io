Miscellaneous Keyboard Services

The keyboard driver provides a number of additional services to Windows and Windows applications. These miscellaneous services are provided by the following functions:

EnableKBSysReq GetKeyboardType
GetBIOSKeyProc GetKeyNameText
GetKBCodePage SetSpeed
