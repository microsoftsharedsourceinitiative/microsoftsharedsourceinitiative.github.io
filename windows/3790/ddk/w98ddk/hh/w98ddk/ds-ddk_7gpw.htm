IDsDriver::AddRef

The IDsDriver::AddRef method increases the object reference count by 1. 

ULONG AddRef( );

Return Values

Returns the new reference count.

Remarks

When the object is initially created, its reference count should be set to one. Each time a function obtains an interface to the object or calls the AddRef method, the reference count should be increased by 1. The application uses the Release method to decrease the object's reference count by 1.

This method is part of the IUnknown interface inherited by the object.
