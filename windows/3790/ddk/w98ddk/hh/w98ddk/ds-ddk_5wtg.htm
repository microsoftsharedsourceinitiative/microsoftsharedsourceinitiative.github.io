DSPROPERTY

DirectSound passes property set data and an application instance identifier to a DirectSound driver using this structure. 

typedef union _DSPROPERTY
{
    struct
    {
        GUID    Set;
        ULONG   Id;
        ULONG   Flags;
        ULONG   InstanceId;
    };
#if defined(_NTDDK_)
    ULONGLONG  Alignment;
#else // !_NTDDK_
    DWORDLONG  Alignment;
#endif // !_NTDDK_
} DSPROPERTY, *PDSPROPERTY;

Members

Set
Uniquely identifies a property set.
Id
Uniquely identifies a property within a property set.
Flags
Unused.
InstanceId
Passes to the driver a value identifying the application that induces the call on the property set.
Alignment
Unused.
