Changing the Model Drop Down List

To change the entries in the pen driver configuration dialog Model dropdown list, you do not need to change any source code. Since the list is populated by information from the Windows 95 registry, you can change the items that appear in the list by changing the .INF file used by Windows 95 startup to create the registry entries when Pen Services for Windows 95 is installed. For instructions on how to change the .INF file, see the Pen Services for Windows 95 OEM Adaptation Kit (OAK) Release Notes.
