The Printer Font Metric Dialog Box

The following error messages are associated with the Printer Font Metric (PFM) dialog box.

List Description
minidriver name :Type must = 128 UniTool issues this message if the value in the Type box does not equal 128.
minidriver name :Points must be > 0 UniTool issues this message if a point size value that is less than or equal to zero is entered.
minidriver name :VertRes must be > 0 UniTool issues this message if a vertical resolution value that is less than or equal to zero is entered.
minidriver name :HorizRes must be > 0 UniTool issues this message if a horizontal resolution value that is less than or equal to zero is entered.
minidriver name :Ascent must be > 0 UniTool issues this message if a value in the Ascent box that is less than or equal to zero is entered.
minidriver name :Weight should be 400 or 700 UniTool issues this message if a value in the weight box that is not equal to either 400 or 700 is entered.
minidriver name :CharSet must be 0,1, or 2 UniTool issues this message if a value in the Char Set box that is not equal to 0, 1, or 2 is entered.
minidriver name :AvgWidth must be > 0 UniTool issues this message if a negative value is entered in the Average Width box.
Kerning Info will be lost! Continue? UniTool issues this message if kerning information will be lost.
