DiDestroyClassInfoList

This API will delete a single, or list of, CLASS_INFO structure(s). 

RETERR
  DiDestroyClassInfoList(
    LPCLASS_INFO  lpci
    );

Return Value

OK

Parameters

lpci
A pointer to a single or list of CLASS_INFO structure(s).

Comments

This function will free any memory associated with the passed in lpci. This function will free any DEVICE_INFO structures also associated with this CLASS_INFO.
