Canceling the Document

The PC_OCD_ABORT box contains the command or commands which the Universal Printer Driver issues to cancel the current document.

Diconix 150 Plus

For the Diconix 150 Plus, the Reset command (ESC@) returns the printer to its default settings and the Cancel command (ESCM) deletes all data from the print buffer.

To specify the Reset and Cancel combination, type the following in the PC_OCD_ABORT box:

\x1B@\x18

HP LaserJet IIP

For the HP LaserJet IIP, the Reset command (ESCE) returns the printer to its default settings.

To specify the Reset command, type the following in the PC_OCD_ABORT box:

\x1BE
