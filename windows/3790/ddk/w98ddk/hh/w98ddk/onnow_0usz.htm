GET_MONITOR_POWER_STATE_CAPS

The GET_MONITOR_POWER_STATE_CAPS function returns the power states that are supported by the monitor and invoked explicitly by the display driver. If this function is not hooked, the VDD uses VESA calls to get the capabilities for the monitor.

GET_MONITOR_POWER_STATE_CAPS does not need to be hooked by most display drivers. It is provided as a convenience for special cases where the miniVDD needs to do something proprietary to control the monitor's power states.

DWORD __cdecl 
  GetMonitorPowerStateCaps(
    DEVNODE  devnode
  );

Parameters

devnode
Specifies the device node for the device.

Return Value

Bitmask of:

CM_POWERSTATE_D0
CM_POWERSTATE_D1
CM_POWERSTATE_D2
CM_POWERSTATE_D3
