PROPSETID_VIDCAP_CAMERACONTROL

The PROPSETID_VIDCAP_CAMERACONTROL property set is used to adjust or inquire about camera control settings. The controls available are a subset of the proposed ITU T.RDC standard. All the properties within this property set are considered optional.

Note that camera device should use the VideoProcAmp property set to adjust qualities such as brightness, contrast, hue, and saturation.

GUID PROPSETID_VIDCAP_CAMERACONTROL
DirectShow Interface IAMCameraControl
Property Structure KSSTRUCT_VIDCAP_CAMERACONTROL

The properties include the following of type KSPROPERTY_VIDCAP_CAMERACONTROL:

KSPROPERTY_CAMERACONTROL_PAN

KSPROPERTY_CAMERACONTROL_TILT

KSPROPERTY_CAMERACONTROL_ROLL

KSPROPERTY_CAMERACONTROL_ZOOM

KSPROPERTY_CAMERACONTROL_EXPOSURE

KSPROPERTY_CAMERACONTROL_IRIS

KSPROPERTY_CAMERACONTROL_FOCUS

All properties within this property set use the same structure to set and retrieve individual properties.

typedef struct {
  KSPROPERTY Property;
  LONG  Value;
  ULONG  Flags;
  ULONG  Capabilities;
} KSPROPERTY_CAMERACONTROL_S, *PKSPROPERTY_CAMERACONTROL_S;

Members

Property
Specifies the property used.
Value
Specifies values unique to that property. See the individual property for information on the range and default value for this member.
Flags
Specifies the flag from KSPROPERTY_CAMERACONTROL_FLAGS_*. See the table below for a list of the flags provided.
Capabilities
Specifies the capabilities from KSPROPERTY_CAMERACONTROL_FLAGS_*. See the table below for a list of the flags provided.

The following table describes the values for KSPROPERTY_CAMERACONTROL_S.Flags and KSPROPERTY_CAMERACONTROL_S.Capabilities.

Flag Description
KSPROPERTY_CAMERACONTROL_
FLAGS_MANUAL		 Specifies that the setting is controlled manually.
KSPROPERTY_CAMERACONTROL_
FLAGS_AUTO		 Specifies that the setting is controlled automatically.
KSPROPERTY_CAMERACONTROL_
FLAGS_ABSOLUTE		 Specifies that the setting is in absolute values.
KSPROPERTY_CAMERACONTROL_
FLAGS_RELATIVE		 Specifies that the setting is in relative values.

Each property also includes a range and default value, the KSPROPERTY_CAMERACONTROL_S.Value. The range for the property set is defined in real world units to allow programmatic control of parameters. Since actual hardware is often limited in how closely it can approximate this ideal, each device can also define a subset of this theoretical range and the size of adjustment steps.

For those properties that involve camera positioning, the point of reference is the camera imaging plane looking outward at the image being captured. Positive positioning values are generally clockwise from the origin, and negative positioning values are counterclockwise from the point of origin.
