Obsolete Functionality

All Printing APIs and Long File Names APIs defined in Windows 3.1 WinNet have no equivalent in the Windows 95 Provider API set. Also, the following Windows 3.1 WinNet APIs are not used by Windows 95:

WNetGetPropertyText  
WNetPropertyDialog
