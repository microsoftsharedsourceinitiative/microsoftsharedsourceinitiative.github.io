Default and Supported Text Qualities

The Text Qualities dialog box identifies the default and supported text qualities. The default text quality appears in the dropdown list box at the top of the dialog box. The supported text qualities are highlighted in the list box.

The following list describes each of the controls in this dialog box.

Control Description
Default Specifies the default text-quality identifier.
List box Lists the available text-quality identifiers for this printer.
Enable All Enables all text-quality identifiers which appear in the list box.
Inhibit All Inhibits all text-quality identifiers which appear in the list box.
Cancel Cancels current edit changes.
OK Saves current edit changes.

Diconix 150 Plus

The Diconix 150 Plus printer supports letter-, near-letter-, and draft-quality text output.

HP LaserJet IIP

The HP LaserJet IIP printer does not support multiple text qualities.
