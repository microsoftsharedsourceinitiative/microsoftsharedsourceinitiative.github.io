Writing Kernel-Mode Drivers

Kernel-mode drivers that conform to the Windows Driver Model (WDM) can run on Windows 98, Windows Millennium Edition, and Windows 2000 and later systems. All WDM drivers must use kernel-mode driver support routines, which are defined by the operating system. See the Kernel-Mode Driver Reference in the Windows DDK for information on these routines.

The Kernel-Mode Driver Reference documents all kernel-mode routines supported by Windows. Most of these routines are supported by WDM on Windows 98 and ME, but a few are not. The reference material for each support routine lists the header file(s) in which the routine is defined. Unless otherwise noted, routines defined in wdm.h can be used by WDM drivers on Windows 98, Windows ME, and Windows 2000 and later. Routines defined in ntddk.h, but not in wdm.h, are not supported on Windows 98 or ME.

In general, the support routines behave similarly on all three operating systems. Any individual differences are noted in the reference page for the relevant routine.
