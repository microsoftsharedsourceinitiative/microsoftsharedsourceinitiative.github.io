VMDAPI_MOUSEDRV_CONSISTENT

include vmd.inc
 
mov     ax, VMDAPI_MOUSEDRV_CONSISTENT
call    pVMD_API_Entry

Discovers whether the current mouse driver has registered its mouse-event callback function with VMOUSE. Uses EAX and Flags.

  • Returns one in AL if the driver has registered the callback function, zero otherwise.
