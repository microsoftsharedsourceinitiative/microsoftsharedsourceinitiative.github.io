PageFile_Read_Or_Write

mov     ch, read_or_write  ; PF_Read_Data or PF_Write_Data
mov     edi, filepos       ; file position, in pages
mov     esi, pPage         ; linear address of memory to touch
VxDcall PageSwap_Read_Or_Write
or      eax, eax
jz      error_handler

Reads a page from disk or writes a page to disk. Uses Flags.

  • Returns a nonzero value in EAX if successful, zero otherwise.
read_or_write
Value indicating whether to read from or write to the swap file. Can be PF_Read_Data or PF_Write_Data.
filepos
Offset in the swap file to read from or write to, in pages.
pPage
Linear address of the memory page to read from or write to.
