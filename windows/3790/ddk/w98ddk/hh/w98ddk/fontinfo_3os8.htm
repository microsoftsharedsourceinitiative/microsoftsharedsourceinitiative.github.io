Set Character Code Box

The SET_CHAR_CODE box contains the command which identifies the next character to be printed.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Character Code command (ESC*cnE) designates the character code that will be associated with the next character to be downloaded.

To specify the Character Code command, type the following in the SET_CHAR_CODE box:

\x1B*c%dE
