The Paper Sizes Dialog Box

The following error messages are associated with the Paper Sizes dialog box.

List Description
minidriver name :Illegal X Size UniTool issues this message if an illegal paper width is entered.
minidriver name :Illegal Y Size UniTool issues this message if an illegal paper height is entered.
minidriver name :Illegal Top Margin UniTool issues this message if an illegal top-margin height is entered.
minidriver name UniTool issues this message if an illegal :Illegal Bottom Margin bottom-margin height is entered.
minidriver name :Illegal Left Margin UniTool issues this message if an illegal left-margin width is entered.
minidriver name :Illegal Right Margin UniTool issues this message if an illegal right-margin width is entered.
minidriver name :Illegal fPaperType bit(s) set UniTool issues this message if the developer selects an illegal paper-type box.
