VxD_REAL_INIT_ENDS

include vmm.inc

VxD_REAL_INIT_ENDS

Defines the end of a real-mode initialization segment. Virtual devices use this macro with in conjunction with the VxD_REAL_INIT_SEG macro to create initialization code for real-mode execution.

  • No return value.
