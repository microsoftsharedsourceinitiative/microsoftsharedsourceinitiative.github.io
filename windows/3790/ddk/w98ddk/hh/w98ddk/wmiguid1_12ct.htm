Comments

A WNODE_EVENT_ITEM is built by a data provider and used to deliver event information to all data consumers who have registered for the specific event being generated. A WNODE is a WNODE_EVENT_ITEM when the WNODE_FLAG_EVENT_ITEM flag is set in the WNODE_HEADER. Depending upon the flag set in the WNODE_HEADER, the event data could be a WNODE_SINGLE_ITEM, WNODE_SINGLE_INSTANCE, WNODE_ALL_DATA, or WNODE_EVENT_REFERENCE. A data provider can attach a severity level from zero to 256 by placing the severity level in the top 8 bits of the Flags in the WNODE_HEADER.
