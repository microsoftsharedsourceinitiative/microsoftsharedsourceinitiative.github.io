New Functionality

In addition to the Windows 3.1 WinNet features, the parameters to the AddConnection and GetUser functions have been extended. A set of network resource enumeration functions have been added, as well as functions to tailor the Shell Explorer property list descriptions. Functions have been added to support connection with authentication and network-specific log on and log off.
