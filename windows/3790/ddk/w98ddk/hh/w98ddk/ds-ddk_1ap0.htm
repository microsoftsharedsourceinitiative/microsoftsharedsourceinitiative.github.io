Architectural Overview

The following diagram shows the relationships between DirectSound and other system audio components.

To obtain the best performance, a DirectSound HAL must be implemented for the sound hardware. The HAL is a set of functions that are implemented by a driver specific to each type of sound hardware. If a DirectSound driver is not installed for the sound hardware, DirectSound uses its HAL emulation layer. This layer uses the Wave APIs to output sound. This allows DirectSound applications to run on any Windows sound device (although the performance is much better when a DirectSound driver is used).
