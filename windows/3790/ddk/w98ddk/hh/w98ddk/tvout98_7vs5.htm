Win32 Application Interface

The Win32 API ChangeDisplaySettingsEx has been modified to contain both a structure to control television parameters and a flag to indicate that the structure is present. CDS_VIDEOPARAMETERS is the flag that indicates that the lParam element is actually a pointer to a VIDEOPARAMETERS structure. This structure can be used for both querying the television capabilities of a driver and hardware set, and for adjusting supported features of the hardware through the driver. Possible features supported are TV standard adjustment, flicker filter adjustment, scaling, positioning, brightness and contrast adjustment, and copy protection support.

The following example queries the current video parameter settings:

#include “tvout.h”
dm.dmSize = sizeof(DEVMODE);
vp.Guid = vpguid;
vp.dwCommand = VP_COMMAND_GET;
if (ChangeDisplaySettingsEx("\\\\.\\Display1",&dm,0,CDS_VIDEOPARAMETERS,&vp) ==  DISP_CHANGE_SUCCESSFUL) 
{
   // process good return code here
}

The possible return values include:

Value Description
DISP_CHANGE_BADPARAM The ChangeDisplaySettingEx calling parameters were incorrect.
DISP_CHANGE_BADESC Driver does not support functionality, or the driver returned an error.
DISP_CHANGE_SUCCESSFUL Driver interface succeeded correctly.
