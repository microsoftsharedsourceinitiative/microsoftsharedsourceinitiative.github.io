Special Uses of INF Files

There are several device installation requirements that can be met by INF files, but which are not part of the installation requirements for the sample INF file, XSCSI.INF. These special requirements of INF files are:

  • Installing a multifunction device.
  • Installing a property provider.
  • Overriding PCMCIA tuples on PC card devices.
