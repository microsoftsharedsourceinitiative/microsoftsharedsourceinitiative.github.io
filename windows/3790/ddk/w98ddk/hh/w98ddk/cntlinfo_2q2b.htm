Specifying Page Control Commands

Each printer provides a unique set of commands that control the printing of a document or a page. These commands are called page-control commands. For example, these commands initialize the device prior to printing a document, initialize the device prior to printing a page, cancel the printing of a document, print multiple copies of a document, and so on.

The PAGECONTROL dialog box contains controls in which you specify these commands as well as controls which specify the order in which the Universal Printer Driver should send the commands to your printer.
