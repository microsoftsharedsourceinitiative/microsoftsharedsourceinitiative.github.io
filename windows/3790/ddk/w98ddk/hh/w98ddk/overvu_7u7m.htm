Description of a Printer Driver

A Microsoft® Windows® printer driver is an executable file that performs the tasks in the following list:

  • Retrieves device capabilities and metrics for an application.
  • Prepares the device for printing (by selecting the requested font, graphics resolution, paper orientation, paper source, and so on).
  • Converts GDI calls into device commands which generate output on a page of printer paper.
