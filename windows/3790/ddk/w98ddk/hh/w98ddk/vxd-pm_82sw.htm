Enable_DevNode_Wakeup

Components that want to enable wake up on a device call the Enable_DevNode_Wakeup service. This service causes the configuration manager to set a flag in the device node to indicate that the device is enabled for wake up and to send a CONFIG_ENABLEWAKEUP message to the device. The configuration manager returns failure if the device does not support wake up.

CONFIGRET
  CONFIGMG_Enable_DevNode_Wakeup(
    DEVNODE  dnDevNode, 
    ULONG  ulFlags
    );

Parameters

dnDevNode
Handle to the device node in which the configuration manager enables wake up.
ulFlags
Reserved. Currently must be zero.
