Stream Categories

The following tables describe both the different types of pins/streams available from the KSProxy capture filter and the format types associated with each stream. The table entries prefaced by "DS" give the DirectShow equivalent in user mode.

The Category GUID and PinName GUID for a given video capture stream type are normally identical. The only case where this is not true is when a given stream Category has more than one instance on a filter. In this case, the Category GUIDs should match, but each pin should be assigned a unique PinName GUID.
