DRV_CONFIGURE

Requests the driver to display its configuration dialog box.

  • DRV_SUCCESS if successful; DRV_FAILURE if the windows handle is invalid.
lParam1
Low-order word contains the handle to a window that will own the dialog box.
lParam2
Should be set to zero.
