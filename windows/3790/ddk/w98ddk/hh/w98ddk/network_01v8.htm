How to Port a Network Driver to Windows 95

You will probably want to find out several things:

  • Differences between Windows for Workgroups and Windows 95. For more information, see Differences from Windows for Workgroups.
  • Miniport overview. For more information, look in the Microsoft Development Library CD.

Microsoft recommends that you write a miniport driver for Windows 95. However, you will need the Windows NT 3.51 DDK to do so.
