Windows 2000 Developers and Windbg for Kernel Debugging on Windows 95

Windows NT/Windows 2000 developers cannot use windbg for kernel debugging on Windows 95. They can only use only WDEB386 or a third-party debugger such as SoftIce/W for kernel debugging.
