TURN_VGA_OFF

TURN_VGA_OFF is an optional function that is called to allow the miniVDD to turn off access to the VGA resources on the given device.

Implement this function only if your device cannot be turned off using the PCI configuration registers, or if your device needs to access hardware registers while its VGA resources are off. An example would be a card that is processing hardware interrupts, and the interrupt service routine needs to touch non-VGA resources on the device. Another example would be for a multifunction card, capable of emulating multiple types of devices. Note, however, that the following caveats apply:

  • If this function is implemented, the decode of the VGA resources must be OFF. That is, reading 3c4h, 3c5h, 3d4h, 3d4h, 3ceh, 3cfh all return 0FFh.
  • If the miniVDD has virtualized interrupts and does not implement TURN_VGA_OFF, then the VDD will never start a secondary display device.
Entry Meaning
ESP+4 Device node.
