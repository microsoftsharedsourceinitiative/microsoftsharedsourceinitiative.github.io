Maximum Font Count

The sMaxFontCount box specifies the maximum number of fonts which may be downloaded at any given point in time.

HP LaserJet IIP

For the HP LaserJet IIP printer, the sMaxFontCount box contains the value -1; this means that the number of fonts that may be downloaded to the printer at any given time is unlimited (except by available printer memory).
