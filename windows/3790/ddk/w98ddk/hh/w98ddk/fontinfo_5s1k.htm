Set Font ID Box

The SET_FONT_ID box contains the command that sets the ID of the font. This command assigns an integer value to a downloadable font. This integer value is the value that the Universal Printer Driver uses to refer to the font.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Assign Font ID command (ESC*cnD) assigns an ID value to a font. This value can be used to select or delete the font after it is downloaded.

To specify the Assign Font ID command, type the following in the SET_FONT_ID box:

\x1B*c%dD
