Windows 98 Display Driver Interface

The Windows® 98 display driver interface has been modified to support the VIDEOPARAMETERS structure. The interface is through the Control function, with an ESC Escape code of 3077(0x0c05). 

WORD Control(
  LPPDEVICE lpDestDev,
  WORD      wFunction,
  LPSTR     lpInData,
  LPSTR     lpOutData
 );

Parameters

lpDestDev
Address of a PDEVICE structure that specifies the destination device.
wFunction
Escape to be carried out. If set to VIDEO_PARAMETER, it indicates that lpInData is a VIDEOPARAMETERS structure.
lpInData
Address of escape-specific input data. May be a pointer to a VIDEOPARAMETERS structure.
lpOutData
Address of escape-specific output data. When using a VIDEOPARAMETERS structure, this value is NULL.

Return Value

A positive value indicates success, while a negative value indicates an error. Zero can indicate either an unimplemented escape or a general error condition.
