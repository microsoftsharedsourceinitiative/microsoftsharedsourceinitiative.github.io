CRT Connection

When using multiple adapters, display problems resulting from cable connections need to be circumvented by the user for the following reasons:

  • The cable connection cannot be detected by the software.
  • A similar problem may result, not only for multimonitor use, but in setups, including Windows® 95 (display card used is unknown).
