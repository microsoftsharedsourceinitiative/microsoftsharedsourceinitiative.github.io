Win32 API Functions for Multimonitor Support

The following Win32® API functions have been added for multimonitor support:

EnumDisplayMonitors

GetSystemMetrics

GetMonitorInfo

MonitorEnumProc

MonitorFromPoint

MonitorFromRect

MonitorFromWindow

SystemParametersInfo

This section also contains the following existing Win32 function:

Enable

Each physical display device is represented to the application by a monitor handle called an HMONITOR. A physical device has the same HMONITOR value throughout its lifetime, even across changes to display settings, as long as it remains a part of the desktop. When a WM_DISPLAYCHANGE message is broadcast, any HMONITOR can have its settings changed in some way, or can be removed from the desktop and become invalid.

A valid HMONITOR is guaranteed to be non-NULL.
