mciGetDriverData

DWORD WINAPI mciGetDriverData(uUINT DeviceID)

This function returns a pointer to the instance data associated with an MCI device.

  • Returns the driver instance data. On error, it returns zero. Since the driver data might be zero, this cannot be verified by the caller unless the instance data is known to be nonzero (for example, a pointer).
uDeviceID
Specifies the MCI device ID.
