Updating the Minidriver Resolution and Measurement Data

A number of the data structures in a minidriver's data table contain measurement values (that is, graphics resolutions, paper sizes, unprintable-region dimensions, line-feed values, and so on). Each of the values is specified in master units. The Update References control indicates whether or not UniTool should update each measurement in each data structure. If this control is checked and you choose the OK button, UniTool will update all measurements; otherwise UniTool will leave the original measurement values. If some of the measurements do not factor evenly into the new master units and the Update References box is checked, UniTool will issue an error message which identifies the faulty value.
