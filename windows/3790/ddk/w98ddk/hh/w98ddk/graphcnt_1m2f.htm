RECT

typedef struct tagRECT {
    short left;   // x-coordinate of top-left corner
    short top;    // y-coordinate of top-left corner
    short right;  // x-coordinate of bottom-right corner
    short bottom; // y-coordinate of bottom-right corner
} RECT;

Contains the coordinates of the top-left and bottom-right corners of a rectangle.
