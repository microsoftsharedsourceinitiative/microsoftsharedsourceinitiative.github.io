PWMI_FUNCTION_CONTROL

The PWMI_FUNCTION_CONTROL routine is a callback into the driver to enable or disable event generation or data block collection. A device should only expect a single enable when the first event or data consumer enables events or data collection, and a single disable when the last event or data consumer disables events or data collection. Data blocks will only receive collection enable/disable if they were registered as requiring it.

typedef NTSTATUS (*PWMI_FUNCTION_CONTROL) (
    IN PDEVICE_OBJECT DeviceObject,
    IN PIRP Irp,
    IN ULONG GuidIndex,
    IN WMIENABLEDISABLEFUNCTION Function,
    IN BOOLEAN Enable
    );

Parameters

DeviceObject
Device whose data or event block is being enabled or disabled.
Irp
To be provided.
GuidIndex
Index into the list of GUIDs provided when the device registered.
Function
Specifies which functionality is being enabled or disabled.
Enable
If TRUE, then the function is being enabled. Otherwise, it is disabled.

Return Value

Returns a status code.
