VKD_Cancel_Paste

include vkd.inc

mov     ebx, VMHandle
VxDCall VKD_Cancel_Paste

Cancels the paste that was started in the virtual machine by the VKD_Start_Paste service. Uses EAX and Flags.

  • No return value.
VMHandle
Handle of the virtual machine.
