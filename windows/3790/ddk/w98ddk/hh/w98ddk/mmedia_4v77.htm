MCI_SAVE_PARMS

typedef struct {
    DWORD  dwCallback;
    DWORD  lpfilename;
} MCI_SAVE_PARMS;

The MCI_SAVE_PARMS structure contains the information for MCI_SAVE message. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
lpfilename
Specifies a far pointer to the buffer containing a null-terminated string.
