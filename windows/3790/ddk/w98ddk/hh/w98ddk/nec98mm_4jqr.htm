Protected Mode API in Multimonitor Environments

To identify which display driver is making the protected mode (PM) API call in a multimonitor environment, each display adapter identification number must be set when the protected mode API call is made.

Protected Mode API

  Register Meaning
Entry
  AX Function number
  EBX Display adapter ID number
  Other Used by each function.
Exit
  All Used by each function.
