KS_AnalogVideoStandard

The KS_AnalogVideoStandard is used to identify the various aspects of analog broadcast video. A number of the property sets for video use these values to select or describe format capabilities.

The following is a list of available values of type KS_AnalogVideoStandard:

KS_AnalogVideo_None (This is a digital sensor)

KS_AnalogVideo_NTSC_M (75 IRE Setup)

KS_AnalogVideo_NTSC_M_J (For Japan, 0 IRE Setup)
