Corrected NABTS Category

NABTS Category output pins provide a stream of decoded FEC-corrected NABTS (North American Broadcast Teletext Standard) data.

When creating the category PinName, PINNAME_VIDEO_NABTS, the following parameters must be filled out. Note that the subformat is different for corrected NABTS and raw NABTS.

Parameter Value
KS Datarange KS_DATARANGE
KS Dataformat KS_DATAFORMAT
KS MajorFormat KS_DATAFORMAT_TYPE_NABTS
KS SubFormat (typical) KSDATAFORMAT_SUBTYPE_NABTS_FEC
KS Specifier KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
DS majortype MEDIATYPE_VBI
DS formattype FORMAT_VBI
