VKD_Remove_Hot_Key

include vkd.inc

mov     eax, hHotKey
VxDCall VKD_Remove_Hot_Key

Removes a defined hot key. Uses Flags.

  • No return value.
hHotKey
Handle of the hot key to remove. This handle must have been previously created using the VKD_Define_Hot_Key service.
