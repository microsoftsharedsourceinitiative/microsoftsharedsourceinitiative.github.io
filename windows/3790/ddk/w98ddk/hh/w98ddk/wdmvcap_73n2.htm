KSPROPERTY_CAMERACONTROL_PAN

The KSPROPERTY_CAMERACONTROL_PAN read/write property is used to set and retrieve the camera pan setting.

The KSPROPERTY_CAMERACONTROL_S.Value member specifies the pan setting in degrees. Values range from –180 to +180, and the default is zero. Positive values are clockwise from the origin (the camera rotates clockwise when viewed from above), and negative values are counterclockwise from the origin. Note that a particular device may only implement a subset of this range.
