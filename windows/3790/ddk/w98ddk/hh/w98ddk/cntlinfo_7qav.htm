Backspacing

The CM_OCD_BS box contains the printer command or commands which cause a backspace. This command is only required for printers that support device fonts which use byte-paired character-translation tables (CTTs). For more information about CTTs, see Creating a Character-Translation Table.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Backspace command (0x08) moves the cursor in the negative x-direction by the width of the current font's space character.

To specify the Backspace command, type the following in the CM_OCD_BS box:

\x08

HP LaserJet IIP

For the HP LaserJet IIP printer, the Backspace command (0x08) moves the cursor in the negative x-direction by the width of the current font's space character.

To specify the Backspace command, type the following in the CM_OCD_BS box:

\x08
