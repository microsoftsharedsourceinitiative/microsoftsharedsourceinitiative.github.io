Windows 2000 First, Windows 95 Second

If your media access control driver will have both Windows NT/Windows 2000 and Windows 95, it is best to develop and perfect the Windows NT/Windows 2000 version first. Code written for Windows NT/Windows 2000 easily translates to Windows 95, but the reverse is not necessarily true. Maintaining a common source base for both environments is painless and encouraged. Of course, the easiest course of action is to write an NDIS miniport that will run the same binary on both platforms.
