MCI_VD_ESCAPE_PARMS

typedef struct {
    DWORD  dwCallback;
    LPCSTR  lpstrCommand;
} MCI_VD_ESCAPE_PARMS;

The MCI_VD_ESCAPE_PARMS structure contains parameters for the MCI_ESCAPE message for videodisc devices. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
lpstrCommand
Specifies a far pointer to a null-terminated buffer containing the command to send to the device.
