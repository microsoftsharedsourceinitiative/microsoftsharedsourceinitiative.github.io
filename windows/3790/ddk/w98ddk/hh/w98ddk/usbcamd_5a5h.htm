CamNewVideoFrame

CamNewVideoFrame, called at the DISPATCH_LEVEL, allows the minidriver to initialize a new video frame context structure.

VOID
  CamNewVideoFrame(
    PVOID  DeviceContext,
    PVOID  FrameContext
    );

Parameters

DeviceContext
Minidriver device context.
FrameContext
Frame context to be initialized.

Return Values

No value is returned.
