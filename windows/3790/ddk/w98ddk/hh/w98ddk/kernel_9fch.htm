VDMAD_Mask_Channel

include vdmad.inc

mov eax, DMA_Handle
VxDcall VDMAD_Mask_Channel

Masks a channel so that it will not attempt any further DMA transfers. Uses flags

  • This service has no return value.
DMA_Handle
Handle of DMA.
