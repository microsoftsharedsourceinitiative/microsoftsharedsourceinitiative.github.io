Virtual Flat-Frame Buffer Device Reference

The virtual flat-frame buffer device (VFLATD) helps manage the video frame buffer for a display driver. VFLATD virtualizes the buffer for display adapters that use bank-switching techniques to access their video memory by mapping it into a smaller block of memory addresses.
