PROPSETID_VIDCAP_TVAUDIO

The PROPSETID_VIDCAP_TVAUDIO property set is used to control settings unique to audio from television sources. This includes secondary audio program (SAP), stereo or mono selection, and volume levels. These controls are generally found on devices external to the system audio mixer.

GUID PROPSETID_VIDCAP_TVAUDIO
DirectShow Interface IAMTVAudio

The property set includes the following of type KSPROPERTY_VIDCAP_TVAUDIO:

KSPROPERTY_TVAUDIO_CAPS

KSPROPERTY_TVAUDIO_MODE
