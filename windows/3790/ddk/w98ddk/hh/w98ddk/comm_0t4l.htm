_VCOMM_Steal_Port

include vcomm.inc

VxDCall _VCOMM_Steal_Port <PortHandle, VMId>

Steals a port, if possible, from other owners. Before the caller steals a port, it must acquire it by calling _VCOMM_Acquire_Port.

  • Returns TRUE if successful, or zero for failure.
PortHandle
Port handle returned by the original call to _VCOMM_Acquire_Port.
VMId
Handle of the VM that owns the port (-1 means the owner is a VxD).
