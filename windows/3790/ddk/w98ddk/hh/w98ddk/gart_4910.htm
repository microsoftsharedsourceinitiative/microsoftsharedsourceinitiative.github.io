PCIMP_GARTExit

The PCIMP_GARTExit service shuts down the miniport driver.

VOID
  PCIMP_GARTExit(
    PVOID  pHandle, 
    PVOID  pRefData 
    );

Parameters

pHandle
PCI device handle. Should be the same value used in the preceding PCIMP_GARTInit call.
pRefData
Specifies the DWORD that was written by the PCIMP_GARTInit service.

Return Values

This service has no return value.

Comments

In response to this call, the miniport driver frees any memory and releases any resources it previously allocated, and disables the hardware.
