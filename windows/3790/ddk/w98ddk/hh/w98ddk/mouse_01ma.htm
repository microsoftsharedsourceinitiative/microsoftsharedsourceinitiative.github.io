Virtual Cursor Drivers

The virtual mouse device (VMOUSE) draws the mouse cursor for non-system virtual machines (VMs). It supports VGA modes 0 through 7 and 0Dh through 13h. In graphics modes, VMOUSE draws an arrow cursor; in text modes, it draws a block cursor.

A display vendor can enhance VMOUSE's cursor-drawing capabilities for non-system VMs by providing a virtual cursor driver (cursor VxD). For example, a cursor VxD might provide additional cursor styles or add support for other video modes. Any number of cursor VxDs can be loaded at a time, each supporting one or more video modes. This section describes the requirements of a cursor VxD.
