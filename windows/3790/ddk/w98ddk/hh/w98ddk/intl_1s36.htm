ImmUnlockIMC

BOOL WINAPI ImmUnlockIMC(HIMC hIMC)

Decrements the lock count for IMC.

If the lock count of IMC was decremented to zero, the return value is FALSE. Otherwise, it is TRUE.

hIMC
Handle of the input context.
