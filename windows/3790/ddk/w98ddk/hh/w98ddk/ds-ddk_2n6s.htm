DirectSound VxD Exports

In addition to the version service, the DirectSound virtual device driver, (Dsound.vxd), exports two services: a registration service and a deregistration service. DirectSound drivers should call these services in response to the appropriate configuration manager messages. Both services accept a pointer to an IDsDriver interface and a doubleword of flags. No flags are currently defined; the value must be zero in this implementation.

With the exception of the registration and deregistration calls, all communication is initiated by DirectSound by calling the IDsDriver interface or other interfaces that DirectSound obtains through IDsDriver.
