PCIGetVersion

The PCIGetVersion function returns a ULONG representing the operating system version.

ULONG
  PCIGetVersion(
    VOID
    );

Parameters

This function takes no parameters.

Return Values

Currently, there are three possible return values, as shown in the following table.

Value Operating System
0x00000400 Windows® 95
0x00000410 Windows® 98
0x00000500 Windows® 2000
