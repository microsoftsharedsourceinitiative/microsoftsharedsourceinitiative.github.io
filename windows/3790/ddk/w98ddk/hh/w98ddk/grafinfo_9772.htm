End Compression

The End Compression box contains the printer command which disables the compression of graphics data.

Diconix 150 Plus

For the Diconix 150 Plus printer, the End box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Set Compression Mode command (ESC*bnM) enables and disables various types of graphics compression. If n is 0, graphics compression is disabled; if n is 1, run-length encoding is enabled; if n is 2, TIFF encoding is enabled; and if n is 3, delta-row compression is enabled.

To specify the Set Compression Mode command to disable TIFF encoding, type the following in the Begin box:

\x1B*b0M
