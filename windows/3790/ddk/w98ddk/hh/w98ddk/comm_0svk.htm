_VCOMM_Register_Enumerator

include vcomm.inc

VxDcall _VCOMM_Register_Enumerator, <OFFSET32 Enumerator>

Registers an enumerator for the currently active device node. This function is called only by plug-and-play enumerators. Uses the C calling convention.

  • Returns TRUE if successful, FALSE otherwise.
Enumerator
Address of a function that, given a device node, enumerates the devices attached to the corresponding port.

For information about plug-and-play enumerators, see “Configuration Manager.”
