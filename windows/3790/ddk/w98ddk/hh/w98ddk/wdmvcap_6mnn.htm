KSPROPERTY_CAMERACONTROL_IRIS

The KSPROPERTY_CAMERACONTROL_IRIS read/write property is used to set and retrieve the iris setting.

The KSPROPERTY_CAMERACONTROL_S.Value member specifies the iris setting expressed as the fstop * 10. Range of values will be provided in a later release.
