Virtual Shell Device (SHELL)

The virtual shell device (SHELL) provides services that virtual devices use to display messages, resolve contention between devices, post Window messages to applications and VxDs, monitor changes to virtual machine properties, and carry out two-way communication with Windows-based applications or DLLs. This section describes the interfaces of the virtual shell device.
