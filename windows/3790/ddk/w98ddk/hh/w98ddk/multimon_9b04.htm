Implementing Multiple Monitor Support

Multiple monitor support is a set of related features that support the use of more than one display device at a time. Many names have been used to describe this capability, including multiple displays, multiple adapters, and multimonitor. This document uses the name multimonitor. Support for multiple monitors is provided under the Microsoft® Windows® 98/Me and Windows® 2000 and later operating systems. Additional application programming interfaces and device driver interfaces are provided under Microsoft DirectX® 5.0. To support multiple monitors, see the Windows DDK available on the Windows DDK Web site.

This section provides information on the following topics:

  • Implementing multimonitor support for Microsoft DirectX programming.
  • Implementing multimonitor support in Microsoft DirectDraw® and Direct3D® Hardware Abstraction Layer(HAL) drivers.
  • Implementing multimonitor support for Windows and Windows NT/Windows 2000 display drivers.
