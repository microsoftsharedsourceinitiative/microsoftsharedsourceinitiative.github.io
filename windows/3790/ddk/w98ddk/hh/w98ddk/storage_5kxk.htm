Partitionable Removable Media

Windows 95 supports existing removable media with MS-DOS compatible partitions. These partitions are usually created by third party supplied FDISK utilities.

Windows 95 FDISK will continue to create partitions on a removable media as long as these devices are INT 13 based. This is the current behavior of FDISK and will not be changed.
