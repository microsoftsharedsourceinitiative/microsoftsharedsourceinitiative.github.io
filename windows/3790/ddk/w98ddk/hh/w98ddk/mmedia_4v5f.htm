MCI_LOAD_PARMS

typedef struct {
    DWORD  dwCallback;
    LPCSTR  lpfilename;
} MCI_LOAD_PARMS;

The MCI_LOAD_PARMS structure contains the information for MCI_LOAD message. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
lpfilename
Specifies a far pointer to a null-terminated string containing the filename of the device element to load.
