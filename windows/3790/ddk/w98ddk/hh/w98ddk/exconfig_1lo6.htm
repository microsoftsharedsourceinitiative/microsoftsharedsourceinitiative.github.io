Connection Configuration

The ISDN connection properties consists of one standard Connection page. Optional vendor-supplied pages can be added. These pages should allow the user to change or view any attributes that pertain to the specific connection in context. Alternatively, settings that are specific to the physical line (that do not change across connections) should be reserved for the Line properties. For example, SPIDs and phone numbers do not change across connections, therefore these settings would be reserved for the Line properties.
