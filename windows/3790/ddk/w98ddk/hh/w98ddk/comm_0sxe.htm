_VCOMM_Enumerate_DevNodes

include vcomm.inc

VxDCall _VCOMM_Enumerate_DevNodes
or      eax, eax
jz      NoMoreDevNodes:
mov     esi, eax
VMMCall List_Get_First
jz      NoMoreDevNodes

DevNodeLoop:
mov     ecx, [eax]
;
; Do something with devnode in ecx, preserve eax, esi
;
VMMCall List_Get_Next
jnz     DevNodeLoop

NoMoreDevNodes:

Returns the handle of the Port_instance list. A VCOMM client can use this list to find out all the current VCOMM device nodes. The caller must not delete or modify any element of this list.

  • Returns the list handle, or zero if there is an error.
