Face Name

The Face Name control contains an ASCII string that specifies the name of the font. If a printer supports multiple fonts with the same font name and point size but different pitches, the string should also contain the pitch value. Pitch values should be expressed in terms of characters-per-inch (cpi). The font name in a minidriver must be unique.

Diconix 150 Plus

For the Diconix 150 Plus printer, this string is Roman(12cpi).
