DRV_QUERYDRVENTRY

Queries the driver-entry text in the registry for this device (typically the name of the Ring3 driver). If this is not a Plug and Play device, this is the name of the key in SYSTEM.INI that the driver is registered under (wave1, midi, aux4, mixer5).

The system intercepts this message and returns the appropriate value without sending the message to the device driver. Most drivers will not use this message.

  • Returns MMSYSERR_NOERROR if successful; otherwise, returns an error value.
dwParam1
Pointer to a buffer to fill with the string.
dwParam2
Buffer size, in bytes.
