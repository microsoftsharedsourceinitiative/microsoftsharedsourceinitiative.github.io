Identifying the First Character

The First Char control in the CTT dialog box contains a value that identifies the first character in the character-translation table. The extended characters used by HP LaserJet IIP printer vary depending upon the individual font being used. In the previous illustration, a character-translation table for a font using the ECMA-94 character set is shown. The first character in the translation table is 128. Characters below this value do not use the character-translation table, and are sent directly to the printer.
