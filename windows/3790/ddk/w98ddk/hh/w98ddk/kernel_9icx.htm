VPICD_Begin_Inst_Page_Swap

VxDcall VPICD_Begin_Inst_Page_Swap

Informs VPICD to begin masking interrupts that must be held off until instance data copies are complete. This service should be called only by the virtual machine manger's instance page-swap code. Uses Flags.

  • No return value.
