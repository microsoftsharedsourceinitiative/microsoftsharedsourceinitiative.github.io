KSPROPERTY_TUNER_STATUS

The KSPROPERTY_TUNER_STATUS read-only property is used to retrieve information about the tuning process, including the current frequency. 

typedef struct {
  KSPROPERTY  Property;
  ULONG  CurrentFrequency;
  ULONG  PLLOffset;
  ULONG  SignalStrength;
  ULONG  Busy;
} KSPROPERTY_TUNER_STATUS_S, *PKSPROPERTY_TVTUNER_STATUS_S;

Members

Property
Specifies the property used.
CurrentFrequency
Specifies the current frequency in Hertz (Hz).
PLLOffset
Specifies the PLL offset in multiples of the tuning granularity. This is used if the KS_TUNER_STRATEGY is KS_TUNER_STRATEGY_PLL.
SignalStrength
Specifies the amplitude of the signal. This is used if the KS_TUNER_STRATEGY is KS_TUNER_STRATEGY_SIGNAL_STRENGTH.
