Loading MiniVDD

Loading the Primary MiniVDD

The primary loading begins after verification of the multimonitor support level. This is given as a parameter during the miniVDD registration. The handling by baseVDD differs, depending on the following support levels.

Support Level Description
Not supported (0). Load as a standard driver (secondary cannot be used as a desktop).
Exclusive for primary (1) or dual usage (3). OK.
Other than (0), (1), (3). Loading failure (fallback mode).

Loading the Secondary miniVDD

When a driver passes all of the following, it is loaded as a secondary driver.

Note:  If a problem is detected during one of the verification processes, loading is terminated and the monitor is set as a desktop monitor. It displays “!” in the device manager.

Verification Process Result Meaning
DEFAULT\Drv2 Check
  Described OK
  Not described Driver is not loaded. ("!" is displayed.)
Multimonitor support condition check at miniVDD registration
  Not supported (0) or exclusive for primary (1) Driver is not loaded. ("!" is displayed.)
  Exclusive for secondary (2) or
