Paper Type

The Paper Size and Paper Source dialog boxes contain an fPaperType button that displays an fPaperType dialog box. This dialog box is used to match paper sizes (legal, letter, and so on) to paper sources (trays, bins, or other paper-feed mechanisms). The corresponding bit must be set in both the PAPER SIZE data structure and the PAPER SOURCE data structure for the combination to be valid. The name assigned to each bit is provided for convenience.

The following list describes the four controls in this dialog box.

Control Description
PS_T_STD Specifies that this paper size corresponds to the standard paper source.
PS_T_LARGE Specifies that this paper size corresponds to the large paper source.
PS_T_ENV Specifies that this paper size corresponds to the envelope paper source.
PS_T_LRGENV Specifies that this paper size corresponds to the large-envelope paper source.

Diconix 150 Plus

For the Diconix 150 Plus printer, PS_T_STD is selected for the 8.5 by 11-inch paper size. This control is also selected for the tractor-feed and sheet-feed paper sources in the Paper Source dialog box.

HP LaserJet IIP

For the HP LaserJet IIP printer, PS_T_STD is selected for the 8.5 by 11-inch paper size. This control is also selected for the upper bin, lower bin, and manual feed paper sources in the Paper Source dialog box.
