The Windows System Debugger

Microsoft® Windows® System Debugger (debugger.exe) is used to test and debug Windows 9x applications, dynamic-link libraries (DLL), and virtual device drivers (VxD) running with the Windows operating system. You can use System Debugger commands to inspect and manipulate memory and registers, control the execution of code, and perform other debugging operations. This document describes the debugger, explains how to start and use the debugger, and provides detailed information about the debugger commands.

Note that previous versions of the system debugger were called wdeb386.exe and wdeb98.exe, and that some versions of the documentation refer to this name instead of debugger.exe.
