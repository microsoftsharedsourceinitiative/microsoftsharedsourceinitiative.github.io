Module Name
Name of the module with this handle value. This corresponds to the memory in this area.
Handle
16-bit value containing the handle to the owner of this object. We call ModuleFindHandle or TaskFindHandle with this value to get to the owner of the object.
Memory Block Size
16-bit value indicating the size, in bytes, of this memory object.
Data Type
Ignore. This value is used to determine the size of the fields.
Block Size:
Total size of the block for this module/handle. Note that module name to handle correspondence need not be unique.
Unknown
Item is GT_UNKNOWN.
Dgroup
Total size of Dgroup elements for this module.
Code
Total size of code for this element.
Data
Total data element size.
Task
Task entry for this module.
Resource
Resource for this module.
Free
Free block.
Module
A module.

Internal, Burger Master, and Reserved are internal and are measured for historical reasons.

Note  If the Memory Type member is not Code or Resource, Data Type is zero.

If the Memory Type is Code, Data, or DGroup, Data Type contains the segment number for the code.

If Memory Type is GT_RESOURCE, Data Type specifies the type of resource.
