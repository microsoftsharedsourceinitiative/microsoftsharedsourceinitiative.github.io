MonitorFromRect

HMONITOR
  WINUSERAPI MonitorFromRect(
  LPCRECT  lprc, 
  DWORD  dwFlags
    );

Parameters

lprc
To be provided.
dwFlags
To be provided.

Return Values

Returns the monitor that intersects lprc. If no monitor intersects lprc, the return value depends upon the dwFlags field as in MonitorFromPoint.
