Linefeeds

The CM_OCD_LF box contains the printer command or commands which cause a linefeed.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Line Feed command (0x0A) causes a linefeed.

To specify the linefeed command, type the following in the CM_OCD_LF box:

\x0A




HP LaserJet IIP



For the HP LaserJet IIP printer, the Line Feed command (0x0A) causes a linefeed.



To specify the Line Feed command, type the following in the CM_OCD_LF box:



\x0A
