IMC_GETCOMPOSITIONFONT

An application sends the IMC_GETCOMPOSITIONFONT message to the IME window retrieve the font to use for displaying intermediate characters in the composition window. The UI window does not receive this message.

If successful, the return value is zero. Otherwise, it is nonzero.

lParam
Address of a LOGFONT structure that defines the font.
