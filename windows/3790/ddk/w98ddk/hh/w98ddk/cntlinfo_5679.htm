Line Spacing Increment Value

The CM_OCD_YM_LINESPACING box contains the printer command or commands which set the line spacing increment value for the printer. This is the amount of increment that occurs at each linefeed. For some printers, the corresponding Units dialog box contains values by which the line spacing increment value command arguments must be multiplied or divided. The Units dialog box also contains values which must be added to command arguments, maximum values for command arguments, and minimum values for command arguments.

Diconix 150 Plus

For the Diconix 150 Plus printer, the Line Spacing command (ESC3n) is required to set the value which the printer will use for subsequent linefeed operations. The variable n specifies a line spacing value in 1/288 inch increments.

To specify the Line Spacing command, type the following in the CM_OCD_YM_LINESPACING box:

\x1B3%c

HP LaserJet IIP

The HP LaserJet IIP printer does not support a command which sets the line-spacing increment value.
