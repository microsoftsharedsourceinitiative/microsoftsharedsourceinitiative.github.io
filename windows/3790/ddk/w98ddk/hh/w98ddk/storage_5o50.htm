Ontrack Computer Systems

This section describes two INT 13h API supported by XBIOS. These commands can be used by a device driver to obtain operating parameters from XBIOS. This will allow the device driver to take over XBIOS and retain full compatibility with it. This section was originally written with FASTDISK in mind but applies to any driver which needs compatibility with Ontrack Disk Manager and Ontrack Drive Rocket software.
