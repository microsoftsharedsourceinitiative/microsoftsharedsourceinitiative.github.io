Supported Paper Sizes

After you specify control information for your printer in the PAGECONTROL and CURSORMOVE dialog boxes, you should specify paper-size information. The Paper Sizes dialog box contains controls which you use to specify printer data such as whether a particular paper size is driver defined or predefined, the paper dimensions, the cursor origin, the unprintable-region dimensions, and so on. From the Printer Data menu, choose the Paper Info command, and then the Size command.

Diconix 150 Plus

For the Diconix 150 Plus printer, there are two supported paper sizes. The first is 8.5 x 11 and the second is defined by the user.

HP LaserJet IIP

For the HP LaserJet IIP printer, there are four supported paper sizes and four supported envelope sizes. The following list identifies each:

  • 8.5 x 11 inches
  • Legal (8.5 x 14 inches)
  • Executive (7.25 x 10.5 inches)
  • A4 (210 x 297 millimeters)
  • Envelope #10 (4.125 x 9 inches)
  • Envelope DL (110 x 220 millimeters)
  • Envelope C5 (162 x 229 millimeters)
  • Envelope C3 (324 x 458 millimeters)

The examples in the following sections are based on the first paper size initialized for each minidriver. In both instances, this is for 8.5 x 11 paper.
