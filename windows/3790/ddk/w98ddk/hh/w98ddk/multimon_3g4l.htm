SET_ADAPTER_POWER_STATE

SET_ADAPTER_POWER_STATE sets the adapter's power state to a new state.

Entry Meaning
ESP+4 Device node.
ESP+8 Power state to set. One of:

CM_POWERSTATE_D0
CM_POWERSTATE_D1
CM_POWERSTATE_D2
CM_POWERSTATE_D3

Exit Meaning
EAX CR_DEFAULT

Routine did nothing.

CR_SUCCESS

Routine set the appropriate power state for the
adapter/display chip set.
