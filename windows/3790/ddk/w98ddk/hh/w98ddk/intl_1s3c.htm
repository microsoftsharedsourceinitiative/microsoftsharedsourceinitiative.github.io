ImmReSizeIMCC

HIMCC WINAPI ImmReSizeIMCC(HMICC hIMCC, DWORD dwSize)

Changes the size of the IMC component.

If successful, the return value is the new HIMCC. Otherwise, it is zero.

hIMCC
Handle of IMC component.
dwSize
New size of IMC component.
