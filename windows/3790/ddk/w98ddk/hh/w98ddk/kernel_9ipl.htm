Building Dynamically Loadable VxDs

Building a dynamically loadable VxD requires a change to the build procedure. The module-definition file should specify that the VxD being linked is a dynamically loadable VxD. This is accomplished by adding the keyword DYNAMIC at the end of the LIBRARY line, following the device's name.
