KSPROPERTY_VIDEOCONTROL_FRAME_RATES

The KSPROPERTY_VIDEOCONTROL_FRAME_RATES read-only property is used to enumerate the available frame rates. The available frame rates are returned in a structure of type KSMULTIPLE_ITEM with individual values given in units of 100ns. This is an optional property.
