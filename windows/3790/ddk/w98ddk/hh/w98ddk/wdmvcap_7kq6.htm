PROPSETID_VIDCAP_VIDEOCOMPRESSION

The PROPSETID_VIDCAP_VIDEOCOMPRESSION properties are used to get and set video compression settings.

GUID PROPSETID_VIDCAP_VIDEOCOMPRESSION
DirectShow Interface IAMVideoCompression

The property set includes the following of type KSPROPERTY_VIDCAP_VIDEOCOMPRESSION:

KSPROPERTY_VIDEOCOMPRESSION_GETINFO (Read Only)

KSPROPERTY_VIDEOCOMPRESSION_KEYFRAME_RATE (Read/Write)

KSPROPERTY_VIDEOCOMPRESSION_PFRAMES_PER_KEYFRAME (Read/Write)

KSPROPERTY_VIDEOCOMPRESSION_QUALITY (Read/Write)

KSPROPERTY_VIDEOCOMPRESSION_OVERRIDE_KEYFRAME (Read/Write)

KSPROPERTY_VIDEOCOMPRESSION_OVERRIDE_FRAME_SIZE (Read/Write)

All the properties that can write settings (all properties except KSPROPERTY_VIDEOCOMPRESSION_GETINFO), use the following structure:

typedef struct {
  KSPROPERTY  roperty;
  LONG  Value;   
} KSPROPERTY_VIDEOCOMPRESSION_S, *PKSPROPERTY_VIDEOCOMPRESSION_S;

Members

Property
Specifies the property used.
Value
Specifies values unique to that property. See the individual property for information on the range and default value for this member.
