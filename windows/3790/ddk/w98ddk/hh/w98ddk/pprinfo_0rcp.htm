Predefined Paper Quality

The Paper Quality Name control which appears adjacent to the Predefined option displays a list box which contains two strings that identify the predefined paper qualities. The following contains these options and a brief description of their meaning.

Control Description
Paper Specifies paper as the output medium (as opposed to transparency).
Transparency Specifies a transparency as the output medium (as opposed to paper).
