Video Stream Extended Headers

Each of the output streams from a video capture driver utilizes an extended header to provide auxiliary information about the stream and current frame contents. For example, image stream headers provide information about the current frame number, number of dropped frames, and field polarity flags. As each frame is completed, the driver must fill in the extended header with the auxiliary information about the frame captured.

Drivers must indicate to the stream class their ability to provide this additional information by setting the HW_STREAM_OBJECT.StreamHeaderMediaSpecific size according to the following table.
