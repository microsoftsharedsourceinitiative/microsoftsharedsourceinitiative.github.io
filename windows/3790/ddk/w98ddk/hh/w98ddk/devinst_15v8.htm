DiInstallDevice

Install the device specified by lpdi. 

RETERR
  DiInstallDevice(
    LPDEVICE_INFO  lpdi
    );

Return Value

Returns OK if successful, otherwise an ERR_DI_xxx error code will be returned. lpdi->Flags will be set to indicate if the system needs to be rebooted or restarted for the device to be started.

Parameters

lpdi
Pointer to a DEVICE_INFO structure for the device being installed. If lpdi->lpSelectedDriver is NULL this function will install a NULL driver for the device, otherwise it installs the driver specified in the driver node.

Comments

This API is the default handler for the DIF_INSTALLDEVICE class installer function. This API will install a device by GenInstalling the install section specified by the lpid->lpSelectedDriver DRIVER_NODE. If Config Manager is running, this API will attempt to dynamically start the device.
