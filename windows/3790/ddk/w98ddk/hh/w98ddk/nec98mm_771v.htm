GET_ADAPTER_POWER_STATE_CAPS

The GET_ADAPTER_POWER_STATE_CAPS function obtains the power saver mode supported by the display adapter. 

ULONG get_adapter_power_state_caps(
  DEVNODE dn
  );

Parameters

dn
Device node of the display adapter.

Return Value

Returns the supported mode.
(0:supported, 1:not supported)

bit 0: STAND BY

bit 1: SUSPEND

bit 2: OFF

bit 3: REDUCED ON

bit 4-bit 31:Reserved
