Supporting Multimonitor in a Windows Display Driver

For Windows® 98, a new driver must be created to support multimonitor features. This driver must be a DIBENG driver and it must adhere to the following guidelines:

  • It must recognize a new parameter for Enable.
  • It must not hook or issue Int 2Fh calls.
  • It must be unloadable.
  • It must not touch VGA registers, except with help from the main VDD.
