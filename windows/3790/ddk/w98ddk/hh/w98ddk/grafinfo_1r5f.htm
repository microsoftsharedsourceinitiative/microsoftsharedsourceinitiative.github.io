Specifying Rectangle-Fill Capabilities

The RECTFILL dialog box contains controls which specify rectangle-fill data and commands for printers with rectangle-fill capabilities. Typically, the only printers which provide these capabilities are printers that support a page-control language such as the Hewlett-Packard PCL. The data supplied in the RECTFILL dialog box describes the printer's minimum and maximum gray levels, the cursor position at the conclusion of a rectangle-fill operation, and so on. The commands supplied in the RECTFILL dialog box specify the rectangle's dimensions and fill pattern.
