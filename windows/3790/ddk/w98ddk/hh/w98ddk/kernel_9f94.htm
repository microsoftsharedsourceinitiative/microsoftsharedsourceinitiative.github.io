DMA Devices

This section describes the interfaces of the virtual direct memory access device (VDMAD). Virtual devices that support hardware that requires direct memory access (DMA) transfers should use the services provided by the virtual DMA device to carry out these transfers.

The virtual DMA device virtualizes direct memory access devices and provides services that other virtual devices use to carry data transfers using DMA channels.
