SET_VDD_BANK (Function 2)

Call With

EBX: Contains the Windows VM Handle.

EBP: Contains the Windows VM's Client Registers.

Return Values

Save anything that you use.

Remarks

The mini-VDD sets to the bank determined when GET_VDD_BANK was called in preparation for 4 plane virtualization. See the sample mini-VDDs to determine how to save the current bank so that it can be restored at when RESET_BANK is called.
