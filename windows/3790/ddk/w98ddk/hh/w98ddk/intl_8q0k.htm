GDI API

The system default TrueType EUDC and typeface-specific TrueType EUDC fonts are hidden fonts from applications. Thus, there is no way to handle the EUDC fonts in the applications, except the following special GDI functions.

Special applications like EUDC editors and Control Panel must call these special functions whenever the applications directly add, modify, or delete the EUDC fonts in real time without any help fromGDI.
