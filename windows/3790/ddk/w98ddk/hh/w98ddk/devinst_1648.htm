RemOldPath Item

RemOldPath=<ldid>[,<ldid>]

Removes the path associated with the given LDID from the path command.

ldid
Can be any of the predefined LDID values or a new value defined in the INF. For a definition of all the predefined LDID values, see the Reference topic about the DestinationDirs section.

For example, if the user installs the new version of Windows into c:\newwin and has an old copy of Windows in c:\windows, the following INF file item removes c:\windows from the path environmental variable: 

RemOldPath=10
