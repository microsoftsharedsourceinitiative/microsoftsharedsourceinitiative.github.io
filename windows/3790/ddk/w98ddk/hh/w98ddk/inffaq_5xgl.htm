FAQ for Installing Your Driver on Windows Millennium

  1. Q: Where can I get more information about creating INFs for Windows® Millennium?

    A: See the Windows Me INF Guide. Also refer to the Windows 95 documentation included with this DDK.

  2. Q: I want to disable version checking and always copy my file. How can I do this?

    A: This is set by a flag in a CopyFile section. See Copy Files (CopyFiles).

  3. Q: I want to copy a file only if it does not exist on a user's system. How can I do this?

    A: Please see Question 2.

  4. Q: How do I avoid overwriting critical system files?

    A: Microsoft has a new feature, System File Protection (SFP) specifically designed to prevent overwriting shared system files. See Protection for System Files.

  5. Q: What files are protected under SFP?

    A: See Protection for System Files.

  6. Q: How do I disable System File Protection.

    A: System File Protection (SFP) cannot be disabled.

  7. Q: How do I write an INF file to install a WDM Driver on both Windows® 2000 and Windows Me?

    A: Please see the example in Writing a Cross Platform INF to Install USB Printer Drivers.

  8. Q: Where can I get a list of install directories or Logical Disk Identifiers (LDIDs)?

    A: Check the DestinatonDirs section in the Windows Me INF Guide.

  9. Q: How can I get my drivers signed?

    A: See Driver Signing
