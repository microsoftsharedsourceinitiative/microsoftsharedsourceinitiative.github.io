Set_DevNode_PowerStateTable

The Power State Table maps system states to device states for a given device. The ACPI driver calls the Set_DevNode_PowerStateTable service to set the Power State Table for a device. 

CONFIGRET
  CONFIGMG_Set_DevNode_PowerStateTable(
    DEVNODE  dnDevNode, 
    PVOID  pPowerTable, 
    ULONG  ulFlags
    );

Parameters

dnDevNode
Handle to the device node from which to set the Power State Table.
pPowerTable
Pointer to the Power State Table for the device represented by the specified device node.
ulFlags
Reserved. Currently must be zero.
