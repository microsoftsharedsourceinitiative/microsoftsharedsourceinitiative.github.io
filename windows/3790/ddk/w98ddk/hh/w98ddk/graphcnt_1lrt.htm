CreateDIBitmap

WORD CreateDIBitmap(void);

Creates a DIB.

No return value.

The export ordinal for this function is 20.

A graphics-device driver must export the CreateDIBitmap function if the RC_DI_BITMAP value is set in the dpRaster member of the driver's GDIINFO structure.

Because GDI does not currently use CreateDIBitmap, implement the function as a stub function that carries out no action and always returns zero.
