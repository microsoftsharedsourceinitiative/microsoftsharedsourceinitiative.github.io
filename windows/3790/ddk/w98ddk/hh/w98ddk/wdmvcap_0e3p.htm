KSPROPERTY_CAMERACONTROL_EXPOSURE

The KSPROPERTY_CAMERACONTROL_EXPOSURE read/write property is used to set and retrieve the exposure setting.

The KSPROPERTY_CAMERACONTROL_S.Value member specifies the exposure setting in seconds using the following formula. For values less than zero, the exposure time is 1/2n seconds. For positive values and zero, the exposure time is 2n seconds. Note that a particular device may only implement a subset of this range.

For example, a device advertises a range of –7 to 1, with stepping set to 1. The corresponding exposure times are:

Value –7 –6 –5 –4 –3 –2 –1 0 1
Seconds 1/128 1/64 1/32 1/16 1/8 ¼ ½ 1 2
