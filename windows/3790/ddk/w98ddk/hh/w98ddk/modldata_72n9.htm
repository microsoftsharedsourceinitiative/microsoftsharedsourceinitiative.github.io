Maximum Number of Fonts per Page

The sMaxFontsPage box contains a value which specifies the maximum number of device fonts that can appear on a single page. If there is no such limit for a printer, this value should be -1.

Diconix 150 Plus

For the Diconix printer, there is no limit to the number of device fonts allowed on a page; therefore, the sMaxFontsPage box contains the value -1.

HP LaserJet IIP

For the HP LaserJet IIP printer, there is no limit to the number of device fonts allowed on a page; therefore, the sMaxFontPage box contains the value -1.
