KSPROPERTY_VIDEOCOMPRESSION_OVERRIDE_FRAME_SIZE

The KSPROPERTY_VIDEOCOMPRESSION_OVERRIDE_FRAME_SIZE read/write property is used to temporarily override the frame rate.

The KSPROPERTY_VIDEOCOMPRESSION_S.Value member specifies the overriding frame size. This property is not supported by capture drivers.
