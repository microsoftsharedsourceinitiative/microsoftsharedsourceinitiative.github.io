Persistent Connections

Persistent connections are connections remembered by Windows and are restored by Windows when (depending upon the network) the user starts up or logs in to a network.

Persistent connections are handled entirely by Windows to present a single, consistent model to the user. Providers should not maintain (explicitly remember or restore) persistent connections.
