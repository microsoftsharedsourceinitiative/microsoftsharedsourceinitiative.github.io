Predefined Translation Tables

The Universal Printer Driver will provide support for some of the common code pages and symbol sets, and allow drivers to support less common ones.

The list of character-translation tables currently supported by the Universal Printer Driver is as follows:

#define CTT_CP437 -1 \\ IBM Code Page 437.

#define CTT_CP850 -2 \\ IBM Code Page 850.
