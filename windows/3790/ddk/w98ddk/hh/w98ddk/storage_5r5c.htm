IOS_Find_Int13_Drive

mov al, UnitNumber  ; int 13h unit number
VxDCall IOS_Find_Int13_Drive
jc error
mov [BDD], edi   ; address of BDD

Returns the BlockDev device descriptor (BDD) for the given INT 13h drive.

  • Returns the address of a BlockDev_Device_Descriptor structure.
