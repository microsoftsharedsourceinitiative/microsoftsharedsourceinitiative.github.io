Specifying Whether the Paper Size Is Predefined or Driver-Defined

The Predefined and Driver Defined buttons specify whether the data that describes the name of a single paper size is selected from the predefined names supplied by the Universal Printer Driver or by the minidriver. This information will be presented to the user (prior to Microsoft® Windows® 3.1, applications had no way of obtaining the name of a paper size other than via its predefined ID value).

Diconix 150 Plus

For the Diconix 150 Plus, the Predefined control is set.

HP LaserJet IIP

For the HP LaserJet IIP, the Predefined control is set.
