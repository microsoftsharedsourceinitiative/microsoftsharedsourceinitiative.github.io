Videoport VBI Category

Videoport VBI Category streams are used to transfer VBI streams from an analog decoder directly to DirectDraw surfaces through a hardware connection.

When creating the category PinName, PINNAME_VIDEO_VIDEOPORT_VBI, the following parameters must be filled out.

Parameter Value
KS Datarange KSDATARANGE
KS Dataformat KSDATAFORMAT
KS MajorFormat KSDATAFORMAT_TYPE_VIDEO
KS SubFormat (typical) KSDATAFORMAT_SUBTYPE_VPVBIVideo
KS Specifier KSDATAFORMAT_SPECIFIER_NONE
Extended Header Size 0
Required Property Sets KSPROPSETID_VPVBIConfig
Required Event Sets KSEVENTSETID_VPVBINotify
DS majortype None
DS formattype None
