Floating Point Registers

Windows Me does not preserve floating-point registers when switching between kernel and user mode.

Use of MMX or floating point in kernel mode corrupts the register state of applications. A driver that modifies floating-point registers will therefore negatively affect applications that operate on floating point numbers. A driver must verify that it is running on Windows 2000 before executing MMX or FP instructions.
