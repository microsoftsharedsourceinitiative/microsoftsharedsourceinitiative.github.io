Absolute-Vertical Cursor Movement

The CM_OCD_YM_ABS box contains the printer command or commands which move the cursor vertically with respect to the page origin. For most printers, the corresponding Units dialog box will contain values that the Universal Printer Driver will use to calculate the argument for the absolute-vertical cursor-movement command.

Diconix 150 Plus

The Diconix 150 Plus printer does not support an absolute-vertical cursor-movement command; therefore, the CM_OCD_YM_ABS box is empty.

HP LaserJet IIP

For the HP LaserJet IIP printer, the Vertical Cursor Positioning command (ESC*pnY) moves the cursor to a new position with respect to the page origin. The variable n specifies the distance of movement in 1/300 inch units.

To specify the absolute Vertical Cursor Positioning command, type the following in the CM_OCD_YM_ABS box:

\x1B*p%dY
