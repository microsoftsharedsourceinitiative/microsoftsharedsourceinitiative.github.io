CONFIG_SET_POWERSTATE_D1, CONFIG_SET_POWERSTATE_D2, and CONFIG_SET_POWERSTATE_D3

The configuration manager sends the CONFIG_SET_POWERSTATE_D1, CONFIG_SET_POWERSTATE_D2, and CONFIG_SET_POWERSTATE_D3 messages to a device driver to tell it to switch to a low power state. After the device driver completes this message, the configuration manager sends this message to the device driver's parent drivers so they can signal the physical hardware to go into the specified low power state and possibly remove power from the device.

The configuration manager never sends CONFIG_SET_POWERSTATE_Dx for a power state not supported according to the device's current capabilities mask. Instead, the configuration manager calls the driver with the next highest power state supported in the capabilities mask.
