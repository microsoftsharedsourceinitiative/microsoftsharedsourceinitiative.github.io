DRV_PenPlayStop

Forces the pen driver out of playback mode.

  • Always returns DRV_SUCCESS.
lParam1
Should be set to zero.
lParam2
Should be set to zero.

An application must send this message on completion of the playback or on playback failure. This resets the playback mode. A new playback cannot be started without resetting the playback mode.
