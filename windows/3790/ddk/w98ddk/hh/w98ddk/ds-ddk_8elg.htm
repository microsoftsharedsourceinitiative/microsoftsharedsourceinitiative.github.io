IDsDriverPropertySet

The IDsDriverPropertySet methods allow clients to query for support of a particular property, retrieve the current value of a property, and set the current value of a property.

The IDsDriverPropertySet interface contains the following methods:

  • AddRef
  • Get
  • QueryInterface
  • QuerySupport
  • Release
  • Set
