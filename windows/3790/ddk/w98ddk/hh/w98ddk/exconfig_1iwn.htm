NDIS WAN Miniport Driver Configuration and Registry Information for Windows 95

In the current implementation of the NDIS WAN and WAN TAPI Service Provider (WANTSP) components, the only avenue for adapter configuration is through the Advanced page of the WAN (ISDN) adapter, accessed through the Network control panel. ISDN adapters are currently treated like network cards; this documentation describes how to add support for configuration property pages accessible through TAPI for WAN devices, specifically, ISDN adapters.
