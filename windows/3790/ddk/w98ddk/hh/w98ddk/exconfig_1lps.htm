Line Page

The Line page includes settings that relate to the physical line. These settings should be independent of any specific connection information, such as channel speed or dialing time-outs. Changing these settings will probably require a system restart.

Friendly name and icon.
The icon is to be determined. The picture currently uses an internal adapter as a placeholder. The friendly name of the line is derived from the physical line number (base-zero) and converted to a friendlier term. For example, Line 0 would be displayed as 1st Line.
Switch Type.
This combination box contains the list of switches supported by the adapter. These switches include the following:

Channel settings.
This list view contains a list of all the B-channels for the physical line. There are three columns in this list view:
Channel:
Indicates the channel number (zero based).
Phone Number:
Specifies the phone number of the channel. This is displayed in TAPI’s canonical address form.
SPID:
Indicates the SPID of the channel.
Modify.
The button invokes the Channel Settings dialog. This allows the user to assign a phone number and SPID to the selected channel. If no channel is selected, this button is disabled.
