IFSMgr_Win32ToDosTime

IFSMgr_Win32ToDosTime(
 _FILETIME Win32Time
 )

This service converts the time passed in Win32 format in UTC to DOS time format in local time.

Win32Time
Win32 time filled in the _FILETIME structure.
  • Returns the time in DOS format filled in the dostime_t structure.
