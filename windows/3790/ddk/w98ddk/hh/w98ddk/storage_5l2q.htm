IOS Protected-Mode Functions

IOS protected-mode functions are provided by IOS_PM_API_Handler, the interface routine for use with the Windows protected mode application interface. Clients retrieve the address of this function by using the Get PM API Vector Interrupt 2Fh, Function 1684h which returns a Ring 3 callable vector.
