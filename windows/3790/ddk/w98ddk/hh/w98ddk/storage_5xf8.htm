IFSMgr_GetVersion

An FSD that wants to register with the IFS manager must call this service to verify that it is running with the correct version of the IFS manager. This function is not C callable.

Nothing.
Carry flag clear if no error, set if IFS manager is not loaded.
[EAX] = Version number of the IFS manager if no error.
Register Usage

EAX, Flags.
