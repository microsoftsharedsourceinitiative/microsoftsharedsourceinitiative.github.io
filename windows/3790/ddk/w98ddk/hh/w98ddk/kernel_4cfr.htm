Virtual Power Device

The virtual power device (VPOWERD) provides power management support for the Windows 95 operating system. It works in conjunction with the system BIOS or other power management software to provide a system-independent interface to power management services. This means virtual devices and device drivers written for Windows 95 use VPOWERD services for power management rather than access the power management software directly.

VPOWERD supports power management software written to the Advanced Power Management (APM) specifications, versions 1.0 and 1.1. VPOWERD does not support the UPS functionality provided by APM 1.1.
