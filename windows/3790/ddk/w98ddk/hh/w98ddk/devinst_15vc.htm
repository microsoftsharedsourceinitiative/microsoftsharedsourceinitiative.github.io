DiSelectDevice

This is the default handle for the DIF_SELECTDEVICE class installer message. 

RETERR
  DiSelectDevice(
    LPDEVICE_INFO  lpdi
    );

Return Value

OK if a driver is selected successfully, otherwise one of the ERR_DI_ error codes will be returned

Parameters

lpdi
A device info structure for the device that is having a driver selected.

Comments

This routine will handle the UI for allowing a user to select a driver for the device defined by the passed in lpdi. By using the Flags field of the lpdi structure, the caller can specify special handling of the UI, such as allowing selecting from OEM disks.
