Installing the Miniport Driver

The PCI class driver loads GART miniport drivers. In order for the class driver to recognize and load a miniport driver, the following entries must be present in the registry.

Registry Key Description
HKLM,System\CurrentControlSet\
Services\VxD\PCI\GARTMiniports\
xxxxyyyy\Name		 Name of the device or chip set (not a file name).
HKLM,System\CurrentControlSet\
Services\VxD\PCI\GARTMiniports\
xxxxyyyy\Path		 Path to GART miniport driver file.
HKLM,System\CurrentControlSet\
Services\VxD\PCI\GARTMiniports\
xxxxyyyy\Instance		 Instance number, usually 0x00000001.

In the preceding table, xxxxyyyy stands for a DEV-VEN ID; xxxx is a manufacturer-supplied number, and yyyy is a vendor ID obtained from PCI SIG. More information can be found at the PCI-SIG Web site.
