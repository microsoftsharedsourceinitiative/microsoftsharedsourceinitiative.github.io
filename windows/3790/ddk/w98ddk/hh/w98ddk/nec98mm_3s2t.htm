SET_ADAPTER_POWER_STATE

The SET_ADAPTER_POWER_STATE function manages power for the display adapter. 

CONFIGRET set_adapter_power_state(
  DEVNODE dn,
  DWORD PowerState
  );

Parameters

dn
Device node of the display adapter.
PowerState
Mode setting. One of the following:

00h: ON

01h: STAND BY

02h: SUSPEND

04h: OFF

08h: REDUCED ON

Return Value

Returns CR_SUCCESS if successful, or CR_FAILURE if an error has occurred.
