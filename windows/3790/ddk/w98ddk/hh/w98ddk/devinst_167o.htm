Display INF File Extensions

The installation disk for your Windows® 95 display driver must contain:

  • A Windows® 95 format INF text file
  • All DRV and VxD files copied and used by the INF

Display INF files use the standard format defined for all device types in Windows 95. For general information about the INF format and syntax, see Using the Device Information (INF) File Editor. This section documents entries specific to display driver INF files.
