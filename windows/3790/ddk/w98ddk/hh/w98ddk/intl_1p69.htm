Structure of IME

The Windows 95 IME must provide two components: the IME conversion interface and the IME user interface. The IME conversion interface is provided as a set of functions that are exported from the IME module. These functions are called by IMM. The IME user interface is provided as a set of windows. These windows receive messages and provide the IME's user interface.
