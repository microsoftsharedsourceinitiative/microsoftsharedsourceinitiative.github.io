WODM_PAUSE

Sent to a waveform output device driver to pause playback of a waveform.

  • The return value is an error code, or zero (MMSYSERR_NOERROR) if the operation is successful. Possible error codes are:
MMSYSERR_NOTENABLED The driver failed to load or initialize.

DWORD dwParam1
Unused.
DWORD dwParam2
Unused.

If a waveform is currently playing, the driver should stop playing immediately without resetting

its position. If playback has not yet begun, the driver should remember that it is in a paused state and delay playback until it receives a WODM_RESTART message. Data blocks received with the WODM_WRITE message should be placed in the output queue. If output is already paused, this message should have no effect, and the driver should return MMSYSERR_NOERROR.
