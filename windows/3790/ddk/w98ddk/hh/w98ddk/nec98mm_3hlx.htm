VDD_SET_POWER_STATE

The VDD_SET_POWER_STATE function controls power management settings.

Parameters

EDX: PowerState

01h: ON

02h: STANDBY

03h: SUSPEND

04h: OFF

EBX: device ID

Return Value

EAX:

CR_SUCCESS

CR_FAILURE
