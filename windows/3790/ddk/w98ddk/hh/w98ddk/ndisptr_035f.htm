Windows 98/Me: Windows DDK

NDIS Miniport Drivers

To create and test an NDIS miniport driver, refer to the Windows® 2000 DDK documentation. A miniport driver created in this manner is binary compatible and will run on both the Windows 2000 and Windows® Me operating systems. However, a few features are available for miniport drivers in the Windows 2000 operating system that are not available on Windows Me.

IrDA (Infrared) Miniports

To create and test an NDIS miniport driver based on the Infrared communications port, refer to the Windows® 2000 DDK documentation in Part 2 of the Design Guide in Network Drivers.
