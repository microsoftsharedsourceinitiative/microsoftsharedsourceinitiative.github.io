Property Sets for Video Capture

The following property sets are defined for video capture. Each property set definition begins with the GUID used to access the property set, along with the DirectShow™ interface used to access the properties from user mode applications.

The video capture property sets include:

PROPSETID_VIDCAP_VIDEOPROCAMP

PROPSETID_TUNER

PROPSETID_VIDCAP_TVAUDIO

PROPSETID_VIDCAP_CROSSBAR

PROPSETID_VIDCAP_VIDEODECODER

PROPSETID_VIDCAP_VIDEOENCODER

PROPSETID_VIDCAP_CAMERACONTROL

PROPSETID_VIDCAP_VIDEOCOMPRESSION

PROPSETID_VIDCAP_VIDEOCONTROL
