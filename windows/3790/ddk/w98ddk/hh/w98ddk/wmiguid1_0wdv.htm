Standard Data Blocks Requirements

In order to provide a consistent mechanism for management and performance monitoring applications to obtain standard blocks of data, a set of data blocks are defined and must be supported by various driver types. For example, a required block for a serial driver might include bytes sent and bytes received and might be assigned a fixed GUID. All WMI-compliant serial drivers would be required to support this GUID.
