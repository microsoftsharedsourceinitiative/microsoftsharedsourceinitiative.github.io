DVM_GETVIDEOAPIVER

Notifies a video capture driver to supply the version of the video capture command set it uses.

  • Returns DV_ERR_OK if successful.
dwParam1
Address of a DWORD to be filled with the version. Set this value to VIDEOAPIVERSION.
dwParam2
Not used.
