Xlat_API_Exec_Int

Xlat_API_Exec_Int  IntNum

Reflects the specified interrupt into V86 mode. When the interrupt returns, the V86MMGR_Xlat_API service terminates the script and returns to the virtual device that called it.

  • No return value.
IntNum
Interrupt to execute.
