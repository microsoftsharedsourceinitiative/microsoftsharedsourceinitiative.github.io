PROPSETID_VIDCAP_VIDEOCONTROL

The PROPSETID_VIDCAP_VIDEOCONTROL property set is used to control additional aspects of the capture process such as enumerating available frame rates and image orientation.

The property set includes the following of type KSPROPERTY_VIDCAP_VIDEOCONTROL:

KSPROPERTY_VIDEOCONTROL_CAPS (Read Only)

KSPROPERTY_VIDEOCONTROL_FRAME_RATES (Read Only)

KSPROPERTY_VIDEOCONTROL_MODE (Read/Write)

The property set also uses the following enumerated control flags:

typedef enum {
    KS_VideoControlFlag_FlipHorizontal        = 0x0001,
    KS_VideoControlFlag_FlipVertical          = 0x0002,
    KS_VideoControlFlag_ExternalTriggerEnable = 0x0010,
    KS_VideoControlFlag_Trigger               = 0x0020,
} KS_VideoControlFlags;
