System Sleep States and Device Drivers

From the device driver's perspective, the operating system puts the computer into a sleeping state in the following stages:

  • Query Phase

    The configuration manager asks each driver whether it can go to sleep.

  • Sleep Phase

    The configuration manager tells each driver to go into a low power state.

  • Signal Hardware

    The ACPI driver or the Advanced Power Management (APM) driver signals the core logic to go to sleep.
