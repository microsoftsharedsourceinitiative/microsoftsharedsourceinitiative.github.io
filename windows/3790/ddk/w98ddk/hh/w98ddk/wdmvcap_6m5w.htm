KSPROPERTY_VIDEOPROCAMP_CONTRAST

The KSPROPERTY_VIDEOPROCAMP_CONTRAST read/write property is used to set and retrieve the contrast or luma gain setting.

The KSPROPERTY_VIDEOPROCAMP_S.Value member specifies the contrast value expressed as a gain factor * 100. The Value range is from zero to 10000, and the default is 100 (1x). Note that a particular device may only implement a subset of this range.
