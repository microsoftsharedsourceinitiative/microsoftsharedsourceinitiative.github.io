KS_ANALOGVIDEOINFO

The KS_ANALOGVIDEOINFO structure is used with the format FORMAT_AnalogVideo.

typedef struct tagKS_AnalogVideoInfo {
  RECT   rcSource;
  RECT   rcTarget;
  DWORD   dwActiveWidth;
  DWORD   dwActiveHeight;
  REFERENCE_TIME  AvgTimePerFrame;
} KS_ANALOGVIDEOINFO, *PKS_ANALOGVIDEOINFO;

Members

rcSource
Specifies the portion of the active video signal to use. The maximum width is 720, and the height can vary with the TransmissionStd value.
rcTarget
Specifies the destination rectangle. This value and rcSource are relative to the dwActiveWidth and dwActiveHeight.
dwActiveWidth
Currently set to 720 (ITUR-601 active samples per line) but may change with the introduction of HDTV.
AvgTimePerFrame
Specifies the average time per frame, in 100ns units.
