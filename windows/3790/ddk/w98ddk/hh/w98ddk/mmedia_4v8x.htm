MCI_WAVE_DELETE_PARMS

typedef struct {
    DWORD  dwCallback;
    DWORD  dwFrom;
    DWORD  dwTo;
} MCI_WAVE_DELETE_PARMS;

The MCI_WAVE_DELETE_PARMS structure contains parameters for the MCI_DELETE message for waveform audio devices. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
dwFrom
Specifies the starting position for the delete.
dwTo
Specifies the end position for the delete.
