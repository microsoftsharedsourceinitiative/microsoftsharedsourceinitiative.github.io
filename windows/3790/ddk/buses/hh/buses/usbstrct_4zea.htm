_URB_PIPE_REQUEST

USB client drivers set up this structure to clear a stall condition on an endpoint.

struct _URB_PIPE_REQUEST { 
  struct _URB_HEADER  Hdr;
  USBD_PIPE_HANDLE  PipeHandle ;
} ;

Members

Hdr
Pointer to a _URB_HEADER structure that specifies the URB header information. Hdr.Function must be URB_FUNCTION_RESET_PIPE or URB_FUNCTION_ABORT_PIPE, and Hdr.Length must be sizeof(_URB_PIPE_REQUEST).
PipeHandle
Specifies the handle to the pipe that wasreturned by the host controller driver when a configuration was selected.

Headers

Declared in usb.h. Include usb.h.

Comments

Other members that are part of this structure but not described here should be treated as opaque and considered to be reserved for system use.
