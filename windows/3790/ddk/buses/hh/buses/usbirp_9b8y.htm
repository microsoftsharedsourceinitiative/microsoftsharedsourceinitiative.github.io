IOCTL_INTERNAL_USB_GET_PORT_STATUS

Operation

This I/O request queries the status of the PDO. This IOCTL must be sent at IRQL = PASSIVE_LEVEL.

Input

Parameters.Others.Argument1 should be a pointer to a ULONG to be filled in with the port status flags.

Output

Parameters.Others.Argument1 points to a ULONG that has the port status flags filled in. The flags can be one or both of USBD_PORT_ENABLED, USBD_PORT_CONNECTED.

I/O Status Block

The bus or port driver sets Irp->IoStatus.Status to STATUS_SUCCESS or the appropriate error status.

Headers

Defined in usbioctl.h. Include usbioctl.h.
