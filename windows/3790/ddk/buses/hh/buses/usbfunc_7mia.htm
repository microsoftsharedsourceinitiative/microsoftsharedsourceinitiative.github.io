USBD_GetPdoRegistryParameter

The USBD_GetPdoRegistryParameter routine retrieves registry values that indicate the state of the USB stack. This routine is obsolete in Windows XP and later operating systems. Do not use. 

NTSTATUS
USBD_GetPdoRegistryParameters (
  IN PDEVICE_OBJECT  PhysicalDeviceObject,
  IN OUT PULONG  ComplienceFlags,
  IN OUT PULONG  DiagnosticFlags,
  IN OUT PULONG  DeviceHackFlags
  )

Parameters

PhysicalDeviceObject
Specifies the device object for the device.
ComplienceFlags
Indicates, when non-zero, that support for non-compliant devices is enabled.
DiagnosticFlags
Indicates, when non-zero, that the USB stack is in diagnostic mode.
DeviceHackFlags
For system use only. Do not use.

Return Value

The USBD_GetPdoRegistryParameter returns STATUS_SUCCESS when the operation succeeds or an appropriate error status when the operation fails.

Headers

Declared in usbdlib.h. Include usbdlib.h.

Comments

Callers must be running at IRQL < DISPATCH_LEVEL.
