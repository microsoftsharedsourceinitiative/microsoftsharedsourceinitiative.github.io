USB_DEVICE_SPEED

The USB_DEVICE_SPEED enumeration type is used in conjunction with the USB_DEVICE_INFORMATION_0 structure and the QueryDeviceInformation routine to report the speed of a USB device. 

typedef enum _USB_DEVICE_SPEED {
  UsbLowSpeed = 0,
  UsbFullSpeed,
  UsbHighSpeed
} USB_DEVICE_SPEED;

Enumerators

UsbLowSpeed
Indicates a low-speed USB 1.1-compliant device.
UsbFullSpeed
Indicates a full-speed USB 1.1-compliant device.
UsbHighSpeed
Indicates a high-speed USB 2.0-compliant device.

Headers

Declared in usbhcdi.h. Include usbhcdi.h.
