System Support for Buses: Windows DDK

Bus Driver Interfaces Available to USB Clients

In Windows XP and later operating systems, USB client drivers can query the hub driver for an interface rather than linking directly to usbd.sys, in order to perform certain operations.

For a description of the structures associated with the interfaces exposed by the USB bus driver, see Interface Structures Exposed by the USB Bus Driver.

For a description of the routines associated with the interfaces exposed by the USB bus driver, see Routines Associated with Interfaces Exposed by the USB Bus Driver.
