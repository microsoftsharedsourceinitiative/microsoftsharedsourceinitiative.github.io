USB_HUB_NAME

This structure stores the hub's symbolic device name.

typedef struct _USB_HUB_NAME {
  ULONG  ActualLength;
  WCHAR  HubName[1];
} USB_HUB_NAME,  *PUSB_HUB_NAME;

Members

ActualLength
Size of the entire data structure in bytes.
HubName
Specifies the Unicode string containing the hub's symbolic device name.

Headers

Declared in usbioctl.h. Include usbioctl.h.
