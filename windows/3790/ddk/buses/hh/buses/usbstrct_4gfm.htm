USB_NODE_CONNECTION_INFORMATION_EX

The USB_NODE_CONNECTION_INFORMATION_EX structure is used in conjunction with the IOCTL_USB_GET_NODE_CONNECTION_INFORMATION_EX request to obtain information about the connection associated with the indicated USB port.

typedef struct _USB_NODE_CONNECTION_INFORMATION_EX {
  ULONG  ConnectionIndex;
  USB_DEVICE_DESCRIPTOR  DeviceDescriptor;
  UCHAR  CurrentConfigurationValue;
  UCHAR  Speed;
  BOOLEAN  DeviceIsHub;
  USHORT  DeviceAddress;
  ULONG  NumberOfOpenPipes;
  USB_CONNECTION_STATUS  ConnectionStatus;
  USB_PIPE_INFO  PipeList[0];
} USB_NODE_CONNECTION_INFORMATION_EX, *PUSB_NODE_CONNECTION_INFORMATION_EX;

Members

ConnectionIndex
Contains a value greater than or equal to 1 that specifies the number of the port whose connection information is to be reported.
DeviceDescriptor
Reports the USB device descriptor returned by the device during enumeration.
CurrentConfigurationValue
Identifies the ID of the device’s configuration.
Speed
Indicates, when zero, that the device connected to the port is a low or full speed device. A non-zero value indicates that the device is a high-speed device.
DeviceIsHub
Indicates, when TRUE, that the device connected to the port is a hub.
DeviceAddress
Contains the bus-relative address of the device that is connected to the port.
NumberOfOpenPipes
Indicates the number of open pipes on the device that is connected to the port.
ConnectionStatus
Contains a value of type USB_CONNECTION_STATUS that indicates the status of the port.
PipeList
Contains an array of structures of type USB_PIPE_INFO that hold information about each pipe associated with the connected device.

Headers

Declared in usbioctl.h. Include usbioctl.h.
