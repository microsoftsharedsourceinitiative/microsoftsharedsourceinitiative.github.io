UsbBuildSelectConfigurationRequest

UsbBuildSelectConfigurationRequest formats an URB with the parameters necessary to select a configuration on a USB device.

VOID
  UsbBuildSelectConfigurationRequest(
    IN PURB  Urb,
    IN USHORT  Length,
    IN PUSB_CONFIGURATION_DESCRIPTOR  ConfigurationDescriptor
    );

Parameters

Urb
Pointer to an URB to be formatted as a select configuration request.
Length
Specifies the size, in bytes, of the URB.
ConfigurationDescriptor
Pointer to an initialized USB configuration descriptor that identifies the configuration to be set on the device. If NULL, the device will be set into its unconfigured state.

Return Value

None.

Headers

Defined in usbdlib.h. Include usbdlib.h.

Comments

This function is actually a macro defined in usblib.h. Callers of this routine must be running at IRQL <= DISPATCH_LEVEL.
