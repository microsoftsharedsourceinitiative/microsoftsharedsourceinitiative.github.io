IOCTL_USB_GET_NODE_CONNECTION_INFORMATION_EX

Operation

This I/O request queries the USB bus driver for a USB_NODE_CONNECTION_INFORMATION_EX structure that contains information about the connection associated with the indicated port.

This IOCTL must be sent at an IRQL of PASSIVE_LEVEL.

Input

AssociatedIrp.SystemBuffer should be a pointer to a USB_NODE_CONNECTION_INFORMATION_EX structure. On input, the ConnectionIndex member of this structure must contain a number greater than or equal to 1 that indicates the number of the port whose connection information is to be reported. The bus driver returns connection information in the remaining members of USB_NODE_CONNECTION_INFORMATION_EX.

Output

The USB bus driver will return information about the indicated connection in the USB_NODE_CONNECTION_INFORMATION_EX structure pointed to by AssociatedIrp.SystemBuffer.

I/O Status Block

The bus driver sets Irp->IoStatus.Status to STATUS_SUCCESS or the appropriate error status.

Headers

Defined in usbioctl.h. Include usbioctl.h.
