System Support for Buses: Windows DDK

Port Driver Interfaces Available to Vendor-Supplied Bus Drivers

In Windows XP and later operating systems, vendor-supplied bus drivers can directly access interfaces exposed by the port driver.

For a description of the structures associated with the interfaces exposed by the USB bus driver, see Interface Structures Exposed by the USB Port Driver.

For a description of the routines associated with the interfaces exposed by the USB bus driver, see Routines Associated with Interfaces Exposed by the USB Port Driver.
