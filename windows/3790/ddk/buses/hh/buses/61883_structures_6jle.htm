UNIT_DDI_VERSION

The UNIT_DDI_VERSION structure is used in conjunction with the Av61883_GetUnitInfo request to retrieve the current 61883 DDI version.

typedef struct _UNIT_DDI_VERSION {
  OUT ULONG  Version;
} UNIT_DDI_VERSION, *PUNIT_DDI_VERSION;
