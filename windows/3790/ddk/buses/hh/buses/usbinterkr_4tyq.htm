System Support for Buses: Windows DDK

Routines Associated with Interfaces Exposed by the USB Port Driver

The routines in version 0 of the port driver interface are identical in function to the corresponding routines in version 0 of the bus driver interfaces:

InterfaceReference

InterfaceDereference

Version 1 of the port driver interface contains all of the routines present in version 0 plus the following routines:

CreateUsbDevice

GetPortHackFlags

GetUsbDescriptors

InitializeUsbDevice

QueryDeviceInformation

RestoreUsbDevice

Version 2 of the port driver interface contains all of the routines present in version 1 plus the following routines:

GetControllerInformation

ControllerSelectiveSuspend

GetExtendedHubInformation

GetRootHubSymbolicName

GetDeviceBusContext

Initialize20Hub

Version 3 of the port driver interface contains all of the routines present in version 2 plus the following routine:

RootHubInitNotification

Version 4 of the port driver interface contains all of the routines present in version 3 plus the following routine:

FlushTransfers.

Version 5 of the port driver interface contains all of the routines present in version 4 plus the following routine:

SetDeviceHandleData
