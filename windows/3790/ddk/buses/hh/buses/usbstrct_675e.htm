USB_EXTHUB_INFORMATION_0

The USB_EXTHUB_INFORMATION_0 structure is used in conjunction with the GetExtendedHubInformation routine to retrieve information about the ports on a hub device. 

typedef struct _USB_EXTHUB_INFORMATION_0 {
  ULONG  InformationLevel;
  ULONG  NumberOfPorts;
  USB_EXTPORT_INFORMATION_0  Port[255];
} USB_EXTHUB_INFORMATION_0, *PUSB_EXTHUB_INFORMATION_0;

Members

InformationLevel
Indicates the level of information that the caller requests. For this structure, the level should be set to 0.
NumberOfPorts
Indicates, on return, the number of ports on the hub. The maximum number of ports that a hub can have is 255.
Port
Contains an array of USB_EXTPORT_INFORMATION_0 structures, each of which contains information about a port on the hub.

Headers

Declared in hubbusif.h. Include hubbusif.h.
