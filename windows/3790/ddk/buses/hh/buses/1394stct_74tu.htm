GET_LOCAL_HOST_INFO7

The GET_LOCAL_HOST_INFO7 structure contains the data returned by a REQUEST_GET_LOCAL_HOST_INFO request with u.GetLocalHostInformation.nLevel set to GET_HOST_DMA_CAPABILITIES.

typedef struct _GET_LOCAL_HOST_INFO7 {
  ULONG  HostDmaCapabilities;
  ULARGE_INTEGER  MaxDmaBufferSize;
} GET_LOCAL_HOST_INFO7, *PGET_LOCAL_HOST_INFO7;

Members

HostDmaCapabilities
See GET_LOCAL_HOST_INFO2 for the list of host controller capability flags that can be assigned to this member.
MaxDmaBufferSize
Specifies the maximum DMA buffer size supported by the host controller. This value is determined by multiplying page size by the number of map registers supported by the host controller. This value does not necessarily represent the buffer size supported by the system as a whole. To be safe, the caller should compare this value with the value for the maximum DMA buffer size returned by IoGetDmaAdapter and use the minimum of the two values.
