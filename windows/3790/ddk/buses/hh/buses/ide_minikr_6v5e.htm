PHW_ADAPTER_STATE

The PHW_INITIALIZE routine prototype declares a routine that saves or restores the state of the miniport's HBA. 

typedef
BOOLEAN
(*PHW_ADAPTER_STATE) (
  IN PVOID  DeviceExtension,
  IN PVOID  Context,
  IN BOOLEAN  SaveState
  );

Parameters

DeviceExtension
Pointer to the miniport driver's per-HBA storage area.
Context
Reserved for system use.
SaveState
Indicates, when TRUE, that the miniport should save the current state of the HBA until the HwScsiAdapterState routine is called again with SaveState set to FALSE to restore the saved state.

Return Value

The routine declared by this prototype returns TRUE if it successfully saved or restored the HBA state, FALSE otherwise.

Headers

Declared in either srb.h or storport.h. Include srb.h or storport.h.

Comments

Only SCSI miniports use this prototype. Miniports that work with the StorPort driver do not use the routine that is declared by this prototype.

For more information on the routine declared by this prototype, see HwScsiAdapterState.
