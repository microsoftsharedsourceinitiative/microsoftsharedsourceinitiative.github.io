TUPLE_REQUEST

The TUPLE_REQUEST structure is used in conjunction with the IOCTL_GET_TUPLE_DATA request to retrieve tuple data from a PC card's attribute memory. 

typedef struct _TUPLE_REQUEST {
  USHORT  Socket;
} TUPLE_REQUEST, *PTUPLE_REQUEST;

Members

Socket
Indicates the socket number.

Headers

Declared in ntddpcm.h. Include ntddpcm.h.
