Battery Devices: Windows DDK

Responding to Battery Class Driver Queries

The miniclass driver must provide the following three BatteryMiniXxx routines, which report battery status:

  • BatteryMiniQueryTag
  • BatteryMiniQueryInformation
  • BatteryMiniQueryStatus

The BatteryClassIoctl routine in the class driver calls these miniclass driver routines when it receives IOCTLs requesting information about the batteries.
