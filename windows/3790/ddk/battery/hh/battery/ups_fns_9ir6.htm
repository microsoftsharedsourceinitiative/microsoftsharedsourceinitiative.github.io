UPS Minidriver Functions

This section describes the following functions, which are exported by UPS minidrivers:

UPSCancelWait

UPSGetState

UPSInit

UPSStop

UPSTurnOff

UPSWaitForStateChange

These functions are called by the system-supplied UPS service. For more information, see Writing UPS Minidrivers.
