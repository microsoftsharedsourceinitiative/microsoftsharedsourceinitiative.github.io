Installing UPS Minidrivers

UPS minidrivers are user-mode DLLs that can be installed using Microsoft® Windows® Installer (described in Platform SDK documentation) or an INF file (see Creating an INF File).

One responsibility of the installation operation is to create UPS\ServiceProviders registry entries.
