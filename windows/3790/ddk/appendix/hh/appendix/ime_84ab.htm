Appendix: Windows DDK

East Asian Platforms

On Microsoft® Windows NT® 4.0 or Windows® 9.x non-Asian platforms, there is no IME support. Starting from Windows 2000, IME is supported on all language operating systems. When developing applications, please make sure your code can handle Asian languages. Pay attention to the following things:

  • Test all windows that accept character input to make sure they can accept and handle non-ASCII characters correctly.
  • Make sure the font can handle all the different language input characters.
  • Make sure that all your dialogs still look good and are big enough for your strings.
