Appendix: Windows DDK

Designed for Windows XP Logo Program Requirements

Requirements for the Windows Logo Program for hardware that apply specifically for systems or peripherals that will receive the "Designed for Windows XP" are defined in Microsoft Windows Logo Program System and Device Requirements, Version 2.0,
available on the Windows Logo Program for Hardware Web site.

Windows Hardware Quality Labs (WHQL) is providing the Hardware Compatibility Tests (HCT) for Windows XP. For information, see the WHQL Web site.
