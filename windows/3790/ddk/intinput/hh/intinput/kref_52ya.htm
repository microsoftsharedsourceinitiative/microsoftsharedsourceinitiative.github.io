CONNECT_DATA (Kbdclass)

CONNECT_DATA specifies information that Kbdclass uses to connect to a keyboard port.

typedef struct _CONNECT_DATA {
  IN PDEVICE_OBJECT  ClassDeviceObject;
  IN PVOID  ClassService;
} CONNECT_DATA, *PCONNECT_DATA;

Members

ClassDeviceObject
Pointer to a Kbdclass upper-level class filter device object (filter DO).
ClassService
Specifies the Kbdclass class service routine KeyboardClassServiceCallback.

Headers

Declared in kbdmou.h. Include kbdmou.h.

Comments

The keyboard class driver uses this structure with an IOCTL_INTERNAL_KEYBOARD_CONNECT request.

For more information about connecting to a keyboard port, see Connect a Class Service Callback and a Filter Service Callback to a Device.
