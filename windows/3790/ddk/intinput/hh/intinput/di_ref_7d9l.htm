DirectInput Kernel-Mode Service Macros

DirectInput provides a set of macros that contain entry points to kernel-mode services. These entry points are:

Windows Services

VJOYD_Register_Device_Driver

VJOYD_GetPosEx_Service

DirectInput Services

VJOYD_GetInitParams_Service

VJOYD_Poll_Service

Device Services

VJOYD_Escape_Service

VJOYD_CtrlMsg_Service

Force Feedback Device Services

VJOYD_DestroyEffect_Service

VJOYD_DownloadEffect_Service

VJOYD_GetEffectStatus_Service

VJOYD_GetFFState_Service

VJOYD_SendFFCommand_Service

VJOYD_SetGain_Service

VJOYD_StartEffect_Service

VJOYD_StopEffect_Service

Interrupt Polling Service

VJOYD_DeviceUpdateNotify_Service

Screen Saver Service

VJOYD_JoystickActivity_Service

Registry Access Services

VJOYD_OpenTypeKey_Service

VJOYD_OpenConfigKey_Service

Note  If you need to open registry keys, you should use the VJOYD_OpenConfigKey_Service and VJOYD_OpenTypeKey_Service service macros instead of opening the registry keys directly. Using these services ensures that the correct registry branch is opened. In addition, the service macros will be supported in future versions of DirectInput when the underlying registry data may be structured differently.
