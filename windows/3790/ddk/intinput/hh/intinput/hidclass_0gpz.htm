Interface That Is Part of a Composite USB Device and Supports More Than One Top-Level Collection

The following hardware ID formats are used if the parent input device is enumerated as a composite interface USB device and an interface supports more than one top-level collection:

  • Vendor Hardware ID Formats

    HID\Vid_v(4)&Pid_d(4)&Rev_r(4)&MI_z(2)&Colb(2)

    HID\Vid_v(4)&Pid_d(4)&MI_z(2)&Colb(2)

  • System Hardware ID Formats (Internal Use Only)

    HID_DEVICE_UP:p(4)_U:u(4)

    HID_DEVICE

The fields that are in italics are device-specific values which are defined in Hardware ID Field Definitions for Top-Level Collections.
