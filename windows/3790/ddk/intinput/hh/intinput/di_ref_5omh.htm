Interactive Input Devices: Windows DDK

IDirectInputEffectDriver

The IDirectInputEffectDriver interface exposes the following methods:

DestroyEffect

DeviceID

DownloadEffect

Escape

GetEffectStatus

GetForceFeedbackState

GetVersions

SendForceFeedbackCommand

SetGain

StartEffect

StopEffect

In addition, all COM interfaces inherit the following IUnknown interface methods:

AddRef

QueryInterface

Release

These three methods allow additional interfaces to be added to the DirectInputEffectDriver object without affecting the functionality of the original interface.
