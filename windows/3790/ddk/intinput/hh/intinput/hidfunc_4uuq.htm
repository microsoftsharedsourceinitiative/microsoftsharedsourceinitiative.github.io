HIDClass Support Routines

This section describes the system-supplied routines that user-mode applications, kernel-mode drivers, and HID minidrivers use to operate devices in the HIDClass device setup class.

As a general rule, user-mode applications can use the HidD_Xxx and HidP_Xxx routines described in this section. However, kernel-mode drivers can only call HidP_Xxx routines.

HID minidrivers use only the HidRegisterMiniDriver routine.

For more information about system support for HIDClass devices, see the following:
