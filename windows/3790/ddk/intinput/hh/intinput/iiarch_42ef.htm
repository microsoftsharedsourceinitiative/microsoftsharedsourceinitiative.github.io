System-Supplied Drivers for Non-HIDClass Input Devices

This section describes the class drivers and related lower-level drivers for non-HIDClass input devices.

Microsoft® Windows® 2000 and later versions supply the following drivers for keyboard and mouse devices attached to PS/2 and serial ports.

Device Type Driver Type Driver
Keyboard Class driver kbdclass.sys
Mouse Class driver mouclass.sys
PS/2 (i8042) port Function driver i8042prt.sys
Serial port Function driver and filter driver serial.sys and serenum.sys

Windows Me supplies the following drivers for keyboard, mouse, and game port devices attached to PS/2, serial, and game ports.

Device Type Driver Type Driver
Keyboard Class driver vkd.vxd
Mouse Class driver vmd.vxd
Game port Class driver vjoyd.vxd
Serial port, PS/2 port, and game port Class driver Minidrivers
