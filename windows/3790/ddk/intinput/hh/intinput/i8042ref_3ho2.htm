I8042prt Keyboard Major I/O Requests

This section describes the I8042prt-specific operation of the following major I/O requests that I8042prt supports for a keyboard device:

IRP_MJ_CREATE (I8042prt Keyboard)

IRP_MJ_DEVICE_CONTROL (I8042prt Keyboard)

IRP_MJ_INTERNAL_DEVICE_CONTROL (I8042prt Keyboard)

IRP_MJ_PNP (I8042prt Keyboard)

IRP_MJ_POWER (I8042prt Keyboard)

IRP_MJ_SYSTEM_CONTROL (I8042prt Keyboard)

For information about how I8042prt handles the generic operation of these requests, see IRP Major Function Codes.
