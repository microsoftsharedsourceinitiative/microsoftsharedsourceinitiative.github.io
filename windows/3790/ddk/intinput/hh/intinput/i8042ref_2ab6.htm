IRP_MJ_INTERNAL_DEVICE_CONTROL (I8042prt Keyboard)

When Sent

Kbdclass and I8042prt use IRP_MJ_INTERNAL_DEVICE_CONTROL requests to operate a keyboard device.

Input Parameters

Request-specific.

Output Parameters

Request-specific.

I/O Status Block

Status block members are set to request-specific values. If I8042prt does not support the request, it sets the Status member to STATUS_INVALID_DEVICE_REQUEST.

Headers

Defined in wdm.h and ntddk.h. Include wdm.h or ntddk.h.

Operation

For information about the internal device control requests that I8042prt supports for a keyboard device, see I8042prt Keyboard Internal Device Control Requests.

If I8042prt does not support the request, it completes the request with a status of STATUS_INVALID_DEVICE_REQUEST.
