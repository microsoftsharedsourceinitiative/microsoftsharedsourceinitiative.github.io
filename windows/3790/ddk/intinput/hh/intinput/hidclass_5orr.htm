HID Descriptors

A HID descriptor specifies the number, type, and size of physical descriptors and report descriptors that are associated with a HIDClass device. The HID descriptor also specifies a country or region code that localizes the hardware (however, most hardware is not localized).

For information about obtaining a HID descriptor, see the IOCTL_HID_GET_DEVICE_DESCRIPTOR request and the HID_DESCRIPTOR structure.
