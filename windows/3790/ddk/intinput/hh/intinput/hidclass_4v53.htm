Interactive Input Devices: Windows DDK

Game Port HIDClass-Related Samples

The DDK includes the following samples that support game port devices:

  • The system-supplied HID minidriver for game port (I/O 201 port) devices in Microsoft® Windows® 2000 and later versions (hidgame.sys).

    The hidgame.sys minidriver calls HIDRegisterMiniDriver to bind itself to the HID class driver. The HID class driver creates a functional device object (FDO) for each game port physical device object (PDO) that is created by gameenum.sys. The HID class driver also creates a PDO for each HID collection that the game port PDO supports. The HID class driver communicates with a game port PDO through hidgame.sys by using the HID minidriver IOCTLs.

    For general information about the operation of a HID minidriver, see Operation of a HID Minidriver.

    For information about building hidgame.sys and using the Hardware Wizard to install a game port device that uses hidgame.sys, see hid\hidgame\hidgame.htm.

  • The game port bus driver that is used with hidgame.sys to support game port devices in Windows 2000 and later versions (gameenum.sys).

    This driver creates a PDO for an input device that is attached to a game port device.

    For information about building gameenum.sys, see hid\gameenum\gamesys.htm.
