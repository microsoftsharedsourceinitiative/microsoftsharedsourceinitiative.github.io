HIDClass Devices

This section describes the operation of devices in the system-supplied HIDClass device setup class.

The system-supplied HID class driver is the WDM function driver and bus driver for the HIDClass device setup class. The HID class driver is designed to use HID minidrivers to access input devices. This approach makes it possible for the HID class driver to operate USB HID devices and input devices attached to buses or ports other than a USB bus. Examples of such devices include keyboard, mouse, and joystick devices attached to PS/2, serial, and game ports. The executable image of the HID class driver is hidclass.sys.
