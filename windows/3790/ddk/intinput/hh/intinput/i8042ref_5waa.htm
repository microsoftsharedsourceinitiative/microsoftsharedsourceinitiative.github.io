I8042prt Mouse Internal Device Control Requests

This section describes the internal device control requests that I8042prt supports for a mouse device:

IOCTL_INTERNAL_I8042_HOOK_MOUSE

IOCTL_INTERNAL_I8042_MOUSE_START_INFORMATION

IOCTL_INTERNAL_I8042_MOUSE_WRITE_BUFFER

IOCTL_INTERNAL_MOUSE_CONNECT

IOCTL_INTERNAL_MOUSE_DISCONNECT

IOCTL_MOUSE_QUERY_ATTRIBUTES

If I8042prt does not support the request, it completes the request with a status of STATUS_INVALID_DEVICE_REQUEST.
