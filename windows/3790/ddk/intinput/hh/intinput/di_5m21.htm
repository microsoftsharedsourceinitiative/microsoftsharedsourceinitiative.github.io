Gchand.dll, Services and Support

Gchand.dll is an InProcHandler that provides a layer of abstraction between Joy.cpl and the game controller property sheets. Generally, Gchand.dll does little more than simplify the Windows API call PropertySheet by reducing the number of parameters and enforcing others. As shown in Creating Your Property Sheet, to include Gchand.dll, simply add an InProcHandler32 entry to your property sheet DLL's HKEY_CLASSES_ROOT\CLSID entry in the registry. The "(default)" key should then contain the location of Gchand.dll.

Gchand.dll provides services using the COM interface IDIGameCntrlPropSheet. For information on the methods accessed by this interface, see IDIGameCntrlPropSheet.
