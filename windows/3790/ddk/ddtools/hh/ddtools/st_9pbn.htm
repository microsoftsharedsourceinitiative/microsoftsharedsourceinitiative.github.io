Building and Installing the tracedrv Sample

To build the sample driver tracedrv.sys:

  1. Add the following line at the end of the driver's sources file: 
    RUN_WPP= $(sources) -km

    See Invoking the WPP preprocessor.

  2. Use the DDK Build utility to build the driver in the \tracedrv\driver directory of the Windows DDK.

To install the sample, use the Add Hardware wizard and the tracedrv.inf file that is provided with the tracedrv sample.
