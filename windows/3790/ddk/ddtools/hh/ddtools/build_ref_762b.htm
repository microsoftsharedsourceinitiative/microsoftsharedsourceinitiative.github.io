BUILD_CONSUMES

The BUILD_CONSUMES macro instructs the Build utility to delay building this directory until the directory containing the corresponding BUILD_PRODUCES macro has been built.

For any given string, the directory whose sources file contains

BUILD_PRODUCES=String

will be built before any directories whose sources file contains

BUILD_CONSUMES=String

For details, see Building on a Multiprocessor Computer.
