Device Path Exerciser Tests

This section describes the basic and optional open operations and tests that you can include in a Device Path Exerciser test session.

The Device Path Exerciser begins by opening the devices using the Basic Open Operations and, optionally, the other open operation. When an open operation returns a handle to a device, Device Path Exerciser uses the handle to perform tests on the device from all selected test categories.
