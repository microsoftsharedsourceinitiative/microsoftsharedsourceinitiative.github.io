DevCon Return Codes

DevCon returns an integer that can be used in programs and scripts to determine the success of a DevCon command (for example, return = devcon hwids *).

The following table lists and describes the return codes.

Return code Meaning
0 Success
1 Requires reboot
2 Failure
3 Syntax error
