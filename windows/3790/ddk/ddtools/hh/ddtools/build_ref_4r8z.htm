MFC_FLAGS

The MFC_FLAGS macro specifies the Microsoft Foundation Classes (MFC) compiler command-line options.

If you have extra command line options that you want to pass to the compiler, you can put them in MFC_FLAGS. They will only affect programs that use MFC. That can be useful if you have a global project.

For example, you could set MFC_FLAGS equal to some macro name or compiler #define instruction that you want to add. Alternatively, you can use the C_DEFINES macro.
