Bug Check 0x93: INVALID_KERNEL_HANDLE

The INVALID_KERNEL_HANDLE bug check has a value of 0x00000093. This indicates that an invalid or protected handle was passed to NtClose.

Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 The handle passed to NtClose
2 0: the caller tried to close a protected handle

1: the caller tried to close an invalid handle

3 Reserved
4 Reserved

Cause

This error indicates that some kernel code (for example, a server, redirector, or another driver) attempted to close an invalid handle or a protected handle.
