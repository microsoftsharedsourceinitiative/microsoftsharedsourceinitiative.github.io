PASS0_SERVERDIR

Use the PASS0_SERVERDIR macro to specify an output location for the server part that the Microsoft IDL Compiler (MIDL) generates. By default, the results of pass zero are output to the TARGETPATH subdirectory. Use this macro to change that default.

To create the server part in the current subdirectory, set PASS0_SERVERDIR to dot ("."):

PASS0_SERVERDIR=.

Headers can be output to any location. If your #include directives do not specify the correct location, however, the tools later used by the Build utility will not be able to find the headers.
