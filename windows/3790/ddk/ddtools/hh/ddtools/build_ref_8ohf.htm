BASEDIR

The BASEDIR environment variable specifies the base of the build product's source tree. By default, this is set to the root directory where the DDK is installed.

Note  The directory specified by BASEDIR must not contain spaces. If you use the Build utility with directories that contain spaces, the error messages generated may not make it clear what is wrong.
