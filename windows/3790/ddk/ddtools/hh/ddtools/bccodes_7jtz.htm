Bug Check 0x98: END_OF_NT_EVALUATION_PERIOD

The END_OF_NT_EVALUATION_PERIOD bug check has a value of 0x00000098. This indicates that the Windows trial period has ended.

Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 The low-order 32 bits of the installation date
2 The high-order 32 bits of the installation date
3 The trial period (in minutes)
4 Reserved

Cause

Your Windows installation is an evaluation unit with an expiration date. The trial period is over.
