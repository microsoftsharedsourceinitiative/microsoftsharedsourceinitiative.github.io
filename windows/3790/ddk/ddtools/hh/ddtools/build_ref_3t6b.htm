PASS0_HEADERDIR

Use the PASS0_HEADERDIR macro to specify the output location of the headers generated by MC and the Microsoft IDL Compiler (MIDL). By default, everything built in pass zero is output to the TARGETPATH subdirectory. Use this macro to change that default.

To create the headers in the current subdirectory, set PASS0_HEADERDIR to dot ("."):

PASS0_HEADERDIR=.

Headers can be output to any location. If your #include directives do not specify the correct location, however, the tools later used by the Build utility will not be able to find the headers.
