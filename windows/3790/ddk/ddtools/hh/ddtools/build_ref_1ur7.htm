_objects.mac File

The Build utility puts a file in the obj subdirectory called _objects.mac that contains the object dependencies for each platform.

A typical _objects.mac file looks like this:

386_OBJECTS= \
    $(_OBJ_DIR)\i386\x.obj \
    $(_OBJ_DIR)\i386\y.obj

PASS0_386_OBJECTS=

IA64_OBJECTS= \
    $(_OBJ_DIR)\ia64\x.obj \
    $(_OBJ_DIR)\ia64\y.obj

PASS0_IA64_OBJECTS=

This file is created by the Build utility. You do not need to edit it. Every time the sources file gets updated, this file is regenerated.
