Build Utility Environment Variables

The values of the following environment variables are set in the build environment window. Some of these enviroment variables can also be set in the sources files. For an overview of how these macros are used, see Using Macros and Environment Variables.

The environment variables initialized when the build environment is created are described in the following topics:

AMD64_FLAGS

ATL_INC_PATH

BASEDIR

BUILD_ALT_DIR

BUILD_DEFAULT

BUILD_DEFAULT_TARGETS

BUILD_MAKE_PROGRAM

BUILD_MULTIPROCESSOR

C_DEFINES

CRT_INC_PATH

CRT_LIB_PATH

DDK_INC_PATH

DDK_LIB_DEST

DDK_LIB_PATH

DDK_TARGET_OS

DDKBUILDENV

DRIVER_INC_PATH

NO_BINPLACE

NTDEBUG

OAK_INC_PATH

SDK_INC_PATH

SDK_LIB_DEST

SDK_LIB_PATH

VERIFIER_DDK_EXTENSIONS

WDM_INC_PATH

The environment variable described in the following topic is not automatically created, but can be created by using the set command:

BUILD_OPTIONS
