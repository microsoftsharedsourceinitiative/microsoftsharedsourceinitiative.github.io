Bug Check 0xE4: WORKER_INVALID

The WORKER_INVALID bug check has a value of 0x000000E4. This indicates that an executive worker item was found in memory which must not contain such items.

Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 Code position indicator
2 Address of worker item
3 Start of pool block
4 End of pool block

Cause

This is usually caused by a driver freeing memory which still contains an executive worker item.
