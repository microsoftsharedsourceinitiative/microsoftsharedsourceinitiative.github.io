Driver Development Tools: Windows DDK

Disabler

Disabler, or disabler.exe, is a command-line tool that repeatedly disables and re-enables (or, optionally, removes/refreshes) a Device Manager object for use in testing how that driver responds to many load/unload events. This test works on both Microsoft® Windows® 2000 and Microsoft Windows 98.

Note  WHQL is currently using this tool for some of their programs.
