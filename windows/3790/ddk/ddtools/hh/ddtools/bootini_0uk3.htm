Boot Options on x86-based Systems

Computers with x86 architecture that run NT-based Windows operating systems store boot options in boot.ini, a hidden text file at the root of the system partition.

The following sections briefly describe the boot.ini file and explain the aspects of boot options that are specific to x86-based systems.
