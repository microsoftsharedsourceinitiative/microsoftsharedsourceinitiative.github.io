makefile File

NMAKE uses makefile files to generate dependency and command lists. A standard makefile file in every sample driver's source code directory directs NMAKE to the master macro definition file, makefile.def. The makefile you supply for your own source code should do the same. The makefile.def file is included with the DDK.
