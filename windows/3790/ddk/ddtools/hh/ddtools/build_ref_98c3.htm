BUILD_DEFAULT_TARGETS

Use the BUILD_DEFAULT_TARGETS environment variable to specify the default target platform for which you are building. Assuming that cross compilers existed, you could set it to -386 on an Itanium and build x86 binaries.

Usually, you can just set it to the platform on which you are hosted. For example:

c:\> build -386

means the same as:

c:\> set BUILD_DEFAULT_TARGETS=-386
c:\> build
