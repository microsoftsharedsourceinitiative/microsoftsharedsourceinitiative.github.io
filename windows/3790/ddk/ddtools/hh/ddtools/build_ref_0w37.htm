RCCODEPAGE

The RCCODEPAGE macro is used to set the code pages that are associated with your build.

The value of RCCODEPAGE is passed to the Resource Compiler (RC) as the argument of its -C parameter.

This can be useful if you want to control the code pages used (for instance, when building products for different language markets).
