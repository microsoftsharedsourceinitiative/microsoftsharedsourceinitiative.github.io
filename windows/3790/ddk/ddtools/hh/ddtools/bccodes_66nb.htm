Bug Check 0xE3: RESOURCE_NOT_OWNED

The RESOURCE_NOT_OWNED bug check has a value of 0x000000E3. This indicates that a thread tried to release a resouce it did not own.

Parameters

The following parameters are displayed on the blue screen.

Parameter Description
1 Address of resource
2 Address of thread
3 Address of owner table (if it exists)
4 Reserved
