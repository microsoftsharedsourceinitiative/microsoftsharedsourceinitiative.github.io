Tailored Random Test

The Tailored Random Test is included in the Random Test.

The Tailored Random Test uses the results of the Random Test to examine the drivers response to IOCTL or FSCTL requests in more detail. The Tailored Random Test probes areas that the Random Test missed and those on which the driver did not respond as expected based on the status returned by the Random Test calls.
