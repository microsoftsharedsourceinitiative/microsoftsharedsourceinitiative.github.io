SYNCHRONIZE_BLOCK

The SYNCHRONIZE_BLOCK macro instructs the Build utility to build a certain directory first.

When the Build utility finds this macro in a sources file, it will stop building all other directories until this directory is built. To use this macro, specify the following in your sources file:

SYNCHRONIZE_BLOCK=1

For details, see Building on a Multiprocessor Computer.
