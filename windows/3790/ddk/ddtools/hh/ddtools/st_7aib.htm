WPP Utilities and Files

This section describes the basic operation of the WPP utilities and their related files. For information about all the options supported by these utilities, see the Platform SDK documentation.

A consumer of trace information uses the following WPP command-line utilities to control a software trace and convert the logged messages to a human-readable form:

Tracelog Utility

Tracepdb Utility

Tracefmt Utility

The following types of files are associated with the operation of these utilities:

Trace Message Format File

Summary Message File

Formatted Trace Message File
