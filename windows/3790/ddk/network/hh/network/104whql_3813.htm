Process Structure and Run-time Library Routines

WHQL restricts NDIS drivers—that are not also NDIS-WDM—from calling the following process structure routines and run-time library routines:

Restricted routine Error message
PsCreateSystemThread Do not use this call.
PsTerminateSystemThread Do not use this call.
RtlDeleteRegistryValue Do not use this call.
RtlQueryRegistryValues Do not use this call.
RtlWriteRegistryValue Do not use this call.
