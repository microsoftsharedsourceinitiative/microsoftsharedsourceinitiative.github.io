WSABUFEX

The WSABUFEX structure contains information for a Windows Sockets extended buffer. 

typedef struct _WSABUFEX {
  u_long  len; 
  char FAR *  buf; 
  HANDLE  handle; 
} WSABUFEX, FAR * LPWSABUFEX;

Members

len
Size, in bytes, of the buffer at buf.
buf
Pointer to the buffer.
handle
Handle to the buffer. This handle was returned from the WSPRegisterMemory call.

Headers

Defined in Ws2san.h. Include Ws2san.h. Requires Windows Sockets version 2.2.
