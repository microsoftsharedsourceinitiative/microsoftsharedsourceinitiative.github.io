TdiBuildNetbiosAddress

TdiBuildNetbiosAddress sets up a TA_NETBIOS_ADDRESS structure for the caller. 

VOID 
  TdiBuildNetbiosAddress(
    IN PUCHAR  NetbiosName,
    IN BOOLEAN  IsGroupName,
    IN OUT PTA_NETBIOS_ADDRESS  NetworkName
    );

Parameters

NetbiosName
Pointer to 16-byte buffer containing a NetBIOS name.
IsGroupName
Specifies TRUE if the given name is a group address.
NetworkName
Pointer to a caller-supplied buffer, which must be at least sizeof(TA_NETBIOS_ADDRESS) bytes in length.

TdiBuildNetbiosAddress fills in the members of this structure.

Headers

Declared in TdiKrnl.h. Include TdiKrnl.h.
