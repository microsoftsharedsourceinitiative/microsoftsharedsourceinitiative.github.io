INetCfgBindingPath::EnumBindingInterfaces

The EnumBindingInterfaces method retrieves an enumeration of a collection of binding interfaces contained in a binding path.

HRESULT
  EnumBindingInterfaces(
    OUT IEnumNetCfgBindingInterface  **ppenumInterface  
    );

Parameters

ppenumInterface
Pointer to a buffer that receives a pointer to the IEnumNetCfgBindingInterface interface. IEnumNetCtgBindingInterface enumerates the INetCfgBindingInterface interfaces that are contained in a binding path.

Return Value

Returns zero (S_OK) if successful; otherwise, returns NETCFG_* codes that are defined in Netcfgx.h. If the returned pointer points to NULL, EnumBindingInterfaces returns E_POINTER.

Headers

Defined in Netcfgx.h. Include Netcfgx.h.

Comments

The IEnumNetCfgBindingInterface interface is a standard COM enumerator. To locate more information on COM enumerators and their methods, see the IEnumXXXX topic in the Platform SDK.
