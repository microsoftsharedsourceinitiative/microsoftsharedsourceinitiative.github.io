NdisBufferLength

NdisBufferLength returns the length, in bytes, of a given mapped buffer.

ULONG
  NdisBufferLength(
    IN PNDIS_BUFFER  Buffer
    );

Parameters

Buffer
Pointer to the buffer descriptor mapping the buffer.

Return Value

NdisBufferLength returns the byte count of the mapped buffer.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

Callers of NdisBufferLength must be running at IRQL <= DISPATCH_LEVEL.
