Operational OIDs

The following table summarizes the operational power management OIDs.

Length Q S Name
Arr   O OID_PNP_ADD_WAKE_UP_PATTERN

Wake-up pattern that the miniport driver should load into the NIC
Variable O   OID_PNP_CAPABILITIES

The NIC's wake-up capabilities
ULONG O O OID_PNP_ENABLE_WAKE_UP

The NIC wake-up capabilities that the miniport driver should enable
ULONG O   OID_PNP_QUERY_POWER

A query to determine whether the miniport driver can transition the NIC to a specified device power state
Arr   O OID_PNP_REMOVE_WAKE_UP_PATTERN

Wake-up pattern that the miniport driver should remove from the NIC
ULONG   O OID_PNP_SET_POWER

A command to set the NIC to a specified power state
Arr O   OID_PNP_WAKE_UP_PATTERN_LIST

The wake-up pattern currently loaded in the NIC
