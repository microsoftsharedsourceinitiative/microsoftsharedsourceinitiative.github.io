WAN Objects

This section describes two sets of object identifiers (OIDs) that are used to support WAN miniport drivers in either standard or Connection-oriented NDIS (CoNDIS) environments. If drivers work in a standard WAN environment, they use WAN OIDs for WAN miniport drivers. If drivers work in a CoNDIS WAN environment, they use WAN OIDs for CoNDIS WAN miniport drivers. Standard and CoNDIS WAN OIDs can be used to get or set characteristics of their associated NDIS drivers and WAN NICs.

The WAN OIDs are defined in ntddndis.h. When building a driver that supports these OIDs, include ntddndis.h and ndiswan.h.
