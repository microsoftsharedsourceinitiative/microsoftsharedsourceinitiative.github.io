OID_WW_TAC_COMPRESSION

The OID_WW_TAC_COMPRESSION OID requests the miniport driver to return a Boolean indicating whether the miniport driver supports network-layer compression. A value of TRUE indicates network-layer compression is in use. A value of FALSE indicates network-layer compression is not in use.

Sets and indications are irrelevant for this OID.
