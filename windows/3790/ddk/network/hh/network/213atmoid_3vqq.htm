Operational Characteristics for ATM Miniport Drivers

The following chart summarizes the OIDs used to get the operational characteristics of ATM miniport drivers and/or their NICs. Note that an ATM miniport driver handles such requests in its MiniportCoRequest function.

Length Q S Name
4 O   OID_ATM_ASSIGNED_VPI

The VPI value (if any) to be used for all cells on the interface.
Var O   OID_ATM_GET_NEAREST_FLOW

The nearest supported values of media parameters.
6 M   OID_ATM_HW_CURRENT_ADDRESS

The current hardware address of the NIC
8 O   OID_ATM_ILMI_VPIVCI

The VPI/VCI (if any) assigned to the Interim Local Management Interface
4 O   OID_ATM_MAX_AAL0_PACKET_SIZE

The maximum packet size supported for AAL0
4 O   OID_ATM_MAX_AAL1_PACKET_SIZE

The maximum packet size supported for AAL1
4 0   OID_ATM_MAX_AAL34_PACKET_SIZE

The maximum packet size supported for AAL 3/4
4 M   OID_ATM_MAX_AAL5_PACKET_SIZE

The maximum packet size supported for AAL5
4 M   OID_ATM_MAX_ACTIVE_VCI_BITS

The number of usable VCI bits
4 M   OID_ATM_MAX_ACTIVE_VPI_BITS

The number of usable VPI bits
4 M   OID_ATM_MAX_ACTIVE_VCS

The maximum active VCs supported by the miniport driver
8 O   OID_ATM_SIGNALING_VPIVCI

The VPI/VCI (if any) assigned to the signaling VC
4 M   OID_ATM_SUPPORTED_AAL_TYPES

The supported AAL types
4 M   OID_ATM_SUPPORTED_SERVICE_CATEGORY

The supported service categories.
8 O   OID_ATM_SUPPORTED_VC_RATES

The minimum and maximum cell transmission rates
