Handling Wake-Up Events

A miniport driver does not handle a wake-up event detected by its NIC. When a NIC detects an enabled wake-up event, it asserts a bus-specific wake-up line. The Power Manager then sends a power IRP to NDIS, which, in response, sends the miniport driver an OID_PNP_SET_POWER OID that requests the miniport driver to put its NIC in the highest-powered (D0) state.
