Connection-Oriented Objects

This section describes the OIDs for connection-oriented objects, both operational and statistics types.

The connection-oriented OIDs are defined in ntddndis.h. When building a driver that supports these OIDs, include ntddndis.h.

These OIDs are described in the following topics:
