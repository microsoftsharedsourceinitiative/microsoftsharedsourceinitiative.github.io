Windows Sockets Helper Functions

This section describes the system-defined WSHXxx functions that a transport driver provides in a user-mode dynamic-link library (DLL) to support the Windows Sockets 2.0 interface.

For an overview of the relationship between a transport driver's Windows Sockets helper DLL and higher layers of the Windows Sockets interface, see Windows Sockets Helper DLL Architecture. For more information about the Windows Sockets 2.0 interface to applications, see the Platform SDK.

The functions described in this section are listed in alphabetical order.
