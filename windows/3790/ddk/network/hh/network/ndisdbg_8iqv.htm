Debugging NDIS Drivers

Accurate and plentiful debugging information is the key to successfully debugging network drivers.

You can use NDIS debugger extensions to set up NDIS debug tracing and, with Windows 2000 and later versions of the operating system, to examine NDIS data structures. When you set up NDIS debug tracing, you are actually enabling one or more levels of DbgPrint statements within NDIS, which, in most cases, provides enough information to successfully debug most network driver problems.
