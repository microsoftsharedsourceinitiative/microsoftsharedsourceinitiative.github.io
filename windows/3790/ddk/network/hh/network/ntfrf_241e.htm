IEnumNetCfgComponent::Clone

The Clone method creates another IEnumNetCfgComponent interface that contains the same enumeration state as the current IEnumNetCfgComponent interface.

HRESULT
  Clone(
    OUT IEnumNetCfgComponent  **ppEnum
    );

Parameters

ppEnum
Pointer to a buffer that receives a pointer to the new IEnumNetCfgComponent interface.

Return Value

Returns one of the following values:

S_OK
The method succeeded.
E_POINTER
The ppEnum parameter is not a valid pointer.
E_OUTOFMEMORY
Insufficient memory exists to perform the operation.
E_UNEXPECTED
The method failed for unknown reasons.

Headers

Defined in Netcfgx.h. Include Netcfgx.h.
