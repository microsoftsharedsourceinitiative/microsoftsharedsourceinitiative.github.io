Registering a Shutdown Function

NDIS 5.1 miniport drivers should register an entry point for a shutdown function when calling NdisMRegisterMiniport. Other miniport drivers should call NdisMRegisterAdapterShutdownHandler to register a shutdown function.

A miniport driver passes to NdisMRegisterAdapterShutdownHandler the address of its MiniportShutdown function, the miniport driver adapter handle supplied by NDIS to MiniportInitialize, and the context to be passed to MiniportShutdown when it is called. NdisMRegisterAdapterShutdownHandler is typically called last from MiniportInitialize, just before the miniport driver returns from MiniportInitialize.

For more information about the shutdown handler, see Shutdown Handler.
