Values For NDIS Debugging Options in Windows 98

You can enable different levels of debug tracing in various NDIS components by using the following global debugging symbols:

ndistraceimpt
The trace-important flag. Setting to 1 (enabled) causes trace output to be emitted to the debugger. While debugging, this flag should always be enabled.
ndisdebuglevel
Selects the level or amount of debug tracing in NDIS components that are selected with the ndisdebugsystems value. The following values specify the levels that you can select:
Level Description Value
INFORMATION All available debug information. This is the highest level of trace. 0x00000000
LOG Log information. 0x00000800
WARNING Warnings. 0x00001000
ERROR Errors. 0x00002000
FATAL Fatal errors, which can cause the operating system to crash. This is the lowest level of trace. 0x00003000

ndisdebugsystems
Selects various NDIS components to debug trace. The following values specify the NDIS components that you can select:
Component Description Value
INITIALIZE Handles adapter initialization. 0x00000001
CONFIGURATION Handles adapter configuration. 0x00000002
SEND Handles sending data over the network. 0x00000004
RECEIVE Handles receiving data from the network. 0x00000008
MEMORY Handles memory management. 0x00000010
FILTER Handles filter operations. 0x00000020
PROTOCOL Handles protocol operations. 0x00000040
REQUEST Handles requests. 0x00000080
UNLOAD Handles unloading operations. 0x00000100
WORK ITEM Handles work-item operations. 0x00000200
OPEN Handles operations that open reference objects. 0x00000400
LOCKS Handles locking operations. 0x00000800
PLUG AND PLAY Handles Plug and Play operations. 0x00001000
PM Handles Power Management operations. 0x00002000
RESET Handles resetting operations. 0x00004000
Windows Machine
Instrumentation		 Handles Windows Machine Instrumentation operations. 0x00008000
CoSTATUS Handles Connection-Oriented status. 0x00010000
REFERENCE Handles reference operations. 0x00020000
ALL Handles all NDIS components. 0xFFFFFFFF

You can select more than one NDIS component. If more than one component is selected, OR the data values together.
