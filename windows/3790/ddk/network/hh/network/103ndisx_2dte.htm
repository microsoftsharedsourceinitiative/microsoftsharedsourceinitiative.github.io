NdisEqualAnsiString

NdisEqualAnsiString compares two ANSI strings and returns whether they are equal.

BOOLEAN 
  NdisEqualAnsiString(
    IN PNDIS_STRING  String1,
    IN PNDIS_STRING  String2,
    IN BOOLEAN  CaseInSensitive
    );

Parameters

String1
Pointer to an NDIS_STRING type that describes the first ANSI string. The NDIS_STRING type is a counted string in the system-default character set.
String2
Pointer to an NDIS_STRING type that describes the second ANSI string.
CaseInSensitive
If TRUE, case should be ignored when doing the comparison.

Return Value

NdisEqualAnsiString returns TRUE if the two ANSI strings are equal.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

Callers of NdisEqualAnsiString must be running at IRQL = PASSIVE_LEVEL.
