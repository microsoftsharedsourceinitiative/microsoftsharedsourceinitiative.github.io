NDIS WAN Operations that Support Telephonic Services

This section describes how NDIS WAN miniport drivers implement telephonic services using the functions available through the NDISTAPI driver and NDISWAN driver. A TAPI-capable NDIS WAN miniport driver brings up lines, sets up calls and closes calls using TAPI and WAN OIDs

The NDISTAPI driver communicates with telephony applications through a telephony service provider. For more information, see the Telephony Application Programming Interface (TAPI) in the Platform SDK.
