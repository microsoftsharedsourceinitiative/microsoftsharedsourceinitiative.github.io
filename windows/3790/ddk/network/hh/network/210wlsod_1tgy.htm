OID_WW_TAC_UNIQUE_SDU_TAG

The OID_WW_TAC_UNIQUE_SDU_TAG OID requests the miniport driver to return a unique (transaction) token to be used in a subsequent OID_WW_TAC_SEND_COMMAND request. This unique token is returned as an unsigned integer. This token can then be used by the protocol and miniport driver to demultiplex the NIC’s command response. For more information, see OID_WW_TAC_GET_RESPONSE.

Sets and indications are irrelevant for this operation.
