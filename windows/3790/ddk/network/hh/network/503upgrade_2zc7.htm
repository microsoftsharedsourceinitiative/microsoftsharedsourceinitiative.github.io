One-to-One ID Mapping

Note: vendor-supplied network upgrades are not supported in Windows XP (SP1 and later), Windows Server 2003, and later operating systems.

An entry in an OemXxx section of a netmap.inf file that specifies one-to-one ID mapping has the following format:

preupgrade-ID=postupgrade-ID

For example:

netservice=netservice_2000

A one-to-one ID mapping must be used for network protocols, services, and clients. Either one-to-one ID mapping or one-to-many ID mapping can be used for network adapters.
