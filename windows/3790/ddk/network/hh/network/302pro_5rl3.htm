Dynamic Binding in a Protocol Driver

A protocol driver must support dynamic binding to underlying NICs by providing both a ProtocolBindAdapter function and a ProtocolUnbindAdapter function.

When a NIC becomes available, NDIS calls the ProtocolBindAdapter function of any protocol driver that can bind itself to that NIC. When a NIC is removed, NDIS calls the ProtocolUnbindAdapter function of any protocol driver that is bound to that NIC.
