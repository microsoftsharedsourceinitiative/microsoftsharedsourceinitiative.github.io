Task Offload Objects

This section describes the OIDs that support TCP/IP task-offload operations. For a description of such operations, see NDIS Object Identifiers.

The following chart summarizes the OIDs described in this section.

Length Q S Name
Arr M M OID_TCP_TASK_OFFLOAD

When queried, requests a miniport driver to report the task-offload capabilities of its NIC. When set, requests the miniport driver to enable the specified task-offload capabilities of its NIC.
Arr   M OID_TCP_TASK_IPSEC_ADD_SA

Requests a miniport driver to add one or more security associations (SAs) to its NIC.
Arr   M OID_TCP_TASK_IPSEC_ADD_UDPESP_SA

Requests a miniport driver to add one or more UDP-ESP security associations (SAs) and possibly a parser entry.
4   M OID_TCP_TASK_IPSEC_DELETE_SA

Requests a miniport driver to delete an SA from its NIC.
4   M OID_TCP_TASK_IPSEC_DELETE_UDPESP_SA

Requests a miniport driver to delete a UDP-ESP SA and possibly a parser entry.
