Querying Miniport Driver Information

The NdisXxx function that a protocol driver calls to query miniport driver information, as well as the MiniportXxx function used to satisfy such a query, depends on whether the miniport driver is connectionless or connection-oriented.

A miniport driver can be queried directly from user mode.
