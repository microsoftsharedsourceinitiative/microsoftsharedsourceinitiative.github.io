Standard Miniport Driver OIDs Registered with WMI

OID_GEN_HARDWARE_STATUS
OID_GEN_MEDIA_SUPPORTED
OID_GEN_MEDIA_IN_USE

OID_GEN_MAXIMUM_FRAME_SIZE

OID_GEN_VENDOR_ID
OID_GEN_VENDOR_DESCRIPTION
