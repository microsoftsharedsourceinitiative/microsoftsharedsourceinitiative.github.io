Indicating and Transferring Received Data

To indicate one or more receive packets to higher drivers, connectionless miniports call NdisMIndicateReceivePacket; connection-oriented packets call NdisMCoIndicateReceivePacket.

Legacy connectionless miniports that do not support NdisMIndicateReceivePacket call a medium-specific NdisMXxxIndicateReceive function to indicate a single packet to higher drivers.
