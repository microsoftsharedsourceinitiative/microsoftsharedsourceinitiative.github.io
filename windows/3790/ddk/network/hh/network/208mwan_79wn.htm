Implementing NDIS WAN Miniport Drivers

This section provides details about implementing NDIS WAN miniport drivers.

Developers of NDIS WAN miniport drivers should also refer to other Miniport Drivers sections on non-WAN connectionless miniport drivers and to the Network Reference for details on the many NDIS functions.

For information about the differences between NDIS WAN miniport drivers and non-WAN connectionless miniport drivers, see Differences Between NDIS WAN and non-WAN Connectionless Miniport Drivers.
