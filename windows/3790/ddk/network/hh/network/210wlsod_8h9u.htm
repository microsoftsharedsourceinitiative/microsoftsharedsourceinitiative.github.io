OID_WW_CDPD_AREA_COLOR

The OID_WW_CDPD_AREA_COLOR OID requests the miniport driver to return the current area color code of the acquired channel. An area color code is an integer of type INT that identifies the Mobile Data Intermediate System to which the user’s base station is connected.

If any bound protocol has enabled indications with OID_WW_GEN_INDICATION_REQUEST, the miniport driver must call NdisMIndicateStatus to notify such protocol(s) if the area color state changes. It can ignore the TriggerValue member that was specified in the OID_WW_GEN_INDICATION_REQUEST.

Sets are irrelevant for this OID.
