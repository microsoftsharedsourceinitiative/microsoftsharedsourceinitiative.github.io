OIDs for Connection-Oriented Call Managers and Clients

The following chart summarizes the OIDs that connection-oriented clients can send to call managers or MCM drivers and that call managers or MCM drivers can send to connection-oriented clients.

Length Q S Name
Varies   O OID_CO_ADD_PVC
Adds a PVC to the call manager's list of configured PVCs
Varies   O OID_CO_DELETE_PVC
Deletes a PVC from the call manager's list of configured PVCs
      OID_CO_GET_CALL_INFORMATION
Reserved for future use
Varies   O OID_CO_ADD_ADDRESS
Supplies an alias host address to the call manager
Varies   O OID_CO_DELETE_ADDRESS
Deletes an alias host address from the call manager
Varies O   OID_CO_GET_ADDRESSES
Returns a list of addresses from the call manager
0   O OID_CO_ADDRESS_CHANGE
Notifies clients of a change in the call manager's address list
0   O OID_CO_SIGNALING_ENABLED
Notifies clients that the call manager is ready to make calls and dispatch incoming calls
0   O OID_CO_SIGNALING_DISABLED
Notifies clients that the call manager cannot make calls and dispatch incoming calls
0   M OID_CO_AF_CLOSE
Notifies clients that they should close all address families that they have open with the call manager
