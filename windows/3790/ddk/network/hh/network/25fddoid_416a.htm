FDDI Operational Characteristics

The following chart summarizes the mandatory OIDs used to get or set FDDI operational characteristics for NDIS drivers and NICs.

Length Q S Name

Permanent station address
6 M   OID_FDDI_LONG_CURRENT_ADDR

Multicast list

Maximum list size
2 M   OID_FDDI_SHORT_PERMANENT_ADDR
Permanent station address. This OID is not supported in Windows XP (SP1 and later), Windows Server 2003, and later operating systems.
