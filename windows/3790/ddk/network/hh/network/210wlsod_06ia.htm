OID_WW_GEN_BATTERY_LEVEL

The OID_WW_GEN_BATTERY_LEVEL OID requests the miniport driver to return the current charge of the battery as follows:

Value Meaning
0-100 The charge of the battery as a percentage, where 100 means it is fully charged.
-1 The charge state is unknown.

If any bound protocol has enabled indications with OID_WW_GEN_INDICATION_REQUEST, the miniport driver must call NdisMIndicateStatus to notify such protocol(s) according to the TriggerValue that was specified in the OID_WW_GEN_INDICATION_REQUEST.

Sets are irrelevant for this OID.
