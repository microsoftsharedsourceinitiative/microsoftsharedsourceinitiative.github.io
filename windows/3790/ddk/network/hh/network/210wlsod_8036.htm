OID_WW_CDPD_CS_SERVICE_STATUS

The OID_WW_CDPD_CS_SERVICE_STATUS OID requests the miniport driver to return which CDPD service is currently in use. The value is returned as an integer.

One of the following values designates which service is currently in use:

Value Meaning
0 Packet-switched CDPD
1 CS-CDPD via AMPS
2 CS-CDPD via PSTN
-1 Unknown

All other values are reserved.

Sets and indications are irrelevant for this OID.
