WSHGetProviderGuid

The WSHGetProviderGuid function returns the GUID that identifies the protocols supported by the helper DLL. This function is called only during setup of the WSH DLL.

INT
  WSHGetProviderGuid(
    IN LPWSTR  ProviderName,
    OUT LPGUID  ProviderGuid
    );

Parameters

ProviderName
Pointer to a zero-terminated string specifying the name of the provider.
ProviderGuid
Pointer to a variable in which WSHGetProviderGuid returns the GUID that identifies the protocols supported by the helper DLL. A GUID can be generated with UUIDGEN.EXE. For more information about this utility, see the Platform SDK.

Return Value

WSHGetProviderGuid returns NO_ERROR for success or WSAEINVAL if the string at ProviderName does not designate this helper DLL.

Headers

Declared in wsahelp.h. Include wsahelp.h.
