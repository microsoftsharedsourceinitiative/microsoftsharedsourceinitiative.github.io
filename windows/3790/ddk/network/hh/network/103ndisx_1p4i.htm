NdisGetSystemUpTime

NdisGetSystemUpTime returns the number of milliseconds that have elapsed since the system was booted.

VOID 
  NdisGetSystemUpTime(
    OUT PULONG  pSystemUpTime
    );

Parameters

pSystemUpTime
Pointer to a caller-supplied variable in which this function returns the system uptime.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

NdisGetSystemUpTime is typically called by protocol drivers.

Callers of NdisGetSystemUpTime can be running at any IRQL.
