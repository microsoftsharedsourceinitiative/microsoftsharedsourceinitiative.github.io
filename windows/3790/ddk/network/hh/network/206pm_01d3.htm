Handling an OID_PNP_SET_POWER OID

NDIS sends an OID_PNP_SET_POWER request to notify a miniport driver that its NIC will be making a transition from the working state to a sleeping state or from a sleeping state to the working state. OID_PNP_SET_POWER can be preceded by an OID_PNP_QUERY_POWER request.
