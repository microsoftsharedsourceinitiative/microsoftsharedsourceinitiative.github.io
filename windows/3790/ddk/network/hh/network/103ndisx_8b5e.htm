NdisDestroyBlockPool

NdisDestroyBlockPool releases resources for a block of memory pool.

VOID
  NdisDestroyBlockPool(
    IN NDIS_HANDLE  BlockPoolHandle
    );

Parameters

BlockPoolHandle
Handle to the block to release.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

The NdisCreateBlockPool function is called to create the block.

Callers of NdisDestroyBlockPool run at IRQL <= DISPATCH_LEVEL.
