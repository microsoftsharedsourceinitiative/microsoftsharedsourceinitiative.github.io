NDIS Debug Extensions For Windows Me

For Windows Me, you can use a checked version of the NDIS library of functions (Ndis.vxd) plus debugging options in NDIS to produce debug output for various network drivers. You enable most of these debugging options in the registry. You can set one debugging option only during a debugger session; it is not available in the registry. You can also use debugging options in the network class installer (Netdi.dll) to produce debug output during network setup and installation.

The following topics describe debugging options that you set in the registry and the debugging option that you set in the debugger:

Setting NDIS Debugging Options in the Registry

Setting NDIS Debugging Options in the Debugger

Setting Debugging Options in the Network Class Installer

Note  You must disable System File Protection (SFP) before you copy the checked version of Ndis.vxd to the Windows\System directory of the target computer. To ensure that SFP is disabled, either start from the Microsoft Windows Me startup floppy or start Windows Me in safe mode. To start in safe mode, press F8 when Windows Me starts, and then select safe mode from the menu of boot options.
