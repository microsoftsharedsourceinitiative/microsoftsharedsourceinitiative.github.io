OID_802_11_REMOVE_WEP

The OID_802_11_REMOVE_WEP OID requests the miniport driver to remove the wired equivalent privacy (WEP) key at the specified key index. The key index is defined as follows:

typedef  ULONG    NDIS_802_11_KEY_INDEX;

For more information about the WEP key, see OID_802_11_ADD_WEP.
