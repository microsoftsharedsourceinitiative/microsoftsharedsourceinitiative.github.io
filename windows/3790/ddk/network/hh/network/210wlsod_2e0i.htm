OID_WW_CDPD_CS_DEFAULT_DIAL_CODE

As a query, the OID_WW_CDPD_CS_DEFAULT_DIAL_CODE OID requests the miniport driver to return the current default circuit-switched CDPD dial code for establishing connections.

As a set, this OID requests the miniport driver to set the current default circuit-switch CDPD dial code for establishing connections.

In either case, this request uses a WW_CDPD_CS_DEFAULT_DIAL_CODE structure, defined as equivalent to the NDIS_VAR_DATA_DESC structure. The buffer contains the default dial code for circuit-switched CDPD service. This dial code is encoded per circuit-switched CDPD implementation guidelines. The maximum length of the dial code is 20 bytes.

Indications are irrelevant for this OID.
