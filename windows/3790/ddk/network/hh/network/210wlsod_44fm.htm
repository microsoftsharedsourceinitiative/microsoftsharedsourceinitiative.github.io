OID_WW_TAC_DCHANNEL_TABLE

The OID_WW_TAC_DCHANNEL_TABLE OID requests the miniport driver to return the DataTAC-dynamic-channel list that is currently in use by the NIC.

This OID uses the WW_TAC_CHANNEL_TABLE structure, that was previously defined in OID_WW_TAC_CHANNEL_TABLE.

Protocols use this OID to request the dynamic-DataTAC-channel list. The channel-numbering scheme and contents of the channel-attributes array are defined by the DataTAC RD-LAP protocol standard.

Typically, this OID can be used to provide a control panel application with a channel list. The application can then present the list to the user.

Sets and indications are irrelevant for this OID.
