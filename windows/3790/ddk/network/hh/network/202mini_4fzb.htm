NDIS-Supplied Query and Set Completion Functions

The following table summarizes the query and set completion functions provided by NDIS for miniport drivers.

Function Definition
NdisMCoRequestComplete
  Indicates that a prior call to MiniportCoRequest has completed.
NdisMQueryInformationComplete
  Indicates that a prior call to MiniportQueryInformation has completed.
NdisMSetInformationComplete
  Indicates that a prior call to MiniportSetInformation has completed.
