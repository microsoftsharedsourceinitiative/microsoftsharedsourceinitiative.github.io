Intermediate Driver Platform-Specific Differences

The following NDIS functions are not supported on Windows 98/Me:

  • NdisIMCopySendPerPacketInfo
  • NdisIMCopySendCompletePerPacketInfo
  • NdisIMCancelInitializeDeviceInstance

NDIS 5.1 features for intermediate drivers (packet stacking, query pending I/O count) are supported only on Windows XP and later versions.
