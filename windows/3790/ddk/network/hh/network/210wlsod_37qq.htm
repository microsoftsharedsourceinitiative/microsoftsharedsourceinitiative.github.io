Wireless WAN Objects

This section describes the system-defined OIDs for wireless WAN media. These OIDs include a set of general OIDs defined for wireless WAN media, as well as OIDs that are specific to a given medium.

The wireless WAN OIDs are defined in ntddndis.h. When building a driver that supports these OIDs, include ntddndis.h.

Wireless Operational Characteristics

Network-Dependent Wireless Objects
