NdisSetPacketStatus

NdisSetPacketStatus sets status for a packet.

VOID 
  NdisSetPacketStatus(
    IN PNDIS_PACKET  Packet,
    IN NDIS_STATUS  Status,
    IN NDIS_HANDLE  Handle,
    IN ULONG  Code
    );

Parameters

Packet
Pointer to an NDIS_PACKET structure that is the descriptor for a packet.
Status
Specifies the NDIS_STATUS_Xxx code describing the status to set for the packet.
Handle
Handle to a context area. This context area determines a particular packet on a particular NIC.
Code
Specifies a code value describing the status to set for the packet.

Headers

Declared in Ndis.h. Include Ndis.h.

Comments

Callers of NdisSetPacketStatus run at IRQL <= DISPATCH_LEVEL.
