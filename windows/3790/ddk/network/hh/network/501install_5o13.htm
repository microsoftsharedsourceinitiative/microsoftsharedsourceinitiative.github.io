Ndi Values and Keys Not Used in Windows 2000 and Later Versions

The following Ndi registry keys and values are available in Windows 95/98/Me but are not used in Windows 2000 and later versions of the operating system. This information is provided to aid developers who are migrating network drivers from Windows 95/98/Me to Windows 2000 and later versions of the operating system.

In particular, Windows 2000 does not support Ndi\param-key-name\resc and Ndi\Params\param-key-name\flag values. This means that a user cannot specify adapter resources through the Advanced properties page.
