Ardis Wireless OIDs

The Ardis network conforms to current DataTAC technology. Therefore, Ardis-specific wireless OIDs are obsolete and should not be used. Definitions for these OIDs are still included for informational and historical purposes. Use DataTAC-specific wireless OIDs in DataTAC Wireless OIDs instead.
