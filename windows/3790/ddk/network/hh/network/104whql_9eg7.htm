Prohibited Kernel-Mode Routines for NDIS Drivers

WHQL restricts NDIS drivers—that are not also NDIS-WDM—from calling the following kernel-mode support routines. If an NDIS driver calls one of these routines, WHQL fails the NDIS driver and, in most cases, returns an error message that simply states, "Do not use this call." Otherwise, the error message specifies the use of alternate NDIS functions.

Note  WHQL allows NDIS-WDM drivers to call the routines in the following list.
