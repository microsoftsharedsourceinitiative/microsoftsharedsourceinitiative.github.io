Installing NDIS Debugger Extensions

Perform the following procedure to install NDIS Debugger extensions so that you can either enable debug tracing or use other extensions:

To install the NDIS Debugger extensions

  1. Depending on whether the target computer is running a checked or free build of the operating system, copy either the checked or free version of Ndiskd.dll respectively from the Debugger Tools for Windows package to the directory that contains the debugger to be used. For example, the debugger could be either Windbg.exe or i386kd.exe.
  2. Copy the corresponding Ndis.sys symbols files (for example, the Ndis.pdb and Ndis.dbg files) to the symbols directory of the host computer.
