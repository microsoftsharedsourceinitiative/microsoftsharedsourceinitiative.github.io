Initializing a NIC

From its MiniportInitialize function, a miniport driver must initialize its NIC. This process involves three major steps:

  1. The miniport driver allocates memory for the adapter-specific context area in which it typically maintains the NIC state.
  2. The miniport driver reads configuration information and stores this information in the adapter-specific context area.
  3. The miniport driver registers the NIC with NDIS.
