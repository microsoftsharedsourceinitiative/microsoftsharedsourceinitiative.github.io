Receiving Data

To indicate one or more receive packets to higher-level drivers, connectionless miniports call NdisMIndicateReceivePacket; and connection-oriented packets call NdisMCoIndicateReceivePacket.

Legacy connectionless miniports that do not support NdisMIndicateReceivePacket call a medium-specific NdisMXxxIndicateReceive function to indicate a single packet to higher-level drivers.
