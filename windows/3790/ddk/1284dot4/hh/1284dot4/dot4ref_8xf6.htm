DOT4_DRIVER_CMD

A DOT4_DRIVER_CMD structure is used with all Dot4 function I/O control codes to specify command information.

typedef struct _DOT4_DRIVER_CMD
{
  CHANNEL_HANDLE  hChannelHandle;
  ULONG  ulSize;
  ULONG  ulOffset;
  ULONG  ulTimeout;
} DOT4_DRIVER_CMD, *PDOT4_DRIVER_CMD;

Members

hChannelHandle
Specifies a Dot4 channel handle that is obtained by using an IOCTL_DOT4_OPEN_CHANNEL request.
ulSize
Specifies the size of a client-allocated buffer that is used with read and write requests.
ulOffset
Specifies the initial byte offset in the client-allocated buffer that is used with read and write requests.
ulTimeout
Specifies a timeout value, in milliseconds, for the request.

Headers

Declared in d4drvif.h. Include d4drvif.h.

Comments

The setting of the members is request-specific.
