DOT4_WMI_XFER_INFO

A DOT4_WMI_XFER_INFO structure is used to output information about data flow over Dot4 channels.

typedef struct _DOT4_WMI_XFER_INFO
{
  ULONG  ulStreamBytesWritten;
  ULONG  ulStreamBytesRead;
  ULONG  ulPacketBytesWritten;
  ULONG  ulPacketBytesRead;
} DOT4_WMI_XFER_INFO, *PDOT4_WMI_XFER_INFO;

Members

ulStreamBytesWritten
Specifies the total number of data bytes written over stream-type channels. The value returned does not include the IEEE 1284.4 packet headers.
ulStreamBytesRead
Specifies the total number of data bytes read over stream-type channels. The value returned does not include the IEEE 1284.4 packet headers.
ulPacketBytesWritten
Specifies the total number of data bytes written over packet-type channels. The value returned does not include the IEEE 1284.4 packet headers.
ulPacketBytesRead
Specifies the total number of bytes read over packet-type channels. The value returned does not include the IEEE 1284.4 packet headers.
