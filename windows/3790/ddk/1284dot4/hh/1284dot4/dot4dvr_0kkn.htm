The Microsoft Dot4 Driver

This documentation is provided to support existing devices only. Microsoft recommends that hardware vendors design new multifunction print devices using composite USB interfaces, and strongly recommends against using IEEE 1284.4 interfaces. This documentation might be removed from all future versions of the Windows DDK. Microsoft-supplied drivers that support IEEE 1284.4 interfaces might be removed from all future versions of the Windows operating system.

This section includes the following topics about the Dot4 driver, the system-supplied function and bus driver for the Dot4 device setup class in Microsoft® Windows® 2000 and later:
