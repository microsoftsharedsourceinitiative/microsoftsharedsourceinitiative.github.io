Dot4 Socket

A Dot4 socket is a logical endpoint for a Dot4 channel. A Dot4 channel is associated with an IEEE 1284.4 primary socket on the host system (the IEEE 1284.4 primary device) and an IEEE 1284.4 secondary socket on the peripheral (an IEEE 1284.4 secondary device).

A client binds a primary socket to a secondary socket by using an IOCTL_DOT4_CREATE_SOCKET request. A client uses a DOT4_DC_CREATE_DATA structure to input and output information.

A Dot4 socket can be a packet-type socket or a stream-type socket. The Dot4 driver transmits data over a packet-type socket using independent packets. The maximum amount of data that can be transmitted over a packet-type socket by a single read or write request is 65529 bytes. The Dot4 driver transmits data over a stream-type socket in a continuous, ordered sequence of packets.
