IEEE 1284.4 Devices: Windows DDK

Vendor-Supplied Drivers for Dot4 Devices

A vendor usually does not have to supply drivers for a Dot4 device because:

  • Microsoft provides the Dot4 driver, which is the system function and bus driver for the Dot4 device setup class.
  • The Dot4 driver provides the necessary functionality to directly operate a Dot4 device's child devices, or Dot4 functions. A client uses the Dot4 function I/O control codes that the Dot4 driver supports to operate a Dot4 function.

A vendor only has to supply a WDM function driver for a Dot4 function if the vendor chooses to provide an upper-level interface for the Dot4 function that the Dot4 driver does not provide. However, Microsoft requires that the function driver only use the services of the Dot4 driver to actually operate the Dot4 function.
