Dot4 Functions Registry Key

The following Functions key is located under a Dot4 device's Plug and Play hardware device key:

Functions key
Contains a set of Function name subkeys for a Dot4 device.
Functions\<Function name> key
Specifies the name of a Dot4 function supported by the Dot4 device. Each Function name corresponds to an independent function supported by the device. The Dot4 driver creates a named physical device object for each function name. The device name has the format "\Device\DOT4\<Function name><Bus number>". A Function name key has the following entry value:
Description entry value (REG_SZ)
Contains function type.
