DOT4_DC_DESTROY_DATA

A DOT4_DC_DESTROY_DATA structure is used with an IOCTL_DOT4_DESTROY_SOCKET request.

typedef struct _DOT4_DC_DESTROY_DATA
{
    unsigned char  bHsid;
} DOT4_DC_DESTROY_DATA, *PDOT4_DC_DESTROY_DATA;

Members

bHsid
Specifies the socket ID of the IEEE 1284.4 primary socket to delete.

Headers

Declared in d4drvif.h. Include d4drvif.h.
