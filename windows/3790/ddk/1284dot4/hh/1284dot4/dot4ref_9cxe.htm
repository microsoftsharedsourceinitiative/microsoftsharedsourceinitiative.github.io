IOCTL_DOT4_WAIT_FOR_CHANNEL

Operation

A client uses an IOCTL_DOT4_WAIT_FOR_CHANNEL request to wait for a Dot4 device to open a specified Dot4 channel.

The client can specify that the Dot4 driver add Dot4 activity broadcasting immediately after the channel is opened.

Input

Irp->AssociatedIrp.SystemBuffer is a pointer to a client-allocated DOT4_DRIVER_CMD structure, whose members must be set to zero.

The Parameters.DeviceIoControl.InputBufferLength member of the IRP's current I/O location is set to the size, in bytes, of a DOT4_DRIVER_CMD structure.

Irp->MdlAddress is a pointer to a client-allocated MDL buffer for a DOT4_DC_OPEN_DATA structure that inputs and outputs the following information:

bHsid
Specifies an IEEE 1284.4 primary socket ID, which is obtained using an IOCTL_DOT4_CREATE_SOCKET request.
fAddActivity
Specifies whether to add activity broadcasting on a channel immediately after it is opened. The fAddActivity member is set to TRUE (nonzero) to add activity broadcasting, otherwise fAddActivity is set to FALSE (zero).

The Parameters.DeviceIoControl.OutputBufferLength member of the IRP's current I/O stack location is set to the size, in bytes, of a DOT4_DC_OPEN_DATA structure.

Output

Irp->MdlAddress is a pointer to the client-allocated MDL buffer for the DOT4_DC_OPEN_DATA structure that outputs the following information:

hChannelHandle
Specifies a channel handle.

Headers

Defined in d4drvif.h. Include d4drvif.h.

I/O Status Block

The Information member is set to the size, in bytes, of a DOT4_DC_OPEN_DATA structure.

The Status member is set to one of the generic status values for Dot4 IOCTLS or to one of the following request-specific values:

STATUS_BUFFER_TOO_SMALL
The Parameters.DeviceIoControl.OutputBufferLength member of the IRP's current I/O stack location is less than the size, in bytes, of a DOT4_DC_OPEN_DATA structure.
STATUS_NO_MEMORY
The system did not successfully map the client-allocated MDL buffer for the DOT4_DC_OPEN_DATA structure that inputs and outputs information.

Headers

Defined in d4drvif.h. Include d4drvif.h.
