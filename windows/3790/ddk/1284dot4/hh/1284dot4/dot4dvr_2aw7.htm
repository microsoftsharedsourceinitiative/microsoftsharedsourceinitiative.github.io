Dot4 Timeout Registry Key

The following Timeout key is located under a Dot4 device's Plug and Play hardware device key:

Timeout key
Contains the following timeout entry values for a Dot4 device:
InterByteTimeout entry value (REG_DWORD)
Specifies the maximum time that can occur between packet parts before a timeout on the Dot4 data link can occur. The default value is 500 (in milliseconds).
StandardPacketTimeout entry value (REG_DWORD)
The value of the StandardPacketTimeout entry value specifies the maximum time that a peripheral can take to reply to a command packet. The default value is 3000 (in milliseconds).
CreditTimeout entry value (REG_DWORD)
Specifies the maximum time Dot4 waits to be granted credits by a Dot4 device. The default value is 5000 (in milliseconds).

The Dot4 driver creates a Timeout key if one does not exist. If an entry values does not exist under the Timeout key, The Dot4 driver creates the entry value and sets it to the specified default value.
