Tape Class Driver Routines

The system-supplied tape class driver handles generic tape operations and provides OS-specific support to individual tape miniclass drivers.

This section describes the routines exported by the tape class driver for use by tape miniclass drivers. The routines are listed in alphabetical order.

These routines are defined in minitape.h.
