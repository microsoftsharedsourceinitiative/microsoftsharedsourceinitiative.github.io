FEATURE_DATA_MRW

The FEATURE_DATA_MRW structure contains information about the MRW feature. 

typedef struct _FEATURE_DATA_MRW {
  FEATURE_HEADER Header;
  UCHAR  Write:1;
  UCHAR  Reserved1:7;
  UCHAR  Reserved2[3];
} FEATURE_DATA_MRW, *PFEATURE_DATA_MRW;

Members

Header
Contains a FEATURE_HEADER structure with header information for this feature descriptor.
Write
Indicates, if set to 1, that the device can format discs using the MRW format and write to discs that have been formatted in this manner. See the SCSI Multimedia - 3 (MMC-3) specification for more information.
Reserved1
Reserved.
Reserved2
Reserved.

Headers

Declared in ntddmmc.h. Include ntddmmc.h.
