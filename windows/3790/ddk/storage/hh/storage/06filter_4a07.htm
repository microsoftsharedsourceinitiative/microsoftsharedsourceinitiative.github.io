Storage Filter Driver's DriverEntry Routine

Like any other kernel-mode driver, the DriverEntry routine of an SFD must define its Dispatch and other entry points in the input driver object. If necessary, an SFD can allocate a driver object extension of appropriate size by calling IoAllocateDriverObjectExtension, copy the input registry path into the driver extension for later use, and initialize the driver extension with other driver-determined data, if any.

For more information about a PnP driver's DriverEntry routine, see Plug and Play.
