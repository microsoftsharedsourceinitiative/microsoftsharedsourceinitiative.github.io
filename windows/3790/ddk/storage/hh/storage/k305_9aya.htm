ChangerClassCreateClose

The ChangerClassCreateClose routine is called by a changer minidriver to allow the class driver perform device-independent operations needed to create or close a device. 

NTSTATUS
ChangerClassCreateClose (
  IN PDEVICE_OBJECT  DeviceObject,
  IN PIRP  Irp
  )

Parameters

DeviceObject
Pointer to the device object of the device to be created or closed.
Irp
Pointer to the I/O request packet (IRP) that initiated the create or close operation.

Return Value

If the operation succeeds, the ChangerClassCreateClose routine returns STATUS_SUCCESS. Otherwise the routine returns one of the following status values.

Status Meaning
STATUS_NO_SUCH_DEVICE The device object does not have a properly initialized device extension.
STATUS_DEVICE_BUSY The device is already open. Only one open at a time is allowed.

Headers

Declared in mcd.h. Include mcd.h and ntddchgr.h.
