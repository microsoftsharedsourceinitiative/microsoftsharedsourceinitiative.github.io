ChangerClassFreePool

The ChangerClassFreePool routine frees pool memory previously allocated using ChangerClassAllocatePool. 

VOID
  ChangerClassFreePool(
    IN PVOID  PoolToFree
    );

Parameters

PoolToFree
Pointer to the block of memory to be freed.

None
