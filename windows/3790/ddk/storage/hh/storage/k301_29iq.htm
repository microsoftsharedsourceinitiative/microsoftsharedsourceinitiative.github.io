HwStorBuildIo

The HwStorBuildIo routine does unsynchronized processing of an SRB before passing it to HwStorStartIo.

BOOLEAN
HwStorBuildIo(
    IN PVOID  DeviceExtension,
    IN PSCSI_REQUEST_BLOCK  Srb
    );

Parameters

DeviceExtension
Pointer to the miniport driver's per-HBA storage area.
Srb
Pointer to the SCSI request block (SRB) to be processed.

Return Value

HwStorBuildIo returns TRUE to inform the caller that the SRB can be forwarded to the HwStorStartIo routine. HwStorBuildIo returns FALSE to inform the caller that the SRB should not be passed on to HwStorBuildIo. Usually, in such cases, the miniport completes the SRB from within HwStorBuildIo by calling StorPortNotification with a notification type of RequestComplete.

Headers

Declared in storport.h. Include storport.h.

Comments

The name HwStorBuildIo is just a placeholder. The actual prototype of this routine is defined in storport.h as follows:

typedef
BOOLEAN
(*PHW_BUILDIO) (
  IN PVOID  DeviceExtension,
  IN PSCSI_REQUEST_BLOCK  Srb
  );
