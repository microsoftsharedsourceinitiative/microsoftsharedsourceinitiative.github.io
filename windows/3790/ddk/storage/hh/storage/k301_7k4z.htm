StorPortQuerySystemTime

The StoriPortQuerySystemTime routine obtains the current system time.

VOID 
  StorPortQuerySystemTime(
    OUT PLARGE_INTEGER  CurrentTime
    );

Parameters

CurrentTime
Pointer to the current time, on return.

Return Value

None

Headers

Declared in srb.h. Include miniport.h and scsi.h.

Comments

The system time returned in CurrentTime is the number of 100-nanosecond intervals that have elapsed since January 1, 1601. System time is typically updated approximately every ten milliseconds. This value is computed for the GMT time zone. Callers of StorPortQuerySystemTime can be running at any IRQL.
