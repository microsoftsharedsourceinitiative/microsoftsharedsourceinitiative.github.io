Changer Class Driver Routines

This section describes the changer class driver routines that a changer miniclass driver can call. The routines are listed in alphabetical order.

The changer class driver routines are declared in mcd.h. The structures and constants required by a changer class driver are declared in ntddchgr.h.

For information about the changer class/miniclass architecture, see Changer Drivers.
