Storage Devices: Windows DDK

IOCTL_STORAGE_RELEASE

Operation

Releases a device previously reserved for the exclusive use of the caller on a bus that supports multiple initiators and the concept of reserving a device, such as a SCSI bus.

Input

None

Output

None

I/O Status Block

The Information field is set to zero. The Status field is set to STATUS_SUCCESS, or possibly to STATUS_INVALID_DEVICE_REQUEST.

Headers

Defined in ntddstor.h. Include ntddstor.h.
