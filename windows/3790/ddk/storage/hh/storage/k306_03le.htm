FEATURE_DATA_DISC_CONTROL_BLOCKS

The FEATURE_DATA_DISC_CONTROL_BLOCKS structure holds an array of structures of type FEATURE_DATA_DISC_CONTROL_BLOCKS_EX, each element of which contains the data reported for a the Disc Control Block feature. 

typedef struct _FEATURE_DATA_DISC_CONTROL_BLOCKS {
  FEATURE_HEADER  Header;
  FEATURE_DATA_DISC_CONTROL_BLOCKS_EX  Data[0];
} FEATURE_DATA_DISC_CONTROL_BLOCKS, *PFEATURE_DATA_DISC_CONTROL_BLOCKS;

Members

Header
Contains a FEATURE_HEADER structure with header information for this feature descriptor.
Data
Contains zero, one, or more disk control blocks. Each disk control block is contained in a FEATURE_DATA_DISC_CONTROL_BLOCKS_EX structure.

Headers

Declared in ntddmmc.h. Include ntddmmc.h.

Comments

This structure holds data for the feature named "Disc Control Blocks" by the SCSI Multimedia - 3 (MMC-3) specification. Devices that support this feature can do reads and writes of disc control blocks.
