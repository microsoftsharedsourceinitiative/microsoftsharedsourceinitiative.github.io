SET_PARTITION_INFORMATION_MBR

The SET_PARTITION_INFORMATION_MBR structure defines the information in a Master Boot Record (MBR) partition that is not common to both GUID Table Partition (GPT) and MBR partitions.

typedef SET_PARTITION_INFORMATION SET_PARTITION_INFORMATION_MBR;

Headers

Declared in ntddk.h and ntdddisk.h. Include ntddk.h or ntdddisk.h.

Comments

The SET_PARTITION_INFORMATION_MBR structure is actually equivalent to the SET_PARTITION_INFORMATION structure.
