IOCTL_CDROM_SET_VOLUME

Operation

The driver resets the volume for its device's audio ports.

Input

The buffer at Irp->AssociatedIrp.SystemBuffer contains the VOLUME_CONTROL value(s) to be set. Parameters.DeviceIoControl.InputBufferLength in the I/O stack location indicates the size in bytes of the buffer, which must be >= sizeof(VOLUME_CONTROL).

Output

None

I/O Status Block

The Information field is set to zero. The Status field is set to STATUS_SUCCESS, or possibly to STATUS_BUFFER_TOO_SMALL, STATUS_DEVICE_NOT_READY, STATUS_IO_TIMEOUT, STATUS_IO_DEVICE_ERROR, STATUS_VERIFY_REQUIRED, or STATUS_INVALID_DEVICE_REQUEST.

Headers

Defined in ntddcdrm.h. Include ntddcdrm.h.
