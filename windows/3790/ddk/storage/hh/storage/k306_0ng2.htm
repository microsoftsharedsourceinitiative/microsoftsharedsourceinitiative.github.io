FEATURE_DATA_CDRW_CAV_WRITE

The FEATURE_DATA_CDRW_CAV_WRITE structure contains information about the CD-RW CAV Write feature. 

typedef struct _FEATURE_DATA_CDRW_CAV_WRITE {
  FEATURE_HEADER  Header;
  UCHAR  Reserved1[4];
} FEATURE_DATA_CDRW_CAV_WRITE, *PFEATURE_DATA_CDRW_CAV_WRITE;

Headers

Declared in ntddmmc.h. Include ntddmmc.h.

Comments

This structure holds data for the feature named "CD-RW CAV Write" by the SCSI Multimedia - 3 (MMC-3) specification. Devices that support this feature can perform writes on CD-RW media in CAV mode.
