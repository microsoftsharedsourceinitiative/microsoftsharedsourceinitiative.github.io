StorPortReadPortBufferUshort

The StorPortReadPortBufferUshort routine reads a value from a specified port address. 

VOID
  StorPortReadPortBufferUchar(
    IN PUSHORT  Port,
    IN PUSHORT  Buffer,
    IN ULONG  Count
    )

Parameters

Port
Pointer to the address from which to read.
Buffer
Pointer to the buffer that receives the data that is read.
Count
Number of data items to be read. Each data item has a size of sizeof(USHORT).

Return Value

None

Headers

Declared in storport.h. Include storport.h.

Comments

For more information, see ScsiPortReadPortBufferUshort. For a non-buffered version of this routine, see StorPortReadPortUshort.
