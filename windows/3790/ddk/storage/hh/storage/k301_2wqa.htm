StorPortInitialize

The StorPortInitilize routine initializes the operating-system-supplied Storport driver, saving the HwInitializationData value to use later when initializing the adapter. 

ULONG
  StorPortInitialize(
    IN PVOID  Argument1,
    IN PVOID  Argument2,
    IN PHW_INITIALIZATION_DATA  HwInitializationData,
    IN PVOID  HwContext
    );

Parameters

Argument1
Is the first pointer with which the operating system called the miniport driver's DriverEntry routine.
Argument2
Is the second pointer with which the operating system called DriverEntry.
HwInitializationData
Pointer to the initialization and configuration information supplied by DriverEntry.
HwContext
Is the address of a context value to be passed to the miniport driver's HwStorFindAdapter routine. Only legacy miniport drivers that scan the bus for HBAs rather than receiving configuration information from the port driver can use this parameter to store state between calls to HwStorFindAdapter.

Return Value

StorPortInitialize returns a status value that is used as the return value from the miniport driver's DriverEntry routine.

Headers

Declared in storport.h. Include storport.h.

Comments

This routine must be called from the miniport driver's DriverEntry routine.

Because Storport miniport drivers must support PnP, the Storport driver does not use the HwContext parameter passed to StorPortInitialize.
