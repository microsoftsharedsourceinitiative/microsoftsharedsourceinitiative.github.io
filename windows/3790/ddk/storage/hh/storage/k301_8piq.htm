StorPortReadPortUchar

The StorPortReadPortUchar routine reads a value from a specified port address 

UCHAR
  StorPortReadPortUCHAR(
    IN PUCHAR  Port
    );

Parameters

Port
Pointer to the address from which to read.

Return Value

StorPortReadPortUchar returns an unsigned character of data.
