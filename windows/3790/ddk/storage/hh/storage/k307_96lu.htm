Storage Devices: Windows DDK

IOCTL_DISK_GET_DRIVE_LETTERS

Operation

None. This IOCTL is obsolete. Windows 2000 and later operating systems store persistent names for volumes rather than hard drives. For more information on how to retrieve persistent names for volumes, see IOCTL_MOUNTMGR_QUERY_POINTS.
