FEATURE_DATA_DVD_RECORDABLE_WRITE

The FEATURE_DATA_DVD_RECORDABLE_WRITE structure holds information for the DVD-R/RW Write feature.

typedef struct _FEATURE_DATA_DVD_RECORDABLE_WRITE {
  FEATURE_HEADER Header;
  UCHAR  Reserved1:2;
  UCHAR  DVD_RW:1;
  UCHAR  TestWrite:1;
  UCHAR  Reserved2:3;
  UCHAR  BufferUnderrunFree:1;
  UCHAR  Reserved3:1;
  UCHAR  Reserved4[3];
} FEATURE_DATA_DVD_RECORDABLE_WRITE, *PFEATURE_DATA_DVD_RECORDABLE_WRITE;

Members

Header
Contains a FEATURE_HEADER structure with header information for this feature descriptor.
Reserved1
Reserved.
DVD_RW
Indicates, when set to 1, that the device supports writing and erasing on DVD-RW media. For more information on this feature see the SCSI Multimedia - 3 (MMC-3) specification.
TestWrite
Indicates, when set to 1, that the device is capable of performing test writes. When set to zero, the device cannot perform test writes.
Reserved2
Reserved.
BufferUnderrunFree
Indicates, when set to 1, that the device can perform under-run-free recording.
Reserved3
Reserved.
Reserved4
Reserved.

Headers

Declared in ntddmmc.h. Include ntddmmc.h.

Comments

This structure holds data for the feature named "DVD-R Write" by the SCSI Multimedia - 3 (MMC-3) specification. Devices that support this feature can write data to a write-once DVD media in "Disc-at-Once" mode.
