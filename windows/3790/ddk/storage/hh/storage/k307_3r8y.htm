IOCTL_DISK_PERFORMANCE_OFF

Operation

The IOCTL_DISK_PERFORMANCE_OFF request disables the counters that were enabled by previous calls to IOCTL_DISK_PERFORMANCE. This request is available in Windows® XP and later operating systems. Caller must be running at IRQL = PASSIVE_LEVEL.

Input

None

Output

None

I/O Status Block

The Information member is set to zero. The Status member is set to STATUS_SUCCESS if the operation was successful. Otherwise, the Status member is set to the appropriate error code.

Headers

Defined in ntdddisk.h. Include ntdddisk.h
