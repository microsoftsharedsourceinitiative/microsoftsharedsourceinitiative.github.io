New for USB Drivers

Windows XP features new USB driver and stack architectures. For more information, see USB Driver Stack for Windows XP and Later and USB Device Stack for Windows XP and Later. In Windows XP, the generic parent facility formerly provided by the USB hub driver, usbhub.sys, is now provided by a separate generic parent driver, usbccgp.sys. For more information, see USB Common Class Generic Parent Driver. The new generic parent driver supports certain features of USB content security. For more information, see Content Security Features in the Composite Client Generic Parent Driver.
