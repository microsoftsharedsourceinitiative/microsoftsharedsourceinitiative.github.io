Windows DDK

New for Display and Print Device Drivers

Windows Server 2003 includes several new functions of interest to display driver writers, video miniport driver writers, and printer driver writers.

Display Driver and Video Miniport Driver Functions

EngBugCheckEx – Brings down the system in a controlled manner when the caller discovers an unrecoverable error.

VideoPortFlushRegistry – Flushes registry keys and values associated with the video miniport driver.

Printer Driver Functions

DrvSplDeviceCaps – Queries a printer for its capabilities.

GetJobAttributes – Gets information about a print job.

The Windows Server 2003 version of Unidrv introduces support for PCL XL color. For details, see PCL XL (PCL-6) Vector Graphics Support.
