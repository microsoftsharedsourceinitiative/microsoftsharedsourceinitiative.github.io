Windows DDK

New for Graphics Drivers

The following display-specific updates are new in Windows XP:

  • DualView can be enabled in video miniport drivers for adapters that can drive two or more different display devices at the same time. For more information, see Supporting DualView.
  • Microsoft® DirectDraw® and Direct3D® are DirectX® 8.0 Beta-compatible. For more information, see DirectX 8.0 Release Notes.
  • The video port driver provides support for bus-master DMA in video miniport drivers. For more information, see Video Port Driver Support for Bus-Master DMA.
  • GDI provides support for events in display drivers. For more information, see GDI Event Services.
  • The video port driver provides support for events in video miniport drivers. For more information, see Using Events in Video Miniport Drivers.
  • The video port driver provides support for spin locks in video miniport drivers. For more information, see Using Spin Locks in Video Miniport Drivers.
  • DirectX Video Acceleration is a specification for the hardware acceleration of digital video decoding processing. For more information, see DirectX Video Acceleration.

The following printer-specific updates are new in Windows XP:

  • Developers of Pscript printer minidrivers can now add support for two features: AddEuro and TrueGray.
  • Windows XP supports versioning in Unidrv and Pscript printer minidrivers, as well as in IHV monolithic printer drivers. This feature ensures that when users upgrade to newer operating system versions, the most recent printer files will be installed. For more information, see Specifying Printer Driver Version Numbers.
  • Several COM interfaces for printer UI and rendering plug-ins are new for Windows XP. The new interfaces for UI plug-ins include:

    IPrintOemUI2 COM Interface

    IPrintCoreUI2 COM Interface

    The new interfaces for rendering plug-ins include:

    IPrintOemPS2 COM Interface

    IPrintOemUni2 COM Interface

    IPrintCorePS2 COM Interface

  • The Microsoft Unidrv printer driver now supports the following vector graphics capabilities: HP-GL/2 monochrome (PCL-5e) and PCL XL monochrome (PCL-6 and pseudo-vector graphics). For more information, see Using Unidrv Vector Graphics Capabilities.
  • The installation of a port monitor no longer requires an older-style Windows NT® 3.1 INF file. Port monitor installation can now be done from within an ordinary printer driver INF file.
