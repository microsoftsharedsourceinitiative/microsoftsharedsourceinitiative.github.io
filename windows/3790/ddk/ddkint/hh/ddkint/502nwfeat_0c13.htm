Windows DDK

New for Network Drivers

Windows Server 2003 includes new support for offloading the processing of UDP-encapsulated ESP packets. The following two object identifiers (OIDs) have been added to support this function:

OID_TCP_TASK_IPSEC_ADD_UDPESP_SA

OID_TCP_TASK_IPSEC_DELETE_UDPESP_SA
